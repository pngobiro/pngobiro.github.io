2.

A. 0.406
B. 0.812
C. 2.22
D. 0.482
Show Answer

Correct Answer: D. 0.482 (closest to standard)

Note: The options provided in the source text seem to have typos or variations. The theoretical ripple factor for a full-wave bridge rectifier is $\gamma = \sqrt{(\frac{V_{rms}}{V_{dc}})^2 - 1} \approx 0.482$.

Standard value is 0.482. Option D is often cited as 0.48 or 0.482.

3.
Identify the primary function of a Diacs?
A. Pulse generators
B. Triggering device
C. Surge protection devices
D. Power thyristor
Show Answer

Correct Answer: B. Triggering device

A DIAC (Diode for Alternating Current) is a bidirectional trigger diode that conducts current only after its breakdown voltage has been exceeded. It is primarily used to trigger TRIACs in phase control circuits (like dimmers).

Holding current ($I_H$) is the minimum anode current required to maintain the SCR in the conducting state after the gate signal is removed. If current drops below this level, the SCR turns off.

10.

B. Immediately fall to zero

19.
What is the frequency of the sinusoidal signal that occupies five horizontal divisions and three vertical divisions on a Cathode Ray Oscilloscope for one complete cycle, if the time base is set at 1 millisecond per division?
A. 100 Hz
B. 2000 Hz
C. 1000 Hz
D. 200 Hz
Show Answer

Correct Answer: D. 200 Hz

Period ($T$) = Horizontal divisions $\times$ Time/div
$T = 5 \text{ div} \times 1 \text{ ms/div} = 5 \text{ ms} = 0.005 \text{ s}$.
Frequency ($f$) = $1/T = 1/0.005 = 200 \text{ Hz}$.

27.
The open-loop gain of an operational amplifier is $10^5$. An input signal of 1 millivolt is applied to the inverting input with the non-inverting input connected to the ground. The supply voltages are $\pm 10$ volt. Calculate the output of the amplifier?
A. +10 V
B. +100 V
C. -100 V
D. -10 V
Show Answer

Correct Answer: D. -10 V

Theoretical output = Gain $\times$ Diff Input.
$V_{out} = A_{ol} \times (V_+ - V_-) = 10^5 \times (0 - 0.001) = -100$ V.
However, the Op-Amp saturates at the supply rails ($\pm 10$ V). Thus, the output is limited to -10 V.

41.
Explain the construction of triac.
Show Answer

Answer:

A TRIAC (Triode for Alternating Current) construction features:

  • Structure: It is effectively two SCRs connected in inverse-parallel (back-to-back) with a common gate connection.
  • Layers: It is a five-layer device (n-p-n-p-n structure).
  • Terminals: It has three terminals: Main Terminal 1 (MT1), Main Terminal 2 (MT2), and Gate (G).
  • Operation: Unlike an SCR, it can conduct current in both directions when triggered by either a positive or negative gate pulse, making it a bilateral device suitable for AC power control.

44.
Explain the functions of the following microprocessor registers: (i) Stack pointer, (ii) Program counter.
Show Answer

Answer:

(i) Stack Pointer (SP): A specialized 16-bit register that stores the memory address of the top of the stack. It is automatically incremented or decremented during PUSH and POP operations, or subroutine calls, to manage temporary data storage and return addresses.

(ii) Program Counter (PC): A 16-bit register that holds the address of the next instruction to be fetched and executed from memory. After fetching an instruction, the PC is automatically incremented to point to the subsequent instruction.

Then expression becomes $(X+C)(X+D)$.

Applying the law: $X + CD$.

Substituting back: $F = AB + CD$.

46.
State five (5) reasons why universal logic gates are preferred in construction of digital electronics circuit.
Show Answer

Answer:

  1. Versatility: Universal gates (NAND and NOR) can be used to implement ANY Boolean function or other logic gate (AND, OR, NOT).
  2. Standardization: Using one type of gate reduces the variety of ICs required in inventory (Reduced Bill of Materials).
  3. Cost Effectiveness: Buying large quantities of a single chip type is cheaper than small quantities of mixed types.
  4. Simplification of Fabrication: In IC manufacturing (e.g., CMOS), NAND/NOR gates are often the fundamental building blocks and are easier/smaller to fabricate than AND/OR gates.
  5. Propagation Delay: Sometimes using universal gates can result in uniform propagation delays across the circuit.

48.
Define the following terms as used in memories: (i) Memory cell, (ii) Word size, (iii) Memory word, (iv) Volatile memory, (v) Address.
Show Answer

Answer:

  1. Memory Cell: The fundamental building block of computer memory capable of storing a single bit of information (0 or 1).
  2. Word Size: The number of bits that a CPU can process or transfer in a single operation (e.g., 8-bit, 16-bit, 32-bit).
  3. Memory Word: A group of bits (cells) stored at a specific address that are accessed together as a unit.
  4. Volatile Memory: Memory that requires power to maintain the stored information. Data is lost when power is turned off (e.g., RAM).
  5. Address: A unique binary number assigned to each memory location (word) allowing the CPU to locate and access specific data.

49.

(a) A $1K \times 8$ Bits Memory is Implemented using $256 \times 4$ Bits Memory chips.

i. Determine the number of chips and address lines required. (2.5 marks)

ii. Draw also the memory arrangement. (5 marks)

(b) Draw also the logic implementation of a full adder. (2.5 marks)

(c) Draw a full adder truth table and Compute the Boolean equation from the K-map. (5 marks)

Show Answer

Answer:

(a) Memory Implementation

Target: $1K \times 8$ (1024 words, 8 bits/word).
Available Chips: $256 \times 4$ (256 words, 4 bits/word).

i. Calculations:

  • Number of Chips:
    • Horizontal expansion (for bits): Need 8 bits, chip has 4. $8/4 = 2$ columns.
    • Vertical expansion (for addresses): Need 1K (1024), chip has 256. $1024/256 = 4$ rows.
    • Total Chips = $2 \text{ cols} \times 4 \text{ rows} = \mathbf{8 \text{ chips}}$.
  • Address Lines:
    • Total capacity $1K = 1024 = 2^{10}$. Total address lines needed = 10 lines ($A_0 - A_9$).
    • Chip capacity $256 = 2^8$. Chip address lines = 8 lines ($A_0 - A_7$).
    • Select lines: $10 - 8 = 2$ lines ($A_8, A_9$) used for decoder to select rows.

ii. Memory Arrangement (Description):

  • Arrange chips in 4 rows and 2 columns.
  • Connect Address lines $A_0-A_7$ to all 8 chips in parallel.
  • Connect Data inputs/outputs: Column 1 handles $D_0-D_3$, Column 2 handles $D_4-D_7$.
  • Use a 2-to-4 Decoder connected to $A_8$ and $A_9$.
    • Output $Y_0$ enables Row 1 chips (Address 0-255).
    • Output $Y_1$ enables Row 2 chips (Address 256-511).
    • Output $Y_2$ enables Row 3 chips (Address 512-767).
    • Output $Y_3$ enables Row 4 chips (Address 768-1023).

(b) Full Adder Logic Implementation

A Full Adder adds 3 bits (A, B, Cin) and produces Sum and Carry Out.

  • Sum: $S = A \oplus B \oplus C_{in}$ (Requires two XOR gates).
  • Carry: $C_{out} = AB + C_{in}(A \oplus B)$ (Requires AND and OR gates).
