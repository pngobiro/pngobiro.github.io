In 8086 microprocessors, the data bus and address bus are multiplied in order to?

A. 0.3125 V
B. 3.125 V
C. 0.78125 V
D. 1.5625 V
Show Answer

Correct Answer: D. 1.5625 V

A UJT has a resistance between Base 1 and Base 2 $R_{BB} = 10 K\Omega$ and Resistance of Base 2 $R_{B2} = 4 K\Omega$. Calculate its intrinsic stand-off ratio.
A. 0.6
B. 0.4
C. 2.5
D. 5/3
Show Answer

Correct Answer: A. 0.6

Calculation:
$R_{BB} = R_{B1} + R_{B2}$
$R_{B1} = R_{BB} - R_{B2} = 10K - 4K = 6K\Omega$.

41.

a) State three methods used to turn an SCR off.

b) Outline two multiplexing methods commonly used in fibre optics.

Show Answer

Answer:

(a) SCR Turn-off Methods (Commutation):

  1. Natural Commutation (AC Line Commutation): Occurs when the AC supply voltage reverses polarity, reducing anode current below holding current.
  2. Forced Commutation: Using external components (capacitors/inductors) to force current to zero. Examples: Class A (Self), Class B (Resonant), Class C (Complementary).
  3. Anode Current Interruption: Physically breaking the circuit or diverting current using a parallel switch.

(b) Fiber Optic Multiplexing Methods:

  1. Wavelength Division Multiplexing (WDM): Transmitting multiple signals at different wavelengths (colors) of light simultaneously over the same fiber.
  2. Time Division Multiplexing (TDM): Interleaving bits or bytes from different signals in time domains.
42.
Simplify the following Boolean Algebra equation: $$Y=\bar{A} B C+A \bar{B} C+A B \bar{C}+A B C$$
Show Answer

Answer:

Using the property $X + X = X$, we can duplicate the term $ABC$ to combine with others.

$Y = \bar{A}BC + ABC + A\bar{B}C + ABC + AB\bar{C} + ABC$

Group terms:

  • $\bar{A}BC + ABC = BC(\bar{A} + A) = BC$
  • $A\bar{B}C + ABC = AC(\bar{B} + B) = AC$
  • $AB\bar{C} + ABC = AB(\bar{C} + C) = AB$

Result: $Y = AB + BC + AC$ (This is the Carry output equation of a Full Adder).

43.
Using a NOR gate, implement an AND gate and a NAND gate.
Show Answer

Answer:

1. AND Gate using NOR:

Logic: $(A+A)' = A'$ (Invert A), $(B+B)' = B'$ (Invert B). Then NOR the inverted inputs: $((A')+(B'))' = ((AB)')' = AB$.

Requires 3 NOR gates: Two for inversion, one for combining.

2. NAND Gate using NOR:

Logic: Create AND as above, then invert the output.

Requires 4 NOR gates: 3 for AND, 1 for NOT.

48.
Calculate the overall system reliability for 1000 hours operating period if the mean time between failures for a power supply oscillators and amplifier are 100,000, 10,000 and 40,000 respectively.
Show Answer

Answer:

Assuming components are in series (system fails if any fails).

Failure rate $\lambda = \frac{1}{MTBF}$.

  • $\lambda_1 = 1/100,000 = 10^{-5}$
  • $\lambda_2 = 1/10,000 = 10 \times 10^{-5} = 10^{-4}$
  • $\lambda_3 = 1/40,000 = 2.5 \times 10^{-5}$

Total $\lambda_{sys} = \lambda_1 + \lambda_2 + \lambda_3 = 10^{-5} + 10 \times 10^{-5} + 2.5 \times 10^{-5} = 13.5 \times 10^{-5}$ failures/hour.

Reliability $R(t) = e^{-\lambda_{sys} t}$ for $t = 1000$.

$R(1000) = e^{-(13.5 \times 10^{-5}) \times 1000} = e^{-0.135}$.

$R(1000) \approx 0.8737$ or **87.37%**.

49.

a) Two memories are organized as follows: $64 K \times 8$ and $256 K \times 4$. Calculate:

i) The word size for each memory device.

ii) The number of locations for each memory device.

iii) The capacity in bits for each memory device.

b) With the aid of a labeled schematic block diagram of a ramp Analogue to Digital Converter and explain its operation.

Show Answer

Answer:

(a) Memory Calculations

Device 1: $64 K \times 8$

  • i) Word size: 8 bits (1 byte).
  • ii) Number of locations: $64K = 64 \times 1024 = 65,536$ locations.
  • iii) Capacity in bits: $65,536 \times 8 = 524,288$ bits (512 Kbits).

Device 2: $256 K \times 4$

  • i) Word size: 4 bits (1 nibble).
  • ii) Number of locations: $256K = 256 \times 1024 = 262,144$ locations.
  • iii) Capacity in bits: $262,144 \times 4 = 1,048,576$ bits (1 Mbit).

(b) Ramp (Single Slope) ADC

Block Diagram Components:

  • Comparator (Op-Amp)
  • AND Gate
  • Counter (Binary Counter)
  • Ramp Generator (Integrator or DAC driven by counter)
  • Clock Generator
  • Latch/Register (Output)

Operation:

  1. Start: A "Start Conversion" pulse resets the counter to zero.
  2. Ramp Generation: A linear ramp voltage starts rising from 0V. Alternatively, a DAC connected to the counter generates a staircase ramp.
  3. Comparison: The analog input voltage ($V_{in}$) is compared with the ramp voltage ($V_{ramp}$).
    • As long as $V_{in} > V_{ramp}$, the comparator output is High.
    • This enables the AND gate, allowing clock pulses to pass to the counter.
  4. Counting: The counter counts up, representing the digital value corresponding to the time/voltage.
  5. Stop: When $V_{ramp}$ exceeds $V_{in}$, the comparator output goes Low.
    • This disables the AND gate, stopping the clock.
    • The counter stops counting.
  6. Output: The final count stored in the counter is proportional to the input voltage ($V_{in}$) and is latched as the digital output.
