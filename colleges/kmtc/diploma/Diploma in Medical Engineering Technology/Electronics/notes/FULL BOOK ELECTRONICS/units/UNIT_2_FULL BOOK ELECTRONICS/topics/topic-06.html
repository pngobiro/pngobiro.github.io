üìä Experimental Setup

The static characteristics of an NPN transistor connected in common-Collector configuration can be determined by using the circuit shown in Fig. 2.24.

  • Ammeters are included in series with the base and emitter circuits to measure $\mathrm{I_B}$ and $\mathrm{I_E}$.
  • Voltmeters are connected across B and C to measure voltage $\mathrm{V_{CB}}$ and across E and C to measure $\mathrm{V_{CE}}$.
  • The two variable dc supplies are used for biasing. The collector terminal is common to both the input (CB) and output (CE) circuits.

üì• Input Characteristics

It is a graph of input base current $\mathrm{I_B}$ versus input voltage $\mathrm{V_{CB}}$, when $\mathrm{V_{CE}}$ is constant.

Voltage Relationship

From Fig. 2.24, we can deduce the relationship between the voltages:

$$\mathrm{V_{CB}} = \mathrm{V_{CE}} - \mathrm{V_{BE}}$$

As $\mathrm{V_{CB}}$ is increased, $\mathrm{V_{BE}}$ is reduced, thereby reducing $\mathrm{I_B}$. It is also found that as $\mathrm{V_{CB}}$ increases, $\mathrm{I_B}$ is decreased. This relationship is shown in Fig. 2.25.

Similarity to CE

üîÑ Current Transfer Characteristic

It indicates how $\mathrm{I_E}$ varies with changes in $\mathrm{I_B}$ when $\mathrm{V_{EC}}$ is kept constant at a given value.

The current gain of the transistor in CC configuration is:

$$\gamma = \frac{I_{\mathrm{E}}}{I_{\mathrm{B}}}$$

Key Points

  • ‚úì

    Common Collector (CC) configuration has high input resistance and low output resistance.

  • ‚úì

    CC configuration provides high current gain ($\gamma$) but voltage gain less than unity.

  • ‚úì

    CC is primarily used for impedance matching applications.

  • ‚úì

    Common Emitter (CE) is the only configuration providing both high current and voltage gain, making it ideal for amplification.
