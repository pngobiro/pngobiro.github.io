Problem 1

A variation of $0.02\mathrm{V}$ across a $3.8\mathrm{V}$ Zener produces a current change from $20\mathrm{mA}$ to $22\mathrm{mA}$. Calculate the dynamic resistance of the device?

‚ñ∂ View Solution

Given: The change in Zener voltage $= \Delta V_{Z} = 0.02\mathrm{V}$

The change in Zener current $= \Delta I_{Z} = 22 - 20 = 2\mathrm{mA}$

The dynamic resistance:

$$\mathrm{r_Z} = \Delta \mathrm{V_Z} / \Delta \mathrm{I_Z} = 0.02 / 2 \times 10^{-3} = 10\Omega$$

Problem 2

A $4.7\mathrm{V}$ Zener has a resistance of $15\Omega$. What is the terminal voltage, when the current is $20\mathrm{mA}$?

‚ñ∂ View Solution

Given: $\mathrm{V_Z} = 4.7$ volts; $\mathrm{r_Z} = 15\Omega$ and $\mathrm{I_Z} = 20\mathrm{mA} = 20\mathrm{X}10^{-3}\mathrm{A}$.

We know that the terminal voltages of a Zener diode,

$$\mathrm {Then } \mathrm {V} _ {\mathrm {Z}} = \left(\mathrm {V} _ {\mathrm {Z}} + \mathrm {I} _ {\mathrm {Z}}. \mathrm {r} _ {\mathrm {Z}}\right) = 4. 7 + (2 0 \times 1 0 ^ {- 3}) \times 1 5 = 5 \mathrm {V}.$$

Problem 3

The input of a Half-wave rectifier is $V = 200 \sin 40 t$. If $R_{L} = 1K\Omega$, and forward resistance of the diode is $30 \Omega$, find (i) The dc current through the diode, (ii) The ac value of current in the circuit, (iii) dc output voltage, (iv) dc power input, (v) Rectifier efficiency?

‚ñ∂ View Solution

Given: $\mathrm{V_i} = 200 \sin 40 \, \mathrm{t}$ (i) then $\mathrm{V_m} = 200 \, \mathrm{V}$,

$$\mathrm{I_m} = \mathrm{V_m} / \mathrm{R_f} + \mathrm{R_L} = 200 / 1030 = 194 \, \mathrm{mA}$$

(i) $\mathrm{I_{dc}} = \mathrm{I_m} / \pi = 194 / \pi = 62.8 \, \mathrm{mA}$

(ii) $\mathrm{I_{rms}} = \mathrm{I_m} / 2 = 194 / 2 = 97 \, \mathrm{mA}$

(iii) Output dc voltage $= I_{dc} \times R_L = 62.8 \times 10^{-3} \times 1 \times 10^{+3} = 62.8 \, \mathrm{V}$

(iv) Input ac Power $= P_{ac} = (I_{rms})^2 \times R_F + R_L = 0.097 \times 0.097 \times 1030 = 9.69 \, \mathrm{W}$

(v) Output dc Power $= P_{dc} = (I_{dc})^2 \times R_L = (0.0628)^2 \times 1 \times 10^{+3} = 3.94 \, \mathrm{W}$

(vi) Rectifier Efficiency $= \eta = (P_{dc} / P_{ac}) \times 100 = 3.94 / 9.69 \times 100 = 40.66$

Problem 4

In a Half-Wave Rectifier, the input is $300 \sin 314 t$. Find its average output voltage. (VTU 99)

‚ñ∂ View Solution

Given: $\mathrm{V_m} = 300 \, \mathrm{V}$

Average output voltage $= V_{dc} = V_m / \pi = 300 / 3.14 = 95.5 \, \mathrm{V}$

Problem 5

In a Centre-tap Full-Wave-Rectifier, the load resistance is $1.5 \, \mathrm{K}\Omega$. Each has a forward resistance of $6\,\Omega$. The voltage across half the secondary winding is $220 \, \mathrm{V}$. Find the Ripple Factor and Rectifier Efficiency.

‚ñ∂ View Solution

Given: $\mathrm{V_{rms}} = 220 \, \mathrm{V}$

(i) Peak value of Voltage $= V_m = \sqrt{2} \times V_{rms} = \sqrt{2} \times 220 = 311 \, \mathrm{V}$

(ii) Peak value of load current $I_m = V_m / R_f + R_L = 311 / (6 + 1500) = 0.207 \, \mathrm{A}$

(iii) DC load current $I_{dc} = 2I_m / \pi = 2 \times 0.207 / 3.14 = 0.132 \, \mathrm{A}$

(iv) RMS value of load current $I_{rms} = I_m / \sqrt{2} = 0.207 / \sqrt{2} = 0.146 \, \mathrm{A}$

(v) Ripple Factor $= \gamma = \sqrt{\left(\frac{I_{rms}}{I_{dc}}\right)^2 - 1} = \sqrt{\left(\frac{0.146}{0.132}\right)^2 - 1} = 0.473$

(vi) Rectification Efficiency, $\eta = (P_{dc} / P_{ac}) = (I_{dc}^2 \times R_L) / (I_{rms}^2 (R_F + R_L))$

$$= \left[ (0.132)^2 \times 1500 \right] / (0.146)^2 \times (6 + 1500) = 0.814 \quad \text{or} \quad 81.4\%$$

Problem 6

In a Bridge Rectifier the input is from $230 \, \mathrm{V}$, $50 \, \mathrm{Hz}$ mains. Calculate D.C. output voltage.

‚ñ∂ View Solution

Given: Input Voltage $= 230 \, \mathrm{V}$ (r.m.s)

Peak value of voltage $V_m = \sqrt{2} \times r.m.s = \sqrt{2} \times 230 = 325 \, \mathrm{V}$

$\therefore$ D.C. Output Voltage $V_{dc} = 2V_m / \pi = (2 \times 325) / \pi = 207 \, \mathrm{V}$

Problem 7

If a capacitor of $1000\ \mu\mathrm{F}$ is used as a filter in (a), calculate the Ripple Factor and (b) D.C. Output. (March 99, V.T.U.)

‚ñ∂ View Solution

Given: $C = 1000\ \mu\mathrm{F}$

Taking Load Resistance $R_L = 100\ \mathrm{ohms}$

(a) Ripple Factor $\gamma = 1 / (4\sqrt{3}\mathrm{fCR_L})$

$$= 1 / (4 \times \sqrt{3} \times 50 \times 1000 \times 10^{-6} \times 100) = 0.028$$

(b) D.C. Output

$$V_{dc} = \left(\frac{4\ \mathrm{f}\ R_L\ \mathrm{C}}{4\ \mathrm{f}\ R_L\ \mathrm{C} + 1}\right) V_m$$

Now, $4\mathrm{fCR_L} = (4 \times 50 \times 100 \times 1000 \times 10^{-6}) = 20$

$\therefore V_{dc} = (20/21) \cdot V_m = (20/21) \cdot 325 = 309.5\ \mathrm{V}.$

Problem 8

Calculate the value of capacitor has to be used for the capacitor filter of a full wave rectifier to get a ripple of $0.01\%$. The rectifier supplies a load of $3\ \mathrm{K}\Omega$ while the supply frequency is $50\ \mathrm{Hz}$.

‚ñ∂ View Solution

Given: $\gamma = 0.01\%$, $R_L = 3\ \mathrm{K}\Omega$, and $f = 50\ \mathrm{Hz}$

For a capacitor filter in full wave rectifier circuit $\gamma = 1 / (4\sqrt{3}\ \mathrm{f}\ \mathrm{CR_L}) \times 100\%$

$$0.01 = 1 / (4\sqrt{3} \times 50 \times \mathrm{C} \times 3000) \times 100$$

or $C = 1.39 \times 10^{-5}\ \mathrm{F} = 13.94\ \mu\mathrm{F}$

Problem 9

In a Full Wave Rectifier with a capacitor filter, the load current from the circuit operating from $230\ \mathrm{V}$, $50\ \mathrm{Hz}$ supply is $10\ \mathrm{mA}$. Estimate the value of capacitor required to keep the ripple factor less than $1\%$. (VTU July 2004)

‚ñ∂ View Solution

Given: $V_2(\mathrm{rms}) = 230\ \mathrm{V}, \quad f = 50\ \mathrm{Hz}, \quad I_L = 10\ \mathrm{mA}, \quad \gamma = 1\% = 0.01$

Then $V_{rm} = \sqrt{2} \cdot V_2(\mathrm{rms}) = \sqrt{2} \times 230 = 325.269\ \mathrm{V}$

$V_{dc} = 2 V_{rm} / \pi$ for full wave rectifier or $I_{dc} = 2 V_{rm} / \pi R_L$

or $R_L = 2 V_{rm} / \pi I_L = (2 \times 325.269) / (\pi \times 10 \times 10^{-3}) = 20.7\ \mathrm{K}\Omega$

Then the value of Capacitor:

$$\gamma = 1 / (4\sqrt{3}\ \mathrm{f}\ \mathrm{CR_L})$$
$$0.01 = 1 / (4\sqrt{3} \times 50 \times \mathrm{C} \times 20.7 \times 10^{-3}) \times 100$$

or $C = 1.39 \times 10^{-5}\ \mathrm{F} = 13.94\ \mu\mathrm{F}$

Problem 10

The output voltage of a d.c. power supply system varies from $40\mathrm{V}$ to $39.6\mathrm{V}$ when the load current is raised from zero to its maximum specified level. The voltage also increases to $40.5\mathrm{V}$ when the a.c. supply increases by $10\%$. Calculate the load and source effects, and the load and line regulation.

‚ñ∂ View Solution

(1) Load effect $= \Delta V_{o}$ for $\Delta I_{L(\max)}$

$$= 40 - 39.6 = 0.4\mathrm{V} = 400\mathrm{mV}$$

(2) Load regulation $= \left[\left(\Delta V_{o} \text{ for } \Delta I_{L(\max)}\right) / V_{o}\right] \times 100 = [400\mathrm{mV} / 40] \times 100 = 1.0\%$

(3) Source effect $= \Delta V_{o}$ for $10\%$ change in $V_{0}$

$$= 40.5 - 40 = 0.5\mathrm{V} = 500\mathrm{mV}$$

(4) Line regulation $= (\Delta V_{o}$ for $10\%$ change in $V_{0}) / V_{o} \times 100$

$$= (500\mathrm{mV} / 40) \times 100 = 1.25\%$$

Problem 11

For a Zener regulator shown in Fig. 1.33, calculate the range of input voltage for which output remain constant? Given: $V_{Z} = 6.1\mathrm{V}$, $I_{Zmin} = 2.5\mathrm{mA}$, $I_{Zmax} = 25\mathrm{mA}$, $r_{Z} = 0\,\Omega$

‚ñ∂ View Solution

Given: $R_{L} = 1\,\mathrm{k}\Omega, \ V_{Z} = 6.1\,\mathrm{V}$

Then

$$I_{L} = \frac{V_{Z}}{R_{L}} = \frac{6.1}{1000} = 6.1\,\mathrm{mA} \quad \text{(is always constant)}$$

For $V_{in(min)}$:

$I_{Z} = I_{Zmin} = 2.5\,\mathrm{mA}$

$I = I_{Zmin} + I_L = 2.5\,\mathrm{mA} + 6.1\,\mathrm{mA} = 8.6\,\mathrm{mA}$

$V_{in(min)} = V_Z + I \cdot R_S$

$$= 6.1 + 8.6 \times 10^{-3} \times 2.2 \times 10^3 = 6.1 + 18.92 = 25.02\mathrm{V}$$

For $V_{in(max)}$:

$I_{Z} = I_{Zmax} = 25\,\mathrm{mA}$

$I = I_{Zmax} + I_L = 25\,\mathrm{mA} + 6.1\,\mathrm{mA} = 31.1\,\mathrm{mA}$

$V_{in(max)} = V_Z + I \cdot R_S$

$$= 6.1 + 31.1 \times 10^{-3} \times 2.2 \times 10^3 = 6.1 + 68.42 = 74.52\mathrm{V}$$

Thus the range of input voltage is $25.02\mathrm{V}$ to $74.52\mathrm{V}$, for which the output will be constant.

Problem 12

Design a zener voltage regulator for the following specifications: Output voltage $V_{0} = 5V$, Input voltage $V_{i} = 12 \pm 3V$, $I_{Zmin} = 10 \, \text{mA}$, Load current $I_{L} = 20 \, \text{mA}$, and Zener wattage $P_{Z} = 500 \, \text{mW}$.

‚ñ∂ View Solution

Given: $V_{\text{in(min)}} = 12 - 3 = 9\mathrm{V}$, $V_{\text{in(max)}} = 12 + 3 = 15\mathrm{V}$, $I_L = 20\mathrm{mA}$ (should be constant), $V_O = V_Z = 5\mathrm{V}$, $P_Z = 500\mathrm{mW}$.

(i) The maximum power dissipation is corresponding to $I_{Z\max}$.

$$P_{Z}(\text{given}) = V_{Z} I_{Z\max} \quad \Rightarrow \quad 500 \times 10^{-3} = 5 \times I_{Z\max}$$

$I_{Z\max} = 100 \, \text{mA}$

(ii) Since $I_L = \text{constant}$, For $V_{\text{in(max)}}$, $I_Z = I_{Z\max}$

$$I = I_L + I_{Z\max} = 20 + 100 = 120 \, \text{mA}$$

and $V_{\text{in(max)}} = V_Z + I R_{\min}$

$$R_{\min} = (15 - 5) / (120 \times 10^{-3}) = 83.33 \, \Omega$$

When $R = R_{\min}$, for $V_{\text{in(max)}}$ = the current I will be at its maximum i.e., $I_Z$ will be at its maximum.

(iii) To calculate $R_{\max}$, I must be minimum i.e., $I_Z$ must be minimum $= 10 \, \text{mA}$.

$$I = I_L + I_{Z\min} = 20 + 10 = 30 \, \text{mA}$$
$$V_{\text{in(min)}} = V_Z + I \times R_{\max}$$
$$R_{\max} = (9 - 5) / (30 \times 10^{-3}) = 133.33 \, \Omega$$

Thus R must be greater than $83.33 \, \Omega$ and less than $133.33 \, \Omega$ for proper regulating action.

Problem 15

A 24 V, 600 mW Zener diode is used for providing a 24 V stabilized supply to a variable load. If the input voltage is 32 V, calculate (i) The value of series resistance required. (ii) Diode current when the load is 1200 Œ©. (VTU 2003)

‚ñ∂ View Solution

Given: $V_Z = 24V, P_Z = 600mW, V_{in} = 32V$.

Max Zener Current $I_{Zmax} = P_Z / V_Z = 0.6 / 24 = 25 \text{mA}$.

(i) Series Resistance $R_S$: To ensure safe operation at no load ($I_L=0$), $I_S = I_{Zmax}$.

$R_S = (V_{in} - V_Z) / I_{Zmax} = (32 - 24) / 0.025 = 8 / 0.025 = 320 \Omega$.

(ii) Load Current $I_L = V_Z / R_L = 24 / 1200 = 20 \text{mA}$.

Total current $I_T = (V_{in} - V_Z) / R_S = (32 - 24) / 320 = 25 \text{mA}$.

Diode Current $I_Z = I_T - I_L = 25 - 20 = 5 \text{mA}$.
