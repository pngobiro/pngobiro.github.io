1.
In 8086 microprocessors, the data bus and address bus are multiplied in order to?
A. Increase the speed of a microprocessor
B. Reduce the number of pins
C. Connect more peripheral chips
D. To increase the efficiency of a microprocessor
Show Answer

Correct Answer: B. Reduce the number of pins

Explanation: Multiplexing the data and address buses (e.g., AD0-AD15) allows the same physical pins to carry address information at one time and data at another, thereby reducing the total pin count of the IC package.

2.
What is resolution as used in digital to analog converter?
A. It is the comparison between the actual output of the converter and its expected output
B. It is the deviation between the ideal straight-line output and the output of the converter
C. It is the smallest analog output increment in the digital input
D. It is the ability to resolve between forward and reverse steps when sequenced over its entire range
Show Answer

Correct Answer: C. It is the smallest analog output increment in the digital input

Explanation: Resolution refers to the smallest change in voltage (or current) that the DAC can produce, corresponding to a change in the Least Significant Bit (LSB). Ideally $V_{ref} / 2^n$.

3.
The basic building blocks of the arithmetic unit in a digital computer are?
A. Subtractors

Explanation: Adders (Half Adders and Full Adders) are the fundamental circuits used to perform addition. Subtraction, multiplication, and division are typically derived from addition operations (e.g., using 2's complement for subtraction).

8.
The cycle required to fetch and execute an instruction in an 8086 microprocessor is referred to as a
A. Clock cycle
B. Memory cycle
C. Machine cycle

Correct Answer: D. Instruction cycle

Correct Answer: D. 3.125V (Check Calculation)

Calculation:
Input 0101 = $5_{10}$.
Max value ($2^4 - 1$) = 15.
$V_{out} = V_{ref} \times \frac{\text{Decimal Input}}{2^n} = 5 \times \frac{5}{16} = 1.5625V$.
Wait, common formula variations exist. Let's check options.
Option A: 5/16 = 0.3125 (This is 1 LSB, code 0001).
If code is 0101 (5), output should be $5 \times 0.3125 = 1.5625V$.
Option B/D are 3.125V. $3.125 / 0.3125 = 10$. Code 1010.
If logic implies 5V is max scale (code 1111), then 5 * 5/15 = 1.66V.
Re-evaluation: Maybe code is 1010? 3.125V corresponds to $10/16 * 5$.
Or maybe $V_{ref}$ is 10V? No, question says 5V.

For 0101 (5): $5 * 5 / 16 = 1.5625V$. This isn't an option.
Maybe the question assumes code 1010 (Decimal 10)? $10/16 * 5 = 3.125V$.

12.
A Schmitt trigger can be used as a __________.
A. Square wave generator
B. Comparator
C. Both square wave generator and comparator
D. Sawtooth generator
Show Answer

Correct Answer: C. Both square wave generator and comparator

Explanation: A Schmitt trigger is a comparator with hysteresis. It is used to square up sine waves (Comparator function) and, when combined with an RC network, acts as a Square Wave Generator (Relaxation Oscillator).

15.
After firing an SCR, the gating pulse is removed. The current will ___________.
A. remain the same
B. Immediately fall to zero
C. Rise up
D. Rise a little and then fall to zero
Show Answer

Correct Answer: A. remain the same

Explanation: The SCR is a latching device. Once it is triggered (fired) into conduction, the gate loses control. The anode current will continue to flow as long as it is above the Holding Current, even if the gate pulse is removed.

17.
The average time to repair any fault in a system is two hours. Calculate the value of maintainability for a time of four hours?
A. 0.02
B. 2
C. 0.135
D. 0.865
Show Answer

Correct Answer: D. 0.865

Explanation: Maintainability $M(t)$ is often modeled as $M(t) = 1 - e^{-t/MTTR}$.
Here, $MTTR = 2$ hours, $t = 4$ hours.
$M(4) = 1 - e^{-4/2} = 1 - e^{-2}$.
$e^{-2} \approx 0.135$.
$M(4) = 1 - 0.135 = 0.865$.
This represents an 86.5% probability that the system will be repaired within 4 hours.

Show Answer

a) SCR Turn-off Methods (Commutation)

  1. Natural Commutation: When the AC supply voltage reverses polarity (Class F).
  2. Forced Commutation: Using an external circuit to force current to zero (e.g., Class B - Resonant Pulse).
  3. Anode Current Interruption: Physically breaking the circuit (Switch).

  1. Immunity to EMI/RFI: Not affected by electromagnetic interference.
  2. High Bandwidth: Can carry vast amounts of data at high speeds.
  3. Low Attenuation: Signals can travel long distances without repeaters.
42.
Simplify the following Boolean Algebra equation (5 marks)
$$Y = \bar{A}BC + A\bar{B}C + AB\bar{C} + ABC$$
Show Answer

Step 1: Group terms with ABC
$Y = \bar{A}BC + A\bar{B}C + AB\bar{C} + ABC + ABC + ABC$ (Idempotent law: X+X=X)
Group 1: $\bar{A}BC + ABC = BC(\bar{A}+A) = BC(1) = BC$
Group 2: $A\bar{B}C + ABC = AC(\bar{B}+B) = AC$
Group 3: $AB\bar{C} + ABC = AB(\bar{C}+C) = AB$

Result: $Y = AB + BC + AC$

49.

a) Two memories are organized as follows: $64K \times 8$ and $256K \times 4$. Calculate:
i) The word size for each memory device. (2 marks)
ii) The number of locations for each memory device. (2 marks)
iii) The capacity in bits for each memory device. (4 marks)

b) With the aid of a labeled schematic block diagram of a ramp Analogue to Digital Converter and explain its operation. (12 marks)

Show Answer

a) Memory Calculations

  • i) Word Size:
    • Memory A ($64K \times 8$): 8 bits
    • Memory B ($256K \times 4$): 4 bits
  • ii) Number of Locations:
    • Memory A: 64K = $64 \times 1024 = 65,536$ locations.
    • Memory B: 256K = $256 \times 1024 = 262,144$ locations.
  • iii) Capacity in Bits:
    • Memory A: $65,536 \times 8 = 524,288$ bits (512 Kbits).
    • Memory B: $262,144 \times 4 = 1,048,576$ bits (1 Mbit).

b) Ramp ADC (Single Slope)

Block Diagram: Contains a Ramp Generator, Comparator, Control Logic, Counter, and Clock.

      Vin (+)
        |
        +----[ Comparator ]<-------+
                 |                 |
                 v                 |
Start ----> [ Control Logic ]      |
                 |      ^          |
       Clock ----+      |          |
                 v      |          |
            [ Binary Counter ]     |
                 |                 |
                 v                 |
            [     DAC    ]---------+
                 |
            Digital Out

Operation:

  1. A start pulse resets the counter to 0.
  2. The DAC (or ramp generator) produces a voltage proportional to the counter value (a stepping ramp).
  3. This ramp voltage ($V_{dac}$) is compared to the analog input $V_{in}$.
  4. As long as $V_{dac} < V_{in}$, the comparator output allows the clock pulses to increment the counter.
  5. When $V_{dac}$ exceeds $V_{in}$, the comparator switches state, stopping the clock.
  6. The value currently in the counter is the digital representation of $V_{in}$.
