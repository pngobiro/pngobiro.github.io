INTRODUCTION TO POST-MORTEM

A post-mortem examination, also known as an autopsy, is the examination of a body after death. The aim of a post-mortem is to determine the cause of death. Post-mortems are carried out by pathologists. Post-mortems provide useful information about how, when and why someone died. They enable pathologists to obtain a better understanding of how diseases spread. A post-mortem examination will be carried out if it has been requested by a coroner because the cause of death is unknown, or following a sudden, violent or unexpected death. Or a hospital doctor-to find out more about an illness or the cause of death, or to further medical research and understanding. There are two different types of post-mortem.

i. Coroner's post-mortem examination
A coroner is a judicial officer responsible for investigating deaths in certain situations. The main aim of a post-mortem requested by a coroner is to find out how someone died and decide whether an inquest is needed.

ii. Hospital post-mortem examination
These post-mortems are requested by hospital doctors to provide more information about an illness or the cause of death, or to further medical research.

In conclusion after a post- mortem, the pathologist writes a report of the findings. If the postmortem was requested by the coroner, the coroner or coroner's officer will let you know the cause of death determined by the pathologist. If you want a full copy of the pathologist's report, you can request this from the coroner's office, but there may be a fee.

Access Database Procedures:

1. Database Creation:

  • Open Access > Blank Database.
  • Filename: Research Data 2023. Location: Your Desktop Folder. Click Create.

2. Table Creation (registered samples):

  • Create > Table Design.
  • Fields:
    • County (Short Text) - Primary Key (Right-click field > Primary Key).
    • Surname (Short Text)
    • Other Name (Short Text)
    • Age (Number)
    • Sex (Short Text)
    • Specimen (Short Text)
    • Results (Short Text)
  • Save as registered samples. Go to Datasheet View and type the data.

3. Import Excel:

  • External Data Tab > New Data Source > From File > Excel.
  • Browse to COUNTYMPS...xlsx.
  • Select "Import the source data into a new table".
  • Select Worksheet: Original data.
  • Check "First Row Contains Column Headings".
  • Primary Key: Choose "Choose my own primary key" > Select County.
  • Finish Import.

4. Queries (Design View / SQL):

a) Total negatives and positives result MPS basis sex:

  • Create > Query Design > Add registered samples.
  • Add fields: Sex, Results.
  • Click Totals (Sigma Σ icon) in ribbon.
  • "Group By" Sex and Results.
  • Add Results again (or Count field), change "Group By" to Count.
  • SQL View:
    SELECT Sex, Results, Count(*) AS Total FROM [registered samples] GROUP BY Sex, Results;

b) Total positives/negatives aged 15+ (Adults):

  • Create > Query Design.
  • Add Results.
  • Add Age. Under Criteria for Age, type >=15.
  • Click Totals.
  • Results: Group By. Age: Where (or just leave grouped if counting records).
  • Add Count field.
  • SQL View:
    SELECT Results, Count(*) AS Total FROM [registered samples] WHERE Age >= 15 GROUP BY Results;

5. Report:

  • Select the query created in step (b).
  • Create > Report.
  • Title: MPS RESEARCH REPORT 2023.
  • Add a Label control: "Interpretation: The data shows X positives among adults..."
  • Save.
