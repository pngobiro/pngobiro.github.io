Step-by-step MS Access Procedures:

1. Database Setup:

  • Open Access > Blank Database.
  • Filename: clinic database 001 (replace 001). Save to Desktop.

2. Table Creation:

  • Create > Table Design.
  • Enter Field Names and Data Types exactly as listed.
  • Properties:
    • ID Number: Field Size = 10.
    • Name/Surname: Field Size = 15.
    • Telephone: Data Type = Number, Field Size = Long Integer.
    • D.O.B: Format property = Medium Date.
    • Payment method: Field Size = 20.
    • Nationality: Field Size = 25.
  • Primary Key: Right-click ID Number > Primary Key.
  • Save table as PATIENT TABLE.

3. Lookup Wizard & Data Entry:

  • Lookup Wizard: In Design View, click Payment method. Under Data Type, select "Lookup Wizard". Choose "I will type in the values that I want". Enter: NHIF, Cash, Credit, Insurance. Finish.
  • Go to Datasheet View. Enter the 5 records provided.

4. Formatting & Arrangement:

  • Sort: Click dropdown arrow on Surname column > Sort A to Z.
  • Move Columns: Click column header D.O.B to select it. Click and drag it to the left, dropping it immediately after Surname. Do the same for Telephone if needed to adjust position.

5. Form Creation:

  • Select PATIENT TABLE. Create > Form.
  • Save as PATIENTS ENTRIES.
  • Insert Picture: Open Form in Design View. Design Tab > Insert Image > Browse > Select any image file > Draw box to place it.

6. Report Creation:

  • Create > Report Wizard.
  • Select Table: PATIENT TABLE.
  • Select Fields: ID Number, Name, Telephone number, Payment method.
  • Finish.

7. Make Table Query (PAYMENT TABLE):

  • Create > Query Design > Add Patient Table.
  • Add Fields: ID Number, Surname, Payment method, NHIF.
  • In Ribbon, change query type to Make Table.
  • Table Name: PAYMENT TABLE.
  • Run the query (Red Exclamation Mark). Click Yes to paste rows.
  • Save query (optional, e.g., MakePaymentTable).

8. Age Query:

  • Create > Query Design > Add Patient Table.
  • Add all fields or just relevant ones.
  • Under D.O.B criteria: Type > #31/12/1972# (Assuming "after 1972" means from Jan 1, 1973).
  • Save as AGE QUERY.

9. Macro:

  • Create > Macro.
  • Action: QuitAccess.
  • Save as ExitMacro.

Section 2: Web Design

10 Marks
2.

You have been identified to create a webpage to help in COVID-19 vaccine distribution for Kitui County.

Generate a HTML code for the following output:

KITUI COUNTY GOVERNMENT

COVID - 19 TRACKING

ENTER YOUR DETAILS FOR QUICK RESPONSE

Name:

ID No:

Location:

Sub location:

Telephone No:

Show HTML Code

Copy the following code into Notepad and save as covid_tracking.html:

<!DOCTYPE html>
<html>
<head>
    <title>Kitui County COVID-19 Tracking</title>
</head>
<body>

    <!-- Centered Main Heading -->
    <h1 align="center">KITUI COUNTY GOVERNMENT</h1>

    <!-- Centered Sub-heading -->
    <h2 align="center">COVID - 19 TRACKING</h2>

    <hr>

    <!-- Section Heading -->
    <h3>ENTER YOUR DETAILS FOR QUICK RESPONSE</h3>

    <!-- Data Entry Form -->
    <form>
        <p>
            <label for="name">Name:</label>
            <input type="text" id="name" name="name">
        </p>
        
        <p>
            <label for="idno">ID No:</label>
            <input type="text" id="idno" name="idno">
        </p>
        
        <p>
            <label for="location">Location:</label>
            <input type="text" id="location" name="location">
        </p>
        
        <p>
            <label for="subloc">Sub location:</label>
            <input type="text" id="subloc" name="subloc">
        </p>
        
        <p>
            <label for="tel">Telephone No:</label>
            <input type="number" id="tel" name="tel">
        </p>
        
        <p>
            <input type="submit" value="Submit">
            <input type="reset" value="Clear">
        </p>
    </form>

</body>
</html>
