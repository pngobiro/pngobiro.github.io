1.
In Ms Access, how can you crate relationship between two tables in the GUI?
A. Drag the foreign key of s table into the primary key if another table
B. Drag any field from parent table and drop on child table
C. Drag the primary key if a table into foreign key of another table
D. Any of these can be done to create relationship
Show Answer

Correct Answer: C. Drag the primary key if a table into foreign key of another table

The standard way to create a relationship is to drag the Primary Key from the parent table and drop it onto the Foreign Key in the child table.
