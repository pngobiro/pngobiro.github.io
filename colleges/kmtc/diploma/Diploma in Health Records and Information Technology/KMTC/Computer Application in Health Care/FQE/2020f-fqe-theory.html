Section 1: Multiple Choice Questions

30 Marks
1.
In database design, the stage where one gathers and lists all necessary fields for the database project is
a) Data depilation
b) Data refinement
c) Establishing relationship
d) None of the above
Show Answer

Correct Answer: a) Data definition (likely typo "depilation" in source) or d) None of the above

This phase is typically called "Requirements Gathering" or "Data Definition". Given the typo "depilation", "None of the above" might be the intended strict answer, but contextually it refers to defining data requirements.

2.
Which of the following fields in database has a width of 8 bytes?
a) Memo
b) Name
c) Number
d) Data/time
Show Answer

Correct Answer: d) Date/time

In MS Access, the Date/Time data type uses 8 bytes. "Number" size varies (Byte, Integer, Long Integer, Double), but Double is 8 bytes.

3.
A ____ name must be unique within a database
a) Record
b) Character
c) Table
d) Character
Show Answer

Correct Answer: c) Table

Table names must be unique within the same database.

4.
The key that uniquely identifies each record in database is called
a) Field name
b) Primary key
c) Candidate key
d) Secondary key
Show Answer

Correct Answer: b) Primary key

5.
One to many relationships between student and subject tables means
a) One student can have many subjects
b) One subject can have more students
c) Many subjects can have many students
d) Many students can have many subjects
Show Answer

Correct Answer: a) One student can have many subjects (or b depending on direction)

Usually, Student-Subject is Many-to-Many. In a 1:M context, it implies one entity relates to many of the other. Option A is the standard interpretation of "One (Student) to Many (Subjects)".

6.
A part of a database that stores only one type of data is ____
a) Record
b) Field
c) Query
d) Report
Show Answer

Correct Answer: b) Field

A field (column) holds a specific type of data (e.g., just dates, just names).

7.
Which of the following may not be included in a database?
a) Data presented in a table in MS Word document
b) Data in MS Excel spreadsheet
c) A PowerPoint presentation
d) A telephone directory
Show Answer

Correct Answer: c) A PowerPoint presentation

While modern DBs can store BLOBs (files), conceptually a PowerPoint presentation is not "structured data" used to build a database, unlike tables in Word/Excel or directories.

8.
A collection of related records in a database is known as
a) File
b) Record
c) Relationship
d) Table
Show Answer

Correct Answer: d) Table (or a) File)

In relational DBs, it's a Table. In traditional file systems, it's a File.

9.
Each record is constituted by a number of individual data items called
a) Rations
b) Data types
c) Fields
d) Tables
Show Answer

Correct Answer: c) Fields

10.
The complete information about an activity in a database is called ____
a) Data
b) Record
c) Field
d) Information
Show Answer

Correct Answer: b) Record

A record contains all fields related to one entity/activity.

11.
What field type will you select if you need to enter long text in that field
a) Currency
b) Text
c) Memo
d) Hyperlink

Correct Answer: c) Memo

"Memo" (now Long Text) is used for long alphanumeric data.

Correct Answer: d) All the above

20.
The size of a field with data type as number cannot be
a) 2
b) 8
c) 4
d) 16
Show Answer

Correct Answer: d) 16

Number sizes: Byte (1), Integer (2), Long Integer (4), Double (8), Decimal (16 - specific case, but typically 16 is associated with replication IDs or Decimals, whereas 2, 4, 8 are standard number sizes. In MCQ context, 16 is often the outlier for standard integer/float types).

b) Double click on the relationship line
c) Both A and B
d) None of the above

Correct Answer: c) Both A and B
