WORD PROCESSING

Definition and Concepts

Definition: Word-processing is essentially typing, editing, and manipulation of a document in a desired form.

Units of the Document:

Since word-processing is concerned with preparation of a document (in a desired form), it is essential to know the units of the document:

  1. Character: It refers to the alphabets, numerical digits, punctuations and other special symbols which are commonly used in the text.
  2. Word: A word is group of characters that are separated from other group of characters by some delimiters like, comma, full stop and space.
  3. Sentence: A sentence is a group of words preceded and followed by appropriate delimiting characters.
  4. Paragraph: It is a group of one or more sentences, paragraphs are separated by leaving blank lines between them.
  5. Pages: It is the amount of text that can be printed on one page of a paper.
  6. Chapters: It is a collection of pages
  7. Documents and files: It is a group of chapters. Usually, a complete document may be very short such as a memo (or letter) or very long such as a book consisting of several chapters. These documents are referred as Files.

Features of Word-processing

  1. Word-wrap: In word-processing packages, the text can be continuously typed and the computer automatically starts a fresh line when a line is filled up. This is called word-wrap.
  2. Cursor Control: The four directional keys of the keyboard help in locating the text for editing.
  3. Editing: Words or lines can be entered (inserted) or deleted. Any word can be replaced by a new word throughout the file. A block of text can be moved or copied.
  4. Formatting: The text formatting refers to the way the text is desired to appear on a page. This includes setting margins, line spacing, font specifications, foot-notes, page numbers, etc.
  5. Spell-checker: Word-processing packages provide checking of spellings facility.
  6. Thesaurus: It provides synonyms (or words with similar meanings).
  7. Macros: A macro is a character or word that represents a series of keystrokes, saving time by replacing common combinations.
  8. Printing: It gives a hard copy of the text.
  9. File management: This facility allows to create, delete, move and search for files.
  10. Mail merge printing: This facility helps in printing the same original letter with different addresses.

MS-WORD

MS- Word is a word processor. It is an application used to create, edit, print and save a document. It allows the user to insert pictures, tables, charts, drawings and features that will make the text richer and more interactive.

Starting MS-Word: Start → All Programs → MS-office → MS-word

The MS-Word Interface

The default Word document includes the following layout tools:

  • Title bar: displays the document name and the application.
  • Menu bar: Contains the list of menus available inside word.
  • Standard toolbar: provides shortcuts for frequently performed tasks.
  • Formatting toolbar: Contains a list of formatting options.
  • Horizontal and Vertical rulers: used for measurement purposes.
  • White page area: is the space where you type, edit and format your document.
  • Insertion point: is the blinking vertical line that indicates where text or graphics will be placed.
  • Task pane: provides shortcuts to commonly used tasks.
  • Scroll bars: are used to move up and down or left and right in a document.
  • Status bar: displays details such as the page number, line number, column number etc.

Standard Toolbar: Function of commonly used buttons

  • New: Creates a new blank document.
  • Open: Opens or finds an existing file.
  • Save: Saves the active file.
  • Print: Prints the active file.
  • Print preview: Shows how the document will look when you print it.
  • Spelling and Grammar: Checks for spelling, grammar and writing style.
  • Cut: Removes the selection and places it on the clipboard.
  • Copy: Copies the selected item(s) to the clipboard.
  • Paste: Places the content of the clipboard at the insertion point.
  • Format painter: Copies the format from selected text and applies to other text.
  • Undo: Reverses the last command.
  • Redo: Reverses the action of the Undo button.
  • Insert table: Inserts a table into the document.
  • Insert Excel worksheet: Inserts an Excel spreadsheet into the Word document.
  • Columns: Changes the number of columns in a document.
  • Drawing: Displays or hides the Drawing toolbar.
  • Zoom: Enlarge or reduce the display of the active document.

Formatting Toolbar

  • Style menu: Allows you to make your text Bold, Italic, underlined, etc.
  • Font: Allows you to change the font.
  • Font size: Allows you to change the font size.
  • Bold, Italic, Underline: Applies bold, italic, or underline formatting.
  • Alignment: Aligns your text to the left, center, right, or justifies it.
  • Line spacing: Sets the amount of space between lines.
  • Numbering, Bullets: Creates a numbered or bulleted list.
  • Increase / Decrease indent: Increases or decreases the paragraph indentation.
  • Outside Border: Adds a border around a text selection.
  • Highlight colour: Changes the colour behind a text selection.
  • Font colour: Changes the colour of the text.

Features of FILE MENU

  1. New: Opens new Word file.
  2. Open: Opens existing files.
  3. Save: Saves the file with one name.
  4. Save as: Saves the file with more than one name or format.
  5. Page set up: Sets margins, paper size, Orientation.
  6. Print: Gets the printout (Specified pages and no. of copies).
  7. Properties: Gives details about the document.
  8. Send: Sends the file document to internet mail.
  9. Exit: Closes Ms-word.

Features of EDIT MENU

  • Undo (Ctrl +Z): Will take the previous command.
  • Redo (Ctrl + Y): Will take the opposite action of undo.
  • Cut (Ctrl + X): Can cut a selected text.
  • Copy (Ctrl + C): Can copy a selected text.
  • Paste (Ctrl + V): Can paste the selected text.
  • Find (Ctrl + F): Used to find text in the file.
  • Replace (Ctrl + H): Used to replace any part of text.
  • Go To (Ctrl + G): Cursor goes to the specified page, line, or paragraph.

FEATURES OF FORMAT MENU

  • Font: used to change the font face, style, size etc.
  • Paragraph: Used to set the space between paragraphs and lines.
  • Bullets: Used to highlight points in number wise / bullet wise.
  • Borders and Shades: Used to apply different borders.
  • Columns: Used to write the text in columns.
  • Drop Cap: Used to drop the first letter of a paragraph across multiple lines.
  • Change case: Used to change the case of the text (e.g., uppercase, lowercase).

Some important features of Tools menu:

  1. Spelling and Grammar (F7): Used to check the spelling and grammar.
  2. Thesaurus (Shift + F7): Used to know the synonym (meaning) of the word.
  3. Mail merge: used to type letters with the same matter but different addresses.

MS WORD - working with files

Creating a New document:

  1. Click the new Blank document button on the standard tool bar.
  2. (or) From the Menu bar, choose File → New, and select Blank document.

Opening Existing documents:

  1. Click the open button found on the Standard tool bar.
  2. (or) From the Menu bar, select File → Open.

Saving a document:

  1. Click the Save button on the Tool bar.
  2. (or) From the Menu bar, select File → Save.
  3. (or) Follow the key sequence Ctrl + S.

Printing a document:

  1. Select File → Print.
  2. (or) Click on the Print button on the Standard Tool bar.
  3. (or) Click Ctrl + P.

Closing a document:

  1. Select File → Close.
  2. Click on the small X found on the right top next to the Menu bar.

MS WORD - Editing Text

Typing and Inserting Text:

To enter text, position the insertion point and type. Word offers two modes for adding text:

  • Insert mode: characters typed are inserted into the text, pushing existing text to the right.
  • Overtype mode: the text you type will replace the existing text.

Note: The Insert key is a toggle key between these two modes.

Selecting Text:

To change the format of text, it must be first highlighted. To highlight, click and drag the mouse over the desired text.

Shortcuts for selecting text:

  • Whole word: Double click within the word.
  • Whole paragraph: Triple click within the paragraph.
  • Sentence: Ctrl + click in a sentence.
  • Entire document: Select all (Ctrl + A).

Deleting text

Use the BACKSPACE or the DELETE key to delete text.

  • ✓ BACKSPACE key will delete text to the left of the cursor.
  • ✓ DELETE key will erase text to the right.
  • ✓ To delete a large section of text, highlight it and press the DELETE key.

Moving, copying and pasting text

  • Cut text (Ctrl + X): Highlight the text and select Edit → Cut or click the cut button.
  • Copy text (Ctrl + C): Highlight the text and select Edit → Copy or click the copy button.
  • Paste Text (Ctrl + V): Move the cursor to the desired location and select Edit → Paste or click the paste button.
