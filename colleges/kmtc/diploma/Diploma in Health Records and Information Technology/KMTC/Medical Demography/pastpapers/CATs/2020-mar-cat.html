The following table relates to number of women and births that had occurred at least 12 months before a survey was carried out in 1979 in Kenya. Mid-year population was 76,398,000.

Age Group Women Births
15-193,777,000756,900
20-243,101,0001,046,000
25-292,636,000819,000
30-342,161,000565,000
35-391,793,000353,000

45-491,222,00016,500

Using the above information, calculate to one decimal place:

  1. Age Specific Fertility Rate (ASFR)
  2. Total Fertility Rate (TFR)
  3. Crude Birth Rate (CBR)
  4. General Fertility Rate (GFR)
View Calculations

Total Births: 756,900 + 1,046,000 + 819,000 + 565,000 + 353,000 + 141,000 + 16,500 = 3,697,400

Total Women (15-49): 3,777,000 + 3,101,000 + 2,636,000 + 2,161,000 + 1,793,000 + 1,484,000 + 1,222,000 = 16,174,000

a) ASFR (Births / Women × 1000):

  • 15-19: (756,900 / 3,777,000) × 1000 = 200.4
  • 20-24: (1,046,000 / 3,101,000) × 1000 = 337.3
  • 25-29: (819,000 / 2,636,000) × 1000 = 310.7
  • 30-34: (565,000 / 2,161,000) × 1000 = 261.5
  • 35-39: (353,000 / 1,793,000) × 1000 = 196.9
  • 40-44: (141,000 / 1,484,000) × 1000 = 95.0
  • 45-49: (16,500 / 1,222,000) × 1000 = 13.5

b) TFR:

Sum of ASFR = 200.4 + 337.3 + 310.7 + 261.5 + 196.9 + 95.0 + 13.5 = 1,415.3

TFR = (Sum of ASFR × 5) / 1000 = (1415.3 × 5) / 1000 = 7.1 children per woman

c) CBR (Total Births / Mid-year Pop × 1000):

CBR = (3,697,400 / 76,398,000) × 1000 = 48.4 per 1000

d) GFR (Total Births / Total Women 15-49 × 1000):

GFR = (3,697,400 / 16,174,000) × 1000 = 228.6 per 1000

(Note: Population figures in table provided as '000, expanded here for clarity or calculation).

Question: Using the direct method of standardization, standardize the immigrant's crude rate using the population of Uganda as your standard. (10 Marks)

View Steps

Step 1: Calculate Age Specific Rates for Immigrants (Index)

  • 0-4: 31/26,000 = 0.00119
  • 5-14: 6/23,000 = 0.00026
  • 15-44: 36/127,000 = 0.00028
  • 45-64: 50/28,000 = 0.00179
  • 65+: 55/5,000 = 0.01100

Step 2: Apply Index Rates to Standard Population (Uganda) to get Expected Cases

  • 0-4: 0.00119 × 2,000,000 = 2,380
  • 5-14: 0.00026 × 4,100,000 = 1,066
  • 15-44: 0.00028 × 9,500,000 = 2,660
  • 45-64: 0.00179 × 5,800,000 = 10,382
  • 65+: 0.01100 × 18,000,000 = 198,000

Step 3: Calculate Standardized Rate

Total Expected Cases = 2,380 + 1,066 + 2,660 + 10,382 + 198,000 = 214,488

Total Standard Population = 2m + 4.1m + 9.5m + 5.8m + 18m = 39,400,000

Standardized Rate = (214,488 / 39,400,000) × 1000 = 5.44 per 1000
