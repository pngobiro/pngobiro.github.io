Section 3: Database Creation (70 Marks)

1.
Open the computer and create a folder on the desktop with DHRI and the last three digits of your exam number eg DHRI001. (3 marks)
Show Practical Solution

Step-by-Step Procedure:

  1. Minimize all windows to view the Desktop.
  2. Right-click on an empty space on the Desktop.
  3. Select New > Folder.
  4. Type the name as requested (e.g., DHRI001 if your number ends in 001).
  5. Press Enter to save the folder name.
2.
Using Microsoft Access, create a database called Student Registration consisting of two tables: Students and Faculty. (5 marks)
Show Practical Solution

Step-by-Step Procedure:

  1. Launch Microsoft Access.
  2. Select Blank Database.
  3. In the "File Name" box, type: Student Registration.
  4. Click the folder icon to browse and select the folder you created on the Desktop (DHRI001) as the save location.
  5. Click Create.
3.
The Student table has the following fields with the following properties: (10 marks)
  • Student ID - data type is Text, field size is 4 and Caption is ID #. It is also the primary key.
  • Last Name is Text, field size is 15
  • First Name is Text, field size is 15
  • Course is Text, field size is 30
  • Lecturer ID is Text, field size is 4
  • Marks is Number and its field size is Integer
Show Practical Solution

Step-by-Step Procedure:

  1. In the open database, go to the Create tab and click Table Design.
  2. Enter the fields and properties as follows:
    • Field Name: Student ID | Data Type: Short Text
      • Field Properties (Bottom pane): Set Field Size to 4. Set Caption to ID #.
      • Right-click "Student ID" row and select Primary Key.
    • Field Name: Last Name | Data Type: Short Text
      • Properties: Field Size: 15.
    • Field Name: First Name | Data Type: Short Text
      • Properties: Field Size: 15.
    • Field Name: Course | Data Type: Short Text
      • Properties: Field Size: 30.
    • Field Name: Lecturer ID | Data Type: Short Text
      • Properties: Field Size: 4.
    • Field Name: Marks | Data Type: Number
      • Properties: Field Size: Select Integer from dropdown.
  3. Click the Save icon (floppy disk) or press Ctrl+S.
  4. Name the table Student and click OK.
4.
The Faculty table has the following fields with the following properties: (10 marks)
  • Lecturer ID - is Text, the field size is 4, the Caption is Fac. ID and it is the primary key.
  • Last Name - is Text and the field size is 15
  • First Name is Text, field size is 15
  • Office Phone is Text, field size is 4 and the Caption is Phone
Show Practical Solution

Step-by-Step Procedure:

  1. Go to Create tab > Table Design.
  2. Enter the fields:
    • Lecturer ID (Short Text): Size 4, Caption Fac. ID. Set as Primary Key.
    • Last Name (Short Text): Size 15.
    • First Name (Short Text): Size 15.
    • Office Phone (Short Text): Size 4, Caption Phone.
  3. Save the table as Faculty.
  4. Close the table design tab.

Create a form with all fields on the Student table. Name the form STUDENT DATA (10 marks).
Show Practical Solution

Step-by-Step Procedure:

  1. In the Navigation Pane, click once on the Student table to highlight it.

  6. Click OK.

7.
Create a relationship between the student table and faculty table and save (10 marks)
Show Practical Solution

Step-by-Step Procedure:

  1. Go to the Database Tools tab.
  2. Click Relationships.
  3. In the "Show Table" dialog, double-click Faculty and Student to add them to the workspace. Close the dialog.
  4. Identify the common field: Lecturer ID.
    • Faculty table has Lecturer ID (Primary Key).
    • Student table has Lecturer ID (Foreign Key).
  5. Click and drag Lecturer ID from the Faculty table and drop it on top of Lecturer ID in the Student table.
  6. In the "Edit Relationships" window, check the box Enforce Referential Integrity (this is best practice, though often just creating the line is accepted in exams).
  7. Click Create. A line connecting the two tables should appear.
  8. Click Save and close the Relationships tab.
8.
Create two queries for Student. One that lists every student's First Name, Last Name, Course and Marks (in that order). The second query will list the First Name, Last Name, Marks for every student whose Lecturr's ID is 2222. (10 marks)
Show Practical Solution

Query 1 Procedure:

  1. Go to Create > Query Design.
  2. Add the Student table.
  3. Double-click fields in this order: First Name, Last Name, Course, Marks.
  4. Save the query as Student List Query.

Query 2 Procedure:

  1. Go to Create > Query Design.
  2. Add the Student table.
  3. Add fields: First Name, Last Name, Marks, and Lecturer ID.
  4. In the Criteria row under the Lecturer ID column, type: "2222".
  5. Uncheck the "Show" box for Lecturer ID if you only want to see names and marks, though usually keeping it confirms the result.
  6. Run the query to verify it lists Ouma Juliet (63) and Gonda Larry (52).
  7. Save the query as Lecturer 2222 Query.
