Section 2: Ms. Access (40 Marks)

1.
Open a blank Ms. Access Database and save it in the folder "Final Exams" as "My Database" and the last three digits of your college number e.g. My Database698 (1 Mark)
Show Practical Solution
  1. Open Access.
  2. Select Blank Database.
  3. Name: My Database[YourNumber].
  4. Location: "Final Exams" folder on Desktop.
  5. Create.
2.
Import the Ms. Excel table "students' performance" that you created to the Database and save it as "Performance Table. (2 Marks)
Show Practical Solution
  1. In Access, go to External Data > New Data Source > From File > Excel.
  2. Browse to "students' performance.xlsx".
  3. Select "Import the source data into a new table in the current database".
  4. Follow Wizard: Check "First Row Contains Column Headings".
  5. Choose Primary Key: You can choose "Student Number" if it's unique, or let Access add ID. Given Q3 creates another table with Student Number, it's good practice to index this.
  6. Name table: Performance Table.
  7. Finish.
3.
Create a table and name it "STUDENTDATA" with appropriate fields and data types as below; (6 marks)
Field NameData typeField Properties
Student NumberShort TextField size 25

Show Practical Solution
  1. Create > Table Design.
  2. Student Number: Short Text, Size 25.
  3. Full Name: Short Text, Size 25.
  4. Contacts: Short Text, Size 12.
  5. County: Short Text.
    • To make it a Combo Box: Click on the Lookup tab in Field Properties.

    • Row Source: "Nakuru";"Mombasa";"Vihiga";"Nairobi" (based on data in Q5).
  6. Save as STUDENTDATA.

6.
Change the settings of the StudentDataF Form as follows;
  1. Back Colour: Green 4 (2 Marks)
  2. Border width = 4 pt (2 Marks)
  3. Border colour = black (2 Marks)
  4. Font name = Aerial (2 Marks)
  5. Font size = 16 (2 Marks)
Adjust the rows and columns appropriately to fit the contents.
Show Practical Solution
  1. Open StudentDataF in Design View.
  2. Back Color: Click on the Detail section background. In Property Sheet > Format > Back Color, select a Green shade (e.g., Green 4 or similar system color).
  3. Border Width/Color: Click on the text box controls (hold Shift to select all). In Property Sheet > Format:
    • Border Width: 4pt.
    • Border Color: Black (#000000).
  4. Font: With controls selected, in Property Sheet > Format:
    • Font Name: Arial (Question says "Aerial", interpret as Arial).
    • Font Size: 16.
  5. Resize the text boxes to ensure data is visible with the larger font.
7.
Create an appropriate relationship for the two tables (2 Marks)
Show Practical Solution
  1. Database Tools > Relationships.
  2. Add Performance Table and STUDENTDATA.
  3. Drag Student Number from one table to Student Number (or STUDENT NUMBER) in the other.
  4. Create Relationship (One-to-One if numbers are unique in both, or One-to-Many).
8.
Create a query that captures students from Vihiga County displaying their Registration Number, Names and scores for English and Biology (4 Marks)
Show Practical Solution
  1. Create > Query Design.
  2. Add both tables.
  3. Select Fields:
    • Student Number (Registration Number).
    • Full Name (Names).
    • English.
    • Biology.
    • County (to filter).
  4. In Criteria under County, type "Vihiga".
  5. Uncheck "Show" for County if you only want to display the requested fields.
  6. Save as Vihiga Query.

From the query above, create a report and Name in "Vihiga Report". Format your report to fit the page and look appealing (4 Marks)
Show Practical Solution
  1. Select Vihiga Query.
  2. Create > Report Wizard (or Report).
  3. Finish.
  4. Save as Vihiga Report.
  5. In Layout View, adjust column widths so they don't spill over to a second page.
