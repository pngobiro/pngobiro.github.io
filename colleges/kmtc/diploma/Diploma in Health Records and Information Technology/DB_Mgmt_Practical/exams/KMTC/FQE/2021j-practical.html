Section 1: Microsoft Access

1.
You are the new HRIO in Baringo County and you have been asked by the MOH to come up with a database to manage the paper records. Create a database with the following field names. (5 marks)
Admission No.Data Type
Patient nameText
Admission dateDate/Time
IllnessText
Ward NumberText
GenderText
Payment methodCurrency
NationalityText
Covid vaccineYes/no
Show Practical Solution

Step-by-Step Solution:

  1. Open Microsoft Access and select Blank Database.
  2. Name the database (e.g., Baringo_HRIO_Database) and click Create.
  3. Go to View > Design View. Save the table as Patients.
  4. Enter the field names and data types as specified:
    • Note: "Admission No." is likely the Primary Key (AutoNumber or Text depending on format, usually Text/Number for records).
    • Set "Payment method" to Text (Standard practice) or Currency if it refers to amount paid, but the sample data later shows "Mpesa", "NHIF", "Cash". Correction: The question asks for "Currency" but the data in Q2 implies a method name. Exam Strategy: Follow the instruction table strictly (Currency) initially, or if data entry fails later, modify design to Text. Given the data in Q2 ("Mpesa"), the instruction "Currency" is likely a typo in the exam paper. Best Action: Set to Short Text to match Q2 data.
  5. Save the Table Design.

Step-by-Step Solution:

  2. Define the fields based on the data:
    • Doctor Name: Short Text
    • Mobile Number: Short Text (to handle hyphens)
    • Address: Short Text

5.
Make Doctors ID to be the primary key (2 marks)
Show Practical Solution

Solution:

  1. Open Doctors table in Design View.
  2. Click on the Doctor ID field selector (gray box to the left of the field name).
  3. Click the Primary Key icon in the Design ribbon (Key icon).
  4. A key symbol should appear next to Doctor ID.
  5. Save the table.
6.
Create a relationship between the two tables (2 marks)
Show Practical Solution

Solution:

  1. Go to Database Tools tab > Relationships.
  2. Add both tables (patients table and Doctors table).
  3. Identify the common field. Both tables have a Ward field.
    • Note: Normally relationships are on ID fields (e.g., DoctorID in patients table), but based on the provided schemas, Ward is the only common data point linking them, implying a many-to-many relationship based on location, or linking doctors to wards.
  4. Drag Ward from Doctors table to Ward (or Ward Number) in patients table.
  5. Click Create in the Edit Relationships dialog box.
7.
Create a query showing all male patients and name it Gender query (3 marks)
Show Practical Solution

Solution:

  1. Go to Create > Query Design.
  2. Add the patients table.
  3. Add all fields to the grid (double click title bar of table and drag to grid).
  4. In the Criteria row under the Gender field, type "M".
  5. Run the query to verify it shows only Male patients.
  6. Save the query as Gender query.
8.
Create a form for patients table and use it to enter one new patient (4 marks)
Show Practical Solution

Solution:

  1. Select patients table in the navigation pane.

  3. Save the form as Patients Form.
  4. In Form View, navigate to a new record (bottom arrow with star icon).
  5. Enter details for a new dummy patient (e.g., Adm: 9999, Name: Test, etc.).
  6. Save the record.

10.
Create a query to create a new table showing the following from the patients table: Covid vaccine, admission number, patients name, gender and payment method. Name it Payment query. (5 marks)
Show Practical Solution

Solution:

  1. Go to Create > Query Design.
  2. Add patients table.
  3. Double click the specific fields requested: Covid vaccine, Admission No, Patient name, Gender, Payment method.
  4. In the Design ribbon, change the query type from Select to Make Table.
  5. In the dialog box, enter Table Name: Payment query (as requested, though typically queries and tables have distinct names, follow the exam instruction).
  6. Click Run (Red Exclamation mark).
  7. Confirm "You are about to paste X rows into a new table". Click Yes.
  8. Save the query as Make Payment Table (or as instructed).
11.
Create a macro to open the doctors form (5 marks)
Show Practical Solution

Solution:

  1. Go to Create > Macro.
  2. In the Action dropdown, select OpenForm.
  3. In the "Form Name" argument, select Doctors Form from the dropdown.
  4. Save the macro as OpenDoctors.
  5. Run it to test (it should open the form).

Section 2: Microsoft PowerPoint

1.
Start a new Microsoft PowerPoint presentation, start a new black presentation. Your first slide is going to be a title slide. (4 marks)
Show Practical Solution

Solution:

  1. Open Microsoft PowerPoint.
  2. Select Blank Presentation.
  3. The default first slide is automatically a Title Slide layout.
2.
Write the text as shown in preview below:
COVID - 19
Pfizer Lttd
VACCINES FOR LIFE
Show Practical Solution

Solution:

  1. Click inside the "Click to add title" placeholder.
  2. Type: COVID - 19.

  4. Type: Pfizer Lttd press Enter, then type VACCINES FOR LIFE.
3.
Make the font of COVID-19 and VACCINES FOR LIFE, Arial Black and size 88. (1 mark)
Show Practical Solution

Solution:

  1. Highlight the text COVID - 19.
  2. Go to Home tab > Font group. Change Font to Arial Black and Size to 88.
  3. Highlight the text VACCINES FOR LIFE.
  4. Apply the same settings (Arial Black, Size 88).
4.
Insert a second slide with a layout of bulleted list. Write the text as shown below:

  • Fast adaptability
  • 95% efficacy
  • Based in Italy

Solution:

  1. Go to Home > New Slide.
  2. Select the Title and Content layout (standard for lists).
  3. In the Title placeholder, type Pfizer Lttd.
  4. In the Content placeholder, type the list items provided. Bullet points are active by default in this layout.

Show Practical Solution

Solution:

  1. Highlight all the bullet points in the second slide.
  2. Go to Home tab > Font Color icon (A with a colored bar).
  3. Select a Green color from the palette.
6.
Insert a third slide and use the following data to present a multiple bar chart in the slide (4 marks)
EfficacyJanuaryFebruaryMarch

Show Practical Solution

Solution:

  1. Insert a new slide (Title and Content).
  2. Click the Insert Chart icon in the center of the content placeholder (or Insert > Chart).

    • Series headers: January, February, March
    • Category rows: Pfizer, Moderna, Sputnik, Rwandese
    • Data points: Enter corresponding values.
  5. Close the Excel window.
