41.
A student sat for exams in five paper and score the following, 64, 56, 70, 67, 73. Calculate the variance and standard deviation of the scores (5 marks)
Show Answer

Answer:

Data ($X$): 64, 56, 70, 67, 73 ($n=5$).

1. Calculate Mean ($\bar{x}$):
$\bar{x} = (64+56+70+67+73)/5 = 330/5 = 66$.

2. Calculate Deviations ($x - \bar{x}$) and Squared Deviations ($x - \bar{x})^2$:
64 - 66 = -2 -> 4
56 - 66 = -10 -> 100
70 - 66 = 4 -> 16
67 - 66 = 1 -> 1
73 - 66 = 7 -> 49

3. Sum of Squares: $4 + 100 + 16 + 1 + 49 = 170$.

4. Variance ($s^2$):
$s^2 = \sum(x-\bar{x})^2 / (n-1)$
$170 / 4 = 42.5$.

5. Standard Deviation ($s$):
$s = \sqrt{42.5} \approx 6.52$.

42.
In the year 2018, Mwangaza hospital which had 45 beds allocated throughout the year recorded 11,680 in-patient days. A total of 98 patients died while 2,238 were discharged during the periods. From the above information, calculate, (5 marks)

a) Available bed days (1 mark)

b) % occupancy (1 mark)

c) Average daily vacant beds(1 mark)

d) ALOS (1 mark)

e) TOB (1 mark)

Show Answer

Answer:

Given: Beds = 45, Year 2018 (365 days), IPSD = 11,680, Deaths = 98, Discharges = 2,238.

a) Available Bed Days (ABD):
$45 \times 365 = 16,425$.

b) % Occupancy:
$(IPSD / ABD) \times 100$
$(11,680 / 16,425) \times 100 = 71.1\%$.

c) Average Daily Vacant Beds:
Vacant Bed Days = $16,425 - 11,680 = 4,745$.
Avg Daily = $4,745 / 365 = 13$ beds.
Alternatively: $45 \times (1 - 0.711) = 13$.

d) ALOS (Average Length of Stay):
$IPSD / (Discharges + Deaths)$
$11,680 / (2238 + 98) = 11,680 / 2,336 = 5$ days.

e) TOB (Turnover Rate):
$(Discharges + Deaths) / Bed Count$
$2,336 / 45 = 51.9$ patients per bed.

47.
Calculate the geometric mean for the following data 10, 12, 12, 13, 15, 19 (5 marks)
Show Answer

Answer:

Formula: $GM = \sqrt[n]{x_1 \cdot x_2 \cdot ... \cdot x_n}$

$n = 6$.

Product = $10 \times 12 \times 12 \times 13 \times 15 \times 19$
= $5,335,200$.

$GM = \sqrt[6]{5,335,200}$

Using logs:
$\sum \log x = \log 10 + \log 12 + \log 12 + \log 13 + \log 15 + \log 19$
$= 1 + 1.079 + 1.079 + 1.114 + 1.176 + 1.279 = 6.727$.

Mean Log = $6.727 / 6 = 1.121$.

$GM = \text{Antilog}(1.121) \approx \mathbf{13.2}$.

Answer:

Data Preparation:

a) Mean Height:
$\bar{x} = \sum fx / \sum f$
$\bar{x} = 8520 / 50 = 170.4$ cm.

b) Median Height:
Position = $N/2 = 50/2 = 25$.
25th value is in the class 170-174.
$L = 169.5$ (Lower class boundary).
$CF = 21$ (Cumulative freq before).
$f = 10$ (Freq of median class).
$i = 5$ (Class width).
$Median = L + (\frac{N/2 - CF}{f}) \times i$
$= 169.5 + (\frac{25 - 21}{10}) \times 5$
$= 169.5 + (\frac{4}{10}) \times 5$
$= 169.5 + 2 = 171.5$ cm.

c) Graphical Estimation of Mode:
1. Draw a Histogram with frequency on y-axis and Class Boundaries on x-axis.
2. Identify the Modal Class (highest bar): 175-179 ($f=11$).
3. Draw diagonal lines from the top corners of the modal bar to the adjacent corners of the neighboring bars.
4. Drop a perpendicular line from the intersection of diagonals to the x-axis.
Calculation Check for Graph Verification:
$Mode = L + (\frac{f_1 - f_0}{2f_1 - f_0 - f_2}) \times i$
Modal Class: 175-179. $L=174.5, f_1=11, f_0=10, f_2=5$.
$Mode = 174.5 + (\frac{11-10}{22-10-5}) \times 5$
$= 174.5 + (\frac{1}{7}) \times 5 = 174.5 + 0.71 = 175.2$ cm.
(The graph should point to approx 175.2 cm).
