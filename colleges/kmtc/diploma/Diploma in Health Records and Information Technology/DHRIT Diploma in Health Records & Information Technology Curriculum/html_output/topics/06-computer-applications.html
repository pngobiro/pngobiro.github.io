08. MODULE TITLE: COMPUTER APPLICATIONS

Module Overview

TIME: 120 HOURS

CREDITS: 12 CREDITS

MODULE COMPETENCE

To enable the learner to collect, compute, analyze, interpret ad disseminate data for better healthcare service delivery.

Units Covered:

  • UNIT 1: Introduction to ICT & Ethics (20 HOURS)
  • UNIT 2: Computer Application Packages (20 HOURS)
  • UNIT 3: Computer Operating Systems (20 HOURS)
  • UNIT 4: System Analysis & Design (20 HOURS)
  • UNIT 5: Database Management Systems (20 HOURS)
  • UNIT 6: Computer Networking (20 HOURS)

Module Outcomes

At the end of the module the learner will be able to:

  1. Apply basic skills in computer allocation packages
  2. Utilize basic statistical skills for evidence -based decision making
  3. Present data using various presentation packages
  4. Analyze and process data using basis ICT skills

Module Content

DHR1108 COMPUTER APPLICATIONS I (20 HOURS)

INTRODUCTION TO INFORMATION COMMUNICATION TECHNOLOGY & ETHICS

INTRODUCTION

This module unit is intended to introduce the trainee to Information Communication Technology, equipping him/her with the necessary knowledge, skills and attitude to enable him/her professionally work with ICT facilities.

GENERAL OBJECTIVES

By the end of this module unit, the trainee should be able to:-

  • a) appreciate the role of ICT in health records field
  • b) appreciate the ethical and legal implication in the use of ICT
  • c) appreciate the basic skills necessary in the use of ICT
INTRODUCTION TO COMPUTERS
Specific Objectives

By the end of this topic, the trainee should be able to:-

  • a) explain the meaning and importance of a computer
  • b) describe the components of a computer
  • c) describe computer hardware
  • d) describe computer software
  • e) explain the historical evolution of computers
  • f) describe the classification of computers
  • g) explain the benefits and challenges of computers
Content
  • Meaning and importance of a computer
  • Historical evolution of computers
  • Computer components (Computer hardware, Computer Software)
  • Classification of computers (super computers, mainframe computers, micro computers, mini computers)
  • Benefits and challenges in the use of computers
COMPUTER HARDWARE AND ORGANISATION

Content includes: input devices, output devices, central processing unit, storage devices, bus system, others.

COMPUTER SOFTWARE

Content includes: Meaning and importance of computer software, Classification of computer software (System software, Application software).

COMPUTER SAFETY

Content includes: Meaning and importance of computer safety and ergonomics, Computer safety (hardware safety, software safety).

ICT ETHICS AND LEGISLATIONS

Content includes: Meaning and importance of ICT ethics, Unethical behaviour in ICT, Reasons for unethical behaviour, Control measures, Sources of ICT legislations, ICT legislations and policies (ICT policy, Data Protection, Copyright, Information Privacy), Enforcement measures.

DHR1208 COMPUTER APPLICATION II (20 HOURS)

UNIT 02: COMPUTER APPLICATION PACKAGES

INTRODUCTION

This module unit is designed to equip the trainee with the practical skills on the use of basic computer application packages.

GENERAL OBJECTIVES

By the end of this topic, the trainee should be able to:

  • a) understand the basic concept of computer application packages
  • b) appreciate and use word processing application software
  • c) use spreadsheet application software
  • d) use database application software
  • e) appreciate and use presentation application software
WORD PROCESSING

Content includes: Definition, concepts, features, document formatting, printing, mail merging, table creation, templates, multiple column formats, utilities (sorting, spell checking, thesaurus), object formulae, macros, and advanced features (table of contents, indexes, email).

SPREADSHEET

Content includes: Worksheet layout, formatting, cell data types, formulae and functions, charting, simple database management, printing, utilities (sorting, filtering), and macros.

DATABASE

Content includes: Database structure, creating a database, retrieval and editing, sorting and indexing, logic and arithmetic functions, querying, reporting and labeling, screen/view formatting, and linking of databases.

DHR2108: COMPUTER APPLICATIONS III (20 HOURS)

UNIT 03: OPERATING SYSTEMS

INTRODUCTION

This module unit is intended to equip the trainee with knowledge, skills and attitudes to enable him/her use operating system in a computing environment.

GENERAL OBJECTIVES

By the end of the module unit the trainee should be able to:

  • a) understand the principles of operating systems
  • b) appreciate the functions of operating systems
  • c) use operating systems in a computer environment
INTRODUCTION OPERATING SYSTEMS

Content includes: Meaning and importance, types (Traditional, Multiprocessor, Distributed), functions, and introduction to Ms Windows Components.

DHR2208: COMPUTER APPLICATIONS IV (20 HOURS)

UNIT 04: SYSTEM ANALYSIS & DESIGN

INTRODUCTION

This module unit is intended to equip the trainee with the knowledge, skills and attitudes to enable him/her to undertake systems analysis and design.

GENERAL OBJECTIVES

By the end of this module unit, the trainee should be able to:

  • a) understand systems concepts
  • b) understand systems analysis and design phases
  • c) use systems analysis design tools and techniques
  • d) apply systems development methodologies
INTRODUCTION TO SYSTEMS ANALYSIS AND DESIGN

Content includes: Meaning of terms (system, information, information system), components, types of information systems (Transaction processing, Management information, Decision support, Expert systems), and roles of stakeholders.

SYSTEMS DEVELOPMENT LIFE CYCLE (SDLC)

Content includes: Meaning of SDLC, and stages (problem definition, feasibility study, systems analysis, design and development, implementation, maintenance and review).

DHR3108: COMPUTER APPLICATIONS V (20 HOURS)

UNIT 05: DATABASE MANAGEMENT SYSTEMS

INTRODUCTION

This module unit is designed to equip the trainee with knowledge, skills and attitude that will enable him/her understand the management of databases.

GENERAL OBJECTIVES

By the end of this module unit, the trainee should be able to:-

  • a) appreciate database design and development
  • b) understand the components of database management systems
  • c) develop and manipulate a database
INTRODUCTION TO DATABASE MANAGEMENT SYSTEMS

Content includes: Meaning, components, historical evolution, traditional vs database approaches, advantages, and role of key players.

QUERYING A DATABASE

Content includes: Meaning of a database query, features of SQL, categories of SQL, and designing and using statements to query a database.

DHR3208: COMPUTER APPLICATIONS VI (20 HOURS)

UNIT 06: COMPUTER NETWORKING

INTRODUCTION

This module unit is designed to equip the trainee with knowledge, skills and attitude that will enable him/her install and manage computer networks.

GENERAL OBJECTIVES

By the end of this module unit, the trainee should be able to:

  • a) understand the concepts of computer networks
  • b) understand the various data transmission techniques
  • c) appreciate the use and application of internet
INTRODUCTION TO COMPUTER NETWORKS

Content includes: Meaning, components, types, role, topologies, and categories.

NETWORK MEDIA

Content includes: Electrical properties, importance of different media, types (bounded, unbounded), wireless networks, and advantages/disadvantages of different media.

INTERNET

Content includes: Configuring computer for internet use, demonstrating file transfer, and network browsing.
