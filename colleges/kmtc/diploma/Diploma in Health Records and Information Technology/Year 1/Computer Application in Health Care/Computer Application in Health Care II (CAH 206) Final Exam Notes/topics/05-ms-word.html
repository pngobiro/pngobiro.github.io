Word Processing and Word Processor

1. Word processing

Word processing is the activity of manipulating text, characters, words and sentences so as to make the final document free of errors and also attractive to look at. It enables one to create, edit, revise, retrieve, save and printing of documents or store for future use.

2. Word processor

  • This is a computer application package used in manipulation of text, characters, words and sentences so as to make the final document free of errors and also attractive to look at.
  • Also defined as a "computer application package used in processing of words."
  • We can also define it as a computer application software that enable a person to create, save, edit, format and print text documents.

Advantages of Word Processors

Advantages of word processors over the conventional method of document preparation:

  1. Documents can be stored for future use
  2. Typing is easy and efficient because some actions are automated
  3. Has special editing tools e.g. spelling & grammar check, thesaurus Etc.
  4. Have superior formatting features e.g. underline, bold, italic, color Etc.
  5. One can use Copy, Cat and pest commands i.e. reuse text without having to retype
  6. One can easily insert tables and illustration e.g. pictures, clip art, charts auto shapes Etc.
  7. One can print as many copies of the document as he/she want
  8. One can correct or edit text without wasting papers

Getting Started with MS Word

Starting MS Word

One can start MS Word in the following three ways

  1. Click start button → then select All Programs → then choose Microsoft Office → then select on the Microsoft word 2007
  2. Double click on the MS word icon in the Desktop if it's available and it will open word
  3. Open any word Document and it will start MS Word automatically

Elements found in Ms Word

  1. Title Bar-located at the top of the window, Display name of the application and the name of the document
  2. Tab Bar - located below title bar, list available options in Ms Word
  3. Tool Bar - located below the Tab bar, provide access to commands used in Ms Word (for word processing task)
  4. Formatting toolbar Ruler - located below Tool Bar, it is used to set tab stops and indent
  5. Scroll Bar and scroll buttons - used to view different areas of the active document, up and downwards or side by side
  6. Status Bar - located across the bottom of the window, display information about the active document e.g. current page number, number of words Etc.
  7. View buttons - located directly to the right side of the status bar, provide a quick way of moving between various page view

HOME TAB contains command buttons that you can use to carry out many of the commonly used word commands.

You can access all Ms Word commands through the Tab Bar.

* When you want to save a document, click Office button, → then save or save As (or press CRT + S at the same time)

* In the SAVE IN dialog box, select where you want to save the document

* When naming a document, don't use the following characters / : ; *, ?, <, >

Text Manipulation

Ways of inserting text

Word is a WYSIWYG (what you see is what you get) word processor. This mean that what you see on the screen is exactly what will be printed.

Using the ENTER key

It is used to:-

  1. End a paragraph
  2. End a short line E.g. a title, heading or a list of items
  3. Create a blank line

Using the Tab key

Use to align text to specific column positions. Space bar creates a jagged appearance when printed.

Using the Backspace and Delete Keys

  • Backspace key is used to delete characters to the left of the cursor key
  • Delete key is used to delete characters to the right of the cursor key

Inserting text in insert mode and overwrite mode

  • By default, word operates in insert mode. New text does not replace existing text but bushes it to the right as you type.
  • When you need to overwrite text, change to overwrite mode by pressing the Insert Key (note that the status bar OVR is now active)

Select text with a mouse

  • Any amount of text - Hold down the mouse button and drug over the text
  • A word - double click the word
  • A line of text - move the mouse pointer to the left margin of the line until it changes to a right-pointing arrow, and then click
  • Multiple line of text - move the mouse pointer to the left margin of the line until it changes to a right-pointing arrow, and then drag up or down
  • A sentence - Hold down the CTRL Key and click anywhere in the sentence
  • A paragraph - triple-click anywhere in the paragraph or - move the mouse pointer to the left margin of the paragraph until it changes to a right-pointing arrow, and double-click the mouse.
  • Multiple paragraphs - double click and while holding down the Shift Key, drag up or down /or/ move the mouse pointer to the left margin of the paragraph until it changes to a right-pointing arrow, and double-click the mouse. then while holding down the shift key drag up or down
  • A large block of text - click the start, scroll to the end of the selection and while holding down SHIFT Key, click at the end
  • An entire document - press CTRL + A, at the same time

Deleting text

Use BACKSPACE key to delete characters to the last or use DELETE key to delete characters to the right of the insertion pointer.

Replacing text

One can type the new text to replace the old text.

Text Editing Features

Copying and moving(cut) text

We use CUT to move text and COPY to make another copy of text. We can either use the mouse by right clicking on the selected text or we can use the Tool Bar in the Home Tab on the clipboard group.

Text formatting

In the Home Tab, Font section, one can format font:

  • Size
  • Style
  • Color
  • Underline
  • Super script and Subscripts ETC.

Find and replace

One can find text in the document by using the FIND feature in the tool bar in edit section of the Home Tab.

Spell check and thesaurus

  • Thesaurus is used to find synonyms, antonyms or related words
  • Spelling errors are underlined in red while grammatical errors are underline in green
  • Ms Word checks the following errors:
    • Spelling errors
    • Repeated words
    • Irregular capitalization
    • Errors in sentence structure

Paragraph Formatting

1. Paragraph numbering and bulleting

Select text you want to add bullet/numbering and click bulleting/numbering. One can increase or reduce indent of numbering.

2. Indent and Tab

Are use to align text within a paragraph.

3. Word wrap

Word Wrap enable one to easily place text around pictures, shapes and tables with any position or style that you want e.g.

  • Square
  • Tight
  • Behind or in front of text Etc.

4. Drop cap

Use to create a large dropped initial capital latter. It is sometimes used when starting a paragraph at the beginning of a chapter in novels. To create, click the paragraph you want to begin with a drop cap, on the Insert tab, → click Drop Cap in the text section for the dialog box to appear. Then select the one you want.

Page Formatting

1. Page margins

Page margins are used to restrict the printable area of a page. When changing the page margin, we follow the following steps:-

  1. Switch to page layout tab, → select margins on page setup section and select the option available /or/
  2. Point to a margin boundary on the horizontal ruler or vertical ruler to change each one of them
  3. When the pointer changes to a margin height pointer, drag the margin boundary

To specify exact margin measurements, select the "page layout" tab and click the margins, type the required values for each of the margin needed to be changed.

2. Insert page breaks

Click page Layout Tab, → select BREAKS in the page setup section.

3. Setting column layout

Columns are used to format document into layout, like the columns of a newspaper. To quickly format text into columns, select the text to be included in the columns, click on the columns icon in the page layout tab, → page setup section, and highlight the number of columns you want.

4. Adding header and footer

Header and footer are typically used in printed documents. They appears above the top margin or below the bottom of each page margin respectively. One can include graphics or text e.g. page number, date, company logo, document title or authors name etc. To insert page header and/or footer, follow the following steps:

  1. Click Insert Tab, → go to header & footer section, the select header or/\& footer then
  2. Select the format of page header/footer you want
  3. Enter text, graphics Etc., in the header/footer

5. Page orientation

Page orientation refers to the direction the page text is laid out on the page Eg. Portrait or landscape.

  • Portrait - text is laid across the upright page
  • Landscape - across the horizontal page

Steps are: Click the Page Layout Tab, → select page setup section, → then select orientation.

Tables

It is made up of ROWS and COLUMNS of cells that you can fill with text and graphics. You can use tables to:

  • Align text in columns
  • Sort text
  • Perform simple calculation

Create a table

Select Insert Tab, → click table in table section. Select the number of rows and columns of the table and then click okay.

Edit table

One can perform the following editing features on a table namely:-

  • Resizing Row/Column size - using mouse to drag the boundaries of the Row/Column /or/ use the Layout Tab, → Cell Size Section
  • Insert Row/Column - using the mouse, select Row/Column that you want to add a Row(above or below) or Column (to the right or left), then right click on it and then select add Row/ Column /or/ use the Layout Tab, → Rows & Columns Section
  • Delete cells/Rows/Columns - using the mouse, select Row/Column that you want to delete, right click on it and then select delete Row/Column /or/ use the Layout Tab, → Rows & Columns Section
  • Merge Cells/Rows/Columns - using the mouse, select Cells/Row/Column that you want to merge, right click on it and then select merge Row/Column or use /or/ use the Layout Tab, → Merge Section → merge
  • Split Cells/Rows/Columns - using the mouse, select Cells/Row/Column that you want to merge, right click on it and then select merge Row/Column or use /or/ use the Layout Tab, → Merge Section → split

Formatting table

You can use all the text formatting options to format TEXT within a table. To give a table more polish look, you can use:

  • Borders - table Design Tab, → then Table Styles
  • Shading - table Design Tab, → then Table Styles

Performing arithmetic calculation on tables

One can perform simple arithmetic on data entered in a table by following the steps below:

  1. Click the cell in which you want the results to appear
  2. On the Layout Tab, → Data section, select formula
  3. In the paste function box, click a function e.g. sum
  4. In the Number Format box, enter a format for number, E.g. Decimal or Percentage
  5. In the Formula box, type the cell references in the parentheses and then click OK button E.g. =Sum(Left)

Other Features

1. Sorting

Used to sort text, numbers or dates within a table or in the list either in ascending or descending order.

2. Mail merge

Used when you want to send out a standard yet personalized latter to several people at the same time. Instead of repeating the latter with different address, you can create a form containing the address list and then merge it with either Envelope, labels, latter, E mails, Directories Etc. When creating the Mail Merge, follow the following steps:-

  1. Create the main document E.g. Envelop or a Latter
  2. Create the Data Source → Mailings Tab, → Start Mail Merge section, Select Recipients. One can either:
    • Type new list
    • Use an existing list
    • Or select from Outlook content
  3. Insert mail Merge Field
  4. Previewing & fine tuning the merge Document

3. Inserting a picture objects

Word Processing documents may have pictures objects that can be Drawing (Clip Art), Charts (graphs), Photographs, Etc. Ms Word comes with a Clip Art Gallery that includes a wide variety of Clip Art that make it easy for you to dress up your document with professional designed images E.g. people, nature, human activity, maps, photographs etc. one can :-

  • Insert a picture from another file. → click Insert Tab, → select Picture in Illustration section
  • Insert a Clip Art file. → click Insert Tab, → select Clip Art in Illustration section. In the Clip Art Dialog box, select the one you want
  • Draw objects → click Insert Tab, → select Shapes in Illustration section and select any object to draw

4. Word art

This is used to add special effects to the text. With Word Art, one can:

  • Create shadowed text
  • Create skewed text
  • Create Rotated Text
  • Create Stretched text
  • Create text fitted to predefined shapes

Changing normal text to Word Art: Select the text to change into Word Art, usually a short phrase. Then click the Word Art icon in the Insert Tab, → Text section.

5. Section Breaks

A Section Break is a mark you insert to show the End of a section, E.g. book chapter. The section includes such information as margins, page orientation, header, footer and sequence of page numbering. There are three types of section breaks in Ms Word:

  • Next Page - start new section on new page
  • Continuous - start new section on the same page
  • Odd or Even Pages - start new section on the next Odd or Even page
  • Column break

To start Section Break, go to Page Layout Tab, → Breaks, in the Page Setup section.

6. Table of contents

Table of Content (TOC) provides an overview of the topics discussed in a document and is used to quickly navigate to a give topic in a document. It uses the Heading Style to determine deferent levels. To create TOC, you 1st ensure that you apply similar heading styles (Header 1 through Heading 9) to the heading you want to include in the Table of Content and the :-

  1. Move to the top of the document
  2. Click References Tab, → select Table of Content
  3. Scroll down the list of available styles

7. Printing a document/s

Printing is the process of making a hard copy of the document on a real paper. Before printing, the following factors should be considered:-

  • Previewing the document to display each page as it would appear when printed
  • Before printing, one should know the page orientation of the document
    • Portrait-across paper in the upright position
    • Landscape - across the paper in horizontal position
  • Print options E.g.
    • Print a range of pages
    • Print multiple pages
    • Print only ODD or EVEN pages
    • Print the document on different paper SIZE

Print several documents together.
