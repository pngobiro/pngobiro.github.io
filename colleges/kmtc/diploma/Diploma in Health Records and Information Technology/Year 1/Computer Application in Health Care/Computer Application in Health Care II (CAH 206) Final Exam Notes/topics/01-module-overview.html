Computer Application in Health Care II (CAH 206)

Module Competence

This module is designed to enable the learners set up a computer, carry out basic maintenance and apply appropriate computer application packages in health records management and communication.

Module Outcomes

By the end of this module, the learner should be able to;

  1. Carry out basic hardware connectivity and maintenance
  2. Understand and explain the computer application software
  3. Use Ms. Word for daily activities
  4. Apply Ms. Excel in Health Records Management
  5. Use PowerPoint for data presentation
  6. Browse internet and communicate by email
