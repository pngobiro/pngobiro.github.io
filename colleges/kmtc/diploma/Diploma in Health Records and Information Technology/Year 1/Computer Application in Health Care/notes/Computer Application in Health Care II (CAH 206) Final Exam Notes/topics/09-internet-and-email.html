Internet and E-mails (Computer Networking)

1. Definitions

Network

It's a set of connected things or people and its' size is determined by the number of items/people connected to it, their locations/area size (space between them), the needs of the users (capabilities/data band) and the available technology e.g. inter-county highways, telephone network etc.

Computer network

It's formed when two or more computers are connected together to enable communication among them e.g. send messages, share files and programs.

Internet

It's the world biggest computer network (a vast network of networks). Internet comprises of the hardware that sent information/data from one place to another and the software that govern (glues) the system together. Network that form the internet are individually run by different entities namely the: government agencies, universities, commercial and voluntary organization etc. No single organization owns or controls the internet, though there is an internet society (The World Wide Web Consortium - W3C) that set and co-ordinates standards and protocols for the internet.

2. History of the internet

Stage 1 - The beginning

In 1969, the Defense Advance Research Project Agency (DARPA) of the USA Department of Defense (DoD) built the 1st network known as Advance Research Project Agency Network (ARPANET). Stage 2 - Internet start running

In 1982, the Defense Communication Agency (DCA) and DARPA established a protocol known as Transmission Control Protocol and Internet Protocol (TCP/IP) as the standard protocol used on the network (ARPANET) and the conversion to TCP/IP on the ARPANET was completed on Jun. 1983 and the term "INTERNET" come into use. In 1989, ARPANET was split into two networks namely MILNET (military network) and ARPANET (for non-DOD sites).

Stage 3 - National Science Foundation

In 1986, the DOD assigned the ARPANET project to the National Science Foundation (NSF) and a network connecting five supercomputers centers was established by the NSF to enhance computer and network research capabilities in the USA. The NSF network and other networks makes up the largest network of networks; the Internet.

Stage 4 - The World Wide Web (WWW)

The 1st public distribution of the World Wide Web (WWW) client Software on the Internet began in 1989 and became public in 1992. The WWW client software was created by Tim Berners at CERN. He proposed a hypertext system enabling scientist studying high-energy physics to share information more efficiently. The WWW uses the Internet Protocol http:/ to present information through multimedia format; ie graphics, sound, animation and video.

3. Advantages and Uses of the Internet

Advantages of the internet

  • Speedy communication
  • Inexpensive communication
  • It provides convenient communication

Uses of internet

  • Communication services: e.g. E-mail, Chatting, Conferencing
  • Commerce and industries: It facilitate trade and is replacing traditional practices e.g. Tel, Fax, Letters. It also enables E-business and E-commerce (Online shopping, Online banking).
  • Health: It provides information, daily news and other resources about health.
  • News and information: One can get headlines information and in-depth stories on various topics.
  • Job opportunity: With the use of internet, one can find an ideal job.
  • Education: Virtual learning, reading books and magazines, and in-service training.
  • Software download: One can download and use a program from the internet.
  • Finding people and organization: One can find people or an organization by using the internet.

4. Internet Connection

The following elements are required for one to access the internet:

  1. Computer
  2. Modem: A modem is device that form a connection (via telephone or cable-TV lines), between your computer & a computer network that provides internet access.
  3. Internet connection:
    • Dial-up connection: It accesses the internet on a per-use basis & uses a modem.
    • Direct connection: Provide continuous, "Always on" access to the internet. Direct connection are commonly referred to as Broadband connection.
  4. Internet software (Web-Browser): Communication software is needed to manage your connection to the internet. Examples include Internet Explorer, Opera Mini, Mozilla Firefox, Netscape.

5. How the Web Works

What makes up the Web(Internet)

Internet is made-up of 3 components namely:-

  • Web site: It's made up of collection of web pages.
  • Web pages: It's a single page in a website. It provides a single screen of information which may contain text, images, animation, sound, video etc.
  • Links: These are Hypertext or Hypermedia that is used to move from one page to another.

Hypertext transmission protocol (HTTP)

This is the protocol that Web-client and servers computers use to communicate with each other. The web works under the popular client-server model.

Hypertext markup language (HTML)

This is the language the web uses to creating and recognizing hypermedia documents. Documents are usually named with the suffix ".html".

Uniform resource locator (URL)

URL is the Internet address of a particular Web Page. It has two basic parts: a portion specifying the method of file access (e.g., http://) and a portion specifying the internet location of the file.

6. Surfing and Searching the Web

Surfing/Browsing the internet

You start up your browser in the same way as any other software. To request a web page, you can type a "URL" in the address bar, use the "open page" menu, or use keyboard shortcuts.

Searching the web (Using Search Engines)

Search engines allow the users to type keywords that are run against a database. They use software "spiders" to crawl the web and build its database. When you enter a query, your input is checked against the search engine's keyword indexes. Examples of search engines are Google, Yahoo, MSN, etc.

Common surfing problems

When using web, common problems includes URL not working, dead links, slow loading content, and telecommunication breakdown. You might also encounter error codes like 404 - file not found, 500 - server error, etc.

Web directories

This is a hierarchical organized Index of subjects that allow the web searcher to browse through a list of web sites by subject. Examples include Yahoo and Open Directory Project (dmoz).

7. Electronic Mails (E-mail)

It's a type of mail that is electronic, whether it's transmitted over the internet or not.

E-mail software

E-mail allows one to compose, send and receive e-mails. The most e-mail software includes: - Microsoft Outlook, Netscape communicator, Eudora etc. The e-mails are stored in separate folders described below:

FolderDescription
InboxStore incoming e-mails.
Outbox/sentContain sent messages.
DraftEmails that are being worked on and not yet ready to be sent.
Deleted itemsEmails that have been deleted.

Composing an e-mail

This is the process of writing an email. Most header elements in the new e-mail window are:

  • To: address of your recipient
  • Subject: a few words outlining the nature of your message
  • CC: copy an email to other recipient
  • BCC: copy an email to other recipient without any indication of that action appearing in the main recipient's copy
  • Attachments: to attach a file to your email

Sending, receiving and forwarding an E-mail

Once one has finish composing an email, he/she can send it immediately or save it as draft. Once an email has been sent, the sent message automatically appears in the "sent items/outbox".

Attaching documents to E-mail

One can attach a file, document, picture, sound or a movie to an email before sending it.

Mail Etiquette

When writing an email, the following measures should be taken into consideration:

  • Keep emails short and clear
  • Avoid forwarding other people massages
