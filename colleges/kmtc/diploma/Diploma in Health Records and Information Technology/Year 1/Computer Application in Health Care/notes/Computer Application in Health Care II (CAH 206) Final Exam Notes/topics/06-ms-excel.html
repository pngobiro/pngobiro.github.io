Spreadsheet Fundamentals

Definition of Spreadsheet

A spreadsheet is a computer application package used for manipulating numerical information. It consists of a series of Rows and Columns. The intersection of a row and a column is called a CELL. Figures and text can be inserted into the cell. Every cell is referenced by the Column Label and Row Number.

Manual verses electronic spreadsheet

MANUAL

  • Manual calculation
  • Difficult to correct or edit figures
  • Hard to insert charts, pictures and graphics

ELECTRONIC

  • Electronic/automatic calculation
  • One can easily play with figures
  • Easy to insert charts, pictures and graphs

Facilities provided by electronic spreadsheet

  • Formulas - used to calculate new values automatically E.g. sum, product, average Etc.
  • Data formatting features - both text and numerical data can be formatted using predefined format E.g. fonts, size, styles, decimals, currency, date Etc.
  • The ability to adjust column widths and row heights, hide rows/column and freeze panes
  • Automatic recalculation - when you change one figure, the rest of the figures are automatically recalculated by the computer to correspond with the different inputs
  • Data sorting and filtering capabilities
  • Chart making E.g. line, bar, charts, pie charts or histograms
  • Data validation
  • Printing of entire or portion of a document

Components of a spreadsheet

  • Work sheet - cells organized into Rows and Column. Groups of worksheets is called Workbook
  • Data base - list of items stored in a worksheet
  • A graph/chart - can be created from data stored in a worksheet

Application/uses of spreadsheet

  • Used for storing data records
  • Used for numerical calculation
  • Researchers can compile and analyze their results using spreadsheet
  • Teachers can compile their student's marks and produce overall results
  • Data management. E.g. Sorting, Filtering and tallying
  • "What if" feature can be used to forecast and outcome.

Working with Worksheets

Features of Excel Window

  • Worksheet - one sheet in a workbook. It's made up of Rows and Columns of information
  • Workbook - this are user file that hold information. It consist of several worksheet
  • Column - vertical divisions of a worksheet that are identified by letters ( A, B, C, ..... Z, AA, AB, ...)
  • Row - horizontal divisions of a worksheet that are identified by numbers. They are 65536 Rows in a worksheet
  • Cell - intersection of a Row and a Column. It has an address that consist of a column letter and a row number. The current (Active) cell is highlighted by a thick borders and its address is displayed in the address bar.
  • Label - text entry into the worksheet that consist of alphanumeric characters that provide descriptive information (E.g. Names) and have no numerical/mathematical significance in Excel
  • Value - data that has numerical/mathematical significance E.g. Numbers, Date, Time

Performing Calculations

Formulas are used to perform calculation e.g. Addition, subtraction, multiplication and division. Formulas include cell addresses that reference to the cell on which you want to perform the calculation and the mathematical operation. The more complex calculations are left to Excel Function. The formula begins with the Equals sign (=) which tell Excel that it's a calculation operation.

Order of Operations

The natural order of performing mathematical operation are:-

  1. Whatever is in the Brackets
  2. Calculate the Exponential (^)
  3. Calculate Multiplication(*) and Division(/)
  4. Calculate Addition(+) and Subtraction(-)

In case of Multiplication/division, Addition/Subtraction, the operations will be executed in the order they appear in the formula from left to right.

Entering Formulas

  1. Select the cell where you will place the formula
  2. Type the Equal sign (=)
  3. Enter the appropriate values (), when finish, press Enter

Using the Auto Calculate on the status Bar

Select the cells and looking at the Status Bar, to select different formulas, right-click the Auto-calculate area of the Status Bar.

Displaying Formulas

Formulas are not normally displayed in the cell; instead results of the calculation are displayed. To view the formula, select the cell and look at the Formula Bar. One can also go to View Tab, → formula. To edit the formula, use the Formula Bar.

Excel Error Messages

Error occur when a formula cannot resolve to a Value answer. Some of the errors include:

Error Code Description
#DIV/0 The formula is attempting to divide by zero
#N/A Refers to a cell that contain no value or #N/A entry (null)
#NAME? Does not recognize a Function, Range or Name
#NULL! The formula refers to an invalid Range (remember to use commas and not space btw functions argument)
#NUM! Problem with the number used in the formula
#REF! Cell reference in the formula is incorrect
#VALUE! Wrong type of argument/operation e.g. wrong syntax of the formula
CIRCULAR Appears on status bar, the formula refers to the same cell in which the formula is written

Cell Referencing

Cell referencing is the location identity of a cell or a range of cells on the worksheet. They include:

Type Refers to
B6 Cell column "B", row "6"
A10:A20 Range of cells in column "A" from row "10" through "20"
B15:E15 Range of cell in Row 15 from columns "B" through "E"
5:5 All cells in row 5
5:10 All cells in row 5 through 10
H:H All cells in column "H"
H:J All cells in column "H" through "J"
Sales!(C20:C30) Information stored in sheet "Sales" from range C20 - C 30

  2. Click the name box in the formula bar
  3. Type the name of the cell e.g. "Rongai" and press Enter /OR/
  4. Right click the selected range and select "NAME RANGE"

Relative Cell Reference

Formula are based on their position relative to the Cell/Range that contains the formula. When you copy a formula that uses Relative cell reference, the pasted formula will refer to a different cell, but with the same relative position with the original formula.

Absolute Cell Reference

The cell referred to is the same, no matter which cell refers to it. To make a cell reference absolute, type the dollar sign" $ " before the column latter and before the Row number e.g. =$A$2 i.e. B2=$A$2*10.

Mixed Cell Reference

A reference that is only partially absolute e.g. A$2 or $A2 is referred to as mixed cell reference. When used/copied to another cell, only the relative part of the reference is adjusted.

Using Functions

A function is a ready-made formula, built into the spreadsheet to perform a series of operations on a specified value/range of value e.g. =AVERAGE(A1:A10). There are many functions, categorized into:

Financial Mathematical Trigonometrical
Date Time Statistical
lookup Reference Text
Logical Database Information Function

A function consist of three element:

  1. The " = " sign - indicates the start of a function
  2. Name - e.g. SUM → indicates the operation to be performed
  3. The Arguments - values to be acted upon by the function and are enclosed in brackets

Commonly used excel function are:-

  • AVERAGE - calculates the mean/average e.g. =AVERAGE(A1:A10)
  • COUNT - count the number of cells that contain values in the selected range e.g. =COUNT(B1:B5)
  • MAX - returns the maximum value in a range of cells e.g. =MAX(B4:B20)
  • MIN - =MIN(D1:D6) - Returns the minimum value
  • SUM - =SUM(A1:B5)
  • IF - =IF(A5>1500, "Bonus", "No Bonus") - place a conditional function in a cell
  • PMT - =PMT(0.125/12, 120, 150,000) - Calculates the monthly repayment over time e.g. ( 120 monthly repayment)at 12.5%(0.125) interest par year (0.125/12) for Kshs 150,000/=

Charts and Data Management

Charts

There are many types of charts namely:-

  • Pie chart - used to show the relationship among parts of a whole so that one can compare items readily
  • Column chart - illustrate individual values at a point in time and is suitable for side-by side comparison of numerical values
  • Bar chart - similar to column chart but display data horizontally rather than vertically
  • Line chart - display linear and unbroken change in large values. use to illustrate trends and changes of values over time
  • XY(Scatter) chart - similar to line chart, use to show heavy occurrence or absence of data
  • Area chart - show change in values over time. Use to give a bigger sense of the size of a change or concept

One can:

  • Create and save a chart - select the data that you want to include in the chart including the column and/or row labels that you want to include in the chart
  • Select a chart objects - E.g. title, legend, data series etc.
  • Add new chart data - point to the edge of the selected range until the pointer changes to an arrow, the drag the data anywhere onto the chart and release the mouse button
  • Change the chart style - click on the chart and choose the new chart type you want
  • Add a title and legend
  • Format chart title and legend
  • Format the chart Axes
  • Format chart data series

Sorting and Filtering Data

  • Sorting - it's arranging data according to a specified field, in ascending or descending order
  • Filtering data - it's displaying data that only meet a given criteria. It's when you have a large worksheet and you are only interested in a small portion.
