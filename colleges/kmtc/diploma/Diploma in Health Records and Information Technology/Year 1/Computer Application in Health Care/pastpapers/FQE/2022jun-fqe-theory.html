Section 1: Multiple Choice Questions

30 Marks
1.
The database designed to consist of multiple table that are linked together through matching data stored in each table is called ____
A. Object oriented database
B. Hierarchical database
C. Relational database
D. Network database
Show Answer

Correct Answer: C. Relational database

A relational database organizes data into tables which can be linked (related) based on data common to each.

2.
In E-R diagram attributes are represented by:
A. Triangle
B. Square
C. Rectangle
D. Ellipse
Show Answer

Correct Answer: D. Ellipse

In Entity-Relationship diagrams: Rectangles represent Entities, Ellipses represent Attributes, and Diamonds represent Relationships.

3.
Count function in SQL returns the number of ____
A. Groups
B. Distinct values
C. Columns
D. Values
Show Answer

Correct Answer: D. Values (or Rows)

COUNT(*) returns the number of rows. COUNT(column_name) returns the number of non-null values in that column.

4.
The statement in SQL which allows change of the definition of a table is ____
A. Select
B. Update
C. Alter
D. Create
Show Answer

Correct Answer: C. Alter

The ALTER TABLE statement is used to add, delete, or modify columns in an existing table.

5.
What is the advantage of the database management approach?
A. Data redundancy increases
B. Data is integrated and can be accessed by multiple programs
C. Data is dependent on programs
D. None of the above
Show Answer

Correct Answer: B. Data is integrated and can be accessed by multiple programs

DBMS reduces redundancy and allows shared access to integrated data.

6.
Which term is Row synonymous with?
A. Field
B. Column
C. Record
D. Relation
Show Answer

Correct Answer: C. Record

In a relational database table, a row represents a single record.

7.
In a one to many relationship, the entity that is on the one side of the relationship is called a ____ entity.
A. Parent
B. Child
C. Instance
D. Subtype
Show Answer

Correct Answer: A. Parent

The "One" side is the Parent (or Primary) table, and the "Many" side is the Child (or Related) table.

8.

A. Make table
B. Define table
C. Alter table
D. Create table
Show Answer

Correct Answer: D. Create table

9.
In database management, what do you mean by one to many relationship between student and class table?
A. One class may have many students
B. Many classes may have many classes
C. Many students may have many classes
D. One student can have many classes
Show Answer

Correct Answer: A. One class may have many students

Assuming "Class" refers to a specific course session, one class contains many students. (Note: In many real-world scenarios, this is Many-to-Many, but in basic modeling contexts, Option A is the standard intended answer for 1:M).

10.
To set a password, you first need to open the database in ____ mode.
A. Password
B. Sharing
C. Exclusive
D. Read Only
Show Answer

Correct Answer: C. Exclusive

In MS Access, you must open the database in "Open Exclusive" mode to set an encryption password.

11.
Referential integrity means:
A. Do not enter value in the primary key field of a child table if that value does not exist in the primary key of the parent table
B. Do not enter a value in the foreign key field of a parent table if that value does not exist in the primary key of the child table
C. Do not enter a value in the foreign key field of a child table if that value does not exist in the primary key of the parent table.
D. All of the above
Show Answer

Correct Answer: C. Do not enter a value in the foreign key field of a child table if that value does not exist in the primary key of the parent table.

This ensures that every record in the child table points to a valid, existing record in the parent table.

12.
Which of the following data types is used to attach Microsoft files, graphics files, sound files and pdf files to a table?
A. Memo
B. Hyperlink
C. OLE Object
D. Text
Show Answer

Correct Answer: C. OLE Object

Object Linking and Embedding (OLE) allows embedding documents and other objects. (Note: Newer Access versions use the "Attachment" data type, but OLE Object is the classic answer).

13.
____ is a collection of related fields?
A. Record
B. File

D. Relation

Correct Answer: A. Record

Fields (columns) grouped together form a Record (row).

16.
In a relational schema, each tuple is divided into fields called:
A. Domains
B. Queries
C. Relations
D. All of the above
Show Answer

Correct Answer: A. Domains (Technically: Attributes)

Tuples (rows) are divided into Attributes (columns). "Domains" refers to the set of allowable values for an attribute. In the absence of "Attributes" as an option, "Domains" is the most related structural concept among the choices.
