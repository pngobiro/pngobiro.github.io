Section 1: MS PowerPoint

Presentation
1.

Create a folder on the desktop and name it with your examination number. In your folder, prepare and save the following work under the file name PowerPoint.

  1. Open a blank presentation, in the first slide, enter the title as 'Kenya Medical Training College", sub title as "Department of Health Records and Information". (2 marks)
  2. Slide 2
    • Title: Disease Patterns
    • Text:
      • Endemic
      • Sporadic
      • Hyper endemic
      • Epidemic
      • Outbreak
      • Cluster
      • Pandemic
  3. Slide 3
    • Title: Pandemic
    • Text: Corona Virus (in a text box) (4 marks)

Formatting:

  1. Apply the 'facet' theme of the presentation. (4 marks)
  2. For slide 1: apply a 'push up' transition. (4 marks)
  3. For slide 2:
    • (i) Apply 'float In' animation of the disease pattern. (2 marks)
    • (ii) Apply "wheel" animation under "entrance" cluster. (2 marks)
  4. Insert date and footer "epidemiology" in your presentation. (4 marks)
Show Practical Steps

PowerPoint Steps:

1. Setup & Content:

  • Open PowerPoint. Save As PowerPoint in your folder.
  • Slide 1: Type Title and Subtitle as instructed.
  • Slide 2: New Slide (Title and Content). Type Title "Disease Patterns". Type the list in the content box.
  • Slide 3: New Slide (Title Only). Type Title "Pandemic". Insert > Text Box. Draw box and type "Corona Virus".

2. Formatting:

  • Theme: Design Tab > Themes > Search/Select "Facet" (Green geometric pattern usually).
  • Transition (Slide 1): Select Slide 1. Transitions Tab > Select "Push". Effect Options > "From Bottom" (Push Up).
  • Animations (Slide 2):
    • Select the list text. Animations Tab > Select "Float In".
    • Note on part (ii): "Wheel animation under entrance cluster". If this refers to the same text or a specific part, select the text again > Add Animation > Entrance Effects > Wheel.
  • Footer: Insert Tab > Header & Footer. Check "Date and time". Check "Footer" and type "epidemiology". Click "Apply to All".

Section 2: MS Access

Student Fees Database
2.

Create a database in your folder and name it Access.

Activity: You have been posted to a newly established KMTC campus where the accountant needs help in creating a working database to manage students' fees. The requirements are as given below:

1. Table 1: Student details

Admission number Should allow numerics only and it is automatically generated (2 mks)
Names Capture three names and should not allow a null entry (1 mk)
Gender Provide a drop down menu of male and female (2 mks)
Marital status Provide a drop down menu for single, married, divorced, widow, widower and separated (2 mks)
Age Only a number not exceeding 60 years (1 mark)
Department Provide a drop down menu of "health records and information, clinical medicine, nursing and physiotherapy (1 mk)
Programme Provide a drop down menu of HRI-certificate, HRI- preservice diploma, HRI-upgrading diploma, Clinical Medicine, Nursing-KRN, Nursing-KRCHN and Physiotherapy.

2. Table 2: Fee Payment

Admission number Foreign Key linking to Student details.
Fees Due Equal to KShs. 90,000 by default. Any other figure entered should be rejected with a note "enter the approved fees" (2 mks)
Fees Paid Must be equal to or greater than KShs. 40,000. A lesser amount entered shall be rejected with a note, "Note students must pay a minimum of KShs, 40,000 for admission to proceed. (2 mks)
Date By default it should capture the current date automatically after an entry (2 mks)
Balance Difference between fees due and feed paid (2 mks)

NB: All the currency fields should indicate a cumulative total at the bottom of the table (2 mks)

  1. Create a query for HRI and Nursing departments capturing students with a balance equal to or greater than KShs. 30,000 with the following fields: Admission Number, Names, Department, Programme, Fees paid and balance. (8 marks)
  2. Forms:
    • Create a form for student details using the form wizard and choose the ‘columnar’ layout for your form. (5 marks)
    • Insert a button to close window with the appropriate picture on the button. (2 marks)
    • Insert a button to 'find record' in your form using the relevant picture. (2 marks)
    • Apply red colour to the field labels and size 'to fit'. (2 marks)
    • Insert a picture of your choice in the form. (2 marks)
Show Access Steps

MS Access Procedures:

1. Table 1: Student details

  • Admission number: Data Type = AutoNumber (automatically generated) OR Number (Long Integer) if you want specific numbers. Instructions say "numerics only... automatically generated", suggesting AutoNumber. Set as Primary Key.
  • Names: Short Text. Field Property Required = Yes.
  • Gender: Short Text. Lookup Wizard > I will type values > Male; Female.
  • Marital status: Short Text. Lookup Wizard > Type values (Single, Married, etc).
  • Age: Number. Validation Rule: <=60. Validation Text: "Age must not exceed 60".
  • Department: Lookup Wizard > Type values.
  • Programme: Lookup Wizard > Type values.

2. Table 2: Fee Payment

  • Fees Due: Currency. Default Value: 90000. Validation Rule: =90000. Validation Text: "enter the approved fees".
  • Fees Paid: Currency. Validation Rule: >=40000. Validation Text: "Note students must pay a minimum of KShs, 40,000 for admission to proceed."
  • Date: Date/Time. Default Value: Date().
  • Balance: Calculated Field (Access 2010+). Expression: [Fees Due] - [Fees Paid].
  • Totals Row: Open table in Datasheet View. Home Tab > Records Group > Totals. Select "Sum" at bottom of Currency columns.

3. Query:

  • Create > Query Design. Add both tables (link via Admission Number).
  • Fields: Admission Number, Names, Department, Programme, Fees Paid, Balance.
  • Criteria for Department: Like "*HRI*" OR Like "*Nursing*" (or specify exact strings).
  • Criteria for Balance: >=30000.

4. Form (Student Details):

  • Create > Form Wizard. Select Student details table. Layout: Columnar.
  • Buttons: Design View > Button Control.
    • Action: Form Operations > Close Form. Show Picture.
    • Action: Record Navigation > Find Record. Show Picture.
  • Labels: Select all Labels (on the left). Format Tab > Font Color: Red. Layout Tab > Size > To Fit.
  • Picture: Design Tab > Insert Image.

Section 3: Web Design

HTML Coding
3.

As a Health Records and Information officer in the new campus you have been asked by your supervisor to prepare a website to market the new campus. Using HTML language, develop a website with the following features:

Kenya Medical Training College - New campus

Programmes offered in this campus are:

  1. Health Records and Information
  2. Nursing
  3. Clinical Medicine
  4. Physiotherapy

Welcome and be part of our success story

Save this website in your folder as website.

Show HTML Code

Copy the code below into Notepad and save as website.html:

<!DOCTYPE html>
<html>
<head>
    <title>KMTC New Campus</title>
</head>
<body>

    <!-- Heading 1 -->
    <h1>Kenya Medical Training College - New campus</h1>

    <!-- Heading 2 -->
    <h2>Programmes offered in this campus are:</h2>

    <!-- Ordered List (a, b, c...) -->
    <ol type="a">
        <li>Health Records and Information</li>
        <li>Nursing</li>
        <li>Clinical Medicine</li>
        <li>Physiotherapy</li>
    </ol>

    <!-- Paragraph Text -->
    <p>Welcome and be part of our success story</p>

</body>
</html>
