Database Creation (MS Access)

70 Marks
1.

Task Requirements:

  1. Open the computer and create a folder on the desktop with DHRI and the last three digits of your exam number (e.g., DHRI001). (3 marks)
  2. Using Microsoft Access, create a database called Student Registration consisting of two tables: Students and Faculty. (5 marks)
  3. The Student table has the following fields with the following properties: (10 marks)
    • Student ID - data type is Text, field size is 4 and Caption is ID #. It is also the primary key.
    • Last Name is Text, field size is 15.
    • First Name is Text, field size is 15.
    • Course is Text, field size is 30.
    • Lecturer ID is Text, field size is 4.
    • Marks is Number and its field size is Integer.
  4. The Faculty table has the following fields with the following properties: (10 marks)
    • Lecturer ID - is Text, the field size is 4, the Caption is Fac. ID and it is the primary key.
    • Last Name - is Text and the field size is 15.
    • First Name is Text, field size is 15.
    • Office Phone is Text, field size is 4 and the Caption is Phone.
  5. Create a form with all fields on the Student table. Name the form STUDENT DATA. (10 marks)
  6. Enter the data below into the database. (10 marks)

Step-by-step MS Access Procedures:

1. Setup:

  • Create folder DHRI001 on desktop.
  • Open Access > Blank Database > Name: Student Registration.

2. Creating 'Student' Table:

  • Create > Table Design.
  • Field Definitions:
    • Student ID: Short Text. Field Size: 4. Caption: ID #. Right-click > Primary Key.
    • Last Name: Short Text. Field Size: 15.
    • First Name: Short Text. Field Size: 15.
    • Course: Short Text. Field Size: 30.
    • Lecturer ID: Short Text. Field Size: 4.
    • Marks: Number. Field Size: Integer (or Long Integer, but Integer specified).
  • Save table as Student.

3. Creating 'Faculty' Table:

  • Create > Table Design.
  • Field Definitions:
    • Lecturer ID: Short Text. Field Size: 4. Caption: Fac. ID. Right-click > Primary Key.
    • Last Name: Short Text. Field Size: 15.
    • First Name: Short Text. Field Size: 15.
    • Office Phone: Short Text. Field Size: 4. Caption: Phone.
  • Save table as Faculty.

4. Creating Relationship:

  • Database Tools > Relationships.
  • Add both tables.
  • Drag Lecturer ID from Faculty table to Lecturer ID in Student table.
  • Check "Enforce Referential Integrity". Click Create.
  • Note: Enter Faculty data FIRST to avoid referential integrity errors, or create relationship after data entry if allowed, but strict RI requires Parent (Faculty) records to exist before Child (Student) records reference them.

5. Data Entry:

  • Open Faculty table first (Parent table). Enter the 5 lecturer records.
  • Open Student table (Child table). Enter the 6 student records.

6. Creating Form:

  • Select Student table.
  • Create > Form Wizard (or Form).
  • Select all fields.
  • Save form as STUDENT DATA.

7. Creating Queries:

Query 1 (All Students):

  • Create > Query Design > Add Student table.
  • Add fields in order: First Name, Last Name, Course, Marks.
  • Save as Query1 (or descriptive name).

Query 2 (Lecturer 2222):

  • Create > Query Design > Add Student table.
  • Add fields: First Name, Last Name, Marks, Lecturer ID.
  • In Criteria row under Lecturer ID, type "2222".
  • Uncheck "Show" for Lecturer ID if you only want to see names and marks, or leave it to verify.
  • Save as Query2.
