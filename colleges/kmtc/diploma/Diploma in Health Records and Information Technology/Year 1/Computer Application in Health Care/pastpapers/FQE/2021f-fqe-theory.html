1.
To start Microsoft PowerPoint application
A. Click on start >> progress >> all programs >> Ms power point
B. Hit ctrl+e then type ppoint.exe and enter
C. Click start >> run then type power point then press enter
D. Hit ctrl+s then type power point
Show Answer

Correct Answer: A. Click on start >> programs >> all programs >> Ms power point

Note: Option C usually requires typing "powerpnt" (not power point with space) in the Run dialog, making A the most standard GUI method described.

2.
PowerPoint presentations are widely used for the following except
A. Note outlines for teachers
B. Project presentations by students
C. Communication of planning
D. Mail set up and postage
Show Answer

Correct Answer: D. Mail set up and postage

This is a function of Email clients like Outlook, not PowerPoint.

3.
What PowerPoint feature will one use to apply motion affects to different objects of a slide
A. Slide transition
B. Slide design
C. Animation objects
D. Animation scheme
Show Answer

Correct Answer: C. Animation objects (Custom Animation)

To apply motion to specific objects (text, images), we use Animations. Transitions apply to the slide itself.

What does router do in a network

25.
Which of the following layer is not a network support layer
A. Transport layer
B. Network layer
C. Data link layer
D. Physical layer
Show Answer

Correct Answer: A. Transport layer

In the OSI model, Layers 1-3 (Physical, Data Link, Network) are often called Network Support layers. Layers 5-7 (Session, Presentation, Application) are User Support layers. The Transport Layer (Layer 4) links the two groups.

26.
Which of the following is not a type of server

B. Application server
C. Cloud server
D. Bus server
Show Answer

Correct Answer: D. Bus server

Bus is a network topology, not a server type.

27.
Identify the odd one out of the following terms below
A. Virus
B. Identity theft
C. Hackers
D. Spam
Show Answer

Correct Answer: B. Identity theft

Identity theft is a crime/outcome. Virus (malware), Hackers (attackers), and Spam (junk mail) are vectors or agents of security threats.

28.
Which of the following devices is used to transfer data between LAN
A. Repeater
B. Hub
C. Routers
D. Bridges
Show Answer

Correct Answer: D. Bridges (or C. Routers)

Bridges connect two LAN segments. Routers connect different networks (e.g., LAN to WAN). Given "transfer data between LAN" (singular or segments), Bridges is the classic definition for connecting LAN segments. Routers connect distinct IP networks.

29.
How many types of IP addresses are there
A. Two
B. Three
C. One
D. Four
Show Answer

Correct Answer: A. Two

IPv4 and IPv6.

30.
Which of the following protocol sends electronic mail to the receiver's end
A. FTP
B. SMTP
C. POP 3
D. Both A and B
Show Answer

Correct Answer: B. SMTP

Simple Mail Transfer Protocol is used for sending mail. POP3/IMAP are for receiving.
