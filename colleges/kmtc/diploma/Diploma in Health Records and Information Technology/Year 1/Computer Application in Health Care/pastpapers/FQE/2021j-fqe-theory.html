Section 1: Multiple Choice Questions

30 Marks
1.
The process of finding errors and fixing them within a program is called?
A. Compiling
B. Debugging
C. Scanning
D. Executing
Show Answer

Correct Answer: B. Debugging

2.
Which tool do you use to create a query object?
A. Simple query wizard
B. Database wizard
C. Table query wizard
D. Simple filer wizard
Show Answer

Correct Answer: A. Simple query wizard

3.
Which of the following is the ascending order of data hierarchy?
A. Byte-Bit-Field-Record-File-Database
B. Bit-Byte-Field-Record-Database-File
C. Bit-Byte-Field-Record-File-Database
D. Bit-Byte-Record-Field-File-Database
Show Answer

Correct Answer: C. Bit-Byte-Field-Record-File-Database

Smallest to largest: Bit -> Byte (Character) -> Field -> Record -> File (Table) -> Database.

4.
Database access levels are specified so as to define who can access what in a database. It is identified through?
A. Password
B. Lock settings
C. Status
D. User ID
Show Answer

Correct Answer: D. User ID

The User ID identifies the user, while the password authenticates them. Access levels (permissions) are assigned to the User ID.

5.
A search value can be exact valve or it can be?
A. Wild card character
B. Relationships
C. Logical operator
D. Comparison operation
Show Answer

Correct Answer: A. Wild card character

Wildcards (like * or ?) allow searching for patterns rather than exact values.

10.
If a create 'Student' field' in 'Fees' table to store 'Student_id' of 'Student' table, then this 'Student' field in 'Fees' table is called?
A. Native key
B. Foreign key
C. Composite key
D. Primary key

Correct Answer: B. Foreign key

A key that links to the Primary Key of another table.

11.
The complete information about an entity in a database is called?
A. Record
B. Field
C. Data
D. Information
Show Answer

Correct Answer: A. Record

12.
The input mask wizard does not include the ability to do which of the following?
A. Store symbols with data
B. Use preset masks
C. Define placeholder
D. Force all uppercase
Show Answer

Correct Answer: D. Force all uppercase

Note: The Input Mask syntax uses the > character to force uppercase. However, usually, the Wizard focuses on placeholders and storing literals. In standard Access exams, this question often targets "Force all uppercase" as the outlier or "Define placeholder" if aiming at what the wizard *automates* vs what you type manually. Given the standard options, the wizard *does* handle presets, storing symbols, and placeholders. Forcing uppercase is a formatting feature often handled by the `Format` property, though `Input Mask` can do it too. In this context, B, A, and C are core wizard steps.

16.
Using the keyboard ____ will return a record in which the field value equals a defined text.
A. Like
B. Between
C. Is null
D. Equal
Show Answer

Correct Answer: D. Equal (or "=")

"Like" is for patterns, "Between" for ranges, "Is Null" for empty fields.

19.
In Ms PowerPoint, which of the following is not a part of slide design?
A. Design template
B. Design view

PowerPoint has "Normal View", "Slide Sorter", etc. Access has "Design View". PowerPoint has "Slide Design" pane which includes templates, schemes, etc.

What lets you create new presentation by selecting ready-made fort colour and graphic effects?
A. Animation scheme
B. Design template

21.
Which of the following is not one of PowerPoint views?
A. Presentation view
B. Slide show view
C. Slide views
D. Transition view
Show Answer

Correct Answer: D. Transition view

"Presentation view" is vague but "Slide Show" is valid. "Slide view" was an old view. "Transition view" does not exist as a view mode (it's a ribbon tab).
