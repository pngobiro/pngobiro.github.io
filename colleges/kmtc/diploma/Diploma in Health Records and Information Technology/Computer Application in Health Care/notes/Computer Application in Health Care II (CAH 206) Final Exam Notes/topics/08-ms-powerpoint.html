Microsoft PowerPoint

Introduction to Presentation Packages

Is a package used to prepare visual aids that can be used in meetings, discussions, forums, seminars e.t.c. or a package used to create designs that can be used for advertising events in form of slides.

Examples of Presentation Packages

These include:

  • Ms PowerPoint
  • Corel presentation
  • Soft maker presentation
  • Adobe premier

Terminologies

  1. Slides - This are individual screens / pages in a presentation.
  2. Transition - Refers to the replacement or wiping effects that PowerPoint uses when one slide is replaced by the next one during a slide show.
  3. Slide Show - Is the final view of the presentation on a full screen where all slides are shown one after the other.
  4. Animation - Is an effect applied to text or graphic so that they look as if in motion.
  5. Timing - This refers to the duration one slide takes on the screen before it's replaced by the next one during a slide show.
  6. Master slide - is the first slide within a presentation.

Creating a Presentation

Starting PowerPoint: Start → Program → MS PowerPoint

This opens a PowerPoint starter dialog-box which shows the different ways that may be used to create a presentation.

Presentation may be created in four (4) different ways.

  1. AutoContent Wizard
  2. Design Template
  3. Blank Presentation
  4. From an Existing File

Auto content Wizard

This enables the user to create a new presentation out of sample lines already created and saved in the computer. That is a kind of a template that gives guidelines and design that may be used to create a new presentation and it is done by way of replacing the information on slides with your new ideas that will form the new presentation.

To use AutoContent Wizard:

  • Start PowerPoint & click on the Office Button
  • Click on New from the list of commands, a New Presentation dialog box will appear
  • Under the Templates option, click on Installed templates Command
  • Select a theme of your choice from the list that appears then click Create

Design Template

This the one used to design a presentation using different background provided by PowerPoint. The selected background is applied on the slide known as the Master Slide. This method is commonly used to create a one-slide presentation such as a poster or an advertisement and when printed, it may appear like a Word Processing document.

To do this:

  • Start PowerPoint & click on the Office Button
  • Click on New from the list of commands, a New Presentation dialog box will appear
  • Under the Templates option, click on Installed themes Command
  • Select a theme of your choice from the list that appears then click Create

NB: Instructions include click to add text / click to add sub-title etc

From An Existing Presentation

This enables one to create a presentation in cases where identical presentations need be. Layout and design of existing presentation are used for a new one.

To do this:

  • Start PowerPoint & click on the Office Button
  • Click on New from the list of commands, a New Presentation dialog box will appear
  • Under the Templates option, click on New from existing Command
  • A New from existing presentation dialog box will appear.
  • Select / choose the location of the saved presentation that you want to use then click on Create new
  • Replace the information on the new slide with new information i.e. your own ideas

Blank Presentation

This allows the user to create a presentation from scratch, i.e. every element of the presentation (ideas, layout, text, graphics...) is determined by the user.

To do this:

  • Start PowerPoint & click on the Office Button
  • Click on New from the list of commands, a New Presentation dialog box will appear
  • Under the Templates option, click on Blank and recent Command
  • Select Blank presentation from the list that appears then click Create

Managing Presentations

Saving a Presentation

  • Click on Office button to display a list of commands
  • Move to "Save As" option to obtain a sub menu
  • From the sub menu click on PowerPoint 97_2003 presentation option
  • A save as dialog box will be displayed
  • Type the File Name in the File Name box
  • Select the name of the Folder / Drive in which to save the file
  • Select your favored destination then click Save.

Viewing Presentation

A PowerPoint Presentation can be viewed in different ways depending on the operation carried out. That is:

  1. Normal View
  2. Slide Sorter View
  3. Notes Page View
  4. Slide Show View

Normal View

This view is normally very useful during the initial stages of creating a presentation. One can quickly move to a particular slide by clicking on its slide number from the left panel for the actual slide to be displayed on the right panel.

Slide Sorter View

Is a view that displays all the slides in the presentation on one screen. It acts as the Print Preview to the presentation. It is very important when we wish to add / delete / remove slides.

Notes Page View

This view divides the screen into 2 parts - the slide at the top and the notes option at the bottom. The notes part displays the flow of ideas while that on the top displays the actual slide being worked on. The notes page view is used to edit the speaker notes as they will look when you print them out.

Slide Show View

Is the final view of a presentation that is shown to the audience where slides are shown one after the other on full screen. The slides may be made to 'transmit' automatically or manually by the mouse.

Switching Between Views

There are two ways to do this:

  1. Go to View tab then click on the suitable view under the Presentation views group.
  2. Click on the viewing commands on the right hand side of the status bar.

Working with Slides

Inserting Slides

After creating the first slide of a presentation (the master slide), most slides may be inserted to complete the show.

  • On the Home tab, click on arrow pointing downwards on the New slide command under the Slides group
  • Select the slide layout of your choice
  • The new slide will automatically be inserted in your presentation.
  • Enter text in your new slide
  • Or press Ctrl + M on the keyboard
  • Or right click on the last slide then from the shortcut menu click on New slide command

Slide Layout

This refers to how you want your information to be arranged on your slide. There are different types of slide layout depending on the type of information or data to be entered in a slide.

a. Inserting Tabular Information

This is used when information has to be presented in tables for clarity e.g. a price list.

b. Graphical Information

Helps in presenting information using graphs and charts e.g. comparison of some products e.t.c

c. Smart Art Graphic

Helps in presenting information in form of a list, Process, Cycle, Hierarchy, Relationship, Matrix and Pyramid.

d. Inserting Picture from file

This is used when you want to insert a picture that you have saved in your computer.

e. Inserting Clip Arts

Clip arts are used to make slides more attractive as well as easily understood.

f. Inserting Media Clip

This is used when you want to insert a short video / media clip in your presentation.

Deleting Slides

Is the removal of unwanted slides from a presentation. When deleted, a slide is completely removed with all its contents.

  • Go to View tab
  • Select Slide Sorter View under the Presentation views group
  • Select the slide to be deleted by clicking on it once
  • Press Delete key on the keyboard or Right click on the slide then click Delete slide

Running a Slide Show

A Slide Show is usually the final viewing of a presentation that is shown on the full screen where all the info in all slides is displayed in a sequence by which they are show, corresponding with the flow of ideas as set by the user. A slide show may be run either-

  1. Manually
  2. Automatically

Manually

A slide show is said to be running manually where a mouse click helps to move from one slide to another i.e. the user runs the show by clicking on the mouse. To do this, you can click "Slide Show" on the View tab, press F5, or click the Slide Show button on the status bar, then click the left mouse button to advance.

Automatically

Is the running of the show from the 1st slide to the last unassisted. This will involve initial setting, timings and transitions in the slides.

Transitions and Animations

Setting a Transition

This shows a professional progression of slides whereby each slide will appear and disappear automatically in a set style.

To do this:

  • Go to View tab and select Slide Sorter View.
  • While on Slide sorter view, click on the Animations tab.
  • Select the first slide then select the suitable transition under the Transition to this slide group.
  • Select the transition sound and speed.
  • Move to the Advance slide part and activate the Automatically after checkbox then set the time.
  • Select the next slide and apply the above settings.
  • Go to Slide show tab, click on Set up slide show command and ensure "Using timings if present" is selected.
  • Click OK.

Setting Animation Effects

Animation is that effect set where the slides appear as if in motion. Effects are applied to objects, texts and graphs in order to create suspense and/or arouse interest in the viewers. It is applied per object per slide.

To do this:

  • Display the Slides in Normal View then click on the slide that you want to work on.
  • Select the Object/Part you want to work on i.e. title, subtitle, text, clip art e.t.c
  • Click on the Animations tab then click on Custom animation under animations group.
  • A Custom animation dialog box will appear, click on the add effect command then from the options i.e. Entrance, Emphasis and Exit select the suitable animation effect(s).
  • Repeat the above procedure to all the other objects in the slide.
  • Click OK once done and repeat for other slides.
