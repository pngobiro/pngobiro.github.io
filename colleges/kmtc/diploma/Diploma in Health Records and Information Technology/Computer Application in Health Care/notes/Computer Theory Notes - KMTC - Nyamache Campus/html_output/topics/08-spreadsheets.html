MS-EXCEL Spreadsheets

Introduction to Spreadsheets

Spreadsheet is software that helps to substitute the paper worksheets in the offices. Spreadsheet displays data in the form of rows and columns. An intersection of row and column is known as a cell.

MS-Excel is a window based spreadsheet developed by Microsoft corporation. It includes all features of a spreadsheet package like recalculation, graphs and functions. It also provides many Mathematical, Financial and Statistical functions. Thus it is used in many scientific and engineering environments for analysing data. Excel can even hold graphic objects like pictures and images.

Some important features of MS-Excel:

  1. Window based application: Excel like all other applications has Toolbars, Shortcut Menus, Auto correct, Online help and Wizards.
  2. Workbooks: Workbooks are the files in which worksheets related to a project are held.
  3. OLE support: Object linking and Embedding is a feature through which Excel can contain any object like a document, a picture etc.
  4. Maintaining high volume of data: Excel can contain large volume of data. A worksheet can contain 65536 rows and 256 columns. A single cell can contain a maximum of 255 characters. One workbook can contain a maximum of 256 worksheets.
  5. Availability of functions: Several Mathematical, financial & statistical functions are available in an Excel package.
  6. Availability of Charts and Graphs: MS-Excel allows users to view data entered as tables in a graphical form as charts, which helps the user to easily understand, analyze data and compare data.
  7. Data Analysis Tools: MS-Excel provides a set of data analysis tools called Analysis Tool pack.
  8. Sorting capability: Excel has the capability of sorting any data in Ascending or Descending order.
  9. Auto fill feature: Excel has the feature which allows filling cells with repetitive data such as chronological dates or numbers and repeated text.

Getting started with Excel

An Excel document is called a workbook. By default, Excel workbook contains 3 worksheets designated as sheet 1, sheet 2, sheet 3. The extension name of excel workbook is .xls

We can start excel in many ways:

  1. Start → Programs → Microsoft office → Microsoft Excel → hit enter.
  2. Start → Run → Type Excel → hit enter
  3. Double click on the Microsoft application Icon.

Components of the Excel window:

  • Rows, columns and cell: In a worksheet rows are numbered from top to bottom (1 to 65,536) and columns are labeled with letters from left to right (A to IV - 256 columns).
  • Title bar: Contains the name of the program and the default name of the workbook (Book1).
  • Menu bar: Contains menus that include all the commands.
  • Tool Bar: Shortcuts for menu items. Standard and formatting toolbars are displayed by default.
  • Active cell: The cell in which you are currently working.
  • Formula bar: displays the contents of the active cell.
  • Name box: displays the cell address of the active cell (e.g., B6).
  • Worksheet area: The middle portion of screen where information or data can be entered.
  • Status bar: located at the very bottom of the screen, displays brief information.
  • Sheet tabs: appear above the status bar displaying the names of the worksheets.

Working with Data

Creating a New Workbook:

On the File menu, click New. The new workbook task pane appears at the right side of the screen. Click blank workbook. A new workbook with 3 worksheets appears.

Entering and Editing Data:

  1. Select the cell by clicking on it.
  2. Type in the information.
  3. Press the Enter key.

The easiest way to edit the contents of a cell is to select the cell and then retype the entry. Alternatively, to edit the data in a cell, press F2, or double-click the cell.

Formatting a worksheet:

Changing the style or appearance of data in a worksheet is called formatting.

  • Aligning data: By default, any text you enter in Excel is aligned to the left and any value or number is aligned to the Right. Use the alignment buttons on the formatting tool bar to change this.
  • Formatting Numbers: To format the number in a cell (e.g., change decimal places, add currency symbols), go to Format → Cell → Number tab.
  • Formatting Text: Text can be formatted using the buttons on the formatting toolbar (Font, font size, bold, Italic etc.).

Inserting Rows & Columns:

To insert a row, select the row below where you want the new row, and on the Insert menu, click rows. To insert a column, select the column to the right of where you want the new column, and on the Insert menu, click columns.

Changing Row height and Column width:

To resize a column or row, rest the mouse pointer on the boundary on the right side (for columns) or bottom (for rows), and drag the boundary until the required size is obtained. Or use Format → Column → Width / Format → Row → Height.

Sorting:

Highlight the cells that should be sorted and click the sort Ascending (A-Z) button or Sort Descending (Z-A) button found on the Standard toolbar.

Printing and Page Breaks:

To print the worksheet, select File → Print. To set a manual page break, select the row you want to appear just below the page break and choose Insert → Page Break.

Saving a workbook:

To save a workbook, go to File → Save (or Ctrl+S). Type the name and click Save.

Formulas and Functions

Formulas:

A formula is an equation that is used to perform calculations on data in a worksheet. In Excel, a formula starts with an equal (=) sign. MS-Excel evaluates the formula according to the order of precedence of the operators.

Example: To add the values in cells B2, C2, and D2 and display the result in F2, click on cell F2 and type =(B2 + C2 + D2) then press Enter.

Functions:

A function is a built-in, readymade formula that accepts data, perform calculations and returns results. To specify a range of cells, a colon (:) is used between the first and last cell addresses. Example: =AVERAGE(B1:B10).

Common Functions:

  • SUM(range): gives the sum of the values in a specified range.
  • ABS(number): gives the absolute value of the number.
  • FACT(number): gives the factorial of the number.
  • SQRT(number): gives the square root of the number.
  • LOG(number): gives the logarithm of the number.
  • AVERAGE(range): calculates the average of the values.
  • STDEV(range): calculates the standard deviation.
  • MAX(range): gives the maximum value within the range.
  • COUNT(range): counts how many numbers are in the list.

Function Wizard:

The Function Wizard is a tool in Excel that helps a user enter formulas in the correct format. To use it, select the cell, then choose Insert → Function. The wizard will guide you through selecting a function and entering its arguments.

Creating Charts

Charts in Excel are used to represent data pictorially. Common types include Column, Bar, Line, and Pie charts.

Steps to Create a Chart:

  1. Enter the data in a table.
  2. Open the Insert Menu and select Chart.
  3. In the Chart Wizard - Step 1 of 4, select the type of chart you want.
  4. In Step 2 of 4, enter the chart source data range.
  5. In Step 3 of 4, enter the chart options like titles and labels.
  6. In Step 4 of 4, choose the location for the chart and press Finish.

Statistical Analysis Tools

Microsoft Excel provides a set of data analysis tools called the Analysis ToolPak. To access these tools, click Data Analysis on the Tools menu. If the command is not available, you need to load the Analysis ToolPak add-in program. This toolpak includes tools for ANOVA, Correlation, Covariance, Descriptive Statistics, Histograms, Regression, t-tests, and more.
