Step-by-step Excel Procedures:

1. Data Entry:

  • Enter headers in Row 1. Enter data in columns A to F.
  • Enter "Maximum" and "Minimum" labels in column A below the data (e.g., A12, A13).

2. Calculations:

  • Total (G2): =SUM(B2:F2). Drag fill handle down.
  • Average (H2): =AVERAGE(B2:F2). Drag fill handle down.
  • Maximum (B12): =MAX(B2:B10). Drag across to other columns.
  • Minimum (B13): =MIN(B2:B10). Drag across to other columns.

3. Remarks (Nested IF Function):

In cell I2, enter the following formula:

=IF(H2>=80, "Excellent", IF(H2>=70, "Good", IF(H2>=60, "Above Average", IF(H2>=50, "Average", IF(H2>=40, "Poor", "Very Poor")))))

Drag the fill handle down to apply to all students.

Step-by-step MS Access Procedures:

1. Database Setup:

  • Create Blank Database named My Database[001].

2. Import Excel:

  • External Data > New Data Source > From File > Excel.
  • Select students' performance.xlsx.
  • Select "Import the source data into a new table".
  • Check "First Row Contains Column Headings".
  • Set Primary Key (Choose "Student Number").
  • Finish. Rename table to Performance Table if needed.

3. Creating STUDENTDATA Table:

  • Create > Table Design.
  • Fields:
    • Student Number: Short Text (Size 25). Set as Primary Key.
    • Full Name: Short Text (Size 25).
    • Contacts: Short Text (Size 12).
    • County: Short Text. In "Lookup" tab (lower pane):
      • Display Control: Combo Box.
      • Row Source Type: Value List.
      • Row Source: "Nakuru";"Mombasa";"Vihiga";"Nairobi".
  • Save as STUDENTDATA.

4. Creating Forms:

  • Select STUDENTDATA table > Create > Form. Save as StudentDataF.
  • Select Performance Table > Create > Form. Save as PerformanceF.

5. Form Formatting (StudentDataF):

  • Open StudentDataF in Design View.
  • Back Color: Click on Detail section background > Format Tab > Shape Fill > Select Green.
  • Border: Select input boxes > Format Tab > Shape Outline (Black) > Line Thickness (4pt).
  • Font: Select all controls (Ctrl+A) > Format Tab > Font: Arial, Size: 16.
  • Resize boxes to make text visible.

6. Relationships:

  • Database Tools > Relationships.
  • Add both tables.
  • Drag Student Number from one table to the other.
  • Enforce Referential Integrity. Create.

7. Query (Vihiga Students):

  • Create > Query Design. Add both tables.
  • Select fields: Student Number, Full Name (from STUDENTDATA), English, Biology (from Performance Table).
  • Criteria under County (add field if needed, uncheck show): "Vihiga".
  • Save query.

8. Report:

  • Select the query. Create > Report.
  • Save as Vihiga Report. Adjust layout in Layout View.
