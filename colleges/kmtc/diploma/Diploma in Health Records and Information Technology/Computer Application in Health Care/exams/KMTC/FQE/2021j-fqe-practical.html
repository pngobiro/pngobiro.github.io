Step-by-step MS Access Procedures:

1. Creating Patients Table:

  • Create > Table Design.
  • Enter Fields: Admission No. (Text/Number), Patient name, Admission date (Date/Time), etc.
  • Note: Set 'Payment method' to Text initially if entering text values like "Mpesa", or keep as Currency if intended for amounts (but the data shows text "Mpesa", "NHIF", etc., so use Short Text instead of Currency, despite the question saying Currency).
  • Save as patients table.

2. Creating Doctors Table:

  • Create > Table Design.
  • Enter Fields: Doctor Name, Mobile Number, etc.
  • Right-click Doctor ID row > Primary Key.
  • Save as Doctors table.

3. Relationships:

  • To link them, they need a common field. Looking at the data, Ward appears in both tables. Alternatively, the patients table might need a Doctor ID foreign key added, but based on strict instructions, we might link via Ward (though not ideal for normalization).
  • Database Tools > Relationships. Add both tables. Drag common field (e.g., Ward) to create link.

4. Gender Query:

  • Create > Query Design > Add patients table.
  • Add all fields.
  • Criteria under Gender: "M".
  • Save as Gender query.

5. Make Table Query (Payment Query):

  • Create > Query Design > Add patients table.
  • Fields: Covid vaccine, Admission No., Patient name, Gender, Payment method.
  • Ribbon > Make Table. Table Name: Payment Table (or similar).
  • Save query as Payment query. Run it to create the table.

6. Macro:

  • Create > Macro.
  • Action: OpenForm. Form Name: [Select Doctors Form].
  • Save.

Step-by-step PowerPoint Procedures:

1. Slide 1 (Title):

  • Type text: "COVID - 19", "Pfizer Ltd", "VACCINES FOR LIFE".
  • Select "COVID - 19" and "VACCINES FOR LIFE".
  • Home Tab > Font: Arial Black, Size: 88.

2. Slide 2 (Bulleted List):

  • New Slide > Title and Content.
  • Title: "Pfizer Ltd".
  • Content: Type the points.
  • Select the bullet points > Home Tab > Font Color > Green.

3. Slide 3 (Chart):

  • New Slide > Title and Content.
  • Click Insert Chart icon in the placeholder.
  • Select Column > Clustered Column (Multiple Bar).
  • In the Excel popup window:
    • Categories (Rows) = Pfizer, Moderna, Sputnik, Rwandese.
    • Series (Columns) = January, February, March.
    • Enter the data values accordingly.

4. Saving:

  • File > Save As > Desktop.
  • Filename: Pfizer 001.
