Section 1: Multiple Choice Questions

1.
In Ms Access, how can you create relationship between two tables in the GUI?
A. Drag the foreign key of a table into the primary key of another table
B. Drag any field from parent table and drop on child table
C. Drag the primary key of a table into foreign key of another table
D. Any of these can be done to create relationship
Show Answer

Correct Answer: C. Drag the primary key of a table into foreign key of another table

Explanation: To create a relationship in Microsoft Access, you typically drag the Primary Key field from the parent table (the "one" side) and drop it onto the corresponding Foreign Key field in the child table (the "many" side) within the Relationships window.

2.
The small button with three dots displayed at the right of field properties box is called
A. Make button
B. Expression button
C. Build button
D. None of the above
Show Answer

Correct Answer: C. Build button

Explanation: The button with three dots (...) is known as the Build button. Clicking it opens the Expression Builder or other relevant property builders (like Input Mask Wizard or Query Builder) depending on the property selected.

3.
The amend center of access file that appears when we create or open the Ms Access database file
A. Database window
B. Query window
C. Switch window
D. Design view window
Show Answer

Correct Answer: A. Database window

Explanation: In older versions of Access (like 2003), the main container that appeared upon opening a database was called the Database Window. In newer versions, this has been replaced by the Navigation Pane, but based on the options provided, Database window is the correct legacy term.

4.
How can you define a field so that when entering data, ##### is displayed?
A. Indexed
B. Validation role
C. IME mode
D. Input mask
Show Answer

Correct Answer: D. Input mask

Explanation: The Input Mask property controls how data is entered and displayed. By setting the input mask to "Password", characters typed are displayed as asterisks or other masking characters. The question refers to displaying placeholder characters or masking input.

5.
In which method we don't need to specify the field type while creating a new table
A. Design view
B. Created table by entering data
C. While using wizard
D. Report view
Show Answer

Correct Answer: B. Created table by entering data

Explanation: When you create a table by simply entering data (often in Datasheet View), Access automatically guesses and assigns the data type based on the value you type (e.g., if you type a date, it becomes a Date/Time field).

6.
What does the show check box in query design window indicate?
A. It indicates whether the field is to be used or not
B. It indicates whether the field is to be displayed in query result or not
C. It indicates whether the field name is to be displayed in query result or not
D. It indicates whether the field s to be used for grouping results or not
Show Answer

Correct Answer: B. It indicates whether the field is to be displayed in query result or not

Explanation: The "Show" checkbox in the query design grid determines visibility. You might want to include a field to filter results (Criteria) or sort by it, but hide it from the final output by unchecking this box.

7.
Which of the following option creates a Drop Down list of values?
A. OLE object
B. Hyperlink
C. Memo
D. Lookup Wizard
Show Answer

Correct Answer: D. Lookup Wizard

Explanation: The Lookup Wizard is used to create a field that allows you to choose values from another table or a custom list, typically presented as a drop-down combo box.

8.
Which kind of query that displays its own dialogue box prompting you for entering information before executing
A. Dialogue query
B. Select query
C. Parameter query
D. Crosstab query
Show Answer

Correct Answer: C. Parameter query

Explanation: A Parameter Query is designed to be flexible; it pauses execution to ask the user for input (criteria), such as a date range or ID, via a dialog box.

9.
Which operator is used to filter and display rows when all (two or more) conditions are Satisfied
A. IN
B. AND
C. ALL
D. RANGE
Show Answer

Correct Answer: B. AND

Explanation: The AND operator requires that all connected conditions must be true for the record to be selected.

  • OR: Requires only one condition to be true.
  • IN: Checks if a value exists in a list.
10.
The upper part of the query design window in Ms Access has two parts shows;
A. Filed name with field size
B. Field name with relationship
C. Fields and criteria
D. Sorting optio
Show Answer

Correct Answer: B. Field name with relationship

Explanation: The upper pane of the Query Design window displays the Tables (showing field names) and the Relationships (join lines) between them. The lower pane is the design grid showing fields, criteria, sort order, etc.

11.
Which feature is used to duplicate a control formatting
A. Manager
B. Control
C. Pointer
D. Wizard
Show Answer

Correct Answer: C. Pointer

Explanation: While the specific tool is called the Format Painter, the icon is a paintbrush, and using it involves changing the mouse pointer to apply the format. Among the given options, "Pointer" is the closest reference to the action of clicking to paint formats.

12.
Database locking refer to?
A. Preventing two individual from changing the same data at the same time
B. Using a password to prevent unauthorized access
C. Preventing accidents delete by using password lock
D. Ensuring data is not lost before exiting database
Show Answer

Correct Answer: A. Preventing two individual from changing the same data at the same time

Explanation: Record locking is a mechanism used in multi-user databases to prevent data inconsistencies (concurrency control). It ensures that when one user is editing a record, others cannot edit the same record simultaneously.

13.
How do you make data field DOB to store the dates like "2002/10/15"?
A. By entering dates in yyyy, mm, dd order
B. By setting the format property to 'yyyy/mm/dd'
C. By setting the input mask property to "yyyy/mm/dd"
D. By allowing access to pick date from calendar option
Show Answer

Correct Answer: B. By setting the format property to 'yyyy/mm/dd'

Explanation: The Format property determines how data is displayed. Setting it to 'yyyy/mm/dd' ensures dates appear in that specific order regardless of how they are stored internally.

14.
Which of the following data type is more commonly called Boolean?
A. True/false
B. Date/time
C. Attachment
D. Hyperlink
Show Answer

Correct Answer: A. True/false

Explanation: A Boolean data type represents two states: True or False (or Yes/No, On/Off). In Access, this is the Yes/No data type.

15.
Which linking code can be used to link a web page to kmtc.ac.ke?
A. <link="kmtc.ac.ke">KMTC </A>
B. <br> "kmtc.ac.ke" >< KMTC </A>
C. <URL="kmtc.c.ke">KMTC </A>
D. <AHREF="kmte.ac.ke"> KMTC </A>
Show Answer

Correct Answer: D. <AHREF="kmte.ac.ke"> KMTC </A>

Explanation: Although containing typos in the option (AHREF instead of A HREF, and kmte instead of kmtc), option D follows the closest syntax for an HTML anchor tag: <a href="url">Link Text</a>.

Which one of the following is a html tag

C. <ahref/>

23.
Computer memory is measured in terms of
A. Bytes
B. Bits
C. Units
D. Megabytes
Show Answer

Correct Answer: A. Bytes

Explanation: The Byte is the fundamental unit of measurement for computer memory storage. While bits are smaller and Megabytes are multiples, "Bytes" is the base unit.

24.
What type of chart is useful comparing values over categories?
A. Pie chart
B. Dot graph
C. Column chart
D. Line chart
Show Answer

Correct Answer: C. Column chart

Explanation: A Column chart (or Bar chart) is specifically designed to compare values across different categories. Pie charts show parts of a whole, and Line charts show trends over time.
