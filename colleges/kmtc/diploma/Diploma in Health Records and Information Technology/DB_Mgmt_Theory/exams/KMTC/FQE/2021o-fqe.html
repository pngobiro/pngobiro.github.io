Section 1: Multiple Choice Questions

1.
A collection of interrelated records is called a?
A. Database
B. Spreadsheet
C. Management information system
D. Text file
Show Answer

Correct Answer: A. Database

Explanation: A database is defined as a structured collection of data (interrelated records) stored and accessed electronically.

2.
Microsoft Access is a ______
A. Network Database Model
B. RDBMS
C. ORDBMS
D. OODBMS
Show Answer

Correct Answer: B. RDBMS

Explanation: Microsoft Access is a Relational Database Management System (RDBMS) that combines the relational Microsoft Jet Database Engine with a graphical user interface and software-development tools.

3.
Which one of the following is not an object of Ms-access?
A. Macro
B. Row
C. Screen input
D. File structure
Show Answer

Correct Answer: B. Row

Explanation: The standard objects in Microsoft Access are Tables, Queries, Forms, Reports, Macros, and Modules. A Row (or record) is a component of a Table, not a standalone database object.

4.
Which option allows us to build a new table by entering data directly into a tabular grid?
A. Datasheet View
B. Design View
C. Linked Table
D. Grid View
Show Answer

Correct Answer: A. Datasheet View

Explanation: Datasheet View displays the table in a grid format (rows and columns) and allows you to enter data directly, upon which Access can automatically define the field types.

5.
External database is _______
A. Database created using DBMS package
B. Database created in MS-Word
C. Database created in EXCEL
D. All of the above
Show Answer

Correct Answer: A. Database created using DBMS package

Explanation: A true database is created using a DBMS package. While Excel can store list data, it is a spreadsheet, not a database management system. MS Word is a word processor. Therefore, in the context of database definitions, A is the correct answer.

6.
A primary key in any table has the properties _______
A. Not Null
B. No Duplicates
C. Both A and B above
D. None of above
Show Answer

Correct Answer: C. Both A and B above

Explanation: A Primary Key must uniquely identify each record. Therefore, it cannot contain duplicate values (Unique) and it cannot be left empty (Not Null).

7.
Report generators are used to _______
A. Store data input by a user
B. Retrieve information from files
C. Answer queries
D. Both (b) and (c)
Show Answer

Correct Answer: D. Both (b) and (c)

Explanation: Reports are used to present data in a formatted way. To do this, they must retrieve information from the database tables and often effectively answer queries by presenting the specific data requested in a readable format.

8.
The logical data structure with a one-to-many relationship is a _______
A. Tree
B. Chain
C. Network
D. All of these
Show Answer

Correct Answer: A. Tree

Explanation: The Hierarchical model (often represented as a tree structure) is characterized by parent-child relationships where one parent can have many children, but a child has only one parent (1:Many).

9.
In a database, related fields are grouped to _______
A. File
B. Bank
C. Menu
D. Data record
Show Answer

Correct Answer: D. Data record

Explanation: A Record is a collection of related fields (e.g., Name, Age, Address) treated as a single unit.

10.
In one-to-many relationship the table in 'one' side is called ______ and on 'many' side is called ______
A. Child, Parent
B. Parent, Child
C. Brother, Sister
D. Father, Son
Show Answer

Correct Answer: B. Parent, Child

Explanation: In a relationship, the table containing the primary key (One side) is the Parent table, and the table containing the foreign key (Many side) is the Child table.

11.
The database environment has all the following components except _______
A. Database administrator
B. Database
C. Users
D. Separate files
Show Answer

Correct Answer: D. Separate files

Explanation: The database approach is designed specifically to replace the traditional file-processing systems. Therefore, "Separate files" (unintegrated data) is not a component of a database environment, which focuses on integrated data.

12.
Which one of the following is not a type of data type?
A. OLE Object
B. Number
C. Field size
D. Currency
Show Answer

Correct Answer: C. Field size

Explanation: OLE Object, Number, and Currency are valid data types in Access. Field Size is a property of a data type (e.g., specifying "Long Integer" for a Number type), not a data type itself.

13.
Which database handles full text data, image, audio and video?
A. Multimedia database
B. Video on demand database
C. Graphics database
D. Transaction database
Show Answer

Correct Answer: A. Multimedia database

Explanation: A Multimedia database is designed specifically to store, manage, and retrieve multimedia data such as text, images, audio, and video.

14.
What do you mean by one to many relationships between Teacher and Class table?
A. One class may have many students
B. One teacher can have many classes
C. Many classes may have many teachers
D. Many teachers may have many classes
Show Answer

Correct Answer: B. One teacher can have many classes

Explanation: In a "One-to-Many" relationship from Teacher to Class, a single Teacher record can be associated with multiple Class records.

15.
Define the term of input mask
A. It is a field property that controls the value of a record and sets in a specific format
B. It is an association among several entities
C. It's a collection of related fields that represent a single entity
D. It is a collection of related record
Show Answer

Correct Answer: A. It is a field property that controls the value of a record and sets in a specific format

Explanation: An Input Mask is a property that controls how data is entered by providing a template (e.g., `(___) ___-____` for phone numbers), ensuring data conforms to a specific format.

16.
In one-to-many relationship the table on 'many' side is called
A. Parent
B. Child
C. Sister
D. Master
Show Answer

Correct Answer: B. Child

Explanation: The table on the "Many" side of a relationship is referred to as the Child or Related table.

17.
In which state one gathers and lists all the necessary fields for the database design project.
A. Data Definition
B. Data Refinement
C. Establishing Relationship
D. None of The Above
Show Answer

Correct Answer: A. Data Definition

Explanation: This phase typically involves defining the data requirements, which includes listing all necessary fields, their types, and properties. It roughly corresponds to the Data Definition or Requirements Analysis phase.

18.
Which of the following enables us to view data from a table based on a specific criterion?
A. Form
B. Query
C. Macro
D. Report
Show Answer

Correct Answer: B. Query

Explanation: A Query is specifically designed to retrieve and view data based on specific criteria (filters) from one or more tables.

19.
Which database object is ideal to view, change, and analyze data in different ways?
A. Table
B. Form
C. Query
D. Report
Show Answer

Correct Answer: B. Form

Explanation: While queries analyze data, Forms provide an interface specifically "ideal to view, change, and analyze" data record by record in a user-friendly way. Tables allow this too but are less "ideal" for end-users. Queries are primarily for analysis/retrieval, not necessarily "changing" data as conveniently as forms.

20.
Which data type allows alphanumeric characters and special symbols to be entered?
A. Text
B. Memo
C. Auto number
D. Mixed
Show Answer

Correct Answer: A. Text

Explanation: The Text (or Short Text) data type allows any combination of letters, numbers, and symbols.

21.
The size of Yes No field is always _______
A. 1 bit
B. 1 byte
C. 1 kilobyte
D. 1 megabyte
Show Answer

Correct Answer: A. 1 bit

Explanation: A Yes/No field requires only 1 bit of storage (0 or 1).

22.
It is used to establish an association between related tables.
A. Line
B. Relationship
C. Primary Key
D. Records
Show Answer

Correct Answer: B. Relationship

Explanation: A Relationship is the association established between common fields (Primary and Foreign keys) in two tables.

23.
This key that uniquely identifies each record is called:
A. Primary Key
B. Key Record
C. Unique Key
D. Field Name
Show Answer

Correct Answer: A. Primary Key

Explanation: The Primary Key is the unique identifier for each record in a database table.

24.
Which field type will you select if you need to enter long text in that field?
A. Text
B. Memo
C. Currency
D. Hyperlink
Show Answer

Correct Answer: B. Memo

Explanation: The Memo data type (or Long Text) is used for storing large amounts of alphanumeric data (sentences and paragraphs).

25.
Which of the following is not a type of MS Access database object?
A. Table
B. Form
C. Worksheets
D. Modules
Show Answer

Correct Answer: C. Worksheets

Explanation: Worksheets are Excel objects. Access objects include Tables, Queries, Forms, Reports, Macros, and Modules.

26.
In which method we don't need to specify the field type and size while creating a new Table?
A. Design View
B. While using Wizard
C. Create Table By Entering Data
D. All of Above
Show Answer

Correct Answer: C. Create Table By Entering Data

Explanation: When creating a table by entering data (Datasheet View), Access infers the data types automatically based on the content entered.

27.
The size of yes/no field is always _______
A. 1 Bit
B. 1 Byte
C. 1 Character
D. 1 KB
Show Answer

Correct Answer: A. 1 Bit

Explanation: Duplicate of Question 21.

28.
The size of a field with number data type cannot be _______
A. 2
B. 4
C. 8
D. 16
Show Answer

Correct Answer: D. 16

Explanation: Standard Access numeric field sizes are Byte (1 byte), Integer (2 bytes), Long Integer (4 bytes), Single (4 bytes), and Double (8 bytes). While Replication ID uses 16 bytes, in the context of standard numeric choices taught in introductory courses, 16 is often considered the outlier or "incorrect" standard size compared to 2, 4, and 8.

29.
How can you edit a relationship already established between two tables?
A. Right click the relationship line and choose Edit Relationship
B. Double click the relationship line
C. Both of above
D. None of above
Show Answer

Correct Answer: C. Both of above

Explanation: You can open the Edit Relationships dialog box by either double-clicking the relationship line or right-clicking it and selecting "Edit Relationship".

30.
Which of the following option creates a Drop down List of values?
A. Ole Object
B. Hyperlink
C. Memo
D. Lookup Wizard
Show Answer

Correct Answer: D. Lookup Wizard

Explanation: The Lookup Wizard is used to create a field that allows the user to select a value from a drop-down list.

31.
What can be the maximum length of a text field?
A. 120
B. 255
C. 265
D. 75
Show Answer

Correct Answer: B. 255

Explanation: The maximum number of characters in a standard Text (Short Text) field in Access is 255.

32.
What Cascade Delete option is used for?
A. It is used to delete all the records of all tables in a database
B. It repeats the recent delete operation to all the records of current table
C. It ensures that all the related records will be deleted automatically when the record from parent table is deleted
D. It is used to delete all the records of related tables in a database
Show Answer

Correct Answer: C. It ensures that all the related records will be deleted automatically when the record from parent table is deleted

Explanation: Cascade Delete Related Records ensures referential integrity by automatically deleting any child records when the corresponding parent record is deleted.

33.
While creating relationship using mouse, how the mouse pointer looks like?
A. A thick plus sign
B. A thin plus sign
C. An outlined rectangle
D. Arrow with Plus sign
Show Answer

Correct Answer: C. An outlined rectangle

Explanation: When dragging a field to create a relationship in Access, the pointer typically changes to show a small rectangle representing the field being dragged.

34.
The columns in a Microsoft access table are also called _______
A. Rows
B. Records
C. Fields
D. Columns
Show Answer

Correct Answer: C. Fields

Explanation: In Access terminology, columns are referred to as Fields, while rows are referred to as Records.

35.
In the context of Microsoft Access, which of the following statement is true?
A. Reports are used to retrieve and calculate data from tables
B. Queries can be printed and presented as the information
C. Queries can include calculated fields that do not exist in table
D. Reports and forms are similar but forms are used to print but reports to display on screen only
Show Answer

Correct Answer: C. Queries can include calculated fields that do not exist in table

Explanation: Queries allow you to create new fields using formulas (calculated fields) that are not stored in the underlying tables. Option D is incorrect (reversed), B is possible but not the primary purpose, and A is partially true but describing queries better.

36.
Which of the following is not a field type in Microsoft Access?
A. Memo
B. Hyperlink
C. Ole Object
D. Lookup Wizard
Show Answer

Correct Answer: D. Lookup Wizard

Explanation: Lookup Wizard is a tool/helper that appears in the data type list, but it is not a data type itself; it creates a Text or Number field with lookup properties.

37.
While creating relationship, what happens when you release mouse pointer after you drop the primary key of a table into foreign key of another table?
A. A relationship is created
B. Edit relationship dialog box appears
C. A connector symbol is created
D. A connecting Line is visible
Show Answer

Correct Answer: B. Edit relationship dialog box appears

Explanation: Upon releasing the mouse, Access opens the Edit Relationships dialog box, allowing you to define the relationship type and enforce referential integrity before the relationship is actually created.

38.
What is a form?
A. Forms are used only to view information
B. Forms can be previewed on the screen, but they are usually printed
C. Forms generally provide a detailed took at records and usually for the people who actually work with the database
D. Forms are often used to group and summarize data, and are often for the people who don't work with the database but use its information for other business task
Show Answer

Correct Answer: C. Forms generally provide a detailed took at records and usually for the people who actually work with the database

Explanation: Forms are the primary interface for data entry and viewing, designed for users interacting directly with the database records.

39.
What is the size limitation for an access database?
A. 255 gigabytes
B. 1 gigabyte
C. 266 gigabytes
D. 1000 gigabytes
Show Answer

Correct Answer: B. 1 gigabyte (or 2GB in modern versions)

Explanation: Older versions of Access (like Access 97) had a 1 GB limit. Modern versions (Access 2000 and later) have a 2 GB limit. Among the options provided, 1 GB is the only plausible answer representing a known limit.

40.
_______ enables the user to automatically insert numbers in a sequential or random number.
A. Auto Number
B. Look up Wizard
C. Hyperlink
D. Ole object
Show Answer

Correct Answer: A. Auto Number

Explanation: The AutoNumber data type automatically generates a unique sequential (or random) number for each new record added to the table.
