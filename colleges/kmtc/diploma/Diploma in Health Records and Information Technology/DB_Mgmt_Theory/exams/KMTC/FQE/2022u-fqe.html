Section 1: Multiple Choice Questions

1.
The database designed to consist of multiple table that are linked together through matching data stored in each table is called
A. Object oriented database
B. Hierarchical database
C. Relational database
D. Network database
Show Answer

Correct Answer: C. Relational database

Explanation: A Relational Database Management System (RDBMS) organizes data into tables (relations) which are linked together based on common fields (matching data), such as Primary and Foreign keys.

2.
In E-R diagram attributes are represented by:
A. Triangle
B. Square
C. Rectangle
D. Ellipse
Show Answer

Correct Answer: D. Ellipse

Explanation: In standard Entity-Relationship (ER) diagrams:

  • Ellipses: Represent Attributes.
  • Rectangles: Represent Entities.
  • Diamonds: Represent Relationships.
3.
Count function in SQL returns the number of __________
A. Groups
B. Distinct values
C. Columns
D. Values
Show Answer

Correct Answer: D. Values

Explanation: The `COUNT()` function returns the number of rows or non-null values in a specific column. `COUNT(*)` returns the total number of rows (records).

4.
The statement in SQL which allows change of the definition of a table is __________
A. Select
B. Update
C. Alter
D. Create
Show Answer

Correct Answer: C. Alter

Explanation: The ALTER TABLE statement is a Data Definition Language (DDL) command used to add, delete, or modify columns in an existing table.

  • UPDATE: Changes data within the table (DML).
  • CREATE: Makes a new table.
  • SELECT: Retrieves data.
5.
What is the advantage of the database management approach?
A. Data redundancy increases
B. Data is integrated and can be accessed by multiple programs
C. Data is dependent on programs
D. None of the above
Show Answer

Correct Answer: B. Data is integrated and can be accessed by multiple programs

Explanation: A primary advantage of DBMS is data integration and sharing, allowing multiple users and applications to access the same data source efficiently. It also reduces redundancy and ensures data independence.

6.
Which term is Row synonymous with?
A. Field
B. Column
C. Record
D. Relation
Show Answer

Correct Answer: C. Record

Explanation: In relational database terminology:

  • Row: Tuple or Record.
  • Column: Attribute or Field.
  • Table: Relation.
7.
In a one to many relationship, the entity that is on the one side of the relationship is called a __________ entity.
A. Parent
B. Child
C. Instance
D. Subtype
Show Answer

Correct Answer: A. Parent

Explanation: In a one-to-many relationship, the "One" side is the Parent (or Master) table, and the "Many" side is the Child (or Detail) table.

B. Define table

D. Create table

9.
In database management, what do you mean by one to many relationship between student and class table?
A. One class may have many students
B. Many classes may have many classes
C. Many students may have many classes
D. One student can have many classes
Show Answer

Correct Answer: A. One class may have many students

Explanation: In a standard school database model, a single Class (the "One" side) contains multiple Students (the "Many" side), but a student typically belongs to one administrative class at a time.

10.
To set a password, you first need to open the database in __________ mode.
A. Password
B. Sharing
C. Exclusive
D. Read Only
Show Answer

Correct Answer: C. Exclusive

Explanation: In Microsoft Access, to set or remove a database password, you must open the file in Exclusive Mode to ensure no other users are currently accessing it.

11.
Referential integrity means:
A. Do not enter value in the primary key field of a child table if that value does not exist in the primary key of the parent table
B. Do not enter a value in the foreign key field of a parent table if that value does not exist in the primary key of the child table
C. Do not enter a value in the foreign key field of a child table if that value does not exist in the primary key of the parent table
D. All of the above
Show Answer

Correct Answer: C. Do not enter a value in the foreign key field of a child table if that value does not exist in the primary key of the parent table

Explanation: Referential Integrity ensures that a relationship between tables remains valid. Specifically, every Foreign Key value in the child table must match an existing Primary Key value in the parent table (or be Null).

12.
Which of the following data types is used to attach Microsoft files, graphics files, sound files and pdf files to a table?
A. Memo
B. Hyperlink
C. OLE Object
D. Text
Show Answer

Correct Answer: C. OLE Object

Explanation: OLE Object (Object Linking and Embedding) is the legacy data type in Access used for binary files like images, Word documents, and PDFs. In newer versions, the "Attachment" data type is preferred, but OLE Object is the standard answer for this context.

Explanation: A Record is a collection of related fields (data items) that describe a single entity (e.g., Name, Age, ID for one student).

In a relational schema, each tuple is divided into fields called:
A. Domains
B. Queries
C. Relations
D. All of the above
Show Answer

Correct Answer: A. Domains (Technically Attributes)

Explanation: In relational theory, a tuple (row) is made up of attributes (columns). Attributes draw their values from Domains (the set of permitted values). Among the options provided, "Domains" is the closest term related to the definition of fields/columns in this context, although "Attributes" would be the most precise term.

25.
Which of the following are five built in functions provided by SQL?
A. SUM, AVG, MIN, MAX, MULT
B. COUNT, SUM, AVG, MAX, MIN
C. SUM, AVG, MULT, DIV, MIN
D. SUM, AVG, MIN, MAX, NAME
Show Answer

Correct Answer: B. COUNT, SUM, AVG, MAX, MIN

Explanation: These are the standard aggregate functions in SQL:
- COUNT(): Returns number of rows.
- SUM(): Returns sum of numeric column.
- AVG(): Returns average value.
- MAX(): Returns largest value.
- MIN(): Returns smallest value.
Note: "MULT" and "DIV" are not standard aggregate functions.

A. Input data
B. Output data
C. Sort and filter
D. All of the above
Show Answer

Correct Answer: D. All of the above

Explanation: Queries are versatile tools in Access used to:
- Input/Modify data (Action queries like Append/Update).
- Output data (Select queries).
- Sort and Filter records for viewing or reporting.

28.
Which field type will you select if you need to enter long text in that field?
A. Text
B. Memo
C. Currency
D. Hyperlink
Show Answer

Correct Answer: B. Memo

Explanation: The Memo data type (now called "Long Text" in newer Access versions) is used for long alphanumeric data, storing up to 65,536 characters, whereas standard Text (Short Text) is limited to 255 characters.

Explanation: Worksheets are the primary documents in Microsoft Excel. Access objects include Tables, Queries, Forms, Reports, Macros, and Modules.

30.
In which method don't we need to specify the field type and size while creating a new Table?
A. Design View
B. While using Wizard
C. Create Table By Entering Data
D. All of Above
Show Answer

Correct Answer: C. Create Table By Entering Data

Explanation: When creating a table by entering data directly (Datasheet view), Access interprets the data entered and automatically assigns the appropriate data type and size, unlike Design View where you must explicitly define them.
