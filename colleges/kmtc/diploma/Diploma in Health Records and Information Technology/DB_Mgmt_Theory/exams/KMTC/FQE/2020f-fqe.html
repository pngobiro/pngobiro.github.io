Section 1: Multiple Choice Questions (30 Marks)

1.
In database design, the stage where one gathers and lists all necessary fields for the database project is
a) Data depilation
b) Data refinement
c) Establishing relationship
d) None of the above
Show Answer

Correct Answer: [Data Definition/Requirements Gathering]

Explanation: The option "a) Data depilation" appears to be a typo for "Data definition". In the database design process, requirements analysis involves gathering data needs, which leads to defining the fields (Data Definition). Based on context from similar past papers, Option A is likely intended to be the correct answer despite the typo.

2.
Which of the following fields in database has a width of 8 bytes?
a) Memo
b) Name
c) Number
d) Data/time
Show Answer

Correct Answer: d) Data/time

Explanation: In Microsoft Access, the Date/Time data type uses 8 bytes of storage. Standard "Number" fields like Long Integer use 4 bytes, though Double precision uses 8 bytes. "Memo" is variable length.

3.
A ______ name must be unique within a database
a) Record
b) Character
c) Table
d) Character
Show Answer

Correct Answer: c) Table

Explanation: Table names must be unique within the same database object container. You cannot have two tables with the exact same name.

4.
The key that uniquely identifies each record in database is called
a) Field name
b) Primary key
c) Candidate key
d) Secondary key
Show Answer

Correct Answer: b) Primary key

Explanation: The Primary Key is the specific field (or combination of fields) chosen to uniquely identify every record in a table.

5.
One to many relationships between student and subject tables means
a) One student can have many subjects
b) One subject can have more students
c) Many subjects can have many students
d) Many students can have many subjects
Show Answer

Correct Answer: a) One student can have many subjects

Explanation: A "One-to-Many" relationship implies a single entity on one side connects to multiple entities on the other. If defined from Student to Subject, it means one student is associated with many subjects.

6.
A part of a database that stores only one type of data is
a) Record
b) Field
c) Query
d) Report
Show Answer

Correct Answer: b) Field

Explanation: A Field (column) is defined by a specific data type (e.g., text, number, date) and stores only that type of data for all records.

7.
Which of the following may not be included in a database?
a) Data presented in a table in MS Word document
b) Data in MS Excel spreadsheet
c) A PowerPoint presentation
d) A telephone directory
Show Answer

Correct Answer: c) A PowerPoint presentation

Explanation: A PowerPoint presentation is a slide show format, not a structured data container. Word tables, Excel spreadsheets, and telephone directories are all forms of structured data collections (databases in the general sense).

8.
A collection of related records in a database is known as
a) File
b) Record
c) Relationship
d) Table
Show Answer

Correct Answer: d) Table

Explanation: In a relational database, a collection of related records is called a Table (historically referred to as a File).

9.
Each record is constituted by a number of individual data items called
a) Rations
b) Data types
c) Fields
d) Tables
Show Answer

Correct Answer: c) Fields

Explanation: A record is composed of multiple Fields, where each field holds a specific piece of data about the entity.

10.
The complete information about an activity in a database is called
a) Data
b) Record
c) Field
d) Information
Show Answer

Correct Answer: b) Record

Explanation: A Record contains the complete set of information (all fields) related to a single entry or activity.

11.
What field type will you select if you need to enter long text in that field
a) Currency
b) Text
c) Memo
d) Hyperlink
Show Answer

Correct Answer: c) Memo

Explanation: The Memo data type (or Long Text) is designed for storing large amounts of alphanumeric text, unlike the standard Text field which has a shorter limit.

12.
Which of the following is not an MS Access database object?
a) Modules
b) Worksheet
c) Table
d) Form
Show Answer

Correct Answer: b) Worksheet

Explanation: Worksheet is a component of Microsoft Excel. Access objects include Tables, Forms, Modules, Queries, Reports, and Macros.

13.
In which method do we need not to specify the field type and size when creating a new table?
a) Using wizard
b) Design new
c) Create table by entering data
d) All the above
Show Answer

Correct Answer: c) Create table by entering data

Explanation: When creating a table by entering data directly (Datasheet view), Access automatically assigns field types and sizes based on the input.

14.
The option that creates a drop-down list of values is
a) Lookup wizard
b) Hyperlink
c) Memo
d) Ole object
Show Answer

Correct Answer: a) Lookup wizard

Explanation: The Lookup Wizard is used to create a field that displays a list of values (combo box) for the user to choose from.

15.
The maximum length of a text field is
a) 75
b) 100
c) 234
d) 255
Show Answer

Correct Answer: d) 255

Explanation: The standard Text field (Short Text) in Access has a maximum capacity of 255 characters.

16.
When we create or open the MS Access Database file, we get to
a) Design view window
b) Database window
c) Switchboard
d) Query window
Show Answer

Correct Answer: b) Database window

Explanation: In traditional Access versions, opening a file brings you to the Database Window (or Navigation Pane in newer versions), which displays all database objects.

17.
Columns in MS Access table are also called
a) Records
b) Rows
c) Pillars
d) Fields
Show Answer

Correct Answer: d) Fields

Explanation: In relational database terminology, columns are synonymous with Fields (attributes).

18.
Which of the following describes a FORM in MS Access?
a) Is an input screen for viewing centering data
b) Is used to make print out from Access tables
c) Is optional once we can just report
d) Is used for storing special data
Show Answer

Correct Answer: a) Is an input screen for viewing centering data

Explanation: A Form serves as a user interface (input screen) for entering, viewing, and modifying data ("centering" likely typo for "entering").

19.
How can you create a query in MS Access?
a) Use query wizard
b) Drag and drop fields
c) Type the SQL
d) All the above
Show Answer

Correct Answer: d) All the above

Explanation: Queries can be created using the Wizard, Design View (drag and drop), or SQL View (typing code).

20.
The size of a field with data type as number cannot be
a) 2
b) 8
c) 4
d) 16
Show Answer

Correct Answer: d) 16

Explanation: Standard number field sizes are 1 (Byte), 2 (Integer), 4 (Long Integer/Single), and 8 (Double). 16 is typically not a standard Number field size option in basic configuration.

21.
You can edit a relationship already established between two tables by
a) Right click of the relationship line and then choose edit relationship
b) Double click on the relationship line
c) Both A and B
d) None of the above
Show Answer

Correct Answer: c) Both A and B

Explanation: Both methods (right-clicking and double-clicking the relationship line) open the "Edit Relationships" dialog box.

22.
Which one of the following is not a field type in MS Access?
a) Ole Object
b) Memo
c) Look up wizard
d) Hyperlink
Show Answer

Correct Answer: c) Look up wizard

Explanation: Lookup Wizard is a tool to configure a field, not a data type itself (it usually results in a Text or Number field with lookup properties).

23.
Which of the following relationship cannot be applied in MS Access database?
a) One to many
b) Many to many
c) One to many
d) Two by two table relationship
Show Answer

Correct Answer: d) Two by two table relationship

Explanation: Standard database relationships are One-to-One, One-to-Many, and Many-to-Many. "Two by two" is not a valid relationship type.

24.
Which field type stores photos in MS Access
a) Memo
b) Hyperlink
c) Picture
d) Ole object
Show Answer

Correct Answer: d) Ole object

Explanation: OLE Object (Object Linking and Embedding) is used to store binary data like photos, documents, etc.

25.
Which data type should we use for storing a few paragraphs of a report?
a) Para
b) Memo
c) Text
d) Desc
Show Answer

Correct Answer: b) Memo

Explanation: Memo fields are capable of storing long text (up to ~65,000 characters), making them suitable for paragraphs.

26.
In database, data is held in which object?
a) Tables
b) Forms
c) Reports
d) Queries
Show Answer

Correct Answer: a) Tables

Explanation: Tables are the primary storage objects in a database where actual data is held.

27.
Which of the following shows the final database results in a more presentable way?
a) Forms
b) Queries
c) Reports
d) Tables
Show Answer

Correct Answer: c) Reports

Explanation: Reports are designed specifically for formatting, summarizing, and presenting data for printing or viewing.

28.
How many characters can you type in maximum while writing the field name?
a) Any number
b) 68
c) 60
d) 64
Show Answer

Correct Answer: d) 64

Explanation: In Microsoft Access, the maximum length for an object name (like a field name) is 64 characters.

29.
To sort records in a table,
a) Open table, click sort button on tool bar, choose field to sort, then click ok
b) Open table, click field to sort, then click sort button on toolbar
c) Click the field heading and sort in ascending or descending order
d) All the above
Show Answer

Correct Answer: d) All the above

Explanation: Access provides multiple ways to sort: selecting a field and using the toolbar buttons, or using the dropdown arrows on the field headers.

30.
Commands gather data from database are stored in
a) Forms
b) Queries
c) Reports
d) Tables
Show Answer

Correct Answer: b) Queries

Explanation: Queries are saved requests (commands) to retrieve, gather, and manipulate data from the database.
