Section 1: Multiple Choice Questions (30 Marks)

1.
To start Microsoft PowerPoint application
a) Click on start >> progress >> all programs >> Ms power point
b) Hit ctrl+e then type ppoint.exe and enter
c) Click start >> run then type power point then press enter
d) Hit ctrl+s then type power point
Show Answer

Correct Answer: a) Click on start >> progress >> all programs >> Ms power point

Explanation: The standard Windows method to open an application is Start > All Programs > Microsoft Office > Microsoft PowerPoint. Note: "progress" in the option is likely a typo for "Programs". The Run command for PowerPoint is `powerpnt`, not "power point".

2.
PowerPoint presentations are widely used for the following except
a) Note outlines for teachers
b) Project presentations by students
c) Communication of planning
d) Mail set up and postage
Show Answer

Correct Answer: d) Mail set up and postage

Explanation: PowerPoint is a presentation software used for visual communication. Mail setup and postage are functions of email clients (like Outlook) or word processors (Mail Merge in Word), not PowerPoint.

3.
What PowerPoint feature will one use to apply motion affects to different objects of a slide
a) Slide transition
b) Slide design
c) Animation objects
d) Animation scheme
Show Answer

Correct Answer: d) Animation scheme

Explanation: Animation Schemes (or Custom Animation) are used to apply motion effects to specific text or objects within a slide. Slide Transition applies to the slide itself.

4.
What feature will on use to apply motion effects in between a slide exists and another enters
a) Slide transition
b) Slide design
c) Animation objects
d) Animation scheme
Show Answer

Correct Answer: a) Slide transition

Explanation: Slide Transitions control the visual effect that occurs when moving from one slide to the next during a presentation.

5.
In which menu can you find features like slide design and slide layout
a) Insert menu
b) Format menu
c) Tools menu
d) Slide show menu
Show Answer

Correct Answer: b) Format menu

Explanation: In older versions of PowerPoint (like 2003, which uses menus), Slide Design and Slide Layout were located under the Format menu.

6.
Which shortcut key inserts a new slide in current presentation
a) Ctrl+N
b) Ctrl+M
c) Ctrl+S
d) Ctrl+Z
Show Answer

Correct Answer: b) Ctrl+M

Explanation: Ctrl+M adds a new slide. Ctrl+N creates a new presentation document.

7.
What is the best way to create another copy of a slide
a) Click the slide then pres ctrl+A and paste in new slide
b) From insert menu choose duplicate slide
c) Redo everything on a new slide that you had done on previous slide
d) On the slide panel click the slide, press ctrl+C then hit ctrl+Z
Show Answer

Correct Answer: b) From insert menu choose duplicate slide

Explanation: The most efficient way to copy a slide is using the "Duplicate Slide" command (often in the Insert or Home menu depending on version, or Ctrl+D). Ctrl+A selects all objects, not the slide itself. Ctrl+C then Ctrl+V (not Z) would also work.

8.
In a PowerPoint presentation
a) Sound clips can be inserted but not movie clips
b) Movie clips can be inserted but not sound clips
c) Both cannot be inserted
d) Both can be inserted
Show Answer

Correct Answer: d) Both can be inserted

Explanation: PowerPoint supports inserting both audio (sound clips) and video (movie clips) into slides.

9.
Step by step instructions written to solve any problem is called
a) Pseudocode
b) Algorithm
c) Assembler
d) Class
Show Answer

Correct Answer: b) Algorithm

Explanation: An algorithm is a process or set of rules to be followed in calculations or other problem-solving operations.

10.
Diagrammatic or symbolic representation of an algorithm is called
a) Data flow diagram
b) E.R diagram
c) Flow chart
d) Algorithm diagram
Show Answer

Correct Answer: c) Flow chart

Explanation: A flowchart is a visual representation of an algorithm, using standard symbols to denote different types of instructions.

11.
Object oriented programming method is followed in
a) C programming language
b) C++ programming language
c) C# programming language
d) Both (b) and (c)
Show Answer

Correct Answer: d) Both (b) and (c)

Explanation: C is a procedural language. Both C++ and C# are Object-Oriented Programming (OOP) languages.

12.
Informed high level description of an algorithm in English is called
a) Fraction
b) Class
c) Pseudocode
d) Program
Show Answer

Correct Answer: c) Pseudocode

Explanation: Pseudocode is a plain language description of the steps in an algorithm or another system, intended for human reading rather than machine execution.

13.
The geometrical figure [rectangle] shown here in a flow chart represents
a) Start/stop
b) Looping
c) Processing
d) Connector
Show Answer

Correct Answer: c) Processing

Explanation: In standard flowchart symbols, a Rectangle represents a Process step (calculation or action). Ovals are for Start/Stop, and Rhombuses are for Decisions/Looping.

14.
This geometrical figure (circle) in a flow chart represents
a) Connector
b) Alternate process
c) Looping
d) Data
Show Answer

Correct Answer: a) Connector

Explanation: A Circle is used as an on-page connector to jump from one point in the flowchart to another.

15.
In Ms-access to open new database press
a) Ctrl+N
b) Ctrl+O
c) Ctrl+Alt+N
d) Alt+N
Show Answer

Correct Answer: a) Ctrl+N

Explanation: Ctrl+N is the standard shortcut in Microsoft Office applications to create a New document or database.

16.
The checking operation on input data is called the
a) Validation of data
b) Verification of data
c) Cross check
d) Control of data
Show Answer

Correct Answer: a) Validation of data

Explanation: Data Validation is the process of ensuring that data entered into a system meets specific criteria (e.g., format, type, range) before it is processed.

17.
The type of a query that extract data and put them into separate tables is called
a) A parameter query
b) A make-table query
c) An update query
d) A cross tab query
Show Answer

Correct Answer: b) A make-table query

Explanation: A Make-Table Query retrieves data from one or more tables and creates a new table containing that data.

18.
What is a firewall in computer network?
a) The physical boundary of network
b) An operating system of computer network
c) A system designed to prevent unauthorized access
d) A web browsing software
Show Answer

Correct Answer: c) A system designed to prevent unauthorized access

Explanation: A Firewall is a network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules, acting as a barrier against unauthorized access.

19.
What is the benefit of computer networking
a) File sharing
b) Easier access to resources
c) Easier backups
d) All of the above
Show Answer

Correct Answer: d) All of the above

Explanation: Computer networking allows for file sharing, resource sharing (printers, drives), and centralized data management which facilitates easier backups.

20.
Which of the following is not a networking device
a) Gateways
b) Linux
c) Routers
d) Firewalls
Show Answer

Correct Answer: b) Linux

Explanation: Linux is an Operating System, not a networking hardware device. Gateways, Routers, and Firewalls (hardware appliances) are networking devices.

21.
DHCP server provides .... To the client
a) Protocol
b) IP address
c) MAC address
d) Network address
Show Answer

Correct Answer: b) IP address

Explanation: The primary function of a DHCP (Dynamic Host Configuration Protocol) server is to dynamically assign IP addresses to clients on the network.

22.
A set of rules that govern all aspects of information communication is called
a) Server
b) Internet
c) Protocol
d) OSI model
Show Answer

Correct Answer: c) Protocol

Explanation: A Protocol is a standard set of rules that allow electronic devices to communicate with each other.

23.
What is the meaning of bandwidth in network
a) Transmission capacity of a communication channel
b) Connected computers in the network
c) Class of IP used in network
d) Spread of data exchange
Show Answer

Correct Answer: a) Transmission capacity of a communication channel

Explanation: Bandwidth refers to the maximum rate of data transfer across a given path, representing the transmission capacity of the connection.

24.
What does router do in a network
a) Forwards a packet to all outgoing links
b) Forwards a packet to the next outgoing link
c) Determines on which outing link a packet is to be forwarded
d) Forwards a packet to all outgoing links except the originated link
Show Answer

Correct Answer: c) Determines on which outing link a packet is to be forwarded

Explanation: The primary function of a router is routing: determining the best path (link) for a packet to travel to reach its destination. While it does forward packets (B), the core distinction is the decision-making process (determining the route).

25.
Which of the following layer is not a network support layer
a) Transport layer
b) Network layer
c) Data link layer
d) Physical layer
Show Answer

Correct Answer: a) Transport layer

Explanation: In the OSI model, layers 1-3 (Physical, Data Link, Network) are often called the network support layers because they deal with the physical aspects of moving data. Layers 5-7 are user support layers. Layer 4 (Transport) links the two groups.

26.
Which of the following is not a type of server
a) Proxy server
b) Application server
c) Cloud server
d) Bus server
Show Answer

Correct Answer: d) Bus server

Explanation: "Bus" refers to a network topology, not a type of server. Proxy, Application, and Cloud servers are real types of servers.

27.
Identify the odd one out of the following terms below
a) Virus
b) Identify theft
c) Hackers
d) Spam
Show Answer

Correct Answer: c) Hackers

Explanation: Hackers are human agents who perform actions. Virus, Identity Theft (typo "Identify theft"), and Spam are either software tools, outcomes, or content used/generated by actors.

28.
Which of the following devices is used to transfer data between LAN
a) Repeater
b) Hub
c) Routers
d) Bridges
Show Answer

Correct Answer: c) Routers

Explanation: Routers are used to connect different networks (LANs) together. Bridges can also connect LAN segments, but Routers are the primary device for interconnecting distinct Local Area Networks.

29.
How many types of IP addresses are there
a) Two
b) Three
c) One
d) Four
Show Answer

Correct Answer: a) Two

Explanation: There are two main versions of IP addresses currently in use: IPv4 and IPv6.

30.
Which of the following protocol sends electronic mail to the receiver's end
a) FTP
b) SMTP
c) POP3
d) Both A and B
Show Answer

Correct Answer: b) SMTP

Explanation: SMTP (Simple Mail Transfer Protocol) is responsible for sending emails from a client to a server or between servers. POP3 is used for receiving/retrieving emails.
