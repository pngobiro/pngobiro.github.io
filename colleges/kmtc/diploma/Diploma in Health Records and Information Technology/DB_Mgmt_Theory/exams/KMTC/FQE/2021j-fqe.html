Section 1: Multiple Choice Questions (30 Marks)

1.
The process of finding errors and fixing them within a program is called?
a. Compiling
b. Debugging
c. Scanning
d. Executing
Show Answer

Correct Answer: b. Debugging

Explanation: Debugging is the systematic process of identifying, analyzing, and removing errors (bugs) from software code.

  • Compiling: Converting source code into machine code.
  • Scanning: Reading input or checking for viruses.
  • Executing: Running the program.
2.
Which tool do you use to create a query object?
a. Simple query wizard
b. Database wizard
c. Table query wizard
d. Simple filer wizard
Show Answer

Correct Answer: a. Simple query wizard

Explanation: In Microsoft Access, the "Simple Query Wizard" is a specific tool used to create basic SELECT queries by selecting fields from tables.

3.
Which of the following is the ascending order of data hierarchy?
a. Byte-Bit-Field-Record-File-Database
b. Bit-Byte-Field-Record-Database-File
c. Bit-Byte-Field-Record-File-Database
d. Bit-Byte-Record-Field-File-Database
Show Answer

Correct Answer: c. Bit-Byte-Field-Record-File-Database

Explanation: The data hierarchy is organized from smallest to largest:

  1. Bit: 0 or 1.
  2. Byte: A character (8 bits).
  3. Field: A data item (e.g., Name).
  4. Record: A collection of fields (e.g., a student's profile).
  5. File (Table): A collection of records.
  6. Database: A collection of related files.
4.
Database access levels are specified so as to define who can access what in a database. It is identified through?
a. Password
b. Lock settings
c. Status
d. User ID
Show Answer

Correct Answer: d. User ID

Explanation: The User ID (Authentication) is the primary identifier used to determine a user's identity. Once identified, the system checks the permissions (Authorization) associated with that ID to grant specific access levels.

5.
A search value can be exact value or it can be?
a. Wild card character
b. Relationships
c. Logical operator
d. Comparison operation
Show Answer

Correct Answer: a. Wild card character

Explanation: When searching in a database, you can specify an exact match (e.g., "John") or use Wildcard characters (e.g., "*John*" or "J?") to represent one or more unknown characters.

6.
In which method we don't need to specify the field type and size while creating a new table?
a. Design view
b. Data sheet view
c. Create table by entering data
d. While using wizard
Show Answer

Correct Answer: c. Create table by entering data

Explanation: When creating a table by simply entering data (often in Datasheet View in older contexts or "Enter Data" options), Access attempts to automatically infer the data type and size based on the values you type, rather than requiring you to define them explicitly first.

7.
The size of Yes/No field is always?
a. 1 character
b. 1 bit
c. 1 byte
d. 1 kB
Show Answer

Correct Answer: b. 1 bit

Explanation: A Yes/No field (Boolean) requires only 1 bit of storage to represent two states (0 for No/False, 1 for Yes/True/On).

8.
Which kind of query that displays its own dialog box prompting you for entering information before executing.
a. Dialog query
b. Parameter query
c. Crosstab query
d. Select query
Show Answer

Correct Answer: b. Parameter query

Explanation: A parameter query prompts the user for input (like a date range or a specific name) using a dialog box each time the query is run.

9.
Which field type can store photos?
a. Photo link
b. Expression builder
c. Hyper link
d. OLE object
Show Answer

Correct Answer: d. OLE object

Explanation: OLE (Object Linking and Embedding) Object fields are used in Microsoft Access to store complex binary data such as images (photos), documents, and graphs. Note: Newer versions also support "Attachment" fields which are more efficient.

10.
If a create 'Student' field' in 'Fees' table to store 'Student_id' of 'Student' table, then this 'Student' field in 'Fees' table is called?
a. Native key
b. Foreign key
c. Composite key
d. Primary key
Show Answer

Correct Answer: b. Foreign key

Explanation: A Foreign Key is a field in one table that links to the Primary Key of another table, establishing a relationship between them. In this case, the Student ID in the Fees table references the unique Student ID in the Student table.

11.
The complete information about an entity in a database is called?
a. Record
b. Field
c. Data
d. Information
Show Answer

Correct Answer: a. Record

Explanation: A record (or tuple) contains all the data fields related to a single entity (e.g., all details for one specific patient).

12.
The input mask wizard does not include the ability to do which of the following?
a. Store symbols with data
b. Use preset masks
c. Define placeholder
d. Force all uppercase
Show Answer

Correct Answer: d. Force all uppercase

Explanation: The Input Mask Wizard guides you through selecting a mask (preset), choosing a placeholder character, and deciding whether to store the literal symbols. Forcing text to uppercase is typically done using the Format property (using the `>` symbol), not the Input Mask Wizard itself.

13.
This data type allows alphanumeric characters and special symbols?
a. Currency
b. Memo
c. Auto number
d. Text
Show Answer

Correct Answer: d. Text

Explanation: The Text (or Short Text) data type is designed to store alphanumeric characters (letters, numbers, and symbols). "Memo" also stores text but is used for very long descriptions (Long Text). "Text" is the standard answer for general alphanumeric data.

14.
Query design has two parts. The upper part shows?
a. Name of fields, field types and size
b. Sorting check boxes
c. Tables with fields and relationships between tables
d. Reports and macros to query
Show Answer

Correct Answer: c. Tables with fields and relationships between tables

Explanation: In Access Query Design View, the upper pane displays the tables added to the query and the relationship lines between them. The lower pane (grid) is where you define fields, criteria, and sorting.

15.
Any report, including basic report can be modified in?
a. Layout view
b. Table view
c. Report view
d. Print view
Show Answer

Correct Answer: a. Layout view

Explanation: Reports can be modified in Design View and Layout View. Layout View allows you to visually modify the report (resize columns, move fields) while viewing live data. Report View and Print Preview are for viewing only.

16.
Using the keyboard ________ will return a record in which the field value equals a defined text.
a. Like
b. Between
c. Is null
d. Equal
Show Answer

Correct Answer: d. Equal

Explanation: The "Equal" operator (or simply typing the value) is used to find records that match a defined text exactly.

17.
In Ms Access ‘lookup wizard’ is use to?
a. Select value from excel sheets
b. Select value from another table
c. Identify tables to be selected
d. Select from preview values
Show Answer

Correct Answer: b. Select value from another table

Explanation: The Lookup Wizard creates a field that allows you to choose a value from another table or from a list of values you type in. It is commonly used to create foreign keys and drop-down lists.

The basic elements of a form or a report are called?

What lets you create new presentation by selecting ready-made fort colour and graphic effects?

Which of the following is not one of PowerPoint views?

22.
A function inside another function is called?
a. Duplicate function
b. Nested function
c. Round function
d. Inner function
Show Answer

Correct Answer: b. Nested function

Explanation: When a function is used as an argument within another function (e.g., `=IF(AND(A1>0, A1<10), ...)`), it is called a nested function.

23.
Which function will you use to enter current time in a worksheet cell?
a. = Current Time ( )
b. = time ( )
c. = now ( )
d. = today ( )
Show Answer

Correct Answer: c. = now ( )

Explanation:

  • =NOW() returns the current date and time.
  • =TODAY() returns only the current date.
  • =TIME(hour, minute, second) constructs a time from specific numbers.

Note: To display only time with =NOW(), you would format the cell as Time.

Long text can be broken down into many times within a cell. You can do this through?

Multiple calculations can be made in a single formula using?

27.
The correct sequence of HTML tags for starting a web page is?
a. Head, Title, HTML, Body
b. HTML, Body, Title, Head
c. HTML, Head, Title, Body
d. HTML, Head, Title, Body
Show Answer

Correct Answer: c. HTML, Head, Title, Body

Explanation: The standard structure of an HTML document is:

<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Content...
  </body>
</html>

Thus, the sequence of opening tags is HTML -> Head -> Title, and then Body comes after Head closes.

28.
Which of the following tags is used to insert the targets heading in HTML?
a. (h3)
b. (h1)
c. (h5)
d. (h6)
Show Answer

Correct Answer: b. (h1)

Explanation: Assuming "targets" is a typo for "largest" or refers to the main/primary heading of a page, <h1> is the correct tag. Headings range from <h1> (largest/most important) to <h6> (smallest).

29.
How to create an unordered list (a list with the list items in bullets) in HTML?
a. (uL)
b. (oL)
c. (L1)
d. (1)
Show Answer

Correct Answer: a. (uL)

Explanation: The <ul> tag defines an Unordered List (bullet points). <ol> defines an Ordered List (numbers).

30.
Which characterized is used to represent the closing of a tag in HTML?
a. \
b. !
c. /
d. ()
Show Answer

Correct Answer: c. /

Explanation: A forward slash / is used in the closing tag. For example: Opening tag <p>, Closing tag </p>.
