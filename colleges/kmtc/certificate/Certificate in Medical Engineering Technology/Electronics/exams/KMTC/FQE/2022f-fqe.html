Section 1: Multiple Choice Questions

40 Marks
1.
A logic gate is electronic circuit which
a) Alternate between 0 and I values
b) Works on binary algebra
c) Allows electron flow only in one direction
d) Makes logic decision
Show Answer

Correct Answer: d) Makes logic decision

Explanation: A logic gate is defined as a physical device implementing a Boolean function; it performs a logical operation on one or more binary inputs and produces a single binary output, essentially making a "decision" based on the input combination.

2.
The output of a 2-input OR gate is zero only when
a) Either input is 0
b) Both inputs are 1
c) Both inputs are 0
d) Either input is 1
Show Answer

Correct Answer: c) Both inputs are 0

Explanation: The truth table for an OR gate states that the output is Low (0) only if all inputs are Low (0). If any input is High (1), the output is High (1).

3.
The number $1110_2$ is equivalent to decimal number
a) One thousand
b) Four
c) Sixteen
d) Fourteen
Show Answer

Correct Answer: d) Fourteen

Explanation: $1110_2 = (1 \times 2^3) + (1 \times 2^2) + (1 \times 2^1) + (0 \times 2^0) = 8 + 4 + 2 + 0 = 14_{10}$.

4.
Digital system operates on which number system
a) Binary
b) Decimal
c) Octal
d) Hexadecimal
Show Answer

Correct Answer: a) Binary

Explanation: Digital electronics use voltage levels to represent states, typically ON/OFF or High/Low, which corresponds directly to the Binary number system (Base 2).

5.
The frequency of oscillation of an astable multivibrator depends mainly on
a) Resistor-capacitor values of the circuit
b) Value of collector resistor
c) Value of transistor gain
d) Width of input pulse
Show Answer

Correct Answer: a) Resistor-capacitor values of the circuit

Explanation: The timing of the unstable states is determined by the charging and discharging of the capacitors through the base resistors. $T \approx 0.69(R_1C_1 + R_2C_2)$.

6.
An example of a non-sinusoidal oscillator is
a) Hartley oscillator
b) Bistable multivibrator
c) Colpitts oscillator
d) Tuned oscillator
Show Answer

Correct Answer: b) Bistable multivibrator

Explanation: Multivibrators produce square waves (non-sinusoidal). Hartley, Colpitts, and Tuned oscillators are designed to produce sinusoidal (sine wave) outputs.

7.
An oscillator differs from an amplifier because it
a) Requires no input signal
b) Has more gain
c) Requires no dc supply
d) Always has the input and output
Show Answer

Correct Answer: a) Requires no input signal

Explanation: An oscillator generates an output signal (AC) from a DC supply without requiring an external AC input signal, unlike an amplifier which increases the magnitude of an input signal.

8.
An astable multivibrator has
a) One stable state
b) Two stable states
c) No stable state
d) Three stable states
Show Answer

Correct Answer: c) No stable state

Explanation: An astable multivibrator continuously switches between two quasi-stable states, never settling in a stable state. Monostable has one, Bistable has two.

9.
An XOR gate has an output only when
a) Both input is at logic 1
b) Any input is at logic 1 but not both
c) Both inputs are at logic 0
d) Any input is at logic 0 but not both
Show Answer

Correct Answer: b) Any input is at logic 1 but not both

Explanation: The Exclusive-OR (XOR) gate outputs High (1) only when the inputs are different (one High, one Low). If both are High or both are Low, the output is Low.

10.
A multivibrator has how many stable states
a) One (Monostable)
b) Two (Bistable)
c) None (Astable)
d) All of the above
Show Answer

Correct Answer: d) All of the above

Explanation: The term "multivibrator" covers a family of circuits: Astable (0 stable states), Monostable (1 stable state), and Bistable (2 stable states).

11.
When an SCR is forward biased, what is needed to cause it conduct
a) A drain signal
b) A source current
c) A source-drain signal
d) A gate signal
Show Answer

Correct Answer: d) A gate signal

Explanation: An SCR (Silicon Controlled Rectifier) blocks current in the forward direction until a positive pulse is applied to the Gate terminal.

12.
In a class A amplifier the output current flows for
a) A part of the cycle of the input signal
b) The full cycle of the input signal
c) Half the cycle of the input signal
d) Â¾ of the circle of the input signal
Show Answer

Correct Answer: b) The full cycle of the input signal

Explanation: Class A amplifiers are biased such that the active device conducts for the entire $360^\circ$ of the input cycle.

13.
__________ is the device which is fabricated as two SCRs in parallel with common gate terminal
a) Field effect transistor
b) Triac
c) Diac
d) Unijuction transistor
Show Answer

Correct Answer: b) Triac

Explanation: A TRIAC behaves like two SCRs connected in inverse parallel (back-to-back) with a common gate, allowing it to switch AC current.

14.
A thermocouple is a __________ type of transducer
a) Variable resistance
b) Variable inductance
c) Voltage generating (Active)
d) Voltage divider
Show Answer

Correct Answer: c) Voltage generating

Explanation: A thermocouple is an active transducer that generates a small voltage (EMF) proportional to the temperature difference between its junctions (Seebeck effect).

15.
An SCR is turned off by
a) Interrupting its anode current
b) Reversing polarity of its anode-cathode voltage
c) Reduce anode current below holding current
d) All of the above
Show Answer

Correct Answer: d) All of the above

Explanation: Turning off an SCR (Commutation) requires reducing the anode current below the holding current ($I_H$). This can be done by opening the circuit (interrupting), applying a reverse voltage, or shunting the current.

16.
The output of a class A amplifier is
a) Is not distortion free
b) Consists of positive pulses only
c) Is like output of a full wave rectifier
d) Comprises of short duration pulses
Show Answer

Correct Answer: a) Is not distortion free

Explanation: While Class A has the lowest distortion of all classes, practical amplifiers still introduce some distortion. Options b, c, and d describe Class B/C behaviors. Thus, "a" is the most accurate statement by exclusion, or implies "Is essentially distortion free" (if a typo exists), but as written, it states a practical fact.

17.
__________ is two transistors with a control element
a) Diac
b) Thyristor (SCR)
c) Triac
d) Diode
Show Answer

Correct Answer: b) Thyristor (SCR)

Explanation: A Thyristor (SCR) can be modeled as a two-transistor analogy (an NPN and a PNP connected in a regenerative feedback loop) with a gate terminal as the control element.

18.
An operational amplifier can be connected as the following except
a) Inverting amplifier
b) Non-inverting amplifier
c) Oscillator
d) Summing amplifier
Show Answer

Correct Answer: c) Oscillator

Explanation: Inverting, Non-inverting, and Summing are fundamental amplifier configurations (topology). An Oscillator is a distinct application or circuit function built using an op-amp, but not a basic connection configuration in the same category.

19.
The following amplifier is used for amplifying low frequency signals
a) Audio amplifier
b) T.V amplifier
c) D.C amplifier
d) Power amplifier
Show Answer

Correct Answer: a) Audio amplifier

Explanation: Audio amplifiers operate in the audio frequency range (20Hz - 20kHz), which is considered low frequency compared to RF/Video (TV) signals.

20.
When a transistor is driven into saturation ideally the output is
a) 2Vcc
b) 3Vcc
c) 1/2 Vcc
d) Vcc
Show Answer

Correct Answer: 0V (Ideally)

Explanation: When a transistor is in saturation (acting as a closed switch), the voltage across the collector-emitter ($V_{CE}$) drops to near zero ($\approx 0.2V$ or ideally 0V). The options provided ($Vcc$, $2Vcc$, etc.) suggest a confusion with Cut-off (where output is $Vcc$). If forced to choose based on the paper marking, 'd' might be intended for "Cut-off", but for Saturation, the correct physical answer is ~0V.

21.
Which of the following statements is true about the field effect transistor?
a) It has a high output impedance
b) It does not offer any resistance
c) It has a low input impedance
d) It has a high input impedance
Show Answer

Correct Answer: d) It has a high input impedance

Explanation: The primary characteristic of FETs (JFETs and MOSFETs) is their extremely high input impedance (Mega-ohms to Giga-ohms) because the gate is insulated (MOSFET) or reverse-biased (JFET).

22.
Comparing the size of the BJT and the FET
a) BJT is larger than the FET
b) They are the same size
c) BJT is smaller than the FET
d) Depends on application
Show Answer

Correct Answer: a) BJT is larger than the FET (FET is smaller)

Explanation: FETs (specifically MOSFETs) occupy significantly less silicon area than BJTs, which is why they are the preferred technology for high-density Integrated Circuits (ICs).

23.
Enhancement only MOSFET refers to a set which
a) Operates only in enhancement mode
b) Operates only with positive voltage
c) Operates only with negative voltage
d) Operates both negative and positive mode
Show Answer

Correct Answer: a) Operates only in enhancement mode

Explanation: Enhancement-mode MOSFETs are "Normally Off" devices. They have no channel at zero gate bias and conduct only when a gate voltage is applied (Enhancement mode). Depletion MOSFETs can operate in both modes.

24.
Improper biasing of a transistor circuit leads to
a) Excessive heat production
b) Distortion in the output signal
c) Faulty location of load line
d) Heavy loading of the transistor
Show Answer

Correct Answer: b) Distortion in the output signal

Explanation: If the Q-point (bias point) is not centered on the load line, the output signal may be clipped (cut off or saturated) during its swing, causing severe distortion.

25.
Transistor amplifiers use __________ feedback
a) Positive feedback
b) Negative feedback
c) Both positive and negative feedback
d) No feedback
Show Answer

Correct Answer: b) Negative feedback

Explanation: Negative feedback is used in amplifiers to improve stability, linearity, and bandwidth, and to reduce distortion. Positive feedback is used in oscillators.

26.
The following statements are true about BJTS. Which one is false
a) Base is thinly doped
b) Collector is wider and thickly doped
c) Base is sandwiched between collector and emitter
d) Emitter current is equal to collector current
Show Answer

Correct Answer: d) Emitter current is equal to collector current

Explanation: Emitter current is the sum of Collector and Base currents ($I_E = I_C + I_B$). While $I_C \approx I_E$ (since $I_B$ is small), strictly speaking, they are not equal ($I_E > I_C$).

27.
Majority carriers in an PNP transistor are
a) Holes
b) Electrons
c) Neutrons
d) Both electrons and holes
Show Answer

Correct Answer: a) Holes

Explanation: In PNP transistors, the Emitter and Collector are P-type materials where Holes are the majority charge carriers.

28.
Transistor as a switch operates at
a) Cut off region only
b) Saturation region only
c) Between cut off and saturation region
d) Active region only
Show Answer

Correct Answer: c) Between cut off and saturation region (Contextual)

Explanation: A switch operates by toggling between the Cut-off region (OFF state) and Saturation region (ON state). It avoids the Active region.

29.
The channel of a JFET is between
a) Gate and drain
b) Drain and source
c) Gate and source
d) Input and outputs

Correct Answer: b) Drain and source

Explanation: The current conduction path (channel) in a JFET lies between the Drain and Source terminals.

A single stage transistor amplifier contains __________ and associated circuitry.

Explanation: By definition, a single stage amplifier uses one active amplifying device (transistor).

31.
Which one of the following is the correct definition of power rating of a resistor?

32.
The following are factors considered while selecting a capacitor, which one is not?
a) Tolerance
b) Stability
c) Leakage current
d) Capacitive reactance $X_c$
Show Answer

Correct Answer: d) Capacitive reactance $X_c$

Explanation: You select a capacitor based on its physical properties (Value, Voltage Rating, Tolerance, Leakage, Temp Coefficient). Reactance ($X_c$) is a calculated value dependent on frequency ($1/2\pi fC$) during operation, not a manufacturing spec you select directly.

33.
The output of unfiltered half-wave is
a) Steady dc voltage
b) Pulsating dc voltage
c) Smooth dc voltage
d) None of the above
Show Answer

Correct Answer: b) Pulsating dc voltage

Explanation: A half-wave rectifier passes only the positive half-cycles, resulting in a series of pulses (pulsating DC) before filtering.

34.
What is the function of a zener diode in power supply circuit?
a) An amplifier
b) Rectifier
c) Voltage regulator
d) Resistor
Show Answer

Correct Answer: c) Voltage regulator

Explanation: Zener diodes are designed to maintain a constant voltage across their terminals when reverse-biased in the breakdown region, making them ideal for voltage regulation.

35.
What is extrinsic conduction?

d) It is reversely biased P-N junction

Correct Answer: b) Introduction of an external impurity to a pure semi-conductor

36.
An ideal PN junction diode is one which behaves as a perfect __________ when forward biased.
a) Conductor
b) Insulator
c) Resistance material
d) None of the above

Correct Answer: a) Conductor (Short Circuit)

37.
In the breakdown region, zener diode behaves like __________ source
a) Constant resistance
b) Constant power
c) Constant voltage
d) Constant current
Show Answer

Correct Answer: c) Constant voltage

Explanation: In breakdown, the voltage across the Zener diode remains nearly constant ($V_Z$) despite large changes in current.

38.
In reverse biased PN junction __________ carriers cause leakage current?
a) Majority carriers
b) Minority carriers
c) Both majority and minority carriers
d) Either the majority or minority carriers
Show Answer

Correct Answer: b) Minority carriers

Explanation: Leakage current in reverse bias is due to thermally generated minority carriers (holes in N-type, electrons in P-type) drifting across the junction.

39.
The transducer that convert the input signal into the output signal, which is a discrete function of time is known as
a) Active
b) Analogue
c) Digital
d) Pulse

Explanation: A digital transducer produces a discrete (digital) output signal (e.g., binary pulses), as opposed to an analog transducer which produces a continuous signal.

40.
When a trivalent impurity is added to a semiconductor trivalent becomes __________ of electrons
a) Donor
b) Acceptor
c) P-type
d) N-type

Section 2: Short Answer Questions

1.
Name the five gates commonly used and state their differences. (5 marks)
Show Answer

Answer:

  1. AND Gate: Output is HIGH only if all inputs are HIGH. (Logical Multiplication)
  2. OR Gate: Output is HIGH if at least one input is HIGH. (Logical Addition)
  3. NOT Gate (Inverter): Output is the inverse of the input (HIGH if input is LOW, and vice versa).
  4. NAND Gate: Output is LOW only if all inputs are HIGH (Inverted AND).
  5. NOR Gate: Output is HIGH only if all inputs are LOW (Inverted OR).

Note: XOR and XNOR are also common, but the first 5 are basic/universal.

4.
Name any three classes of amplifier and state their differences. (5 marks)
Show Answer
  1. Class A: Conducts for the full cycle ($360^\circ$). High linearity, low efficiency.
  2. Class B: Conducts for half the cycle ($180^\circ$). Used in Push-Pull to cover full cycle. Higher efficiency than A, crossover distortion exists.
  3. Class C: Conducts for less than half cycle ($<180^\circ$). Highest efficiency, high distortion. Used in RF applications.

Section 3: Long Answer Questions

20 Marks
1.

a) Draw a labelled diagram of a centre-tapped full-wave rectifier and explain how the circuit works. (10 marks)

b) name and explain the functions of the terminals of a JFET. (6 marks)

c) Sketch a resistance temperature graph if a thermistor and explain its shape. (4 marks)

Show Answer

a) Centre-Tapped Full-Wave Rectifier

      T1 (Transformer)
       +--------+----> D1 ----+
       |        |             |
AC   ==+==      +             +-----> + Vout
In     |      (CT)            |
       |        +----[ RL ]---+-----> - Vout (GND)
       +--------+             |
                |             |
                +----> D2 ----+

Operation:

  • Positive Half-Cycle: The top end of the secondary winding is positive relative to the center tap. D1 is forward biased and conducts. Current flows through Load RL. D2 is reverse biased.
  • Negative Half-Cycle: The bottom end is positive relative to the center tap. D2 is forward biased and conducts. Current flows through Load RL in the same direction. D1 is reverse biased.
  • Result: Both halves of the AC cycle are converted to DC pulses in the same direction.

b) JFET Terminals

  • Source (S): The terminal through which majority carriers enter the channel. Equivalent to the Emitter in BJT.
  • Drain (D): The terminal through which majority carriers leave the channel. Equivalent to the Collector in BJT.
  • Gate (G): The control terminal. By applying a reverse voltage to the Gate-Source junction, the width of the depletion layer is varied, controlling the flow of current between Source and Drain.

c) Thermistor Resistance-Temperature Graph (NTC)

Shape: Non-linear exponential decay.

Resistance (R)
^
| *
|  *
|   *
|    *
|      *
|         *
|             * * *
+--------------------> Temperature (T)

Explanation: For a Negative Temperature Coefficient (NTC) thermistor, resistance decreases as temperature increases. The change is large at low temperatures and smaller at high temperatures, following an exponential relationship.
