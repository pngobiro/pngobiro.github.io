                            <p style="margin-top: 1rem;">Carbohydrates are the sugars, starches, and dietary fiber that occur in plant foods and dairy products. Carbohydrates are mainly found in plant foods. They also occur in dairy products in the form of a milk sugar called lactose.</p>
                            <p>Carbohydrates contain carbon, hydrogen, and oxygen atoms. Oxygen and hydrogen are present in the same proportion as in water.</p>
                            <p>Two basic compounds make up carbohydrates: Aldehydes, which are double-bonded carbon and oxygen atoms, plus a hydrogen atom, and ketones, which are double-bonded carbon and oxygen atoms, plus two additional carbon atoms.</p>

                        <div class="content-card">
                            <h3 class="card-title">Classification of Carbohydrates</h3>

                                            <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Galactose:</strong> Most readily available in milk and dairy products.</span></li>

Sometimes called ‚Äòmalt sugar‚Äô and is found in cereals such as barley where it is formed during germination.</span></li>
                                        </ul>

                                    <p class="vuln-text"><strong>Sources:</strong> Cereals (wheat, rice, oats), Pulses (peas, beans, lentils), Nuts, Vegetables, Gelatine (animal exception).</p>

                            <ul class="enhanced-list">

                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Pregnant women require more than normal to cater for the growing baby</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Nursing mothers requires more than normal milk production during breast-feeding (lactation)</span></li>

                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Oils:</strong> Liquid at room temperature</span></li>
                            </ul>
                            <p>Fats are composed of the elements: carbon, oxygen, hydrogen. These molecules make up molecules of glycerol and fatty acids which combine to form fat molecules known as triglycerol (sometimes called a triglyceride).</p>

                            <h3 class="card-title" style="margin-top: 1.5rem;">Types of Fats</h3>
                            
                            <div class="feature-grid">
                                <div class="feature-card">
                                    <div class="feature-icon">ü´í</div>
                                    <h4 class="feature-title">Monounsaturated Fats</h4>
                                    <p class="feature-text">Have one double bond. Found in olive, peanut, canola oils, avocados, almonds, hazelnuts, pumpkin seeds.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üåª</div>
                                    <h4 class="feature-title">Polyunsaturated Fats</h4>
                                    <p class="feature-text">Have more than one double bond. Found in sunflower, corn, soybean, flaxseed oils, walnuts.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üêü</div>
                                    <h4 class="feature-title">Omega-3 Fats</h4>
                                    <p class="feature-text">Body cannot make these. Found in fish, flax seeds, walnuts, canola or soybean oil.</p>
                                </div>
                            </div>

                            <h3 class="card-title" style="margin-top: 1.5rem;">Functions of Fats</h3>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Provides a convenient and concentrated source of energy</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Surrounds and protects vital organs e.g kidney, glands</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Forms insulating layer (adipose tissue) beneath the skin</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Forms part of the structure of cell membranes</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Provides a source of fat-soluble vitamins A,D,E, and K</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Provides texture and flavour in food</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Provides a feeling of fullness (satiety)</span></li>
                            </ul>
                        </div>
                    </section>

                    <section class="content-section" id="vitamins">
                        <div class="section-header">
                            <div class="section-number">04</div>
                            <h2 class="section-title">
                                <span class="title-icon">üíä</span>
                                Micronutrients: Vitamins
                            </h2>
                        </div>

                        <div class="content-card content-card--accent">
                            <h3 class="card-title">Fat-Soluble Vitamins</h3>
                            
                            <div class="collapsible-section">
                                <details>
                                    <summary class="collapsible-trigger">
                                        <span class="trigger-icon">‚ñ∂</span>
                                        Vitamin A (Retinol)
                                    </summary>
                                    <div class="collapsible-content">
                                        <p><strong>Roles:</strong> Regulation of gene expression, promotes development of immune cells (Natural Killer Cells), antioxidant, promotes good vision (rhodopsin), promotes healthy skin, growth and maintenance of bones and teeth.</p>
                                        <p><strong>Sources:</strong></p>
                                        <ul class="enhanced-list">
                                            <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Retinol (Animal):</strong> Milk, cheese, eggs (yolk), butter, oily fish, Liver, kidney.</span></li>
                                            <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Beta-carotene (Plant):</strong> Carrots, spinach, apricots, parsley, cabbage, tomatoes, prunes.</span></li>
                                        </ul>
                                        <div class="callout callout--warning">
                                            <div class="callout-header"><span class="callout-icon">‚ö†Ô∏è</span><h4 class="callout-title">Warning</h4></div>
                                            <div class="callout-content"><p>Too much vitamin A is poisonous. Pregnant women are advised not to eat liver due to high retinol which could lead to defects in unborn baby.</p></div>
                                        </div>
                                    </div>
                                </details>
                            </div>

                            <div class="collapsible-section">
                                <details>
                                    <summary class="collapsible-trigger">
                                        <span class="trigger-icon">‚ñ∂</span>
                                        Vitamin D (Cholecalciferol)
                                    </summary>
                                    <div class="collapsible-content">
                                        <p><strong>Functions:</strong> Required for proper formation of bones and teeth. Helps promote absorption of calcium and phosphorus.</p>
                                        <p><strong>Sources:</strong> Liver, Fish-liver oils, Oily fish (herring, sardine), Sunlight (converts dehydro-cholesterol to cholecalciferol).</p>
                                        <p><strong>Deficiency:</strong> Rickets in children, Osteomalacia in adults, Osteoporosis.</p>
                                        <p><strong>Toxicity:</strong> Loss of calcium from bone, loss of appetite, nausea, vomiting, psychological depression.</p>
                                    </div>
                                </details>
                            </div>

                            <div class="collapsible-section">
                                <details>
                                    <summary class="collapsible-trigger">
                                        <span class="trigger-icon">‚ñ∂</span>
                                        Vitamin E (Tocopherol)
                                    </summary>
                                    <div class="collapsible-content">
                                        <p><strong>Roles:</strong> Antioxidant, prevents damage to lungs/RBCs/WBCs, necessary for normal nerve development, acts as blood thinner.</p>
                                        <p><strong>Sources:</strong> Vegetable oils, salad dressings, whole grain cereals, green leafy vegetables, nuts, seeds, peanut butter, wheat germ.</p>
                                        <p><strong>Deficiency:</strong> Liver disease, premature babies with fragile RBCs, loss of muscle coordination/vision/immune functions.</p>
                                        <p><strong>Toxicity:</strong> Increases effects of anticoagulants.</p>
                                    </div>
                                </details>
                            </div>

                            <div class="collapsible-section">
                                <details>
                                    <summary class="collapsible-trigger">
                                        <span class="trigger-icon">‚ñ∂</span>
                                        Vitamin K
                                    </summary>
                                    <div class="collapsible-content">
                                        <p><strong>Functions:</strong> Promotes synthesis of blood clotting proteins, acts in bone formation. Produced by bacteria in large intestines.</p>
                                        <p><strong>Sources:</strong> Turnip greens, cauliflower, spinach, liver, broccoli, kales, cabbage.</p>
                                        <p><strong>Deficiency:</strong> Rare but seen in infants or after prolonged antibiotics.</p>
                                        <p><strong>Toxicity:</strong> Rupture of RBCs, jaundice.</p>
                                    </div>
                                </details>
                            </div>
                        </div>

                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Selenium:</strong> Antioxidant. Deficiency: Heart disease risk.</span></li>

                    <section class="content-section" id="supplements">
                        <div class="section-header">
                            <div class="section-number">06</div>
                            <h2 class="section-title">
                                <span class="title-icon">üíä</span>
                                Supplements & Healthy Eating
                            </h2>
                        </div>

                        <div class="content-card">
                            <h3 class="card-title">Food Supplements</h3>
                            <p>A food supplement is a preparation that is intended to supply a nutrient that is missing from a diet. It is intended to be taken by mouth as a pill, capsule, tablet or liquid.</p>
                            
                            <h4 class="card-title">Principles of Supplementation</h4>
                            <ol class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Read carefully the labels.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Professionally determined individual needs govern specific supplement usage.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">All nutrient work together to promote good health.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Food is the best source of nutrients.</span></li>
                            </ol>

                            <div class="callout callout--danger">
                                <div class="callout-header">
                                    <span class="callout-icon">‚ò£Ô∏è</span>
                                    <h4 class="callout-title">Supplement Toxicity</h4>
                                </div>
                                <div class="callout-content">
                                    <p>Taking large amounts of vitamins can lead to hypervitaminosis.</p>
                                    <ul>
                                        <li>Too much vit D can harm your kidneys</li>
                                        <li>Vit A overdose causes nausea, vomiting, dizziness, blurry vision</li>
                                        <li>Excess calcium settles in arteries causing obstruction</li>
                                    </ul>
                                </div>
                            </div>
                        </div>

                        <div class="content-card content-card--gradient">
                            <div class="card-header-with-icon">
                                <span class="card-icon">ü•ó</span>
                                <h3 class="card-title">Guidelines on Healthy Eating</h3>
                            </div>
                            <p>A healthy diet helps to protect against malnutrition in all its forms, as well as non-communicable diseases (NCDs).</p>
                            
                            <div class="feature-grid">
                                <div class="feature-card">
                                    <div class="feature-icon">üçé</div>
                                    <h4 class="feature-title">Fruits & Veg</h4>
                                    <p class="feature-text">At least 400g (5 portions) per day.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üßÇ</div>
                                    <h4 class="feature-title">Salt</h4>
                                    <p class="feature-text">Less than 5g per day (iodized).</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üç¨</div>
                                    <h4 class="feature-title">Sugars</h4>
                                    <p class="feature-text">Limit free sugars to less than 10% of energy.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üçî</div>
                                    <h4 class="feature-title">Fats</h4>
                                    <p class="feature-text">Total fat < 30%. Saturated < 10%. Trans-fats < 1%.</p>
                                </div>
                            </div>
                        </div>

                        <div class="content-card">

                            <div class="process-steps">
                                <div class="process-step">
                                    <div class="step-number">01</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Grilling</h5>

                                </div>
                                <div class="process-step">
                                    <div class="step-number">02</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Steaming</h5>
                                        <p class="step-text">Food placed in a steamer over hot liquid. Good way to retain nutrients (water-soluble vitamins).</p>
                                    </div>
                                </div>
                                <div class="process-step">
                                    <div class="step-number">03</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Searing</h5>
                                        <p class="step-text">Browning of food using small amounts of fat over high heat to caramelize the outside.</p>
                                    </div>
                                </div>
                                <div class="process-step">
                                    <div class="step-number">04</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Boiling</h5>
                                        <p class="step-text">Cooking in water at 100¬∞C. Most water-soluble vitamins are destroyed.</p>
                                    </div>
                                </div>
                                <div class="process-step">
                                    <div class="step-number">05</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Sauteing</h5>
                                        <p class="step-text">Cooking in very little oil or fat until tender.</p>
                                    </div>
                                </div>
                                <div class="process-step">
                                    <div class="step-number">06</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Poaching</h5>
                                        <p class="step-text">Submerged in water between 71¬∞C and 82¬∞C. Example: cooking githeri.</p>
                                    </div>
                                </div>
                                <div class="process-step">
                                    <div class="step-number">07</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Broiling</h5>
                                        <p class="step-text">Source of heat comes from the top (ovens).</p>
                                    </div>
                                </div>
                            </div>

                            <h3 class="card-title" style="margin-top: 2rem;">Importance of Food Processing</h3>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">It helps to preserve food and prevent it from deterioration e.g milk.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Many food are inedible unless processed in some manner e.g coffee, cocoa, etc.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">It helps to remove toxins occurring naturally in foods e.g the hydrogen cyanide removed by fermenting cassava.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">To make the food taste better.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">To make food stay longer e.g drying cassava, plantain, etc.</span></li>
                            </ul>

                    <!-- Maternal Nutrition Section -->
                    <section class="content-section" id="maternal-nutrition">
                        <div class="section-header">
                            <div class="section-number">01</div>
                            <h2 class="section-title">
                                <span class="title-icon">ü§∞</span>
                                Maternal Nutrition
                            </h2>
                        </div>

                        <div class="content-card content-card--accent">
                            <div class="card-header-with-icon">
                                <span class="card-icon">üçé</span>
                                <h3 class="card-title">Pregnancy & Breastfeeding Guidelines</h3>
                            </div>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Eat <strong>two additional small meals or snacks</strong> each day to provide energy and nutrition for the growing baby.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Consume a minimum of <strong>2 to 3 liters of liquids</strong> per day.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Eat nutritious foods: milk, fresh fruit/vegetables, meat, fish, eggs, grains, groundnuts, peas, and beans.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Avoid:</strong> Taking tea or coffee with meals (interferes with nutrient use).</span></li>
                            </ul>

                            <h4 class="card-title" style="margin-top: 1.5rem;">Supplementation Schedule</h4>
                            <div class="process-steps">
                                <div class="process-step">
                                    <div class="step-number">Fe</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Iron & Folic Acid</h5>
                                        <p class="step-text">Take during pregnancy and continue for 6‚Äì12 weeks following delivery to prevent anemia.</p>
                                    </div>
                                </div>
                                <div class="process-step">
                                    <div class="step-number">A</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Vitamin A</h5>
                                        <p class="step-text">Take immediately after delivery or within 6 weeks so baby receives it in breast milk.</p>
                                    </div>
                                </div>
                                <div class="process-step">
                                    <div class="step-number">I</div>
                                    <div class="step-content">
                                        <h5 class="step-title">Iodized Salt</h5>
                                        <p class="step-text">Use to help the baby‚Äôs brain and body develop well.</p>
                                    </div>
                                </div>
                            </div>
                        </div>

                        <div class="content-card">
                            <h3 class="card-title">Adolescent Mothers</h3>
                            <p>Adolescent mothers need extra care, more food, and more rest than older mothers because they need to nourish their own growing body as well as the baby‚Äôs.</p>
                            <div class="callout callout--note">
                                <div class="callout-header"><span class="callout-icon">‚ÑπÔ∏è</span><h4 class="callout-title">Key Actions</h4></div>
                                <div class="callout-content">
                                    <ul class="enhanced-list">
                                        <li class="list-item"><span class="list-marker"></span><span class="list-content">Learn HIV status.</span></li>
                                        <li class="list-item"><span class="list-marker"></span><span class="list-content">Attend all clinic appointments.</span></li>
                                        <li class="list-item"><span class="list-marker"></span><span class="list-content">Take medicines as advised by health provider.</span></li>
                                    </ul>
                                </div>
                            </div>
                        </div>
                    </section>

                    <!-- Infant Nutrition Section -->
                    <section class="content-section" id="infant-nutrition">
                        <div class="section-header">
                            <div class="section-number">02</div>
                            <h2 class="section-title">
                                <span class="title-icon">ü§±</span>
                                Infant Nutrition
                            </h2>
                        </div>

                        <div class="content-card content-card--gradient">
                            <div class="card-header-with-icon">
                                <span class="card-icon">‚ö°</span>
                                <h3 class="card-title">Immediate Breastfeeding</h3>
                            </div>
                            <div class="feature-grid">
                                <div class="feature-card">
                                    <div class="feature-icon">‚è±Ô∏è</div>
                                    <h4 class="feature-title">Within 1 Hour</h4>
                                    <p class="feature-text">Begin breastfeeding within the first hour. Hold newborn skin-to-skin immediately for one hour.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">ü•õ</div>
                                    <h4 class="feature-title">Colostrum</h4>
                                    <p class="feature-text">First thick yellowish milk. Protects from illness and helps pass meconium.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üö´</div>
                                    <h4 class="feature-title">No Water</h4>
                                    <p class="feature-text">Do not give water or other liquids. Dangerous for newborn.</p>
                                </div>
                            </div>
                        </div>

                        <div class="content-card">
                            <h3 class="card-title">Exclusive Breastfeeding (0-6 Months)</h3>
                            <div class="callout callout--success">
                                <div class="callout-header"><span class="callout-icon">‚úÖ</span><h4 class="callout-title">Definition</h4></div>
                                <div class="callout-content">
                                    <p>Feeding the baby <strong>ONLY breast milk</strong> for the first 6 months, without weaning. No water, no herbs, no other liquids.</p>
                                </div>
                            </div>
                            <div class="vulnerability-grid">
                                <div class="vulnerability-card">
                                    <div class="vuln-icon">üõ°Ô∏è</div>
                                    <h4 class="vuln-title">Benefits</h4>
                                    <p class="vuln-text">Complete food for 6 months. Protects against diarrhea and respiratory infections.</p>
                                </div>
                                <div class="vulnerability-card">
                                    <div class="vuln-icon">‚ö†Ô∏è</div>
                                    <h4 class="vuln-title">Risks of Mixed Feeding</h4>
                                    <p class="vuln-text">Diarrhea, pneumonia, malnutrition. Weaning too early causes GIT abrasions where infection can enter.</p>
                                </div>
                            </div>
                        </div>

                        <div class="content-card">
                            <h3 class="card-title">Breastfeeding on Demand</h3>
                            <p>Breastfeed as often as the baby wants, day and night (every 2-3 hours). The more the baby suckles, the more milk is produced.</p>
                            
                            <h4 class="card-title">Signs of Hunger (Early)</h4>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Restlessness</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Opening mouth and turning head</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Putting tongue in and out</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Suckling on fingers/fists</span></li>
                            </ul>
                            <p><em>Note: Crying is a late sign or may mean discomfort, not necessarily hunger.</em></p>

                            <div class="callout callout--tip">
                                <div class="callout-header"><span class="callout-icon">üí°</span><h4 class="callout-title">Foremilk vs. Hindmilk</h4></div>
                                <div class="callout-content">
                                    <p>Let baby finish one breast before offering the other.</p>
                                    <ul>
                                        <li><strong>Foremilk:</strong> More water, satisfies thirst.</li>
                                        <li><strong>Hindmilk:</strong> More fat, satisfies hunger.</li>
                                    </ul>
                                </div>
                            </div>
                        </div>
                    </section>

                    <!-- School Age Nutrition Section -->
                    <section class="content-section" id="school-age">
                        <div class="section-header">
                            <div class="section-number">03</div>
                            <h2 class="section-title">
                                <span class="title-icon">üéí</span>
                                School-Going Children (6-12 Years)
                            </h2>
                        </div>

                        <div class="content-card">
                            <p>Steady but slow growth rate. Usually eat 4-5 times a day. Habits influenced by family, friends, and media.</p>
                            
                            <h4 class="card-title">5 Essential Nutrients</h4>
                            <div class="feature-grid">
                                <div class="feature-card">
                                    <div class="feature-icon">ü•©</div>
                                    <h4 class="feature-title">Protein</h4>
                                    <p class="feature-text">Failure to grow, irritability, weak immunity. Sources: Poultry, lean meat, eggs, dairy, legumes.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">ü©∏</div>
                                    <h4 class="feature-title">Iron</h4>
                                    <p class="feature-text">Deficiency leads to cognitive/motor delays. Sources: Lean meat, dark poultry, leafy greens.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">ü¶¥</div>
                                    <h4 class="feature-title">Vitamin D</h4>
                                    <p class="feature-text">Bone health. Sources: Fish, dairy, egg yolks, sunlight.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">ü•õ</div>
                                    <h4 class="feature-title">Calcium</h4>
                                    <p class="feature-text">Bone development, muscle function. Sources: Dairy, fortified cereals.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üêü</div>
                                    <h4 class="feature-title">Omega-3s</h4>
                                    <p class="feature-text">Cognitive development. Sources: Fish, fish oil.</p>
                                </div>
                            </div>

                            <h4 class="card-title" style="margin-top: 1.5rem;">Mealtime Hints</h4>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Always serve breakfast (Fruit, milk, toast, cereal).</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Set good examples.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Let children help with meal planning.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Serve meals at the table without distractions (TV, phones).</span></li>
                            </ul>
                        </div>
                    </section>

                    <!-- Adolescent Nutrition Section -->
                    <section class="content-section" id="adolescence">
                        <div class="section-header">
                            <div class="section-number">04</div>
                            <h2 class="section-title">
                                <span class="title-icon">üèÉ</span>
                                Adolescence
                            </h2>
                        </div>

                        <div class="content-card content-card--gradient">
                            <div class="card-header-with-icon">
                                <span class="card-icon">üìà</span>
                                <h3 class="card-title">Growth & Needs</h3>
                            </div>
                            <p>Period of rapid growth (Girls: 11-14, Boys: 13-16). High physiological stress for girls due to menstruation.</p>
                            
                            <h4 class="card-title">Nutritional Requirements</h4>
                            <div class="vulnerability-grid">
                                <div class="vulnerability-card">
                                    <div class="vuln-icon">‚ö°</div>
                                    <h4 class="vuln-title">Energy</h4>
                                    <p class="vuln-text">Higher needs due to BMR, activity, and growth. Boys need more than girls.</p>
                                </div>
                                <div class="vulnerability-card">
                                    <div class="vuln-icon">üí™</div>
                                    <h4 class="vuln-title">Protein</h4>
                                    <p class="vuln-text">For lean body mass accumulation. Inadequacy leads to delayed sexual maturation and linear growth reduction.</p>
                                </div>
                                <div class="vulnerability-card">
                                    <div class="vuln-icon">ü¶¥</div>
                                    <h4 class="vuln-title">Calcium</h4>
                                    <p class="vuln-text">Needs greater than childhood/adulthood due to skeletal growth.</p>
                                </div>
                                <div class="vulnerability-card">
                                    <div class="vuln-icon">ü©∏</div>
                                    <h4 class="vuln-title">Iron</h4>
                                    <p class="vuln-text">Blood volume expansion + muscle mass. Menstruation adds needs for girls.</p>
                                </div>
                                <div class="vulnerability-card">
                                    <div class="vuln-icon">üß¨</div>
                                    <h4 class="vuln-title">Zinc</h4>
                                    <p class="vuln-text">Essential for growth and sexual maturation.</p>
                                </div>
                            </div>
                        </div>

                        <div class="content-card">
                            <h3 class="card-title">Nutrition Related Problems</h3>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Acne vulgaris:</strong> Skin disease involving hair follicles.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Anemia:</strong> Iron deficiency.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Obesity:</strong> Excess weight gain.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Eating Disorders:</strong></span>
                                    <ul class="enhanced-list">
                                        <li class="list-item"><span class="list-marker">‚ó¶</span><span class="list-content">Anorexia nervosa (Fear of gaining weight, low body weight).</span></li>
                                        <li class="list-item"><span class="list-marker">‚ó¶</span><span class="list-content">Bulimia nervosa (Binge eating followed by loss of control).</span></li>
                                        <li class="list-item"><span class="list-marker">‚ó¶</span><span class="list-content">Binge eating disorder.</span></li>
                                    </ul>
                                </li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content"><strong>Osteoporosis predisposition:</strong> Weak bones later in life.</span></li>
                            </ul>
                        </div>
                    </section>

                                <span class="title-icon">üë¥</span>
                                Elderly Nutrition
                            </h2>
                        </div>

                        <div class="content-card">
                            <h3 class="card-title">Physiological Changes (>51 Years)</h3>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Decline in immune function, hormone production, muscle mass.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Reduced efficiency of heart and kidneys.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Lower efficiency in vitamin/mineral absorption.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Slight reduction in height.</span></li>
                            </ul>

                            <h4 class="card-title" style="margin-top: 1.5rem;">Specific Nutritional Requirements</h4>
                            <div class="table-wrapper">
                                <table class="content-table">
                                    <thead>
                                        <tr>
                                            <th>Nutrient</th>
                                            <th>Requirement/Note</th>
                                        </tr>
                                    </thead>
                                    <tbody>
                                        <tr>
                                            <td><strong>Energy</strong></td>
                                            <td>Lower needs due to reduced lean mass and metabolic rate.</td>
                                        </tr>
                                        <tr>
                                            <td><strong>Fiber</strong></td>
                                            <td>Crucial to prevent constipation and diverticulitis.</td>
                                        </tr>
                                        <tr>
                                            <td><strong>Calcium & Vit D</strong></td>
                                            <td>Increase intake to slow bone loss (Men >70, all women).</td>
                                        </tr>
                                        <tr>
                                            <td><strong>Vitamin B12</strong></td>
                                            <td>Low stomach acid reduces absorption. Use fortified foods/supplements.</td>
                                        </tr>
                                        <tr>
                                            <td><strong>Iron</strong></td>
                                            <td>Higher levels no longer needed for post-menopausal women.</td>
                                        </tr>
                                        <tr>
                                            <td><strong>Fluids</strong></td>
                                            <td>8 glasses/day to prevent dehydration and constipation.</td>
                                        </tr>
                                    </tbody>
                                </table>
                            </div>

                    <!-- Definition of Malnutrition -->
                    <section class="content-section" id="definitions">
                        <div class="section-header">
                            <div class="section-number">01</div>
                            <h2 class="section-title">
                                <span class="title-icon">üìñ</span>
                                What is Malnutrition?
                            </h2>
                        </div>

                        <div class="content-card">
                            <p>Malnutrition means poor nutrition. This can be divided into:</p>
                            
                            <div class="definitions-grid">
                                <div class="definition-card definition-card--first">
                                    <div class="definition-icon">‚¨áÔ∏è</div>
                                    <div class="definition-content">
                                        <h4 class="definition-term">Under nutrition</h4>
                                        <p class="definition-text">When one does not eat enough of one or more nutrients that the body requires, then the body does not have enough nutrients to produce energy, grow and repair the tissues and protect itself from diseases. This results in people becoming underweight. They may become too thin (Wasted) or too short (Stunted).</p>
                                    </div>
                                </div>

                                <div class="definition-card definition-card--multiple">
                                    <div class="definition-icon">‚¨ÜÔ∏è</div>
                                    <div class="definition-content">
                                        <h4 class="definition-term">Over nutrition</h4>
                                        <p class="definition-text">When one takes diets that provide excess energy and lack physical exercise. This results in over nutrition and may lead to nutrition related diseases that develop over a long period of time e.g. heart disease, and diabetes.</p>
                                    </div>
                                </div>

                                <div class="definition-card">
                                    <div class="definition-icon">üî¨</div>
                                    <div class="definition-content">
                                        <h4 class="definition-term">Micronutrient deficiencies</h4>
                                        <p class="definition-text">This occurs when the body lacks one or more micronutrients.</p>
                                    </div>
                                </div>
                            </div>
                        </div>
                    </section>

                    <!-- Signs of Malnutrition -->
                    <section class="content-section" id="clinical-forms">
                        <div class="section-header">
                            <div class="section-number">02</div>
                            <h2 class="section-title">
                                <span class="title-icon">ü©∫</span>
                                Clinical Signs of Malnutrition
                            </h2>
                        </div>

                        <div class="content-card content-card--accent">
                            <h3 class="card-title">1. Marasmus</h3>
                            <p>Marasmus is a severe form of protein-energy malnutrition that results when a person does not consume enough protein and calories. It leads to wasting, or the loss of body fat and muscle.</p>
                            
                            <div class="feature-grid">
                                <div class="feature-card">
                                    <div class="feature-icon">ü¶¥</div>
                                    <h4 class="feature-title">Appearance</h4>
                                    <p class="feature-text">Extremely emaciated (thin), no fat on body, ribs are visible. "Old man's appearance".</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üß¨</div>
                                    <h4 class="feature-title">Skin & Hair</h4>
                                    <p class="feature-text">Thin, flaccid skin hanging in loose folds (especially buttocks). Hair becomes sparse, dry, brittle, yellow/reddish.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üíß</div>
                                    <h4 class="feature-title">State</h4>
                                    <p class="feature-text">Frequent dehydration. Alert and irritable. Usually has appetite.</p>
                                </div>
                            </div>

                            <div class="collapsible-section">
                                <details>
                                    <summary class="collapsible-trigger">
                                        <span class="trigger-icon">‚ñ∂</span>
                                        Complications & Causes
                                    </summary>
                                    <div class="collapsible-content">
                                        <p><strong>Complications:</strong> Bradycardia (Slow heart rate beat), Hypotension (Low blood pressure), Hypothermia (Low body temperature). Diarrhea, measles, or respiratory infections can be fatal.</p>
                                        <p><strong>Causes:</strong> Not having enough nutrition, consuming wrong nutrients, health conditions affecting absorption. In places where food is scarce, extended breastfeeding (>6 months) without solid food can increase risk.</p>
                                    </div>
                                </details>
                            </div>
                        </div>

                        <div class="content-card content-card--gradient">
                            <h3 class="card-title">2. Kwashiorkor</h3>
                            <p>Kwashiorkor is another severe form of protein-energy malnutrition where the main deficiency is protein.</p>
                            
                            <div class="feature-grid">
                                <div class="feature-card">
                                    <div class="feature-icon">üéà</div>
                                    <h4 class="feature-title">Oedema</h4>
                                    <p class="feature-text">Severe swelling beginning in legs/feet, generalized to hands/arms ("moon face"). Masks muscle loss.</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üßí</div>
                                    <h4 class="feature-title">Appearance</h4>
                                    <p class="feature-text">Distended/bulging abdomen. Changes in hair color to yellow/orange. Skin lesions (cracked, peeling).</p>
                                </div>
                                <div class="feature-card">
                                    <div class="feature-icon">üò¥</div>
                                    <h4 class="feature-title">Behavior</h4>
                                    <p class="feature-text">Apathetic and lethargic (inactive). Irritable when handled. Poor appetite.</p>
                                </div>
                            </div>

                            <div class="callout callout--danger">
                                <div class="callout-header">
                                    <span class="callout-icon">üö®</span>
                                    <h4 class="callout-title">Critical Note</h4>
                                </div>
                                <div class="callout-content">
                                    <p>Kwashiorkor needs immediate diagnosis and treatment, as it can quickly become life-threatening. Liver problems can occur with kwashiorkor, but they are rare with marasmus.</p>
                                </div>
                            </div>
                        </div>
                    </section>

                        <div class="content-card content-card--alternate">
                            <div class="card-header-with-icon">
                                <span class="card-icon">ü¶†</span>
                                <h3 class="card-title">Nutrition and Infection</h3>
                            </div>
                            <p>Infection and malnutrition have always been intricately linked. Malnutrition is the primary cause of immunodeficiency worldwide. Five infectious diseases account for more than one-half of all deaths in children aged <5 years, most of whom are undernourished.</p>
                            <p>Malnutrition and nutritional alterations are common complications of human immunodeficiency virus (HIV) infection.</p>
                            
                            <h4 class="card-title" style="margin-top: 1.5rem;">Dangers of Malnutrition</h4>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Reduced muscle and tissue mass.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Decreased mobility and stamina due to muscle wasting.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Breathing difficulties and increased risk of chest infection/respiratory failure.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Wounds take longer to heal and illnesses take longer to recover from.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Slower immune response increasing risk of infection.</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Difficulty staying warm (risk of hypothermia).</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Poor libido and fertility problems.</span></li>
                            </ul>
                        </div>
                    </section>

                                Community Health Role
                            </h2>
                        </div>

                        <div class="content-card">
                            <h3 class="card-title">Building Partnership</h3>
                            <p>Effective partnership is essential for community-based solutions. The causes of malnutrition are multi-sectoral (Planning, Agriculture, Industry, Economy, Education, etc.). Action is needed nationally, locally and within families.</p>

                            <h3 class="card-title" style="margin-top: 2rem;">Role of Community Health Assistants (CHAs)</h3>
                            <ul class="enhanced-list">
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Provide IEC on nutrition services available at health facility and community levels</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Screen, identify and make referrals for malnutrition</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Conduct growth monitoring for under-fives</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Follow-up and defaulter tracing for clients with malnutrition</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Referrals for micronutrient supplementation</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Promote, protect and support exclusive breastfeeding for the first six months of life and sustained breastfeeding for the first two years and beyond within the community</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Carry out health promotion by providing information, education and communication (IEC) about healthy diet for people in all stages of the life cycle, particularly among vulnerable populations</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Promote use of improved home-based recipes and preparation methods for locally available foods, including home fortification</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Liaise and collaborate with other sectors to address food and nutrition security at household level</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Maternal nutrition, screening, advice, referral for further management</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Distribute iron and folic supplements</span></li>
                                <li class="list-item"><span class="list-marker"></span><span class="list-content">Distribute de-wormers</span></li>
                            </ul>
                        </div>

1. Core Quiz Engine

/**
 * PDF to HTML Quiz Engine v1.0
 * Handles rendering, interaction, and scoring for JSON-based quizzes.
 */

(function () {
    'use strict';

    // State management
    const state = {
        quizData: null,
        currentQuestionIndex: 0,
        userAnswers: {}, // { questionId: answer(s) }
        score: 0,
        timer: null,
        timeRemaining: 0,
        isReviewMode: false
    };

    // Registry for JSONP loading
    window.quizRegistry = {};
    window.registerQuiz = function (id, data) {
        window.quizRegistry[id] = data;
    };

    /**
     * Initialize Quiz
     */
    function init() {
        const container = document.querySelector('.quiz-container');
        if (!container) return;

        const quizId = container.dataset.quizId;
        const quizJsPath = container.dataset.quizJs;
        const quizJsonPath = container.dataset.quizSrc;

        if (window.quizRegistry[quizId]) {
            loadQuiz(window.quizRegistry[quizId]);
        } else if (quizJsPath) {
            // Load via JSONP (works with file://)
            const script = document.createElement('script');
            script.src = quizJsPath;
            script.onload = () => {
                if (window.quizRegistry[quizId]) {
                    loadQuiz(window.quizRegistry[quizId]);
                } else {
                    showError('Quiz data not found in registry.');
                }
            };
            script.onerror = () => showError('Failed to load quiz script.');
            document.body.appendChild(script);
        } else if (quizJsonPath) {
            // Load via Fetch (requires server)
            fetch(quizJsonPath)
                .then(res => res.json())
                .then(data => loadQuiz(data))
                .catch(err => showError(err.message));
        }
    }

    function loadQuiz(data) {
        state.quizData = data;
        state.timeRemaining = data.settings?.timeLimit || 0;
        
        // Shuffle if enabled
        if (data.settings?.shuffleQuestions) {
            state.quizData.questions = shuffleArray([...data.questions]);
        }

        renderStartScreen();
    }

    /**
     * Rendering Functions
     */
    function renderStartScreen() {
        const content = document.querySelector('.quiz-content');
        const loading = document.querySelector('.quiz-loading');
        
        loading.hidden = true;
        content.hidden = false;
        
        const { metadata, questions } = state.quizData;
        const timeText = metadata.estimatedTime ? `${metadata.estimatedTime} mins` : 'Unlimited';

        content.innerHTML = `
            <div class="quiz-start-card">
                <div class="quiz-info-grid">
                    <div class="info-item"><span>‚ùì</span> ${questions.length} Questions</div>
                    <div class="info-item"><span>‚è±Ô∏è</span> ${timeText}</div>
                    <div class="info-item"><span>üéØ</span> ${metadata.passingScore}% to Pass</div>
                </div>
                <button class="quiz-button quiz-button--primary" id="start-btn" style="width:100%; margin-top:1.5rem">
                    Start Quiz
                </button>
            </div>
        `;

        document.getElementById('start-btn').addEventListener('click', startQuiz);
    }

    function startQuiz() {
        state.currentQuestionIndex = 0;
        state.userAnswers = {};
        state.isReviewMode = false;
        renderQuestion();
        
        if (state.timeRemaining > 0) {
            startTimer();
        }
    }

    function renderQuestion() {
        const question = state.quizData.questions[state.currentQuestionIndex];
        const content = document.querySelector('.quiz-content');
        const total = state.quizData.questions.length;

        let html = `
            <div class="quiz-progress">
                <div class="quiz-progress-bar">
                    <div class="quiz-progress-fill" style="width: ${((state.currentQuestionIndex) / total) * 100}%"></div>
                </div>
                <span class="quiz-progress-text">Question ${state.currentQuestionIndex + 1} of ${total}</span>
            </div>
            
            <div class="quiz-question" id="question-card">
                <div class="question-header">
                    <span class="question-number">${state.currentQuestionIndex + 1}</span>
                    <div class="question-meta-wrapper" style="flex:1">
                        ${question.topicName ? `<span class="question-topic-badge">${question.topicName}</span>` : ''}
                        <h3 class="question-text">${question.question}</h3>
                    </div>
                    <span class="question-points">${question.points} pts</span>
                </div>
                
                ${renderMedia(question.media)}
                
                <div class="quiz-options">
                    ${renderInput(question)}
                </div>

                ${state.isReviewMode ? renderFeedback(question) : ''}
            </div>

            <div class="quiz-actions">
                ${!state.isReviewMode && question.hint ? `
                    <button class="hint-button" onclick="this.nextElementSibling.hidden = !this.nextElementSibling.hidden">
                        üí° Show Hint
                    </button>
                    <div class="hint-content" hidden>${question.hint}</div>
                ` : '<div></div>'}
                
                <div class="quiz-nav-buttons">
                    ${state.currentQuestionIndex > 0 ? `
                        <button class="quiz-button quiz-button--secondary" id="prev-btn">Previous</button>
                    ` : ''}
                    
                    ${state.isReviewMode 
                        ? (state.currentQuestionIndex < total - 1 
                            ? `<button class="quiz-button quiz-button--primary" id="next-btn">Next</button>`
                            : `<button class="quiz-button quiz-button--primary" id="results-btn">Back to Results</button>`)
                        : (state.currentQuestionIndex < total - 1
                            ? `<button class="quiz-button quiz-button--primary" id="next-btn">Next</button>`
                            : `<button class="quiz-button quiz-button--primary" id="submit-btn">Submit Exam</button>`)
                    }
                </div>
            </div>
        `;

        content.innerHTML = html;
        bindEvents(question);
    }

    function renderInput(q) {
        // Safe check for options shuffle
        let options = q.options ? [...q.options] : [];
        if (state.quizData.settings?.shuffleOptions && !state.isReviewMode) {
            options = shuffleArray(options);
        }

        const saved = state.userAnswers[q.id];

        switch (q.type) {
            case 'multiple-choice':
            case 'true-false':
                return options.map(opt => `
                    <label class="quiz-option ${saved === opt.id ? 'selected' : ''} ${state.isReviewMode ? getOptionClass(q, opt.id) : ''}">
                        <input type="radio" name="q-${q.id}" value="${opt.id}" ${saved === opt.id ? 'checked' : ''} ${state.isReviewMode ? 'disabled' : ''}>
                        <span class="option-text">${opt.text || (opt.id === 'true' ? 'True' : 'False')}</span>
                    </label>
                `).join('');

            case 'multiple-select':
                return options.map(opt => `
                    <label class="quiz-option ${(saved || []).includes(opt.id) ? 'selected' : ''} ${state.isReviewMode ? getOptionClass(q, opt.id) : ''}">
                        <input type="checkbox" name="q-${q.id}" value="${opt.id}" ${(saved || []).includes(opt.id) ? 'checked' : ''} ${state.isReviewMode ? 'disabled' : ''}>
                        <span class="option-text">${opt.text}</span>
                    </label>
                `).join('');

            case 'fill-blank':
                return `<div class="fill-blank-container">
                    ${q.blanks.map(b => {
                        const val = saved?.[b.id] || '';
                        const statusClass = state.isReviewMode ? (checkBlank(b, val) ? 'correct' : 'incorrect') : '';
                        return `<div style="margin-bottom:10px">
                            <label>Blank ${b.id}: </label>
                            <input type="text" class="blank-input ${statusClass}" 
                                   data-blank-id="${b.id}" 
                                   value="${val}" 
                                   ${state.isReviewMode ? 'readonly' : ''}
                                   placeholder="Type answer...">
                            ${state.isReviewMode && statusClass === 'incorrect' ? `<span style="color:var(--color-success); font-size:0.8em; margin-left:10px">Correct: ${b.acceptedAnswers[0]}</span>` : ''}
                        </div>`;
                    }).join('')}
                </div>`;

            case 'matching':
                if (state.isReviewMode) {
                    return `<div class="matching-review">
                        ${q.pairs.map(p => {
                            const userMatchId = saved?.[p.id];
                            const userMatchText = q.pairs.find(x => x.right === userMatchId || x.id === userMatchId)?.right || 'No Answer'; // Simplified logic
                            return `<div class="matching-item" style="display:flex; justify-content:space-between; margin-bottom:0.5rem">
                                <span>${p.left}</span>
                                <span>‚ûù</span>
                                <span style="font-weight:bold">${p.right}</span>
                            </div>`;
                        }).join('')}
                    </div>`;
                }
                // Simplified matching for standard view (Dropdowns)
                return `<div class="matching-container-simple">
                    ${q.pairs.map(p => `
                        <div class="matching-row" style="margin-bottom:1rem">
                            <span class="matching-left"><strong>${p.left}</strong></span>
                            <select class="matching-select" data-pair-id="${p.id}">
                                <option value="">Select match...</option>
                                ${q.pairs.map(opt => `<option value="${opt.id}" ${saved?.[p.id] === opt.id ? 'selected' : ''}>${opt.right}</option>`).join('')}
                            </select>
                        </div>
                    `).join('')}
                </div>`;

            default:
                return '<p>Unsupported question type</p>';
        }
    }

    function renderMedia(media) {
        if (!media) return '';
        const pathPrefix = document.location.pathname.includes('/topics/') ? '../' : '';
        const url = media.url?.startsWith('http') ? media.url : pathPrefix + (media.url || '');
        
        if (media.type === 'image') return `<img src="${url}" alt="${media.alt}" class="quiz-media-img" style="max-width:100%; border-radius:8px; margin-bottom:1rem">`;
        return '';
    }

    function renderFeedback(q) {
        const isCorrect = checkAnswer(q, state.userAnswers[q.id]);
        return `
            <div class="question-feedback ${isCorrect ? 'correct' : 'incorrect'}">
                <span class="feedback-icon">${isCorrect ? '‚úÖ' : '‚ùå'}</span>
                <div class="feedback-text">
                    <strong>${isCorrect ? 'Correct!' : 'Incorrect'}</strong>
                    <p>${q.explanation || ''}</p>
                </div>
            </div>
        `;
    }

    /**
     * Logic & Events
     */
    function bindEvents(q) {
        // Navigation
        document.getElementById('prev-btn')?.addEventListener('click', () => {
            saveCurrentAnswer(q);
            state.currentQuestionIndex--;
            renderQuestion();
        });

        document.getElementById('next-btn')?.addEventListener('click', () => {
            saveCurrentAnswer(q);
            state.currentQuestionIndex++;
            renderQuestion();
        });

        document.getElementById('submit-btn')?.addEventListener('click', () => {
            saveCurrentAnswer(q);
            calculateScore();
        });

        document.getElementById('results-btn')?.addEventListener('click', () => {
            renderResults();
        });

        // Input Listeners
        if (q.type === 'multiple-choice' || q.type === 'true-false') {
            document.querySelectorAll('input[type="radio"]').forEach(el => {
                el.addEventListener('change', (e) => {
                    document.querySelectorAll('.quiz-option').forEach(l => l.classList.remove('selected'));
                    e.target.closest('label').classList.add('selected');
                });
            });
        }
        
        if (q.type === 'multiple-select') {
            document.querySelectorAll('input[type="checkbox"]').forEach(el => {
                el.addEventListener('change', (e) => {
                    e.target.closest('label').classList.toggle('selected', e.target.checked);
                });
            });
        }
    }

    function saveCurrentAnswer(q) {
        if (state.isReviewMode) return;

        if (q.type === 'multiple-choice' || q.type === 'true-false') {
            const selected = document.querySelector(`input[name="q-${q.id}"]:checked`);
            if (selected) state.userAnswers[q.id] = selected.value;
        } 
        else if (q.type === 'multiple-select') {
            const selected = Array.from(document.querySelectorAll(`input[name="q-${q.id}"]:checked`)).map(el => el.value);
            state.userAnswers[q.id] = selected;
        }
        else if (q.type === 'fill-blank') {
            state.userAnswers[q.id] = {};
            document.querySelectorAll('.blank-input').forEach(input => {
                state.userAnswers[q.id][input.dataset.blankId] = input.value.trim();
            });
        }
        else if (q.type === 'matching') {
            state.userAnswers[q.id] = {};
            document.querySelectorAll('.matching-select').forEach(sel => {
                state.userAnswers[q.id][sel.dataset.pairId] = sel.value;
            });
        }
    }

    /**
     * Scoring Logic
     */
    function calculateScore() {
        let totalPoints = 0;
        let earnedPoints = 0;
        let topicScores = {};

        state.quizData.questions.forEach(q => {
            totalPoints += q.points;
            const isCorrect = checkAnswer(q, state.userAnswers[q.id]);
            
            let qPoints = 0;
            if (isCorrect === true) qPoints = q.points;
            else if (typeof isCorrect === 'number') qPoints = isCorrect; // Partial credit

            earnedPoints += qPoints;

            // Topic Breakdown for Comprehensive Exams
            if (q.topicId) {
                if (!topicScores[q.topicId]) {
                    topicScores[q.topicId] = { name: q.topicName, total: 0, earned: 0 };
                }
                topicScores[q.topicId].total += q.points;
                topicScores[q.topicId].earned += qPoints;
            }
        });

        const percentage = Math.round((earnedPoints / totalPoints) * 100);
        state.score = percentage;
        
        // Save to LocalStorage
        saveResultToStorage(state.quizData.quizId, percentage);

        renderResults(earnedPoints, totalPoints, percentage, topicScores);
    }

    function checkAnswer(q, userAns) {
        if (!userAns) return false;

        if (q.type === 'multiple-choice') {
            const correctOpt = q.options.find(o => o.isCorrect);
            return userAns === correctOpt.id;
        }
        
        if (q.type === 'true-false') {
            const val = userAns === 'true';
            return val === q.correctAnswer;
        }

        if (q.type === 'multiple-select') {
            const correctIds = q.options.filter(o => o.isCorrect).map(o => o.id);
            const userIds = userAns || [];
            
            // Exact match
            const isExact = correctIds.length === userIds.length && correctIds.every(id => userIds.includes(id));
            if (isExact) return true;

            // Partial credit (if enabled and no wrong answers selected)
            if (q.partialCredit) {
                const wrongSelections = userIds.filter(id => !correctIds.includes(id));
                if (wrongSelections.length > 0) return 0;
                
                const correctSelections = userIds.filter(id => correctIds.includes(id));
                return (correctSelections.length / correctIds.length) * q.points;
            }
            return false;
        }

        if (q.type === 'fill-blank') {
            let correctCount = 0;
            q.blanks.forEach(b => {
                if (checkBlank(b, userAns[b.id])) correctCount++;
            });
            if (correctCount === q.blanks.length) return true;
            return (correctCount / q.blanks.length) * q.points;
        }

        if (q.type === 'matching') {
            let correctCount = 0;
            q.pairs.forEach(p => {
                // In simplified dropdown, value matches pair ID (self-match logic for validation)
                // Actually, in the dropdown generation: value="${opt.id}". 
                // A correct match implies the user selected the option.id that corresponds to the pair.
                // Since p.right is just text, and options are q.pairs, the user selects the ID of the pair that contains the right answer.
                // In this simplified model, user selects the pair object that represents the right side.
                // So if pair.id is 'p1', user should select 'p1' from the dropdown.
                if (userAns[p.id] === p.id) correctCount++;
            });
            if (correctCount === q.pairs.length) return true;
            return (correctCount / q.pairs.length) * q.points;
        }

        return false;
    }

    function checkBlank(blank, val) {
        if (!val) return false;
        const input = blank.caseSensitive ? val : val.toLowerCase();
        return blank.acceptedAnswers.some(ans => {
            const target = blank.caseSensitive ? ans : ans.toLowerCase();
            return input === target;
        });
    }

    function getOptionClass(q, optId) {
        if (q.type === 'multiple-choice') {
            const isCorrect = q.options.find(o => o.id === optId).isCorrect;
            const isSelected = state.userAnswers[q.id] === optId;
            if (isCorrect) return 'correct';
            if (isSelected && !isCorrect) return 'incorrect';
        }
        if (q.type === 'multiple-select') {
            const isCorrect = q.options.find(o => o.id === optId).isCorrect;
            const isSelected = (state.userAnswers[q.id] || []).includes(optId);
            if (isCorrect) return 'correct-outline'; // Show it was a correct option
            if (isSelected && !isCorrect) return 'incorrect';
        }
        return '';
    }

    /**
     * Results Display
     */
    function renderResults(earned, total, percentage, topicScores) {

    function shuffleArray(array) {
        for (let i = array.length - 1; i > 0; i--) {
            const j = Math.floor(Math.random() * (i + 1));
            [array[i], array[j]] = [array[j], array[i]];
        }
        return array;
    }

            const label = card.querySelector('.progress-label');
            if (fill) fill.style.width = '100%';
            if (label) label.textContent = 'Completed';

            // Add Score Display if not present
            if (!card.querySelector('.quiz-card-score')) {
                const scoreDiv = document.createElement('div');
                scoreDiv.className = 'quiz-card-score';
                scoreDiv.innerHTML = `

                // Insert before actions

                card.insertBefore(scoreDiv, actions);
            }

            // Update button text
            const btn = card.querySelector('.quiz-card-btn--primary');
            if (btn) btn.textContent = 'Retake Quiz';

            totalScore += score;
            attempted++;
        }
    });

    // Update Header Stats
    document.getElementById('total-quizzes').textContent = cards.length;
    document.getElementById('completed-quizzes').textContent = attempted;
    const avg = attempted > 0 ? Math.round(totalScore / attempted) : 0;
    document.getElementById('average-score').textContent = `${avg}%`;

    // Filter Logic
    const diffFilter = document.getElementById('difficulty-filter');
    const statFilter = document.getElementById('status-filter');

    function filterGrid() {
        const dVal = diffFilter.value;
        const sVal = statFilter.value;

        cards.forEach(card => {
            const cDiff = card.dataset.difficulty;
            const cStat = card.dataset.status;

            const diffMatch = dVal === 'all' || cDiff === dVal;
            const statMatch = sVal === 'all' || cStat === sVal;

            card.style.display = (diffMatch && statMatch) ? 'block' : 'none';
        });
    }

    if (diffFilter) diffFilter.addEventListener('change', filterGrid);
    if (statFilter) statFilter.addEventListener('change', filterGrid);
