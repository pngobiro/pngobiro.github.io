<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="definitions">Definitions<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#definitions" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<ul>
<li><strong>Supervision</strong>: The process of directing and supporting staff so that they may effectively perform their duties. A management function that is planned and carried out to guide, support and assist staff in carrying out assigned tasks.</li>
<li><strong>Facilitative supervision</strong>: a process that promotes quality at all health system levels by strengthening relationships, focusing on identification and resolution of problems and helping optimize the allocation of resources.</li>
</ul>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="principle-of-facilitative-supervision">Principle of facilitative supervision<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#principle-of-facilitative-supervision" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<ul>
<li>Focus on the performance not the person</li>
<li>Time frame should be set</li>
<li>Need for supervision</li>
<li>Confidentiality</li>
<li>Objectivity and transparency</li>
<li>Focus on the future</li>
</ul>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="difference-between-traditional-and-facilitative-supervision">Difference between traditional and facilitative supervision<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#difference-between-traditional-and-facilitative-supervision" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="qualities-of-a-good-facilitative-supervisor">Qualities of a good facilitative supervisor<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#qualities-of-a-good-facilitative-supervisor" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>

<li>Good communication skills (listening and constructive feedback)</li>
<li>Desire to empower others and provide growth opportunities</li>
<li>Ability to work in teams</li>
<li>Expertise in specific technical knowledge</li>

<li>Empathy</li>
</ol>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="definition">Definition<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#definition" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p><strong>Mentoring</strong>: a process for the informal transmission of knowledge, social capital and the psychosocial support perceived by the recipient as relevant to work, career or professional development.</p>
<p>It entails informal communication, usually face to face during sustained period of time, between a person who is perceived to have less.</p>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="characteristics-of-mentorship">Characteristics of mentorship<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#characteristics-of-mentorship" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>

<li>Relationship may last for a specific period of time</li>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="methods-of-mentorship-for-tier-one-services">Methods of mentorship for tier one services<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#methods-of-mentorship-for-tier-one-services" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h2 level="2" id="1-one-on-one-mentoring">1. One on one mentoring<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#1-one-on-one-mentoring" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h2></div>
<p>This method matches one mentor with one mentee. It allows both of them to develop a personal relationship and provides individual support for the mentee.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h2 level="2" id="2-group-mentoring">2. <p>It requires that a mentor works with 4-6 mentees at one time. The group meets once or twice a month to discuss various topics.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="methods-of-mentorship-for-tier-one-services-cont">Methods of mentorship for tier one services cont..<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#methods-of-mentorship-for-tier-one-services-cont" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h2 level="2" id="3-resource-based-mentoring">3. Resource based mentoring<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#3-resource-based-mentoring" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h2></div>
<p>The mentee initiates the process by asking one of the volunteers for assistance. Mentors add their names to a list of available mentors from which a mentee can choose from. This model has limited support within the organization and may result in mismatched mentor -mentee pairing.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h2 level="2" id="4-training-based-mentoring">4. Training based mentoring<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#4-training-based-mentoring" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h2></div>
<p>This method is directly tied to a training program. A mentor is assigned to a mentee to help develop the specific skills taught in the program.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="methods-of-mentorship-for-tier-one-services-cont">Methods of mentorship for tier one services cont..<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#methods-of-mentorship-for-tier-one-services-cont" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>Training based mentoring is limited because it focuses on the subject at hand and doesn’t help the mentee develop a broader skill set.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h2 level="2" id="5-executive-mentoring">5. Executive mentoring<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#5-executive-mentoring" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h2></div>
<p>This top down model may be the most effective way to create a mentoring culture and cultivate skills and knowledge throughout an organization. It is also an effective succession planning tool because it prevents the knowledge brain drain that would otherwise take place when senior management retires.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="stages-of-mentoring">Stages of mentoring<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#stages-of-mentoring" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>Stage one: the mentor and mentee become acquainted and informally clarify their common interests, shared values, and goals given a high priority.</p>
<p>In this stage, there may be lack of communication or difficulty in communicating. Mentees may be reluctant to trust mentors. The relationship may be like this from one to six meetings.</p>
<p>Stage 2: At this stage, the mentor and mentee communicate initial expectations and agree upon some common procedures and expectations as a starting point.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="stages-of-mentoring-cont">Stages of mentoring cont..<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#stages-of-mentoring-cont" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>In the less likely event that the two individuals may not be compatible, the pair is able to part on a friendly basis. There is more listening, sharing and confiding in each other. Values will be compared and personal concerns will be expressed.</p>
<p><strong>Stage 3</strong>: The mentor and the mentee begin to accomplish the actual purpose of the mentoring.</p>
<p>Gradually needs become fulfilled, objectives are met and natural growth takes place. New challenges are presented and achieved.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="stages-of-mentoring-cont">Stages of mentoring cont..<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#stages-of-mentoring-cont" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>It is the stage of acceptance but is also a stage of change, where a mentee is more likely to exercise self-discipline.</p>
<p>Stage 4: The mentor and the mentee close their mentoring association and redefine their relationship.</p>
<p>Follow up is conducted.</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="benefits-of-mentoring">Benefits of mentoring<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#benefits-of-mentoring" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="definition">Definition<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#definition" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h2 level="2" id="coaching">Coaching<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#coaching" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h2></div>
<p>A training approach that seeks to achieve continuous improvement in performance. This occurs through motivation, modeling, practice, constructive feedback and gradual transfer of skills.</p>
<p>This is a training or development process via which an individual is supported while achieving a specific personal or professional competence result or goal. The individual receiving the coaching may be referred to as a coachee. Coaching is balanced, concrete and respectful</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="characteristics-of-coaching">Characteristics of coaching<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#characteristics-of-coaching" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<ol>
<li>It takes place within the confines of a formal working set up</li>
<li>Focuses on developing individuals within their current jobs</li>
<li>Interest is functional arising out of the need to ensure that individuals can perform the tasks required to the best of their abilities</li>
</ol>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="characteristics-of-coaching-cont">Characteristics of coaching cont...<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#characteristics-of-coaching-cont" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<ol start="4">
<li>Relationship tends to be initiated and driven by an individual manager</li>
<li>Relationship is finite ends as an individual transfers to another job</li>
<li>It allows staff to learn on the job and immediately apply what they are learning and see how it works</li>
</ol>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="steps-in-effective-coaching">Steps in effective coaching<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#steps-in-effective-coaching" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>Step 1. Start by motivating the staff to gain commitment to acquire the new behavior</p>
<p>Step 2. Demonstrate competently and explain the new behavior with an opportunity for the trainee to ask questions</p>
<p>Step 3. Give the trainee an opportunity to practice, apply and demonstrate their ability to perform under the supervisor</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="steps-in-effective-coaching-cont">Steps in effective coaching cont.<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#steps-in-effective-coaching-cont" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>Step 4. Provide constructive feedback by sharing the evaluation of the trainee in a concrete, respectful, two way interchange of ideas</p>
<p>Step 5. Allow the trainee transfer skills gradually, the trainee is provided an opportunity to undertake and demonstrate an increasing number of the sub skills involved in the new behavior after which the trainee becomes competent to carry out the new behavior without supervision.</p>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="definition-of-terms-concepts">Definition of terms &amp; concepts<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#definition-of-terms-concepts" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>A. Supervision: refers to an act by which any person inspects the work of other people, i.e., whether they are working properly or not</p>
<p>B. Also refers to the direct &amp; immediate guidance &amp; control of subordinates in the performance of their tasks</p>
<p>&gt; Its direction, guidance &amp; control of working force with a view to see:
&gt; (i) that they are working according to plan &amp;,
&gt; (ii) are observing &amp; adhering to time schedules.</p>
<p>B. Supervisors: those having authority to exercise independent judgment in hiring, discharging, disciplining, rewarding &amp; taking other actions of a similar nature with respect to employees'</p>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="traditional-supervision">TRADITIONAL SUPERVISION<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#traditional-supervision" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>→ A superficial/Shallow process that only looks at the face value of the problem &amp; not the root cause – no set standards.
→ It is often punitive, fault-finding does not offer solutions.
→ Focuses on individuals rather than processes
→ Does not endeavor to strengthen systems and processes that may cause disconnect between current outputs of health care to the expected standards</p>

→ It’s not continuous but it’s rather intermittent
→ usually creates resentment, suspicion, and disintegration between team members rather than focusing on strengthening team work for quality service delivery.</p>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="principles-of-supervision">Principles of supervision<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#principles-of-supervision" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>→ Right intensity: Lesser the weight of supervision higher the output. So a supervisor must be given optimum work.</p>
<p>→ Personal interest: supervisors should take personal interest in the welfare of the workers under them. Consultations: A good supervisor must consult people under them. They can afford opportunities for free and frank discussion of issues and problems.</p>
<p>→ Loyalty: they must exhibit their loyalty to the subordinates by safeguarding their interests. They should not offend the dignity of workers</p>

<p>→ Informal assessment: Informal appraisal of the performance of the subordinates and pointing out their shortcomings with a view to improve.</p>
<p>→ Broader interest: Supervisors are expected to take interest in management functions like planning, organizing, directing and controlling.</p>
<p>→ Work climate: They can create better work climate for successful achievements. Any strain in work climate will spoil the whole organization.</p>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="why-is-f-s-different-from-t-s">Why is F.S different from t.s<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#why-is-f-s-different-from-t-s" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>→ Focuses on helping staff solve problems through using quality-improvement tools – T.S Does not
→ Focuses on processes rather than on individuals - T.S Does not
→ Assists staff in planning for future quality-improvement goals - T.S Does not
→ Is continuous and builds on past gains while setting higher quality-improvement goals - T.S is not</p>

(Qualities)<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#a-good-facilitative-supervisor-qualities" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>→ Good listener

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="functions-of-supervision">FUNCTIONS OF SUPERVISION<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#functions-of-supervision" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>→ Educative (formative) — professional development of the supervisee

An approach?<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#a-theory-a-model-an-approach" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="6-major-counseling-theories-and-approaches">6 Major Counseling Theories and Approaches<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#6-major-counseling-theories-and-approaches" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p><img src="assets/images/image-20260113-5f5474d0.jpeg" alt="img-51.jpeg"></p>
<p><strong>Humanistic:</strong> Hold that people have within themselves all the resources they need to live healthy and functional lives, &amp; that problems occur as a result of restricted or unavailable problem-solving resources.</p>
<p><img src="assets/images/image-20260113-c7d2031c.jpeg" alt="img-52.jpeg"></p>
<p><strong>Cognitive:</strong> Hold that people experience psychological &amp; emotional difficulties when their thinking is out of sync with reality.</p>
<p><img src="assets/images/image-20260113-ea2be7aa.jpeg" alt="img-53.jpeg"></p>
<p><strong>Behavioral:</strong> Hold that people engage in problematic thinking and behavior when their environment supports it. When an environment reinforces or encourages these problems, they will continue to occur.</p>
<p><img src="assets/images/image-20260113-d168f8c9.jpeg" alt="img-54.jpeg"></p>
<p><strong>Psychoanalytic:</strong> Hold that psychological problems result from the present-day influence of unconscious psychological drives or motivations stemming from past rships &amp; experiences.</p>
<p><img src="assets/images/image-20260113-527da1df.jpeg" alt="img-55.jpeg"></p>
<p><strong>Constructionist:</strong> Hold that knowledge is merely an invented or “constructed” understanding of actual events in the world.</p>
<p><strong>Systemic:</strong> Hold that thinking, feeling &amp; behavior are largely shaped by pressures exerted on people by the social systems within which they live.</p>
<p>89</p>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="4-empathy">4. Empathy<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#4-empathy" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>“It means entering the private perceptual world of the other and becoming thoroughly at home in it. It involves being sensitive, moment by moment, to the changing felt meanings which flow in this other person, to the fear or rage or tenderness or confusion or whatever that he or she is experiencing. It means temporarily living in the other’s life, moving about in it delicately without making judgments.” (Rogers)</p>
<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="empathic-response">Empathic Response<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#empathic-response" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>

→ Use short responses, don’t over elaborate
→ Don’t mimic, remain yourself but allow yourself to feel what they might be feeling</p>

Probing and Encouraging<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#5-probing-and-encouraging" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>

<div data-testid="stHeadingWithActionElements" class="st-emotion-cache-1p9ibxm e1icttdg2"><h1 level="1" id="6-self-disclosure">6. Self Disclosure<span data-testid="stHeaderActionElements" class="st-emotion-cache-gi0tri e1icttdg3"><a href="#6-self-disclosure" class="st-emotion-cache-yinll1 e1icttdg1"><svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><path d="M15 7h3a5 5 0 0 1 5 5 5 5 0 0 1-5 5h-3m-6 0H6a5 5 0 0 1-5-5 5 5 0 0 1 5-5h3"></path><line x1="8" y1="12" x2="16" y2="12"></line></svg></a></span></h1></div>
<p>This is the act of revealing personal information about oneself to others. It can involve sharing thoughts, feelings, experiences, or personal details that are not commonly known or easily discernible by others.</p>
<p><strong>Questions to consider:</strong></p>
<p>i. When do I use self disclosure?
ii. How much is too much? Why does the client want to know?
iii. How might it benefit the client?
iv. How do you feel about giving the information?
v. Why do you feel you should or should not give the information?</p>
