Section 1: Multiple Choice Questions (20 Marks)

1.
Who among the given persons below classified diseases according to Age, Sex and period?
A. John Grant
B. Linnaeus
C. William Cullian
D. William Farr
Show Answer

Correct Answer: D. William Farr

Explanation: William Farr was a medical statistician who emphasized the classification of diseases for statistical purposes, including analysis by age, sex, and period.

B. 8th revision

11.
Which of the following conventions indicates that the code given must never be used alone?
A. Dagger
B. NEC
C. Parenthesis
D. Asterisk
Show Answer

Correct Answer: D. Asterisk

Explanation: An asterisk (*) code represents the manifestation of a disease and must always be accompanied by a dagger (†) code representing the underlying cause.

12.
Used in listing of inclusion and exclusion terms when the words that precede it are not complete terms for assignment to that rubric, which of the following conventions is best described by the statement above?
A. Colon
B. Exclusion term
C. Inclusion term
D. Brace
Show Answer

Correct Answer: A. Colon

Explanation: A colon (:) indicates that the term is incomplete and requires one of the modifiers listed below it to make it assignable to the category.

13.
Which is the most commonly used mortality rule used in filling in the medical certificate of cause of death
A. Rule 1
B. Rule 2
C. General principles
D. Rule 3
Show Answer

Correct Answer: C. General principles

Explanation: The General Principle states that when more than one condition is entered on the certificate, the condition entered alone on the lowest used line of Part I should be selected as the underlying cause of death, if it could have given rise to all the conditions reported above it.

17.
One of the following is NOT an example of speciality based adaptation as used in disease classification
A. Oncology
B. Dermatology
C. ICPM
D. Neurology
Show Answer

Correct Answer: C. ICPM

Explanation: Oncology (ICD-O), Dermatology (ICD-DA), and Neurology (ICD-NA) are specialty-based adaptations of ICD. ICPM is a classification of *procedures*, not a specialty adaptation of the disease classification itself.

18.

B. Classification

Correct Answer: A. Universe

Explanation: In classification theory, the "universe" refers to the entire set of items or entities (e.g., all diseases) that are being classified.

Section 4: Essay Questions (40 Marks)

41.
State two terms used in describing a classification giving examples of each (5 Marks)
Show Answer

Terms:

  1. Rubric (Category): A 3-character code (e.g., A00 Cholera).
  2. Sub-category: A 4-character code (e.g., A00.0 Cholera due to Vibrio cholerae 01, biovar cholerae).
  3. Block: A range of categories (e.g., A00-A09 Intestinal infectious diseases).
  4. Chapter: The highest level of grouping (e.g., Chapter I Certain infectious and parasitic diseases).

49.

A) Discuss five other situations other than the dagger & the asterisk system that permit two ICD codes to be used to describe fully a person's condition (10 Marks)

B) Explain all uses of the international classification of diseases (10 Marks)

Show Answer

A) Situations for Multiple Coding:

  1. Local infections: Use of B95-B97 to identify the infectious agent added to the site code.
  2. Neoplasms with functional activity: Code for the neoplasm + code for the functional activity (e.g., catecholamine production).
  3. Injury and External Cause: Code for the nature of injury (S/T code) + code for the external cause (V/W/X/Y code).
  4. Poisoning: Code for the substance (T code) + code for the intent/circumstance (External cause).
  5. Sequelae: Code for the late effect (condition) + code for the cause of the sequelae.
  6. Use of Z codes: For history of disease or status (e.g., organ transplant status) alongside current condition.

B) Uses of ICD:

  1. Statistical: To compile morbidity and mortality statistics for health planning and epidemiological studies.
  2. Administrative: For indexing medical records to facilitate retrieval.
  3. Reimbursement: For billing and payment systems (DRGs, insurance claims).
  4. Resource Allocation: To determine funding and resource distribution based on disease burden.
  5. Quality Assurance: To audit clinical care and outcomes.
  6. Research: To identify cohorts of patients with specific conditions for clinical trials.
