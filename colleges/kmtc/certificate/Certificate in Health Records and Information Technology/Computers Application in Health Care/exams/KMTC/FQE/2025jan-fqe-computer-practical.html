Section 1: Microsoft Word (35 Marks)

Note: The specific questions sum to 35 marks despite the instruction stating 30.

1.

Create a new Ms Word document and name it as tree planting 0000. Where the last four digits are your examination number. Encrypt your document with the password TTTT. Save this document in a folder on the desktop. Name the folder as February 2024. (2 marks)

Show Practical Steps

Step-by-Step Solution:

  1. Create Folder: Right-click on Desktop > New > Folder. Name it February 2024.
  2. Create Document: Open the folder, right-click inside > New > Microsoft Word Document. Name it tree planting 0000 (replace 0000 with your exam number).
  3. Encrypt Document:
    • Open the Word document.
    • Click File > Info.
    • Click Protect Document > Encrypt with Password.
    • Enter the password TTTT and click OK. Re-enter TTTT to confirm.
  4. Save: Click the Save icon (disk) or press Ctrl + S.

3.

Perform the following formatting tasks:

  1. The address to be aligned to the right margin (1 mark)
  2. Double underline the ref and bold it (2 marks)
  3. Justify align the body of the letter (1 mark)
  4. Highlight with colour red the word "budget allocation" (1 mark)
  5. Make the reference to be heading 1 style (1 mark)
  6. Insert a watermark with the words "high importance" (3 marks)
  7. Insert page number in the format of Page 1 of 1 center aligned (2 marks)
  8. Insert current date and time below the address set to update automatically (2 marks)
Show Practical Steps

Solutions:

  1. Align Address Right: Select "The Ministry of Health... Nairobi". Press Ctrl + R or click the "Align Right" button in the Paragraph group on the Home tab.
  2. Double Underline & Bold Ref:
    • Select "Ref. Tree planting exercise".
    • Press Ctrl + B for Bold.
    • Press Ctrl + Shift + D for Double Underline (or open Font dialog > Underline style > Double).
  3. Justify Body: Select the two main paragraphs. Press Ctrl + J or click "Justify" in the Paragraph group.
  4. Highlight Text: Select the words "budget allocation". On the Home tab, click the Text Highlight Color arrow and select Red.
  5. Heading 1 Style: Select "Ref. Tree planting exercise". On the Home tab, in the Styles group, click Heading 1.
  6. Watermark:
    • Go to Design (or Page Layout) tab.
    • Click Watermark > Custom Watermark.
    • Select "Text watermark".
    • Type "high importance" in the Text box. Click OK.
  7. Page Number:
    • Go to Insert tab > Page Number.
    • Select Bottom of Page.
    • Scroll to find the format "Page X of Y" (usually labeled "Bold Numbers 2" or similar).
    • Ensure it is centered.
  8. Date and Time:
    • Click below the address (Nairobi).
    • Go to Insert tab > Date & Time (Text group).
    • Select a format.
    • Check the box "Update automatically". Click OK.

6.

Copy column D and Paste it in column H. Use this new column to sort the data in the original table so that the numbers are sorted according to terminal digit filing system. Do not delete any data after sorting. (6 marks)

Show Explanation

Understanding Terminal Digit Filing:

Terminal digit filing usually sorts by the last 2 digits (primary), then middle 2 (secondary), then first 2 (tertiary). Since the question asks to sort using the column, it implies using Excel's sort function which sorts Left-to-Right by default (normal numerical/text sort) unless specific terminal digit logic is extracted.

Interpretation for Exam Context: Usually, this simply means "Sort Ascending" based on the File Number, OR extracting the last digits to sort.

Steps (Assuming simple sort based on column H instructions):

  1. Select Column D (D2:D21). Press Ctrl + C.
  2. Click Cell H2. Press Ctrl + V.
  3. Select the entire original table (A2:E21).
  4. Go to Data tab > Sort.
  5. In "Sort by", select File Number (Column D).
  6. Order: Smallest to Largest. Click OK.

Note: If strict "Terminal Digit" is required manually, you would create helper columns extracting the last 2 digits (=RIGHT(H2,2)) and sort by that helper column first.
