Section 1: Microsoft Word (40 Marks)

1.

Create a folder on the desktop with the name KMTC68. After 68, type last 2 digits of your college number e.g. KMTC6877. (2 marks)

Show Practical Steps
  1. Right-click on a blank area of the Desktop.
  2. Select New > Folder.
  3. Type the name KMTC68xx (replacing xx with your last two digits).
  4. Press Enter.
2.

Inside the above folder, create two other folders with the names MSWORD and MSEXCEL followed by your college number e.g. MSWORDC-HRI-98001-567. (2 marks)

Open Ms word and save as prevention in Msword folder you have created (note: instruction implies saving the document from Q3 here). (Marks included in task)

4.

In heading 1 above, make capital letters, bold, underline, colour red, and font 16 (5 marks)

Show Practical Steps
  1. Select the text "The concept of disease prevention and control".
  2. Capital Letters: Press Shift + F3 until UPPERCASE or use the Change Case button (Aa).
  3. Bold: Press Ctrl + B.
  4. Underline: Press Ctrl + U.
  5. Colour Red: Home Tab > Font Color > Red.
  6. Font 16: Home Tab > Font Size > 16.
  7. Apply Heading 1 Style (Optional but good practice): Right-click Heading 1 style > Update Heading 1 to Match Selection.

  3. Italics: Press Ctrl + I.
  4. Underline: Press Ctrl + U.
  5. Colour Blue: Home Tab > Font Color > Blue.
  6. Font 14: Home Tab > Font Size > 14.

Create a table of content above the sentence in heading 1 (7 marks)

4.

Using the IF function, assign the marks in the remark column as EXCELLENT, VERY GOOD, GOOD, FAIR AND POOR (8 marks)

KEY:

  • 75 and above is EXCELLENT
  • 65 to 74 is VERY GOOD
  • 50 to 64 is GOOD
  • 40 to 49 is FAIR
  • Below 40 is FAIL
Show Formula

Assuming Marks are in column B, starting B2.

Formula in C2:

=IF(B2>=75,"EXCELLENT",IF(B2>=65,"VERY GOOD",IF(B2>=50,"GOOD",IF(B2>=40,"FAIR","FAIL"))))

Drag the fill handle down to apply to all students.

In the chart, indicate the title, legend and values on the chart (6 marks)

Show Practical Steps
  1. Enter the data into Excel.
  2. Select the data range (including headers).
  3. Go to Insert tab > Charts group > Pie Chart (2D or 3D).

  6. Legend: Should appear automatically. If not, Chart Design > Add Chart Element > Legend.
