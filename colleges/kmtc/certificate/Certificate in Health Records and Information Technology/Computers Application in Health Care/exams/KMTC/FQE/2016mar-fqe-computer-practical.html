9.

Type the following in the created document (10 Marks)

Developing hospital mobile applications

Hospitals are looking to use mobile health applications to give their patients a more detailed look into their facilities

Functions of mobile health apps

Tracking and monitoring services: There are a number of health institutes that are offering mobile apps that track weight, cholesterol, pain levels, medications, glucose levels and other health metrics that can be utilized by healthcare providers.

Directory services: For most patients, it can be challenging to find their way around some of today's large hospital campuses. Visiting a specialist or a physician at a large hospital facility can lead to missed appointments, delays and frustration. Access to a hospital mobile application that locates a department, physician or building would be a great tool for patients and visitors.

Self-service: The ability to request medication refills, securely message their provider, book appointments, review lab results and pay their balance gives patients incentive to access their information electronically. Not only do these functions improve patient satisfaction scores, but they also present tremendous cost savings for hospitals, as they reduce incoming call volume and the need for 24/7 call centers.

10.

Select the heading "Developing hospital mobile applications" and format it as follows (3 Marks)

  1. Font Color: Red
  2. Style: Bold
  3. Align: Centre
  4. Double underline
  5. Change the text to upper case
  6. Superscript the last word
Show Practical Steps
  1. Select the heading text.
  2. Color: Home tab > Font Color (A with red bar) > Select Red.
  3. Bold: Press Ctrl + B.
  4. Centre: Press Ctrl + E.
  5. Double Underline: Press Ctrl + Shift + D.
  6. Uppercase: Shift + F3 (toggle until UPPERCASE) OR Home > Change Case (Aa) > UPPERCASE.
  7. Superscript: Select the word "APPLICATIONS". Home tab > Click Superscript (X²) icon.

Format the entire document as follows (3 Marks)

  1. Line spacing: 1.5

Show Practical Steps

Mail Merge Process:

  1. Create Main Document: Open Word. Type the letter content.
    • Align the sender details placeholders (NAMES, ADDRESS...) to the right.
    • Type the body text.
  2. Start Mail Merge:
    • Mailings tab > Start Mail Merge > Letters.
  3. Select Recipients:
    • Click Select Recipients > Use an Existing List.
    • Browse to the Excel file created in the previous step. Select Sheet 2.
  4. Insert Merge Fields:
    • Highlight "NAMES" at top right > Insert Merge Field > Select "NAMES".
    • Repeat for ADDRESS, COUNTY, TEL NUMBER.
    • Next to "Dear", insert the "NAMES" field.
  5. Save:
    • File > Save As.
    • Navigate to the folder BRAIN (renamed from WARD in Section 1).
    • Filename: MERGE.
  6. Finish (Optional based on instructions): The instruction says "prepare letters to send to all". Usually, this means saving the main document linked to data, or completing the merge to a new document. Saving as "MERGE" usually implies the main document.
