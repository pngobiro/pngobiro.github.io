Section 1: Multiple Choice Questions (20 Marks)

1.
The term Computer is derived from ___________?
A. Latin
B. German
C. French
D. Arabic
Show Answer

Correct Answer: A. Latin

Explanation: The word "computer" is derived from the Latin word "computare," which means to calculate or to count.

2.
Which is a device that changes information into digital form?
A. Modem
B. Digitizer
C. Mouse
D. Light pen
Show Answer

Correct Answer: B. Digitizer

Explanation: A digitizer allows users to convert analog information (like a drawing or image) into digital coordinates that a computer can process. A modem converts digital signals to analog and vice versa for transmission.

3.
VGA stands for?
A. Vide Graphics Array
B. Visual Graphics Array
C. Volatile Graphics Array
D. Video Graphics Array
Show Answer

Correct Answer: D. Video Graphics Array

Explanation: VGA is a standard for defining color display screens for computers.

4.
An error in computer data is called?
A. Chip
B. Storage device
C. CPU
D. Bug
Show Answer

Correct Answer: D. Bug

Explanation: In computer terminology, a "bug" refers to an error, flaw, or fault in a computer program or system that causes it to produce an incorrect or unexpected result.

5.
Computer software can be defined as:
A. The instruction that tell the computer what to do
B. The computer and its associated equipment
C. An interface between the computer and the network
D. Computer components that act to accomplish a goal
Show Answer

Correct Answer: A. The instruction that tell the computer what to do

Explanation: Software is a collection of instructions and data that tell a computer how to work, in contrast to hardware, which is the physical system.

6.
__________ are attempts by individuals to obtain confidential information from you by falsifying their identity:
A. Computer virus
B. Phishing scams
C. Spyware scams
D. Phishing trips
Show Answer

Correct Answer: B. Phishing scams

Explanation: Phishing is a type of social engineering where an attacker sends a fraudulent message designed to trick a human victim into revealing sensitive information.

7.
Which of the following is an example of storage devices?
A. Magnetic disks
B. Tapes
C. DVDs
D. All of the above
Show Answer

Correct Answer: D. All of the above

Explanation: Magnetic disks (hard drives), Tapes (magnetic tape), and DVDs (optical storage) are all mediums used to store data.

8.
1024 bytes equals:
A. 1KB
B. 2MB
C. 1GB
D. 1TB
Show Answer

Correct Answer: A. 1KB

Explanation: 1 Kilobyte (KB) = 1024 bytes.

9.
From which menu you can insert Header and Footer?
A. Insert menu
B. View menu
C. Format menu
D. Tools menu
Show Answer

Correct Answer: A. Insert menu

Explanation: In modern MS Word interfaces (Ribbon), Header and Footer options are located under the Insert tab.

10.
Thesaurus tools in MS word is used for
A. Spelling suggestions
B. Grammar options
C. Synonyms and Antonyms words
D. All of the above
Show Answer

Correct Answer: C. Synonyms and Antonyms words

Explanation: The Thesaurus (Shift+F7) allows you to look up synonyms (words with the same meaning) and antonyms (words with opposite meaning).

11.
Why are Drop Caps used in documents?
A. To drop all the capital letters
B. To automatically begin each paragraph with capital letters
C. To begin a paragraph with a large dropped initial capital letter
D. None of the above
Show Answer

Correct Answer: C. To begin a paragraph with a large dropped initial capital letter

Explanation: A Drop Cap is a large capital letter at the beginning of a text block that has the depth of two or more lines of regular text.

12.
Borders can be applied to:
A. Cells
B. Paragraph
C. Text
D. All of the above
Show Answer

Correct Answer: D. All of the above

Explanation: In MS Word, you can apply borders to selected text, entire paragraphs, table cells, or even the whole page.

13.
To drag a selected range of data to another worksheet in the same workbook, use the
A. Tab key
B. Alt Key
C. Shift Key
D. Ctrl Key
Show Answer

Correct Answer: B. Alt Key

Explanation: In Excel, if you click and drag a selection, it moves within the sheet. To move it to another sheet, you hold the Alt key while dragging the selection over the sheet tab, which switches the view to that sheet.

14.
What is entered by the function = today ()
A. The date value for the day accordingly to system clock
B. The time value according to the system clock
C. Today's date as Text format
D. All of the above
Show Answer

Correct Answer: A. The date value for the day accordingly to system clock

Explanation: The =TODAY() function returns the current date serial number based on the system clock. It does not include the time (which =NOW() does).

15.
Which of the following is an absolute cell reference?
A. !A !1
B. $A $1
C. #a #1
D. A1
Show Answer

Correct Answer: B. $A $1

Explanation: In Excel, the dollar sign ($) is used to fix the column and/or row reference so it doesn't change when the formula is copied. $A$1 is an absolute reference.

16.
Which symbol must all formula begin with?
A. =
B. +
C. (
D. @
Show Answer

Correct Answer: A. =

Explanation: All formulas in Excel must start with an equals sign (=) to indicate that the cell contains a calculation.

17.
Which of the following is not entered correctly?
A. = 10+50
B. = B7*B1
C. =B7+14
D. 10+50
Show Answer

Correct Answer: D. 10+50

Explanation: Without the leading equals sign (=), Excel treats "10+50" as text or just a value, not a calculation to be performed.

18.
A function inside another function is called a _______ function
A. Nested
B. Round
C. Sum
D. Text
Show Answer

Correct Answer: A. Nested

Explanation: When you use a function as an argument within another function (e.g., =IF(A1>10, SUM(B1:B5), 0)), it is called nesting.

19.
Special effects used to introduce slides in a presentation are called?
A. Effects
B. Custom animation
C. Transitions
D. Present animations
Show Answer

Correct Answer: C. Transitions

Explanation: Transitions are the animation effects that occur when you move from one slide to the next during a presentation. Animations apply to individual elements within a slide.

20.
To give a power point presentation to a large audience, you:
A. Set up your computer in a large auditorium, and click large, auditorium, OK
B. Click the slide you wish the audience to see in normal view, then click the next slide, and so on
C. Choose either view, slide show or slide show, view show
D. Click slide show, ok
Show Answer

Correct Answer: C. Choose either view, slide show or slide show, view show

Explanation: To present, you must enter Slide Show mode (View > Slide Show) which displays the presentation full screen.
