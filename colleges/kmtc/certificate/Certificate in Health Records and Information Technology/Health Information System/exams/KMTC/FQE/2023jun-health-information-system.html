Section 1: Multiple Choice Questions (40 Marks)

1.
Characteristics of an indicator which mean it can be quantified repeatedly is?
A. Validity
B. Reliability
C. Feasibility
D. Significant
Show Answer

Correct Answer: B. Reliability

Explanation: Reliability refers to the consistency of a measure. If an indicator produces the same results when measured repeatedly under the same conditions, it is considered reliable.

2.
Which process do HRIOs use to gather raw data and express it in summary form?
A. Data analysis
B. Data disposal
C. Data aggregation
D. Data collection
Show Answer

Correct Answer: C. Data aggregation

Explanation: Data aggregation is the process of combining individual data points (raw data) into summary totals (e.g., total monthly admissions) for reporting and analysis.

3.
The MOH tool bearing the code number 505 is
A. Master facility list
B. Service delivery report
C. Nutritional monthly report
D. IDSR weekly epidemic monitoring
Show Answer

Correct Answer: B. Service delivery report

Explanation: MOH 505 is the Service Delivery Summary Report used in Kenya's health information system.

4.
Product of data processing is
A. Data
B. Information
C. Software program
D. System
Show Answer

Correct Answer: B. Information

Explanation: Data refers to raw facts. When processed and organized to have meaning, it becomes Information.

5.
Input indicator can also be?
A. Resources
B. Process
C. Output
D. Impact
Show Answer

Correct Answer: A. Resources

Explanation: Input indicators measure the resources (human, financial, material) needed to carry out activities.

6.
The type of data collected through focus group discussion is known as?
A. Primary data
B. Secondary data
C. Quantitative data
D. Qualitative data
Show Answer

Correct Answer: D. Qualitative data

Explanation: Focus groups generate non-numerical, descriptive data (opinions, feelings, experiences), which is qualitative.

7.
Basic correcting mistakes and errors observed during data processing to ensure logical entries on reporting tools is known as
A. Auditing
B. Data editing
C. Data cleaning
D. Manipulations
Show Answer

Correct Answer: C. Data cleaning

Explanation: Data cleaning (or cleansing) involves detecting and correcting (or removing) corrupt or inaccurate records from a record set, table, or database.

8.
Health centres are classified under which tier of care in the current health care organization
A. Tier 2
B. Tier 5
C. Tier 3
D. Tier 4
Show Answer

Correct Answer: A. Tier 2 (or C depending on curriculum context)

Explanation: Under the Kenya Health Sector Referral Strategy: Tier 1 = Community; Tier 2 = Primary Care (Dispensaries & Health Centres); Tier 3 = County Hospitals. Sometimes older classifications placed Health Centres at Tier 3. However, currently, they are generally grouped in Tier 2.

Section 2: Short Answer Questions (40 Marks)

41.

Define the following terms (5 marks)

  • a) Data
  • b) Health information
  • c) Health system
  • d) Health management information system
  • e) Information system
Show Answer
  • Data: Raw, unorganized facts and figures.
  • Health Information: Processed health data that provides meaning and context.
  • Health System: The sum total of all organizations, institutions, and resources whose primary purpose is to improve health.
  • HMIS: A system used to collect, analyze, and disseminate health data for decision-making.
  • Information System: An integrated set of components for collecting, storing, and processing data.
42.

  3. Antenatal care (ANC)
  4. Family Planning
  5. HIV testing and counseling (HTC)

Show Answer

  3. Staffing and workload planning.
  4. Budgeting and resource allocation.
  5. Performance monitoring.

Section 3: Long Answer Questions (20 Marks)

49.
The process of data collection has been facing a lot of challenges in Kenya. Discuss ten (10) challenges and provide measures to overcome them. (20 marks)
Show Answer

Challenges:

  1. Incomplete data records.
  2. Lack of standardized tools.
  3. Poor internet connectivity.
  4. Inadequate trained staff.
  5. Lack of management support.
  6. Timeliness issues (late reporting).
  7. Data entry errors.
  8. Resistance to new technology (EMR).
  9. Insufficient funding for HIS.
  10. Poor feedback mechanisms.

Measures:

  1. Regular staff training.
  2. Providing standard tools.
  3. Improving infrastructure (internet/power).
  4. Hiring more HRIOs.
  5. Sensitizing management on data use.
  6. Automating data validation.
  7. Allocating budget for HIS.
  8. Implementing data quality audits.
