Section A

Question 1

State two differences between software and hardware.

Answer

Two fundamental differences between software and hardware are:

  1. Physical Nature:
    • Hardware:
      • Physical, tangible components
      • Can be physically touched and seen
      • Occupies physical space
      • Examples: CPU, keyboard, monitor
    • Software:
      • Intangible, logical components
      • Cannot be physically touched
      • Exists as programs and data
      • Examples: Operating systems, applications
  2. Function:
    • Hardware:
      • Performs physical operations
      • Provides computing resources
      • Processes data and executes instructions
      • Handles input/output operations
    • Software:
      • Controls hardware operations
      • Provides instructions to hardware
      • Manages data processing
      • Enables user interaction

Question 3

(a) Define the term Hard Disk as used in computers.

(b) Outline three factors to be considered when selecting a hard disk for purchase.

Answer

(a) Hard Disk Definition:

A hard disk is a non-volatile storage device that:

  • Uses magnetic storage technology for data storage
  • Contains rotating platters coated with magnetic material
  • Provides permanent data storage even when power is off
  • Functions as primary storage for operating system and user data

(b) Factors to consider when selecting a hard disk:

  1. Storage Capacity:
    • Available space (GB/TB)
    • Current storage requirements
    • Future storage needs
    • Operating system requirements
  2. Performance Specifications:
    • Rotational speed (RPM):
      • 5400 RPM - Basic use
      • 7200 RPM - Standard
      • 10000+ RPM - High performance
    • Data transfer rates
    • Cache size
    • Access time
  3. Interface Type and Compatibility:
    • SATA interface versions
    • PCIe/NVMe support
    • Motherboard compatibility
    • Power requirements

Question 4

State the component within the CPU that handles each of the following tasks:

(a) Addition and Subtraction;

(b) Temporarily stores the answers from arithmetic and logical operations before processing;

(c) Coordinates all activities in the processor;

(d) Carries out logical operations.

Answer

  1. Addition and Subtraction:

    Component: Arithmetic Logic Unit (ALU)

    • Performs all arithmetic calculations
    • Processes binary numbers
    • Part of the processor's execution unit
  2. Temporary Storage:

    Component: Registers

    • High-speed storage locations in CPU
    • Hold intermediate results
    • Provide quick access to data
    • Various types including accumulator and data registers
  3. Activity Coordination:

    Component: Control Unit

    • Manages instruction execution sequence
    • Controls data flow between components
    • Generates timing and control signals
    • Supervises overall CPU operation
  4. Logical Operations:

    Component: Arithmetic Logic Unit (ALU)

    • Performs Boolean operations (AND, OR, NOT)
    • Handles comparisons
    • Executes bit manipulations
    • Same component as in part (a) but different function

Question 6

A technician in a computer laboratory noted that three computers were not functioning and needed to open them to confirm the source of the problem.

(a) Outline two precautions that the technician must observe when opening the computer.

(b) Explain one way in which he can ensure safety of the data in the affected computers.

Answer

(a) Precautions when opening a computer:

  1. Anti-static Protection:
    • Use anti-static wrist strap:
      • Connect to computer chassis ground point
      • Ensure proper skin contact
      • Verify strap functionality
    • Work on anti-static mat:
      • Place components on grounded mat
      • Keep workspace clear of static-generating materials
      • Maintain proper grounding connections
  2. Power Safety:
    • Ensure complete power disconnection:
      • Properly shut down the system
      • Unplug from power source
      • Wait for capacitors to discharge (5-10 minutes)
    • Verify power isolation:
      • Check all power indicators are off
      • Remove any connected peripherals
      • Keep work area dry and clean

(b) Ensuring data safety:

Create comprehensive data backup before maintenance:

  • System backup:
    • Create complete system image
    • Include operating system files
    • Back up system settings
    • Store configuration information
  • Data protection measures:
    • Copy all user data to external storage
    • Verify backup integrity
    • Store backup in secure location
    • Document backup procedures

An antistatic wrist strap:

A NIC serves as the interface between a computer and the network by:

  • Network identification:

    • Facilitates network addressing

    • Easier system administration

Question 12

(a) Hekima College has opted to set up a computer network based on star topology technology.

(i) Outline two benefits that the college would achieve from the use of the topology.

(ii) With the aid of a diagram, describe the star topology.

(b) Explain how the college would use each of the following network transmission media:

(i) Unshielded twisted pair cable (UTP);

(ii) Fibre Optic Cable.

(c) Explain the purpose of a terminator in a bus topology network.

Answer

(a)(i) Benefits of star topology:

  • Fault Tolerance:
    • Individual node failures don't affect other nodes
    • Easy to identify and isolate problems
    • Network remains operational if one connection fails
  • Easy Management:
    • Centralized administration
    • Simple to add or remove devices
    • Better network monitoring

(a)(ii) Star Topology:

        PC1          PC2
         |            |
         |            |
PC3 ----[HUB/SWITCH]---- PC4
         |            |
         |            |
        PC5          PC6

Star Network Topology

Description:

  • All devices connect to central hub/switch
  • Point-to-point connections
  • Data passes through central device
  • Each device has dedicated connection

(b) Network transmission media usage:

(i) Unshielded Twisted Pair (UTP) Cable:

  • Internal Network Connections:
    • Connecting computers within rooms/floors
    • Distances up to 100 meters
    • Connecting to network printers
    • Linking workstations to wall outlets

(ii) Fiber Optic Cable:

  • Backbone Connections:
    • Between buildings or floors
    • High-speed data transmission
    • Long-distance connections
    • Areas with electromagnetic interference

(c) Purpose of terminator in bus topology:

  • Signal Management:
    • Absorbs signals at cable ends
    • Prevents signal reflection
    • Stops signal bouncing
  • Network Integrity:
    • Maintains signal quality
    • Prevents data corruption
    • Ensures proper network operation

Data Hierarchy Levels

Explanation of levels:

  • Bit: Smallest unit (0 or 1)
  • Field: Single piece of information (e.g., name, age)
