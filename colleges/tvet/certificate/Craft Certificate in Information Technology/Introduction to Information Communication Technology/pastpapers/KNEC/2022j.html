SECTION A (40 marks)

Question 1

Outline four uses of computers in oil refinery industry.

Answer

  • Process control and monitoring of refinery operations
  • Quality control and testing of petroleum products
  • Inventory management of raw materials and finished products
  • Safety monitoring and emergency response systems

Question 2

Describe two limitations of batch processing system.

Answer

  • Delayed processing: Results are not available immediately as data must be collected into batches before processing
  • No immediate error correction: Errors are only discovered after the entire batch is processed, requiring reprocessing of the entire batch

Question 3

A computer technician installed antivirus software to a computer. Explain two functions of this software.

Answer

  • Real-time scanning and protection against malware, viruses, and other threats
  • Regular system scanning to detect and remove existing viruses and malicious software

Question 4

Differentiate between shielded and unshielded twisted pair cable.

Answer

Shielded Twisted Pair (STP):

  • Has an extra layer of metal shielding around the wires
  • Better protection against electromagnetic interference
  • More expensive and less flexible

Unshielded Twisted Pair (UTP):

  • No additional shielding beyond the basic insulation
  • More susceptible to interference
  • Less expensive and more flexible

Question 5

Outline four advantages of sequential file access method.

Answer

  • Simple and easy to implement
  • Efficient for batch processing
  • Good for backup and archival purposes
  • Uses storage space efficiently

Question 6

Alice created a computer file during data processing. Describe two types of files that she could have created.

Answer

1. Data Files:

  • Contains raw data or information
  • Examples: text files, spreadsheets, databases

2. Program Files:

  • Contains executable code or instructions
  • Examples: .exe files, scripts, source code files

Question 7

With the aid of a diagram, describe a star computer network topology.

Answer

            PC1     PC2
             \     /
              \   /
        PC3----HUB----PC4
              /   \
             /     \
           PC5     PC6

Star Topology Diagram

In a star topology:

  • All devices connect to a central hub/switch
  • Each device has its own dedicated connection to the hub
  • If one connection fails, other devices are not affected
  • Easy to add or remove devices
  • Central point of failure at the hub

Question 8

Outline four benefits of using a Magnetic Ink Character Reader (MICR) as a computer input device.

Answer

  • High accuracy in reading magnetic characters
  • Fast processing of large volumes of documents
  • Enhanced security due to unique magnetic ink
  • Widely used in banking for check processing

Question 9

Describe two characteristics of a computer hard disk.

Answer

  • Non-volatile storage (retains data when power is off)
  • Large storage capacity compared to other storage devices

Question 10 (4 marks)

Hope insurance Company has acquired a super computer for its operations. Outline four uses of this computer to the company.

Answer

  • Complex risk analysis and actuarial calculations
  • Large-scale data mining and predictive modeling
  • Processing massive amounts of claims simultaneously
  • Real-time fraud detection analysis

Question 11

(a) (i) Outline three factors that could affect the clarity of voice output from a computer.

(ii) The following are computer input devices:

Keyboard, Mouse, Touch screen, Track ball.

Classify each as either pointing or keying.

(b) Differentiate between software and liveware as used in computers.

(c) Protech Company has initiated the use of a professional code of conduct. Explain three benefits of using this document to the organisation.

Answer

(a) (i) Factors affecting voice output clarity:

  • Quality of the sound card
  • Speaker quality and configuration
  • Environmental noise and acoustics

(ii) Classification of input devices:

Pointing devices:

  • Mouse
  • Touch screen
  • Track ball

Keying devices:

  • Keyboard

(b) Software vs Liveware:

Software:

  • Programs and instructions that run on the computer
  • Non-physical component of computer systems

Liveware:

  • Human component of computer systems
  • Users, operators, and maintenance personnel

(c) Benefits of professional code of conduct:

  • Establishes clear ethical guidelines for employees
  • Helps maintain professional standards and consistency
  • Protects company reputation and builds trust

Question 13

(a) Outline four characteristics of \(2^{\text{nd}}\) generation computers.

(b) Helen intends to buy computer hardware. Explain two factors she could have put into consideration when selecting the hardware.

(c) (i) A law firm intends to computerise its operations. Outline three areas the firm is likely to computerise.

(ii) Differentiate between compiler and interpreter utility programs.

Answer

(a) Characteristics of 2nd generation computers:

  • Used transistors instead of vacuum tubes
  • Smaller size compared to 1st generation
  • More reliable and generated less heat
  • Used assembly language for programming

(b) Hardware selection factors:

  • Compatibility with existing systems and software
  • Cost and budget constraints

(c) (i) Areas for computerization in law firm:

  • Case management and document storage
  • Client billing and accounting
  • Legal research and document preparation

(ii) Compiler vs Interpreter:

Compiler:

  • Translates entire program at once
  • Creates executable file
  • Faster execution

Interpreter:

  • Translates and executes line by line
  • No separate executable file
  • Easier debugging

Question 15

(a) Describe mechanical data processing method as used in computing.

(b) Riztech Company intends to secure its data. Outline three methods the company could use to secure these data.

(c) Differentiate between a CD and DVD computer storage devices.

(d) A company has installed a ring computer network topology. Explain three challenges the company is likely to encounter from using this network.

Answer

(a) Mechanical data processing:

  • Uses mechanical devices to process data
  • Examples include punch cards and mechanical calculators
  • Limited processing capabilities compared to electronic methods

(b) Data security methods:

  • Encryption of sensitive data
  • Regular data backup and recovery procedures
  • Access control and authentication systems

(c) CD vs DVD:

CD (Compact Disc):

  • Storage capacity: up to 700 MB
  • Single layer storage
  • Lower data transfer rate

DVD (Digital Versatile Disc):

  • Storage capacity: 4.7 GB to 17 GB
  • Multiple layer storage possible
  • Higher data transfer rate

(d) Challenges of ring topology:

  • Single point of failure can affect entire network
  • Difficult to add or remove devices
  • Data must pass through each node, increasing latency
