SECTION A (40 marks)

Answer ALL the questions in this section.

Question 1

Describe each of the following types of computer networks:

(i) Personal Area Network (PAN);

(ii) Local Area Network (LAN).

Answer

(i) Personal Area Network (PAN):

  • Smallest network covering personal workspace
  • Typically spans a few meters
  • Connects personal devices like phones, laptops, tablets

(ii) Local Area Network (LAN):

  • Network confined to a limited geographical area
  • Typically within a building or campus
  • Connects computers and devices within an organization

Question 2

Explain each of the following terms as used in computer ethics:

(a) software piracy;

(b) plagiarism.

Answer

(a) Software piracy:

  • Unauthorized copying, distribution, or use of software
  • Violation of software licensing agreements

(b) Plagiarism:

  • Using someone else's work or ideas without proper attribution
  • Presenting others' content as your own

Question 3

Distinguish between backup diskette and backup system as used in computer systems.

Answer

Backup diskette:

  • Physical storage medium for backing up data
  • Limited storage capacity

Backup system:

  • Complete solution for data backup
  • Includes hardware, software, and procedures
  • Can handle multiple backup methods and media

Question 4

(a) Arrange the following in descending order of data hierarchy: record, byte, field, bit, database, file.

(b) It is the company's policy that all computer users should take a five-minute break after working for two to three hours. Explain the reason for this policy.

Answer

(a) Data hierarchy (descending order):

  1. Database
  2. File
  3. Record
  4. Field
  5. Byte
  6. Bit

(b) Break policy reason:

  • To prevent eye strain and repetitive stress injuries
  • To reduce fatigue and maintain productivity

Question 5

The employees of a certain company confirmed that the software that had been installed was user friendly. Explain two reasons that could have led to their confirmation.

Answer

  • Intuitive interface with clear menus and icons
  • Easy to learn and use without extensive training

Question 6

Outline the function of each of the following devices as used in computer networking:

(a) switch;

(b) router.

Answer

(a) Switch:

  • Connects devices within a local network
  • Forwards data packets to specific destination devices

(b) Router:

  • Connects different networks together
  • Determines best path for data packets between networks

Question 7

Describe each of the following data processing modes:

(a) distributed;

(b) multi-programming.

Answer

(a) Distributed processing:

  • Processing spread across multiple computers
  • Tasks shared between networked systems

(b) Multi-programming:

  • Multiple programs running simultaneously
  • CPU switches between programs to maximize efficiency

Question 8

Explain each of the following terms as used in computer files:

(a) logical files;

(b) physical files.

Answer

(a) Logical files:

  • User's view of data organization
  • Abstract representation of data

(b) Physical files:

  • Actual storage of data on disk
  • How data is physically organized on storage media

Question 9

A certain county government has proposed to set up digital villages. Explain two positive impacts this move would have to the people who reside in the villages.

Answer

  • Access to online services and information
  • Digital literacy and skills development opportunities

Question 10

Computer manufacturers have invented new computers that are powered by solar energy. Outline two benefits of this invention.

Answer

  • Environmentally friendly and reduced carbon footprint
  • Cost savings on electricity bills

SECTION B (60 marks)

Answer any FOUR questions from this section.

Question 11

(a) Peru school intends to set up a computer laboratory to be equipped with new computers. Outline three environmental threats that the school should consider.

(b) Explain each of the following terms as used in computer software, giving an example in each case:

(i) operating system;

(ii) system utility.

(c) Mike has been tasked to acquire a printer for the company he works for. Explain three factors that he should consider, other than cost.

Answer

(a) Environmental threats:

  • Dust and dirt contamination
  • Temperature and humidity variations
  • Power fluctuations and surges

(b) Software terms:

(i) Operating system:

  • Basic software that manages hardware and software resources
  • Example: Windows 10, macOS

(ii) System utility:

  • Programs that help maintain and optimize computer system
  • Example: Disk Defragmenter, Antivirus

(c) Printer selection factors:

  • Print quality and resolution
  • Print speed and volume capacity
  • Network connectivity options

Question 12

(a) (i) Explain the terms wireless fidelity as used in computer systems.

(ii) Outline three advantages of using a wireless network to the organization.

(b) Explain each of the following terms as used in computer security:

(i) data protection Act;

(ii) computer misuse Act.

(c) Figure 1 shows the functional structure of a computer system. Use it to answer the question that follows.

Explain the function of each of the parts labelled (i), (ii) and (iii).

Answer

(a) (i) Wireless Fidelity (Wi-Fi):

Technology that allows devices to connect to a network wirelessly using radio waves.

(ii) Advantages of wireless network:

  • Flexibility in device placement
  • Easy expansion of network
  • Reduced cable infrastructure costs

(b) Computer security terms:

(i) Data Protection Act:

  • Legislation governing how personal data is collected and used
  • Ensures privacy and security of personal information

(ii) Computer Misuse Act:

  • Law addressing unauthorized access and modification of computer material
  • Criminalizes hacking and other computer crimes

(c) Computer system components:

  • (i) Input Unit: Accepts data and instructions from users
  • (ii) Processing Unit: Performs computations and controls operations
  • (iii) Output Unit: Delivers processed results to users

Question 13

(a) Differentiate between freeware and shareware as used in software ownership.

(b) Mukaro Company Ltd management experienced a lot of resistance from the employees while trying to automate its systems. Explain three causes of this resistance.

(c) Jacob intends to install a Local Area Network for the organization he works for using a suitable topology.

(i) Explain the term topology.

(ii) Outline three factors other than cost, he should consider before choosing the type of topology to use.

Answer

(a) Software types:

Freeware:

  • Software available for free use
  • No payment required for use

Shareware:

  • Software available for trial before purchase
  • Payment required after trial period

(b) Causes of resistance:

  • Fear of job loss or changes in job roles
  • Lack of training and technical skills
  • Comfort with existing systems

(c) Network topology:

(i) Definition:

  • Physical and logical arrangement of network devices and connections

(ii) Factors to consider:

  • Network size and scalability requirements
  • Reliability and fault tolerance needs
  • Ease of maintenance and troubleshooting

Question 14

(a) Explain two ways in which a hotel could make use of ICT in its sales and marketing department.

(b) Sammy has been tasked to acquire document readers to be installed in an upcoming supermarket's point of sale.

(i) Outline two examples of document readers he could acquire.

(ii) Explain two benefits the supermarket would gain from the use of these devices.

(c) Rose intends to acquire an application program for typesetting her research work.

(i) Identify the appropriate program for her task.

(ii) Outline four reasons that justify the answer in (i).

Answer

(a) ICT in hotel sales and marketing:

  • Online booking and reservation systems
  • Social media marketing and promotion

(b) Document readers:

(i) Examples:

  • Barcode scanners
  • QR code readers

(ii) Benefits:

  • Faster and accurate item scanning
  • Reduced checkout time and errors

(c) Typesetting program:

(i) Appropriate program: LaTeX or Adobe InDesign

(ii) Justification:

  • Professional document formatting
  • Advanced typography control
  • Support for mathematical equations
  • Consistent formatting throughout document

Question 15

(a) Netcape Airways management intends to acquire a real time processing system for the organization. Explain three advantages of this type of data processing system.

(b) Joash has been employed as a technician in the library and archives department of a computer manufacturing company that produces various types of computers. Outline three criteria he could use to classify the computers.

(c) Moses has been assigned the task of installing the organization's wired Local Area Network. Explain three types of network cables that he could use while carrying out the task.

Answer

(a) Advantages of real-time processing:

  • Immediate transaction processing
  • Up-to-date information availability
  • Better customer service and response time

(b) Computer classification criteria:

  • Processing power and speed
  • Size and portability
  • Purpose and application area

(c) Network cable types:

  • Unshielded Twisted Pair (UTP) cable
  • Shielded Twisted Pair (STP) cable
  • Fiber optic cable
