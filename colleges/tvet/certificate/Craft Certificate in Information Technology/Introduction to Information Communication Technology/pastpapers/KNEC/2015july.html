Question 1 (4 marks)

(a) Define the term Information as used in ICT.

(b) Describe sequential file organization method as applied in computers.

Answer

(a) Information: Processed data that is meaningful and useful to the recipient.

(b) Sequential file organization: A method of storing and accessing records in a specific order, one after the other. Records are accessed in the same order they were written.

Question 3 (4 marks)

Distinguish between mechanical and electronic data processing methods.

Answer

Mechanical Data Processing: Uses manual or mechanical devices like calculators and typewriters to process data. It is slow, labor-intensive, and prone to errors.

Electronic Data Processing: Uses computers and electronic devices to process data. It is fast, efficient, and accurate.

Question 11 (15 marks)

(a) Nicholson intends to buy a printer for his home use. Outline three factors he should consider when selecting the hardware. (3 marks)

(b) Explain each of the following types of computer software:

(i) Operating system;

(ii) Utility software;

(iii) Generic software.

(c) Amgni was advised to use a high level programming language to develop a system for a certain organization.

(i) Outline four features of these programming languages.

(ii) Li≈üt four examples of these programming languages.

Answer

(a) Three factors for selecting a printer:

  • Print quality: Consider the resolution and color accuracy for the intended use.
  • Print speed: Evaluate the pages per minute (PPM) for efficient printing.
  • Cost of ownership: Factor in the price of the printer, ink/toner cartridges, and maintenance.

(b) Types of computer software:

  • (i) Operating system: Software that manages computer hardware and software resources, providing essential services for computer programs.
  • (ii) Utility software: Software designed to analyze, configure, optimize, and maintain a computer.
  • (iii) Generic software: General-purpose applications that can be used for a variety of tasks, such as word processing, spreadsheets, and presentations.

(c) High-level programming languages:

(i) Four features:

  • Readability: Easy to understand and write.
  • Portability: Can run on different platforms with minimal changes.
  • Abstraction: Hides low-level details of the hardware.
  • Modularity: Supports breaking down code into reusable modules.

(ii) Four examples:

  • Python
  • Java
  • C++
  • JavaScript

Question 13 (15 marks)

(it) A technicizn at Usitadi college has been tasked with networking of the college computers. Outline the function of each of the following tools that he might use to network th- computers:

(1) RJ45:

(ii) Crimping tool;

(iii) Punch downs;

(iv) Cable tester;

(v) MODEM.

(b) Speed Bank Ltd has automated its operations, Outline a benefit which the bank would acerue from the use of each the following technologies:

(i) Tele-banking:

(ii) EFT.

(c) (i) Describe a reference file as used in computers.

(ii) Explain two disadvantages of time shearing policy in data processing. (4 marks)

Answer

(a) Networking tools and their functions:

  • (i) RJ45: A connector used to terminate twisted pair cables for Ethernet connections.
  • (ii) Crimping tool: A tool used to attach RJ45 connectors to the ends of Ethernet cables.
  • (iii) Punch down tool: A tool used to terminate wires into punch down blocks, such as those found on patch panels and keystone jacks.
  • (iv) Cable tester: A device used to verify the continuity and proper wiring of network cables.
  • (v) MODEM: A device that converts digital signals from a computer into analog signals for transmission over telephone lines, and vice versa.

(b) Benefits of banking technologies:

  • (i) Tele-banking: Provides customers with convenient access to banking services via telephone, reducing the need to visit a branch.
  • (ii) EFT: Enables electronic transfer of funds, improving transaction speed and efficiency.

(c) (i) Reference file: A file that contains static data used for reference purposes, such as postal codes, product codes, or customer information.

(ii) Two disadvantages of time-sharing:

  • Response time can be unpredictable, especially when many users are active.
  • Can be complex to implement and manage, requiring sophisticated scheduling algorithms.
