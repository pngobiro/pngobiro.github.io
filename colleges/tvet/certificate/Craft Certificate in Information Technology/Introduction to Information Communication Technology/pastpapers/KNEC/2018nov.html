SECTION A (40 marks)

Answer ALL the questions in this section.

Question 1

Define each of the following terms as used in Information Communication Technology:

(i) electronic data interchange;

(ii) telecommuting.

Answer

(i) Electronic Data Interchange (EDI):

The automated exchange of business documents between organizations in a standardized electronic format.

(ii) Telecommuting:

Working from a remote location (usually home) using computers and telecommunications technology to maintain contact with the office.

Question 2

Outline four ethical issues that organizations face as a result of embracing information technology.

Answer

  • Privacy and data protection concerns
  • Intellectual property rights and software piracy
  • Employee monitoring and surveillance
  • Information security and confidentiality

Question 3

Karai College has availed computers for their students that are connected to the Internet. Explain two advantages of this move to the students.

Answer

  • Access to vast educational resources and research materials online
  • Opportunity to develop digital literacy and internet research skills

Question 4

Judy intends to purchase software from an authentic source to assist her in accounts operations. Explain two items that should come along with the software.

Answer

  • User manual/documentation - providing instructions on software use and features
  • License key/certificate - proving authenticity and right to use the software

Question 5

Outline four ways in which computer users could protect themselves from eye strain caused by computer radiations.

Answer

  • Use anti-glare screens or filters
  • Take regular breaks (20-20-20 rule)
  • Adjust screen brightness and contrast
  • Maintain proper viewing distance from the screen

Question 6

(a) Explain the term bandwidth as used in computer networks.

(b) Distinguish between transposition errors and transcription errors as used in data processing.

Answer

(a) Bandwidth:

The maximum amount of data that can be transmitted over a network connection in a given time period, usually measured in bits per second (bps).

(b) Transposition vs Transcription errors:

Transposition errors:

  • Occur when digits or characters are reversed in order (e.g., 12 becomes 21)

Transcription errors:

  • Occur when data is copied incorrectly from one medium to another (e.g., typing "teh" instead of "the")

Question 7

Describe each of the following types of visual display units:

(i) cathode ray tube;

(ii) liquid crystal display.

Answer

(i) Cathode Ray Tube (CRT):

  • Uses electron beam to create images on phosphor-coated screen
  • Bulky and heavy construction
  • High power consumption

(ii) Liquid Crystal Display (LCD):

  • Uses liquid crystals to produce images
  • Thin and lightweight design
  • Lower power consumption

Question 8

Explain each of the following computer output devices with respect to their functionality:

(i) printer;

(ii) plotter.

Answer

(i) Printer:

  • Produces text and graphics on paper
  • Suitable for documents, reports, and general printing needs

(ii) Plotter:

  • Produces large-format drawings and technical designs
  • Specializes in vector graphics like architectural plans and engineering drawings

Question 9

The following are tasks that need to be accomplished in an office:

  • production of reports for the company;
  • management of employees records;
  • production of school magazine;
  • manipulation of numeric data collected from a research station.

State the most appropriate application program that could be used to complete each of the tasks.

  • School magazine - Desktop publishing software (e.g., Adobe InDesign)

Answer any FOUR questions from this section.

Question 11

(a) Explain the term computer crime as used in information communication technology.

(b) A researcher carried out a study on the impacts of ICT on the society and concluded that the community has suffered from its negative impacts. Explain three possible findings from this study.

(c) A large manufacturing firm has opened distribution outlets in three regions within the country and intends to connect them using computer networks.

(i) Describe the type of network that you would recommend to the firm.

(ii) Outline three appropriate devices that could be used during installation of such a network.

Answer

(a) Computer crime:

Criminal activities that involve the use of computers or networks to commit illegal acts, such as fraud, theft, or unauthorized access to data.

(b) Negative impacts of ICT:

  • Social isolation and reduced face-to-face interaction
  • Privacy concerns and data security risks
  • Technology addiction and dependence

(c) (i) Recommended network:

Wide Area Network (WAN) because:

  • Connects geographically dispersed locations
  • Supports data sharing across regions
  • Enables centralized management
  • Facilitates real-time communication

(ii) Network devices:

  • Routers for connecting different networks
  • Switches for local network connections
  • Firewalls for network security

Question 12

(a) With the aid of a diagram, describe each of the following network topologies:

(i) mesh topology;

(ii) ring topology.

(b) Distinguish between analogue computers and digital computers with respect to their functionality.

(c) James has been assigned with the responsibility of organizing ICT systems back-ups and maintenance of computer files.

(i) Explain the function of a back-up file;

(ii) Outline three methods that could be used to create back-up files.

Answer

(a) Network topologies:

(i) Mesh topology:

    PC1 ---- PC2
     | \    / |
     |  \  /  |
     |   \/   |
     |   /\   |
     |  /  \  |
     | /    \ |
    PC3 ---- PC4

Mesh Topology

  • Every device has direct connection to every other device
  • Highly reliable but expensive to implement

(ii) Ring topology:

    PC1 --> PC2
     ^      |
     |      v
    PC4 <-- PC3

Ring Topology

  • Devices connected in a closed loop
  • Data travels in one direction

(b) Analogue vs Digital computers:

Analogue computers:

  • Process continuous data
  • Measure physical quantities
  • Give approximate results

Digital computers:

  • Process discrete data (binary)
  • Use mathematical calculations
  • Give precise results

(c) (i) Back-up file function:

To maintain copies of important data as a safeguard against data loss, system failure, or corruption of original files.

(ii) Back-up methods:

  • Full backup - copying all files
  • Incremental backup - copying only changed files
  • Cloud backup - storing files online

Question 13

(a) Describe each of the following computer input devices as used in computer systems:

(i) digitizer;

(ii) punched tag;

(iii) badge reader.

(b) Figure 1 shows the software family tree. Use it to answer the questions that follow.

(i) Identify each of the software labelled A, B, C and D;

(ii) Describe the function of each of the software labelled A and B giving an example of each.

(iii) Explain two methods that could be used to acquire the software labelled C and D.

Answer

(a) Input devices:

(i) Digitizer:

  • Converts drawings or graphics into digital format
  • Often used with a stylus for precise input

(ii) Punched tag:

  • Card with holes representing data
  • Used in older computer systems

(iii) Badge reader:

  • Reads identification cards or badges
  • Used for access control and security

(b) Software family tree:

(i) Identification:

  • A: System Software
  • B: Application Software
  • C: Operating System
  • D: Utility Programs

(ii) Functions:

System Software (A):

  • Controls and manages computer hardware
  • Example: Windows Operating System

Application Software (B):

  • Performs specific user tasks
  • Example: Microsoft Word

(iii) Acquisition methods:

  • Purchase licensed copies
  • Download free/open-source versions

Question 14

(a) Describe each of the following examples of data transmission media:

(i) UTP;

(ii) microwave;

(iii) Fibre optic cable.

(b) Most colleges have set up computer laboratories to be used by the students. Explain three physical security measures that these colleges should adopt to protect the laboratories.

(c) George is an upcoming musician who intends to use computers in his operations. Outline three ways in which he would benefit from the use of computers.

Answer

(a) Data transmission media:

(i) UTP (Unshielded Twisted Pair):

  • Pairs of copper wires twisted together
  • Common in network cabling
  • Cost-effective but susceptible to interference

(ii) Microwave:

  • Wireless transmission using high-frequency waves
  • Used for long-distance communication
  • Line of sight required

(iii) Fibre optic cable:

  • Uses light to transmit data
  • High bandwidth and speed
  • Immune to electromagnetic interference

(b) Physical security measures:

  • Access control systems (locks, security cards)
  • Security cameras and surveillance
  • Fire protection systems

(c) Benefits for musician:

  • Digital music recording and editing
  • Music composition and arrangement
  • Online music distribution and promotion

Question 15

(a) With the aid of a block diagram, outline the stages of data processing cycle.

(b) Computer buses are used to connect different components in a computer system. Explain the role of each of the following computer buses:

(i) address bus;

(ii) data bus;

(iii) control bus.

(c) Loise has been tasked to acquire a computer printer to be used by her organization. Outline four factors that she should consider.

Answer

(a) Data processing cycle:

    +--------+     +------------+     +--------+
    | Input  | --> | Processing | --> | Output |
    +--------+     +------------+     +--------+
         ^                                |
         |                                |
         +--------------------------------+
              Storage/Feedback

Data Processing Cycle

(b) Computer buses:

(i) Address bus:

  • Carries memory addresses
  • Determines which memory location to access

(ii) Data bus:

  • Transfers actual data between components
  • Bidirectional flow of information

(iii) Control bus:

  • Carries control signals
  • Coordinates system operations

(c) Printer selection factors:

  • Print quality and resolution
  • Print speed and volume capacity
  • Cost of printer and consumables
  • Network connectivity options
