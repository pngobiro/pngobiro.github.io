Question 1

(a) Explain the term Information Communication Technology as used in computers.

(b) Describe data preparation as used in data processing.

Answer

(a) Information Communication Technology (ICT):

Technologies used for creating, storing, processing, and communicating information electronically, including computers, software, networks, and communication devices.

(b) Data preparation:

The process of collecting, organizing, and converting raw data into a suitable format for computer processing, including:

  • Data validation
  • Error checking
  • Data formatting

Question 2

(a) State two benefits that an organization gains by using computers.

(b) Ann's computer has a memory capacity of 2 Gigabytes. Convert this memory capacity to Kilobytes.

Answer

(a) Benefits of computers:

  • Increased efficiency and productivity
  • Improved accuracy in data processing

(b) Memory conversion:

2 GB = 2 × 1024 MB = 2 × 1024 × 1024 KB = 2,097,152 Kilobytes

Question 3

Differentiate between transposition error and transcription error as used in data processing.

Answer

Transposition error:

  • Occurs when digits or characters are reversed in order
  • Example: typing "12" as "21"

Transcription error:

  • Occurs when data is copied incorrectly from one medium to another
  • Example: typing "house" as "hause"

Question 6

(a) Outline two characteristics of computer viruses.

(b) Explain the term Trojan as used in ICT.

Answer

(a) Virus characteristics:

  • Self-replication capability
  • Ability to attach to other files

(b) Trojan:

Malicious software that appears legitimate but contains harmful code. Unlike viruses, Trojans don't self-replicate but can create backdoors for unauthorized access.

(b) Copyright Act laws:

  • Protection against unauthorized copying of software
  • Rights of software creators to control distribution

Question 11

(a) With the aid of an example, describe a magnetic scanner as used in computers.

(b) Explain each of the following categories of computer system software:

(i) utility;

(ii) multi-tasking Operating System;

(iii) networking Operating System.

(c) Abdi, a computer technician in a college has bought general-purpose software for use by members of staff. Outline three advantages and three disadvantages of using general-purpose software.

Answer

(a) Magnetic scanner:

Device that reads magnetic ink characters

Example: MICR (Magnetic Ink Character Recognition) used in bank checks

(b) System software categories:

(i) Utility:

  • Programs that help maintain and optimize computer system
  • Example: Antivirus, disk defragmenter

(ii) Multi-tasking OS:

  • Allows multiple programs to run simultaneously
  • Example: Windows 10, macOS

(iii) Networking OS:

  • Manages network resources and communications
  • Example: Windows Server, Linux Server

(c) General-purpose software:

Advantages:

  • Cost-effective compared to custom software
  • Readily available and easy to acquire
  • Well-tested and reliable

Disadvantages:

  • May not meet specific needs perfectly
  • May include unnecessary features
  • Limited customization options

Question 12

(a) Outline the steps followed when installing a computer application software.

(b) With the aid of an example in each case, explain the following terms:

(i) software piracy;

(ii) plagiarism;

(c) Explain three data processing methods.

Answer

(a) Software installation steps:

  1. Check system requirements
  2. Run setup/installation program
  3. Accept license agreement
  4. Choose installation location
  5. Select components to install
  6. Complete installation and restart if required

(b) Terms:

(i) Software piracy:

  • Unauthorized copying or distribution of software
  • Example: Making copies of Windows OS without license

(ii) Plagiarism:

  • Using others' work without proper attribution
  • Example: Copying text from website without citing source

(c) Data processing methods:

  • Batch processing - Processing data in groups
  • Real-time processing - Immediate processing of data
  • Online processing - Processing data over network connections

Question 13

(a) Peter would like to set up a computer network using star topology. Explain the role of each of the following components.

(i) hub;

(ii) cables;

(iii) computers.

(b) Faida has a chain of supermarkets and would like to introduce computers in his operations. Explain two possible uses of these computers.

(c) With the aid of an example in each case, describe each of the following computer devices:

(i) pointing device;

(ii) keying device.

Answer

(a) Star topology components:

(i) Hub:

  • Central connection point
  • Manages data flow between devices

(ii) Cables:

  • Connect devices to hub
  • Carry data between hub and devices

(iii) Computers:

  • End devices in the network
  • Send and receive data through hub

(b) Computer uses in supermarkets:

  • Point of Sale (POS) systems for transactions
  • Inventory management and stock control

(c) Computer devices:

(i) Pointing device:

  • Used to control cursor movement
  • Example: Mouse, trackpad

(ii) Keying device:

  • Used for text and data entry
  • Example: Keyboard, numeric keypad
