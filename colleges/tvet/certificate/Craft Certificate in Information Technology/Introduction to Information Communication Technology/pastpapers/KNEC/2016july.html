Question 1

(a) Define the term firmware as used in computer networks.

(b) Outline two reasons why an invention should be patented.

Answer

(a) Firmware Definition:

Firmware is a specific type of software that is permanently embedded into a hardware device's read-only memory (ROM) or flash memory. It provides low-level control and instructions necessary for the hardware to operate and communicate with other software (like the operating system).

  • Examples: BIOS/UEFI on a motherboard, software on a router or printer.
  • It's typically updated less frequently than application or system software.

(b) Reasons for Patenting an Invention:

  1. Exclusive Rights: A patent grants the inventor the exclusive right to make, use, sell, and import the invention for a set period (usually 20 years), preventing others from commercially exploiting it without permission.
  2. Market Advantage & Commercialization: Patents provide a competitive edge, can attract investment, and allow the inventor to license the technology to others for royalties or sell the patent rights outright.

Question 3

Explain each of the following computer network technologies as used in LAN:

(i) bus topology;

(ii) token ring.

Answer

Computer Network Technologies in LAN:

  1. Bus Topology:
    • All devices connect to a single central cable (bus)
    • Data travels along the entire bus in both directions
    • Simple and inexpensive to install
    • Vulnerable to cable breaks and performance degradation under heavy load
  2. Token Ring:
    • Devices connected in a logical ring configuration
    • Uses token passing for controlled network access
    • More reliable under heavy load than bus topology
    • More complex and expensive to implement

Minicomputers, supercomputers, mainframes, microcomputers

Arrange them in terms of their processing power from lowest to the highest.

Cyber crime refers to criminal activities where a computer or network is the source, tool, target, or place of a crime.

Question 5

(a) Explain the function of each of the following components of a computer CPU:

(i) control unit;

(ii) ALU.

(b) It has been noted that plagiarism is on the increase in submitted projects by ICT students. Explain two measures that could be put in place to deter students from this practice.

Answer

(a) CPU Components Functions:

  1. Control Unit (CU):
    • Directs and coordinates all computer operations
    • Fetches instructions from memory and decodes them
    • Manages data flow between CPU and other components
    • Generates timing and control signals
    • Coordinates execution of instructions
  2. ALU (Arithmetic Logic Unit):
    • Performs all arithmetic operations (addition, subtraction, multiplication, division)
    • Handles logical operations (AND, OR, NOT)
    • Makes comparisons (equal to, greater than, less than)
    • Processes data according to program instructions

(b) Measures to Deter Plagiarism:

  1. Use of Plagiarism Detection Tools:
    • Implement software like Turnitin to check submissions
    • Use code plagiarism detectors for programming assignments
    • Regular scanning of submitted work
  2. Education and Policy Implementation:
    • Teach proper citation and referencing methods
    • Establish clear academic integrity guidelines
    • Communicate consequences of plagiarism
    • Provide workshops on academic writing

Question 8

Most organizations are shifting from the use of twisted pair cables to the use of fiber optic cables in their computer networks. Explain two reasons for this trend.

Answer

Reasons for Shifting to Fiber Optic Cables:

  1. Higher Bandwidth and Speed:
    • Superior data transmission rates (Gbps to Tbps)
    • Better support for demanding applications:
      • Video conferencing
      • Cloud services
      • Large file transfers
      • High-definition multimedia
    • Future-proof infrastructure investment
  2. Signal Quality and Range:
    • Immune to electromagnetic interference (EMI)
    • No signal degradation from electrical noise
    • Longer transmission distances:
      • Several kilometers vs. 100m for twisted pair
      • Ideal for campus networks
      • Efficient for building-to-building connections

Question 10

Explain each of the following file organization methods as used in file management:

(i) random;

(ii) indexed sequential.

Answer

File Organization Methods:

  1. Random (Direct) File Organization:
    • Records stored at calculated addresses using hashing
    • Direct access to records without sequential search
    • Very fast retrieval of individual records
    • May have collision handling requirements
    • Best for frequent random access needs
  2. Indexed Sequential File Organization:
    • Records stored in sequence by key field
    • Index provides quick access to record locations
    • Supports both sequential and random access
    • Requires extra storage for index maintenance
    • Good balance between sequential and random access

Question 12

(a)

(i) Outline two factors that could be considered while choosing a data transmission medium.

(ii) Differentiate between BaveBand and BroadBand coaxial cables.

(b) William intends to computerize operations in his supermarkets. Explain three types of data capture devices that could be used in the system.

(c) Judy wishes to acquire a word processing application for her laptop. Outline three factors she should consider other than cost.

Answer

(a) Data Transmission Medium:

  1. Factors for Choosing a Medium:
    • Bandwidth/Speed Requirements: The amount of data that needs to be transmitted per unit of time. Higher bandwidth needs (e.g., video streaming) require faster media like fiber optics.
    • Distance: The physical distance the signal needs to travel. Different media have different maximum effective distances (e.g., twisted pair ~100m, fiber optics can span kilometers).
    • Cost: The budget available for installation and maintenance. Fiber optic is generally more expensive than twisted pair.
    • Environment/Interference: The level of electromagnetic interference (EMI) in the environment. Fiber optic is immune to EMI, unlike copper cables.
    • Security Requirements: The need to protect data from eavesdropping. Fiber optic is generally more secure than copper cables.
  2. Baseband vs. Broadband Coaxial Cables:
    • Baseband: Transmits a single digital signal at a time over the cable. Uses the entire bandwidth for one channel. Simpler, often used in older LANs (like early Ethernet).
    • Broadband: Transmits multiple analog signals simultaneously over the cable using different frequency channels (Frequency Division Multiplexing). Allows for carrying multiple data streams (e.g., data, voice, video) concurrently. Used in cable television (CATV) systems.

(b) Data Capture Devices for Supermarket:

  1. Barcode Scanners: Read Universal Product Codes (UPCs) on items to quickly identify products and retrieve prices from the database at checkout counters.
  2. Point of Sale (POS) Terminals: Integrated systems often including a touch screen, cash drawer, receipt printer, and card reader to process sales transactions, calculate totals, accept payments, and update inventory.
  3. Magnetic Stripe Readers (MSR) / Chip Readers (EMV): Used at POS terminals to read customer credit or debit card information for payment processing.

(c) Factors for Choosing Word Processing Software (Other than Cost):

  1. Features and Functionality: Does it have the necessary features like advanced formatting options, mail merge, table of contents generation, collaboration tools, grammar/spell check, template availability?
  2. Compatibility: Can it open and save files in common formats (e.g., .docx, .odt, .pdf)? Is it compatible with the operating system (Windows, macOS, Linux)? Does it integrate well with other software Judy uses?
  3. User Interface and Ease of Use: Is the interface intuitive and easy to learn? Does the layout suit Judy's workflow? Is it customizable?

Question 13

(a) Outline three types of computer files as used in data processing.

(b) Itech Ltd has advertised for sale a personal computer with the following specifications; 4 GB RAM, 200 GB hard disk and DVD drive. Explain each of the devices listed in the specification.

(c) Martin has been requested to acquire computer software for Stats Company. Explain three methods that he could use to achieve his objective.

Answer

(a) Types of Computer Files in Data Processing:

  1. Master File: Contains relatively permanent records that are updated periodically. It holds key information about entities (e.g., customer master file with names, addresses; product master file with item details).
  2. Transaction File: Contains temporary data about business events or transactions that occur over a specific period. This data is used to update the master file (e.g., daily sales file, new customer registrations file).
  3. Report File: Contains formatted data extracted and summarized from master or transaction files, ready for presentation to users (e.g., monthly sales report, employee payroll summary).
  4. Backup File: A copy of master, transaction, or other important files created for security purposes to allow recovery in case of data loss or corruption.

(b) Explanation of PC Specifications:

  • 4 GB RAM (Random Access Memory): This is the computer's primary volatile memory used for temporarily storing data and program instructions that the CPU is actively using. 4GB indicates the capacity, affecting how many applications can run smoothly simultaneously and the overall system responsiveness.
  • 200 GB Hard Disk (Hard Disk Drive - HDD): This is the computer's primary non-volatile storage device. It permanently stores the operating system, applications, and user files. 200GB indicates the total storage capacity for long-term data retention.
  • DVD Drive (Digital Versatile Disc Drive): An optical disc drive used for reading data from and possibly writing data to DVDs and CDs. Allows for installing software from discs, watching DVD movies, or backing up data to optical media.

(c) Methods for Acquiring Computer Software:

  1. Purchasing Off-the-Shelf Software: Buying pre-packaged, commercially available software from retail stores or online vendors (e.g., Microsoft Office, Adobe Creative Suite). This is often cost-effective for standard needs but may lack customization.
  2. Developing Custom Software In-House or Outsourcing: Creating software specifically tailored to the company's unique requirements, either by using internal programmers or hiring an external software development company. This offers maximum customization but can be expensive and time-consuming.
  3. Using Open-Source Software: Acquiring software whose source code is freely available for use, modification, and distribution (e.g., Linux operating system, LibreOffice suite). This can be cost-free but may require more technical expertise for support and customization.
  4. Subscribing to Software as a Service (SaaS): Accessing software over the internet on a subscription basis (e.g., Salesforce, Google Workspace). This offers scalability and accessibility but relies on internet connectivity and involves recurring costs.

Question 14

(a)

(i) Define the term repetitive strain injury as used in Information Communication Technology.

(ii) Explain three risk factors that could lead to repetitive strain injury.

(b) Mugosh Ltd wishes to acquire a printer for the office use. Outline four criteria that could be used while selecting the printer other than cost.

(c) Many schools have embraced the use of ICT in their learning activities. Outline three ways that this move could positively impact on learners.

Answer

(a) Repetitive Strain Injury (RSI):

  1. Definition:

    RSI refers to damage to muscles, nerves, and tendons caused by repetitive movements and poor ergonomics during computer use. In ICT, it commonly affects wrists, hands, arms, neck, and shoulders due to prolonged keyboard and mouse use.

  2. Risk Factors:
    • Poor Posture: Incorrect seating position, improper monitor height
    • Repetitive Movements: Extended typing or mouse use without breaks
    • Static Posture: Maintaining fixed position for long periods
    • Inadequate Ergonomics: Poor workstation setup, non-ergonomic equipment
    • Lack of Breaks: Insufficient rest periods and stretching

(b) Printer Selection Criteria:

  1. Print Quality: Resolution (DPI), color accuracy, text sharpness
  2. Speed: Pages per minute for both color and black/white
  3. Connectivity: USB, network, wireless options
  4. Features: Double-sided printing, paper capacity, multi-function capabilities

(c) Positive Impacts of ICT in Learning:

  1. Enhanced Access to Information:
    • Digital libraries and research materials
    • Online educational resources
    • Current and diverse learning content
  2. Interactive Learning:
    • Multimedia presentations
    • Educational software and simulations
    • Online collaboration tools
  3. Skill Development:
    • Digital literacy skills
    • Research and information evaluation
    • Technical proficiency

Question 15

(a)

(i) Define the term systems software as used in computer system.

(ii) Outline two types of systems software giving an example in each case.

(b) Figure 3 shows computer ports used to connect computer peripherals. Use it to answer the questions that follow.

(i) Identify the ports labeled I, II, III and IV.

(ii) State an example of the computer peripheral that would be used on each of the ports.

(c) Furha Company intends to introduce web based application systems for their daily office duties. Outline three ways in which this move would positively affect the employees.

(3 marks)

Answer

(a) Systems Software:

  1. Definition: Systems software is a type of computer software designed to operate and control the computer hardware and to provide a platform for running application software. It manages the computer's resources and performs essential background tasks.
  2. Types and Examples:
    • Operating Systems (OS): Manages hardware, software resources, and provides common services for computer programs.
      • Example: Microsoft Windows, macOS, Linux, Android, iOS.
    • Utility Programs: Perform specific tasks related to managing computer hardware, operating systems, or application software.
      • Example: Antivirus software (e.g., Norton), disk defragmenters, file compression tools (e.g., WinZip), backup software.
    • Device Drivers: Software that allows the operating system to communicate with specific hardware devices.
      • Example: Printer driver, graphics card driver.
    • Firmware: Software embedded directly into hardware devices.
      • Example: BIOS/UEFI on a motherboard.

(b) Computer Ports in Figure 3:

  1. Identification:
    • I: PS/2 Port (typically purple for keyboard, green for mouse)
    • II: VGA Port (Video Graphics Array - DB-15 connector)
    • III: Parallel Port (DB-25 connector)
    • IV: USB Port (Universal Serial Bus)
  2. Example Peripherals:
    • I (PS/2): Older keyboards or mice.
    • II (VGA): Monitors, projectors.
    • III (Parallel): Older printers, scanners.
    • IV (USB): Modern keyboards, mice, printers, flash drives, external hard drives, webcams, etc.

(c) Positive Effects of Web-Based Applications on Employees:

  1. Accessibility and Flexibility: Employees can access applications and data from anywhere with an internet connection, enabling remote work, flexible hours, and access while traveling.
  2. Collaboration: Web-based applications often include built-in collaboration features, allowing multiple employees to work on the same documents or projects simultaneously, improving teamwork and communication.
  3. Reduced IT Burden and Automatic Updates: Employees don't need to install or update software locally, as updates are managed centrally by the provider. This reduces reliance on internal IT support for software maintenance.
