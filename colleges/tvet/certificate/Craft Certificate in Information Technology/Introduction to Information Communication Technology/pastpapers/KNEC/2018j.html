SECTION A (40 marks)

Answer ALL the questions in this section.

Question 1

Explain each of the following file access methods in a computer system:

(i) random access;

(ii) indexed sequential.

Answer

(i) Random Access:

  • Direct access to any record without going through other records
  • Each record has a unique address or key
  • Fast retrieval of specific records

(ii) Indexed Sequential:

  • Combines sequential and indexed access methods
  • Records stored sequentially but with an index
  • Index provides faster access to specific records

Question 2

Explain each of the following types of outputs, giving an example in each case:

(i) softcopy;

(ii) hardcopy.

Answer

(i) Softcopy:

  • Output displayed on screen or stored electronically
  • Example: PDF document, screen display

(ii) Hardcopy:

  • Physical, permanent output on paper or other material
  • Example: Printed document, plotted graph

Question 3

Distinguish between the terms bandwidth and broadband as used in computer networks.

Answer

Bandwidth:

  • Maximum data transfer capacity of a network connection
  • Measured in bits per second (bps)

Broadband:

  • High-speed internet connection using wide bandwidth
  • Allows simultaneous transmission of multiple signals

Question 4

(a) Jacob has been requested to connect all the computers in the college on a local area network. State four networking hardware tools that would assist him to accomplish the task.

(b) A certain organization intends to use web-based applications for its operations. Outline two drawbacks the organization is likely to face from this technology.

Answer

(a) Networking hardware tools:

  • Network switches
  • Network cables (UTP/STP)
  • Network Interface Cards (NICs)
  • Router

(b) Drawbacks of web-based applications:

  • Dependency on internet connectivity
  • Security risks from online threats

Question 5

Merryland Hospital has installed a database management system for its operations. Explain two advantages of this system to the organization.

Answer

  • Centralized data management and easy access to patient records
  • Improved data security and controlled access to sensitive information

Question 6

Mercy has been tasked to enforce the computer systems data security within her organization. Explain two logical measures she could implement to achieve this.

Answer

  • Implementation of user authentication and access control
  • Data encryption for sensitive information

Question 7

Outline four factors that should be considered during the acquisition of computer software.

Answer

  • Cost of software and licensing
  • Hardware compatibility requirements
  • User-friendliness and ease of use
  • Technical support and documentation

Question 8

Differentiate between off-the-shelf and custom made software.

Answer

Off-the-shelf software:

  • Ready-made software available for immediate use
  • General purpose and standardized features
  • Usually less expensive

Custom made software:

  • Developed specifically for a particular organization
  • Tailored to specific requirements
  • Generally more expensive

Question 9

Explain each of the following data processing modes:

(i) batch processing;

(ii) multi-processing.

Answer

(i) Batch processing:

  • Data collected over time and processed as a group
  • No immediate processing of individual transactions

(ii) Multi-processing:

  • Multiple processors working simultaneously
  • Capable of handling multiple tasks at once

Question 10

Explain each of the following terms as applied in computer ethics:

(i) software piracy;

(ii) software patent.

Answer

(i) Software piracy:

  • Unauthorized copying, distribution, or use of software
  • Violation of software licensing agreements

(ii) Software patent:

  • Legal protection for unique software inventions
  • Gives exclusive rights to the inventor

Question 11

(a) Outline three characteristics of a real time transaction processing system.

(b) Universal Serial Bus devices have become popular with computer users. Explain three reasons for this trend.

(c) Organizations face ethical issues as a result of continuous use of Information Communication Technology. Explain three such issues.

Answer

(a) Real time transaction processing characteristics:

  • Immediate processing of transactions
  • Continuous system availability
  • Quick response time

(b) USB popularity reasons:

  • Plug and play capability
  • Hot-swappable devices
  • Universal compatibility

(c) ICT ethical issues:

  • Privacy and data protection concerns
  • Software piracy and intellectual property rights
  • Cybersecurity and information security

Question 12

(a) A certain country has introduced the use of robots in the war zones. Outline five ways in which they could use these robots.

(b) Describe each of the following computer devices:

(i) flatbed scanner;

(ii) graphic tablet.

(c) Amani hospital intends to install a Local Area Network to all its offices that occupy two floors within the same building. Explain three advantages they would achieve from the network.

Answer

(a) Uses of robots in war zones:

  • Reconnaissance and surveillance
  • Bomb detection and disposal
  • Supply delivery in dangerous areas
  • Search and rescue operations
  • Remote combat operations

(b) Computer devices:

(i) Flatbed scanner:

  • Device that captures digital images of flat documents
  • Uses light-sensitive sensors to convert images to digital format

(ii) Graphic tablet:

  • Input device for hand-drawing digital images
  • Uses stylus for precise drawing and design work

(c) LAN advantages for hospital:

  • Efficient sharing of patient records and medical data
  • Centralized resource sharing (printers, storage)
  • Improved communication between departments

Question 13

(a) Outline three elements of a computer file.

(b) Distinguish between a general purpose and a dedicated computer, giving an example of each.

(c) With the aid of a diagram, describe the functional components of the central processing unit in a computer system.

Answer

(a) Elements of a computer file:

  • File name
  • File extension
  • File content/data

(b) Computer types:

General purpose computer:

  • Can perform various types of tasks
  • Example: Personal computer

Dedicated computer:

  • Designed for specific tasks
  • Example: ATM machine

(c) CPU components:

    +----------------+
    |  Control Unit  |
    +----------------+
           |  |
    +----------------+
    |     ALU       |
    +----------------+
           |  |
    +----------------+
    |   Registers   |
    +----------------+

CPU Components

  • Control Unit: Manages and coordinates CPU operations
  • ALU (Arithmetic Logic Unit): Performs calculations and logical operations
  • Registers: Temporary high-speed storage for data and instructions

Question 14

(a) David acquired a computer and its related software for use in carrying out computer-aided design tasks.

(i) Explain the term software.

(ii) Explain two types of system software that he may acquire giving an example of each.

(b) A computer may harm users if they frequently use it for long hours. Outline three health problems that could be caused by such prolonged use giving a remedy for each.

(c) Joshua recommended the use of graphical user interface for first time computer users in the organization. State four features of this interface.

Answer

(a) (i) Software:

Programs and instructions that tell the computer hardware what to do and how to perform specific tasks.

(ii) System software types:

1. Operating System:

  • Manages hardware and software resources
  • Example: Windows, macOS

2. Utility Software:

  • Performs maintenance and system tasks
  • Example: Antivirus, disk defragmenter

(b) Health problems and remedies:

  • Eye strain - Use anti-glare screen and take regular breaks
  • Back pain - Use ergonomic chair and maintain proper posture
  • Carpal tunnel syndrome - Use wrist rest and perform exercises

(c) GUI features:

  • Icons and buttons for visual representation
  • Pull-down menus for easy access to commands
  • Windows for organizing content
  • Point-and-click interface for easy navigation

Question 15

(a) State three advantages of fibre optic cables in computer networking.

(b) Explain three positive impacts that ICT has on society.

(c) Figure 1 shows network topologies applied in a certain college. Use it to answer the questions that follow.

Describe each of the network topologies labelled (i) and (ii).

Answer

(a) Advantages of fiber optic cables:

  • Higher bandwidth and faster data transmission
  • Immune to electromagnetic interference
  • Greater security as signals are difficult to tap

(b) Positive impacts of ICT:

  • Improved communication and connectivity
  • Enhanced access to information and education
  • Increased efficiency in business operations

(c) Network topologies:

Topology (i) - Star Topology:

  • All devices connect to central hub/switch
  • Easy to manage and troubleshoot
  • Failure of one node doesn't affect others

Topology (ii) - Bus Topology:

  • All devices connect to single cable (backbone)
  • Simple and cost-effective
  • Easy to implement for small networks
