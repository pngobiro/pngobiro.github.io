Question 1

Explain each of the following activities as used in data processing:

(i) summarize;

(ii) classify.

Answer

(i) Summarize:

Condensing data into a shorter form while retaining key information and meaning. This might involve calculating totals, averages, or creating concise reports.

(ii) Classify:

Organizing data into categories or classes based on common characteristics or criteria to make it easier to understand and analyze.

Question 2

Distinguish between analogue computers and digital computers.

Answer

Analogue Computers:

  • Process continuous data
  • Work with physical quantities (temperature, pressure, voltage)
  • Give approximate results

Digital Computers:

  • Process discrete data (binary: 0s and 1s)
  • Work with numerical data
  • Give precise results

Question 3

Pitso Company chose to use in-house development method for its application software. Explain two benefits the company would derive from the use of this method.

Answer

  • Customization - Software can be tailored exactly to the company's specific needs and requirements
  • Control - Company has full control over development, modifications, and maintenance of the software

Question 4

Outline four characteristics of second generation computers.

Answer

  • Used transistors instead of vacuum tubes
  • Smaller in size compared to first generation
  • Generated less heat and consumed less power
  • Used assembly language for programming

Question 5

Josh saved each of the following files on his computer:

Josh.rtf, josh.docx, josh.accdb, josh.xls

State the application software that he used to create each of the files.

Answer

  • josh.rtf - Rich Text Format editor (e.g., WordPad)
  • josh.docx - Microsoft Word
  • josh.accdb - Microsoft Access
  • josh.xls - Microsoft Excel

Question 6

Describe each of the following types of computer networks:

(i) wide area network;

(ii) private area network.

Answer

(i) Wide Area Network (WAN):

  • Covers large geographical areas
  • Connects multiple LANs across cities or countries
  • Often uses public communication infrastructure

(ii) Private Area Network:

  • Restricted to specific organization or user group
  • Uses private infrastructure
  • Higher security and controlled access

Question 7

Describe each of the following types of computer software:

(i) open source;

(ii) proprietary.

Answer

(i) Open Source Software:

  • Source code is freely available
  • Can be modified and distributed
  • Usually free to use

(ii) Proprietary Software:

  • Source code is private and protected
  • Cannot be modified by users
  • Usually requires payment for use

Question 8

A certain shop installed a point of sale system for its operations. Outline four input devices that could be included in the system.

Answer

  • Barcode scanner
  • Card reader
  • Keyboard
  • Touch screen display

Question 9

A company that has branches all over the country intends to install a distributed data processing system to support its operations. Explain two benefits of this system to the company.

Answer

  • Improved reliability - system continues to function even if one location fails
  • Better performance - local processing reduces network traffic and response times

Question 10

Differentiate between trojan horse and logic bomb viruses in computers.

Answer

Trojan Horse:

  • Appears as legitimate software
  • Performs malicious actions while appearing harmless
  • Does not replicate itself

Logic Bomb:

  • Activates when specific conditions are met
  • Remains dormant until triggered
  • Often tied to specific dates or events

Question 12

(a) Outline three advantages of peer-to-peer type of computer network.

(b) Describe each of the following types of file organization methods in a computer system:

(i) serial;

(ii) sequential;

(iii) random.

(c) An ICT technician has been tasked to set up a computer network for a certain company. Explain three types of network topologies that he could use.

Answer

(a) Advantages of peer-to-peer network:

  • Low cost - no need for dedicated server
  • Easy to set up and maintain
  • Direct resource sharing between computers

(b) File organization methods:

(i) Serial:

  • Records stored one after another
  • No particular order
  • Must search entire file to find record

(ii) Sequential:

  • Records stored in sorted order
  • Efficient for batch processing
  • Must maintain sort order

(iii) Random:

  • Records stored using addresses
  • Direct access to records
  • Faster retrieval of specific records

(c) Network topologies:

  • Star topology - all devices connect to central hub
  • Bus topology - all devices connect to single cable
  • Ring topology - devices connect in closed loop

Question 15

(a) Describe each of the following devices as used in computer networking:

(i) gateway;

(ii) router;

(iii) switch.

(b) Football associations in the world have integrated the use of Information Communication Technology to monitor ongoing games. Explain three ways in which the technology is used.

(c) Outline three reasons that would necessitate an organization to adopt the use of external hard disks in their operations.

Answer

(a) Network devices:

(i) Gateway:

  • Connects different types of networks
  • Translates between different protocols

(ii) Router:

  • Directs data between networks
  • Determines best path for data packets

(iii) Switch:

  • Connects devices within same network
  • Forwards data to specific devices

(b) ICT in football:

  • Goal-line technology for accurate goal decisions
  • Video Assistant Referee (VAR) system
  • Player performance tracking and analysis

(c) Reasons for external hard disks:

  • Backup and data recovery
  • Additional storage capacity
  • Portable data transfer between locations
