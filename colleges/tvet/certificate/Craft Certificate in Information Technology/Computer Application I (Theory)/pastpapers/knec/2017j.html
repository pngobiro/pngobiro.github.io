Question 2 (X marks)

(a) Explain the function of textbox feature as used in word processing programs.

(b) Explain the term data validation as applied in spreadsheet programs.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Textbox Feature Function:

A textbox allows users to insert text anywhere on a page, independent of the main document flow. It's useful for creating callouts, sidebars, captions, or placing text precisely over or next to graphics, offering more layout flexibility than standard paragraph text.

(b) Data Validation:

Data validation is a feature used to control the type of data or the values that users can enter into a cell. It helps ensure data accuracy and consistency by setting rules (e.g., restricting input to whole numbers, dates within a range, values from a dropdown list) and providing error messages if invalid data is entered.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Reason for Using Table Template:

A database administrator might use a table template to quickly create a new table with predefined fields, data types, and properties relevant to a common subject (e.g., contacts, events, tasks). Question 6 (4 marks)

(a) Differentiate between primary and secondary files as used in a mail merge feature of a word processing program. (2 marks)

(b) Explain a circumstance under which a many-to-many table relationship would be established in a database program. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Primary vs. Secondary Mail Merge Files:

  • Primary File (Main Document/Template): This is the document containing the standard text, formatting, and layout that will be the same for all merged copies (e.g., the letter template). It also contains merge fields (placeholders) where data from the secondary file will be inserted.
  • Secondary File (Data Source): This file contains the variable data that will be inserted into the merge fields of the primary document (e.g., a spreadsheet, database table, or list containing names, addresses, and other personalized information for each recipient).

(b) Circumstance for Many-to-Many Relationship:

A many-to-many relationship is needed when one record in Table A can be related to multiple records in Table B, AND one record in Table B can also be related to multiple records in Table A. A common example is the relationship between Students and Courses: one student can enroll in many courses, and one course can have many students enrolled. This relationship is typically implemented using a third "junction" or "linking" table that contains foreign keys referencing the primary keys of both Table A and Table B.

Question 8 (4 marks)

(a) An examination database stores details of students who sat for an exam. Each record has the following fields:

  • Student ID Number
  • The student passport sized photo
  • Scores for each subject
  • Total Fees Balance

State a suitable data type for each of the fields. (2 marks)

(b) State the function of each of the following folders in an email program:

  1. inbox;
  2. spam.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Suitable Data Types:

  • Student ID Number: Text (if it can contain letters or leading zeros) or Number (if purely numeric and no leading zeros). AutoNumber could also be suitable if it's the primary key.
  • Student passport sized photo: OLE Object or Attachment (depending on the database system).
  • Scores for each subject: Number (likely Integer or Decimal/Double depending on whether scores can have fractions).
  • Total Fees Balance: Currency or Number (Decimal/Double).

(b) Email Folder Functions:

  1. Inbox: The primary folder where new, incoming emails are typically delivered and stored by default.
  2. Spam (or Junk Mail): A folder where the email program automatically filters and places messages suspected of being unsolicited, unwanted bulk email (spam) to keep the main inbox cleaner.

Question 9 (X marks)

(a) Explain the function of slide sorter feature as used in presentation programs.

(b) A company which deals with the importation of second hand motor vehicles has a website with various hyperlinks. Explain the function of this feature on the website.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Slide Sorter Function:

The slide sorter view displays thumbnails (miniature versions) of all slides in the presentation in a grid format. Its main function is to allow the user to easily view the overall flow of the presentation and rearrange the order of slides by dragging and dropping the thumbnails. It's also useful for applying transitions or timings to multiple slides at once.

(b) Hyperlink Function on Website:

Hyperlinks on the company's website allow users to navigate easily between different pages or sections of the site (e.g., linking from a car model listing to its detailed specifications page) or to external resources (e.g., linking to a manufacturer's site or a financing partner). They create a connected structure, making information accessible and improving user experience.
