Question 1 (X marks)

Explain each of the following terms as used in computer security:

(a) trespass;

(b) cracking.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Trespass: In computer security, trespass refers to unauthorized access to computer systems, networks, or data. It involves gaining entry or using resources without permission, similar to physical trespassing onto property.

(b) Cracking: Cracking involves modifying software or systems to remove or disable features, especially copy protection or registration requirements. It is often associated with software piracy and bypassing security measures for unauthorized use.

Question 2 (X marks)

(a) state the function of each of the following features in a word processor:

  1. thesaurus;
  2. format painter.

(b) With the aid of an example, explain absolute reference as used in spreadsheet programs. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Word Processor Feature Functions:

  1. Thesaurus: Provides synonyms (and sometimes antonyms) for selected words, helping users find alternative words to improve vocabulary or avoid repetition.
  2. Format Painter: Copies the formatting (font, size, color, style, etc.) from selected text or an object and applies it to other selected text or objects.

(b) Absolute Reference:

An absolute reference in a spreadsheet is a cell address that remains fixed when a formula is copied to other cells. It is denoted by placing dollar signs ($) before both the column letter and the row number.

Example: If cell C1 contains the formula =A1*$B$1, and this formula is copied down to cell C2, the formula in C2 will become =A2*$B$1. The reference to A1 changes relatively to A2, but the reference to $B$1 remains absolute, always pointing to cell B1 (useful for constants like tax rates or commission percentages).

Question 3 (X marks)

Purity had the following errors displayed in different cells when working on a worksheet. Explain the cause of each of the errors and a possible solution.

(a) #####

(b) #REF!

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Worksheet Error Explanations:

(a) ##### :

  • Cause: The column is too narrow to display the entire numeric value, date, or time formatted in the cell. It can also occur with negative dates/times.
  • Solution: Increase the width of the column containing the cell until the value is fully visible.

(b) #REF! :

  • Cause: The formula contains an invalid cell reference. This often happens when cells, rows, or columns referred to by the formula have been deleted, or when a formula is copied to a location where its relative references become invalid (e.g., referring outside the worksheet boundaries).
  • Solution: Check the formula for invalid references. Correct the references to point to valid cells, or adjust/rewrite the formula. If cells were deleted, you might need to undo the deletion or reconstruct the formula.

Question 4 (X marks)

Distinguish between slide master and notes page as used in presentation programs.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Slide Master vs. Notes Page:

  • Slide Master: A template slide that controls the formatting (fonts, colors, backgrounds, placeholders, logos) for all slides (or specific layouts) within a presentation. Changes made to the slide master are automatically applied to all associated slides, ensuring consistency.
  • Notes Page: A view in presentation software that shows a smaller version of the slide along with an area below it where the presenter can add speaker notes, reminders, or additional details relevant to that specific slide. These notes are typically not visible to the audience during the slideshow but can be printed or viewed by the presenter.

Question 6 (X marks)

(a) Explain the term hyperlink as used in Internet application programs.

(b) State the function of each of the following field properties as applied in database programs:

  1. input mask;
  2. required.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Hyperlink:

A hyperlink (or simply link) is a reference or navigation element in a document (like a webpage or word document) that points to another document, resource, or a specific location within the same document. Clicking on a hyperlink typically takes the user directly to the linked target.

(b) Database Field Property Functions:

  1. Input Mask: Defines a specific pattern for data entry, guiding the user and ensuring data consistency (e.g., formatting phone numbers or postal codes).
  2. Required: Specifies whether a value must be entered into the field for a record to be saved. If set to "Yes", the field cannot be left blank.

Question 7 (X marks)

(a) Outline four features of the graphical user interface as used in computer systems. (2 marks)

(b) Irene applied the following criteria in a query.

  1. #18/04/1998#;
  2. < date() -12 .

Describe the expected output for each of the criteria in (i) and (ii)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Features of Graphical User Interface (GUI):

  • Windows: Rectangular areas on the screen that display applications or documents, allowing multiple tasks to be viewed simultaneously.
  • Icons: Small graphical symbols representing applications, files, folders, or functions, which can be clicked to activate them.
  • Menus: Lists of commands or options presented to the user, often organized in drop-down lists or toolbars.
  • Pointer: An on-screen symbol (usually an arrow) controlled by a pointing device (like a mouse) used to select objects and interact with GUI elements.
  • Widgets/Controls: Standard interactive elements like buttons, checkboxes, radio buttons, sliders, text boxes used for user input and control.

(b) Query Criteria Output:

  1. #18/04/1998#: This criterion selects records where the date field exactly matches April 18, 1998. The '#' symbols typically denote a date literal in database queries.
  2. < date() - 12: This criterion selects records where the date field is earlier than 12 days before the current date (date() usually returns the current system date). It finds records with dates older than roughly 12 days ago.

Question 8 (X marks)

With the aid of a diagram in each case, explain the following types of database models:

(a) hierarchical; (2 marks)

(b) network. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Database Models:

(a) Hierarchical Model: Organizes data in a tree-like structure where records are linked in parent-child relationships. Each child record has only one parent, but a parent can have multiple children. Resembles an organizational chart.

     Root (Parent)
      /|\
     / | \
Child Child Child
 |     |
Child Child

(b) Network Model: An extension of the hierarchical model, allowing records (members) to have multiple parent records (owners). This creates a more flexible graph or lattice structure, representing many-to-many relationships more easily than the strict hierarchy.

 Owner A     Owner B
   / \       / \
  /   \     /   \
Member X   Member Y
  \   /     \   /
   \ /       \ /
 Owner C     Owner D

Question 9 (X marks)

Explain the circumstance under which each of the following data types would be most appropriate in a database program:

(a) auto number; (2 marks)

(b) memo.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Appropriate Circumstances for Data Types:

(a) AutoNumber: Most appropriate for a primary key field where a unique, system-generated identifier is needed for each record. It automatically increments for each new record, ensuring uniqueness and simplifying record identification (e.g., CustomerID, OrderID).

(b) Memo (or Long Text): Most appropriate for fields that need to store large amounts of alphanumeric text, exceeding the character limit of a standard text field (often 255 characters). Useful for detailed descriptions, notes, comments, or lengthy narratives where the exact length is variable and can be substantial.

Question 10 (X marks)

With the aid of an example in each case, explain the following terms as used in Internet programs:

(a) Internet Service Provider;

(b) web browser.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Internet Terms:

(a) Internet Service Provider (ISP): A company that provides individuals and organizations access to the Internet and related services. Users typically connect to the ISP's network, which in turn connects to the broader Internet.

Example: Safaricom, Zuku, Telkom Kenya, Comcast (USA), BT (UK).

(b) Web Browser: A software application used to access, retrieve, and display information resources (primarily webpages) on the World Wide Web. It interprets HTML, CSS, and JavaScript to render content.

Example: Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari.
