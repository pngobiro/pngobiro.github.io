Question 4 (X marks)

Distinguish between a single user and a multi user operating system.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Single-user vs. Multi-user Operating System:

  • Single-user OS: Designed to be used by only one person at a time on a single computer. It does not have sophisticated mechanisms for managing multiple user accounts, permissions, or concurrent access to resources (e.g., MS-DOS, older versions of Windows like 95/98).
  • Multi-user OS: Allows multiple users to access and use the computer system's resources simultaneously, often through different terminals or network connections. It includes features for user account management, security, resource allocation, and preventing interference between users (e.g., Linux, Unix, Windows Server, modern Windows/macOS).

Question 6 (X marks)

Martin has developed a website for an organization and would like to prevent the site from illegal access. Outline four ways in which he can achieve his objective.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Ways to prevent illegal website access:

  • Strong Authentication: Implement robust user authentication mechanisms (strong passwords, multi-factor authentication) for accessing sensitive areas or administrative functions.
  • Access Control and Permissions: Configure proper file and directory permissions on the web server to restrict access based on user roles. Limit administrative access strictly.
  • Firewall Implementation: Use a Web Application Firewall (WAF) to filter malicious traffic, block common attack vectors (like SQL injection, cross-site scripting), and control access based on IP addresses or geographic location.
  • Regular Security Audits and Updates: Keep the web server software, content management system (CMS), plugins, and all related components updated with the latest security patches. Conduct regular security scans and penetration testing to identify vulnerabilities.
  • HTTPS/SSL/TLS Encryption: Encrypt data transmitted between the website and users to prevent eavesdropping and man-in-the-middle attacks.

Write a formula using the If function and cell addresses that will determine the grade for the subject whose code is U001, given that a score of 60 and above is awarded grade 'A', 50 and less than 60 is awarded grade 'B' and less than 50 is awarded grade 'F'.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Formula for Grade in cell D2 (assuming Score is in C2):

=IF(C2>=60, "A", IF(C2>=50, "B", "F"))

Explanation:

  • The outer IF checks if the score in C2 is greater than or equal to 60. If TRUE, it returns "A".
  • If FALSE, the inner IF checks if the score in C2 is greater than or equal to 50. If TRUE, it returns "B".
  • If FALSE (meaning the score is less than 50), it returns "F".

Question 9 (X marks)

Azira would like to purchase a car from a vendor who is abroad. Outline four ways in which she can use the internet to achieve her goal.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Ways Azira can use the internet to purchase a car from abroad:

  • Research and Comparison: Use manufacturer websites, online car marketplaces (e.g., AutoTrader international sites), forums, and review sites to research models, specifications, pricing, and vendor reputation in the vendor's country.
  • Communication with Vendor: Use email, website contact forms, or video conferencing (like Skype, Zoom) to communicate with the vendor, ask questions, negotiate price, and discuss terms.
  • Secure Online Payment/Finance Application: Use secure online payment gateways (if offered and trusted), international bank transfers, or apply for financing online through the vendor or a third-party financial institution.
  • Arranging Inspection and Shipping: Find and hire an independent vehicle inspection service online in the vendor's location. Research and obtain quotes from international car shipping companies via their websites.
  • Checking Regulations: Use government websites (both vendor's country and home country) to check export/import regulations, taxes, duties, and homologation requirements.
