Question 1 (X marks)

State one example of a programming language used in each of the following computer generations.

  1. 1st generation;
  2. 2nd generation;
  3. 3rd generation;
  4. 4th generation.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Programming Languages by Generation:

  1. 1st Generation: Machine Language (Binary code specific to the hardware).
  2. 2nd Generation: Assembly Language (Low-level symbolic language, e.g., using mnemonics like ADD, SUB).
  3. 3rd Generation: High-Level Languages (e.g., FORTRAN, COBOL, BASIC, C, Pascal).
  4. 4th Generation: Very High-Level Languages (Often domain-specific or database-oriented, e.g., SQL, Python, Ruby, Perl).

Question 2 (4 marks)

Differentiate between a mechanical computer mouse and an optical computer mouse.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Mechanical vs. Optical Mouse:

  • Mechanical Mouse: Uses a rubber or metal ball on its underside. As the ball rolls with mouse movement, it turns internal rollers connected to sensors that detect the direction and speed of movement. Requires a clean, flat surface and periodic cleaning of the ball/rollers.
  • Optical Mouse: Uses a light source (usually an LED or laser) and a small sensor (like a tiny camera) to detect movement relative to the surface underneath. It takes rapid snapshots of the surface to track motion. Does not have moving parts like the ball, works on more surfaces, and generally requires less maintenance.

Question 3 (X marks)

Explain one function of each of the following in a computer operating system:

(a) memory management;

(b) process management.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Operating System Functions:

(a) Memory Management: Controls and coordinates the computer's main memory (RAM). It allocates memory space to processes that need it, keeps track of which parts of memory are currently being used and by which process, decides which processes (or parts of processes) to move into and out of memory, and deallocates memory when processes terminate.

(b) Process Management: Manages the execution of processes (running programs). This includes creating and deleting processes, scheduling processes for CPU time, providing mechanisms for process synchronization and communication, and handling process suspension and resumption.

Question 4 (X marks)

(a) Outline the function of each of the following in a report design of a database program:

  1. report wizard; (1 mark)
  2. report design. (1 mark)

(b) Explain the purpose of a primary key in a database. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Database Report Design Functions:

  1. Report Wizard: A guided tool that steps the user through the process of creating a report by asking questions about the desired data source, fields, grouping, sorting, layout, and style, automatically generating a basic report based on the answers.
  2. Report Design (View): A view that allows detailed manual control over the report's layout and appearance. Users can add, remove, resize, and format controls (like text boxes, labels, images), define grouping and sorting levels, add calculations, and customize headers, footers, and sections precisely.

(b) Purpose of Primary Key:

The primary key uniquely identifies each record (row) in a database table. It ensures that every record is distinct and prevents duplicate entries. It is also crucial for establishing relationships between tables, as foreign keys in other tables refer to the primary key.

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Speaker Notes: Allows the presenter to add private notes, reminders, or additional details associated with each slide. These notes are visible to the presenter (e.g., in Presenter View or on printed notes pages) but not to the audience during the main slideshow, aiding delivery.

Question 7 (X marks)

Keeping information secure over the Internet should be a priority for a company. Outline four ways in which a company can achieve this.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Ways to Secure Company Information Online:

  • Strong Authentication: Implement multi-factor authentication (MFA) and enforce strong password policies for all user accounts accessing company systems and data.
  • Data Encryption: Encrypt sensitive data both in transit (using HTTPS/SSL/TLS for websites, VPNs for remote access) and at rest (encrypting databases and storage drives).
  • Firewalls and Network Security: Use firewalls to control network traffic, intrusion detection/prevention systems (IDPS) to monitor for threats, and segment networks to limit potential breach impact.
  • Regular Software Updates and Patching: Keep operating systems, applications, and security software up-to-date with the latest security patches to fix known vulnerabilities.
  • Employee Training: Educate employees about security best practices, phishing scams, social engineering, and safe internet usage policies.

Question 8 (X marks)

Anne was working on a spreadsheet program and she got the following errors:

(a) #NULL!

(b) #N/A

Explain what would have caused each of the errors.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Spreadsheet Error Causes:

(a) #NULL! : This error usually occurs when you specify an intersection of two cell ranges that do not actually intersect, or when you use an incorrect range operator (e.g., using a space instead of a comma or colon where required by a function).

(b) #N/A : This error means "Not Available". It typically occurs in lookup functions (like VLOOKUP, HLOOKUP, MATCH) when the value being looked up cannot be found in the specified lookup range. It can also be intentionally returned by the NA() function.

Question 9 (X marks)

(a) A computer student has been asked to state the options in page set-up that would be used to customize a document in a word document. Outline two of the options she would mention. (2 marks)

(b) The Science scores of students in a class are graded A, B and C and recorded in cells B2 to B20 in a worksheet. Using the cell reference and a function, write a formula that would display the number of students who scored only A.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Page Setup Options for Customization:

  • Margins: Allows setting the amount of white space around the edges of the page (top, bottom, left, right).
  • Orientation: Allows choosing between Portrait (taller) and Landscape (wider) page layout.
  • Paper Size: Allows selecting standard paper sizes (e.g., A4, Letter, Legal) or defining custom dimensions.
  • Columns: Allows formatting text into multiple newspaper-style columns.

(b) Formula to Count "A" Grades:

=COUNTIF(B2:B20, "A")

Question 10 (X marks)

Helen used the following features in an email drafted to her friends:

  1. Attachment;
  2. Bcc .

Explain the use of each of the features.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Email Feature Uses:

  1. Attachment: Allows sending files (like documents, images, spreadsheets) along with the email message. The recipient can download and open these attached files.
  2. Bcc (Blind Carbon Copy): Allows sending a copy of the email to one or more recipients without their email addresses being visible to the primary ('To') recipients, 'Cc' recipients, or other 'Bcc' recipients. It's used for privacy or when sending to large distribution lists.
