Question 4 (X marks)

(a) Differentiate between hanging indent and first line indent as used in word processing.

(b) Faith would like to increase the width of an object inserted in a word processing document. Outline the steps she would follow to achieve her objective. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Hanging Indent vs. First Line Indent:

  • Hanging Indent: The first line of the paragraph is aligned with the left margin, while all subsequent lines are indented. Often used for bibliographies or lists.
  • First Line Indent: Only the first line of the paragraph is indented from the left margin, while the subsequent lines remain aligned with the left margin. Commonly used to mark the start of new paragraphs.

(b) Steps to Increase Object Width:

  1. Click on the inserted object to select it. Selection handles (small squares or circles) will appear around its border.
  2. Position the mouse cursor over a sizing handle on the left or right side of the object. The cursor will change to a double-headed arrow.
  3. Click and drag the handle horizontally outwards (away from the center) to increase the width. Release the mouse button when the desired width is achieved.

  2. Select consecutive objects in a folder using a mouse and the keyboard. (2 marks)

(b) Anita, a computer student used the following formula in a worksheet.

=(a1*c$1)/d$4-$f$1+b2

Identify each of the following cell references from the formula.

  1. Absolute; (1/2 mark)
  2. Mixed; (1 mark)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Windows Tasks Steps:

  1. Move folder using mouse: Right-click and hold on the folder, drag it over the destination drive icon in File Explorer, release the right mouse button, and select "Move Here" from the context menu.

Question 7 (X marks)

(a) With the aid of an example, explain the term web browser as used in the Internet. (2 marks)

(b) John would like to print three slides created in a presentation program on one page. Outline the steps he would follow to achieve his objective. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Web Browser Explanation:

A web browser is a software application used to retrieve, present, and traverse information resources on the World Wide Web. It interprets HTML, CSS, and JavaScript to display webpages, images, videos, and other content.

Example: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari.

(b) Steps to Print Three Slides per Page:

  1. Go to the "File" menu in the presentation program.
  2. Select "Print".
  3. In the Print dialog box, look for the "Settings" or "Print Layout" section.
  4. Under the "Handouts" option (or similar), select the layout that shows "3 Slides" per page (this layout often includes lines for notes next to the slides).
  5. Adjust other print settings as needed (printer, copies) and click "Print".

Question 8 (X marks)

(a) Paul saved a word document in the root directory of drive C: Outline the steps that he would follow to open the document while in the computer desktop folder.

(b) With the aid of a diagram, outline a hierarchical data model as used in databases. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Steps to Open Document from Desktop:

  1. Double-click the "Computer" or "This PC" icon on the desktop (or open File Explorer via the taskbar/Start menu).
  2. In the window that opens, navigate to the C: drive by double-clicking it in the list of drives.
  3. Locate the saved Word document file within the C: drive's root directory.
  4. Double-click the document file icon to open it in Microsoft Word (or the default associated program).

(b) Hierarchical Data Model:

A hierarchical data model organizes data in a tree-like structure. Data is represented as a collection of records connected to one another through links. Each record is a collection of fields. A record type can have only one "parent" record type but can have multiple "child" record types, forming a hierarchy.

     Root (e.g., University)
      |
      +-- Parent (e.g., Department)
      |    |
      |    +-- Child (e.g., Course)
      |    |    |
      |    |    +-- Grandchild (e.g., Student)
      |    |
      |    +-- Child (e.g., Professor)
      |
      +-- Parent (e.g., Library)
           |
           +-- Child (e.g., Book)

(a) Alex a computer teacher has given his computer students an assignment to research on the emerging trends on computer software. State two resource materials they would use to carry out the assignment. (2 marks)

(b) Phanuel intends to use the password 2013 in his email account. He has been advised by his teacher to change it. State two reasons that might have prompted the teacher to advice him to change.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Resource Materials for Research:

  • Academic Journals and Conference Proceedings (e.g., IEEE Xplore, ACM Digital Library)
  • Reputable Technology News Websites and Blogs (e.g., TechCrunch, Ars Technica, vendor blogs)
  • Online Libraries and Databases
  • Recent Textbooks on Software Engineering/Computer Science

(b) Reasons to Change Password "2013":

  • Too Short: Passwords should ideally be longer (e.g., 12+ characters) to increase complexity.
  • Easily Guessable: Using a recent year makes it predictable and vulnerable to dictionary attacks or brute-force attempts.
  • Lacks Complexity: It only contains numbers. Strong passwords should include a mix of uppercase letters, lowercase letters, numbers, and symbols.

Question 10 (X marks)

(a) Paul would like to display total values of a field named marks in a database during report generation. Write the general syntax of the formula that he would use to achieve his objective.

(b) Philip created two tables in a database program as follows.

Explain the type of relationship which he could create between the tables.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) General Syntax for Total in Database Report:

The exact syntax depends on the specific database reporting tool, but it generally looks like one of these:

  • =Sum([marks])
  • SUM(marks)
  • Total([marks])

(Where `[marks]` represents the field name containing the values to be summed).

(b) Relationship Between Tables:

A **One-to-Many** relationship could be created between the `DEPARTMENT` table and the `EMPLOYEE` table. The `Dept_ID` field in the `EMPLOYEE` table acts as a foreign key, linking each employee record to one specific department record in the `DEPARTMENT` table (using `Dept_ID` as the primary key there). One department can have multiple employees, but each employee belongs to only one department according to this structure.
