Question 1 (X marks)

(a) Outline two positive impacts of the Internet to the society.

(b) Differentiate between the terms server and client as used in Internet connectivity. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Positive Impacts of the Internet:

  • Access to Information and Education: Provides vast resources for learning, research, and staying informed on countless topics.
  • Improved Communication: Enables instant communication globally through email, messaging apps, social media, and video calls, connecting people easily.
  • Economic Opportunities: Facilitates e-commerce, online businesses, remote work, and access to global markets.

(b) Server vs. Client:

  • Server: A computer or system that provides resources, data, services, or programs to other computers, known as clients, over a network. (e.g., web server hosting a website, email server managing emails).
  • Client: A computer or application that requests and uses services or resources provided by a server. (e.g., a web browser requesting a webpage, an email application retrieving mail).

Question 3 (X marks)

(a) List two challenges that end users face while using command based user interfaces.

(b) Explain the function of the macro tool in a word processing application.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Challenges of Command-Based Interfaces:

  • Steep Learning Curve: Users must memorize specific commands and syntax, making it difficult for beginners.
  • Low Discoverability: It's hard to know what commands are available or what options they take without prior knowledge or documentation.
  • Less Intuitive: Requires abstract thinking rather than direct manipulation of visual objects (like in GUIs).
  • Error Prone: Minor typing errors in commands can lead to failure or unintended consequences.

(b) Macro Tool Function:

A macro tool allows users to record a sequence of actions or commands (like formatting steps, text insertion, menu selections) and save it under a single name or shortcut key. This recorded sequence (macro) can then be replayed automatically to perform the same set of actions repeatedly, saving time and effort on repetitive tasks.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Primary Key:

(i) The most appropriate field for a primary key is IdNo.

(ii) Reason: An ID number is typically designed to be unique for each student, ensuring that every record in the table can be uniquely identified, which is the fundamental requirement of a primary key. Names can sometimes be duplicated.

(b) Referential Integrity:

Referential integrity is a set of rules enforced by a relational database management system (RDBMS) to ensure that relationships between tables remain consistent. Specifically, it usually means that a foreign key value in one table must match an existing primary key value in the related table, or the foreign key value must be null. This prevents "orphan" records (e.g., an order record referring to a customer ID that no longer exists).

Question 8 (4 marks)

(a) Describe systems software as used in computer systems. (2 marks)

(b) Explain a circumstance under which the Blind Carbon Copy (Bcc) option could be used when sending an email. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Systems Software:

Systems software is a type of computer software designed to operate and control the computer hardware and provide a platform for running application software. It manages the fundamental operations of the computer. Major examples include the Operating System (OS), device drivers, utility software (like disk formatters, file managers), and firmware.

(b) Circumstance for Using Bcc:

Blind Carbon Copy (Bcc) is used when you want to send an email to multiple recipients without revealing the email addresses of the Bcc'd recipients to each other (or to the main 'To' and 'Cc' recipients). A common circumstance is sending a newsletter or announcement to a large list of people while protecting their privacy from each other.
