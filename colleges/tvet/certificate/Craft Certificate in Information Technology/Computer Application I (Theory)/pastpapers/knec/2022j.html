Question 1 (X marks)

Outline two characteristics of each of the following categories of computers:

(a) analog;

(b) digital;

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Analog Computers Characteristics:

  • Represent data as continuous physical quantities (e.g., voltage, pressure, temperature).
  • Process data in a continuous form, often used for simulation or measurement of physical processes.
  • Typically less precise than digital computers.
  • Examples: Slide rule, thermometer (non-digital).

(b) Digital Computers Characteristics:

  • Represent data using discrete digits (usually binary 0s and 1s).
  • Process data in discrete steps using logical operations and arithmetic.
  • Capable of high precision and complex calculations.
  • Examples: Most modern computers (desktops, laptops, smartphones).

Question 4 (X marks)

Explain two advantages of using a relational database to manage data in an organization.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Advantages of using a Relational Database:

  • Data Integrity and Consistency: Enforces rules (like primary keys, foreign keys, data types) to ensure data accuracy, reduce redundancy, and maintain consistent relationships between different pieces of information.
  • Flexibility and Scalability: Data is stored in separate tables based on logical entities, making it easier to modify the database structure, add new data types, or query information in various ways without restructuring the entire database. Tables can be easily added or expanded.
  • Data Sharing and Security: Allows multiple users to access and share data concurrently, with features to control access permissions and ensure data security based on user roles.

Question 6 (X marks)

The management of an organization realized that their computers were infected by malware. Explain two types of such malware that may have infected these computers.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Types of Malware:

  • Viruses: Malicious code that attaches itself to legitimate programs or files. When the host program is run, the virus executes, potentially replicating itself and causing harm (e.g., corrupting files, slowing the system).
  • Worms: Standalone malware that replicates itself to spread to other computers over a network, often exploiting security vulnerabilities. Unlike viruses, they don't need to attach to an existing program. They often consume bandwidth and can carry malicious payloads.
  • Trojan Horses: Malware disguised as legitimate software. Users are tricked into installing it, after which it performs malicious actions (e.g., stealing data, providing backdoor access) without the user's knowledge.
  • Ransomware: Malware that encrypts the victim's files or locks their system, demanding a ransom payment for decryption or unlocking.
  • Spyware: Malware that secretly monitors user activity, collects personal information (keystrokes, browsing habits, login credentials), and sends it to a third party.

Question 7 (X marks)

Distinguish between value axis label and category axis label chart elements as used in spreadsheet programs.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Value Axis vs. Category Axis Labels:

  • Value Axis Label (usually Y-axis): Describes the scale or units of the numerical values being plotted on that axis (e.g., "Sales ($)", "Temperature (°C)", "Count"). It helps interpret the magnitude of the data points or bars.
  • Category Axis Label (usually X-axis): Describes the categories or groups into which the data is divided and plotted along that axis (e.g., "Months", "Product Names", "Regions"). It identifies what each data point or bar represents.
