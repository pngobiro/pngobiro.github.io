Question 1 (X marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Four ways computers may be used in a hospital:

  • Patient Records Management: Storing and retrieving patient medical history, diagnoses, treatments, and billing information (Electronic Health Records - EHR).
  • Medical Imaging: Processing, storing, and viewing images from devices like X-ray, CT scanners, and MRI machines.

  • Medical Research & Data Analysis: Analyzing patient data for research purposes, tracking disease trends, and evaluating treatment effectiveness.

Question 2 (4 marks)

Distinguish between a space bar key and a back space key on a computer keyboard.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Space Bar Key vs. Backspace Key:

  • Space Bar Key: A long key typically at the bottom of the keyboard used primarily to insert a space character between words or characters.
  • Backspace Key: Usually located in the upper-right area of the main keyboard section, this key deletes the character immediately to the left of the text cursor (or deletes the selected text/object).

Question 3 (X marks)

A school administrator intends to use a database to manage school records. Outline four advantages of using the database.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Advantages of using a database for school records:

  • Centralized Data Storage: Keeps all related information (student details, grades, attendance, fees) organized in one place, making it easier to manage and access.
  • Reduced Data Redundancy: Properly designed databases minimize duplication of information, saving storage space and reducing inconsistencies.
  • Improved Data Integrity and Consistency: Database constraints (like data types, validation rules, relationships) help ensure data is accurate and consistent across the system.
  • Efficient Data Retrieval and Reporting: Allows for quick searching, sorting, filtering, and generation of various reports (e.g., class lists, grade reports, fee statements) based on specific criteria.
  • Enhanced Data Security: Databases typically offer features for user authentication and access control, allowing administrators to restrict access to sensitive information based on user roles.

Question 4 (X marks)

Explain the function of each of the following elements in a graphical user interface (GUI) operating system:

(a) dialog box;

(b) menu list.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

GUI Element Functions:

(a) Dialog Box: A temporary window that appears on screen to provide information to the user, request input, or present options for a specific task or command. It requires user interaction (e.g., clicking OK/Cancel, entering data) before the user can return to the main application window.

(b) Menu List: A list of commands or options presented to the user, typically revealed when a menu title (e.g., File, Edit, View) in a menu bar is clicked. It provides an organized way for users to access the application's features and functions.

Question 6 (X marks)

John intends to modify a page border in a document created in a word processing program. State four ways he could use to modify the border.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Ways to modify a page border:

  • Change Style: Select a different line style (e.g., solid, dashed, dotted, double line, decorative).
  • Change Color: Choose a different color for the border line(s).
  • Change Width: Adjust the thickness of the border line(s).
  • Apply to Specific Sections/Pages: Choose whether the border applies to the whole document, the current section, the first page only, or all pages except the first.
  • Adjust Distance from Text: Modify the margin or spacing between the page border and the edge of the page text.
  • Use Art Borders: Select from a gallery of graphical or artistic borders instead of simple lines.

Question 7 (X marks)

The following is a function used in a spreadsheet program. Use it to answer the question that follows.

=SUM(A1:A4)

Identify four components of the function.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Components of the function =SUM(A1:A4):

  1. Equal Sign (=): Indicates the start of a formula in the spreadsheet cell.
  2. Function Name (SUM): Specifies the built-in operation to be performed (in this case, addition).
  3. Parentheses (()): Enclose the arguments required by the function.
  4. Argument(s) (A1:A4): The input(s) the function operates on. Here, A1:A4 represents a cell range, specifying the cells whose values should be summed. The colon (:) indicates a range from the starting cell (A1) to the ending cell (A4).

Question 9 (4 marks)

A lab technician noted that one of the computers was operating abnormally and diagnosed the cause to be malware infection. Outline four likely abnormal operations of the computer.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Likely abnormal computer operations due to malware:

  • Slow Performance: The computer runs significantly slower than usual, applications take longer to load or respond.
  • Frequent Crashes or Freezes: The system or specific applications crash, freeze, or become unresponsive unexpectedly.
  • Unwanted Pop-ups or Advertisements: Excessive pop-up ads appear, especially in web browsers, even when not browsing specific sites.
  • Unexpected System Changes: Browser homepage or search engine changes without user action, new toolbars or icons appear, system settings are modified.
  • Unusual Network Activity: High network traffic even when the user is not actively using the internet, indicating potential data exfiltration or botnet activity.
  • Disabled Security Software: Antivirus or firewall programs are disabled and cannot be re-enabled easily.

Question 10 (4 marks)

Organisations have embraced the use of cloud computing in conducting their businesses.

(a) Explain the term cloud computing. (2 marks)

(b) State two advantages that organisations would gain from this technology.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Cloud Computing:

Cloud computing refers to the delivery of computing services—including servers, storage, databases, networking, software, analytics, and intelligence—over the Internet ("the cloud") on a pay-as-you-go basis. Instead of owning and maintaining physical data centers and servers, organizations can access technology services from a cloud provider as needed.

(b) Advantages of Cloud Computing for Organizations:

  • Cost Savings: Reduces capital expenditure on hardware and infrastructure; organizations pay only for the resources they consume (operational expenditure), often leading to lower overall IT costs.
  • Scalability and Flexibility: Allows organizations to easily scale computing resources up or down based on demand, providing flexibility and agility without needing to procure and provision physical hardware in advance.
  • Accessibility and Collaboration: Services and data stored in the cloud can be accessed from anywhere with an internet connection, facilitating remote work and collaboration among employees.
