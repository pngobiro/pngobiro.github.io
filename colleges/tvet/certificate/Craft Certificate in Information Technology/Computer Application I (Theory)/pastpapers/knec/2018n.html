Question 1 (X marks)

(a) State the effect of applying each of the following mouse techniques to a computer folder:

  1. right click;
  2. double click.

(b) Most computer users opt for operating systems that offer a graphical user interface. Outline two benefits they get from this interface. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Mouse Technique Effects on Folder:

  1. Right Click: Opens a context-sensitive menu displaying actions relevant to the folder (e.g., Open, Copy, Paste, Rename, Delete, Properties).
  2. Double Click: Opens the folder, displaying its contents (files and subfolders).

(b) Benefits of Graphical User Interface (GUI):

  • Ease of Use/Intuitiveness: GUIs use visual elements (icons, menus, windows) that are generally easier for users, especially beginners, to understand and interact with compared to memorizing commands.
  • Discoverability: Features and options are often visible in menus or toolbars, making it easier for users to discover and learn the software's capabilities without needing extensive documentation.
  • Visual Feedback: Provides immediate visual confirmation of actions (e.g., a file icon moving when dragged).

Question 2 (4 marks)

(a) Moses has prepared a word processing document that has four paragraphs. He intends to move the first paragraph to the end of the document. Describe the feature he would use to achieve his objectives. (2 marks)

(b) Explain the purpose of the section break feature in a word processing program. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Feature to Move Paragraph:

Moses would use the Cut and Paste feature. He would select the first paragraph, use the 'Cut' command (Ctrl+X or via menu/toolbar), move the cursor to the end of the document, and then use the 'Paste' command (Ctrl+V or via menu/toolbar).

(b) Purpose of Section Break:

A section break divides a document into sections, allowing different formatting (such as page orientation, margins, columns, headers/footers, page numbering) to be applied to each section independently. This provides more layout control than simple page breaks.

Question 4 (X marks)

(a) John intends to use a macro to apply the same formats to many worksheets in a workbook. Explain one benefit that he would from using this tool.

(b) Differentiate between sorting and filtering data as used in spreadsheet program.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Benefit of Using Macro for Formatting:

Using a macro saves significant time and reduces the chance of errors when applying the same complex or multi-step formatting sequence repeatedly across many worksheets. It ensures consistency with a single click or shortcut key execution.

(b) Sorting vs. Filtering:

  • Sorting: Arranges the rows (records) in a data range based on the values in one or more columns, either in ascending (A-Z, smallest to largest) or descending (Z-A, largest to smallest) order. All rows are still visible, just reordered.
  • Filtering: Hides rows that do not meet specified criteria, displaying only the subset of data that matches the conditions. It does not change the order of the displayed rows but reduces the number of visible rows.

Question 5 (4 marks)

(a) Explain the function of the freeze pane feature in a spreadsheet program (2 marks)

(b) Joan wishes to have a column containing formulas in a worksheet not visible to users. Explain a spreadsheet program feature that she could use to achieve this.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Freeze Panes Function:

The freeze panes feature keeps specific rows (usually top header rows) and/or columns (usually leftmost identifier columns) visible on the screen while scrolling through the rest of the worksheet. This helps keep headings or key identifiers in view when working with large datasets.

(b) Feature to Hide Formula Column:

Joan could use the Hide Column feature. She would right-click on the column letter containing the formulas and select 'Hide' from the context menu. The column will disappear from view but its data remains and can still be used in calculations. It can be unhidden later if needed.

Question 6 (X marks)

(a) Explain one benefit of using forms in a database application.

(b) Explain the term criteria as used in database querying.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Benefit of Using Forms in Database:

Forms provide a user-friendly interface for entering, viewing, and editing data in tables, often presenting one record at a time. This makes data interaction easier and less error-prone compared to working directly in the table's datasheet view, especially for non-technical users.

(b) Criteria in Database Querying:

Criteria are conditions or rules specified in a query to filter records. Only records that meet the defined criteria will be included in the query results. Criteria can be simple (e.g., `City = "Nairobi"`) or complex involving multiple fields, comparison operators (`>`, `<`, `=`), logical operators (AND, OR, NOT), and wildcards.

Question 7 (4 marks)

(a) Big Deal company has opted to use an online word processing program rather than the traditional offline word processors. State two benefits the company could get from using this approach. (2 marks)

(b) Janet wishes to have her personal computer connected to the Internet. Outline two hardware devices required for her to achieve her objective.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Benefits of Online Word Processing:

  • Accessibility and Collaboration: Documents can be accessed from any device with an internet connection, and multiple users can often collaborate on the same document simultaneously in real-time.
  • Automatic Saving and Version History: Changes are typically saved automatically to the cloud, reducing the risk of data loss. Version history often allows reverting to previous states of the document.
  • No Installation/Updates Required: Runs in a web browser, eliminating the need for software installation or manual updates on individual computers.

(b) Hardware for Internet Connection:

  • Modem: Connects the home/office network to the Internet Service Provider (ISP) over the incoming line (e.g., DSL, Cable, Fiber).
  • Router: Directs traffic between the local network devices and the modem/internet. Often includes Wi-Fi capability to allow wireless connections. (Sometimes combined with the modem in a single device).
  • Network Interface Card (NIC) / Wi-Fi Adapter: The component within the computer (or plugged in via USB) that allows it to connect to the router, either via an Ethernet cable (NIC) or wirelessly (Wi-Fi adapter).

Question 8 (X marks)

Jean is going to present her project using equipment provided by the college.

(i) Explain why she may need to package her presentation.

(ii) Outline two ways she would make use of the slide sorter view while preparing her presentation.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(i) Reason to Package Presentation:

Packaging a presentation (using a feature like "Package for CD" or similar) ensures that all linked files (like videos, audio clips, non-standard fonts) are included along with the presentation file itself. This prevents issues where linked media might be missing or fonts might display incorrectly when the presentation is opened on a different computer (like the college's equipment) that doesn't have those specific files or fonts installed.

(ii) Uses of Slide Sorter View:

  • Rearranging Slides: Easily change the order of slides by dragging and dropping their thumbnails to improve the presentation flow.
  • Applying Transitions/Timings: View and apply transition effects or set timings for multiple slides simultaneously.
  • Getting an Overview: See a miniature view of all slides at once to get a sense of the overall structure and visual consistency of the presentation.
  • Organizing into Sections: Some versions allow creating logical sections within the presentation in this view.

(b) Hierarchical Database Model:

Organizes data in a tree-like structure where records are linked in parent-child relationships. Each child record has only one parent, but a parent can have multiple children. Data access typically follows the defined paths from parent to child.

     Root (e.g., Department)
      |
      +-- Child (e.g., Course)
      |    |
      |    +-- Grandchild (e.g., Student)
      |
