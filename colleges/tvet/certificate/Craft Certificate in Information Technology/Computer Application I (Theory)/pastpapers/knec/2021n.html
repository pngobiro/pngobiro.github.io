Question 2 (X marks)

Differentiate between a normal layout view and slide sorter view as applied in a presentation program.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Normal Layout View vs. Slide Sorter View:

  • Normal Layout View: This is the primary editing view. It typically displays the current slide large, along with smaller panes for slide thumbnails/outline and speaker notes. It's used for detailed content creation and editing on individual slides.
  • Slide Sorter View: This view displays thumbnails (miniature images) of all slides in the presentation in a grid format. It's primarily used for getting an overview of the presentation flow, easily rearranging the order of slides, and applying transitions or timings to multiple slides at once.

Question 3 (X marks)

Explain how computer operating system performs each of the following functions:

(a) memory management;

(b) Input/output operations.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Operating System Functions:

(a) Memory Management: The OS manages the computer's primary memory (RAM) by allocating blocks of memory to running processes, keeping track of used and free memory portions, deciding which processes get memory when, and deallocating memory when processes terminate to prevent conflicts and ensure efficient memory utilization.

(b) Input/Output Operations: The OS manages communication between the computer and its peripheral devices (like keyboard, mouse, printer, disk drives). It handles device requests from processes, loads appropriate device drivers, sends commands to devices, receives interrupts, and manages data transfer to/from devices, abstracting the hardware complexity from applications.

Question 4 (X marks)

(a) State the type of database models for each of the following descriptions:

  1. data is organised in two-dimensional tables and the relationship is maintained by storing a common field.
  2. data is organised more like a graph, and is allowed to have more than one parent node.

(b) Outline the purpose of a foreign key in a database program. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Database Model Types:

  1. Relational Model: Data is organized in two-dimensional tables (relations) with rows and columns, and relationships are maintained by storing common fields (foreign keys).
  2. Network Model: Data is organized as a graph where records (nodes) can have multiple parent and child nodes, allowing for more complex many-to-many relationships than the hierarchical model.

(b) Purpose of Foreign Key:

A foreign key is a field (or collection of fields) in one table that uniquely identifies a row of another table (or the same table). Its purpose is to establish and enforce a link between the data in two tables, ensuring referential integrity â€“ meaning that a value in the foreign key column must correspond to an existing value in the primary key column of the related table.

Question 5 (X marks)

Joy used the following features while creating documents using computer programs:

(a) Word wrap;

(b) Text wrap.

Explain the function of each.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Feature Functions:

(a) Word Wrap: Automatically moves text to the beginning of the next line when it reaches the right margin of the document or container, eliminating the need to press Enter at the end of each line within a paragraph.

(b) Text Wrap: Controls how text flows around an object (like an image, shape, or textbox) that is inserted into the document. Options typically include square, tight, through, top and bottom, behind text, and in front of text, allowing text to wrap around the object's boundaries or flow over/under it.

Question 6 (4 marks)

Master slides are used in preparation of presentations. Explain two advantages that are realized from the use of this feature.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Advantages of using Master Slides:

  • Consistency: Ensures a uniform look and feel across all slides (or slides using a specific layout) by applying consistent formatting elements like fonts, colors, backgrounds, logos, and placeholder positions defined in the master.
  • Efficiency: Allows making global changes easily. Modifying an element on the slide master automatically updates all slides based on that master, saving significant time compared to editing each slide individually.

Question 7 (X marks)

Marion was advised to enhance network security by use of firewalls on her computer. Explain two reasons for this advice.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Reasons for using Firewalls:

  • Blocking Unauthorized Access: Firewalls act as a barrier between Marion's computer (or internal network) and the external network (Internet), inspecting incoming and outgoing traffic and blocking connections that don't meet predefined security rules, thus preventing unauthorized access attempts or intrusions.
  • Filtering Malicious Traffic: Firewalls can be configured to identify and block known malicious traffic patterns, prevent access to harmful websites, or stop certain types of malware or network attacks from reaching her computer.

Question 9 (X marks)

A technical institute connected their computers to the Internet. Explain two negative effects this connection would have.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Negative effects of internet connection for a technical institute:

  • Security Risks: Increased exposure to malware, viruses, phishing attacks, and potential unauthorized access to the institute's network and sensitive data (student records, financial information).
  • Distraction and Misuse: Students and staff might use the internet connection for non-educational purposes (social media, gaming, entertainment) during study or work time, reducing productivity and focus. Access to inappropriate content is also a concern.
  • Bandwidth Costs and Management: Providing reliable internet access to many users can incur significant costs, and managing bandwidth usage to ensure fair access for educational purposes can be challenging.
