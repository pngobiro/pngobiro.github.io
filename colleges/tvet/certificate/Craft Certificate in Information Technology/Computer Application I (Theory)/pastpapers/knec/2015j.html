Question 2 (X marks)

(a) Outline the function of each of the following keyboard keys:

  1. F10;
  2. Escape.

(b) Explain the term conditional formatting as used in spreadsheet applications.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Keyboard Key Functions:

  1. F10: Often activates the menu bar in many applications. In some BIOS/UEFI setups, it's used to save settings and exit. Shift+F10 often opens the context menu (like a right-click).
  2. Escape (Esc): Used to cancel or abort the current operation, close dialog boxes, or exit certain modes (like full-screen).

(b) Conditional Formatting:

Conditional formatting is a feature in spreadsheet applications that allows users to automatically apply specific formatting (like colors, icons, or data bars) to cells based on whether their values meet certain criteria or rules. This helps to visually highlight important data, trends, or outliers.

Question 3 (X marks)

Distinguish between auto complete and auto correct as used in word processing applications.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Autocomplete vs. Autocorrect:

  • Autocomplete: Suggests possible completions for words or phrases as the user types, based on common usage or previous entries. The user can accept the suggestion (e.g., by pressing Tab or Enter) to save typing.
  • Autocorrect: Automatically corrects common spelling mistakes, typos, or capitalization errors as the user types, replacing the incorrect text with the presumed correct version based on a predefined dictionary.

Question 4 (X marks)

With the aid of an example in each case, explain the following types of cell references as used in spreadsheet programs:

  1. absolute;
  2. relative.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Cell Reference Types:

  1. Absolute Reference: A cell reference that remains fixed when a formula is copied to other cells. Both the column letter and row number are preceded by a dollar sign ($). Example: $A$1. If a formula containing $A$1 is copied, it will always refer to cell A1.
  2. Relative Reference: A cell reference that adjusts based on its position relative to the cell where the formula is copied. Neither the column letter nor the row number has a dollar sign. Example: A1. If a formula containing A1 is copied one cell down, the reference will change to A2.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Database Field Property Functions:

  1. Input Mask: Defines a specific pattern or format for data entry into a field. It guides the user by showing placeholders and restricting input to match the required format (e.g., for phone numbers like (___) ___-____ or dates like dd/mm/yyyy).

Question 7 (X marks)

With the aid of a diagram, describe each of the following types of relationships as applied in database applications:

  1. one-to-many;
  2. many-to-many.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Database Relationship Types:

  1. One-to-Many: A relationship where one record in Table A can be related to multiple records in Table B, but each record in Table B is related to only one record in Table A. 
    Table A (e.g., Departments)   Table B (e.g., Employees)
+------------+                +-------------+
| DeptID (PK)|-------<--------| EmployeeID (PK)|
| DeptName   |                | EmpName     |
+------------+                | DeptID (FK) |
                             +-------------+
(One Dept has Many Employees)
  2. Many-to-Many: A relationship where one record in Table A can be related to multiple records in Table B, and one record in Table B can also be related to multiple records in Table A. This typically requires a third "junction" or "linking" table. 
    Table A (e.g., Students)    Junction Table (Enrollment)   Table B (e.g., Courses)
+-------------+             +----------------+            +-------------+
| StudentID(PK)|----->-------| EnrollmentID(PK)|------<----| CourseID(PK)|
| StudentName |             | StudentID (FK) |            | CourseName  |
+-------------+             | CourseID (FK)  |            +-------------+
                           +----------------+
(One Student takes Many Courses, One Course has Many Students)

Question 8 (X marks)

Kirundi secondary school intends to introduce the use of the Internet to its students. Explain two negative social security issues that the students are likely to be exposed to.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Negative Social Security Issues from Internet Use:

  • Cyberbullying: Students may be exposed to harassment, threats, or humiliation online through social media, messaging apps, or gaming platforms, impacting their mental well-being and safety.
  • Exposure to Inappropriate Content: Students might accidentally or intentionally encounter content that is violent, sexually explicit, hateful, or promotes dangerous activities, which can be harmful to their development.

Question 9 (X marks)

(a) Moses intends to print a 20 slide presentation and produce four slides per page. Explain the feature that he could use to achieve this objective.

(b) Wireless fidelity (Wi-Fi) has become one of the most popular modes of Internet connection in recent days. Explain two reasons for this trend.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Printing Multiple Slides per Page (Handouts):

Moses should use the "Handouts" feature in the print settings of the presentation software. This feature allows specifying how many slides (e.g., 1, 2, 3, 4, 6, 9) should be printed on each page, often arranging them in a grid format. Selecting the "4 slides per page" layout under Handouts will achieve his objective.

(b) Reasons for Wi-Fi Popularity:

  • Convenience and Mobility: Wi-Fi allows devices to connect to the internet wirelessly within the network's range, eliminating the need for physical cables and enabling users to move around freely with laptops, smartphones, and tablets while staying connected.
  • Ease of Setup and Cost-Effectiveness (for multiple devices): Setting up a Wi-Fi network is relatively straightforward for homes and small businesses. It allows multiple devices to share a single internet connection without requiring extensive cabling for each device, making it cost-effective compared to wired solutions for numerous users.

Question 10 (X marks)

(a) Explain the function of format painter feature as applied in computer applications.

(b) Explain the circumstance under which hide slide feature would be used in a presentation.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Format Painter Function:

The format painter feature allows a user to copy the formatting (like font style, size, color, alignment, borders, etc.) from selected text or an object and quickly apply the exact same formatting to other text or objects. It's a tool for ensuring consistency and saving time when applying complex formatting multiple times.

(b) Circumstance for Using Hide Slide Feature:

The hide slide feature is used when you want to keep a slide in your presentation file but not show it during the main slideshow presentation. This is useful for:

  • Including backup slides with additional details that you might only show if specific questions arise.
  • Temporarily removing a slide from the flow without deleting it.
  • Creating different versions of a presentation for different audiences while keeping all slides in one file.
