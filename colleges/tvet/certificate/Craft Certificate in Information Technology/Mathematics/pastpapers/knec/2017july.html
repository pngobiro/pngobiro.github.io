Question 1 (4 marks)

Define each of the following terms as used in matrices:

(i) diagonal matrix;

(ii) identity matrix.

Answer

(i) Diagonal Matrix: A square matrix in which all the entries outside the main diagonal are zero. The entries on the main diagonal can be any value, including zero.

(ii) Identity Matrix: A square matrix in which all the elements of the main diagonal are ones and all other elements are zeros. It is denoted by \(I\).

Question 2 (4 marks)

Describe each of the following number systems:

(i) hexadecimal;

(ii) octal.

Answer

(i) Hexadecimal: A base-16 number system that uses sixteen distinct symbols: 0-9 to represent values zero to nine, and A, B, C, D, E, F (or a-f) to represent values ten to fifteen. It is commonly used in computing as a more human-friendly representation of binary-coded values.

(ii) Octal: A base-8 number system that uses the digits 0 to 7. Each digit position represents a power of 8. It was previously used in computing but is less common now compared to hexadecimal.

Question 3 (4 marks)

Outline four properties of Poisson probability distribution.

Answer

Four properties of Poisson probability distribution:

  • The events occur independently. The occurrence of one event does not affect the probability of another event occurring.
  • The average rate (mean number of occurrences, denoted by Î») at which events occur is constant for the interval of interest.
  • The probability of an event occurring is proportional to the length of the interval.
  • The probability of more than one event occurring in a very small interval is negligible (close to zero).

Question 4 (4 marks)

Explain each of the following terms as used in statistics:

(i) discrete random variable;

(ii) continuous random variable.

Answer

(i) Discrete Random Variable: A variable whose value is obtained by counting and can only take a finite or countably infinite number of distinct values. Examples include the number of students in a class, the number of cars passing a point in an hour, or the number of defective items in a batch.

(ii) Continuous Random Variable: A variable whose value is obtained by measuring and can take any value within a given range or interval. Examples include height, weight, temperature, or the time taken to complete a task.

Question 5 (4 marks)

Distinguish between explicit function and quadratic function as used in mathematics.

Answer

Explicit Function: An explicit function is one where the dependent variable (usually \(y\)) is expressed directly in terms of the independent variable (usually \(x\)). The relationship is written in the form \(y = f(x)\). For example, \(y = 3x^2 + 2x - 1\) is an explicit function.

Quadratic Function: A quadratic function is a specific type of polynomial function of degree two. Its general form is \(f(x) = ax^2 + bx + c\), where \(a\), \(b\), and \(c\) are constants and \(a \neq 0\). While a quadratic function *can* be written explicitly (as shown in the general form), the term "quadratic" refers specifically to the degree-two nature of the polynomial, whereas "explicit" refers to how the relationship between variables is expressed (dependent variable isolated).

Question 7 (4 marks)

Using binomial theorem, expand the expression \(\left(x^{2}+3\right)^{6}\) up to the fourth term in descending powers of \(x\).

Answer

The binomial expansion of \((a+b)^n\) is given by:

\[ (a+b)^n = \binom{n}{0}a^n b^0 + \binom{n}{1}a^{n-1}b^1 + \binom{n}{2}a^{n-2}b^2 + \binom{n}{3}a^{n-3}b^3 + \dots \]

In this case, \(a = x^2\), \(b = 3\), and \(n = 6\).

We need the first four terms:

Term 1: \(\binom{6}{0}(x^2)^6 (3)^0 = 1 \cdot x^{12} \cdot 1 = x^{12}\)

Term 2: \(\binom{6}{1}(x^2)^{6-1} (3)^1 = 6 \cdot (x^2)^5 \cdot 3 = 6 \cdot x^{10} \cdot 3 = 18x^{10}\)

Term 3: \(\binom{6}{2}(x^2)^{6-2} (3)^2 = \frac{6 \times 5}{2 \times 1} \cdot (x^2)^4 \cdot 9 = 15 \cdot x^8 \cdot 9 = 135x^8\)

Term 4: \(\binom{6}{3}(x^2)^{6-3} (3)^3 = \frac{6 \times 5 \times 4}{3 \times 2 \times 1} \cdot (x^2)^3 \cdot 27 = 20 \cdot x^6 \cdot 27 = 540x^6\)

Therefore, the expansion up to the fourth term in descending powers of \(x\) is:

\[ \left(x^{2}+3\right)^{6} = x^{12} + 18x^{10} + 135x^8 + 540x^6 + \dots \]

Question 8 (4 marks)

(a) Recently, multimedia systems have become very popular among college students. Outline two ways students could use these systems to improve their mathematical skills. (2 marks)

(b) The quadratic equation \(2 x^{2}+(p+1) x+q-2=0\) has its roots as -3 and \(1 / 2\). Determine the value of \(p\) and \(q\) in the equation. (2 marks)

Answer

(a) Using Multimedia Systems for Math Skills:

  • Interactive Simulations and Visualizations: Multimedia systems can offer interactive simulations (e.g., graphing tools, geometry software) that allow students to visualize abstract mathematical concepts, manipulate variables, and see the immediate effects, leading to deeper understanding.
  • Online Tutorials and Educational Videos: Platforms like Khan Academy, YouTube, and specialized educational websites provide video lessons, step-by-step tutorials, and practice exercises covering various math topics, catering to different learning paces and styles.

(b) Finding p and q:

For a quadratic equation \(ax^2 + bx + c = 0\) with roots \(\alpha\) and \(\beta\):

  • Sum of roots: \(\alpha + \beta = -b/a\)
  • Product of roots: \(\alpha \beta = c/a\)

In the given equation \(2 x^{2}+(p+1) x+(q-2)=0\), we have \(a=2\), \(b=(p+1)\), and \(c=(q-2)\). The roots are \(\alpha = -3\) and \(\beta = 1/2\).

Sum of roots:

\[ -3 + \frac{1}{2} = -\frac{(p+1)}{2} \]

\[ -\frac{5}{2} = -\frac{p+1}{2} \]

\[ 5 = p+1 \]

\[ p = 4 \]

Product of roots:

\[ (-3) \times \frac{1}{2} = \frac{q-2}{2} \]

\[ -\frac{3}{2} = \frac{q-2}{2} \]

\[ -3 = q-2 \]

\[ q = -1 \]

Therefore, the values are \(p=4\) and \(q=-1\).

\[ \det(D) = 4(-12) - 1(-16) + 2(-14) \]

\[ \det(D) = -48 + 16 - 28 = -60 \]

Since \(\det(D) \neq 0\), the inverse exists.

\[ C = \begin{bmatrix} +(-12) & -(-16) & +(-14) \\ -(-6) & +(12) & -(18) \\ +(0) & -(20) & +(10) \end{bmatrix} = \begin{bmatrix} -12 & 16 & -14 \\ 6 & 12 & -18 \\ 0 & -20 & 10 \end{bmatrix} \]

\[ D^{-1} = \frac{1}{-60} \begin{bmatrix} -12 & 6 & 0 \\ 16 & 12 & -20 \\ -14 & -18 & 10 \end{bmatrix} = \begin{bmatrix} \frac{-12}{-60} & \frac{6}{-60} & \frac{0}{-60} \\ \frac{16}{-60} & \frac{12}{-60} & \frac{-20}{-60} \\ \frac{-14}{-60} & \frac{-18}{-60} & \frac{10}{-60} \end{bmatrix} = \begin{bmatrix} \frac{1}{5} & -\frac{1}{10} & 0 \\ -\frac{4}{15} & -\frac{1}{5} & \frac{1}{3} \\ \frac{7}{30} & \frac{3}{10} & -\frac{1}{6} \end{bmatrix} \]

Therefore, the inverse matrix is:

\[ D^{-1} = \begin{bmatrix} 1/5 & -1/10 & 0 \\ -4/15 & -1/5 & 1/3 \\ 7/30 & 3/10 & -1/6 \end{bmatrix} \]

Answer

(a) Coding Systems:

(i) EBCDIC (Extended Binary Coded Decimal Interchange Code):

  • Explanation: An 8-bit character encoding developed by IBM, primarily used on their mainframe and mid-range computer systems (like z/OS, OS/400). It is an extension of the earlier BCD code.
  • Format: Each character is represented by 8 bits (1 byte). The 8 bits are often divided into two 4-bit nibbles: the first 4 bits (zone bits) generally define the category of the character (e.g., lowercase, uppercase, number, special character), and the last 4 bits (digit bits) identify the specific character within that category. Unlike ASCII, the letter sequences are not contiguous.

(ii) ASCII (American Standard Code for Information Interchange):

  • Explanation: A character encoding standard for electronic communication. It is the most common character encoding format for text files in computers and on the internet.
  • Format: Standard ASCII uses 7 bits to represent 128 characters (0-127), including control characters, numbers (0-9), uppercase letters (A-Z), lowercase letters (a-z), and punctuation symbols. Extended ASCII uses 8 bits (1 byte) to represent 256 characters, including the standard 128 ASCII characters plus additional symbols, accented characters, and graphical symbols. The first bit (most significant bit) is 0 for standard ASCII and often 1 for extended characters.

(b) Matrix Multiplication Possibility:

For matrix multiplication \(\mathbf{XY}\) to be possible, the number of columns in matrix \(\mathbf{X}\) must be equal to the number of rows in matrix \(\mathbf{Y}\).

  • A is \(3 \times 4\)
  • B is \(4 \times 5\)
  • C is \(4 \times 4\)

(i) AB: A has 4 columns, B has 4 rows. Since the number of columns in A equals the number of rows in B (4 = 4), the operation \(\mathbf{AB}\) is possible. The resulting matrix will have dimensions \(3 \times 5\).

(ii) BC: B has 5 columns, C has 4 rows. Since the number of columns in B (5) is not equal to the number of rows in C (4), the operation \(\mathbf{BC}\) is not possible.

(iii) CA: C has 4 columns, A has 3 rows. Since the number of columns in C (4) is not equal to the number of rows in A (3), the operation \(\mathbf{CA}\) is not possible.
