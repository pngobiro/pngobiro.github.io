Question 1

(a) (i) Outline the format of Extended Binary Coded Decimal (EBCD) code.

(ii) Convert \(673_{10}\) to its BCD equivalent.

(b) State the binomial theorem (assume n is a positive integer).

Answer

(a)(i) Extended Binary Coded Decimal (EBCD) code format:

  • Uses 8 bits to represent each decimal digit
  • First 4 bits (zone bits): usually 1111
  • Last 4 bits: BCD representation of the decimal digit
  • Example: Decimal 5 = 1111 0101 in EBCD

(a)(ii) Converting \(673_{10}\) to BCD:

  • 6 = 0110 in BCD
  • 7 = 0111 in BCD
  • 3 = 0011 in BCD

Therefore, \(673_{10}\) in BCD is: 0110 0111 0011

(b) The Binomial Theorem states that:

For any positive integer n and any real numbers x and y:

\[(x + y)^n = \sum_{k=0}^n \binom{n}{k} x^{n-k} y^k\]

where \(\binom{n}{k}\) is the binomial coefficient given by:

\[\binom{n}{k} = \frac{n!}{k!(n-k)!}\]

Question 2

Using the substitution method determine the value of \(x\) and \(y\) in the following simultaneous equations:

\[\begin{aligned} & x-2y=4 \\ & x^2+xy+y=1 \end{aligned}\]

Answer

Let's solve this step by step:

Given equations:

\[\begin{aligned} & x-2y=4 \quad (1) \\ & x^2+xy+y=1 \quad (2) \end{aligned}\]

1. From equation (1): \(x = 4 + 2y\)

2. Substitute into equation (2):

\((4 + 2y)^2 + (4 + 2y)(y) + y = 1\)

3. Expand:

\(16 + 16y + 4y^2 + 4y + 2y^2 + y = 1\)

4. Collect like terms:

\(6y^2 + 21y + 16 = 1\)

\(6y^2 + 21y + 15 = 0\)

5. Divide by 3 to simplify:

\(2y^2 + 7y + 5 = 0\)

6. Use quadratic formula: \(y = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)

\[y = \frac{-7 \pm \sqrt{7^2 - 4(2)(5)}}{2(2)} = \frac{-7 \pm \sqrt{49 - 40}}{4} = \frac{-7 \pm \sqrt{9}}{4} = \frac{-7 \pm 3}{4}\]

7. Therefore: \(y = \frac{-7 + 3}{4} = \frac{-4}{4} = -1\) or \(y = \frac{-7 - 3}{4} = \frac{-10}{4} = -2.5\)

8. Substitute back to find x:

When \(y = -1\): \(x = 4 + 2(-1) = 4 - 2 = 2\)

When \(y = -2.5\): \(x = 4 + 2(-2.5) = 4 - 5 = -1\)

The solutions are:

  • Solution 1: \(x = 2, y = -1\)
  • Solution 2: \(x = -1, y = -2.5\)

Question 3

(a) Determine the equation of the line passing through the points (1,3) and (3,7).

(b) Using a sketch, describe a Venn diagram as used in set theory.

Answer

(a) Equation of the line passing through (1,3) and (3,7):

1. Calculate the slope (m):

\[m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{7 - 3}{3 - 1} = \frac{4}{2} = 2\]

2. Use the point-slope form \(y - y_1 = m(x - x_1)\) with point (1,3):

\(y - 3 = 2(x - 1)\)

\(y - 3 = 2x - 2\)

\(y = 2x + 1\)

The equation of the line is \(y = 2x + 1\).

(b) Venn Diagram Description:

A Venn diagram is a visual representation of sets and their relationships.

  • A rectangle represents the universal set (U), containing all elements under consideration.
  • Circles or ovals inside the rectangle represent individual sets.
  • Overlapping areas between circles show the intersection of sets (elements common to both).
  • The entire area covered by the circles represents the union of sets (all elements in either set).
  • The area outside a circle but inside the rectangle represents the complement of that set.

(A sketch would typically be drawn here showing these components)

Question 4

Differentiate between a finite set and an infinite set as used in set theory.

Answer

Finite Set:

  • A set that contains a specific, countable number of elements.
  • The process of counting its elements can, in principle, come to an end.
  • Example: The set of days in a week, \(D = \{\text{Monday, Tuesday, ..., Sunday}\}\). The number of elements is 7.
  • Example: The set of letters in the English alphabet, \(A = \{a, b, c, ..., z\}\). The number of elements is 26.

Infinite Set:

  • A set that contains an unlimited number of elements.
  • The process of counting its elements never ends.
  • Example: The set of natural numbers, \(N = \{1, 2, 3, ...\}\).
  • Example: The set of real numbers between 0 and 1, \((0, 1)\).

Question 5

A bag contains three green and five red balls. A second bag contains four green and six red balls. Two balls are drawn at random one from each bag. Determine the probability that:

(i) both are green;

(ii) they are of different colours.

Answer

Let G1, R1 be drawing Green, Red from Bag 1. Let G2, R2 be drawing Green, Red from Bag 2.

Bag 1: 3 Green, 5 Red (Total 8 balls)

Bag 2: 4 Green, 6 Red (Total 10 balls)

Probabilities for Bag 1: \(P(G1) = \frac{3}{8}\), \(P(R1) = \frac{5}{8}\)

Probabilities for Bag 2: \(P(G2) = \frac{4}{10}\), \(P(R2) = \frac{6}{10}\)

(i) Probability that both are green (G1 and G2):

Since the draws are independent:

\[P(G1 \cap G2) = P(G1) \times P(G2) = \frac{3}{8} \times \frac{4}{10} = \frac{12}{80} = \frac{3}{20}\]

(ii) Probability they are of different colours:

This can happen in two ways: (Green from Bag 1 AND Red from Bag 2) OR (Red from Bag 1 AND Green from Bag 2).

Case 1: G1 and R2

\[P(G1 \cap R2) = P(G1) \times P(R2) = \frac{3}{8} \times \frac{6}{10} = \frac{18}{80}\]

Case 2: R1 and G2

\[P(R1 \cap G2) = P(R1) \times P(G2) = \frac{5}{8} \times \frac{4}{10} = \frac{20}{80}\]

Total probability of different colours = P(G1 ∩ R2) + P(R1 ∩ G2)

\[= \frac{18}{80} + \frac{20}{80} = \frac{38}{80} = \frac{19}{40}\]

Question 6

Explain each of the following terms giving an example in each case.

(i) discrete variable;

(ii) continuous variable.

Answer

(i) Discrete Variable:

  • A variable that can only take specific, separate values
  • There are distinct gaps between possible values
  • Examples:
    • Number of students in a class (can only be whole numbers)
    • Number of cars in a parking lot
    • Shoe sizes (comes in specific increments)

(ii) Continuous Variable:

  • A variable that can take any value within a continuous range
  • Can be measured to any degree of precision
  • Examples:
    • Height (can be any value like 1.72348... meters)
    • Time (can be measured to any precision)
    • Temperature (can take any value within its range)

Question 7

Convert each of the following number to the respective number systems showing your working;

(i) \(\mathrm{DE}2\mathrm{FA}_{16}\) to its octal equivalent;

(ii) \(360_{8}\) to its hexadecimal equivalent.

Answer

(i) Converting DE2FA₁₆ to octal:

  1. First convert each hexadecimal digit to 4-bit binary:
    • D = 1101
    • E = 1110
    • 2 = 0010
    • F = 1111
    • A = 1010
  2. Combine binary digits: 1101 1110 0010 1111 1010
  3. Group into sets of three from right: 001 011 110 010 111 110 10
  4. Add leading zeros if needed: 001 011 110 010 111 110 010
  5. Convert each group to octal: 1 3 6 2 7 6 2

Therefore, DE2FA₁₆ = 1362762₈

(ii) Converting 360₈ to hexadecimal:

  1. Convert octal to binary (3 bits per digit):
    • 3 = 011
    • 6 = 110
    • 0 = 000
  2. Combine binary: 011 110 000
  3. Group into sets of four from right: 0111 1000
  4. Convert each group to hexadecimal: 7 8

Therefore, 360₈ = 78₁₆

Question 8

Outline two advantages and two disadvantages of using a questionnaire for data collection.

Answer

Advantages of using questionnaires:

  1. Cost-effective:
    • Cheaper to distribute than conducting interviews
    • Can reach large numbers of respondents simultaneously
    • Minimal staff required for administration
  2. Convenience:
    • Respondents can complete at their own pace
    • Can be distributed electronically or physically
    • Easy to standardize and analyze responses

Disadvantages of using questionnaires:

  1. Limited depth of responses:
    • Cannot probe for additional information
    • Fixed questions may miss important insights
    • No opportunity for clarification
  2. Response quality issues:
    • May have low response rates
    • Respondents might not answer truthfully
    • Questions may be misinterpreted

Question 9

Explain each of the following terms as used in statistics:

(i) Kurtosis;

(ii) Skewness.

Answer

(i) Kurtosis:

  • A measure of the "tailedness" of a probability distribution
  • Describes how peaked or flat a distribution is compared to a normal distribution
  • Types of kurtosis:
    • Mesokurtic (normal): kurtosis = 3
    • Leptokurtic (more peaked): kurtosis > 3
    • Platykurtic (flatter): kurtosis < 3
  • Formula: \[K = \frac{\sum(x-\mu)^4}{n\sigma^4}\]

(ii) Skewness:

  • A measure of asymmetry of a probability distribution
  • Indicates whether data is skewed left or right of the mean
  • Types of skewness:
    • Positive skew: longer tail on right side
    • Negative skew: longer tail on left side
    • Zero skew: symmetrical distribution
  • Formula: \[S = \frac{\sum(x-\mu)^3}{n\sigma^3}\]

Question 10

Solve the following set of simultaneous equations using elimination method.

\[\begin{aligned} & 2x+3y=9 \\ & 4x+y=8 \end{aligned}\]

Answer

Let's solve the system of equations:

\[\begin{aligned} 2x + 3y &= 9 \quad \text{(equation 1)} \\ 4x + y &= 8 \quad \text{(equation 2)} \end{aligned}\]

Step 1: Multiply equation (2) by 3:

\[\begin{aligned} 2x + 3y &= 9 \\ 12x + 3y &= 24 \end{aligned}\]

Step 2: Subtract the first equation from the new equation:

\[12x + 3y - (2x + 3y) = 24 - 9\]

\[10x = 15\]

Step 3: Solve for x:

\[x = \frac{15}{10} = \frac{3}{2}\]

Step 4: Substitute x into equation (1):

\[2(\frac{3}{2}) + 3y = 9\]

\[3 + 3y = 9\]

\[3y = 6\]

\[y = 2\]

Solution:

\[x = \frac{3}{2}, y = 2\]

Verification:

Equation 1: \[2(\frac{3}{2}) + 3(2) = 3 + 6 = 9 \checkmark\]

Equation 2: \[4(\frac{3}{2}) + 2 = 6 + 2 = 8 \checkmark\]

(c) The captain of Kitweli football team is required to choose a committee of 4 members from a team comprising 3 women and 4 men. Determine each of the following:

(i) the number of ways the committee could be chosen;

(ii) the number of ways in which the committee can be chosen comprising 2 men and 2 women;

(iii) the probability that the committee consists of 2 men and 2 women;

Answer

(a) Set Theory Terms:

(i) Subset:

  • A set A is a subset of set B if every element of A is also an element of B
  • Notation: A ⊆ B
  • Example: If A = {1,2} and B = {1,2,3,4}, then A ⊆ B

(ii) Universal Set:

  • The set containing all elements under consideration in a particular context
  • Usually denoted by U or Ω
  • All other sets in the context are subsets of the universal set

(b) Quadratic Function:

  1. Create a table of values:
    x-3-2-1012
    y174-3-4318
  2. Plot points and connect with smooth curve
  3. Solution: y = 0 when 3x² + 4x - 4 = 0
  4. Using quadratic formula: \[x = \frac{-4 \pm \sqrt{16 + 48}}{6} = \frac{-4 \pm \sqrt{64}}{6} = \frac{-4 \pm 8}{6}\]
  5. Therefore x = \(\frac{2}{3}\) or -2

(c) Committee Selection:

(i) Total number of ways to choose committee:

\[C(7,4) = \frac{7!}{4!(7-4)!} = \frac{7!}{4!3!} = 35\text{ ways}\]

Question 12

(a) Explain three types of mathematical functions giving an example in each case.

(b) Using the binomial expansion determine the first four terms in the expansion of

\[(1-2y)^8 \text{ in ascending order of } y.\]

Hence determine the value of \(0.98^8\) (ii) correct to 4 decimal places.

(c) Differentiate between mode and median as used in statistics.

Answer

(a) Types of Mathematical Functions:

  1. Linear Function:
    • Form: f(x) = mx + b
    • Graph is a straight line
    • Example: f(x) = 2x + 3
  2. Quadratic Function:
    • Form: f(x) = ax² + bx + c
    • Graph is a parabola
    • Example: f(x) = x² - 4x + 4
  3. Exponential Function:
    • Form: f(x) = aᵇˣ
    • Shows continuous growth/decay
    • Example: f(x) = 2ˣ

(b) Binomial Expansion:

Expanding (1-2y)⁸ using binomial theorem:

\[(1-2y)^8 = \sum_{r=0}^8 \binom{8}{r}(1)^{8-r}(-2y)^r\]

First four terms:

  1. \(\binom{8}{0}(1)^8(-2y)^0 = 1\)
  2. \(\binom{8}{1}(1)^7(-2y)^1 = -16y\)
  3. \(\binom{8}{2}(1)^6(-2y)^2 = 112y^2\)
  4. \(\binom{8}{3}(1)^5(-2y)^3 = -448y^3\)

Therefore: \[1 - 16y + 112y^2 - 448y^3 + ...\]

For 0.98⁸, let y = 0.02:

\[1 - 16(0.02) + 112(0.02)^2 - 448(0.02)^3\]

\[= 1 - 0.32 + 0.0448 - 0.003584\]

\[= 0.8512\] (4 d.p.)

(c) Mode vs Median:

  • Mode:
    • Most frequently occurring value in a dataset
    • Can have multiple modes (bimodal, multimodal)
    • Not affected by extreme values
  • Median:
    • Middle value when data is arranged in order
    • Splits dataset into two equal halves
    • Less sensitive to outliers than mean

(c) Matrix Operation Proof:

To show: BA + CA = (B + C)A
