Question 1 (4 marks)

(a) Outline three conditions necessary for a probability distribution to be considered as binomial.

(3 marks)

(b) Define the term infinite set as applied in set theory.

(1 mark)

Answer

(a) Conditions for binomial distribution:

  1. There must be a fixed number of trials (n)
  2. Each trial has exactly two possible outcomes (success/failure)
  3. The probability of success (p) remains constant for each trial
  4. The trials are independent of each other

(b) Infinite Set:

An infinite set is a set that contains an unlimited number of elements. The elements cannot be completely counted or listed, and there is no last element. Example: the set of all natural numbers {1, 2, 3, ...}

Question 2 (4 marks)

(a) Using words state the addition rule as used in probability.

(b) Expand the following binomial expression using binomial theorem:

\[(2x+3y)^4\]

Answer

(a) Addition Rule in Probability:

For any two events A and B:

P(A or B) = P(A) + P(B) - P(A and B)

The probability of either event occurring is the sum of their individual probabilities minus the probability of their intersection.

(b) Binomial Expansion:

Using the formula \((a+b)^n = \sum_{r=0}^n \binom{n}{r} a^{n-r} b^r\)

\[(2x+3y)^4 = \binom{4}{0}(2x)^4 + \binom{4}{1}(2x)^3(3y) + \binom{4}{2}(2x)^2(3y)^2 + \binom{4}{3}(2x)(3y)^3 + \binom{4}{4}(3y)^4\]

\[= 16x^4 + 96x^3y + 216x^2y^2 + 216xy^3 + 81y^4\]

Question 3 (4 marks)

The probability that Morine would sell 10, 20, 30 or 40 fish in a particular day is 0.4, 0.3, 0.2 and 0.1 respectively. Determine the number of fish she is expected to sell in a given day.

Answer

Using Expected Value formula: E(X) = Σ(x × P(x))

\[E(X) = (10 × 0.4) + (20 × 0.3) + (30 × 0.2) + (40 × 0.1)\]

\[= 4 + 6 + 6 + 4\]

\[= 20\]

Therefore, Morine is expected to sell 20 fish in a given day.

Question 4 (4 marks)

Explain each of the following terms as used in matrices:

(i) singular matrix;

(ii) inverse matrix.

Answer

(i) Singular Matrix:

  • A square matrix whose determinant is zero
  • Does not have an inverse matrix
  • The rows or columns are linearly dependent

(ii) Inverse Matrix:

  • A matrix that when multiplied by the original matrix gives the identity matrix
  • For matrix A, its inverse A⁻¹ satisfies: AA⁻¹ = A⁻¹A = I
  • Only non-singular (square matrices with non-zero determinant) have inverses

Question 5 (4 marks)

(a) Using the matrix method, solve the following simultaneous equations:

\[p + q = 1\]

\[q - 2p = 7\]

(b) Define the term binomial theorem as used in mathematics.

Answer

(a) Matrix Solution:

Writing in matrix form:

\[\begin{bmatrix}1 & 1 \\ -2 & 1\end{bmatrix}\begin{bmatrix}p \\ q\end{bmatrix}=\begin{bmatrix}1 \\ 7\end{bmatrix}\]

Using inverse matrix method:

\[\begin{bmatrix}p \\ q\end{bmatrix}=\frac{1}{3}\begin{bmatrix}1 & -1 \\ 2 & 1\end{bmatrix}\begin{bmatrix}1 \\ 7\end{bmatrix}\]

Therefore: p = -2, q = 3

(b) Binomial Theorem:

The binomial theorem is a formula that gives the expansion of any power of a binomial expression (a + b)ⁿ as a sum of terms of the form \(\binom{n}{r}a^{n-r}b^r\), where r ranges from 0 to n.

The general form is:

\[(a+b)^n = \sum_{r=0}^n \binom{n}{r} a^{n-r} b^r\]

Question 7 (4 marks)

Using an example in each case differentiate between a transpose of a matrix and a unit matrix.

Answer

Transpose of a Matrix:

A transpose is obtained by converting rows to columns (or columns to rows)

Example:

If \[A=\begin{bmatrix}1 & 2 \\ 3 & 4\end{bmatrix}\]

Then \[A^T=\begin{bmatrix}1 & 3 \\ 2 & 4\end{bmatrix}\]

Unit Matrix:

A square matrix with 1's on the main diagonal and 0's elsewhere

Example:

\[I=\begin{bmatrix}1 & 0 \\ 0 & 1\end{bmatrix}\]

Question 8 (4 marks)

Using a diagram, outline the types of kurtosis curves.

Answer

Types of Kurtosis:

  1. Mesokurtic (Normal)
    • Normal bell-shaped curve
    • Kurtosis = 3
    • Used as reference for other distributions
  2. Leptokurtic
    • More peaked than normal
    • Kurtosis > 3
    • Heavy tails
  3. Platykurtic
    • Flatter than normal
    • Kurtosis < 3
    • Light tails

Question 9 (4 marks)

(a) Using the binomial theorem, determine the fourth term in the following binomial expression:

\[\left[3x-\frac{2}{3}\right]^7\]

(b) Outline two disadvantages of the arithmetic mean.

Answer

(a) Fourth term:

Using the formula: \(\binom{n}{r}a^{n-r}b^r\)

For 4th term, r = 3

\[\binom{7}{3}(3x)^{7-3}(-\frac{2}{3})^3\]

\[= 35(3x)^4(-\frac{2}{3})^3\]

\[= 35(81x^4)(-\frac{8}{27})\]

\[= -840x^4\]

(b) Disadvantages of arithmetic mean:

  • Highly affected by extreme values (outliers)
  • May not represent the typical value in skewed distributions
  • Cannot be calculated for open-ended distributions
  • Not suitable for qualitative data

Question 10 (4 marks)

Convert each of the following number systems to their respective equivalence:

(i) 243₁₀ to excess-3;

(ii) A45₁₆ to octal.

Answer

(i) Converting 243₁₀ to excess-3:

  1. First convert each digit to BCD:
    • 2: 0010
    • 4: 0100
    • 3: 0011
  2. Add 3 to each digit:
    • 2+3=5: 0101
    • 4+3=7: 0111
    • 3+3=6: 0110

Therefore: 243₁₀ = 0101 0111 0110 (excess-3)

(ii) Converting A45₁₆ to octal:

  1. First convert to binary:
    • A = 1010
    • 4 = 0100
    • 5 = 0101
  2. Combined binary: 1010 0100 0101
  3. Group into threes from right: 001 010 100 101
  4. Convert each group to octal: 1 2 4 5

Therefore: A45₁₆ = 1245₈

Question 11 (15 marks)

(a) Define the term universal set as used in set theory.

(b) Differentiate between permutation and combination as applied in probability.

(c) Let \[X=\begin{bmatrix}2 & 1 \\ 1 & 2\end{bmatrix}\] and \[Y=\begin{bmatrix}3 & 2 \\ 4 & -1\end{bmatrix}\]

Evaluate (X+Y)⁻¹

(d) Explain three types of bias that could be applied during data collection.

Answer

(a) Universal Set:

A universal set is the set of all possible elements under consideration in a given context or discussion. It contains all elements relevant to a particular problem or situation.

(b) Permutation vs Combination:

  • Permutation:
    • Arrangement where order matters
    • Formula: P(n,r) = \(\frac{n!}{(n-r)!}\)
    • Used when sequence is important
  • Combination:
    • Selection where order doesn't matter
    • Formula: C(n,r) = \(\frac{n!}{r!(n-r)!}\)
    • Used when only grouping is important

(c) Matrix Solution:

First find X+Y:

\[X+Y=\begin{bmatrix}5 & 3 \\ 5 & 1\end{bmatrix}\]

Then find inverse:

det(X+Y) = 5(1) - 3(5) = -10

\[(X+Y)^{-1}=\frac{1}{-10}\begin{bmatrix}1 & -3 \\ -5 & 5\end{bmatrix}=\begin{bmatrix}-0.1 & 0.3 \\ 0.5 & -0.5\end{bmatrix}\]

(d) Types of Bias in Data Collection:

  1. Selection Bias: When the sample is not representative of the population
  2. Response Bias: When respondents modify their answers due to various factors
  3. Interviewer Bias: When the interviewer influences responses through their behavior or questions
  4. Non-response Bias: When certain groups are less likely to respond to the survey

Question 12 (15 marks)

(a) Differentiate between explicit function and implicit function as used in mathematics.

(b) (i) Define the term random experiment as used in probability.

(ii) A basket contains five (5) white and three (3) black marbles. Two (2) marbles are drawn at random one after the other without replacement. Determine the probability that both marbles drawn are black.

(c) Table 2 shows the age distribution of people in a certain village.

Answer

(a) Explicit vs Implicit Functions:

  • Explicit Function:
    • Variable y is expressed directly in terms of x
    • Form: y = f(x)
    • Example: y = 2x + 3
  • Implicit Function:
    • Relationship between x and y is not solved for y
    • Form: F(x,y) = 0
    • Example: x² + y² = 25

(b) (i) Random Experiment:

A random experiment is a process with a well-defined set of possible outcomes, where the specific outcome cannot be predicted with certainty before the experiment is performed.

(b) (ii) Probability Calculation:

Using multiplication principle:

\[P(\text{both black}) = \frac{3}{8} × \frac{2}{7}\]

\[= \frac{6}{56}\]

\[= \frac{3}{28} \approx 0.107\]

(c) Age Distribution Analysis:

Mean age calculation:

  • Use class midpoints and frequencies
  • Mean = \(\frac{\sum fx}{\sum f}\)
  • Where f = frequency, x = class midpoint

Standard deviation calculation:

\[\sigma = \sqrt{\frac{\sum f(x-\bar{x})^2}{\sum f}}\]

Question 13 (15 marks)

(a) Let A={m,n,o,p}, B={m,n,o,p,q} and C={m,p,r}

Using a Venn diagram in each case, determine the following set operations:

(i) A ∪ B;

(ii) A ∩ C.

(b) Solve the following simultaneous equations graphically:

\[\begin{aligned} y &= x^2-2x+1 \\ y &= 5-2x \end{aligned}\]

(c) Ann's shopping matrix problem

Answer

(a) Set Operations:

(i) A ∪ B = {m,n,o,p,q}

(ii) A ∩ C = {m,p}

(b) Graphical Solution:

  1. Plot y = x² - 2x + 1 (parabola)
  2. Plot y = 5 - 2x (straight line)
  3. Find intersection points
  4. Points of intersection give solutions

(c) Matrix Solution:

(i) Ann's requirements matrix = [1 3 2]

(ii) Prices matrix:

Hurus market = [140 50 100]ᵀ

Naks market = [300 48 80]ᵀ

(iii) Total cost = requirements × prices

Hurus = 1(140) + 3(50) + 2(100) = 490

Naks = 1(300) + 3(48) + 2(80) = 504

Question 15 (15 marks)

(a) Using the graphical method, present the following inequality:

y < 3x + 1

(b) Use the following sets to answer the questions that follow:

P = {a,b,c,d,e}

Q = {c,d,e,f,g}

R = {a,c,d,e,g,h}

U = {a,b,c,d,e,f,g,h,i}

(b) Table 3 shows information from Human Resource office of a certain company regarding its employees.

Qualifications Employees
A level Smith, Jones, William, Brown
Graduates Smith, Brown
Registered associate members Smith, William, Merville, Tyler, Moore, Knight

Use a Venn diagram to represent this information.

(c) (i) Using the two's complement, solve the following binary operation:

01011100₂ - 11011101₂

(ii) Given x=10, y=4 and z=2, determine P if:

\[P=\frac{10y^2+z^3}{x^2}\]

Answer

(a) Computer Coding Terms:

  1. 4-bit BCD code:
    • Binary-Coded Decimal representation
    • Each decimal digit represented by 4 binary bits
    • Uses values 0000 to 1001 only
  2. Parity bit:
    • Extra bit added for error detection
    • Makes total number of 1's even (even parity) or odd (odd parity)
    • Used in data transmission error checking
  3. EBCDIC code:
    • Extended Binary Coded Decimal Interchange Code
    • 8-bit character encoding
    • Used mainly in IBM mainframe computers

(b) Venn Diagram Analysis:

  • X = A-level qualified employees: {Smith, Jones, William, Brown}
  • Y = Graduates: {Smith, Brown}
  • Z = Registered associate members: {Smith, William, Merville, Tyler, Moore, Knight}

(c) Solutions:

(i) Two's complement subtraction:

  1. 01011100₂ (minuend)
  2. Take two's complement of 11011101₂:
    • One's complement: 00100010
    • Add 1: 00100011
  3. Add: 01011100 + 00100011 = 01111111₂

(ii) Value of P:

\[P=\frac{10(4)^2+2^3}{10^2}=\frac{160+8}{100}=1.68\]
