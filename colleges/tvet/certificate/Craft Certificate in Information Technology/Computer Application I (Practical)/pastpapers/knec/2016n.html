TASK 1 (Word Processing) (15 marks)

(a) Open a word processing program and key in the following information as it appears. Save it as Information system in the KNECEXAM folder. (11 marks)

INFORMATION SYSTEM

Many organizations work with large amounts of data. Data are basic values or facts and are organized in a database. Many people think of data as synonymous with information; however, information actually consists of data that has been organized to help answer questions and to solve problems.

An information system is defined as the software that helps organize and analyze data. So, the purpose of an information system is to turn raw data into useful information that can be used for decision making in an organization.

Types of Business Information Systems

  • Manufacturing and production information systems
  • Sales and marketing information systems
  • Finance and accounting information systems
  • Human resources information systems

(b) Apply the following formats in the document created in (a):

  1. title font: Garamond, bold and size 16; (1 mark)
  2. line spacing double.
  3. apply italics to the last paragraph.

(c) Insert a picture of a computer in the middle of the first paragraph.

(d) Save changes to print out later.

Answer

(Practical Task: Requires performing actions in a word processor)

(a) Document Creation: Type the provided text and save as 'Information system'.

(b) Formatting: Apply specified font, bold, size, line spacing, and italics to 'Information system'.

(c) Image Insertion: Insert a computer picture into the first paragraph.

(d) Printout Required: Information system.
