(f) Create a report for the query created in (e). Save the report as FruitsupplyRep (2 marks)

  1. Farmers, Fruits and Fruittransact tables;
  2. Fruitsupply query;

Answer

(Practical Task: Requires performing actions in a database program)

(a) Database Creation: Create database 'juicyfruit'.

(b) Table Creation & Data Entry: Create tables 'farmers', 'fruits', and 'fruittransact'. Enter the provided data (data needs to be extracted from the image placeholders).

(d) Relationships: Create relationships between tables (e.g., 'farmers'.FarmerID to 'fruittransact'.FarmerID, 'fruits'.FruitID to 'fruittransact'.FruitID).

(e) Query Creation: Create query 'fruitsupply' displaying FarmerName, FruitName, NumberDelivered.

(f) Report Creation: Create report 'FruitsupplyRep' based on the 'fruitsupply' query.

(g) Printouts Required: All three tables (datasheet view), fruitsupply query (datasheet view), FruitsupplyRep report.
