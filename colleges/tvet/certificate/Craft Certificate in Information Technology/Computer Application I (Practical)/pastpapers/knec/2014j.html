TASK 1 (Word Processing - Mail Merge)

The manager of Mobility Insurance Company intends to invite potential clients for a function in the company. You have been requested by the manager to prepare the invitation letters using mail merge.

(a) Open a word processing program and create a data source with the following information and save it as insurancedata in the KNECEXAM folder. (4½ marks)

TitleNamesCompanyAddressInsurance Product
MRANDREW LINCOLINALITE2435-0200EDUCATION
MRMATHEWS ALIYellolite6782-0200MEDICAL
MRSLINDA ALICEMERRGITA7819-0100LIFE INSURANCE
MISSALICE YERIEUROTAP37338-0200ACCIDENT INSURANCE

(b) (i) Type the following letter as it appears and save it as insuremaindoc in the KNECEXAM folder. (8 marks)

MOBILITY INSURANCE COMPANY
P.O. BOX 544467-0233
NAIROBI

RE: INSURANCE PRODUCT

Chairman

(ii) Merge the data source created in (a) with the insuremaindoc created in b(i). (1 mark)

(iii) Save the merged document as mergedinsurance in the KNECEXAM folder to print out later:

  1. Insuremaindoc;
  2. mergedinsurance.

Answer

(Practical Task: Requires performing Mail Merge in a word processor)

(a) Data Source Creation: Create and save 'insurancedata'.

(b) Main Document & Merge:

  1. Type and save letter as 'insuremaindoc'.
  2. Perform mail merge using 'insurancedata' and 'insuremaindoc'.
  3. Save merged document as 'mergedinsurance'.

Printouts Required: insuremaindoc, mergedinsurance.

(g) Print out later:

  1. tables;
  2. contribution query;
  3. customer report.

Answer

(Practical Task: Requires performing actions in a database program)

(a) Database Creation: Create database 'Mobility'.

(b) Table Creation: Create tables 'customer' and 'insurance' with specified fields, data types, and sizes.

(c) Relationship: Create a one-to-many relationship between 'insurance' (Insurance Code - PK) and 'customer' (Insurance code - FK).

(d) Form Creation & Data Entry: Create forms 'customer' and 'insurance' and enter the provided data into the respective tables using the forms.

(e) Query Creation: Create query 'contribution' displaying Customer ID, Customer Name, Type of Insurance, Monthly Contribution, Date Insured (assuming Date Insured is the payment date intended).

(f) Report Creation: Create report 'customer' displaying all fields from the customer table.

Printouts Required: Both tables (datasheet view), contribution query (datasheet view), customer report.
