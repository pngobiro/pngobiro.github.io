Question 1 (4 marks)

Outline four acceptors elements that would form a p-type region when a semiconductor is doped.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Acceptor elements are typically elements from Group 13 (or Group III) of the periodic table. When used to dope a Group 14 semiconductor (like Silicon or Germanium), they have one less valence electron than the semiconductor atom. This creates a "hole" (a missing electron in the covalent bond structure), which acts as a positive charge carrier, forming a p-type region.

Four common acceptor elements are:

  • Boron (B)
  • Aluminum (Al)
  • Gallium (Ga)
  • Indium (In)

The operation is $345_8 + 43F_{16}$. To perform this, we first convert both numbers to decimal.

$$ 345_8 = (3 \times 8^2) + (4 \times 8^1) + (5 \times 8^0) $$ $$ = (3 \times 64) + (4 \times 8) + (5 \times 1) $$ $$ = 192 + 32 + 5 = 229_{10} $$

Question 4 (4 marks)

Explain two characteristics of a Static Random Access Memory.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Static Random Access Memory (SRAM) is a type of semiconductor memory that uses bistable latching circuitry (flip-flops) to store each bit. Two key characteristics are:

  1. Speed: SRAM is significantly faster than Dynamic RAM (DRAM). Because it uses flip-flops, the data is readily available as long as power is supplied, and it doesn't require the periodic refreshing that DRAM needs. This high speed makes SRAM suitable for CPU caches (L1, L2, L3) where rapid access to frequently used data is critical for performance.
  2. Volatility (and No Refresh Needed): Like DRAM, SRAM is volatile, meaning it loses its stored data when the power supply is turned off. However, unlike DRAM which uses capacitors that leak charge and require constant refreshing cycles to maintain data, SRAM retains its data indefinitely as long as power is continuously supplied, without needing any refresh operations. This simplifies the control circuitry but contributes to higher power consumption compared to DRAM when idle.
  3. (Other characteristics include: Lower density and higher cost per bit compared to DRAM due to the more complex cell structure (typically 4-6 transistors per bit vs. 1 transistor and 1 capacitor for DRAM), higher power consumption during operation.)

Question 5 (4 marks)

Modern computers use ROM in the manufacture of their memories. Explain two primarily uses of ROM.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Read-Only Memory (ROM) is non-volatile memory, meaning it retains its data even when power is off. This makes it ideal for storing essential software that needs to be available immediately upon system startup.

Two primary uses of ROM in modern computers are:

  1. Firmware Storage (e.g., BIOS/UEFI): ROM chips store the system's firmware, such as the Basic Input/Output System (BIOS) or its successor, the Unified Extensible Firmware Interface (UEFI). This firmware contains the initial code required to start the computer, perform hardware initialization (like the Power-On Self-Test or POST), identify boot devices, and load the operating system from a storage drive (like an SSD or HDD). Because this code must be present before the main OS loads, non-volatile ROM is essential.
  2. Microcode and Embedded System Controllers: ROM can store microcode for processors or instructions for dedicated microcontrollers found in various computer components (e.g., graphics cards, network adapters, storage controllers, peripherals like printers). These embedded controllers often need their operating instructions permanently stored to manage the specific hardware functions they control, independent of the main operating system.

Question 6 (4 marks)

Explain two application areas of gray code in computers.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

Gray code, also known as reflected binary code, is a binary numeral system where two successive values differ in only one bit (binary digit). This property is useful in preventing errors related to transitional states in digital systems.

Two application areas of Gray code in computers are:

  1. Electromechanical Switches and Position Encoders: Gray codes are widely used in rotary and linear position encoders. These devices translate physical position into a digital signal. If standard binary code were used, multiple bits could change simultaneously when moving between adjacent positions (e.g., from 0111 to 1000). Due to mechanical tolerances or timing issues, the sensor might read an intermediate, incorrect value during the transition. Gray code eliminates this problem because only one bit changes between consecutive positions, ensuring that any read value during a transition corresponds to one of the two adjacent valid positions, thus preventing spurious readings.
  2. Error Correction and Digital Communications: The single-bit change property of Gray codes can be advantageous in certain digital communication systems and error correction schemes. It can help minimize the impact of errors, as a single-bit error during transmission is more likely to result in a value that is numerically close to the original value compared to standard binary, where a single-bit error can sometimes lead to a large jump in numerical value. It's also used in ordering states within Karnaugh maps for simplifying Boolean logic circuits.

Question 7 (4 marks)

(a) Computer designers prefers using hexadecimal number system. State two reasons for this.

(b) Define each of the following terms as used in electrical circuits:

(i) overcurrent;

(ii) wattage.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Two Reasons for Using Hexadecimal Number System:

  • Conciseness and Readability: Hexadecimal provides a much shorter and more human-readable way to represent long binary numbers (which computers use internally). Each hexadecimal digit corresponds exactly to a 4-bit binary sequence (nibble), making conversion straightforward and representation compact. For example, the 16-bit binary number `1110 0101 1011 0010` is simply `E5B2` in hexadecimal, which is easier to write, read, and remember.
  • Direct Mapping to Binary: As mentioned, the direct 4-to-1 mapping between binary bits and hexadecimal digits simplifies debugging, memory addressing, and data representation in low-level programming and hardware design. It allows developers to easily visualize the underlying binary patterns represented by the hexadecimal code.

(b) Definitions:

  • (i) Overcurrent: A situation in an electrical circuit where the current flowing through a conductor exceeds its rated capacity or the normal operating current of the circuit or equipment. Overcurrents can be caused by short circuits, ground faults, overloads, or design flaws. They can lead to overheating, damage to equipment, insulation breakdown, and fire hazards. Protective devices like fuses and circuit breakers are used to detect overcurrents and interrupt the circuit.
  • (ii) Wattage: Refers to the rate at which electrical energy is consumed or transferred in a circuit, measured in Watts (W). It represents the electrical power ($P$) and is calculated as the product of voltage ($V$) and current ($I$) in DC circuits ($P = V \times I$), or involves power factor in AC circuits. The wattage rating of a device indicates how much power it consumes during operation or how much power it can safely handle (e.g., a resistor's power rating).

Question 9 (4 marks)

With the aid of a sketch, implement an AND with two inputs using NAND logic gates.

(Sketch to be added in answer section if possible, or described)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

An AND gate has the Boolean expression $Output = A \cdot B$. We can implement this using only NAND gates by utilizing the property that a NAND gate followed by a NOT gate (which can also be made from a NAND gate) results in an AND gate.

Steps:

  1. Feed inputs A and B into the first NAND gate. The output is $\overline{A \cdot B}$.
  2. Feed the output of the first NAND gate ($\overline{A \cdot B}$) into both inputs of a second NAND gate (acting as a NOT gate). The output is $\overline{(\overline{A \cdot B}) \cdot (\overline{A \cdot B})} = \overline{\overline{A \cdot B}} = A \cdot B$.

(a) (i) Four Examples of Read Only Memory (ROM):

  • Mask ROM (MROM): Programmed during manufacturing; cannot be changed afterward. Used for high-volume, stable firmware.
  • Programmable ROM (PROM): Can be programmed once by the user using special equipment (blowing fuses internally). Cannot be erased or reprogrammed.
  • Erasable Programmable ROM (EPROM): Can be erased by exposing it to strong ultraviolet (UV) light through a quartz window on the chip, and then reprogrammed.
  • Electrically Erasable Programmable ROM (EEPROM): Can be erased and reprogrammed electrically, byte by byte, without removing it from the circuit. Slower and less dense than Flash memory.
  • (Flash Memory is often considered a type of EEPROM, offering block-level erasure/programming and higher density, commonly used in SSDs, USB drives, and modern firmware).

(a) (ii) Two Emerging Trends of Electronic Components:

  • Miniaturization and Integration (System-on-a-Chip - SoC): Continuously shrinking component sizes (following Moore's Law trends) allows for more functionality to be packed into smaller spaces. This leads to the integration of multiple system components (CPU, GPU, memory, controllers) onto a single chip (SoC), enabling powerful yet compact devices like smartphones, wearables, and IoT devices.
  • Low Power Consumption and Energy Harvesting: With the rise of mobile computing and IoT, there's a strong focus on developing ultra-low-power components. This extends battery life and enables devices to operate in energy-constrained environments. Coupled with this is the trend towards energy harvesting technologies, where components can scavenge power from ambient sources like light, vibration, or thermal gradients, potentially leading to self-powered devices.

(b) (i) BCD Evaluation: $91 + 79$

  • 91: $9 \rightarrow 1001$; $1 \rightarrow 0001$. So, $1001 \, 0001_{\text{BCD}}$
  • 79: $7 \rightarrow 0111$; $9 \rightarrow 1001$. So, $0111 \, 1001_{\text{BCD}}$

2. Add the BCD numbers using binary addition:

  (Carry) ->   1   1
               1001 0001

             10000 1010

(b) (ii) Differentiate between Power and Energy: (4 marks)

  • Power (P): Power is the rate at which energy is transferred, used, or converted. It measures how quickly work is done or energy is consumed. In electrical circuits, it's typically measured in Watts (W). $P = \frac{Energy}{Time}$.
  • Energy (E or W): Energy is the capacity to do work. It represents a total amount of work done or potential work that can be done over a period. In electrical terms, it's often measured in Joules (J) or kilowatt-hours (kWh). $Energy = Power \times Time$.

In essence, power is about how fast energy is used, while energy is about the total amount used over time.

In the crystal lattice, each atom shares one valence electron with each of its four nearest neighbors, forming strong covalent bonds. A high resistivity indicates that the material strongly resists current flow. It is the reciprocal of conductivity ($\rho = 1/\sigma$). The unit is Ohm-meter ($\Omega \cdot m$).

In essence, conductivity describes the ease of current flow, while resistivity describes the opposition to current flow. They are inversely related material properties.

(a) (i) Define Impedance (Z):

Impedance is the total opposition that a circuit presents to the flow of alternating current (AC) at a particular frequency. (b) (ii) Simplify Boolean Function: $F=\bar{A} B C+A \bar{B} C+A B \bar{C}+A B C+A B C+A B C$

First, note the repeated $ABC$ terms. Using the Idempotent Law ($X+X = X$), we can simplify the expression to:

$$ F = \bar{A}BC + A\bar{B}C + AB\bar{C} + ABC $$

We can use Boolean algebra or a K-map. Using algebra:

Factor out common terms:

$$ F = BC(\bar{A} + A) + A\bar{B}C + AB\bar{C} $$

Since $\bar{A} + A = 1$:

$$ F = BC(1) + A\bar{B}C + AB\bar{C} $$ $$ F = BC + A\bar{B}C + AB\bar{C} $$

Factor out A:

$$ F = BC + A(\bar{B}C + B\bar{C}) $$

The term in the parenthesis is the XOR function ($B \oplus C$).

$$ F = BC + A(B \oplus C) $$

Alternatively, using consensus theorem or K-map simplification often leads to a simpler sum-of-products form. Let's try factoring differently:

$$ F = \bar{A}BC + A\bar{B}C + AB\bar{C} + ABC $$ $$ F = C(\bar{A}B + A\bar{B} + AB) + AB\bar{C} $$ $$ F = C(\bar{A}B + A(\bar{B} + B)) + AB\bar{C} $$ $$ F = C(\bar{A}B + A(1)) + AB\bar{C} $$ $$ F = C(\bar{A}B + A) + AB\bar{C} $$

Using $X + \bar{X}Y = X + Y$:

$$ F = C(A + B) + AB\bar{C} $$ $$ F = AC + BC + AB\bar{C} $$

Let's check the K-map (from Q13, we know $ABC$ is 111, $AB\bar{C}$ is 110, $A\bar{B}C$ is 101, $\bar{A}BC$ is 011):

     BC=00 BC=01 BC=11 BC=10
   +-----+-----+-----+-----+
A=0 |  0  |  0  | (1) |  0  |
   +-----+-----+-----+-----+
A=1 |  0  | (1) | (1) | (1) |
   +-----+-----+-----+-----+

Grouping the 1s:

  • Group 1: Cells (101, 111) -> $AC$
  • Group 2: Cells (011, 111) -> $BC$
  • Group 3: Cells (110, 111) -> $AB$

Simplified Expression (from K-map): $\mathbf{F = AB + BC + AC}$

Note: The algebraic simplification $AC + BC + AB\bar{C}$ is also valid, but $AB + BC + AC$ is the standard minimal SOP form derived from the K-map.

(b) (ii) Explain Battery Terms:

  • (I) Float Charging: A method of charging a secondary (rechargeable) battery where it is continuously connected to a charger that supplies a low, constant voltage just sufficient to counteract the battery's self-discharge rate and maintain it at a fully charged state. It's commonly used for backup power systems (like UPS batteries or emergency lighting) to ensure the battery is always ready without overcharging it significantly.
  • (II) Memory Effect: A phenomenon observed primarily in older Nickel-Cadmium (NiCd) and, to a lesser extent, Nickel-Metal Hydride (NiMH) rechargeable batteries. If these batteries are repeatedly recharged before being fully discharged, they can appear to "remember" the point at which recharging began and subsequently only deliver the capacity used before recharging, effectively reducing their usable capacity. Modern Lithium-ion (Li-ion) batteries do not suffer from this memory effect.
