Question 1 (15 marks)

(a) Explain the functions of the following units found in the Central Processing Unit:

(i) The control unit;

(ii) the arithmetic unit.

(b) Differentiate between RAM and ROM.

(c) (i) Create the following spreadsheet and answer the questions that follow:

STUDENTS NAME SUBJECT
Maths English Biology Total
Main Anders 67 74 71
Ana Truji 58 65 56
Thomas Hardy 50 55 58
Ann Devon 60 62 64
Martin Suma 85 82 81

I) Enter formula for computing the total marks for each student.

II) Identify a formula that can be able to compute the highest marks for each subject.

(ii) Outline the procedure theo would use to copy data from an existing spread sheet workshect to a new worksheet.

(d) Differentiate between source document file and data document file used in word processing mail merge.

Answer

(a) Explain the functions of the following units found in the Central Processing Unit:

(i) The control unit: The control unit is the component of the CPU that manages and coordinates all the activities of the computer system. It fetches instructions from memory, decodes them, and executes them by sending control signals to other components of the CPU and the rest of the computer.

(ii) the arithmetic unit: The arithmetic logic unit (ALU) performs arithmetic and logical operations on data. Arithmetic operations include addition, subtraction, multiplication, and division. Logical operations include AND, OR, NOT, and XOR.

(b) Differentiate between RAM and ROM:

RAM (Random Access Memory): RAM is a type of computer memory that can be accessed randomly; that is, any byte of memory can be accessed without touching the preceding bytes. RAM is volatile, meaning that data stored in RAM is lost when the computer is turned off. RAM is used to store the operating system, application programs, and data that are currently being used by the computer.

ROM (Read-Only Memory): ROM is a type of computer memory that can only be read; it cannot be written to. ROM is non-volatile, meaning that data stored in ROM is not lost when the computer is turned off. ROM is used to store the boot program and other essential system software.

(c) (i) Create the following spreadsheet and answer the questions that follow:

I) Enter formula for computing the total marks for each student: In a spreadsheet program like Microsoft Excel or Google Sheets, you would enter the following formula in the "Total" column for each student: =SUM(B2:D2) (assuming the Maths, English, and Biology marks are in columns B, C, and D, respectively, and the first student's data starts in row 2). You would then copy this formula down to the other students' rows.

II) Identify a formula that can be able to compute the highest marks for each subject: To compute the highest marks for each subject, you would use the MAX function. For example, to find the highest mark in Maths, you would enter the following formula in a separate cell: =MAX(B2:B6) (assuming the Maths marks are in column B and the student data spans rows 2 to 6). You would repeat this for English and Biology, adjusting the column range accordingly.

(ii) Outline the procedure theo would use to copy data from an existing spread sheet workshect to a new worksheet:

  1. Open both the existing spreadsheet and the new worksheet.
  2. Select the data you want to copy from the existing spreadsheet.
  3. Copy the data by pressing Ctrl+C (or Cmd+C on macOS) or by right-clicking and selecting "Copy."
  4. Navigate to the new worksheet.
  5. Select the cell where you want to paste the data.
  6. Paste the data by pressing Ctrl+V (or Cmd+V on macOS) or by right-clicking and selecting "Paste."

(d) Differentiate between source document file and data document file used in word processing mail merge:

Source document file: In word processing mail merge, the source document file (also known as the main document) contains the text and formatting that will be the same for all merged documents. This includes the letter body, headings, and any other static content.

Data document file: The data document file (also known as the data source) contains the variable information that will be inserted into the source document during the mail merge process. This typically includes names, addresses, and other personalized data for each recipient.

Question 3 (15 marks)

(a) State four threats to information stored in a computer that may arise due to connectivity to the internet.

(b) Explain three benefits of using electronic mail instead of ordinary paper mail to send a job application letter.

Answer

(a) State four threats to information stored in a computer that may arise due to connectivity to the internet:

  1. Malware: Viruses, worms, and Trojan horses can infect computers through downloaded files or malicious websites.
  2. Hacking: Unauthorized individuals can gain access to computers and steal or damage information.
  3. Phishing: Attackers can use fake emails or websites to trick users into revealing sensitive information.
  4. Data breaches: Organizations can experience data breaches that expose sensitive information to the public.

(b) Explain three benefits of using electronic mail instead of ordinary paper mail to send a job application letter:

  1. Speed: Email is much faster than traditional mail, allowing applications to be sent and received almost instantly.
  2. Cost-effectiveness: Email is much cheaper than traditional mail, as it eliminates the cost of postage and paper.
  3. Convenience: Email is more convenient for both the sender and the recipient, as it can be sent and received from anywhere with an internet connection.

Question 4 (15 marks)

(a) State the word processing formula that should be used in a table to compute each of the values:

(i) total in a column;

(ii) total in a row;

(iii) mean score in a row of scores.

(b) State three formatting features that may be applied to a numeric entry in a cell of a spreadsheet.

(c) Distinguish between relative cell references and absolute cell references as used in spreadsheets.

Answer

(a) State the word processing formula that should be used in a table to compute each of the values:

(i) total in a column: =SUM(ABOVE)

(ii) total in a row: =SUM(LEFT) or =SUM(RIGHT) depending on the position of the total column.

(iii) mean score in a row: =AVERAGE(LEFT) or =AVERAGE(RIGHT) depending on the position of the scores.

(b) State three formatting features that may be applied to a numeric entry in a cell of a spreadsheet:

  1. Number format: This allows you to specify the number of decimal places, currency symbol, or percentage format.
  2. Alignment: This allows you to align the number to the left, right, or center of the cell.
  3. Font: This allows you to change the font, size, and color of the number.

(c) Distinguish between relative cell references and absolute cell references as used in spreadsheets:

Relative cell references: These references change when a formula is copied to another cell. For example, if cell A1 contains the formula =B1+C1, and you copy this formula to cell A2, the formula will change to =B2+C2.

Absolute cell references: These references do not change when a formula is copied to another cell. To create an absolute cell reference, you add a dollar sign ($) before the column and row. For example, if cell A1 contains the formula =$B$1+$C$1, and you copy this formula to cell A2, the formula will remain =$B$1+$C$1.

Question 5 (15 marks)

(a) Outline the steps that should be followed when creating a database.

(b) List four views in a presentation program.

(c) State three formatting features that may be applied to a graphic created using a desktop publishing program.

Answer

(a) Outline the steps that should be followed when creating a database:

  1. Define the purpose of the database: Determine what information the database will store and how it will be used.
  2. Identify the tables needed: Decide what tables are needed to store the information.
  3. Define the fields in each table: Determine what fields (columns) are needed in each table.
  4. Identify the primary key for each table: Choose a field that will uniquely identify each record in the table.
  5. Establish relationships between tables: Define how the tables are related to each other.
  6. Create the tables and fields: Use a database management system (DBMS) to create the tables and define the fields.
  7. Enter data into the tables: Populate the tables with data.

(b) List four views in a presentation program:

  1. Normal view: This is the main view used for creating and editing slides.
  2. Slide sorter view: This view displays all the slides in the presentation as thumbnails, allowing you to easily rearrange them.
  3. Notes page view: This view displays each slide with any notes that have been added to it.
  4. Slide show view: This view displays the presentation as it will appear to the audience.

(c) State three formatting features that may be applied to a graphic created using a desktop publishing program:

  1. Resizing: Changing the size of the graphic.
  2. Cropping: Removing unwanted parts of the graphic.
  3. Rotation: Rotating the graphic.

Question 6 (Compulsory) (15 marks)

(a) Business organisations are increasingly using electronic communication. Explain three reasons that may account for this increase.

(b) It has been observed that majority of the employees of Peace Limited spend a lot of their time speaking on their cell phones. As the foreman, write a memorandum to the employees requesting them to stop this practice.

(c) (i) Explain the phrase "Annual General Meeting (AGM)."

(ii) Outline four items of agendn that may be discussed in an annual general meeting of a publicly listed company.

Answer

(a) Business organisations are increasingly using electronic communication. Explain three reasons that may account for this increase:

  1. Increased efficiency: Electronic communication allows for faster and more efficient communication compared to traditional methods such as postal mail or face-to-face meetings.
  2. Reduced costs: Electronic communication can significantly reduce communication costs, as it eliminates the need for paper, postage, and long-distance phone calls.
  3. Improved collaboration: Electronic communication tools such as email, instant messaging, and video conferencing facilitate collaboration among employees, regardless of their location.

(b) It has been observed that majority of the employees of Peace Limited spend a lot of their time speaking on their cell phones. As the foreman, write a memorandum to the employees requesting them to stop this practice:

MEMORANDUM

To: All Employees

From: [Your Name], Foreman

Date: [Date]

Subject: Use of Cell Phones During Work Hours

It has come to my attention that many employees are spending a significant amount of time speaking on their cell phones during work hours. This practice is disruptive and negatively impacts productivity.

Therefore, I am requesting that all employees refrain from using their cell phones for personal calls during work hours. If you need to make or receive a call, please do so during your break time or lunch break, and step away from your workstation.

Your cooperation in this matter is greatly appreciated.

(c) (i) Explain the phrase "Annual General Mecting (AGM).":

An Annual General Meeting (AGM) is a mandatory yearly gathering of a company's directors and shareholders. It is a formal meeting where the company's performance is reviewed, financial statements are presented, and important decisions are made.

(ii) Outline four items of agendn that may be discussed in an annual general meeting of a publicly listed company:

  1. Presentation and approval of the annual financial statements.
  2. Election or re-election of directors.
  3. Appointment of auditors and fixing their remuneration.
  4. Declaration of dividends.

Question 7 (15 marks)

(a) State four guidelines that should be followed when using tables to provide information in a report.

(b) Explain three measures that the management of an organisation may take to encourage upward communication.

Answer

(a) State four guidelines that should be followed when using tables to provide information in a report:

  1. Use clear and concise headings: Each column and row should have a clear and concise heading that accurately describes the data it contains.
  2. Use consistent formatting: Use consistent formatting throughout the table, including font, size, and alignment.
  3. Keep the table simple: Avoid using too many columns or rows, and keep the data as simple as possible.
  4. Provide a caption: Each table should have a caption that describes the table's purpose and content.

(b) Explain three measures that the management of an organisation may take to encourage upward communication:

  1. Implement an open-door policy: Encourage employees to approach management with their concerns and ideas.
  2. Conduct regular employee surveys: Use surveys to gather feedback from employees on a variety of topics.
  3. Hold regular meetings with employees: Use meetings to provide employees with updates on company performance and to solicit their feedback.

Question 8 (15 marks)

(a) Distinguish between a conference and a panel discussion.

(b) (i) Explain the term etiquette as used in communication

(ii) Outline four ways in which an interviewee may demonstrate etiquette during an interview.

Answer

(a) Distinguish between a conference and a panel discussion:

Conference: A conference is a large meeting, typically lasting several days, where people with a common interest come together to learn, share information, and network. Conferences typically involve a variety of activities, such as keynote speeches, workshops, and panel discussions.

Panel discussion: A panel discussion is a format where a group of experts discusses a particular topic in front of an audience. The audience can ask questions and participate in the discussion.

(b) (i) Explain the term etiquette as used in communication:

Etiquette in communication refers to the set of rules and customs that govern polite and respectful behavior in communication. It involves being mindful of the other person's feelings and needs, and using appropriate language and tone.

(ii) Outline four ways in which an interviewee may demonstrate etiquette during an interview:

  1. Arriving on time for the interview.
  2. Dressing professionally and appropriately.
  3. Maintaining eye contact and using respectful language.
  4. Listening attentively and answering questions thoughtfully.
