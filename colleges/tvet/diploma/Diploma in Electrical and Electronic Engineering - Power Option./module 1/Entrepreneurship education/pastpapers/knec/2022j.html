Question 1

(a) Computer networks have gained more popularity as compared to the mainframe computers:

(i) distinguish between the mainframe computer and computer network;

(ii) state two reasons that may account for the popularity of computer networks.

(b) State the function of each of the following keys on the computer keyboard:

(i) Tab

(ii) Esc

(iii) Backspace

(c) Outline three utility programs that form part of a programming language.

Answer

This question covers fundamental concepts of computer systems, input devices, and programming tools.

(a)(i) Distinction between mainframe computer and computer network:

Mainframe Computer:

  • A single, powerful computer system
  • Centralized processing and storage
  • Multiple users access through terminals
  • High cost and maintenance

Computer Network:

  • Multiple interconnected computers
  • Distributed processing and storage
  • Resource sharing between independent computers
  • More flexible and scalable

(a)(ii) Reasons for popularity of computer networks:

  • Cost-effective - sharing resources reduces per-user cost
  • Flexibility in scaling and upgrading
  • Enhanced communication and collaboration
  • Distributed processing power and storage

(b) Function of keyboard keys:

  • Tab: Moves the cursor to the next tab stop or indents text in documents
  • Esc (Escape): Cancels current operations or exits from current screens/menus
  • Backspace: Deletes the character to the left of the cursor position

(c) Utility programs in programming languages:

  • Editor - for writing and modifying source code
  • Compiler/Interpreter - translates source code into machine code
  • Debugger - helps find and fix errors in programs
  • Linker - combines object files into executable programs
  • Loader - loads programs into memory for execution

Question 2

(a) State the purpose of each of the following operations on a computer hard disk:

(i) scanning;

(ii) formatting;

(iii) labeling.

(b) Highlight two risks that may be associated with lack of data privacy in an organization.

(c) State two measures that may be taken to detect computer crimes in an organization.

(d) Joan has a bulleted list typed using a word processing application. State the steps that she should follow to format the list into three columns with a vertical line separating the columns.

Answer

This question addresses disk operations, data security, and word processing skills.

(a) Purpose of hard disk operations:

  • Scanning: Checks the disk for errors, bad sectors, and file system integrity; helps identify and fix disk-related problems
  • Formatting: Prepares the disk for data storage by creating a file system and clearing existing data; establishes the basic structure for storing files
  • Labeling: Assigns a unique name or identifier to the disk for easy identification and organization in the system

(b) Risks associated with lack of data privacy:

  • Identity theft and unauthorized access to sensitive information
  • Financial losses due to fraud or theft of financial data
  • Legal liability and compliance violations
  • Damage to organization's reputation and loss of customer trust

(c) Measures to detect computer crimes:

  • Installing and maintaining security monitoring systems
  • Regular system audits and log analysis
  • Implementation of intrusion detection systems (IDS)
  • Employee activity monitoring and access tracking

(d) Steps to format list into three columns with vertical lines:

Select the bulleted list text
Go to the "Layout" or "Page Layout" tab
Click on "Columns" and select "More Columns"
In the Columns dialog box:
  • Set number of columns to 3
  • Check the "Line between" option
  • Adjust spacing if needed
Click "OK" to apply the changes

Question 3

(a) Explain the importance of using macros in a spreadsheet document.

(b) Outline the steps that should be followed to create a pie chart in a spreadsheet package.

(c) State the formula that would be used to compute each of the following in a table created using a word processing application:

(i) total of a column of values;

(ii) total of a row of values;

(iii) mean score in a column of scores;

(iv) multiplication of two values in a row.

Answer

This question covers spreadsheet automation, data visualization, and table calculations in word processing.

(a) Importance of macros in spreadsheets:

  • Automates repetitive tasks to save time and reduce errors
  • Ensures consistency in performing complex operations
  • Allows non-technical users to execute complex procedures
  • Can be shared across users and documents
  • Increases productivity and efficiency in data processing

(b) Steps to create a pie chart:

Select the data range to be included in the chart
Click on the "Insert" tab in the ribbon
Select "Pie Chart" from the Charts group
Choose the desired pie chart style
Customize the chart:
  • Add chart title
  • Add data labels if needed
  • Format colors and legends
  • Adjust layout and style

(c) Table formulas in word processing:

(i) Column total: =SUM(ABOVE)

Sums all values in the column above the current cell

(ii) Row total: =SUM(LEFT)

Sums all values in the row to the left of the current cell

(iii) Mean score: =AVERAGE(ABOVE)

Calculates the average of all values in the column above

(iv) Multiplication: =PRODUCT(A1,B1)

Multiplies values in cells A1 and B1 (replace with actual cell references)

Question 4

(a) Outline the procedure followed when sending an electronic mail that contains an attachment.

(b) A trainee intends to create a database table. State four basic specifications that the trainee should provide during the process.

Answer

This question addresses email communication and database design fundamentals.

(a) Procedure for sending email with attachment:

Open email client or webmail service
Click "Compose" or "New Email" button
Fill in recipient details:
  • To: Primary recipient's email address
  • Cc: Carbon copy recipients (if needed)
  • Bcc: Blind carbon copy recipients (if needed)
Enter subject line describing email content
Write the email message body
Add attachment:
  • Click "Attach" or paperclip icon
  • Browse and select file(s)
  • Wait for upload to complete
Review email and attachment
Click "Send" button

(b) Basic database table specifications:

  • Table Name: Unique, descriptive name for the table
  • Field Names: Names for each column in the table
  • Data Types: Type of data each field will store (text, number, date, etc.)
  • Field Size/Length: Maximum size for each field
  • Primary Key: Field(s) that uniquely identify each record
  • Field Properties: Additional specifications like required/optional, default values
  • Field Constraints: Rules for data validation and integrity
  • Field Description: Documentation of field purpose and usage

These specifications form the foundation of a well-designed database table that ensures data integrity and usability.

Question 5

(a) Some organizations require job applications to be sent via email. State two reasons that may have influenced this requirement.

(b) Explain the circumstances under which each of the following view options is appropriate in a presentation program:

(i) slide show;

(ii) reading view;

(iii) normal view.

(c) State the purpose of each of the following features in a desktop publishing package:

(i) bring forward;

(ii) send backward.

Answer

This question covers digital communication in recruitment, presentation software features, and desktop publishing concepts.

(a) Reasons for requiring email job applications:

  • Efficient organization and sorting of applications through digital filing
  • Cost-effective compared to physical mail handling
  • Easy sharing of applications among recruitment team members
  • Environmental sustainability by reducing paper usage
  • Demonstrates candidate's basic digital literacy skills

(b) Appropriate circumstances for presentation views:

Slide Show View:

  • During actual presentation delivery
  • When practicing presentation timing
  • For testing animations and transitions
  • When displaying full-screen content to audience

Reading View:

  • When reviewing presentation content individually
  • For self-paced content consumption
  • During content proofreading
  • When sharing for individual review

Normal View:

  • During presentation creation and editing
  • When adding or modifying content
  • For organizing slides and outline
  • While working with speaker notes

(c) Desktop publishing features:

Bring Forward:

  • Moves selected object one layer closer to front
  • Used to control object visibility in overlapping arrangements
  • Helps create proper visual hierarchy in layouts

Send Backward:

  • Moves selected object one layer toward back
  • Used to position background elements
  • Helps manage layering of design elements

Question 6 (Compulsory)

(a) State three indicators of a poorly summarized passage.

(b) Outline three reasons that make it necessary to write business reports in an organization.

(c) With respect to workplace communication, outline three challenges that may be associated with working from home.

(d) The communication Assistant at Wetu Works Limited is drafting an advertisement for a vacant position. Outline four types of information that may be contained in this advertisement.

(e) Highlight three situations that may require a person to observe protocol in an organization.

(f) Certain strategies may be adopted when handling know-it-all customers to retain them in a business. Explain two such strategies.

Answer

This question addresses various aspects of business communication, from document preparation to customer service.

(a) Indicators of a poorly summarized passage:

  • Inclusion of unnecessary details and examples
  • Missing key points or main ideas
  • Length exceeding recommended summary size
  • Lack of coherence and logical flow

(b) Reasons for writing business reports:

  • Document important business activities and decisions
  • Provide data for informed decision-making
  • Track progress and performance
  • Meet legal and regulatory requirements

(c) Work-from-home communication challenges:

  • Limited non-verbal communication cues
  • Technical issues affecting virtual meetings
  • Time zone differences in global teams
  • Reduced informal communication opportunities

(d) Information in job advertisements:

  • Job title and position details
  • Required qualifications and experience
  • Key responsibilities and duties
  • Application procedure and deadline
  • Company information and benefits
  • Salary range (if applicable)

(e) Situations requiring protocol observation:

  • Official ceremonies and events
  • Meetings with high-level executives
  • International business interactions
  • Formal presentations and speeches

(f) Strategies for handling know-it-all customers:

  • Active Listening: Pay attention to their concerns and acknowledge their knowledge
  • Respectful Guidance: Tactfully provide additional information or corrections
  • Collaborative Approach: Involve them in finding solutions
  • Professional Expertise: Demonstrate knowledge while maintaining humility

Question 8

(a) Identify the steps that constitute the process of communication.

(b) Outline three activities that may be carried out in an organization in preparation for conducting recruitment interviews.

(c) Distinguish between lateral communication and vertical communication.

Answer

This question addresses communication processes and organizational communication structures.

(a) Steps in the communication process:

1. Sender (Source)

Initiates the communication and formulates the message

2. Encoding

Converting thoughts into communicable symbols or language

3. Message

The encoded information to be transmitted

4. Channel

Medium through which the message is transmitted

5. Decoding

Receiver interprets and understands the message

6. Receiver

Person or group receiving and interpreting the message

7. Feedback

Response or reaction from the receiver

8. Context

Environment or situation in which communication occurs

(b) Recruitment interview preparation activities:

  • Review job descriptions and requirements
  • Prepare standardized interview questions
  • Schedule interview times and venues
  • Brief interview panel members
  • Review candidates' applications and CVs
  • Prepare evaluation criteria and forms
  • Arrange necessary logistics and resources
  • Set up interview room and materials

(c) Lateral vs. Vertical Communication:

Lateral Communication:

  • Occurs between people at the same organizational level
  • Facilitates coordination between departments
  • More informal and collaborative
  • Supports problem-solving and information sharing
  • Examples: peer-to-peer communication, cross-functional teams

Vertical Communication:

  • Flows up and down the organizational hierarchy
  • Involves superiors and subordinates
  • More formal and structured
  • Used for instructions, feedback, and reporting
  • Examples: manager instructions, employee reports

Question 12

(a) Explain each of the following terms as used in entrepreneurship:

(i) entrepreneurial culture;

(ii) enterprise social responsibility.

(b) Distinguish between an entrepreneur and an intrapreneur.

(c) The management of Turkey Works Limited has noticed that the employees are demotivated. Outline two measures that may be taken to address this situation.

Answer

This question examines key entrepreneurship concepts and employee motivation strategies.

(a)(i) Entrepreneurial culture:

An organizational environment that:

  • Encourages innovation and creativity
  • Supports risk-taking and initiative
  • Values opportunity recognition
  • Promotes continuous learning and improvement
  • Rewards entrepreneurial behavior

(a)(ii) Enterprise social responsibility:

A business's commitment to:

  • Ethical business practices
  • Environmental sustainability
  • Community development
  • Social welfare programs
  • Stakeholder interests beyond profit

(b) Entrepreneur vs. Intrapreneur:

Aspect Entrepreneur Intrapreneur
Risk Bears personal financial risk Company bears the risk
Resources Must acquire own resources Uses company resources
Independence Independent decision-maker Works within organization structure
Ownership Owns the business Employee of the organization

(c) Measures to address employee demotivation:

  1. Improve Work Environment:
    • Better working conditions
    • Enhanced workplace facilities
    • Positive organizational culture
    • Team building activities
  2. Career Development:
    • Training opportunities
    • Clear career progression paths
    • Skill development programs
    • Mentorship initiatives
  3. Incentive Programs:
    • Performance-based rewards
    • Recognition programs
    • Competitive compensation
    • Employee benefits
  4. Employee Engagement:
    • Regular feedback sessions
    • Participation in decision-making
    • Open communication channels
    • Work-life balance initiatives
