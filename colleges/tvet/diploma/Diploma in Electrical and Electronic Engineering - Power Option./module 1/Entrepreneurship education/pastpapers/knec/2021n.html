Question 1

(a) Explain each of the following types of computers:

(i) dedicated computer;

(ii) special purpose computer.

(b) Distinguish between computer input devices and computer output devices.

(c) State the function of each of the following computer software:

(i) utility programme;

(ii) operating system.

Answer

This question tests understanding of computer types, input/output devices, and software functions.

(a)(i) Dedicated Computer:

A computer designed for a specific task or application, such as:

  • Controlling industrial machinery
  • Managing network traffic
  • Handling medical imaging

(a)(ii) Special Purpose Computer:

A computer designed to solve a specific class of problems, such as:

  • Weather forecasting
  • Scientific simulations
  • Cryptography

(b) Distinction between input and output devices:

Input Devices:

  • Used to enter data and instructions into the computer
  • Examples: keyboard, mouse, scanner

Output Devices:

  • Used to display or present processed data from the computer
  • Examples: monitor, printer, speakers

(c) Function of computer software:

  • (i) Utility Programme: Performs specific tasks related to managing and maintaining computer resources, such as disk defragmentation, virus scanning, and file compression.
  • (ii) Operating System: Manages computer hardware and software resources, providing a platform for running applications and interacting with the system.

Question 2

(a) State two reasons that make it necessary to partition a computer hard disk.

(b) Outline the steps that should be followed to double underline a statement typed using a word processing application.

(c) With the aid of illustrations, explain the three drop cap options in word processing.

(d) Distinguish between copy and move as used in a word processing document.

Answer

This question covers hard disk partitioning, word processing formatting, and file manipulation.

(a) Reasons to partition a hard disk:

  • To install multiple operating systems on the same disk
  • To separate system files from user data for security and organization
  • To improve disk performance by reducing fragmentation
  • To create a recovery partition for system restoration

(b) Steps to double underline a statement:

Select the statement to be underlined
Go to the "Home" tab in the ribbon
In the "Font" group, click the dropdown arrow next to the "Underline" button
Select "Double Underline" from the options

(c) Drop cap options (with illustrations):

(i) Dropped: The first letter of the paragraph is dropped into the paragraph, occupying multiple lines.

(ii) In Margin: The first letter is placed in the left margin, next to the paragraph.

(iii) None: No drop cap effect is applied.

(d) Distinction between copy and move:

Copy: Creates a duplicate of the selected text or object at the new location, while the original remains unchanged.

Move: Removes the selected text or object from its original location and places it at the new location.

Question 3

(a) Explain the term data security.

(b) State two ways of improving data security in a learning institution.

(c) With the aid of a sketch, explain the structure of a work sheet in a spreadsheet application.

Answer

This question covers data security and spreadsheet application structure.

(a) Data Security:

Data security refers to the measures taken to protect data from unauthorized access, use, disclosure, disruption, modification, or destruction. It involves implementing policies, procedures, and technologies to ensure data confidentiality, integrity, and availability.

(b) Ways to improve data security in a learning institution:

  • Implementing strong password policies and access controls
  • Regularly backing up data and storing backups securely
  • Installing and updating antivirus software
  • Providing data security training to staff and students
  • Implementing firewalls and intrusion detection systems
  • Encrypting sensitive data both in transit and at rest

(c) Structure of a worksheet in a spreadsheet application:

A worksheet consists of a grid of cells organized into rows and columns:

  • Columns: Vertical lines labeled with letters (A, B, C, ...)
  • Rows: Horizontal lines labeled with numbers (1, 2, 3, ...)
  • Cells: Intersection of a row and column, identified by a cell address (e.g., A1, B2)
  • Formula Bar: Displays the content of the active cell, including formulas
  • Name Box: Shows the address of the active cell
  • Sheet Tabs: Allow navigation between multiple worksheets in a workbook

Question 4

(a) Explain two benefits of a computer network to an organization.

(b) State the purpose of each of the following objects in a database application:

(i) query;

(ii) form;

(iii) table.

(c) Highlight three merits of keeping employee records in a computerized database.

Answer

This question covers computer networks and database applications.

(a) Benefits of a computer network:

  • Resource Sharing: Allows multiple users to share hardware, software, and data, reducing costs and improving efficiency.
  • Communication and Collaboration: Facilitates communication and collaboration among employees through email, instant messaging, and shared documents.
  • Centralized Management: Enables centralized management of resources, security, and updates, simplifying administration.
  • Data Accessibility: Provides easy access to data from different locations, improving productivity and decision-making.

(b) Purpose of database objects:

  • (i) Query: Retrieves specific data from one or more tables based on defined criteria.
  • (ii) Form: Provides a user-friendly interface for entering, editing, and viewing data in a table.
  • (iii) Table: Stores data in a structured format with rows and columns.

(c) Merits of computerized employee records:

  • Improved Accuracy: Reduces errors compared to manual record-keeping.
  • Enhanced Security: Provides access controls and data encryption to protect sensitive information.
  • Efficient Retrieval: Allows quick and easy access to employee information.
  • Better Reporting: Facilitates generation of reports and analysis for decision-making.
  • Reduced Storage Space: Eliminates the need for physical storage space.

Question 5

(a) Distinguish between cropping and re-sizing in reference to a graphic in desktop publishing.

(b) Explain the purpose of each of the following features in a presentation package:

(i) slide sorter;

(ii) animation.

(c) The fiber optic cables have gained widespread popularity despite the existence of satellite links. Explain two reasons for this popularity.

Answer

This question covers desktop publishing, presentation software, and network communication technologies.

(a) Cropping vs. Re-sizing:

Cropping: Removes portions of an image, reducing its overall size and changing its composition.

Re-sizing: Changes the dimensions (height and width) of an image without removing any part of it, potentially distorting the image if proportions are not maintained.

(b) Purpose of presentation features:

  • (i) Slide Sorter: Displays all slides in a presentation as thumbnails, allowing easy reordering, deletion, and organization.
  • (ii) Animation: Adds visual effects to text and objects on a slide, controlling how they appear, move, or disappear during a presentation.

(c) Reasons for fiber optic popularity:

  • Higher Bandwidth: Fiber optic cables offer significantly higher bandwidth compared to satellite links, enabling faster data transmission rates.
  • Lower Latency: Fiber optic cables have lower latency than satellite links, resulting in faster response times and reduced delays.
  • Greater Security: Fiber optic cables are more secure than satellite links, as they are less susceptible to eavesdropping and interference.
  • Reliability: Fiber optic cables are less affected by weather conditions and atmospheric interference compared to satellite links.

SECTION B: COMMUNICATION SKILLS

Question 6 (Compulsory)

(a) Outline three consequences of ineffective internal communication in an organization.

(b) Distinguish between message encoding and message decoding in relation to the process of communication.

(c) State three benefits of grapevine communication to an organization.

(d) Write an essay of about 300 words on the topic, "The challenges that are faced by employees who work from home".

Answer

This question covers internal communication, the communication process, and remote work challenges.

(a) Consequences of ineffective internal communication:

  • Decreased employee morale and job satisfaction
  • Reduced productivity and efficiency
  • Increased conflicts and misunderstandings
  • Poor decision-making due to lack of information
  • Missed opportunities and innovation

(b) Message encoding vs. decoding:

Message Encoding: The process by which the sender translates their thoughts and ideas into a message using symbols, language, or other means of communication.

Message Decoding: The process by which the receiver interprets and understands the sender's message, translating the symbols or language back into meaningful thoughts and ideas.

(c) Benefits of grapevine communication:

  • Provides a channel for informal communication and feedback
  • Helps employees feel connected and informed
  • Can spread information quickly
  • May provide insights into employee concerns and attitudes

(d) Essay on challenges of working from home:

Working from home presents several challenges for employees. One major challenge is maintaining work-life balance, as the boundaries between work and personal life can become blurred, leading to overwork and burnout. Another challenge is social isolation, as remote workers may miss out on the social interactions and networking opportunities that come with working in a traditional office setting. Communication can also be a challenge, as remote workers may find it difficult to stay connected with colleagues and managers, leading to misunderstandings and delays. Additionally, remote workers may face technical difficulties, such as unreliable internet connections or lack of access to necessary equipment, which can hinder their productivity. Finally, remote workers may struggle with distractions at home, such as family members, household chores, or other environmental factors, making it difficult to focus on work.

Question 7

(a) Outline three non-verbal indicators of boredom during a face to face conversation.

(b) Explain the difference between an impulse customer and a need-based customer.

(c) State three benefits that the management of an organization may derive from conducting exit interviews.

Answer

This question covers non-verbal communication, customer behavior, and human resource practices.

(a) Non-verbal indicators of boredom:

  • Lack of eye contact or gazing around the room
  • Fidgeting or restlessness
  • Slouching or poor posture
  • Yawning or heavy sighs
  • Lack of facial expressions or a blank stare
  • Doodling or playing with objects

(b) Impulse vs. Need-based customer:

Impulse Customer:

  • Makes unplanned purchases based on immediate desires or emotions
  • Often influenced by marketing and display tactics
  • May not have a specific need for the product

Need-based Customer:

  • Purchases products or services to fulfill a specific need or solve a problem
  • Conducts research and compares options before making a decision
  • Driven by rational factors and practical considerations

(c) Benefits of exit interviews:

  • Gain insights into employee satisfaction and morale
  • Identify areas for improvement in the workplace
  • Obtain feedback on management practices and policies
  • Understand reasons for employee turnover
  • Improve employee retention strategies
  • Identify potential legal or ethical issues

Question 8

(a) Highlight three objectives the management of a business enterprise may seek to achieve through downward communication.

(b) Explain two ways in which an individual may demonstrate etiquette when answering a telephone call.

(c) State three roles played by the secretary in preparation for a meeting.

Answer

This question covers downward communication, telephone etiquette, and secretarial duties.

(a) Objectives of downward communication:

  • To provide instructions and guidelines to employees
  • To communicate organizational goals and objectives
  • To disseminate information about policies and procedures
  • To provide feedback on employee performance
  • To motivate and inspire employees

(b) Telephone etiquette:

  • Answering Promptly: Answer the call within a few rings to show respect for the caller's time.
  • Identifying Self: Clearly state your name and organization to provide clarity and professionalism.
  • Speaking Clearly: Use a clear and polite tone of voice, avoiding slang or jargon.
  • Active Listening: Pay attention to the caller's needs and respond appropriately.
  • Taking Accurate Messages: Record all necessary information accurately and legibly.
  • Ending Politely: Thank the caller for their time and end the conversation courteously.

(c) Roles of the secretary in meeting preparation:

  • Preparing and distributing the meeting agenda
  • Sending out meeting invitations and confirming attendance
  • Arranging the meeting venue and logistics
  • Preparing meeting materials and documents
  • Taking minutes of the meeting

SECTION C: ENTREPRENEURSHIP

Question 9

(a) Some individuals are reluctant to engage in self employment. Outline three reasons that may account for such reluctance.

(b) State three strategies that the management may adopt to establish an entrepreneurial culture in an organization.

(c) Distinguish between a fabian entrepreneur and an innovative entrepreneur.

Answer

This question covers reasons for reluctance in self-employment, strategies for entrepreneurial culture, and types of entrepreneurs.

(a) Reasons for reluctance in self-employment:

  • Fear of financial insecurity and income instability
  • Lack of access to capital and funding
  • Uncertainty about market demand and competition
  • Lack of business skills and experience
  • Fear of failure and potential losses
  • Preference for job security and benefits

(b) Strategies for entrepreneurial culture:

  • Encouraging risk-taking and experimentation
  • Providing resources and support for new ideas
  • Recognizing and rewarding innovation
  • Promoting collaboration and knowledge sharing
  • Creating a culture of learning and continuous improvement

(c) Fabian vs. Innovative entrepreneur:

Fabian Entrepreneur:

  • Cautious and skeptical about change
  • Imitates successful innovations of others
  • Avoids risks and prefers proven methods

Innovative Entrepreneur:

  • Introduces new products, services, or processes
  • Takes calculated risks and embraces change
  • Seeks to create new markets and opportunities

Question 10

(a) Certain guidelines should be followed to enhance the success of brainstorming as a means of generating business ideas. State three such guidelines.

(b) Outline three tangible resources that an entrepreneur may require when starting a business enterprise.

(c) Identify four types of records that may be maintained in a business enterprise.

Answer

This question covers brainstorming guidelines, tangible resources, and business records.

(a) Brainstorming guidelines:

  • Encourage all ideas, no matter how unconventional
  • Defer judgment and criticism
  • Focus on quantity over quality
  • Build on each other's ideas
  • Stay focused on the topic
  • Set a time limit

(b) Tangible resources for entrepreneurs:

  • Financial capital (cash, loans, investments)
  • Physical assets (equipment, machinery, buildings)
  • Inventory (raw materials, finished goods)
  • Technology (software, hardware, IT infrastructure)

(c) Types of business records:

  • Financial records (income statements, balance sheets, cash flow statements)
  • Sales records (invoices, sales reports, customer orders)
  • Inventory records (stock levels, purchase orders, supplier information)
  • Employee records (personnel files, payroll information, performance reviews)
  • Tax records (tax returns, payment records, compliance documents)
