SECTION A (40 marks)

Answer FOUR questions from this section.

Question 1

(a) State the technology used in each of the first four generations of computers.

(b) Explain the purpose of each of the following components of the central processing unit of a computer:

(i) Arithmetic Logic Unit.

(ii) Control Unit.

(c) Explain the term software as used in computing.

Answer

(a) Technologies used in computer generations:

  1. First Generation (1940-1956): Vacuum tubes
  2. Second Generation (1956-1963): Transistors
  3. Third Generation (1964-1971): Integrated Circuits (ICs)
  4. Fourth Generation (1971-present): Microprocessors (VLSI - Very Large Scale Integration)

(b) CPU Components:

(i) Arithmetic Logic Unit (ALU):

  • Performs all arithmetic operations (addition, subtraction, multiplication, division)
  • Executes logical operations (AND, OR, NOT, etc.)
  • Makes comparisons between numbers and data
  • Primary component for mathematical and logical computations

(ii) Control Unit:

  • Coordinates and controls all computer operations
  • Manages data flow between CPU and other components
  • Fetches, decodes, and executes instructions
  • Controls timing and sequence of operations

(c) Software:

Software refers to a collection of programs, data, and instructions that tell a computer how to perform specific tasks. It is the non-tangible component of a computer system that:

  • Controls computer operations
  • Processes data according to specified instructions
  • Enables interaction between users and hardware
  • Can be system software (operating systems) or application software (specific programs)

Question 2

(a) Describe each of the following operating system activities:

(i) formatting;

(ii) ScanDisk.

(b) Consider a list of test scores stored in a column of a spreadsheet in cells K6 to K50. State the formula that should be used to display each of the following based on the scores:

(i) mean score;

(ii) standard deviation;

(iii) highest score;

(iv) number of scores in the list.

Answer

(a) Operating System Activities:

(i) Formatting:

  • Process of preparing a storage device (hard drive, USB drive, etc.) for data storage
  • Creates a file system structure on the storage medium
  • Marks bad sectors to prevent data corruption
  • Can be quick format (removes file system table) or full format (checks for bad sectors)

(ii) ScanDisk:

  • Disk diagnostic tool that checks for file system errors and disk surface problems
  • Identifies and repairs cross-linked files and lost clusters
  • Verifies file system integrity
  • Checks for and marks bad sectors on the disk

(b) Spreadsheet Formulas:

(i) Mean score:

=AVERAGE(K6:K50)

Calculates the arithmetic mean of all scores in the range

(ii) Standard deviation:

=STDEV(K6:K50)

Calculates the standard deviation of the scores

(iii) Highest score:

=MAX(K6:K50)

Returns the highest value in the range

(iv) Number of scores:

=COUNT(K6:K50)

Counts the number of cells containing numerical values in the range

Question 3

(a) Explain each of the following types of computer crimes:

(b) Distinguish between sorting and querying as used in databases.

(c) State the use of each of the following field types in database design:

(i) memo;

(ii) Yes/No.

Answer

(a) Types of Computer Crimes:

  • Hacking: Unauthorized access to computer systems or networks with the intent to steal, modify, or destroy data
  • Software Piracy: Illegal copying, distribution, or use of copyrighted software without proper licensing or permission
  • Data Theft: Unauthorized acquisition of sensitive information such as personal data, credit card details, or trade secrets
  • Identity Theft: Using stolen personal information to impersonate someone else for financial gain or fraud

(b) Sorting vs. Querying in Databases:

Sorting Querying
Arranges records in a specific order (ascending or descending) Retrieves specific records based on certain criteria or conditions
Reorganizes existing data without filtering Filters data to show only records that match specific conditions
Example: Arranging names alphabetically Example: Finding all customers in a specific city

(c) Database Field Types:

(i) Memo Field:

  • Used for storing large amounts of text data
  • Can hold up to 65,535 characters
  • Suitable for comments, descriptions, or notes
  • Cannot be used as a primary key or indexed field

(ii) Yes/No Field:

  • Used for storing boolean values (True/False)
  • Represents binary choices (Yes/No, True/False, On/Off)
  • Takes minimal storage space
  • Commonly used for status flags or checkbox options

Question 4

(a) Explain the purpose of each of the following views in a presentation package:

(i) slide sorter;

(ii) slide show.

(b) State the steps involved in creating a mail merge.

Answer

(a) Presentation Package Views:

(i) Slide Sorter View:

  • Displays multiple slides in a grid format
  • Allows easy rearrangement of slides through drag and drop
  • Provides an overview of the entire presentation
  • Helps in organizing and managing slide sequence
  • Makes it easy to identify missing content or inconsistencies

(ii) Slide Show View:

  • Displays the presentation in full-screen mode
  • Shows slides as they will appear during the actual presentation
  • Enables testing of animations and transitions
  • Allows for rehearsal of timing and delivery
  • Provides access to presenter tools during presentation

(b) Steps for Creating a Mail Merge:

Step 1: Create the Main Document

  • Open word processor
  • Create or open the document template
  • Include placeholders for merge fields

Step 2: Set Up Data Source

  • Create or open data source (e.g., spreadsheet, database)
  • Organize data with appropriate column headers
  • Save the data source file

Step 3: Connect to Data Source

  • Select "Mail Merge" option
  • Choose data source type
  • Navigate to and select data file

Step 4: Insert Merge Fields

  • Place cursor where field should appear
  • Insert appropriate merge fields
  • Format merge fields as needed

Step 5: Preview and Complete Merge

  • Preview merged documents
  • Make any necessary adjustments
  • Complete merge to create final documents

Question 5

(a) State two disadvantages of using electronic mails for business communication.

(b) Describe the Ring computer network topology.

(c) State three advantages of using desktop publishing software.

(d) One of the challenges brought about by the emergence of ICT is e-waste. State three ways of managing e-waste.

Answer

(a) Disadvantages of Electronic Mail in Business:

  • Security concerns - vulnerability to hacking and interception
  • Risk of misinterpretation due to lack of non-verbal cues
  • Potential for information overload and spam
  • Dependency on internet connectivity and power supply

(b) Ring Network Topology:

  • Configuration where each device connects to exactly two other devices, forming a continuous loop
  • Data travels in one direction around the ring
  • Each device acts as a repeater, receiving and forwarding data
  • No central hub or switch required

Characteristics:

  • Equal access for all nodes
  • Simple data routing
  • Good performance with heavy network load
  • Vulnerable to single point of failure

(c) Advantages of Desktop Publishing Software:

  • Professional-quality layout and design capabilities
  • Precise control over text and graphic placement
  • Wide range of templates and design tools available
  • Cost-effective compared to professional printing services
  • Ability to make last-minute changes easily

(d) Managing E-Waste:

  • Recycling:
    • Proper disposal through certified recycling facilities
    • Recovery of valuable materials
    • Safe disposal of hazardous components
  • Refurbishment:
    • Repairing and upgrading old equipment
    • Extending device lifespan
    • Donating to schools or charities
  • Responsible Procurement:
    • Buying energy-efficient devices
    • Choosing products with longer lifespans
    • Selecting environmentally friendly options

Question 7

(a) Explain the purpose served by each of the following steps in the process of communication:

(i) message decoding;

(ii) supply of feedback.

(b) Explain each of the following types of meetings:

(i) board meeting;

(ii) working parties meeting;

(iii) staff meeting.

Answer

(a) Steps in Communication Process:

(i) Message Decoding:

  • Process of interpreting and understanding received messages
  • Converting symbols, words, or signals into meaningful information
  • Influenced by receiver's knowledge, experience, and context
  • Critical for accurate understanding of the message

(ii) Supply of Feedback:

  • Receiver's response to the received message
  • Confirms message reception and understanding
  • Enables two-way communication
  • Helps identify and correct miscommunication

(b) Types of Meetings:

(i) Board Meeting:

  • Formal gathering of organization's board of directors
  • Discusses strategic decisions and policy matters
  • Reviews company performance and governance
  • Usually follows strict protocols and procedures

(ii) Working Parties Meeting:

  • Temporary groups formed for specific projects or tasks
  • Focus on particular problems or initiatives
  • Members chosen for relevant expertise
  • Time-bound with specific objectives

(iii) Staff Meeting:

  • Regular gatherings of employees within a department or organization
  • Updates on operational matters and announcements
  • Platform for sharing information and concerns
  • Promotes team coordination and communication

(a)

(i) Distinguish between customer care and customer service.

(ii) Outline four customer care skills that an entrepreneur should possess.

(b) State four disadvantages of using fax for business communication.

Answer

(a)(i) Customer Care vs. Customer Service:

Customer Care Customer Service
Long-term relationship building Specific transaction or interaction handling
Proactive approach to customer needs Reactive response to customer requests
Focuses on overall customer experience Focuses on specific service delivery
Emotional and personal connection Professional and functional interaction

(a)(ii) Essential Customer Care Skills:

  • Active listening and empathy
  • Clear and effective communication
  • Problem-solving abilities
  • Patience and composure
  • Product/service knowledge

(b) Disadvantages of Fax Communication:

  • Quality Issues:
    • Poor reproduction of images and text
    • Loss of color and detail
  • Technical Limitations:
    • Requires dedicated phone line
    • Paper jams and machine malfunctions
  • Cost Factors:
    • Equipment maintenance expenses
    • Phone line charges
  • Security Concerns:
    • Documents visible to anyone near machine
    • No encryption of transmitted data
