SECTION A: ICT (40 marks)

Answer any FOUR questions from this section.

Question 1

(a) State four devices that are required for internet connectivity in a learning institution.

(b) Outline three ways in which configuration information may be stored in a motherboard.

(c) ABC company has connected its branches in two different countries through a network. However, the management has realised that someone is illegally gaining access to their data. Outline three ways through which the company may curb this vice.

Answer

(a) Required Internet Connectivity Devices:

  1. Router:
    • Directs network traffic
    • Connects local network to internet
    • Manages data flow between networks
  2. Modem:
    • Converts digital signals to analog and vice versa
    • Connects to ISP network
    • Provides internet access point
  3. Network Switch/Hub:
    • Connects multiple devices
    • Enables network expansion
    • Manages local network traffic
  4. Network Interface Cards (NICs):
    • Enables device connectivity
    • Provides network access points
    • Supports wired/wireless connections

(b) Motherboard Configuration Storage Methods:

1. CMOS (Complementary Metal-Oxide Semiconductor):

  • Stores basic system settings
  • Battery-powered for persistence
  • Contains date, time, and boot settings

2. BIOS/UEFI Firmware:

  • Stores system initialization data
  • Contains hardware configuration
  • Manages boot sequence

3. Flash Memory:

  • Stores firmware updates
  • Retains data without power
  • Allows for BIOS updates

(c) Network Security Measures:

1. Implement Strong Firewall Protection:

  • Monitor and filter network traffic
  • Block unauthorized access attempts
  • Set up access control rules

2. Use VPN (Virtual Private Network):

  • Encrypt data transmission
  • Create secure tunnel between branches
  • Prevent data interception

3. Implement Strong Authentication:

  • Use multi-factor authentication
  • Regular password updates
  • Access control policies

Question 2

(a) State four types of action queries used in databases.

(b) Outline three differences between primary and secondary storage.

Answer

(a) Types of Action Queries:

  1. Update Query:
    • Modifies existing records
    • Changes data based on specified criteria
    • Affects multiple records simultaneously
  2. Delete Query:
    • Removes records from database
    • Based on specified conditions
    • Permanent data removal
  3. Append Query:
    • Adds new records to tables
    • Copies data from one table to another
    • Creates new records based on criteria
  4. Make Table Query:
    • Creates new table from query results
    • Stores query output as new table
    • Used for data reorganization

(b) Primary vs Secondary Storage:

Feature Primary Storage Secondary Storage
Access Speed Very fast direct access Slower sequential access
Volatility Volatile (loses data when powered off) Non-volatile (retains data without power)
Capacity Limited capacity (GB range) Large capacity (TB range)
Cost More expensive per GB Less expensive per GB
Examples RAM, Cache Hard drives, SSDs, USB drives

Additional Differences:

  • Primary storage is directly accessible by CPU, secondary requires data transfer to RAM
  • Primary storage is essential for program execution, secondary for long-term data storage
  • Primary storage has faster data transfer rates but higher cost per unit of storage

Question 3

(a) Explain three factors that determine the clarity of a computer screen.

(b) Highlight four computer hard disk maintenance procedures that should be carried out to enhance computer performance.

Answer

(a) Factors Affecting Screen Clarity:

1. Resolution:

  • Number of pixels displayed (e.g., 1920 x 1080)
  • Higher resolution means sharper images
  • Affects text and image sharpness
  • Determines overall display quality

2. Refresh Rate:

  • Frequency of screen updates per second (Hz)
  • Higher rates reduce flickering
  • Affects motion smoothness
  • Impacts eye strain and comfort

3. Contrast Ratio:

  • Difference between brightest and darkest colors
  • Higher contrast improves readability
  • Affects color distinction
  • Impacts overall image quality

(b) Hard Disk Maintenance Procedures:

1. Disk Defragmentation:

  • Reorganizes file fragments
  • Optimizes file storage
  • Improves access speed
  • Reduces file fragmentation

2. Disk Cleanup:

  • Removes temporary files
  • Deletes unnecessary system files
  • Frees up disk space
  • Improves system performance

3. Error Checking:

  • Scans for bad sectors
  • Repairs file system errors
  • Maintains data integrity
  • Prevents data loss

4. Disk Space Management:

  • Regular file organization
  • Removal of duplicate files
  • Maintaining free space
  • Archive unused files

Question 4

(a) Outline two limitations of using a combo box when working with databases.

(b) Distinguish between sorting and filtering as used in spreadsheets.

(c) A student has forgotten the file name and location of a document previously saved in a computer. State four file details that are assigned to a file by the operating system which can assist the student to trace the file.

Answer

(a) Combo Box Limitations:

1. Limited Display Capacity:

  • Can only display a limited number of items at once
  • Scrolling required for long lists
  • May be difficult to navigate large datasets

2. Input Restrictions:

  • Limited to predefined values only
  • Cannot add new values directly
  • May not suit dynamic data requirements

(b) Sorting vs. Filtering in Spreadsheets:

Feature Sorting Filtering
Purpose Rearranges data in specific order Shows only data meeting specific criteria
Data Display Shows all data in new order Hides non-matching records
Data Organization Changes data arrangement Maintains original order
Result Reorganized complete dataset Subset of original data

(c) File Details for Tracing:

1. File Metadata:

  • Date created
  • Date modified
  • Date last accessed
  • File size

2. File Properties:

  • File type/extension
  • File attributes (read-only, hidden, etc.)
  • File owner/creator
  • Associated application

Search Tips:

  • Use Windows Search with file details
  • Filter by date ranges
  • Search by file type
  • Use advanced search options

Question 5

(a) Highlight four views used in a database.

(b) Distinguish between primary and secondary colours as used in desktop publishing.

(c) Highlight two functions of an uninterruptible power supply in a computing environment.

Answer

(a) Database Views:

1. Design View:

  • Create and modify table structure
  • Define field properties
  • Set primary keys and relationships
  • Establish field constraints

2. Datasheet View:

  • Display data in tabular format
  • Enter and edit records
  • Sort and filter data
  • Quick data manipulation

3. Form View:

  • User-friendly data entry interface
  • Display one record at a time
  • Customized data presentation
  • Enhanced data input control

4. Report View:

  • Formatted data presentation
  • Print-ready layout
  • Summarized information
  • Grouped and sorted data

(b) Primary vs. Secondary Colors:

Primary Colors Secondary Colors
Basic, cannot be created by mixing Created by mixing primary colors
Red, Blue, Yellow Green, Orange, Purple
Foundation of color theory Derived from primary colors
Used in base color schemes Provide color variety and depth

(c) UPS Functions:

1. Power Backup:

  • Provides temporary power during outages
  • Allows time for proper system shutdown
  • Prevents data loss from sudden power cuts

2. Power Conditioning:

  • Regulates voltage fluctuations
  • Filters electrical noise
  • Protects against power surges
  • Ensures stable power supply

Question 6 (Compulsory)

(a) Explain the meaning of each of the following terms as used in communication:

(i) listening;

(ii) interview;

(iii) feedback.

(b) Outline four reasons that make it necessary to write minutes of a meeting.

(c) Explain three differences between a written report and an oral report.

(d) The management of Obit Engineering Limited wishes to send a letter to its customers to inform them of a special offer on a certain machine. Highlight four types of information that should be contained in this letter.

Answer

(a) Communication Terms:

(i) Listening:

  • Active process of receiving and interpreting messages
  • Involves full attention and comprehension
  • Includes understanding verbal and non-verbal cues
  • Critical component of effective communication

(ii) Interview:

  • Formal meeting for gathering information
  • Structured conversation with specific purpose
  • Two-way communication between interviewer and interviewee
  • Used for assessment, selection, or information gathering

(iii) Feedback:

  • Response or reaction to received message
  • Indicates understanding or lack thereof
  • Essential for two-way communication
  • Helps improve communication effectiveness

(b) Importance of Meeting Minutes:

  1. Legal Documentation:
    • Official record of decisions
    • Evidence of proceedings
    • Legal reference point
  2. Reference Material:
    • Record of discussions
    • Action items tracking
    • Progress monitoring
  3. Accountability:
    • Task assignments
    • Responsibility allocation
    • Deadline tracking
  4. Communication Tool:
    • Information sharing
    • Decision dissemination
    • Absent member updates

(c) Written vs. Oral Reports:

Aspect Written Report Oral Report
Permanence Permanent record Temporary, unless recorded
Interaction No immediate feedback Immediate feedback possible
Reference Can be referenced later Relies on memory/notes

(d) Special Offer Letter Information:

  1. Product Details:
    • Machine specifications
    • Features and benefits
    • Technical information
  2. Offer Details:
    • Special price/discount
    • Offer duration
    • Terms and conditions
  3. Contact Information:
    • Company details
    • Contact person
    • Contact methods
  4. Call to Action:
    • How to take advantage
    • Response deadlines
    • Next steps

Etiquette Protocol
Social rules of behavior Official rules and procedures
Based on courtesy and manners Based on formal requirements
Flexible and adaptable Strict and standardized
Personal and social interactions Official and ceremonial events

Question 11

(a) Henrick has started a business enterprise without a business plan. State four problems that may arise from this situation.

(b) Explain three rules that should be followed during a brainstorming session in order to generate a good business idea.

Answer

(a) Problems of Starting Without a Business Plan:

1. Financial Management Issues:

  • Poor cash flow management
  • Inadequate funding estimation
  • Unplanned expenses
  • Difficult to secure financing

2. Strategic Direction Lacking:

  • Unclear business objectives
  • No defined target market
  • Absence of growth strategy
  • Poor resource allocation

3. Operational Challenges:

  • Disorganized processes
  • Inefficient resource utilization
  • Poor inventory management
  • Staff coordination issues

4. Market Entry Problems:

  • Inadequate market research
  • Unknown competition
  • Poor marketing strategy
  • Misunderstood customer needs

(b) Brainstorming Rules:

1. No Criticism:

  • Avoid judging ideas immediately
  • Accept all suggestions
  • Create safe environment for sharing
  • Encourage creative thinking

2. Encourage Quantity:

  • Focus on generating many ideas
  • Build on others' suggestions
  • Don't limit thinking
  • Quantity leads to quality

3. Stay Focused:

  • Keep to session objective
  • Maintain time limits
  • Follow structured approach
  • Document all ideas
