SECTION A: INFORMATION COMMUNICATION TECHNOLOGY (40 marks)

Answer any FOUR questions from this section.

(a) State two reasons for having an Information Communication Technology (ICT) department in an organization.

(b) Explain two benefits of using a bar code reader at a point-of-sale terminal.

(c) Explain each of the following types of software installation:

(i) Silent installation;

(ii) Clean installation.

Answer

(a) Two reasons for having an ICT department in an organization are:

  • To manage and maintain the organization's IT infrastructure, ensuring smooth operation of computer systems and networks.
  • To provide technical support to employees, resolving hardware and software issues and ensuring efficient use of IT resources.

(b) Two benefits of using a bar code reader at a point-of-sale terminal are:

  • Faster checkout process: Bar code readers quickly scan product codes, reducing the time it takes to process transactions.
  • Improved accuracy: Scanning bar codes eliminates the risk of manual data entry errors, ensuring accurate pricing and inventory management.

(c)

(i) Silent installation: A type of software installation that requires no user interaction. The installation process is automated and runs in the background without displaying any prompts or dialog boxes.

(ii) Clean installation: A type of software installation that involves completely removing any previous versions of the software and related files before installing the new version. This ensures a fresh and stable installation without conflicts from older files.

Question 2

(a) State two uses of a password on a computer.

(b) A student intends to transfer all files on the desktop of a personal computer to a new folder to be created on the desktop. State the steps that he should follow.

(c) State the steps followed when typing the symbol $\mathrm{H}_{2} \mathrm{O}$ in word processing.

Answer

(a) Two uses of a password on a computer are:

  • To restrict unauthorized access to the computer and its data.
  • To protect user accounts and prevent identity theft.

(b) The steps a student should follow to transfer all files on the desktop to a new folder are:

  1. Right-click on an empty area on the desktop.
  2. Select "New" and then "Folder" to create a new folder.
  3. Give the new folder a name (e.g., "Desktop Backup").
  4. Select all files on the desktop by pressing Ctrl + A.
  5. Right-click on the selected files and choose "Cut".
  6. Open the newly created folder.
  7. Right-click inside the folder and choose "Paste" to move the files into the folder.

(c) The steps followed when typing the symbol $\mathrm{H}_{2} \mathrm{O}$ in word processing are:

  1. Type "H".
  2. Use the "Subscript" formatting option (usually found in the "Font" section of the ribbon or toolbar).
  3. Type "2".
  4. Turn off the "Subscript" formatting option.
  5. Type "O".

Question 3

(a) Outline the procedure for printing a document created using a word processing application.

(b) Explain the relationship between the naming of cells in a spreadsheet and the naming of positions in a graph using the (x, y) coordinates.

Answer

(a) The procedure for printing a document created using a word processing application:

  1. Click on the "File" menu or press Ctrl + P
  2. Select the printer from the list of available printers
  3. Choose the number of copies to print
  4. Select the page range (all pages, current page, or specific pages)
  5. Set print options like orientation, paper size, and quality
  6. Preview the document to ensure correct formatting
  7. Click the "Print" button to start printing

(b) The relationship between cell naming in spreadsheets and graph coordinates:

  • In spreadsheets, cells are named using letters for columns (X-axis) and numbers for rows (Y-axis), e.g., A1, B2, C3
  • In graphs, positions are identified using (x,y) coordinates where:
    • x represents the horizontal position (similar to column letters)
    • y represents the vertical position (similar to row numbers)
  • Both systems use a two-dimensional grid reference system to uniquely identify locations
  • The main difference is that spreadsheets use alphanumeric references while graphs use numeric coordinates

Question 4

(a) (i) Explain the term data validation as used in databases.

(ii) State four ways of validating data during database creation.

(b) Figure 1. Shows a computer network topology. Use it to answer the questions that follow.

(i) Identify the computer network topology.

(ii) State three disadvantages of the topology.

Answer

(a) (i) Data validation, as used in databases, refers to the process of ensuring that data entered into a database meets certain criteria or rules. This helps to maintain the accuracy, consistency, and integrity of the data.

(ii) Four ways of validating data during database creation are:

  • Data type validation: Specifying the type of data that can be entered into a field (e.g., text, number, date).
  • Range validation: Setting limits on the values that can be entered into a field (e.g., a number between 1 and 100).
  • Required field validation: Ensuring that certain fields cannot be left blank.
  • Format validation: Checking that data is entered in a specific format (e.g., email address, phone number).

(b) (i) The computer network topology shown in Figure 1 is a Bus topology.

(ii) Three disadvantages of the Bus topology are:

  • Single point of failure: If the main cable breaks, the entire network goes down.
  • Difficult troubleshooting: Identifying the source of a problem can be difficult.
  • Limited scalability: Adding new devices can degrade performance.

Question 5

(a) Explain the use of "custom page size" option in relation to desktop publishing.

(b) Outline the process of importing text into a desktop publishing (DTP) file.

(c) Outline the process of running slides when presenting content to an audience.

Answer

(a) The "custom page size" option in desktop publishing:

  • Allows users to define non-standard page dimensions beyond preset sizes like A4 or Letter
  • Useful for creating specialized documents such as:
    • Brochures with unique fold patterns
    • Custom-sized banners or posters
    • Special invitation cards or certificates
  • Enables precise control over document dimensions to meet specific printing requirements

(b) The process of importing text into a desktop publishing (DTP) file:

  1. Create or prepare the text in a compatible format (e.g., .txt, .doc, .rtf)
  2. In the DTP software, select "Import" or "Place" from the File menu
  3. Browse and select the text file to import
  4. Choose import options (if available) such as formatting retention
  5. Place the text in the desired location in the document
  6. Format the imported text to match the document's style

(c) The process of running slides when presenting content to an audience:

  1. Launch the presentation software (e.g., PowerPoint, Google Slides)
  2. Open the presentation file
  3. Check presentation settings (resolution, display options)
  4. Enter slideshow mode (usually F5 or presentation icon)
  5. Navigate through slides using:
    • Arrow keys or mouse clicks
    • Presentation remote
    • Keyboard shortcuts (Page Up/Down)
  6. Use presentation tools as needed:
    • Laser pointer
    • Drawing tools
    • Speaker notes
  7. End the presentation (usually Esc key)

SECTION B: COMMUNICATION SKILLS (30 marks)

Answer question SIX (compulsory) and ONE other question from this section.

(a) Explain each of the following terms as used in communication:

(i) paragraph;

(ii) audience;

(iii) brainstorming.

(b) Explain the way in which each of the following forms of upward communication may be applied in an organisation:

(i) open door policy;

(ii) questionnaires and surveys.

(c) The management of Kazi Works Limited has noted reduced motivation levels among the production staff. As the Foreman, you have been requested to investigate the matter and write a report. Assuming that you have completed the investigations, write the report.

Answer

(a) Explanation of communication terms:

(i) Paragraph:

  • A distinct section of writing that focuses on a single main idea or topic
  • Contains a group of related sentences that develop the main point
  • Usually begins with a topic sentence and includes supporting details
  • Helps organize text into logical, manageable units for readers

(ii) Audience:

  • The intended recipients or readers of a communication message
  • Can be individuals, groups, or organizations
  • Understanding the audience helps in:
    • Selecting appropriate content and level of detail
    • Choosing suitable language and tone
    • Determining the most effective communication channel

(iii) Brainstorming:

  • A creative thinking technique used to generate multiple ideas quickly
  • Key characteristics include:
    • No criticism or evaluation during idea generation
    • Emphasis on quantity over quality initially
    • Building on others' ideas is encouraged
    • Wild or unusual ideas are welcome

(b) Application of upward communication forms:

(i) Open door policy:

  • Implementation methods:
    • Set and communicate specific times for employee meetings
    • Create a welcoming office environment
    • Respond promptly to employee concerns
    • Maintain confidentiality when appropriate
  • Benefits:
    • Encourages direct communication
    • Builds trust between management and employees
    • Allows quick resolution of issues

(ii) Questionnaires and surveys:

  • Implementation methods:
    • Design clear, focused questions
    • Use both closed and open-ended questions
    • Ensure anonymity when needed
    • Provide multiple response channels (online, paper)
  • Benefits:
    • Collects structured feedback from many employees
    • Allows quantitative analysis of responses
    • Provides documentation of employee input

Question 8

(a) Distinguish between a sub-committee and a joint committee.

(b) Baraka Enterprises has commissioned a consultancy firm to interview candidates for the position of foreman. Outline six guidelines that the consultant should follow when preparing for the interview.

Answer

(a) Distinction between sub-committee and joint committee:

Sub-committee Joint Committee
  • Formed from members of a main committee
  • Reports to the main committee
  • Handles specific tasks or aspects of work
  • Members are from the same organization
  • Has limited authority delegated by main committee
  • Formed by combining members from different organizations or groups
  • Reports to all participating organizations
  • Handles matters of mutual interest
  • Members represent different stakeholders
  • Has authority agreed upon by all participating groups

(b) Guidelines for interview preparation:

  1. Review Job Requirements:
    • Analyze the position description thoroughly
    • Identify key skills and qualifications needed
    • Understand the role's responsibilities
  2. Prepare Interview Questions:
    • Develop role-specific technical questions
    • Include behavioral and situational questions
    • Ensure questions are legal and appropriate
  3. Create Assessment Criteria:
    • Establish clear evaluation parameters
    • Design scoring system
    • Prepare assessment forms
  4. Arrange Logistics:
    • Schedule appropriate time slots
    • Book suitable interview location
    • Prepare necessary materials
  5. Review Applications:
    • Study candidates' CVs and applications
    • Note points requiring clarification
    • Prepare candidate-specific questions
  6. Plan Interview Structure:
    • Design interview format
    • Allocate time for different sections
    • Prepare introduction and conclusion
