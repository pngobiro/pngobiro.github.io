SECTION A: INFORMATION COMMUNICATION TECHNOLOGY

Answer any FOUR questions from this section.

Question 1

(a) Explain each of the following terms:

(i) computer network;

(ii) multi-user computer.

(b) On the computer keyboard, there are some keys that appear twice where one set is to the left of the keyboard and the other set is to the right of the keyboard.

(i) State two such keys.

(ii) Explain the reason for having such arrangement for those particular keys on the keyboard.

Answer

This question tests understanding of basic computer terms and keyboard layout.

(a) (i) Computer network: A set of interconnected computers that can share resources and data.

(ii) Multi-user computer: A computer that allows multiple users to access it simultaneously.

(b) (i) Two such keys: Shift and Ctrl keys.

(ii) Reason for arrangement: To allow easier access for both left and right-handed users.

Question 2

(a) Distinguish between installing software and launching software.

(b) Outline the steps of creating a folder on the desktop of a personal computer.

(c) State three ways in which audit trails may help improve data security.

Answer

This question tests understanding of software installation, folder creation, and data security.

(a) Installing software: The process of copying the software files to the computer and configuring it for use.

Launching software: The process of starting the software so that it can be used.

(b) Steps to create a folder:

  1. Right-click on the desktop.
  2. Select "New" from the context menu.
  3. Select "Folder" from the submenu.

(c) Ways audit trails improve security:

  • Tracking unauthorized access attempts.
  • Identifying data breaches.
  • Monitoring user activity.

Question 3

(a) An employee noted that the words typed in the upper case had spelling mistakes despite having a spelling and grammar checker in word processing.

(i) Explain the reason for this;

(ii) State the steps that should be followed to ensure that the spell checker is helpful in correcting the words in upper case.

(b) State the purpose of each of the following features in word processing programs:

(i) thesaurus;

(ii) caption.

(c) The cells M5 to M25 in a spreadsheet program contain grades of students in an examination. State the spreadsheet formula that should be used to display each of the following:

(i) the number of students who scored grade A;

(ii) the total number of students in the entire list;

(iii) the total number of students who scored grades $A$ and $B$.

Answer

This question covers spell checkers, word processor features, and spreadsheet formulas.

(a) (i) Reason for spelling mistakes: Spell checkers are often case-sensitive and may not recognize words in all caps.

(ii) Steps to ensure spell checker is helpful: Ensure the spell checker is set to ignore case or manually check words in uppercase.

(b) (i) Thesaurus: Provides synonyms for a selected word.

(ii) Caption: Adds a descriptive title to a table, figure, or other object.

(c) Spreadsheet formulas:

  • (i) Number of students who scored grade A: =COUNTIF(M5:M25,"A")
  • (ii) Total number of students: =COUNTA(M5:M25)
  • (iii) Total number of students who scored grades A and B: =COUNTIF(M5:M25,"A") + COUNTIF(M5:M25,"B")

Question 4

(a) State two benefits of using cables in a local area network instead of using wireless links.

(b) With the aid of examples, explain the use of each of the following operators in databases:

(i) AND

(ii) OR

(c) An employee was trying to open an email but had forgotten the password. Outline the process of regaining access to the email.

Answer

This question covers network cables, database operators, and email password recovery.

(a) Benefits of using cables:

  • Higher bandwidth and data transfer speeds.
  • More secure and less susceptible to interference.

(b) Database operators:

  • (i) AND: Returns true only if both conditions are true. Example: SELECT * FROM employees WHERE salary > 50000 AND age < 30;
  • (ii) OR: Returns true if either condition is true. Example: SELECT * FROM employees WHERE department = "Sales" OR department = "Marketing";

(c) Process of regaining access to email:

  1. Click on "Forgot Password" link.
  2. Follow the prompts to verify identity, usually via email or phone.
  3. Create a new password.

Question 5

(a) Outline the process of designing staff business cards by use of a desktop publishing program.

(b) A motivational speaker intends to set presentation slides to run automatically. State the steps that should be followed to achieve this objective.

(c) Some banks have introduced card less banking on their ATMs. However, the ATMs still have card slots. Explain two advantages of running the ATM card services alongside the card-less ATM services.

Answer

This question covers desktop publishing, presentation automation, and ATM services.

(a) Process of designing business cards:

  1. Open a desktop publishing program
  2. Create a new document with business card dimensions
  3. Add text boxes for name, title, contact information
  4. Add logo or other graphics
  5. Format text and graphics
  6. Save and print

(b) Steps to automate presentation slides:

  1. Open the presentation
  2. Go to the "Transitions" tab
  3. Set the "Advance Slide" option to "After" and specify the time interval
  4. Apply the settings to all slides

(c) Advantages of dual ATM services:

  • Caters to customers who prefer using cards
  • Provides a backup option if card-less services are temporarily unavailable

Question 6 (Compulsory)

(a) Explain each of the following phrases as used in communication:

(i) active listening;

(ii) communication flow.

(b) Explain the way in which each of the following barriers to effective communication may be minimized in an organization:

(i) wrong choice of medium;

(ii) information overload;

(iii) human relations.

(c) Write an essay of about 250 words on the topic, "The Importance of Budgeting in a Business Enterprise".

Answer

This question covers key communication concepts and the importance of budgeting.

(a) (i) Active listening: Paying close attention to what someone is saying and demonstrating understanding through verbal and non-verbal cues.

(ii) Communication flow: The direction and path that information takes as it moves through an organization.

(b) Minimizing communication barriers:

  • (i) Wrong choice of medium: Select the most appropriate medium for the message and audience.
  • (ii) Information overload: Filter and prioritize information, and use summaries.
  • (iii) Human relations: Foster open communication, empathy, and trust.

(c) The Importance of Budgeting in a Business Enterprise:

Budgeting is crucial for business enterprises as it provides a roadmap for financial planning and control. It enables businesses to allocate resources effectively, monitor income and expenses, and make informed decisions. A well-crafted budget helps in setting realistic goals, identifying potential risks, and securing funding. It also facilitates performance evaluation and accountability. By comparing actual results with budgeted figures, businesses can identify areas of improvement and take corrective actions. In essence, budgeting promotes financial stability, growth, and sustainability for business enterprises.
