INSTRUCTIONS TO CANDIDATES

  • This paper consists of EIGHT questions in THREE sections; A, B and C.
  • Follow the instructions provided at the beginning of each section.
  • All answers should be written in the answer booklet provided.
  • Maximum marks for each part of a question are as indicated.
  • Candidates should answer the questions in English.

SECTION A: ICT (40 marks)

Answer FOUR questions from this section.

1. (a) (i) Explain the term booting as used in computers.

(ii) Outline two circumstances that could necessitate warm booting.

(b) Figure 1 shows a section of a typical toolbar in a spreadsheet program. Use it to answer the question that follows.

Explain the function of each of the features labelled (i) and (ii).

(c) James, a technician in an engineering company would like to buy CAD software for office use. Explain three factors that he could put into consideration.

(d) (i) With the aid of a sketch in each case, differentiate between flip over and flip up print options as used in desktop publishing programs.

(ii) John designed a publication with an object and would like to text wrap it. State four text wrapping options available.

Answers:

(a) (i) Booting is the process of starting up a computer and loading the operating system into memory.

(a) (ii) Circumstances for warm booting:

  • When a program freezes or becomes unresponsive
  • After installing new software that requires a restart

(b) Features in Fig. 1:

  • (i) Sort ascending button - Arranges selected data in ascending order
  • (ii) Sort descending button - Arranges selected data in descending order

(c) Factors to consider when buying CAD software:

  • Cost of the software and licensing
  • Hardware requirements and compatibility
  • Features and capabilities needed for the specific work

(d) (i) Print options:

  • Flip over - Rotates the page horizontally (left to right)
  • Flip up - Rotates the page vertically (top to bottom)

(d) (ii) Text wrapping options:

  • Square
  • Tight
  • Through
  • Top and Bottom

2. (a) Explain each of the following data types as used in database programs:

(i) auto number;

(ii) memo.

(b) (i) Figure 2 shows a section of typical toolbar in a presentation program. Use it to answer the question that follows:

Outline the function of the features labelled (i) and (ii).

(ii) Outline one reason that could make an engineer demonstrate a project report using a presentation program.

(c) Figure 3 shows data in a spreadsheet program. Use it to answer the questions that follow:

A B C D
1 MATERIAL CATEGORY PRICE Discount Amount
2 cement A 35,000
3 ballast A 27,000
4 concrete A 22,000
5 iron sheets B 36,000
6 sand A 45,000
7 water C 15,000
8 doors B 30,000
9 DISCOUNT 10%
10 TOTAL PRICE FOR CATEGORY B MATERIALS
11 NUMBER OF MATERIALS IN CATEGORY A

(i) Using cell referencing only, write a function that could be used to:

(I) Calculate the total price for category B materials.

(II) Display the number of category A materials.

(ii) Using cell referencing only write a function that could be used to calculate the discount amount. Assume that a discount of 10% is offered on materials in category B.

(d) Differentiate between datasheet and design view options of a table in a database program.

Answers:

(a) Database data types:

(i) Auto number - Automatically generates a unique sequential number for each record

(ii) Memo - Stores long text or combinations of text and numbers

(b) (i) Toolbar features:

  • (i) Slide sorter view - Shows thumbnails of all slides for easy reorganization
  • (ii) Normal view - Shows the default editing view with slides, outline, and notes

(b) (ii) Reason for using presentation program:

To present complex technical information in a visual and easily understandable format

(c) (i) Spreadsheet functions:

(I) Total price for category B: =SUMIF(B2:B8,"B",C2:C8)

(II) Number of category A materials: =COUNTIF(B2:B8,"A")

(c) (ii) Discount calculation:

=IF(B2="B",C2*$B$9,0)

(d) Database views:

  • Datasheet view - Shows data in rows and columns like a spreadsheet
  • Design view - Shows the structure of the table where you can modify field properties

3. (a) (i) Define each of the following features as used in a word processing program:

(I) Watermark;

(II) Drop cap.

(ii) Differentiate between hanging and first line indent.

(b) Differentiate between shareware and freeware as used in information security.

(c) When interacting with a newly developed database system, Janet, an engineering company technician came across the following database objects:

(i) a graphical interface object to display, edit and insert data;

(ii) an object to select/retrieve a record from one or more tables;

(iii) an object to output data arranged in a specified order;

(iv) an object that organizes information in rows and columns.

Identify the database object described in each case.

(d) Passwords could be used for data security, however they have some weakness. Explain three weaknesses associated with passwords.

Answers:

(a) (i) Word processing features:

(I) Watermark - A faint design or text that appears behind the main text on a page

(II) Drop cap - An enlarged capital letter at the beginning of a paragraph that drops through multiple lines

(a) (ii) Indentation types:

  • Hanging indent - All lines except the first line are indented
  • First line indent - Only the first line is indented

(b) Software types:

  • Shareware - Software that can be tried for free but requires payment for continued use
  • Freeware - Software that is completely free to use without payment

(c) Database objects:

  • (i) Form - Graphical interface for data entry and editing
  • (ii) Query - Tool for selecting and retrieving specific records
  • (iii) Report - Output format for organized data presentation
  • (iv) Table - Basic structure for storing data in rows and columns

(d) Password weaknesses:

  • Can be forgotten or lost
  • Can be guessed if too simple or personal
  • Can be stolen through various hacking methods

4. (a) Explain each of the following information security features:

(i) replication;

(ii) backup.

(b) (i) A certain engineering firm has the following ICT equipment:

  • Plotter
  • Scanner
  • MICR
  • Monitor
  • Sensor
  • Joystick

Classify the equipment as either input or output devices.

(ii) Outline activities that a user could carry out when caring for a computer mouse.

(c) (i) Differentiate between centre and justify text alignment options as used in a word processing program.

(ii) Explain the function of the format painter feature in a word processing program.

(d) Explain the function of each of the following utility software:

(i) disk cleanup;

(ii) defragmenter;

(iii) linker;

(iv) compiler.

Answers:

(a) Information security features:

  • (i) Replication - Creating and maintaining multiple copies of data across different locations
  • (ii) Backup - Creating copies of data that can be used to restore information if original data is lost

(b) (i) Device classification:

Input devices:

  • Scanner
  • MICR (Magnetic Ink Character Recognition)
  • Sensor
  • Joystick

Output devices:

  • Plotter
  • Monitor

(b) (ii) Mouse care activities:

  • Regular cleaning of the mouse and mouse pad
  • Proper storage when not in use
  • Avoiding rough handling

(c) (i) Text alignment:

  • Center - Text is centered between margins
  • Justify - Text is aligned to both left and right margins

(c) (ii) Format painter function:

Copies formatting from one selection to another, allowing consistent formatting across text

(d) Utility software functions:

  • (i) Disk cleanup - Removes unnecessary files to free up disk space
  • (ii) Defragmenter - Reorganizes fragmented files to improve disk performance
  • (iii) Linker - Combines object code files into an executable program
  • (iv) Compiler - Translates high-level code into machine code

5. (a) Outline four challenges of using speech recognition software.

(b) (i) An engineering company with several offices occupying three floors of a building is contemplating automating their operations. Explain three circumstances that would influence the interconnection of their computers.

(ii) With the aid of a diagram in each case, distinguish between star and bus topology as used in computers.

(c) Figure 4 shows an extract from a DTP program. Use it to answer the question that follows:

The Construction Industry

The construction industry is very important for the Kenya economy. It accounts for 5% of the country's GDP and employs about 1 million people with an estimated annual wage bill of Sh32 billion.

According to recent findings by National Bureau of Statistics (NBS), Kenya's economy grew by 4.9% in the first quarter of 2011 thanks to the improved productivity in the construction industry.

NBS reported that the industry contributed Sh12.6 billion to countries GDP in 1st quarter of 2011 buoyed by the large road infrastructure projects currently under way across the country.

The construction was also reflected in cement consumption which rose from 77.3 million tonnes up from 667.1 million tonnes consumed in 2010.

This has seen an increase on the demand for other engineering professionals to be subcontracted in this industry.

Identify four formatting features applied in the extract.

(d) Name two devices used for data processing in the pre-computer era.

Answers:

(a) Challenges of speech recognition software:

  • Difficulty with accents and pronunciation
  • Background noise interference
  • Accuracy issues with technical terms
  • Training requirements for optimal performance

(b) (i) Circumstances influencing computer interconnection:

  • Physical distance between offices
  • Network traffic volume expected
  • Security requirements

(b) (ii) Network topologies:

Star topology - All devices connect to a central hub/switch

Bus topology - All devices connect to a single cable backbone

(c) Formatting features in the extract:

  • Bold heading
  • Justified paragraph alignment
  • Multiple font sizes
  • Consistent spacing between paragraphs

(d) Pre-computer era data processing devices:

  • Abacus
  • Mechanical calculators

SECTION B: COMMUNICATION SKILLS (30 marks)

Answer question SIX (compulsory) and any other ONE question from this section.

6. (a) State two reasons that make it necessary for organizations to place advertisements in the newspaper.

(b) Identify three methods that may be used to convey oral messages in an organization.

(c) Highlight three benefits of using video conferencing as a method of communication.

(d) (i) List two advantages of effective upward communication in an organization.

(ii) State two ways in which the management of an organization may control the harmful effects of grapevine communication.

(e) Most managers prepare and circulate an agenda in preparation for a meeting. Explain two reasons that may account for this practice.

(f) List two areas on which an interviewee may ask questions during an interview, if given a chance.

(g) Explain two ways in which the medium of communication may bar effective transmission of the message.

(h) Outline two methods that may be used to collect data for report writing.

Answers:

(a) Reasons for newspaper advertisements:

  • Wide reach to target audience
  • Cost-effective mass communication

(b) Methods for oral messages:

  • Face-to-face conversations
  • Telephone calls
  • Public address systems

(c) Benefits of video conferencing:

  • Reduces travel costs and time
  • Enables real-time visual communication
  • Facilitates immediate feedback

(d) (i) Advantages of upward communication:

  • Provides feedback from employees
  • Helps identify problems early

(d) (ii) Controlling grapevine communication:

  • Regular official communication
  • Open door policy

(e) Reasons for circulating agenda:

  • Allows participants to prepare adequately
  • Ensures meeting stays focused and organized

(f) Interview question areas:

  • Career growth opportunities
  • Job responsibilities

(g) Communication medium barriers:

  • Technical failures or breakdowns
  • Noise or interference

(h) Data collection methods:

  • Questionnaires
  • Interviews

7. (a) Distinguish between internal and external communication. Give an example in each case.

(b) As the communication assistant in Pengo Engineering Works, state three guidelines you would follow when designing notices for communication.

(c) Mak Engineering Limited has received an assortment of equipment as donation from a local Non-Governmental Organization (NGO). As the communications officer, write a letter to the management of the NGO acknowledging receipt of the equipment and appreciate the kind gesture.

Answers:

(a) Types of communication:

  • Internal communication - Communication within an organization (e.g., memos between departments)
  • External communication - Communication with outside parties (e.g., letters to customers)

(b) Notice design guidelines:

  • Clear and concise language
  • Appropriate font size and style
  • Strategic placement for visibility

(c) Sample acknowledgment letter:

[Your Address]
[Date]

The Director
[NGO Name]
[NGO Address]

Thank you for your kind gesture.

Yours faithfully,

8. (a) Communication can be defined as a process which involves the transmission and accurate replication of ideas ensured by feedback for the purpose of eliciting actions which will accomplish organisational goals.

State two important points emphasized in this definition.

(b) Architect Kwetu has requested for your advice in choosing a medium of communication. Suggest two factors that he should consider when making an appropriate choice.

(c) The management of Bandon Engineering Works Limited emphasizes on presenting written instead of oral reports. Explain three reasons for this emphasis.

Answers:

(a) Important points in definition:

  • Feedback is essential in communication
  • Communication aims to achieve organizational goals

(b) Factors for choosing communication medium:

  • Nature and urgency of the message
  • Target audience accessibility

(c) Reasons for written reports:

  • Provides permanent record for future reference
  • Reduces misinterpretation of information
  • Enables wider distribution to stakeholders

SECTION C: ENTREPRENEURSHIP EDUCATION (30 marks)

Answer any THREE questions from this section.

9. (a) State five reasons that may make a business plan fail.

(b) Outline five strategies that an organization may take to demonstrate its responsibility to the community living around its area of operation.

Answers:

(a) Reasons for business plan failure:

  • Unrealistic financial projections
  • Inadequate market research
  • Poor risk assessment
  • Lack of clear objectives
  • Insufficient implementation strategy

(b) Corporate social responsibility strategies:

  • Environmental conservation initiatives
  • Supporting local education programs
  • Creating employment opportunities
  • Contributing to community projects
  • Sponsoring community events

10. (a) Outline four measures that an organization may adopt to manage its business finances effectively.

(b) State six steps that may be followed to start a small enterprise.

Answers:

(a) Financial management measures:

  • Regular financial auditing
  • Maintaining proper accounting records
  • Budget planning and control
  • Cash flow monitoring

(b) Steps to start small enterprise:

  • Conduct market research
  • Develop business plan
  • Secure funding
  • Register the business
  • Set up business location
  • Hire necessary staff

11. (a) State four contributions of entrepreneurship to the economic development of a country.

(b) Explain three conditions that may make a business opportunity viable.

Answers:

(a) Contributions of entrepreneurship:

  • Job creation
  • Innovation and technological advancement
  • Increased tax revenue
  • Economic growth and development

(b) Conditions for business viability:

  • Market demand for product/service
  • Availability of required resources
  • Favorable economic conditions

12. (a) State four limitations of using e-commerce in business.

(b) Explain three ways in which the political environment of a country may negatively influence entrepreneurial development.

Answers:

(a) E-commerce limitations:

  • Security concerns
  • Limited internet access in some areas
  • Technical infrastructure requirements
  • Customer trust issues

(b) Political environment effects:

  • Unstable policies affecting business planning
  • Excessive regulations hindering growth
  • Political uncertainty affecting investment
