Example 2.14 : A coil with a resistance of $8 \Omega$ is connected in the direct connection mode of Q meter. Resonance occurs when the oscillator frequency is 1.5 MHz and the resonating capacitor is set at 60 pF . Calculate the $\%$ error introduced in the calculated value of Q by the $0.02 \Omega$ insertion resistance.

Solution :

The true value of Q is,

$$\begin{aligned} \mathrm{Q}_{\text {actual }} &=\frac{1}{\omega \mathrm{CR}}=\frac{1}{2\pi \times 15 \times 10^{6} \times 60 \times 10^{-12} \times 8}\\ &=221.05 \end{aligned}$$
$$\begin{aligned} \mathrm{Q}_{\text {obscrved }}&=\frac{1}{\omega \mathrm{C}\left(\mathrm{R}+\mathrm{R}_{\mathrm{SH}}\right)}\\ &=\frac{1}{2\pi \times 15 \times 10^{6} \times 60 \times 10^{-12} \times(8+0.02)}=220.497 \end{aligned}$$
$$\begin{aligned} \% \text { error }&=\frac{\text { true value }- \text { measured value }}{\text { true value }} \times 100\\ &=\frac{221.05-220.497}{221.05} \times 100\\ &=0.25 \% \end{aligned}$$

Example 2.36:A coil with a resistance of $10 \Omega$ is connected in the 'direct connection' in a Q meter. Resonance occurs when the oscillator frequency is 1 MHz and the resonating capacitor is set to 65 pF . Calculate the $\%$ error introduced in the calculated value of Q by $0.02 \Omega$ insertign resistance.

Solution :

The value of Q is,

$$\mathrm{Q}=\frac{\mathrm{X}_{\mathrm{L}}}{\mathrm{R}}=\frac{\mathrm{X}_{\mathrm{C}}}{\mathrm{R}}$$
$$=\frac{1}{\omega \mathrm{CR}}$$

Now $\quad \omega=2\pi \mathrm{f}=2\pi \times 1 \times 10^{6} \mathrm{rad} / \mathrm{sec}$
$\mathrm{C}=65 \mathrm{pF}$
$\mathrm{R}=10 \Omega$

$$\begin{aligned} \mathrm{Q}_{\text {actual }}&=\frac{1}{\omega \mathrm{CR}}=\frac{1}{2\pi \times 10^{6} \times 65 \times 10^{-12} \times 10}\\ &=244.853 \end{aligned}$$
$$\begin{aligned} \mathrm{Q}_{\text {measured }}&=\frac{1}{\omega \mathrm{C}\left(\mathrm{R}+\mathrm{R}_{\mathrm{SH}}\right)}=\frac{1}{2\pi \times 10^{6} \times 65 \times 10^{-12} \times(10+0.02)}\\ &=244.365 \end{aligned}$$
$$\begin{aligned} \% \text { error }&=\frac{\mathrm{Q}_{\text {actual }}-\mathrm{Q}_{\text {measured }}}{\mathrm{Q}_{\text {actual }}} \times 100=\frac{244.853-244.365}{244.853} \times 100\\ &=0.2 \% \end{aligned}$$
