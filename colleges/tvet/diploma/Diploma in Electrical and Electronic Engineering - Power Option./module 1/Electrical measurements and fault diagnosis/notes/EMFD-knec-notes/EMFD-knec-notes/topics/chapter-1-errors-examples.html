EXAMPLE

A moving coil ammeter has a uniform scale with 50 divisions and gives a full-scale reading of 5 A . The instrument can read up to $V$ th of a scale division with a fair degree of certainty. Determine the resolution of the instrument in $m A$.

Solution

Full-scale reading = 5 A
Number of divisions on scale = 50

$$1 \text { scale division }=\frac{5}{50} \times 1000=100 \mathrm{~mA}$$
$$\text { Resolution }=1 / 4 \text { th of a scale division }=\frac{100}{4}=25 \mathrm{~mA}$$

Speed of Response

The quickness of an instrument to read the measured variable is called the speed of response. Alternately, speed of response is defined as the time elapsed between the start of the measurement to the reading taken.

Errors in Measurement

In order to understand the concept of errors in measurement, we should know the two terms that defines the error and these two terms are written below:

True Value

It is not possible to determine the true value of quantity by experiment means. True value may be defined as the average value of an infinite number of measured values when average deviation due to various contributing factor will approach to zero.

Measured Value

It may be defined as the approximated value of true value. It can be found out by taking means of several measured readings during an experiment, by applying suitable approximations on physical conditions.. Static error is defined as the difference of the measured value and the true value of the quantity

Static error = measured value - true value. defined as the difference btn the measured value and true value of quantity i.e $\delta A=A_{m}-A_{t}$
Where,

$$\delta \mathrm{A}=\text { error }$$
$$\begin{aligned} A_{m} &=\text { measured value } \\ A_{t} &=\text { true value } \end{aligned}$$

$\delta \mathrm{A}$ - static error of quantity $A$ under measurement

Sometimes, $\delta \mathrm{A}$ is denoted by $\varepsilon 0$.(absolute error $=up$ or down value)
The relative error is the ratio of absolute error to the true value of the unknown quantity to be measured,

i.e relative error $=\varepsilon_{\mathrm{r}}=\delta \mathrm{A} / \mathrm{At}=\varepsilon 0 / \mathrm{At}=$ Absolute value /true value
Quantity of measurement is provided by relative static error

When the absolute error $\varepsilon 0(=\delta \mathrm{A})$ is negligible, i.e., when the difference between the true value At and the measured value $A m$ of the unknown quantity is very small or negligible then the relative error may be expressed as,

$$\varepsilon_{\mathrm{r}}=\frac{\delta A}{A_{m}}=\frac{\varepsilon_{0}}{A_{m}}$$

The relative error is generally expressed as a fraction, i.e., 5 parts in 1000 or in percentage value,

$$\begin{aligned} \text { i.e., percentage error } &=\varepsilon_{r} \times 100=\frac{\varepsilon_{0}}{A_{m}} \times 100\\ \boldsymbol{\delta} \boldsymbol{A} &=\boldsymbol{A}_{\boldsymbol{m}}-\boldsymbol{A}_{\boldsymbol{t}}\\ \boldsymbol{A}_{\boldsymbol{t}}&=\boldsymbol{A}_{\boldsymbol{m}}-\boldsymbol{\delta} \boldsymbol{A}\\ &=\boldsymbol{A}_{\boldsymbol{m}}-\boldsymbol{\varepsilon}_{\mathbf{o}}\\ &=\boldsymbol{A}_{\boldsymbol{m}}-\boldsymbol{\varepsilon}_{\mathrm{r}} \cdot \mathbf{A}_{\mathbf{t}}\\ \boldsymbol{A}_{m}&=\boldsymbol{A}_{t} /\left(\mathbf{1}+\boldsymbol{\varepsilon}_{\mathbf{r}}\right)\\ \text {So, } \mathbf{A}_{\mathbf{t}}&=\mathbf{A}_{\mathbf{m}} / \mathbf{( 1 - \varepsilon _ { r } )} \end{aligned}$$

The magnitude of a given quantity having a specifiea magnitude $A m$ and a maximum or a limiting error $\pm \delta \mathrm{A}$ must have a magnitude between the limits

$$\begin{array}{cc} & A_{m}-\delta A \text { and } A_{m}+\delta A \\ \text { or } & A=A_{m} \pm \delta A \end{array}$$

Static correction ($\delta C $)

$$\begin{aligned} &\delta C=\mathbf{A}_{t}-\mathbf{A}_{\mathrm{m}}\\ &\mathbf{A}_{\mathrm{t}}=\text { true value }\\ &\mathbf{A}_{\mathrm{m}}=\text { measured value } \end{aligned}$$

For example, the measured value of a resistance of $100 \Omega$ has a limiting error of $\pm 0.5 \Omega$. Then the true value of the resistance is between the limits $100 \pm 0.5$, i.e., 100.5 and $99.5 \Omega$.

EXAMPLE

A 0-25 A ammeter has a guaranteed accuracy of 1 percent of full scale reading. The current measured by this instrument is 10 A . Determine the limiting error in percentage.

Solution

The magnitude of limiting error of the instrument is $\delta A=\varepsilon r \times A=0.01 \times 25=0.25 \mathrm{~A}$
The magnitude of the current being measured is 10 A . The relative error at this current is

$$\varepsilon_{r}=\frac{\delta A}{A}=\frac{0.25}{10}=0.025$$

Therefore, the current being measured is between the limit of

$$A=A_{m}\left(1 \pm \varepsilon_{r}\right)=10(1 \pm 0.025)=10 \pm 0.25 \mathrm{~A}$$
$$\text { The limiting error }=\frac{0.25}{10} \times 100=2.5 \%$$

EXAMPLE

The measurand value of a resistance is $10.25 \Omega$, whereas its value is $10.22 \Omega$. Determine the absolute error of the measurement.

Solution

Measurand value $A_{m}=10.25 \Omega$

EXAMPLE

The measured value of a capacitor is $205.3 \mu \mathrm{F}$, whereas its true value is $201.4 \mu \mathrm{F}$. Determine the relative error.

Solution

Measured value $A_{m}=205.3 \times 10^{-6} \mathrm{~F}$
True value, A $=201.4 \times 10^{-6} \mathrm{~F}$
Absolute error, $\varepsilon_{0}=A_{m}-A$

$$\begin{aligned} &=205.3 \times 10^{-6}-201.4 \times 10^{-6}\\ &=3.9 \times 10^{-6} \mathrm{~F}\\ &=3.9 \times 10^{-6} \mathrm{~F} \end{aligned}$$

Relative error, $\varepsilon_{r}=\frac{\varepsilon_{0}}{A}=\frac{3.9 \times 10^{-6}}{201.4 \times 10^{-6}}=0.0194$ or $1.94 \%$
