Chapter 4: Error Analysis and Calculations

📐 Basic Error Concepts

True Value and Measured Value

True value may be defined as the average value of an infinite number of measured values when average deviation due to various contributing factors will approach zero.

Measured value is defined as the approximated value of true value. It can be found out by taking means of several measured readings during an experiment.

Static Error

Static error is defined as the difference between the measured value and the true value of the quantity:

$\delta A = A_m - A_t$

Where:

  • • $\delta A$ = error
  • • $A_m$ = measured value
  • • $A_t$ = true value

📊 Relative Error

The relative error is the ratio of absolute error to the true value of the unknown quantity to be measured:

$\varepsilon_r = \frac{\delta A}{A_t} = \frac{\varepsilon_0}{A_t}$

When the absolute error $\varepsilon_0$ is negligible, the relative error may be expressed as:

$\varepsilon_r = \frac{\delta A}{A_m} = \frac{\varepsilon_0}{A_m}$

The relative error is generally expressed as:

  • • A fraction (e.g., 5 parts in 1000)
  • • Percentage value: $\varepsilon_r \times 100 = \frac{\varepsilon_0}{A_m} \times 100$

💡 Practical Examples

Example 1: Voltmeter Reading

A voltage has a true value of 1.5 V on an analog indicating meter with a range 0 to 2.5 V shows a voltage of 1.46 V.

Solutions:

  • • Absolute Error = -0.04
  • • Correction = 0.04
  • • Relative error for true value = 2.67%
  • • Relative error for full scale deflection = 1.6%

Example 2: Ammeter Accuracy

A 0-25 A ammeter has a guaranteed accuracy of 1 percent of full scale reading. The current measured by this instrument is 10 A.

Solution:

$\delta A = \varepsilon_r \times A = 0.01 \times 25 = 0.25A$

The relative error at 10 A is:

$\varepsilon_r = \frac{\delta A}{A} = \frac{0.25}{10} = 0.025$

Therefore, the limiting error = 2.5%

📝

Key Points

  • •

    Static error is the difference between measured and true values.

  • •

    Relative error is the ratio of absolute error to true value.

  • •

    Accuracy specifications are often given as percentage of full scale.

  • •

    The limiting error in percentage increases as the measured value decreases.
