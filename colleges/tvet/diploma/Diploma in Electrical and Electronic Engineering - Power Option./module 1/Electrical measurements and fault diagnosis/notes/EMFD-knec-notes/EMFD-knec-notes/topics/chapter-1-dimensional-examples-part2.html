Example 2.5 : The expression for the eddy currents produced in a metallic former moving in the field of a permanent magnet is found as,

$$I_{e}=K \frac{\mathrm{~B} l \mathrm{bA}}{(2 \mathrm{~b}+l) \rho}$$

where

$$\begin{aligned} B&=\text { Flux density }, & l&=\text { Length of former } \\ b&=\text { Width of former }, & A&=\text { Area of former } \\ \rho&=\text { Resistivity of former, } & K&=\text { Constant } \end{aligned}$$

It is suspected that the term angular frequency $\omega$ is missing in the equation. Using LMIT system, for the dimensional analysis, detect the error.

Solution : Let us obtain the dimensions of each quantity on R.H.S. of the equation,

$$\begin{aligned} [\mathrm{B}]&=\frac{[\phi]}{[\mathrm{A}]}=\frac{\left[\mathrm{M} \mathrm{L}^{2} \mathrm{~T}^{-2} \mathrm{I}^{-1}\right]}{\left[\mathrm{L}^{2}\right]}=\left[\mathrm{MT}^{-2} \mathrm{I}^{-1}\right]\\ [l]&=[\mathrm{L}],[\mathrm{b}]=[\mathrm{L}], \mathrm{A}=\left[\mathrm{L}^{2}\right]\\ \therefore \quad [2\mathrm{~b}+l]&=\text { Length }=[\mathrm{L}]\\ \therefore \quad [\rho]&=\text { resistivity }=\frac{\mathrm{RA}}{l}\\ \therefore \quad [\rho]&=\frac{[\mathrm{R}][\mathrm{A}]}{[l]}=\frac{\left[\mathrm{M} \mathrm{L}^{2} \mathrm{~T}^{-3} \mathrm{I}^{-2}\right]\left[\mathrm{L}^{2}\right]}{[\mathrm{L}]}\\ &=\left[\mathrm{M} \mathrm{L}^{3} \mathrm{~T}^{-3} \mathrm{I}^{-2}\right]\\ \text{R.H.S.}&=[\mathrm{B}][l][b][A] / ([2b+l][\rho]) \\ &=[\mathrm{MT}^{-2} \mathrm{I}^{-1}][\mathrm{L}][\mathrm{L}][\mathrm{L}^{2}] / ([\mathrm{L}][\mathrm{M} \mathrm{L}^{3} \mathrm{~T}^{-3} \mathrm{I}^{-2}]) \\ &=[\mathrm{ML}^{4}\mathrm{T}^{-2}\mathrm{I}^{-1}] / [\mathrm{ML}^{4}\mathrm{T}^{-3}\mathrm{I}^{-2}]\\ &=[\mathrm{TI}] \end{aligned}$$

And $\quad$ L.H.S. $=[I]$

To satisfy the equation dimensionally, R.H.S. must be divided by [T] i.e. multiplied by [$\mathrm{T}^{-1}$]. The dimensions of $\omega$ are cycles/sec. i.e. [$\mathrm{T}^{-1}$]. Hence there must be presence of $\omega$ in the numerator of R.H.S. Hence the correct expression for $\mathrm{I}_{e}$ can be written as,

$$\mathrm{I}_{\mathrm{e}}=\frac{\mathrm{K} \omega \mathrm{B} l \mathrm{~b} \mathrm{~A}}{(2 \mathrm{~b}+l) \rho}$$

Example 2.13: Derive the dimensions of resistance, inductance, capacitance and permeability in LMTI system.

Solution : 1) $\mathrm{R}=\frac{\mathrm{E}}{\mathrm{I}} \quad$ and $\quad \mathrm{E}=\frac{\mathrm{W}}{\mathrm{Q}}, \quad \mathrm{Q}=\mathrm{I}$ t

$$\begin{aligned} \mathrm{W} & =\text { Force } \times \text { Distance }=\text { Mass } \times \text { Acceleration } \times \text { Distance }\\ \therefore \quad [\mathrm{W}] & =[\mathrm{M}]\left[\mathrm{LT}^{-2}\right][\mathrm{L}]=\left[\mathrm{ML}^{2} \mathrm{~T}^{-2}\right] \end{aligned}$$
$$\begin{aligned} [\mathrm{Q}] &=[\mathrm{TI}]\\ [\mathrm{E}] &= \frac{[\mathrm{W}]}{[\mathrm{Q}]} = \frac{[\mathrm{ML}^{2}\mathrm{T}^{-2}]}{[\mathrm{TI}]} = [\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-1}]\\ [\mathrm{R}] &= \frac{[\mathrm{E}]}{[\mathrm{I}]} = \frac{[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-1}]}{[\mathrm{I}]} = [\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-2}] \end{aligned}$$

2) $\mathrm{E}=\mathrm{L} \frac{\mathrm{dI}}{\mathrm{dt}}$

$$\begin{aligned} \therefore \quad [\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-1}]&=\frac{[\mathrm{L}][\mathrm{I}]}{[\mathrm{T}]}\\ \therefore \quad [\mathrm{L}]&=[\mathrm{ML}^{2}\mathrm{T}^{-2}\mathrm{I}^{-2}] \end{aligned}$$

3) $\mathrm{C}=\frac{\mathrm{Q}}{\mathrm{E}}$

$$\begin{aligned} [\mathrm{C}] &= \frac{[\mathrm{TI}]}{[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-1}]} = [\mathrm{M}^{-1}\mathrm{L}^{-2}\mathrm{T}^{4}\mathrm{I}^{2}] \end{aligned}$$

4) $\mu=\frac{\mathrm{B}}{\mathrm{H}}=\frac{\phi}{\mathrm{AH}}$ and $\mathrm{H}=\frac{\mathrm{NI}}{l}$

Now $\quad E=N \frac{d \phi}{d t} \quad \ldots \mathrm{N}$ is dimensionless

$$\begin{aligned} \therefore \quad [\mathrm{E}]&=\frac{[\phi]}{[\mathrm{T}]} \text{ i.e. } [\phi]=[\mathrm{ML}^{2}\mathrm{T}^{-2}\mathrm{I}^{-1}]\\ [\mathrm{A}]&=[\mathrm{L}^{2}] \text{ and } [\mathrm{H}]= \frac{[\mathrm{I}]}{[\mathrm{L}]} = [\mathrm{L}^{-1}\mathrm{I}]\\ \therefore \quad [\mu]&= \frac{[\phi]}{[\mathrm{A}][\mathrm{H}]} = \frac{[\mathrm{ML}^{2}\mathrm{T}^{-2}\mathrm{I}^{-1}]}{[\mathrm{L}^{2}][\mathrm{L}^{-1}\mathrm{I}]} = [\mathrm{MLT}^{-2}\mathrm{I}^{-2}] \end{aligned}$$

Example 2.14 : Use dimensional analysis to find whether the following equation is dimensionally correct or not.

$$R_{4}=\frac{R_{2}\left(1+\omega^{2} R_{3}^{2} C_{3}^{2}\right)}{\omega^{2} R_{1} R_{3} C_{3}}$$

where $R_{1}, R_{2}, R_{3}$ and $R_{4}$ are resistances, $C_{3}$ is capacitance while $\omega$ is angular frequency. If the equation is not correct, find the error and correctiti.
[GU : Dec.-2005]

Solution : Let us find the dimensions of R.H.S.

$$\begin{aligned} [\mathrm{R}_{1}]&=[\mathrm{R}_{2}]=[\mathrm{R}_{3}]=\frac{[\mathrm{E}]}{[\mathrm{I}]}=\frac{[\mathrm{W}]/[\mathrm{Q}]}{[\mathrm{I}]}\\ &=\frac{[\mathrm{ML}^{2}\mathrm{T}^{-2}]/[\mathrm{TI}]}{[\mathrm{I}]}=[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-2}] \end{aligned}$$

The dimension of all resistances is same.

$$\begin{aligned} [\omega]&=\text { cycles/second }=\frac{1}{[\mathrm{~T}]}=[\mathrm{T}^{-1}] \quad \text {... cycles are dimensionless}\\ [\mathrm{C}]&=\frac{[\mathrm{Q}]}{[\mathrm{E}]}=\frac{[\mathrm{TI}]}{[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-1}]/[\mathrm{TI}]} = [\mathrm{M}^{-1}\mathrm{L}^{-2}\mathrm{T}^{4}\mathrm{I}^{2}] \end{aligned}$$
$$\begin{aligned} \therefore \quad [\text { R.H.S }]&=\frac{[\mathrm{R}]\left\{1+[\omega]^{2}\left[\mathrm{R}_{3}\right]^{2}\left[\mathrm{C}_{3}\right]^{2}\right\}}{[\omega]^{2}[\mathrm{R}]^{2}\left[\mathrm{C}_{3}\right]} \text { where } [\mathrm{R}]^{2}=[\mathrm{R}_{1}][\mathrm{R}_{3}]\\ &=\frac{[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-2}]\{1+[\mathrm{T}^{-1}]^{2}[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-2}]^{2}[\mathrm{M}^{-1}\mathrm{L}^{-2}\mathrm{T}^{4}\mathrm{I}^{2}]^{2}\}}{[\mathrm{T}^{-1}]^{2}[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-2}]^{2}[\mathrm{M}^{-1}\mathrm{L}^{-2}\mathrm{T}^{4}\mathrm{I}^{2}]} \end{aligned}$$

Now as 1 is dimensionless, the dimensions of $1+\omega^{2} R_{3}^{2} C_{3}^{2}$ is same as the dimensions of $\omega^{2} R_{3}^{2} C_{3}^{2}$.

$$\begin{aligned} \therefore \quad [\text { R.H.S }]&=\frac{[\mathrm{ML}^{2}\mathrm{T}^{-3}\mathrm{I}^{-2}][\mathrm{T}^{-2}][\mathrm{M}^{2}\mathrm{L}^{4}\mathrm{T}^{-6}\mathrm{I}^{-4}][\mathrm{M}^{-2}\mathrm{L}^{-4}\mathrm{T}^{8}\mathrm{I}^{4}]}{[\mathrm{T}^{-2}][\mathrm{M}^{2}\mathrm{L}^{4}\mathrm{T}^{-6}\mathrm{I}^{-4}][\mathrm{M}^{-1}\mathrm{L}^{-2}\mathrm{T}^{4}\mathrm{I}^{2}]}\\ &=\frac{[\mathrm{ML}^{2}\mathrm{T}^{-2}]}{[\mathrm{T}^{-4}\mathrm{ML}^{2}\mathrm{I}^{-2}]} = [\mathrm{ML}^{0}\mathrm{T}^{2}\mathrm{I}^{0}] = [\mathrm{M}^{0}\mathrm{L}^{0}\mathrm{T}^{0}] \quad \text{ (Dimensional Analysis error in source OCR?)} \end{aligned}$$

While $\quad[\text { L.H.S. }]=[\mathrm{R}]=\left[\mathrm{ML}^{2} \mathrm{~T}^{-3} \mathrm{I}^{-2}\right]$

So dimensionally equation is not correct.

So on R.H.S., a quantity must be present to make the dimensions of R.H.S. same as L.H.S.

$$\begin{aligned} \therefore \quad [\mathrm{T}][\mathrm{x}] & =\left[\mathrm{ML}^{2} \mathrm{~T}^{-3} \mathrm{I}^{-2}\right]\\ \therefore \quad [\mathrm{x}] & =\left[\mathrm{ML}^{2} \mathrm{~T}^{-4} \mathrm{I}^{-2}\right] \end{aligned}$$

Thus R.H.S. must be multiplied by $x$, i.e. divided by $x^{-1}$.

$$\mathrm{x}^{-1}=\left[\mathrm{ML}^{2} \mathrm{~T}^{-4} \mathrm{I}^{-2} \mathrm{~J}^{-1}=\left[\mathrm{M}^{-1} \mathrm{~L}^{-2} \mathrm{~T}^{4} \mathrm{I}^{2}\right]\right.$$

But these are the dimensions of capacitor $\mathrm{C}_{3}$ in the given equation. Hence for having dimensional correctness of the equation, R.H.S. must be divided by one more $\mathrm{C}_{3}$. Hence the equation becomes,

$$R_{4}=\frac{R_{2}\left(1+\omega^{2} R_{3}^{2} C_{3}^{2}\right)}{\omega^{2} R_{1} R_{3} C_{3}^{2}}$$
