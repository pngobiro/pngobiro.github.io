Voltmeter-Ammeter Methods and Examples

💡 Voltmeter-Ammeter Method Examples

Example 1.13: A voltmeter having a sensitivity of $1.5 \mathrm{~k} \Omega / v o l t$ reads 80 V on its 150 V range, when connected across an unknown resistor in series with a milliammeter. The ammeter reads 15 mA . Calculate

i) Apparent resistance
ii) Actual resistance of unknown resistor
iii) Error due to loading effect of voltmeter
iv) Percentage relative accuracy.
[JNTU, Dec.-2005]

Solution :

The circuit diagram is shown in the Fig. 1.23.

The total circuit resistance, neglecting resistance of milliammeter is

$$\begin{aligned} \mathrm{R}_{\mathrm{T}} &=\frac{\mathrm{V}}{\mathrm{I}}=\frac{80}{15 \times 10^{-3}}\\ &=5.333 \mathrm{~k} \Omega \end{aligned}$$

i) Thus the apparent value of the resistance is

$$\mathrm{R}_{\mathrm{app}}=5.333 \mathrm{~k} \Omega$$

ii) Let us calculate the actual $\mathrm{R}_{\mathrm{x}}$.

The resistance of the voltmeter be $\mathrm{R}_{\mathrm{v}}$.

$$\begin{aligned} \therefore \quad \mathrm{R}_{\mathrm{v}} &=1.5 \mathrm{~k} \Omega / \text { volt } \times 150 \text { as } 150 \mathrm{~V} \text { is full scale reading.}\\ &=225 \mathrm{~k} \Omega \end{aligned}$$

Thus $\quad \mathrm{R}_{\mathrm{T}}=\mathrm{R}_{\mathrm{x}} \| \mathrm{R}_{\mathrm{v}}$

$$\therefore \quad \mathrm{R}_{\mathrm{T}}=\frac{\mathrm{R}_{\mathrm{x}} \mathrm{R}_{\mathrm{v}}}{\mathrm{R}_{\mathrm{x}}+\mathrm{R}_{\mathrm{v}}}$$
$$\begin{aligned} \therefore \quad 5.333&=\frac{\mathrm{R}_{\mathrm{x}} \times 225}{\mathrm{R}_{\mathrm{x}}+225}\\ \therefore \quad \mathrm{R}_{\mathrm{x}}+225&=42.19 \mathrm{~R}_{\mathrm{x}}\\ \therefore \quad \mathrm{R}_{\mathrm{x}}&=5.462 \mathrm{~k} \Omega \end{aligned}$$

This is the actual value of the unknown resistance.
iii) $\%$ error $=\frac{A_{t}-A_{m}}{A_{t}} \times 100=\frac{5.462-5.333}{5.462} \times 100$

$$=2.36 \%$$

iv) The relative accuracy,

$$\begin{aligned} \% A & =(1-\mid \text { Error } \mid) \times 100=(1-0.0236) \times 100\\ &=97.63 \% \end{aligned}$$

Example 1.11: A 0-100 V voltmeter has 200 scale divisions which can be read to $1 / 2$ division. Determine the resolution of the meter in volt.

Solution :

$$1 \text { scale division }=\frac{\text { Full scale division }}{\text { Number of divisions }}=\frac{100}{200}=0.5 \mathrm{~V}$$
$$\begin{aligned} \therefore \quad \text { Resolution }&=\frac{1}{2} \times \text { Scale division }=\frac{1}{2} \times 0.5\\ &=0.25 \mathrm{~V} \end{aligned}$$

Example 1.9 : An ammeter reads 8.3 A and the true value of the current is 8.5 A . Determine the absolute error and relative percentage error.

Solution :

$\mathrm{A}_{\mathrm{t}}=$ True value $=8.5 \mathrm{~A}$
$\mathrm{A}_{\mathrm{m}}=$ Measured value $=8.3 \mathrm{~A}$

Absolute error $=\mathrm{A}_{\mathrm{t}}-\mathrm{A}_{\mathrm{m}}=8.5-8.3=0.2 \mathrm{~A}$

$$\% \mathrm{e}=\frac{\mathrm{A}_{\mathrm{t}}-\mathrm{A}_{\mathrm{m}}}{\mathrm{A}_{\mathrm{t}}} \times 100=\frac{0.2}{8.5} \times 100=2.35 \%$$

Example 1.10 : A voltmeter reads 111.5 V . The error taken from an error curve is $5.3 \%$. Find the true value of the voltage.

Solution :

$\mathrm{A}_{\mathrm{m}}=111.5 \mathrm{~V}$
$\% \mathrm{e}=5.3 \%$
Now $\% \mathrm{e}=\frac{\mathrm{A}_{\mathrm{t}}-A_{m}}{A_{t}} \times 100$

$$\begin{aligned} \therefore \quad 5.3 &=\frac{\mathrm{A}_{\mathrm{t}}-111.5}{A_{t}} \times 100\\ \therefore \quad 0.053 \mathrm{~A}_{\mathrm{t}}&=\mathrm{A}_{\mathrm{t}}-111.5\\ \therefore \quad A_{t}&=117.74 \mathrm{~V} \end{aligned}$$

Example 1.6 : A 0-50 V voltmeter is specified to be accurate within $\pm 1 \%$ of full scale. Calculate the limiting error when the instrument reading is 15 V .

Solution :

The limiting error at full scale is,

$$\begin{aligned} \delta \mathrm{A} &= \pm 1 \% \text { of } 50\\ &= \pm \frac{1}{100} \times 50\\ &= \pm 0.5 \mathrm{~V} \end{aligned}$$

For a reading of 15 V , it is

$$\% \mathrm{e}=\frac{0.5}{15} \times 100=3.33 \%$$

Thus as reading is less than full scale, the limiting error is more.
