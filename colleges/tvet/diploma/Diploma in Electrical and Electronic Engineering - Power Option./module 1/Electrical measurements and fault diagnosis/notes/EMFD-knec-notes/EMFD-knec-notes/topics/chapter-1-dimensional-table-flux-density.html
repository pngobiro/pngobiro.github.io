EXAMPLES

Magnetic flux

To obtain the dimensions of flux, use can be made of Faraday's law which states that EMF (induced in any circuit) = rate of change of flux

or the EMF $=\frac{\text { flux }}{\text { time }}$

or

$$\text { flux }=\text { EMF } \times \text { time }$$

$\therefore$ dimensionally

$$\begin{aligned} [\emptyset] & =\left[\varepsilon^{-1 / 2} \mathrm{~L}^{1 / 2} \mathrm{M}^{1/2} \mathrm{~T}^{-1}\right] \times[\mathrm{T}] & & \\ & =\left[\varepsilon^{-1/2} \mathrm{~L}^{1/2} \mathrm{M}^{1/2}\right], & & \text { in the ES system } \\ \text { Likewise } \quad [\emptyset] & =\left[\mu^{1/2} \mathrm{~L}^{3/2} \mathrm{M}^{1/2} \mathrm{~T}^{-1}\right], & & \text { in the EM system } \end{aligned}$$

Flux density

Since by definition flux density is flux per unit area

$$\begin{aligned} [\mathrm{B}]=\frac{[\emptyset]}{\left[\mathrm{L}^{2}\right]}=\left[\varepsilon^{-1 / 2} \mathrm{~L}^{-3/2} \mathrm{M}^{1/2}\right], & \text { in the ES system } \\ {[\mathrm{B}]=\left[\mu^{1/2} \mathrm{~L}^{-1/2} \mathrm{M}^{1/2} \mathrm{~T}^{-1}\right],} & \text { in the EM system } \end{aligned}$$

Consider the measurement of inductance of a coil using Hay's bridge ,the expression for the inductance being obtained as

$$L=\frac{R_{2} R_{3} C}{1+\omega^{2} R_{4}^{2} C^{2}}$$

proof that it is dimensionally correct
