üîÅ5.2.1 SUCCESSIVE DIFFERENTIATION

If \(y=f(x)\) then \(\frac{d y}{d x}=f^{\prime}(x)\) is a function of x or constant which can be differentiated once again. \(\frac{d y}{d x}=f^{\prime}(x)\) is the first order derivative of y w.r.t x.

\(\frac{d^{2} y}{d x^{2}}=\frac{d}{d x}\left(\frac{d y}{d x}\right)\) is the second order derivative of y w.r.t x.
i.e the derivate of \(\frac{d y}{d x}\) w.r.t x.

\(\frac{d^{3} y}{d x^{3}}=\frac{d}{d x}\left(\frac{d^{2} y}{d x^{2}}\right)\) is the third order derivative of y w.r.t x and so on.

Notations of successive derivatives

\(\frac{d y}{d x}=y_{1}=y^{\prime}=f^{\prime}(x)=D(y)\)
\(\frac{d^{2} y}{d x^{2}}=y_{2}=y^{\prime \prime}=f^{\prime \prime}(x)=D^{2}(y)\)
...
\(\frac{d^{n} y}{d x^{n}}=y_{n}=y^{(n)}=f^{(n)}(x)=D^{n}(y)\)

WORKED EXAMPLES

PART - A

1. If \(y=e^{3 x}\) find \(\frac{d^{2} y}{d x^{2}}\).

Solution:

\(\frac{d y}{d x} = \frac{d}{dx}(e^{3x}) = 3 e^{3 x}\)
\(\frac{d^{2} y}{d x^{2}} = \frac{d}{dx}(3e^{3x}) = 3 \frac{d}{dx}(e^{3x}) = 3(3 e^{3 x})\)
\(=9 e^{3 x}\)

2. If \(y=\sin 3 x\) find \(\frac{d^{2} y}{d x^{2}}\).

Solution:

\(\frac{d y}{d x} = \frac{d}{dx}(\sin 3x) = 3 \cos 3 x\)
\(\frac{d^{2} y}{d x^{2}} = \frac{d}{dx}(3 \cos 3x) = 3 \frac{d}{dx}(\cos 3x) = 3(-3 \sin 3 x)\)
\( = -9 \sin 3 x\)

PART - B

1. If \(y=a \cos n x+b \sin n x\) find \(y_{2}\).

Solution:

\(y_{1} = \frac{d}{dx}(a \cos n x+b \sin n x) = a(-n \sin n x)+b(n \cos n x)\)
\(=n(-a \sin n x+b \cos n x)\)
\(y_{2} = \frac{d}{dx}(y_1) = n \frac{d}{dx}(-a \sin n x+b \cos n x)\)
\(=n[-a(n \cos n x)+b(-n \sin n x)]\)
\(=n^2(-a \cos n x - b \sin n x)\)
\(=-n^{2}(a \cos n x+b \sin n x)\)
\(=-n^{2} y\)

2. If \(y=A e^{3 x}+B e^{-3 x}\) find \(\frac{d^{2} y}{d x^{2}}\).

Solution:

\(\frac{d y}{d x} = \frac{d}{dx}(A e^{3 x}+B e^{-3 x}) = A(3 e^{3 x})+B(-3 e^{-3 x})\)
\(=3\left(A e^{3 x}-B e^{-3 x}\right)\)
\(\frac{d^{2} y}{d x^{2}} = \frac{d}{dx}(3(Ae^{3x} - Be^{-3x})) = 3\left[A(3 e^{3 x})-B(-3 e^{-3 x})\right]\)
\(=3\left[3 A e^{3 x}+3 B e^{-3 x}\right]\)
\(=9\left[A e^{3 x}+B e^{-3 x}\right]\)
\(=9 y\)

3. If \(y=\tan x\) find \(y_{2}\).

Solution:

\(y_1 = \frac{d y}{d x} = \frac{d}{dx}(\tan x) = \sec ^{2} x\)
\(y_2 = \frac{d^{2} y}{d x^{2}} = \frac{d}{dx}(\sec^2 x) = \frac{d}{dx}[(\sec x)^2]\)
\(=2(\sec x)^{1} \cdot \frac{d}{dx}(\sec x)\) (Using Chain Rule)
\(=2(\sec x) \cdot (\sec x \tan x)\)
\(=2 \sec ^{2} x \tan x\)

PART - C

1. If \(y=x^{2} \cos x\) prove that \(x^{2} y_{2}-4 x y_{1}+\left(x^{2}+6\right) y=0\).

Solution:

\(y=x^{2} \cos x\)
\(y_{1}=x^{2}(-\sin x)+\cos x(2 x)\) (using product rule)
\(=-x^{2} \sin x+2 x \cos x\)
\(y_{2}=\frac{d}{dx}(-x^2 \sin x + 2x \cos x)\)
\( = [-x^2(\cos x) + \sin x(-2x)] + [2x(-\sin x) + \cos x(2)] \)
\(=-x^{2} \cos x-2 x \sin x-2 x \sin x+2 \cos x\)
\(y_2=-x^{2} \cos x-4 x \sin x+2 \cos x\)

Now, substitute into the expression: \(x^{2} y_{2}-4 x y_{1}+\left(x^{2}+6\right) y\)
\( =x^{2}\left[-x^{2} \cos x-4 x \sin x+2 \cos x\right] -4 x\left[-x^{2} \sin x+2 x \cos x\right]+\left(x^{2}+6\right)\left(x^{2} \cos x\right)\)
\( = (-x^{4} \cos x - 4 x^{3} \sin x + 2 x^{2} \cos x) + (4 x^{3} \sin x - 8 x^{2} \cos x) + (x^{4} \cos x + 6 x^{2} \cos x) \)
\( = (-x^4 + x^4)\cos x + (-4x^3 + 4x^3)\sin x + (2 - 8 + 6)x^2 \cos x \)
\( = 0 \cos x + 0 \sin x + 0 x^2 \cos x \)
\( =0 \). Hence proved.

2. If \(y=e^{x} \sin x\), prove that \(y_{2}-2 y_{1}+2 y=0\).

Solution:

\(y =e^{x} \sin x\)
\(y_{1} =e^{x}(\cos x)+\sin x\left(e^{x}\right)\) (using product rule)
\(y_{1} =e^{x} \cos x+y \quad\) .....(1)

Differentiating w.r.t x
\(y_{2} = \frac{d}{dx}(e^x \cos x) + \frac{dy}{dx} = [e^{x}(-\sin x)+\cos x e^{x}]+y_{1}\)
\(y_{2} =-e^{x} \sin x+e^{x} \cos x+y_{1}\)
\(y_{2} =-y+\left(y_{1}-y\right)+y_{1}\) (Substituting \(e^x \cos x = y_1 - y\) from (1))
\(y_{2} =-y+y_{1}-y+y_{1}\)
\(y_2 =-2 y+2 y_{1}\)
\(y_{2} -2 y_{1}+2 y=0\). Hence proved.

3. If \(y=a \cos (\log x)+b \sin (\log x)\) prove that \(x^{2} y_{2}+x y_{1}+y=0\).

Solution:

\(y=a \cos (\log x)+b \sin (\log x)\)

Differentiating w.r.t x
\(y_{1} =a\left[-\sin (\log x) \cdot \frac{1}{x}\right]+b\left[\cos (\log x) \cdot \frac{1}{x}\right]\)
\(y_{1} =\frac{1}{x}[-a \sin (\log x)+b \cos (\log x)]\)
\(x y_{1} =-a \sin (\log x)+b \cos (\log x)\)

Differentiating w.r.t x again (using product rule on LHS):
\(x \frac{d}{dx}(y_1) + y_1 \frac{d}{dx}(x) = -a \cos (\log x) \cdot \frac{1}{x}+b\left(-\sin (\log x) \cdot \frac{1}{x}\right)\)
\(x y_{2}+y_{1}(1) =-a \cos (\log x) \cdot \frac{1}{x}-b \sin (\log x) \cdot \frac{1}{x}\)
\(x y_{2}+y_{1} =\frac{-1}{x}[a \cos (\log x)+b \sin (\log x)]\)
Multiply by x:
\(x(x y_{2}+y_{1}) =- [a \cos (\log x)+b \sin (\log x)]\)
\(x^{2} y_{2}+x y_{1} =-y\)
\(x^{2} y_{2}+x y_{1}+y =0\). Hence proved.

üî¢5.2.2 DIFFERENTIAL EQUATION

Definition: An equation containg differential co-efficients is called differential equation.

Examples:

  1. 1. \(\frac{d y}{d x}+y=x\)
  2. 2. \(x \frac{d^{2} y}{d x^{2}}+1=0\)
  3. 3. \(a \frac{d^{2} y}{d x^{2}}+b \frac{d y}{d x}+c y=d\)

Order of the differential equation:

The order of the highest derivative in the differential equation is called order of the differential equation.

Degree of the differential equation:

The power of the highest derivative in the differential equation is called degree of the differential equation (after the equation has been cleared of radicals and fractions with respect to the derivatives).

Example:

(i) \(7\left(\frac{d^{2} y}{d x^{2}}\right)^{2}+5\left(\frac{d y}{d x}\right)^{5}+7 y=\sin x\)
Order-2, degree -2

Formation of Differential Equation:

A differential equation is obtained by differentiating the function \(f(x, y, c_1, c_2, ...)=0\) as many times as the number of arbitrary constants, followed by elimination of arbitrary constants.

Worked Examples (Differential Equations)

PART -A

1. Find the order and degree of the following differential equations.

(i) \(\left(\frac{d^{3} y}{d x^{3}}\right)^{4}+\frac{d^{2} y}{d x^{2}}+2 \frac{d y}{d x}+7 y=0\)
(ii) \(\left(\frac{d y}{d x}\right)^{2}+x \frac{d y}{d x}=k\)
(iii) \(\sqrt{1+\frac{d y}{d x}}=\frac{d^{2} y}{d x^{2}}\)

Solution

(i) order = 3, degree = 4

(ii) order = 1, degree = 2

(iii) \(\sqrt{1+\frac{d y}{d x}}=\frac{d^{2} y}{d x^{2}}\)
Squaring on both sides
\(\left(1+\frac{d y}{d x}\right)=\left(\frac{d^{2} y}{d x^{2}}\right)^{2}\)
Order -2 , degree -2 .

2. From the differential equation by eliminating the constant.

(i) \(x y=c\)
(ii) \(\sqrt{x}+\sqrt{y}=\sqrt{a}\)

Solution:

(i) \(x y=c\)
Differentiating w.r.t. x:
\(x \cdot \frac{d y}{d x}+y \cdot 1=0\) (using product rule)
\(x \frac{d y}{d x}+y=0\)

(ii) \(\sqrt{x}+\sqrt{y}=\sqrt{a}\)
Differentiating w.r.t. x:
\(\frac{1}{2 \sqrt{x}}+\frac{1}{2 \sqrt{y}} \cdot \frac{d y}{d x}=0\)
Multiply by 2
\(\frac{1}{\sqrt{x}}+\frac{1}{\sqrt{y}} \frac{d y}{d x}=0\)

PART -B

From the different equation by eliminating the constants.

(i) \(y=a x+b\) (Typo: should be \(b^2\)?)
(ii) \(y=m e^{5 x}\)
(iii) \(\frac{\mathrm{x}}{\mathrm{a}}+\frac{\mathrm{y}}{\mathrm{b}}=1\)
(iv) \(y^{2}=4 a x\)

Solution:

(i) \(y=a x+b\) (Assuming b is a constant, not b^2)
\(\frac{d y}{d x}=a(1)+0 = a\)
\(\frac{d^{2} y}{d x^{2}}=0\)

(ii) \(y=m e^{5 x}\) .....(1)
\(\frac{d y}{d x}=m \cdot (5 e^{5 x})\)
\(\frac{d y}{d x}=5 (m e^{5 x})\)
\(\frac{d y}{d x}=5 y\) (from (1))

(iii) \(\frac{x}{a}+\frac{y}{b}=1\)
Differentiating w.r.t x
\(\frac{1}{a}+\frac{1}{b} \frac{d y}{d x}=0\)
Differentiating again w.r.t x
\(0+\frac{1}{b} \frac{d^{2} y}{d x^{2}}=0\)
\(\therefore \frac{d^{2} y}{d x^{2}}=0\)

(iv) \(y^{2}=4 a x\) .....(1)
Differentiating w.r.t. x:
\(2 y \cdot \frac{d y}{d x}=4 a(1) = 4a\)
From (1), \(a = \frac{y^2}{4x}\). Substitute into the differentiated equation:
\(2 y \frac{d y}{d x}=4\left(\frac{y^{2}}{4 x}\right)\)
\(2 y \frac{d y}{d x}=\frac{y^{2}}{x}\)
