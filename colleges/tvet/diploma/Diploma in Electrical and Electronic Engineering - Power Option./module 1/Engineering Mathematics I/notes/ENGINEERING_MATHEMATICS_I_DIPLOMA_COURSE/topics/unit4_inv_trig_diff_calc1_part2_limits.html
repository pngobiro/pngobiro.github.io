üìà4.2 LIMITS

Mathematical quantities can be divided into two (i) constants and (ii) Variables

The quantitity which does not change in its value is called constant.

Constants can be divided (i) Absolute constants and (ii) Arbitrary constants

Absolute constants are those which retain in values at any time and at any place.

Examples:

Numbers \(5,-7,3, \frac{-11}{27}, \pi, e, \sqrt{2}, \sqrt{3}, \ldots \ldots\) etc. are absolute constants.

Arbitrary constants are those which are constants in a particular problem but changes from problem to problem. They are represented as \(a, b, c, f, g, h, \ldots \ldots\) etc.

Examples:

(i) \(ax+by+c=0\) - a, b, c

(ii) \(x^{2}+y^{2}+2 g x+2 f y+c=0\) - f, g, c. ...ex.

Variables are those which vary in their values. They are represented as \(u, v, w, x, y, \theta, \ldots \ldots\)

Variables can be divided into two (i) Independent variable and (ii) Dependent varaible.

In the equation \(y=x^{2}\), x is the independent variable and y is the dependent variable depending on x.

For example, if x takes the value 2, y takes the value 4. If x takes -3, y takes 9 ... etc.

The relation or equation or connection between two variables is called a function and is denoted as \(f(x)\).

Example: \(y=f(x)=x^{2}\) or \(\sin x\).

The value of \(f(x)\) at \(x=a\) is \(f(a)\) and is called the functional value of \(f(x)\) at \(x=a\).

If for every function, if the functional value at every point of its domain exists, there is no need to study Limit chapter and from that the famous CALCULUS.

For example, \(f(x)=\frac{x^{2}-4}{x-2}\) at \(x=2\) i.e, \(f(2)\) does not exist, since \(f(2)=\frac{0}{0}\), the indeterminant quantity. Similarly there are two more indeterminant quantities \(\frac{\infty}{\infty}\) and \(\infty-\infty\).

But, as x approaches 2, \(f(x)=\frac{x^{2}-4}{x-2}\) will approach 4.

Hence the approachment value 4 of the function \(\frac{x^{2}-4}{x-2}\) as x approaches 2 is called the limit value of the function.

Definition of Limit:

When the variable x approaches a constant a and if the function \(f(x)\) approaches a constant \(l\), then \(l\) is called the limit value of \(f(x)\) as x approaches a and is denotd as

\(\operatorname{Lt}_{x \rightarrow \mathrm{a}} \mathrm{f}(\mathrm{x})=l\)

Results:

  1. 1) \(\underset{x \rightarrow a}{\operatorname{Lt}}[f(x) \pm g(x)]=\underset{x \rightarrow a}{\operatorname{Lt}} f(x) \pm \underset{x \rightarrow a}{\operatorname{Lt}} g(x)\)
  2. 2) \(\underset{x \rightarrow a}{\operatorname{Lt}}[k f(x)]=k \underset{x \rightarrow a}{\operatorname{Lt}} f(x)\), where k is a constant
  3. 3) \(\underset{x \rightarrow a}{\operatorname{Lt}}[f(x) \cdot g(x)]=\underset{x \rightarrow a}{\operatorname{Lt}} f(x) \cdot \underset{x \rightarrow a}{\operatorname{Lt}} g(x)\)
  4. 4) \(\underset{x \rightarrow a}{\operatorname{Lt}}\left[\frac{f(x)}{g(x)}\right]=\frac{\underset{x \rightarrow a}{\operatorname{Lt}} f(x)}{\underset{x \rightarrow a}{\operatorname{Lt}} g(x)}\), provided \(\underset{x \rightarrow a}{\operatorname{Lt}} g(x) \neq 0\)

Formulae:

  1. 1) \(\underset{x \rightarrow a}{\operatorname{Lt}} \frac{x^{n}-a^{n}}{x-a}=n a^{n-1}\), for all values of n.
  2. 2) \(\underset{\theta \rightarrow 0}{\operatorname{Lt}} \frac{\sin \theta}{\theta}=1\)
  3. 3) \(\underset{\theta \rightarrow 0}{\operatorname{Lt}} \frac{\tan \theta}{\theta}=1\), \(\theta\) in radians

Now,
\(\underset{\theta \rightarrow 0}{\operatorname{Lt}} \frac{\sin n \theta}{\theta} =\underset{\theta \rightarrow 0}{\operatorname{Lt}}\left[\frac{\sin n \theta}{n \theta} \cdot n\right]\)
\(=n \operatorname{Lt}_{\theta \rightarrow 0}\left(\frac{\sin n \theta}{n \theta}\right) \quad\) [By Result 2]
\(=n . 1 \quad\) [By Formula 2]
\(=n\)

Similarly, \(\underset{\theta \rightarrow 0}{\operatorname{Lt}} \frac{\tan n \theta}{\theta}=n\)

WORKED EXAMPLES

PART - A

1. Evaluate \(\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{3 \mathrm{x}^{2}+2 \mathrm{x}+1}{5 \mathrm{x}^{2}+6 \mathrm{x}+7}\).

Solution:

\(\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{3 x^{2}+2 x+1}{5 x^{2}+6 x+7} =\frac{3(0)^{2}+2(0)+1}{5(0)^{2}+6(0)+7}\)
\(=\frac{1}{7}\)

2. Evaluate \(\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{3 x^{2}+4 x}{5 x-7 x^{2}}\)

Solution:

\(\operatorname{Lt}_{x \rightarrow 0} \frac{3 x^{2}+4 x}{5 x-7 x^{2}} =\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{x(3 x+4)}{x(5-7 x)}\)
\(=\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{3x+4}{5-7x}\)
\(=\frac{3(0)+4}{5-7(0)}\)
\(=\frac{4}{5}\)

3. Evaluate: \(\underset{\theta \rightarrow 0}{\operatorname{Lt}} \frac{\sin 7 \theta}{\theta}\)

Solution:

\(\underset{\theta \rightarrow 0}{\operatorname{Lt}} \frac{\sin 7 \theta}{\theta}=\underset{\theta \rightarrow 0}{\operatorname{Lt}}\left[\frac{\sin 7 \theta}{7 \theta} \cdot 7\right]\)
\(=7 \operatorname{Lt}_{\theta \rightarrow 0} \frac{\sin 7 \theta}{7 \theta}\)
\(= 7 \times 1\)
\( = 7\)

PART - B

1. Evaluate: \(\operatorname{Lt}_{x \rightarrow 1} \frac{x^{2}+x-2}{x^{2}-4 x+3}\)

Solution:

\(\operatorname{Lt}_{x \rightarrow 1} \frac{x^{2}+x-2}{x^{2}-4 x+3}=\frac{1+1-2}{1-4(1)+3}=\frac{2-2}{4-4}=\frac{0}{0}\) Indeterminent

\(\operatorname{Lt}_{x \rightarrow 1} \frac{x^{2}+x-2}{x^{2}-4 x+3} =\underset{x \rightarrow 1}{\operatorname{Lt}} \frac{(x+2)(x-1)}{(x-1)(x-3)}\)
\(=\underset{x \rightarrow 1}{\operatorname{Lt}} \frac{x+2}{x-3}\)
\(=\frac{1+2}{1-3}=\frac{3}{-2}=-\frac{3}{2}\)

2. Evaluate \(\underset{x \rightarrow 2}{\operatorname{Lt}} \frac{x^{10}-2^{10}}{x-2}\).

Solution:

\(\operatorname{Lt}_{x \rightarrow 2} \frac{x^{10}-2^{10}}{x-2} =10 \cdot 2^{10-1} \quad \left[\text { By F1 } \underset{\mathrm{x} \rightarrow \mathrm{a}}{\mathrm{Lt}} \frac{\mathrm{x}^{\mathrm{n}}-\mathrm{a}^{\mathrm{n}}}{\mathrm{x}-\mathrm{a}}=\mathrm{na}^{\mathrm{n}-1}\right]\)
\(=10 \cdot 2^{9}\)
\(=10 \times 512\)
\(=5120\)

3. Evaluate \(\operatorname{Lt}_{x \rightarrow a} \frac{\sqrt{x}-\sqrt{a}}{x-a}\).

Solution:

\(\operatorname{Lt}_{x \rightarrow a} \frac{\sqrt{x}-\sqrt{a}}{x-a} =\underset{x \rightarrow a}{\operatorname{Lt}} \frac{x^{1 / 2}-a^{1 / 2}}{x-a}\)
\(=\frac{1}{2} \cdot a^{1 / 2^{-1}}\)
\(=\frac{1}{2} a^{-1 / 2}\)
\(=\frac{1}{2} \cdot \frac{1}{a^{1 / 2}}=\frac{1}{2 \sqrt{a}}\)

4. Evaluate \(\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{\sin 5 x}{4 x}\)

Solution:

\(\operatorname{Lt}_{x \rightarrow 0} \frac{\sin 5 x}{4 x}=\frac{1}{4} \underset{x \rightarrow 0}{\operatorname{Lt}} \frac{\sin 5 x}{x}=\frac{1}{4} \underset{x \rightarrow 0}{\operatorname{Lt}}\left[\frac{\sin 5 x}{5 x} \times 5\right]\)
\(=\frac{1}{4} \times 5 \underset{x \rightarrow 0}{\operatorname{Lt}} \frac{\sin 5 x}{5 x} \quad \left[\begin{array}{l}\text { By } \mathrm{F}_{2} \\ \operatorname{Lt}_{\theta \rightarrow 0} \frac{\sin \theta}{\theta}=1\end{array}\right]\)
\(=\frac{5}{4} \cdot 1=\frac{5}{4}\)

5. Evaluate \(\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{\tan ^{2} 5 x}{x^{2}}\).

Solution:

\(\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{\tan ^{2} 5 x}{x^{2}} =\underset{x \rightarrow 0}{\operatorname{Lt}}\left(\frac{\tan 5 x}{x}\right)^{2}\)
\(=\underset{x \rightarrow 0}{\operatorname{Lt}}\left[\frac{\tan 5 x}{5 x} \cdot 5\right]^{2}\)
\(=\left(\underset{x \rightarrow 0}{\operatorname{Lt}} \frac{\tan 5 x}{5 x}\right)^{2} \times 5^2 \quad \left[\begin{array}{l}\text { By } \mathrm{F}_{3} \\ \operatorname{Lt}_{\theta \rightarrow 0} \frac{\tan \theta}{\theta}=1\end{array}\right]\)
\(=(1)^2 \times 25\)
\(=25\)

PART - C

1. Evaluate \(\underset{x \rightarrow 3}{\operatorname{Lt}} \frac{x^{5}-243}{x^{2}-9}\).

Solution:

\(\operatorname{Lt}_{x \rightarrow 3} \frac{x^{5}-243}{x^{2}-9} =\underset{x \rightarrow 3}{\operatorname{Lt}} \frac{x^{5}-3^{5}}{x^{2}-3^{2}}\)
\(=\underset{x \rightarrow 3}{\operatorname{Lt}}\left[\frac{x^{5}-3^{5}}{x-3} \times \frac{x-3}{x^{2}-3^{2}}\right]\)
\(=\underset{x \rightarrow 3}{\operatorname{Lt}}\left[\frac{x^{5}-3^{5}}{x-3} / \frac{x^{2}-3^{2}}{x-3}\right]\)
\(=\frac{\operatorname{Lt}_{x \rightarrow 3} \frac{x^{5}-3^{5}}{x-3}}{\operatorname{Lt}_{x \rightarrow 3} \frac{x^{2}-3^{2}}{x-3}} \quad\) [By Result 4 & Formula 1]
\(=\frac{5 \cdot 3^{5-1}}{2 \cdot 3^{2-1}}=\frac{5 \cdot 3^{4}}{2 \cdot 3^{1}}\)
\(=\frac{5}{2} \times 3^{3}=\frac{5}{2} \times 27=\frac{135}{2}\)

2. Evaluate : \(\underset{x \rightarrow 4}{\operatorname{Lt}} \frac{x^{7 / 3}-4^{7 / 3}}{x^{2 / 3}-4^{2 / 3}}\)

Solution:

\(\operatorname{Lt}_{x \rightarrow 4} \frac{x^{7 / 3}-4^{7 / 3}}{x^{2 / 3}-4^{2 / 3}} =\underset{x \rightarrow 4}{\operatorname{Lt}}\left[\frac{x^{7 / 3}-4^{7 / 3}}{x-4} \times \frac{x-4}{x^{2 / 3}-4^{2 / 3}}\right]\)
\(=\underset{x \rightarrow 4}{\operatorname{Lt}}\left[\frac{x^{7 / 3}-4^{7 / 3}}{x-4} / \frac{x^{2 / 3}-4^{2 / 3}}{x-4}\right]\)
\(=\frac{\operatorname{Lt}_{x \rightarrow 4} \frac{x^{7 / 3}-4^{7 / 3}}{x-4}}{\operatorname{Lt}_{x \rightarrow 4} \frac{x^{2 / 3}-4^{2 / 3}}{x-4}}\)
\(=\frac{\frac{7}{3} \cdot 4^{\frac{7}{3}-1}}{\frac{2}{3} \cdot 4^{\frac{2}{3}-1}}=\frac{\frac{7}{3} \cdot 4^{4 / 3}}{\frac{2}{3} \cdot 4^{-1 / 3}}\)
\(=\frac{7}{3} \times \frac{3}{2} \times 4^{4 / 3-(-1 / 3)}=\frac{7}{2} \times 4^{5 / 3}\)

3. Evaluate \(\underset{\theta \rightarrow 0}{\mathrm{Lt} \frac{\sin 8 \theta}{\tan 3 \theta}}\).

Solution:

\(\underset{\theta \rightarrow 0}{\mathrm{Lt}} \frac{\sin 8 \theta}{\tan 3 \theta} =\underset{\theta \rightarrow 0}{\mathrm{Lt}} \left[ \frac{\sin 8\theta}{\theta} \times \frac{\theta}{\tan 3\theta} \right]\)
\(=\underset{\theta \rightarrow 0}{\mathrm{Lt}} \left[ \frac{\sin 8 \theta}{8\theta} \times 8 \times \frac{3\theta}{\tan 3\theta} \times \frac{1}{3} \right]\)
\( = \left( \underset{\theta \rightarrow 0}{\mathrm{Lt}} \frac{\sin 8 \theta}{8\theta} \right) \times \left( \underset{\theta \rightarrow 0}{\mathrm{Lt}} \frac{3\theta}{\tan 3\theta} \right) \times \frac{8}{3} \)
\( = (1) \times (1) \times \frac{8}{3} = \frac{8}{3} \)
