üî¢2.2 DE-MOIVRE'S THEOREM

DeMoivre's Theorem: (Statement only)

  1. (i) If ' n ' is an integer positive or negative then \((\cos \theta+i \sin \theta)^{n}=\cos n \theta+i \sin n \theta\).
  2. (ii) If ' n ' is a fraction, then \(\cos n \theta+i \sin n \theta\) is one of the values of \((\cos \theta+i \sin \theta)^{n}\).

Results:

  1. 1) \((\cos \theta+i \sin \theta)^{n}=\cos n \theta+i \sin n \theta\)
  2. 2) \((\cos \theta+i \sin \theta)^{-n}=(\cos n \theta-i \sin n \theta)\)
  3. 3) \(\frac{1}{\cos \theta+i \sin \theta}=(\cos \theta+i \sin \theta)^{-1}=\cos \theta-i \sin \theta\)
  4. 4) \(\frac{1}{\cos \theta-i \sin \theta}=(\cos \theta-i \sin \theta)^{-1}=\cos \theta+i \sin \theta\)
  5. 5) \(\sin \theta+i \cos \theta=\cos \left(\frac{\pi}{2}-\theta\right)+i \sin \left(\frac{\pi}{2}-\theta\right)\)

üí°Note

  1. 1) \((\cos \theta_{1}+i \sin \theta_{1})(\cos \theta_{2}+i \sin \theta_{2})=\cos \left(\theta_{1}+\theta_{2}\right)+i \sin \left(\theta_{1}+\theta_{2}\right)\)
  2. 2) \((\cos \theta_{1}+i \sin \theta_{1})(\cos \theta_{2}+i \sin \theta_{2})(\cos \theta_{3}+i \sin \theta_{3})=\cos \left(\theta_{1}+\theta_{2}+\theta_{3}\right)+i \sin \left(\theta_{1}+\theta_{2}+\theta_{3}\right)\)
  3. 3) \((\cos \theta_{1}+i \sin \theta_{1})(\cos \theta_{2}+i \sin \theta_{2}) \ldots\left(\cos \theta_{n}+i \sin \theta_{n}\right) = \cos \left(\theta_{1}+\theta_{2}+\ldots \ldots . .+\theta_{n}\right)+i \sin \left(\theta_{1}+\theta_{2}+\ldots . .+\theta_{n}\right)\)

WORKED EXAMPLES

PART - A

1. If \(z=\cos 30^{\circ}+i \sin 30^{\circ}\) what is the value of \(z^{3}\).

Solution:

\(z^{3} =\left[\cos 30^{\circ}+\mathrm{i} \sin 30^{\circ}\right]^{3}\)
\(=\cos 90^{\circ}+\mathrm{i} \sin 90^{\circ}\)
\(=0+\mathrm{i}(1)=\mathrm{i}\)

2. If \(z=\cos \frac{\pi}{2}+i \sin \frac{\pi}{2}\) what is the value of \(z^{8}\).

Solution:

\(z^{8} =\left[\cos \frac{\pi}{2}+\mathrm{i} \sin \frac{\pi}{2}\right]^{8}\)
\(=\cos 8\left(\frac{\pi}{2}\right)+\mathrm{i} \sin 8\left(\frac{\pi}{2}\right)\)
\(=\cos 4 \pi+\mathrm{i} \sin 4 \pi\)
\(=1+\mathrm{i}(0)=1\)

3. If \(z=\cos 45^{\circ}-\mathrm{i} \sin 45^{\circ}\) what is the value of \(\frac{1}{z}\).

Solution:

\(\frac{1}{z}=z^{-1}\)
\(=\left[\cos 45-\mathrm{i} \sin 45^{\circ}\right]^{-1}\)
\(=\cos 45^{\circ}+\mathrm{i} \sin 45^{\circ}\)
\(=\frac{1}{\sqrt{2}}+\mathrm{i} \frac{1}{\sqrt{2}}\)

4. If \(\frac{1}{z}=\cos 60^{\circ}+i \sin 60^{\circ}\) what is the value of \(z\).

Solution:

\(z =\frac{1}{\frac{1}{z}}\)
\(=\frac{1}{\cos 60^{\circ}+i \sin 60^{\circ}}=\left(\cos 60^{\circ}+i \sin 60^{\circ}\right)^{-1}\)
\(=\cos 60^{\circ}-\mathrm{i} \sin 60^{\circ}\)
\(=\frac{1}{2}-\mathrm{i} \frac{\sqrt{3}}{2}\)

5. Find the value of \(\frac{\cos 3 \theta+i \sin 3 \theta}{\cos \theta-i \sin \theta}\).

Solution:

\(\frac{\cos 3 \theta+\mathrm{i} \sin 3 \theta}{\cos \theta-\mathrm{i} \sin \theta} =(\cos 3 \theta+\mathrm{i} \sin 3 \theta)(\cos \theta-\mathrm{i} \sin \theta)^{-1}\)
\(=(\cos 3 \theta+\mathrm{i} \sin 3 \theta)(\cos \theta+\mathrm{i} \sin \theta)\)
\(=\cos (3 \theta+\theta)+\mathrm{i} \sin (3 \theta+\theta)\)
\(=\cos 4 \theta+\mathrm{i} \sin 4 \theta\)

6. Simplify: \((\cos 20^{\circ}+\mathrm{i} \sin 20^{\circ})(\cos 30^{\circ}+\mathrm{i} \sin 30^{\circ})(\cos 40^{\circ}+\mathrm{i} \sin 40^{\circ})\)

Solution:

\((\cos 20^{\circ}+\mathrm{i} \sin 20^{\circ})(\cos 30^{\circ}+\mathrm{i} \sin 30^{\circ})(\cos 40^{\circ}+\mathrm{i} \sin 40^{\circ})\)
\(=\cos \left(20^{\circ}+30^{\circ}+40^{\circ}\right)+\mathrm{i} \sin \left(20^{\circ}+30^{\circ}+40^{\circ}\right)\)
\(=\cos 90^{\circ}+\mathrm{i} \sin 90^{\circ}\)
\(=0+\mathrm{i}(1)=\mathrm{i}\)

7. If \(x=\cos \theta+i \sin \theta\) find \(x+\frac{1}{x}\).

Solution:

Given: \(x=\cos \theta+\mathrm{i} \sin \theta\)
\(\Rightarrow \quad \frac{1}{x}=\cos \theta-i \sin \theta\)
\(\therefore x+\frac{1}{x} =\cos \theta+i \sin \theta+\cos \theta-i \sin \theta\)
\(=2 \cos \theta\)

8. If \(x=\cos \alpha+i \sin \alpha\) and \(y=\cos \beta+i \sin \beta\) find \(xy\).

Solution:

\(xy =(\cos \alpha+i \sin \alpha)(\cos \beta+i \sin \beta)\)
\(=\cos (\alpha+\beta)+i \sin (\alpha+\beta)\)

9. If \(a=\cos \alpha+i \sin \alpha\) and \(b=\cos \beta+i \sin \beta\) find \(\frac{a}{b}\).

Solution:

\(\frac{a}{b} =a(b)^{-1}\)
\(=(\cos \alpha+i \sin \alpha)(\cos \beta+i \sin \beta)^{-1}\)
\(=(\cos \alpha+i \sin \alpha)(\cos \beta-i \sin \beta)\)
\(=\cos (\alpha-\beta)+i \sin (\alpha-\beta)\)

10. Find the product of \(3\left(\cos \frac{\pi}{3}+\mathrm{i} \sin \frac{\pi}{3}\right)\) and \(4\left(\cos \frac{\pi}{6}+\mathrm{i} \sin \frac{\pi}{6}\right)\).

Solution:

\(3\left(\cos \frac{\pi}{3}+\mathrm{i} \sin \frac{\pi}{3}\right) \times 4\left(\cos \frac{\pi}{6}+\mathrm{i} \sin \frac{\pi}{6}\right)\)
\(=12\left[\cos \left(\frac{\pi}{3}+\frac{\pi}{6}\right)+\mathrm{i} \sin \left(\frac{\pi}{3}+\frac{\pi}{6}\right)\right]\)
\(=12\left[\cos \frac{\pi}{2}+\mathrm{i} \sin \frac{\pi}{2}\right]\)
\(=12[0+i(1)]=12i\)

PART - B

1. If \(x=\cos \theta+i \sin \theta\) find the value of \(x^{m}+\frac{1}{x^{m}}\).

Solution:

Given: \(x=\cos \theta+\mathrm{i} \sin \theta\)
\(\Rightarrow x^{m}=(\cos \theta+\mathrm{i} \sin \theta)^{m}=\cos m \theta+\mathrm{i} \sin m \theta\)
also, \(\frac{1}{x^{m}}=\left(x^{m}\right)^{-1}\)
\(=[\cos m \theta+\mathrm{i} \sin m \theta]^{-1}\)
\(\Rightarrow \frac{1}{x^{m}}=\cos m \theta-\mathrm{i} \sin m \theta\)
\(\therefore x^{m}+\frac{1}{x^{m}}=\cos m \theta+i \sin m \theta+\cos m \theta-i \sin m \theta=2 \cos m \theta\)

2. If \(a=\cos \alpha+i \sin \alpha\) and \(b=\cos \beta+i \sin \beta\) find \(a b+\frac{1}{a b}\).

Solution:

Given: \(a=\cos \alpha+\mathrm{i} \sin \alpha\)
& \(b=\cos \beta+i \sin \beta\)
\(a b=(\cos \alpha+i \sin \alpha)(\cos \beta+i \sin \beta)\)
\(\Rightarrow a b=\cos (\alpha+\beta)+i \sin (\alpha+\beta)\)
and \(\frac{1}{a b}=\cos (\alpha+\beta)-i \sin (\alpha+\beta)\)
\(\therefore a b+\frac{1}{a b} =\cos (\alpha+\beta)+i \sin (\alpha+\beta)+\cos (\alpha+\beta)-i \sin (\alpha+\beta)\)
\(=2 \cos (\alpha+\beta)\)

3. Prove that \((\sin \theta+\mathrm{i} \cos \theta)^{n}=\cos n\left(\frac{\pi}{2}-\theta\right)+\mathrm{i} \sin n\left(\frac{\pi}{2}-\theta\right)\).

Solution:

We have \(\quad \sin \theta=\cos \left(\frac{\pi}{2}-\theta\right)\)
& \(\cos \theta=\sin \left(\frac{\pi}{2}-\theta\right)\)
LHS: \((\sin \theta+i \cos \theta)^{n}=\left[\cos \left(\frac{\pi}{2}-\theta\right)+i \sin \left(\frac{\pi}{2}-\theta\right)\right]^{n}\)
\(=\cos n\left(\frac{\pi}{2}-\theta\right)+i \sin n\left(\frac{\pi}{2}-\theta\right)\) = RHS

4. If \(x=\cos 3 \alpha+i \sin 3 \alpha, y=\cos 3 \beta+i \sin 3 \beta\) find the value \(\sqrt[3]{xy}\).

Solution:

Given: \(x=\cos 3 \alpha+i \sin 3 \alpha\)
& \(y=\cos 3 \beta+i \sin 3 \beta\)
\(x y=(\cos 3 \alpha+i \sin 3 \alpha)(\cos 3 \beta+i \sin 3 \beta)\)
\(=\cos (3 \alpha+3 \beta)+i \sin (3 \alpha+3 \beta)\)
\(\Rightarrow x y=\cos 3(\alpha+\beta)+i \sin 3(\alpha+\beta)\)
\(\sqrt[3]{x y}=(x y)^{1 / 3}\)
\(=[\cos 3(\alpha+\beta)+i \sin 3(\alpha+\beta)]^{1 / 3}\)
\(=\cos \frac{1}{3} \cdot 3(\alpha+\beta)+i \sin \frac{1}{3} \cdot 3(\alpha+\beta)\)
\(=\cos (\alpha+\beta)+i \sin (\alpha+\beta)\)

5. If \(a=\cos \alpha+i \sin \alpha, b=\cos \beta+i \sin \beta\) and \(c=\cos \gamma+i \sin \gamma\) find the value of \(\frac{a b}{c}\).

Solution:

Given: \(a=\cos \alpha+i \sin \alpha\)
\(b=\cos \beta+i \sin \beta\)
& \(c=\cos \gamma+i \sin \gamma\)
\(\therefore \frac{a b}{c}=a b(c)^{-1}\)
\(=(\cos \alpha+i \sin \alpha)(\cos \beta+i \sin \beta)(\cos \gamma+i \sin \gamma)^{-1}\)
\(=(\cos \alpha+i \sin \alpha)(\cos \beta+i \sin \beta)(\cos \gamma-i \sin \gamma)\)
\( = (\cos(\alpha+\beta) + i \sin(\alpha+\beta)) (\cos \gamma-i \sin \gamma) \)
\(\Rightarrow \frac{a b}{c}=\cos (\alpha+\beta-\gamma)+i \sin (\alpha+\beta-\gamma)\)

PART - C

1. Simplify: \(\frac{(\cos 2 \theta+\mathrm{i} \sin 2 \theta)^{3}(\cos 3 \theta-\mathrm{i} \sin 3 \theta)^{4}}{(\cos 3 \theta+\mathrm{i} \sin 3 \theta)^{2}(\cos 4 \theta+\mathrm{i} \sin 4 \theta)^{-3}}\)

Solution:

\(\frac{(\cos 2 \theta+i \sin 2 \theta)^{3}(\cos 3 \theta-i \sin 3 \theta)^{4}}{(\cos 3 \theta+i \sin 3 \theta)^{2}(\cos 4 \theta+i \sin 4 \theta)^{-3}}\)
\(=\frac{(\cos \theta+\mathrm{i} \sin \theta)^{3 \times 2}(\cos \theta+\mathrm{i} \sin \theta)^{4 \times-3}}{(\cos \theta+\mathrm{i} \sin \theta)^{2 \times 3}(\cos \theta+\mathrm{i} \sin \theta)^{-3 \times 4}}\)
\(=\frac{(\cos \theta+\mathrm{i} \sin \theta)^{6}(\cos \theta+\mathrm{i} \sin \theta)^{-12}}{(\cos \theta+\mathrm{i} \sin \theta)^{6}(\cos \theta+\mathrm{i} \sin \theta)^{-12}}\)
\(=\frac{(\cos \theta+\mathrm{i} \sin \theta)^{-6}}{(\cos \theta+\mathrm{i} \sin \theta)^{-6}}\)
\(=(\cos \theta+\mathrm{i} \sin \theta)^{0}\)
\(=\cos 0+\mathrm{i} \sin 0\)
\(=1+\mathrm{i}(0)=1\)

2. Simplify : \(\frac{(\cos 2 \theta+\mathrm{i} \sin 2 \theta)^{3}(\cos 4 \theta-\mathrm{i} \sin 4 \theta)^{3}}{\cos 3 \theta+\mathrm{i} \sin 3 \theta}\) when \(\theta=\frac{\pi}{9}\).

Solution:

\(\frac{(\cos 2 \theta+\mathrm{i} \sin 2 \theta)^{3}(\cos 4 \theta-\mathrm{i} \sin 4 \theta)^{3}}{\cos 3 \theta+\mathrm{i} \sin 3 \theta}\)
\(=\frac{(\cos \theta+\mathrm{i} \sin \theta)^{3 \times 2}(\cos \theta+\mathrm{i} \sin \theta)^{3 \times-4}}{(\cos \theta+\mathrm{i} \sin \theta)^{3}}\)
\(=\frac{(\cos \theta+\mathrm{i} \sin \theta)^{6}(\cos \theta+\mathrm{i} \sin \theta)^{-12}}{(\cos \theta+\mathrm{i} \sin \theta)^{3}}\)
\(=\frac{(\cos \theta+\mathrm{i} \sin \theta)^{-6}}{(\cos \theta+\mathrm{i} \sin \theta)^{3}}\)
\(=(\cos \theta+\mathrm{i} \sin \theta)^{-6-3}\)
\(=(\cos \theta+\mathrm{i} \sin \theta)^{-9}\)
\(=\cos (-9 \theta)+i \sin (-9 \theta)\)
\(=\cos 9 \theta-\mathrm{i} \sin 9 \theta \quad\) when \(\theta=\frac{\pi}{9}\)
\(=\cos 9\left(\frac{\pi}{9}\right)-\mathrm{i} \sin 9\left(\frac{\pi}{9}\right)\)
\(=\cos \pi-\mathrm{i} \sin \pi\)
\(=-1-\mathrm{i}(0)=-1\)

LHS : \(\left(\frac{\cos \theta+i \sin \theta}{\sin \theta+i \cos \theta}\right)^{4}\)

\( = \left(\frac{(\cos \theta+i \sin \theta)^{1 - (-1)}}{i}\right)^{4} = \left(\frac{(\cos \theta+i \sin \theta)^{2}}{i}\right)^{4} \)
\( = \frac{((\cos \theta+i \sin \theta)^{2})^4}{i^4} = \frac{(\cos \theta+i \sin \theta)^{8}}{1} \)
\(=\cos 8 \theta+i \sin 8 \theta=\text { RHS }\)

\( = \left[ (-1) (\cos \theta+i \sin \theta)^{1 - (-1)} \right]^4 = \left[ (-1) (\cos \theta+i \sin \theta)^{2} \right]^{4} \)
\( = (-1)^4 ((\cos \theta+i \sin \theta)^{2})^{4} \)
\( = (1) (\cos \theta+i \sin \theta)^{8} \)

4. Prove that \(\left[\frac{1+\cos \theta+i \sin \theta}{1+\cos \theta-i \sin \theta}\right]^{n}=\cos n \theta+i \sin n \theta\).

Solution:

Let \(z=\cos \theta+\mathrm{i} \sin \theta\)
\(\Rightarrow \frac{1}{Z}=\cos \theta-\mathrm{i} \sin \theta\)
LHS : \(\left[\frac{1+\cos \theta+i \sin \theta}{1+\cos \theta-i \sin \theta}\right]^{\mathrm{n}}\)
Consider the numerator: \(1 + \cos \theta + i \sin \theta = (1+\cos \theta) + i \sin \theta = 2 \cos^2(\theta/2) + i (2 \sin(\theta/2)\cos(\theta/2)) = 2 \cos(\theta/2)[\cos(\theta/2) + i \sin(\theta/2)]\)
Consider the denominator: \(1 + \cos \theta - i \sin \theta = (1+\cos \theta) - i \sin \theta = 2 \cos^2(\theta/2) - i (2 \sin(\theta/2)\cos(\theta/2)) = 2 \cos(\theta/2)[\cos(\theta/2) - i \sin(\theta/2)]\)
LHS = \( \left[ \frac{2 \cos(\theta/2)(\cos(\theta/2) + i \sin(\theta/2))}{2 \cos(\theta/2)(\cos(\theta/2) - i \sin(\theta/2))} \right]^n = \left[ \frac{\cos(\theta/2) + i \sin(\theta/2)}{\cos(-\theta/2) + i \sin(-\theta/2)} \right]^n \)
\( = \left[ \cos(\theta/2 - (-\theta/2)) + i \sin(\theta/2 - (-\theta/2)) \right]^n = [\cos \theta + i \sin \theta]^n \)
\( = z^n \)
\(=\cos n \theta+\mathrm{i} \sin \mathrm{n} \theta=\mathrm{RHS}\)

5. Show that \(\left[\frac{1+\sin A+i \cos A}{1+\sin A-i \cos A}\right]^{n}=\cos n\left(\frac{\pi}{2}-A\right)+i \sin n\left(\frac{\pi}{2}-A\right)\).

Solution:

Let \(z=\sin \mathrm{A}+\mathrm{i} \cos \mathrm{A}\)
\(\Rightarrow z=\cos \left(\frac{\pi}{2}-\mathrm{A}\right)+\mathrm{i} \sin \left(\frac{\pi}{2}-\mathrm{A}\right)\)
\(\therefore \frac{1}{Z}=\cos \left(\frac{\pi}{2}-\mathrm{A}\right)-\mathrm{i} \sin \left(\frac{\pi}{2}-\mathrm{A}\right)=\sin \mathrm{A}-\mathrm{i} \cos \mathrm{A}\)

LHS \(\left[\frac{1+\sin A+i \cos A}{1+\sin A-i \cos A}\right]^{\mathrm{n}}\)
\(=\left[\frac{1+z}{1+\frac{1}{z}}\right]^{\mathrm{n}}\)
\(=\left[\frac{1+z}{\frac{z+1}{z}}\right]^{n}\)
\(=\left[\frac{z(1+z)}{(1+z)}\right]^{n}\)
\( =(z)^{n} \)
\( =\left[\cos \left(\frac{\pi}{2}-\mathrm{A}\right)+\mathrm{i} \sin \left(\frac{\pi}{2}-\mathrm{A}\right)\right]^{\mathrm{n}} \)
\( =\cos n\left(\frac{\pi}{2}-\mathrm{A}\right)+\mathrm{i} \sin n\left(\frac{\pi}{2}-\mathrm{A}\right) = RHS \)

6. If \(a=\cos \theta+i \sin \theta, b=\cos \phi+i \sin \phi\) prove that
(i) \(\cos (\theta+\phi)=\frac{1}{2}\left[a b+\frac{1}{a b}\right]\)
(ii) \(\sin (\theta-\phi)=\frac{1}{2 \mathrm{i}}\left[\frac{a}{b}-\frac{b}{a}\right]\)

Solution:

Given \(a=\cos \theta+\mathrm{i} \sin \theta\)
& \(b=\cos \phi+i \sin \phi\)

Now, \(ab=(\cos \theta+\mathrm{i} \sin \theta)(\cos \phi+\mathrm{i} \sin \phi)\)
\(\Rightarrow ab=\cos (\theta+\phi)+\mathrm{i} \sin (\theta+\phi)\) .....(1)
also, \(\frac{1}{ab}=(ab)^{-1}=[\cos (\theta+\phi)+\mathrm{i} \sin (\theta+\phi)]^{-1}\)
\(\Rightarrow \frac{1}{ab}=\cos (\theta+\phi)-\mathrm{i} \sin (\theta+\phi)\) .....(2)

\(\therefore(1)+(2) \Rightarrow\)
\(ab+\frac{1}{ab}=\cos (\theta+\phi)+\mathrm{i} \sin (\theta+\phi)+\cos (\theta+\phi)-\mathrm{i} \sin (\theta+\phi)\)
\(\Rightarrow ab+\frac{1}{ab}=2 \cos (\theta+\phi)\)
\(\Rightarrow \cos (\theta+\phi)=\frac{1}{2}\left[a b+\frac{1}{a b}\right]\)

(ii) \(\frac{a}{b}=a(b)^{-1}\)
\(=(\cos \theta+i \sin \theta)(\cos \phi+i \sin \phi)^{-1}\)
\(=(\cos \theta+\mathrm{i} \sin \theta)(\cos \phi-\mathrm{i} \sin \phi)\)
\(\Rightarrow \frac{a}{b}=\cos (\theta-\phi)+\mathrm{i} \sin (\theta-\phi)\).....(3)
also, \(\frac{b}{a}=\left(\frac{a}{b}\right)^{-1}=[\cos (\theta-\phi)+\mathrm{i} \sin (\theta-\phi)]^{-1}\)
\(\Rightarrow \frac{b}{a}=\cos (\theta-\phi)-\mathrm{i} \sin (\theta-\phi)\) .....(4)

(3) \(-(4) \Rightarrow\)
\(\frac{a}{b}-\frac{b}{a}=\cos (\theta-\phi)+\mathrm{i} \sin (\theta-\phi)-(\cos (\theta-\phi)-\mathrm{i} \sin (\theta-\phi))\)
\(\Rightarrow \frac{a}{b}-\frac{b}{a}=2 \mathrm{i} \sin (\theta-\phi)\)
\(\Rightarrow \sin (\theta-\phi)=\frac{1}{2 \mathrm{i}}\left[\frac{a}{b}-\frac{b}{a}\right]\)

7. If \(a=\cos x+i \sin x, b=\cos y+i \sin y\) prove that
(i) \(\sqrt{a b}+\frac{1}{\sqrt{a b}}=2 \cos \left(\frac{x+y}{2}\right)\)
(ii) \(\sqrt{\frac{a}{b}}+\sqrt{\frac{b}{a}}=2 \cos \left(\frac{x-y}{2}\right)\)

Solution:

Given: \(a=\cos x+i \sin x\)
& \(b=\cos y+i \sin y\)

(i) Now, \(ab=(\cos x+i \sin x)(\cos y+i \sin y)\)
\(\Rightarrow a b=\cos (x+y)+i \sin (x+y)\)
\(\therefore \sqrt{a b}=(a b)^{1 / 2}=[\cos (x+y)+i \sin (x+y)]^{1 / 2}\)
\(\Rightarrow \sqrt{a b}=\cos \left(\frac{x+y}{2}\right)+i \sin \left(\frac{x+y}{2}\right)\) .....(1)
also, \(\frac{1}{\sqrt{a b}}=(\sqrt{a b})^{-1}=\left[\cos \left(\frac{x+y}{2}\right)+i \sin \left(\frac{x+y}{2}\right)\right]^{-1}\)
\(\Rightarrow \frac{1}{\sqrt{a b}}=\cos \left(\frac{x+y}{2}\right)-i \sin \left(\frac{x+y}{2}\right)\) .....(2)
\(\therefore(1)+(2) \Rightarrow\)
\(\sqrt{a b}+\frac{1}{\sqrt{a b}}=\cos \left(\frac{x+y}{2}\right)+i \sin \left(\frac{x+y}{2}\right)+\cos \left(\frac{x+y}{2}\right)-i \sin \left(\frac{x+y}{2}\right)\)
\(\Rightarrow \sqrt{a b}+\frac{1}{\sqrt{a b}}=2 \cos \left(\frac{x+y}{2}\right)\)

(ii) \(\frac{a}{b}=a(b)^{-1} =(\cos x+i \sin x)(\cos y+i \sin y)^{-1}\)
\(=(\cos x+i \sin x)(\cos y-i \sin y)\)
\(\Rightarrow \quad \frac{a}{b} =\cos (x-y)+i \sin (x-y)\)

\(\therefore \sqrt{\frac{a}{b}}=\left(\frac{a}{b}\right)^{1 / 2}=[\cos (x-y)+i \sin (x-y)]^{1 / 2}\)
\(\Rightarrow \quad \sqrt{\frac{a}{b}}=\cos \left(\frac{x-y}{2}\right)+i \sin \left(\frac{x-y}{2}\right)\) .....(3)
also, \(\quad \sqrt{\frac{b}{a}}=\left(\sqrt{\frac{a}{b}}\right)^{-1}=\left[\cos \left(\frac{x-y}{2}\right)+i \sin \left(\frac{x-y}{2}\right)\right]^{-1}\)
\(\Rightarrow \quad \sqrt{\frac{b}{a}}=\cos \left(\frac{x-y}{2}\right)-i \sin \left(\frac{x-y}{2}\right)\) .....(4)

\(\therefore(3)+(4) \Rightarrow\)
\(\sqrt{\frac{a}{b}}+\sqrt{\frac{b}{a}}=\cos \left(\frac{x-y}{2}\right)+i \sin \left(\frac{x-y}{2}\right)+\cos \left(\frac{x-y}{2}\right)-i \sin \left(\frac{x-y}{2}\right)\)
\(\Rightarrow \sqrt{\frac{a}{b}}+\sqrt{\frac{b}{a}}=2 \cos \left(\frac{x-y}{2}\right)\)

8. If \(a=\cos \alpha+i \sin \alpha, b=\cos \beta+i \sin \beta\) and \(c=\cos \gamma+i \sin \gamma\) find the value of \(\frac{a b}{c}-\frac{c}{a b}\).

Solution:

Given:
\(a=\cos \alpha+i \sin \alpha\)
\(b=\cos \beta+i \sin \beta\)
& \(c=\cos \gamma+i \sin \gamma\)

Now,
\(\frac{a b}{c}= a b(c)^{-1}\)
\(=(\cos \alpha+i \sin \alpha)(\cos \beta+i \sin \beta)(\cos \gamma+i \sin \gamma)^{-1}\)
\(=(\cos \alpha+i \sin \alpha)(\cos \beta+i \sin \beta)(\cos \gamma-i \sin \gamma)\)
\(\Rightarrow \quad \frac{a b}{c} =\cos (\alpha+\beta-\gamma)+i \sin (\alpha+\beta-\gamma)\) .....(1)
also, \(\frac{c}{a b} =\left[\frac{a b}{c}\right]^{-1}\)
\(=[\cos (\alpha+\beta-\gamma)+i \sin (\alpha+\beta-\gamma)]^{-1}\)
\(\Rightarrow \frac{c}{a b} =\cos (\alpha+\beta-\gamma)-i \sin (\alpha+\beta-\gamma)\) .....(2)

\(\therefore(1)-(2) \Rightarrow\)
\(\frac{a b}{c}-\frac{c}{a b}=\cos (\alpha+\beta-\gamma)+i \sin (\alpha+\beta-\gamma)-(\cos (\alpha+\beta-\gamma)-i \sin (\alpha+\beta-\gamma))\)
\(\Rightarrow \frac{a b}{c}-\frac{c}{a b}=2 \operatorname{isin}(\alpha+\beta-\gamma)\)

9. If \(x+\frac{1}{x}=2 \cos \theta\) prove that (i) \(x^{n}+\frac{1}{x^{n}}=2 \cos n \theta \quad\) (ii) \(x^{n}-\frac{1}{x^{n}}=2 i \sin n \theta\).

Solution:

Given: \(x+\frac{1}{x}=2 \cos \theta\)
\(\frac{x^{2}+1}{x}=2 \cos \theta\)
\(x^{2}+1=2 x \cos \theta\)
\(x^{2}-2 x \cos \theta+1=0\)

Here \(a=1, b=-2 \cos \theta \& c=1\)
\(\therefore x=\frac{-b \pm \sqrt{b^{2}-4 a c}}{2 a}\)
\(=\frac{2 \cos \theta \pm \sqrt{(-2 \cos \theta)^{2}-4 \times 1 \times 1}}{2 \times 1}\)
\(=\frac{2 \cos \theta \pm \sqrt{4 \cos ^{2} \theta-4}}{2}\)
\(=\frac{2 \cos \theta \pm \sqrt{4\left(\cos ^{2} \theta-1\right)}}{2}\)
\(=\frac{2 \cos \theta \pm \sqrt{4\left(-\sin ^{2} \theta\right)}}{2}\)
\(=\frac{2 \cos \theta \pm \mathrm{i} 2 \sin \theta}{2}\)
\(=\frac{2[\cos \theta \pm \mathrm{i} \sin \theta]}{2}\)
\(\Rightarrow x =\cos \theta \pm \mathrm{i} \sin \theta\)

Consider \(x=\cos \theta+\mathrm{i} \sin \theta\)
\(\therefore x^{n}=(\cos \theta+i \sin \theta)^{n}=\cos n \theta+i \sin n \theta\)
& \(\frac{1}{x^{n}}=\cos n \theta-i \sin n \theta\)

(i) \(x^{n}+\frac{1}{x^{n}}=\cos n \theta+i \sin n \theta+\cos n \theta-i \sin n \theta\)
\(\Rightarrow x^{n}+\frac{1}{x^{n}}=2 \cos n \theta\)

(ii) \(x^{n}-\frac{1}{x^{n}}=\cos n \theta+i \sin n \theta-(\cos n \theta-i \sin n \theta)\)
\(\Rightarrow x^{n}-\frac{1}{x^{n}}=2 i \sin n \theta\)

10. Show that \((1+i)^{n}+(1-i)^{n}=2^{\frac{n+2}{2}} \cos \frac{n \pi}{4}\).

Solution:

Let \(1+\mathrm{i}=\mathrm{r}(\cos \theta+\mathrm{i} \sin \theta)=\mathrm{r} \cos \theta+\mathrm{i} r \sin \theta\)
Equating real & imaginary parts on both sides
\(r \cos \theta=1 \quad \& \quad r \sin \theta=1\)

Now, \((r \cos \theta)^{2}+(r \sin \theta)^{2}=(1)^{2}+(1)^{2}\)
\(\Rightarrow r^{2} \cos ^{2} \theta+r^{2} \sin ^{2} \theta=1+1\)
\(\Rightarrow r^{2}\left(\cos ^{2} \theta+\sin ^{2} \theta\right)=2\)
\(\Rightarrow r^{2}(1) =2\)
\(\Rightarrow r^{2}=2 \quad \Rightarrow \quad r=\sqrt{2}\)

Also, \(\frac{r \sin \theta}{r \cos \theta}=\frac{1}{1}\)
\(\Rightarrow \tan \theta=1\)
\(\Rightarrow \theta=\tan ^{-1}(1)\)
\(\Rightarrow \theta=\frac{\pi}{4}\)
\(\therefore 1+\mathrm{i}=\sqrt{2}\left[\cos \frac{\pi}{4}+\mathrm{i} \sin \frac{\pi}{4}\right]\) .....(1)

Similarly we can prove that
\(1-\mathrm{i}=\sqrt{2}\left[\cos \frac{\pi}{4}-\mathrm{i} \sin \frac{\pi}{4}\right]\). .....(2)

LHS: \((1+i)^{n}+(1-i)^{n}\)
\(=\left[\sqrt{2}\left(\cos \frac{\pi}{4}+\mathrm{i} \sin \frac{\pi}{4}\right)\right]^{\mathrm{n}}+\left[\sqrt{2}\left(\cos \frac{\pi}{4}-\mathrm{i} \sin \frac{\pi}{4}\right)\right]^{\mathrm{n}}\)
\(=(\sqrt{2})^{n}\left[\cos \frac{\pi}{4}+\mathrm{i} \sin \frac{\pi}{4}\right]^{\mathrm{n}}+(\sqrt{2})^{n}\left[\cos \frac{\pi}{4}-\mathrm{i} \sin \frac{\pi}{4}\right]^{\mathrm{n}}\)
\(=(\sqrt{2})^{n}\left[\left(\cos \frac{n \pi}{4}+i \sin \frac{n \pi}{4}\right)+\left(\cos \frac{n \pi}{4}-i \sin \frac{n \pi}{4}\right)\right]\)
\(=2^{\frac{n}{2}}\left[2 \cos \frac{n \pi}{4}\right]\)
\(=2^{\frac{n}{2}+1} \cos \frac{n \pi}{4}\)
\(=2^{\frac{n+2}{2}} \cos \frac{n \pi}{4}=\text { RHS }\)
