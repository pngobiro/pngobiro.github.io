Question 1 (20 marks)

(a) Prove the identity:

$\frac{\tan \theta+\cot \theta}{\sec \theta \operatorname{cosec} \theta}=1$. (5 marks)

(b) Solve the equation:

$\cos 2 \theta+3=5 \cos \theta$ for $0^{\circ} \leq \theta \leq 360^{\circ}$. (7 marks)

(c) Given that angles A, B and C are included angles of a triangle, prove that:

$\sin A+\sin B+\sin C=4 \cos \frac{A}{2} \cos \frac{B}{2} \cos \frac{C}{2}$. (8 marks)

Answer

This question tests trigonometric identities and solving trigonometric equations.

(a) Prove $\frac{\tan \theta+\cot \theta}{\sec \theta \operatorname{cosec} \theta}=1$.

Express LHS in terms of $\sin \theta$ and $\cos \theta$:

Numerator: $\tan \theta + \cot \theta = \frac{\sin \theta}{\cos \theta} + \frac{\cos \theta}{\sin \theta} = \frac{\sin^2 \theta + \cos^2 \theta}{\sin \theta \cos \theta} = \frac{1}{\sin \theta \cos \theta}$.

Denominator: $\sec \theta \csc \theta = \frac{1}{\cos \theta} \cdot \frac{1}{\sin \theta} = \frac{1}{\sin \theta \cos \theta}$.

LHS = $\frac{\text{Numerator}}{\text{Denominator}} = \frac{1/(\sin \theta \cos \theta)}{1/(\sin \theta \cos \theta)} = 1$. (Proven)

(b) Solve $\cos 2 \theta+3=5 \cos \theta$ for $0^{\circ} \leq \theta \leq 360^{\circ}$.

Use the identity $\cos 2\theta = 2\cos^2 \theta - 1$.

$(2\cos^2 \theta - 1) + 3 = 5 \cos \theta$

$2\cos^2 \theta + 2 = 5 \cos \theta$

Rearrange into a quadratic equation in $\cos \theta$: $2\cos^2 \theta - 5\cos \theta + 2 = 0$.

Let $x = \cos \theta$. Then $2x^2 - 5x + 2 = 0$.

Factor the quadratic: $(2x - 1)(x - 2) = 0$.

So, $2x - 1 = 0 \implies x = 1/2$ or $x - 2 = 0 \implies x = 2$.

Substitute back $x = \cos \theta$.

Case 1: $\cos \theta = 1/2$. The principal value (acute angle) is $\theta = \arccos(1/2) = 60^{\circ}$. Since $\cos \theta$ is positive, solutions are in the 1st and 4th quadrants.

1st Quadrant: $\theta_1 = 60^{\circ}$.

4th Quadrant: $\theta_2 = 360^{\circ} - 60^{\circ} = 300^{\circ}$.

Case 2: $\cos \theta = 2$. This is impossible, as the range of $\cos \theta$ is $[-1, 1]$.

The solutions are $\theta = 60^{\circ}$ and $\theta = 300^{\circ}$.

(c) Prove $\sin A+\sin B+\sin C=4 \cos \frac{A}{2} \cos \frac{B}{2} \cos \frac{C}{2}$ for a triangle.

In a triangle, $A+B+C = \pi$ (or $180^{\circ}$). So, $C = \pi - (A+B)$.

Consider the LHS: $\sin A + \sin B + \sin C$.

Use sum-to-product for $\sin A + \sin B = 2 \sin\left(\frac{A+B}{2}\right) \cos\left(\frac{A-B}{2}\right)$.

LHS = $2 \sin\left(\frac{A+B}{2}\right) \cos\left(\frac{A-B}{2}\right) + \sin C$.

Since $A+B = \pi - C$, $\frac{A+B}{2} = \frac{\pi}{2} - \frac{C}{2}$. Therefore, $\sin\left(\frac{A+B}{2}\right) = \sin\left(\frac{\pi}{2} - \frac{C}{2}\right) = \cos\left(\frac{C}{2}\right)$.

Also, use the double angle formula $\sin C = 2 \sin\left(\frac{C}{2}\right) \cos\left(\frac{C}{2}\right)$.

Substitute these into the LHS expression:

LHS = $2 \cos\left(\frac{C}{2}\right) \cos\left(\frac{A-B}{2}\right) + 2 \sin\left(\frac{C}{2}\right) \cos\left(\frac{C}{2}\right)$

Factor out $2 \cos\left(\frac{C}{2}\right)$:

LHS = $2 \cos\left(\frac{C}{2}\right) \left[ \cos\left(\frac{A-B}{2}\right) + \sin\left(\frac{C}{2}\right) \right]$.

Now use $\frac{C}{2} = \frac{\pi}{2} - \frac{A+B}{2}$. So, $\sin\left(\frac{C}{2}\right) = \sin\left(\frac{\pi}{2} - \frac{A+B}{2}\right) = \cos\left(\frac{A+B}{2}\right)$.

LHS = $2 \cos\left(\frac{C}{2}\right) \left[ \cos\left(\frac{A-B}{2}\right) + \cos\left(\frac{A+B}{2}\right) \right]$.

Use the sum-to-product identity $\cos X + \cos Y = 2 \cos\left(\frac{X+Y}{2}\right) \cos\left(\frac{X-Y}{2}\right)$.

Let $X = \frac{A-B}{2}$ and $Y = \frac{A+B}{2}$.

$\frac{X+Y}{2} = \frac{(A-B)/2 + (A+B)/2}{2} = \frac{A/2 - B/2 + A/2 + B/2}{2} = \frac{A}{2}$.

$\frac{X-Y}{2} = \frac{(A-B)/2 - (A+B)/2}{2} = \frac{A/2 - B/2 - A/2 - B/2}{2} = \frac{-B}{2}$.

So, $\cos\left(\frac{A-B}{2}\right) + \cos\left(\frac{A+B}{2}\right) = 2 \cos\left(\frac{A}{2}\right) \cos\left(\frac{-B}{2}\right) = 2 \cos\left(\frac{A}{2}\right) \cos\left(\frac{B}{2}\right)$ (since $\cos(-x) = \cos x$).

Substitute this back into the expression for LHS:

LHS = $2 \cos\left(\frac{C}{2}\right) \left[ 2 \cos\left(\frac{A}{2}\right) \cos\left(\frac{B}{2}\right) \right]$

LHS = $4 \cos\left(\frac{A}{2}\right) \cos\left(\frac{B}{2}\right) \cos\left(\frac{C}{2}\right)$. (Proven)

Identities and equations were solved/proven using standard trigonometric rules and sum-to-product formulas.

Question 2 (20 marks)

(a) Solve the following equations:

(i) $\log_{4}(x+3)+\log_{4}(2-x)=1$; (5 marks)

(ii) $5^{2 x+2}=3^{5 x-1}$. (5 marks)

(b) Solve for $x$ in the equation:

$3^{2 x}+3^{x+1}-4=0$. (5 marks)

(c) Use the elimination and/or substitution method to solve the following simultaneous equation:

$\frac{1}{x}-\frac{2}{y}-\frac{2}{z}=0$

$\frac{2}{x}+\frac{3}{y}+\frac{1}{z}=1$

$\frac{3}{x}-\frac{1}{y}-\frac{3}{z}=3$ (5 marks)

Answer

This question covers solving logarithmic and exponential equations, including those reducible to quadratic form, and solving systems of linear equations involving reciprocals.

(a)

(i) Solve $\log_{4}(x+3)+\log_{4}(2-x)=1$.

Combine the logarithms on the LHS: $\log_4[(x+3)(2-x)] = 1$.

Convert to exponential form: $(x+3)(2-x) = 4^1 = 4$.

Expand the product: $2x - x^2 + 6 - 3x = 4$.

$-x^2 - x + 6 = 4$.

Rearrange into standard quadratic form: $x^2 + x - 2 = 0$.

Factor the quadratic: $(x+2)(x-1) = 0$.

Potential solutions are $x=-2$ and $x=1$.

Check the domain of the original logarithms: $x+3 > 0 \implies x > -3$ and $2-x > 0 \implies x < 2$.

Both $x=-2$ and $x=1$ satisfy the domain conditions $(-3 < x < 2)$.

Solutions: $x = -2, 1$.

(ii) Solve $5^{2 x+2}=3^{5 x-1}$.

Take logarithms of both sides (natural log ln or base 10 log):

$\ln(5^{2x+2}) = \ln(3^{5x-1})$.

Use the power rule of logarithms: $(2x+2)\ln 5 = (5x-1)\ln 3$.

$2x \ln 5 + 2 \ln 5 = 5x \ln 3 - \ln 3$.

Group terms with $x$: $2 \ln 5 + \ln 3 = 5x \ln 3 - 2x \ln 5$.

$2 \ln 5 + \ln 3 = x (5 \ln 3 - 2 \ln 5)$.

$x = \frac{2 \ln 5 + \ln 3}{5 \ln 3 - 2 \ln 5}$.

Calculate approximate value: $\ln 5 \approx 1.6094$, $\ln 3 \approx 1.0986$.

$x \approx \frac{2(1.6094) + 1.0986}{5(1.0986) - 2(1.6094)} = \frac{3.2188 + 1.0986}{5.4930 - 3.2188} = \frac{4.3174}{2.2742} \approx 1.8984$.

Solution: $x = \frac{\ln 25 + \ln 3}{\ln 243 - \ln 25} = \frac{\ln 75}{\ln(243/25)} \approx 1.898$.

(b) Solve $3^{2 x}+3^{x+1}-4=0$.

Rewrite the equation using index laws: $(3^x)^2 + 3^1 \cdot 3^x - 4 = 0$.

Let $y = 3^x$. The equation becomes $y^2 + 3y - 4 = 0$.

Factor the quadratic: $(y+4)(y-1) = 0$.

So, $y = -4$ or $y = 1$.

Substitute back $y = 3^x$.

Case 1: $3^x = -4$. This has no real solution, as $3^x$ must be positive.

Case 2: $3^x = 1$. Since $3^0 = 1$, the solution is $x = 0$.

Solution: $x = 0$.

(c) Solve the system of equations.

Let $a = 1/x$, $b = 1/y$, $c = 1/z$. The system becomes:

(1) $a - 2b - 2c = 0$

(2) $2a + 3b + c = 1$

(3) $3a - b - 3c = 3$

Eliminate $c$.

From (2), $c = 1 - 2a - 3b$. Substitute into (1) and (3).

Substitute into (1): $a - 2b - 2(1 - 2a - 3b) = 0$

$a - 2b - 2 + 4a + 6b = 0 \implies 5a + 4b = 2 \quad (4)$

Substitute into (3): $3a - b - 3(1 - 2a - 3b) = 3$

$3a - b - 3 + 6a + 9b = 3 \implies 9a + 8b = 6 \quad (5)$

Now solve the system in $a$ and $b$:

(4) $5a + 4b = 2$

(5) $9a + 8b = 6$

Multiply (4) by 2: $10a + 8b = 4 \quad (4')$

Subtract (5) from (4'): $(10a + 8b) - (9a + 8b) = 4 - 6$.

$a = -2$.

Substitute $a=-2$ into (4): $5(-2) + 4b = 2 \implies -10 + 4b = 2 \implies 4b = 12 \implies b = 3$.

Substitute $a=-2$ and $b=3$ into the expression for $c$: $c = 1 - 2a - 3b = 1 - 2(-2) - 3(3) = 1 + 4 - 9 = -4$.

Now find $x, y, z$:

$a = 1/x \implies -2 = 1/x \implies x = -1/2$.

$b = 1/y \implies 3 = 1/y \implies y = 1/3$.

$c = 1/z \implies -4 = 1/z \implies z = -1/4$.

Solutions: $x = -1/2$, $y = 1/3$, $z = -1/4$.

Solutions involve applying logarithm properties, solving quadratic equations (sometimes after substitution), and systematic elimination for linear systems.

Question 3 (20 marks)

(a) Given the complex numbers $z_{1}=3+5 j, z_{2}=4-6 j, z_{3}=5+2 j$. Determine in the form $a+b j$:

(i) $z_{1} z_{2}$; (3 marks)

(ii) $z$ if $\frac{1}{z}=\frac{1}{z_{1}}+\frac{1}{z_{2}}+\frac{1}{z_{3}}$. (7 marks)

(b) (i) Express $\sin ^{4} \theta$ in terms of cosines of multiples of $\theta$; (5 marks)

(ii) Find all the solutions to the equation $z^{4}=2+2 \sqrt{3} j$, giving the answers in the form $a+b j$. (5 marks)

Answer

This question covers complex number arithmetic, including multiplication and addition of reciprocals, using De Moivre's theorem (implicitly) for trigonometric power reduction, and finding roots of complex numbers.

(a)

(i) Calculate $z_1 z_2$.

$z_1 z_2 = (3+5j)(4-6j) = 3(4) + 3(-6j) + 5j(4) + 5j(-6j)$

$= 12 - 18j + 20j - 30j^2 = 12 + 2j - 30(-1)$

$= 12 + 2j + 30 = 42 + 2j$.

(ii) Find $z$ if $\frac{1}{z}=\frac{1}{z_{1}}+\frac{1}{z_{2}}+\frac{1}{z_{3}}$.

First find the reciprocals:

$\frac{1}{z_1} = \frac{1}{3+5j} = \frac{1(3-5j)}{(3+5j)(3-5j)} = \frac{3-5j}{3^2 - (5j)^2} = \frac{3-5j}{9 - (-25)} = \frac{3-5j}{34}$.

$\frac{1}{z_2} = \frac{1}{4-6j} = \frac{1(4+6j)}{(4-6j)(4+6j)} = \frac{4+6j}{4^2 - (6j)^2} = \frac{4+6j}{16 - (-36)} = \frac{4+6j}{52} = \frac{2+3j}{26}$.

$\frac{1}{z_3} = \frac{1}{5+2j} = \frac{1(5-2j)}{(5+2j)(5-2j)} = \frac{5-2j}{5^2 - (2j)^2} = \frac{5-2j}{25 - (-4)} = \frac{5-2j}{29}$.

Now add the reciprocals:

$\frac{1}{z} = \left(\frac{3}{34} - \frac{5}{34}j\right) + \left(\frac{2}{26} + \frac{3}{26}j\right) + \left(\frac{5}{29} - \frac{2}{29}j\right)$

$\frac{1}{z} = \left(\frac{3}{34} + \frac{1}{13} + \frac{5}{29}\right) + j \left(-\frac{5}{34} + \frac{3}{26} - \frac{2}{29}\right)$

Find common denominators. For real part: LCD(34, 13, 29) = 13 * 34 * 29 = 12818. For imaginary part: LCD(34, 26, 29) = 13 * 17 * 2 * 29 = 12818.

Real Part: $\frac{3(13)(29) + 1(34)(29) + 5(13)(34)}{12818} = \frac{1131 + 986 + 2210}{12818} = \frac{4327}{12818}$.

Imaginary Part: $\frac{-5(13)(29) + 3(17)(29) - 2(13)(17)}{12818} = \frac{-1885 + 1479 - 442}{12818} = \frac{-848}{12818} = \frac{-424}{6409}$.

So, $\frac{1}{z} = \frac{4327}{12818} - \frac{424}{6409}j$. (Approximation: $0.3376 - 0.0662j$)

Now find $z = \frac{1}{1/z}$.

$z = \frac{1}{\frac{4327}{12818} - \frac{848}{12818}j} = \frac{12818}{4327 - 848j}$

$z = \frac{12818 (4327 + 848j)}{(4327 - 848j)(4327 + 848j)} = \frac{12818 (4327 + 848j)}{4327^2 + 848^2}$

$z = \frac{12818 (4327 + 848j)}{18722929 + 719104} = \frac{12818 (4327 + 848j)}{19442033}$

$z = \frac{12818 \times 4327}{19442033} + j \frac{12818 \times 848}{19442033}$

$z \approx 2.853 + 0.560 j$ (Approximate calculation)

(b)

(i) Express $\sin ^{4} \theta$ in terms of cosines of multiples of $\theta$.

Use the identity $\sin^2 \theta = \frac{1 - \cos 2\theta}{2}$.

$\sin^4 \theta = (\sin^2 \theta)^2 = \left(\frac{1 - \cos 2\theta}{2}\right)^2 = \frac{1}{4}(1 - \cos 2\theta)^2$

$= \frac{1}{4}(1 - 2\cos 2\theta + \cos^2 2\theta)$

Now use the identity $\cos^2 A = \frac{1 + \cos 2A}{2}$ with $A=2\theta$. So $\cos^2 2\theta = \frac{1 + \cos 4\theta}{2}$.

$\sin^4 \theta = \frac{1}{4}\left(1 - 2\cos 2\theta + \frac{1 + \cos 4\theta}{2}\right)$

$= \frac{1}{4}\left(\frac{2 - 4\cos 2\theta + 1 + \cos 4\theta}{2}\right)$

$= \frac{1}{8}(3 - 4\cos 2\theta + \cos 4\theta)$.

(ii) Find roots of $z^{4}=2+2 \sqrt{3} j$.

Convert the complex number $w = 2+2\sqrt{3}j$ to polar form $r(\cos \phi + j \sin \phi)$.

$r = |w| = \sqrt{2^2 + (2\sqrt{3})^2} = \sqrt{4 + 4(3)} = \sqrt{4+12} = \sqrt{16} = 4$.

$\phi = \arg(w)$. Since both real and imaginary parts are positive, it's in the 1st quadrant. $\tan \phi = \frac{2\sqrt{3}}{2} = \sqrt{3}$. So $\phi = \arctan(\sqrt{3}) = \frac{\pi}{3}$ radians (or $60^\circ$).

$w = 4\left(\cos \frac{\pi}{3} + j \sin \frac{\pi}{3}\right)$.

We want to find $z$ such that $z^4 = w$. Let $z = R(\cos \Theta + j \sin \Theta)$.

By De Moivre's Theorem, $z^4 = R^4 (\cos 4\Theta + j \sin 4\Theta)$.

$R^4 (\cos 4\Theta + j \sin 4\Theta) = 4\left(\cos \left(\frac{\pi}{3} + 2k\pi\right) + j \sin \left(\frac{\pi}{3} + 2k\pi\right)\right)$ for $k=0, 1, 2, 3$.

Equating magnitudes: $R^4 = 4 \implies R = \sqrt[4]{4} = (2^2)^{1/4} = 2^{1/2} = \sqrt{2}$.

Equating arguments: $4\Theta = \frac{\pi}{3} + 2k\pi \implies \Theta = \frac{\pi}{12} + \frac{2k\pi}{4} = \frac{\pi}{12} + \frac{k\pi}{2}$.

Find the four roots by plugging in $k=0, 1, 2, 3$:

$k=0: \Theta_0 = \frac{\pi}{12}$. $z_0 = \sqrt{2}\left(\cos \frac{\pi}{12} + j \sin \frac{\pi}{12}\right)$.

$k=1: \Theta_1 = \frac{\pi}{12} + \frac{\pi}{2} = \frac{\pi + 6\pi}{12} = \frac{7\pi}{12}$. $z_1 = \sqrt{2}\left(\cos \frac{7\pi}{12} + j \sin \frac{7\pi}{12}\right)$.

$k=2: \Theta_2 = \frac{\pi}{12} + \pi = \frac{13\pi}{12}$. $z_2 = \sqrt{2}\left(\cos \frac{13\pi}{12} + j \sin \frac{13\pi}{12}\right)$.

$k=3: \Theta_3 = \frac{\pi}{12} + \frac{3\pi}{2} = \frac{\pi + 18\pi}{12} = \frac{19\pi}{12}$. $z_3 = \sqrt{2}\left(\cos \frac{19\pi}{12} + j \sin \frac{19\pi}{12}\right)$.

To express in $a+bj$ form, we need values for $\cos(\pi/12)$ etc. $\cos(\pi/12) = \cos(15^\circ) = \cos(45^\circ-30^\circ) = \cos 45 \cos 30 + \sin 45 \sin 30 = \frac{\sqrt{2}}{2}\frac{\sqrt{3}}{2} + \frac{\sqrt{2}}{2}\frac{1}{2} = \frac{\sqrt{6}+\sqrt{2}}{4}$. $\sin(\pi/12) = \sin(15^\circ) = \sin(45-30) = \sin 45 \cos 30 - \cos 45 \sin 30 = \frac{\sqrt{2}}{2}\frac{\sqrt{3}}{2} - \frac{\sqrt{2}}{2}\frac{1}{2} = \frac{\sqrt{6}-\sqrt{2}}{4}$.

$z_0 = \sqrt{2} \left( \frac{\sqrt{6}+\sqrt{2}}{4} + j \frac{\sqrt{6}-\sqrt{2}}{4} \right) = \frac{\sqrt{12}+2}{4} + j \frac{\sqrt{12}-2}{4} = \frac{2\sqrt{3}+2}{4} + j \frac{2\sqrt{3}-2}{4} = \frac{\sqrt{3}+1}{2} + j \frac{\sqrt{3}-1}{2}$.

Similarly for other roots using angles $105^\circ, 195^\circ, 285^\circ$.

$z_1 (\frac{7\pi}{12}=105^\circ): \sqrt{2}(\cos 105 + j \sin 105) = \sqrt{2}(-\sin 15 + j \cos 15) = \frac{1-\sqrt{3}}{2} + j \frac{1+\sqrt{3}}{2}$.

$z_2 (\frac{13\pi}{12}=195^\circ): \sqrt{2}(\cos 195 + j \sin 195) = \sqrt{2}(-\cos 15 - j \sin 15) = \frac{-1-\sqrt{3}}{2} + j \frac{1-\sqrt{3}}{2}$.

$z_3 (\frac{19\pi}{12}=285^\circ): \sqrt{2}(\cos 285 + j \sin 285) = \sqrt{2}(\sin 15 - j \cos 15) = \frac{\sqrt{3}-1}{2} + j \frac{-1-\sqrt{3}}{2}$.

Complex number operations and finding roots require careful application of definitions and theorems.

Question 4 (20 marks)

(a) Convert the polar equation $r=4 a \cot \theta \operatorname{cosec} \theta$ into a Cartesian equation. (4 marks)

(b) (i) Given that $A \cosh x+B \sinh x=3 e^{x}-4 e^{-x}$ where A and B are constants, determine the values of A and B ; (6 marks)

(ii) Find the logarithmic form of $\cosh ^{-1} x$. (4 marks)

(c) Determine the equations of the tangents to the circle $x^{2}+y^{2}-4 x-2 y-8=0$ which are parallel to the line $3 x+2 y=0$. (6 marks)

Answer

This question involves converting between polar and Cartesian coordinates, working with hyperbolic functions and their definitions, finding inverse hyperbolic functions, and determining tangent lines to a circle.

(a) Convert $r=4 a \cot \theta \operatorname{cosec} \theta$ to Cartesian.

Use the relationships $x = r \cos \theta$, $y = r \sin \theta$, $r^2 = x^2+y^2$.

Rewrite the equation in terms of $\sin \theta$ and $\cos \theta$:

$r = 4a \frac{\cos \theta}{\sin \theta} \frac{1}{\sin \theta} = \frac{4a \cos \theta}{\sin^2 \theta}$.

Multiply both sides by $\sin^2 \theta$: $r \sin^2 \theta = 4a \cos \theta$.

Since $y = r \sin \theta \implies \sin \theta = y/r$ and $x = r \cos \theta \implies \cos \theta = x/r$.

$r \left(\frac{y}{r}\right)^2 = 4a \left(\frac{x}{r}\right)$.

$r \frac{y^2}{r^2} = \frac{4ax}{r}$.

$\frac{y^2}{r} = \frac{4ax}{r}$.

Multiply by $r$ (assuming $r \neq 0$): $y^2 = 4ax$.

This is the equation of a parabola opening to the right.

(b)

(i) Find A and B for $A \cosh x+B \sinh x=3 e^{x}-4 e^{-x}$.

Use the definitions: $\cosh x = \frac{e^x + e^{-x}}{2}$, $\sinh x = \frac{e^x - e^{-x}}{2}$.

$A\left(\frac{e^x + e^{-x}}{2}\right) + B\left(\frac{e^x - e^{-x}}{2}\right) = 3e^x - 4e^{-x}$.

Multiply by 2: $A(e^x + e^{-x}) + B(e^x - e^{-x}) = 6e^x - 8e^{-x}$.

Group terms with $e^x$ and $e^{-x}$: $(A+B)e^x + (A-B)e^{-x} = 6e^x - 8e^{-x}$.

Equate coefficients:

$A + B = 6 \quad (1)$

$A - B = -8 \quad (2)$

Add (1) and (2): $2A = -2 \implies A = -1$.

Substitute $A=-1$ into (1): $-1 + B = 6 \implies B = 7$.

So, $A=-1$ and $B=7$.

(ii) Find the logarithmic form of $\cosh ^{-1} x$.

Let $y = \cosh^{-1} x$. Then $x = \cosh y$. By definition, $x \ge 1$.

$x = \frac{e^y + e^{-y}}{2}$.

$2x = e^y + e^{-y}$.

Multiply by $e^y$: $2x e^y = (e^y)^2 + e^0 = (e^y)^2 + 1$.

Rearrange into a quadratic equation in $e^y$: $(e^y)^2 - 2x e^y + 1 = 0$.

Let $u = e^y$. Then $u^2 - 2xu + 1 = 0$.

Use the quadratic formula for $u$: $u = \frac{-(-2x) \pm \sqrt{(-2x)^2 - 4(1)(1)}}{2(1)}$.

$u = \frac{2x \pm \sqrt{4x^2 - 4}}{2} = \frac{2x \pm \sqrt{4(x^2 - 1)}}{2} = \frac{2x \pm 2\sqrt{x^2 - 1}}{2}$.

$u = x \pm \sqrt{x^2 - 1}$.

So, $e^y = x \pm \sqrt{x^2 - 1}$.

Take the natural logarithm: $y = \ln(x \pm \sqrt{x^2 - 1})$.

Since $y = \cosh^{-1} x$ is usually defined as the non-negative branch, we need to determine which sign to take.

Note that $(x + \sqrt{x^2 - 1})(x - \sqrt{x^2 - 1}) = x^2 - (x^2 - 1) = 1$.

So, $x - \sqrt{x^2 - 1} = \frac{1}{x + \sqrt{x^2 - 1}}$.

Then $\ln(x - \sqrt{x^2 - 1}) = \ln\left(\frac{1}{x + \sqrt{x^2 - 1}}\right) = -\ln(x + \sqrt{x^2 - 1})$.

The two solutions are $y = \ln(x + \sqrt{x^2 - 1})$ and $y = -\ln(x + \sqrt{x^2 - 1})$.

The principal value (non-negative) corresponds to the positive logarithm: $\cosh^{-1} x = \ln(x + \sqrt{x^2 - 1})$.

(c) Find tangents to $x^{2}+y^{2}-4 x-2 y-8=0$ parallel to $3 x+2 y=0$.

Find the center and radius of the circle. Complete the square:

$(x^2 - 4x) + (y^2 - 2y) = 8$.

$(x^2 - 4x + 4) + (y^2 - 2y + 1) = 8 + 4 + 1$.

$(x-2)^2 + (y-1)^2 = 13$.

Center $C = (2, 1)$, Radius $R = \sqrt{13}$.

Question 5 (20 marks)

(a) (i) Prove that, if $x$ is so small that its cube and higher powers can be neglected $\sqrt{\frac{1+x}{1-x}}=1+x+\frac{1}{2} x^{2}$. (6 marks)

(ii) By taking $x=\frac{1}{9}$, prove that $\sqrt{5}$ is approximately equal to $\frac{181}{81}$. (4 marks)

(b) Use binomial theorem to calculate $\sqrt{0.998}$ correct to four significant figures. (4 marks)

(c) The load $F=\frac{\mathrm{kbd}^{3}}{\mathrm{~L}}$. Use binomial theorem to approximate the percentage increase in load $F$ when b is increased by $2\%$, $d$ is increased by $4\%$ and the length $L$ is reduced by $5\%$. (6 marks)

Answer

This question focuses on binomial expansions for approximation and calculating percentage changes.

(a)

(i) Approximate $\sqrt{\frac{1+x}{1-x}} = (1+x)^{1/2}(1-x)^{-1/2}$.

Use $(1+u)^k \approx 1 + ku + \frac{k(k-1)}{2} u^2$.

$(1+x)^{1/2} \approx 1 + \frac{1}{2}x + \frac{\frac{1}{2}(\frac{1}{2}-1)}{2} x^2 = 1 + \frac{1}{2}x - \frac{1}{8}x^2$.

$(1-x)^{-1/2} \approx 1 + (-\frac{1}{2})(-x) + \frac{-\frac{1}{2}(-\frac{1}{2}-1)}{2} (-x)^2 = 1 + \frac{1}{2}x + \frac{-\frac{1}{2}(-\frac{3}{2})}{2} x^2 = 1 + \frac{1}{2}x + \frac{3}{8}x^2$.

Multiply the expansions, keeping terms up to $x^2$:

$(1 + \frac{1}{2}x - \frac{1}{8}x^2)(1 + \frac{1}{2}x + \frac{3}{8}x^2)$

$\approx 1(1 + \frac{1}{2}x + \frac{3}{8}x^2) + \frac{1}{2}x(1 + \frac{1}{2}x) - \frac{1}{8}x^2(1)$

$\approx 1 + \frac{1}{2}x + \frac{3}{8}x^2 + \frac{1}{2}x + \frac{1}{4}x^2 - \frac{1}{8}x^2$

$\approx 1 + (\frac{1}{2}+\frac{1}{2})x + (\frac{3}{8} + \frac{2}{8} - \frac{1}{8})x^2$

$\approx 1 + x + \frac{4}{8}x^2 = 1 + x + \frac{1}{2}x^2$. (Proven)

(ii) Use $x=1/9$ to approximate $\sqrt{5}$.

Evaluate $\sqrt{\frac{1+x}{1-x}}$ with $x=1/9$:

$\sqrt{\frac{1+1/9}{1-1/9}} = \sqrt{\frac{10/9}{8/9}} = \sqrt{\frac{10}{8}} = \sqrt{\frac{5}{4}} = \frac{\sqrt{5}}{2}$.

Now use the approximation $1 + x + \frac{1}{2}x^2$ with $x=1/9$:

$1 + \frac{1}{9} + \frac{1}{2}\left(\frac{1}{9}\right)^2 = 1 + \frac{1}{9} + \frac{1}{2}\left(\frac{1}{81}\right) = 1 + \frac{1}{9} + \frac{1}{162}$.

Combine the terms: $\frac{162}{162} + \frac{18}{162} + \frac{1}{162} = \frac{181}{162}$.

We have the approximation $\frac{\sqrt{5}}{2} \approx \frac{181}{162}$.

Multiply by 2: $\sqrt{5} \approx 2 \times \frac{181}{162} = \frac{181}{81}$. (Proven)

(b) Calculate $\sqrt{0.998}$ using binomial theorem.

$\sqrt{0.998} = (1 - 0.002)^{1/2}$.

Use the expansion $(1+u)^k \approx 1 + ku + \frac{k(k-1)}{2}u^2 + \frac{k(k-1)(k-2)}{6}u^3$.

Here $u = -0.002$ and $k = 1/2$.

Term 1: $1$.

Term 2: $ku = \frac{1}{2}(-0.002) = -0.001$.

Term 3: $\frac{k(k-1)}{2} u^2 = \frac{\frac{1}{2}(-\frac{1}{2})}{2} (-0.002)^2 = -\frac{1}{8} (0.000004) = -0.0000005$.

Term 4: $\frac{k(k-1)(k-2)}{6} u^3 = \frac{\frac{1}{2}(-\frac{1}{2})(-\frac{3}{2})}{6} (-0.002)^3 = \frac{3/8}{6} (-0.000000008) = \frac{1}{16} (-8 \times 10^{-9}) = -0.5 \times 10^{-9}$. (Negligible for 4 sig figs).

$\sqrt{0.998} \approx 1 - 0.001 - 0.0000005 = 0.9989995$.

Correct to four significant figures: $0.9990$.

(c) Percentage change in $F=\frac{\mathrm{kbd}^{3}}{\mathrm{~L}}$.

Let the original values be $F_0, b_0, d_0, L_0$. $F_0 = \frac{kb_0 d_0^3}{L_0}$.

The new values are $b = b_0(1+0.02)$, $d = d_0(1+0.04)$, $L = L_0(1-0.05)$.

The new load $F = \frac{k b d^3}{L} = \frac{k [b_0(1.02)] [d_0(1.04)]^3}{L_0(0.95)}$.

$F = \frac{kb_0 d_0^3}{L_0} \frac{(1.02)(1.04)^3}{(0.95)} = F_0 \frac{(1.02)(1.04)^3}{(0.95)}$.

We need to approximate $(1.04)^3$ and $(0.95)^{-1} = (1-0.05)^{-1}$.

Using $(1+x)^n \approx 1+nx$ for small $x$.

$(1+0.04)^3 \approx 1 + 3(0.04) = 1 + 0.12 = 1.12$.

$(1-0.05)^{-1} \approx 1 + (-1)(-0.05) = 1 + 0.05 = 1.05$.

$F \approx F_0 (1.02)(1.12)(1.05)$.

Multiply the factors: $1.02 \times 1.12 = 1.1424$.

$1.1424 \times 1.05 \approx 1.1424 (1 + 0.05) = 1.1424 + 0.05712 = 1.19952$.

$F \approx F_0 (1.19952)$.

The change is $F - F_0 \approx F_0 (1.19952 - 1) = 0.19952 F_0$.

Percentage change = $\frac{F - F_0}{F_0} \times 100\% \approx 0.19952 \times 100\% = 19.952\%$.

Approximate percentage increase is $20.0\%$.

Alternatively, using calculus (small changes): $\ln F = \ln k + \ln b + 3 \ln d - \ln L$. Differentiating: $\frac{\delta F}{F} = \frac{\delta b}{b} + 3 \frac{\delta d}{d} - \frac{\delta L}{L}$. Percentage change $\approx (\% \text{ change in } b) + 3(\% \text{ change in } d) - (\% \text{ change in } L)$. $\approx (2\%) + 3(4\%) - (-5\%) = 2\% + 12\% + 5\% = 19\%$. The binomial method gives a slightly more refined answer by including second-order terms implicitly in the multiplication. $19.952\%$ is likely the expected answer using the factors.

Binomial theorem is used for direct approximation and estimating effects of small percentage changes.

Question 6 (20 marks)

(a) Differentiate the following functions:

(i) $\log _{e}\left(\frac{1+x}{1-x}\right)$; (4 marks)

(ii) $\sin ^{-1}\left(\frac{5}{3} x\right)$. (4 marks)

(b) A curve is defined by the parametric equations $x=\sin ^{2} t$ and $y=\cos t$. Determine the value of $\frac{d y}{d x}$ at the point of $t=\frac{\pi}{3}$. (5 marks)

(c) A particle P moves in a straight line. After t seconds, the displacement in metres of P from a fixed point O on the line is given by $s=t^{3}-2 t^{2}+4 t$. Calculate the:

(i) distance between P and O given that the time $\mathrm{t}=2$; (2 marks)

(ii) times at which the velocity of P equals 4 metres per second. (3 marks)

(d) Determine the point of inflexion on the curve $y=x^{3}-3 x^{2}-2$. (2 marks)

Answer

This question covers differentiation techniques including logarithmic properties, chain rule, inverse trigonometric functions, parametric differentiation, and applications to motion (velocity) and curve analysis (points of inflexion).

(a) Differentiate.

(i) $y = \ln\left(\frac{1+x}{1-x}\right)$.

Using logarithm properties: $y = \ln(1+x) - \ln(1-x)$.

Differentiate term by term:

$\frac{dy}{dx} = \frac{d}{dx}(\ln(1+x)) - \frac{d}{dx}(\ln(1-x))$

$\frac{dy}{dx} = \frac{1}{1+x} \cdot (1) - \frac{1}{1-x} \cdot (-1) = \frac{1}{1+x} + \frac{1}{1-x}$.

Combine the fractions: $\frac{dy}{dx} = \frac{(1-x) + (1+x)}{(1+x)(1-x)} = \frac{2}{1-x^2}$.

(ii) $y = \sin^{-1}\left(\frac{5}{3} x\right)$.

Use the standard derivative $\frac{d}{du}(\sin^{-1} u) = \frac{1}{\sqrt{1-u^2}}$ and the chain rule.

Let $u = \frac{5}{3}x$. Then $\frac{du}{dx} = \frac{5}{3}$.

$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = \frac{1}{\sqrt{1-u^2}} \cdot \frac{5}{3}$.

Substitute back $u = \frac{5}{3}x$:

$\frac{dy}{dx} = \frac{1}{\sqrt{1-(\frac{5}{3}x)^2}} \cdot \frac{5}{3} = \frac{1}{\sqrt{1-\frac{25}{9}x^2}} \cdot \frac{5}{3}$.

Simplify the term under the square root: $\sqrt{\frac{9-25x^2}{9}} = \frac{\sqrt{9-25x^2}}{3}$.

$\frac{dy}{dx} = \frac{1}{(\sqrt{9-25x^2})/3} \cdot \frac{5}{3} = \frac{3}{\sqrt{9-25x^2}} \cdot \frac{5}{3} = \frac{5}{\sqrt{9-25x^2}}$.

(b) Parametric differentiation: $x=\sin ^{2} t$, $y=\cos t$. Find $\frac{dy}{dx}$ at $t=\frac{\pi}{3}$.

$\frac{dy}{dx} = \frac{dy/dt}{dx/dt}$.

$\frac{dy}{dt} = \frac{d}{dt}(\cos t) = -\sin t$.

$\frac{dx}{dt} = \frac{d}{dt}(\sin^2 t)$. Use chain rule: let $u=\sin t$, $x=u^2$. $\frac{dx}{dt} = \frac{dx}{du}\frac{du}{dt} = (2u)(\cos t) = 2 \sin t \cos t = \sin 2t$.

$\frac{dy}{dx} = \frac{-\sin t}{2 \sin t \cos t} = -\frac{1}{2 \cos t}$ (assuming $\sin t \neq 0$).

Evaluate at $t=\frac{\pi}{3}$ ($60^\circ$):

$\cos(\pi/3) = 1/2$.

$\frac{dy}{dx} \bigg|_{t=\pi/3} = -\frac{1}{2 (1/2)} = -1$.

(c) Motion analysis: $s=t^{3}-2 t^{2}+4 t$.

(i) Distance at $t=2$. Distance from O is the magnitude of displacement $|s|$.

$s(2) = (2)^3 - 2(2)^2 + 4(2) = 8 - 2(4) + 8 = 8 - 8 + 8 = 8$ metres.

Distance from O at $t=2$ is 8 metres.

(ii) Times when velocity $v = 4$ m/s.

Velocity $v = \frac{ds}{dt} = \frac{d}{dt}(t^3 - 2t^2 + 4t) = 3t^2 - 4t + 4$.

Set $v=4$: $3t^2 - 4t + 4 = 4$.

$3t^2 - 4t = 0$.

$t(3t - 4) = 0$.

Solutions are $t=0$ seconds and $3t - 4 = 0 \implies t = 4/3$ seconds.

The velocity is 4 m/s at $t=0$ s and $t=4/3$ s.

(d) Point of inflexion for $y=x^{3}-3 x^{2}-2$.

Points of inflexion occur where the second derivative changes sign, typically where $y''=0$.

$y' = \frac{dy}{dx} = 3x^2 - 6x$.

$y'' = \frac{d^2y}{dx^2} = 6x - 6$.

Set $y'' = 0$: $6x - 6 = 0 \implies 6x = 6 \implies x = 1$.

Find the corresponding y-value: $y(1) = (1)^3 - 3(1)^2 - 2 = 1 - 3 - 2 = -4$.

Check if concavity changes: $y''' = 6$. Since $y''' \neq 0$, the concavity changes at $x=1$.

The point of inflexion is $(1, -4)$.

Derivatives are found using various rules, and applied to find gradients, velocity, and points of inflexion.

Question 7 (20 marks)

(a) Evaluate the following integrals:

(i) $\int_{1}^{2} \frac{x+1}{x\left(x^{2}+1\right)} d x$; (7 marks)

(ii) $\int_{0}^{\frac{\pi}{4}} \sin 5 x \sin 3 x d x$. (5 marks)

(ii) Use integration to determine the area of the region in (a) (i). (Should be b(i)) (5 marks)

Answer

This question involves evaluating definite integrals using partial fractions and trigonometric product-to-sum identities, and finding the area between curves.

(a) Evaluate integrals.

(i) $\int_{1}^{2} \frac{x+1}{x\left(x^{2}+1\right)} d x$.

Use partial fractions for the integrand: $\frac{x+1}{x(x^2+1)} = \frac{A}{x} + \frac{Bx+C}{x^2+1}$.

$x+1 = A(x^2+1) + (Bx+C)x = Ax^2 + A + Bx^2 + Cx$.

$x+1 = (A+B)x^2 + Cx + A$.

Equate coefficients:

Coefficient of $x^2$: $A+B = 0 \implies B = -A$.

Coefficient of $x$: $C = 1$.

Constant term: $A = 1$.

From $B=-A$, we get $B = -1$.

So, $\frac{x+1}{x(x^2+1)} = \frac{1}{x} + \frac{-x+1}{x^2+1} = \frac{1}{x} - \frac{x}{x^2+1} + \frac{1}{x^2+1}$.

The integral is: $\int_{1}^{2} \left( \frac{1}{x} - \frac{x}{x^2+1} + \frac{1}{x^2+1} \right) dx$.

Integrate term by term:

$\int \frac{1}{x} dx = \ln|x|$.

$\int \frac{x}{x^2+1} dx$. Let $u=x^2+1$, $du=2xdx$. Integral is $\frac{1}{2} \int \frac{du}{u} = \frac{1}{2}\ln|u| = \frac{1}{2}\ln(x^2+1)$.

$\int \frac{1}{x^2+1} dx = \arctan(x)$.

The definite integral is: $\left[ \ln|x| - \frac{1}{2}\ln(x^2+1) + \arctan(x) \right]_1^2$.

$= \left( \ln 2 - \frac{1}{2}\ln(2^2+1) + \arctan 2 \right) - \left( \ln 1 - \frac{1}{2}\ln(1^2+1) + \arctan 1 \right)$.

$= \left( \ln 2 - \frac{1}{2}\ln 5 + \arctan 2 \right) - \left( 0 - \frac{1}{2}\ln 2 + \frac{\pi}{4} \right)$.

$= \ln 2 - \frac{1}{2}\ln 5 + \arctan 2 + \frac{1}{2}\ln 2 - \frac{\pi}{4}$.

$= \frac{3}{2}\ln 2 - \frac{1}{2}\ln 5 + \arctan 2 - \frac{\pi}{4}$.

Approximate value: $\frac{3}{2}(0.6931) - \frac{1}{2}(1.6094) + 1.1071 - \frac{3.1416}{4}$

$\approx 1.0397 - 0.8047 + 1.1071 - 0.7854 = 0.5567$.

(ii) $\int_{0}^{\frac{\pi}{4}} \sin 5 x \sin 3 x d x$.

Use the product-to-sum identity: $\sin A \sin B = \frac{1}{2}[\cos(A-B) - \cos(A+B)]$.

Let $A=5x, B=3x$. $A-B=2x$, $A+B=8x$.

$\sin 5x \sin 3x = \frac{1}{2}[\cos(2x) - \cos(8x)]$.

The integral is: $\int_{0}^{\pi/4} \frac{1}{2}(\cos 2x - \cos 8x) dx = \frac{1}{2} \int_{0}^{\pi/4} (\cos 2x - \cos 8x) dx$.

$= \frac{1}{2} \left[ \frac{\sin 2x}{2} - \frac{\sin 8x}{8} \right]_0^{\pi/4}$.

$= \frac{1}{2} \left[ \left(\frac{\sin(2\pi/4)}{2} - \frac{\sin(8\pi/4)}{8}\right) - \left(\frac{\sin 0}{2} - \frac{\sin 0}{8}\right) \right]$.

$= \frac{1}{2} \left[ \left(\frac{\sin(\pi/2)}{2} - \frac{\sin(2\pi)}{8}\right) - (0 - 0) \right]$.

$= \frac{1}{2} \left[ \frac{1}{2} - \frac{0}{8} \right] = \frac{1}{2} \left[ \frac{1}{2} \right] = \frac{1}{4}$.

(b) Area between $y=(x+1)^{2}$ and $y=3 x+3$.

(i) Sketch:

$y=(x+1)^2 = x^2+2x+1$ is a parabola opening upwards, vertex at $(-1, 0)$.

$y=3x+3$ is a straight line with y-intercept 3 and x-intercept -1.

Find points of intersection: Set equations equal.

$(x+1)^2 = 3x+3$.

$x^2+2x+1 = 3x+3$.

$x^2 - x - 2 = 0$.

$(x-2)(x+1) = 0$.

Intersection points at $x=2$ and $x=-1$.

If $x=2$, $y = 3(2)+3 = 9$. Point $(2, 9)$. Check parabola: $y=(2+1)^2 = 3^2 = 9$.

If $x=-1$, $y = 3(-1)+3 = 0$. Point $(-1, 0)$. Check parabola: $y=(-1+1)^2 = 0^2 = 0$.

Sketch: Draw the parabola and the line intersecting at $(-1,0)$ and $(2,9)$. The line is above the parabola between these points.

(ii) Determine the area.

Area $A = \int_{x_1}^{x_2} (\text{Upper curve} - \text{Lower curve}) dx$.

Here $x_1 = -1$, $x_2 = 2$. Upper curve is $y=3x+3$. Lower curve is $y=(x+1)^2$.

$A = \int_{-1}^{2} [(3x+3) - (x+1)^2] dx$.

$A = \int_{-1}^{2} [3x+3 - (x^2+2x+1)] dx$.

$A = \int_{-1}^{2} (3x+3 - x^2 - 2x - 1) dx = \int_{-1}^{2} (-x^2 + x + 2) dx$.

Question 8 (20 marks)

(a) Find the slope of the tangent to the curve $(x+3)^{2}-4(y-2)^{2}=9$ at the point $(2,4)$. (5 marks)

(b) Given that $z=\frac{1}{\sqrt{x^{2}+y^{2}}}$, determine:

(i) $\frac{\partial^{2} z}{\partial x^{2}}$; (5 marks)

(ii) $\frac{\partial^{2} z}{\partial y^{2}}$; (Note: Similar to (i) by symmetry) (3 marks)

(iii) $\frac{\partial^{2} z}{\partial x \partial y}$. (Mistyped? Should be $\partial^2 z / \partial x \partial y$) (7 marks)

(c) Determine the stationary values of the function $f(x, y)=x^{3}-3 x^{2}-4 y^{2}+2$. (Structure unclear, assume finding and classifying stationary points)

Answer

This question involves implicit differentiation, partial differentiation (including second-order derivatives), and finding/classifying stationary points of a function of two variables.

(a) Slope of tangent to $(x+3)^{2}-4(y-2)^{2}=9$ at $(2,4)$.

Differentiate implicitly with respect to $x$:

$\frac{d}{dx}[(x+3)^2] - \frac{d}{dx}[4(y-2)^2] = \frac{d}{dx}[9]$.

$2(x+3)(1) - 4[2(y-2)] \frac{dy}{dx} = 0$.

$2(x+3) - 8(y-2) \frac{dy}{dx} = 0$.

$8(y-2) \frac{dy}{dx} = 2(x+3)$.

$\frac{dy}{dx} = \frac{2(x+3)}{8(y-2)} = \frac{x+3}{4(y-2)}$.

Evaluate the slope at the point $(2,4)$:

Slope $m = \frac{dy}{dx} \bigg|_{(2,4)} = \frac{2+3}{4(4-2)} = \frac{5}{4(2)} = \frac{5}{8}$.

(b) Partial derivatives of $z=(x^{2}+y^{2})^{-1/2}$.

First derivatives:

$\frac{\partial z}{\partial x} = -\frac{1}{2}(x^2+y^2)^{-3/2} \cdot (2x) = -x(x^2+y^2)^{-3/2}$.

$\frac{\partial z}{\partial y} = -\frac{1}{2}(x^2+y^2)^{-3/2} \cdot (2y) = -y(x^2+y^2)^{-3/2}$.

(i) $\frac{\partial^{2} z}{\partial x^{2}} = \frac{\partial}{\partial x}[-x(x^2+y^2)^{-3/2}]$. Use product rule.

$= (-1)(x^2+y^2)^{-3/2} + (-x) \left[ -\frac{3}{2}(x^2+y^2)^{-5/2} \cdot (2x) \right]$.

$= -(x^2+y^2)^{-3/2} + (-x) [-3x(x^2+y^2)^{-5/2}]$.

$= -(x^2+y^2)^{-3/2} + 3x^2(x^2+y^2)^{-5/2}$.

Factor out $(x^2+y^2)^{-5/2}$:

$= (x^2+y^2)^{-5/2} [ -(x^2+y^2)^1 + 3x^2 ]$.

$= (x^2+y^2)^{-5/2} [ -x^2-y^2 + 3x^2 ] = (x^2+y^2)^{-5/2} [ 2x^2-y^2 ]$.

$\frac{\partial^{2} z}{\partial x^{2}} = \frac{2x^2-y^2}{(x^2+y^2)^{5/2}}$.

(ii) $\frac{\partial^{2} z}{\partial y^{2}}$. By symmetry (swapping x and y in the expression for $\frac{\partial^{2} z}{\partial x^{2}}$):

$\frac{\partial^{2} z}{\partial y^{2}} = \frac{2y^2-x^2}{(x^2+y^2)^{5/2}}$.

(iii) $\frac{\partial^{2} z}{\partial x \partial y} = \frac{\partial}{\partial x}\left(\frac{\partial z}{\partial y}\right) = \frac{\partial}{\partial x}[-y(x^2+y^2)^{-3/2}]$.

Treat $-y$ as a constant with respect to $x$.

$= (-y) \left[ -\frac{3}{2}(x^2+y^2)^{-5/2} \cdot (2x) \right]$.

$= (-y) [-3x(x^2+y^2)^{-5/2}]$.

$= 3xy(x^2+y^2)^{-5/2} = \frac{3xy}{(x^2+y^2)^{5/2}}$.

(Note: We can verify $\frac{\partial^{2} z}{\partial y \partial x} = \frac{\partial}{\partial y}[-x(x^2+y^2)^{-3/2}] = (-x)[-\frac{3}{2}(x^2+y^2)^{-5/2}(2y)] = 3xy(x^2+y^2)^{-5/2}$, confirming they are equal).

(c) Stationary points of $f(x, y)=x^{3}-3 x^{2}-4 y^{2}+2$.

Find first partial derivatives and set them to zero.

$f_x = \frac{\partial f}{\partial x} = 3x^2 - 6x$.

$f_y = \frac{\partial f}{\partial y} = -8y$.

Set $f_x=0$: $3x^2 - 6x = 0 \implies 3x(x-2) = 0$. So $x=0$ or $x=2$.

Set $f_y=0$: $-8y = 0 \implies y=0$.

The stationary points occur when both derivatives are zero. This gives two points:

Point 1: $(0, 0)$.

Point 2: $(2, 0)$.

Find second partial derivatives to classify the points.

$f_{xx} = \frac{\partial^2 f}{\partial x^2} = 6x - 6$.

$f_{yy} = \frac{\partial^2 f}{\partial y^2} = -8$.

$f_{xy} = \frac{\partial^2 f}{\partial x \partial y} = 0$.

Calculate the discriminant $D = f_{xx} f_{yy} - (f_{xy})^2 = (6x-6)(-8) - (0)^2 = -48(x-1)$.

Classify Point 1: $(0, 0)$.

$f_{xx}(0,0) = 6(0)-6 = -6$.

$D(0,0) = -48(0-1) = 48$.

Since $D > 0$ and $f_{xx} < 0$, the point $(0, 0)$ is a local maximum.

The value at this maximum is $f(0,0) = 0^3 - 3(0)^2 - 4(0)^2 + 2 = 2$.

Classify Point 2: $(2, 0)$.

$f_{xx}(2,0) = 6(2)-6 = 6$.

$D(2,0) = -48(2-1) = -48$.

Since $D < 0$, the point $(2, 0)$ is a saddle point.

The value at this point is $f(2,0) = 2^3 - 3(2)^2 - 4(0)^2 + 2 = 8 - 12 + 2 = -2$.

The question asks for stationary *values*. The stationary values are the function values at the stationary points. The stationary values are 2 (local maximum) and -2 (at a saddle point).

Implicit and partial differentiation rules are applied, and the second derivative test is used to classify stationary points for a function of two variables.
