Question 1 (20 marks)

(a) Prove the identities:

(i) $\quad \tanh 2 x=\frac{2 \tanh x}{1+\tanh ^{2} x}$; (4 marks)

(ii) $\cosh 3 x=4 \cosh ^{3} x-3 \cosh x$. (5 marks)

(b) (i) Express $\operatorname{sech}^{-1} x$ in logarithmic form; (4 marks)

(ii) Given that $\sinh ^{-1} x=\ln \left(x+\sqrt{x^{2}+1}\right)$, find the real root of the equation $\operatorname{sech}^{-1} x=\sinh ^{-1} x$. (7 marks)

Answer

This question tests knowledge of hyperbolic function identities and the logarithmic forms of inverse hyperbolic functions.

(a) Prove identities.

(i) Prove $\tanh 2 x=\frac{2 \tanh x}{1+\tanh ^{2} x}$.

Start with $\tanh(A+B) = \frac{\tanh A + \tanh B}{1 + \tanh A \tanh B}$.

Let $A=x$ and $B=x$.

$\tanh(x+x) = \tanh 2x = \frac{\tanh x + \tanh x}{1 + \tanh x \tanh x} = \frac{2 \tanh x}{1 + \tanh^2 x}$. (Proven)

(ii) Prove $\cosh 3 x=4 \cosh ^{3} x-3 \cosh x$.

Use $\cosh(A+B) = \cosh A \cosh B + \sinh A \sinh B$.

$\cosh 3x = \cosh(2x+x) = \cosh 2x \cosh x + \sinh 2x \sinh x$.

Use double angle identities: $\cosh 2x = 2\cosh^2 x - 1$ and $\sinh 2x = 2\sinh x \cosh x$.

$\cosh 3x = (2\cosh^2 x - 1)\cosh x + (2\sinh x \cosh x)\sinh x$.

$\cosh 3x = 2\cosh^3 x - \cosh x + 2\sinh^2 x \cosh x$.

Use the identity $\sinh^2 x = \cosh^2 x - 1$.

$\cosh 3x = 2\cosh^3 x - \cosh x + 2(\cosh^2 x - 1)\cosh x$.

$\cosh 3x = 2\cosh^3 x - \cosh x + 2\cosh^3 x - 2\cosh x$.

$\cosh 3x = 4\cosh^3 x - 3\cosh x$. (Proven)

(b)

(i) Express $\operatorname{sech}^{-1} x$ in logarithmic form.

Let $y = \operatorname{sech}^{-1} x$. Then $x = \operatorname{sech} y = \frac{1}{\cosh y}$.

Since $y = \operatorname{sech}^{-1} x$, the range implies $y \ge 0$, and the domain is $0 < x \le 1$.

From $x = 1/\cosh y$, we have $\cosh y = 1/x$.

Recall the logarithmic form $\cosh^{-1} u = \ln(u + \sqrt{u^2 - 1})$ for $u \ge 1$.

Here $u = 1/x$. Since $0 < x \le 1$, $u = 1/x \ge 1$.

$y = \cosh^{-1}(1/x) = \ln\left(\frac{1}{x} + \sqrt{(\frac{1}{x})^2 - 1}\right)$.

$y = \ln\left(\frac{1}{x} + \sqrt{\frac{1}{x^2} - 1}\right) = \ln\left(\frac{1}{x} + \sqrt{\frac{1-x^2}{x^2}}\right)$.

$y = \ln\left(\frac{1}{x} + \frac{\sqrt{1-x^2}}{|x|}\right)$. Since $0

$y = \ln\left(\frac{1 + \sqrt{1-x^2}}{x}\right)$.

So, $\operatorname{sech}^{-1} x = \ln\left(\frac{1 + \sqrt{1-x^2}}{x}\right)$.

(ii) Solve $\operatorname{sech}^{-1} x=\sinh ^{-1} x$.

Substitute the logarithmic forms:

$\ln\left(\frac{1 + \sqrt{1-x^2}}{x}\right) = \ln\left(x+\sqrt{x^{2}+1}\right)$.

Since $\ln$ is a one-to-one function, the arguments must be equal:

$\frac{1 + \sqrt{1-x^2}}{x} = x+\sqrt{x^{2}+1}$.

This equation looks difficult to solve directly. Let's reconsider the definitions.

Let $y = \operatorname{sech}^{-1} x = \sinh^{-1} x$.

Then $x = \operatorname{sech} y$ and $x = \sinh y$.

So, $\operatorname{sech} y = \sinh y$.

$\frac{1}{\cosh y} = \sinh y$.

$1 = \sinh y \cosh y$.

Use $\sinh y \cosh y = \frac{1}{2} \sinh 2y$.

$1 = \frac{1}{2} \sinh 2y \implies \sinh 2y = 2$.

Use the definition $\sinh u = (e^u - e^{-u})/2$. Let $u=2y$.

$\frac{e^{2y} - e^{-2y}}{2} = 2 \implies e^{2y} - e^{-2y} = 4$.

Multiply by $e^{2y}$: $(e^{2y})^2 - 1 = 4e^{2y}$.

Let $z = e^{2y}$. Then $z^2 - 1 = 4z \implies z^2 - 4z - 1 = 0$.

Use quadratic formula for $z$: $z = \frac{-(-4) \pm \sqrt{(-4)^2 - 4(1)(-1)}}{2(1)} = \frac{4 \pm \sqrt{16 + 4}}{2} = \frac{4 \pm \sqrt{20}}{2}$.

$z = \frac{4 \pm 2\sqrt{5}}{2} = 2 \pm \sqrt{5}$.

Since $z = e^{2y}$, $z$ must be positive. Both $2+\sqrt{5}$ and $2-\sqrt{5}$ need checking. $\sqrt{5} \approx 2.236$. $2-\sqrt{5}$ is negative.

So, $e^{2y} = 2 + \sqrt{5}$.

$2y = \ln(2 + \sqrt{5})$.

$y = \frac{1}{2} \ln(2 + \sqrt{5})$.

We need the value of $x$. We know $x = \sinh y$.

From $e^{2y} = 2+\sqrt{5}$, we have $e^y = \sqrt{2+\sqrt{5}}$.

$x = \sinh y = \frac{e^y - e^{-y}}{2} = \frac{\sqrt{2+\sqrt{5}} - 1/\sqrt{2+\sqrt{5}}}{2}$.

$x = \frac{(2+\sqrt{5}) - 1}{2\sqrt{2+\sqrt{5}}} = \frac{1+\sqrt{5}}{2\sqrt{2+\sqrt{5}}}$.

Let's check $x = \operatorname{sech} y = 1/\cosh y$.

$\cosh y = \frac{e^y + e^{-y}}{2} = \frac{\sqrt{2+\sqrt{5}} + 1/\sqrt{2+\sqrt{5}}}{2} = \frac{(2+\sqrt{5})+1}{2\sqrt{2+\sqrt{5}}} = \frac{3+\sqrt{5}}{2\sqrt{2+\sqrt{5}}}$.

$x = \frac{1}{\cosh y} = \frac{2\sqrt{2+\sqrt{5}}}{3+\sqrt{5}}$.

Are these $x$ values the same? Let's square the first expression for x:

$x^2 = \frac{(1+\sqrt{5})^2}{4(2+\sqrt{5})} = \frac{1 + 2\sqrt{5} + 5}{8 + 4\sqrt{5}} = \frac{6 + 2\sqrt{5}}{8 + 4\sqrt{5}} = \frac{2(3 + \sqrt{5})}{4(2 + \sqrt{5})} = \frac{3 + \sqrt{5}}{2(2 + \sqrt{5})}$.

$x^2 = \frac{(3+\sqrt{5})(2-\sqrt{5})}{2(2+\sqrt{5})(2-\sqrt{5})} = \frac{6 - 3\sqrt{5} + 2\sqrt{5} - 5}{2(4-5)} = \frac{1 - \sqrt{5}}{2(-1)} = \frac{\sqrt{5}-1}{2}$.

Now square the second expression for x:

$x^2 = \frac{4(2+\sqrt{5})}{(3+\sqrt{5})^2} = \frac{8+4\sqrt{5}}{9+6\sqrt{5}+5} = \frac{8+4\sqrt{5}}{14+6\sqrt{5}} = \frac{4(2+\sqrt{5})}{2(7+3\sqrt{5})} = \frac{2(2+\sqrt{5})}{7+3\sqrt{5}}$.

$x^2 = \frac{2(2+\sqrt{5})(7-3\sqrt{5})}{(7+3\sqrt{5})(7-3\sqrt{5})} = \frac{2(14-6\sqrt{5}+7\sqrt{5}-15)}{49-9(5)} = \frac{2(-1+\sqrt{5})}{49-45} = \frac{2(\sqrt{5}-1)}{4} = \frac{\sqrt{5}-1}{2}$.

The expressions are consistent. So $x^2 = (\sqrt{5}-1)/2$.

Since $x = \sinh y$ and $y > 0$ (because $e^{2y}=2+\sqrt{5}>1$), $x$ must be positive.

$x = \sqrt{\frac{\sqrt{5}-1}{2}}$.

Approximate value: $x = \sqrt{\frac{2.236-1}{2}} = \sqrt{\frac{1.236}{2}} = \sqrt{0.618} \approx 0.786$.

The real root is $x = \sqrt{\frac{\sqrt{5}-1}{2}}$.

Identities are proven using definitions and double angle formulas. The equation requires equating arguments of logarithms or using function definitions, leading to solving a quadratic in an exponential term.

Question 2 (20 marks)

(a) Prove the identity:

$\frac{1-\cos \theta}{\sin \theta}=\frac{1}{\operatorname{cosec} \theta+\cot \theta}$ (5 marks)

(b) Given that A, B and C are angles of a triangle, prove that

$\sin 2 \mathrm{~A}+\sin 2 \mathrm{~B}+\sin 2 \mathrm{C}=4 \sin A \sin B \sin C$ (8 marks)

(c) (i) Express $5 \sin \theta-12 \cos \theta$ in the form $\mathrm{R} \sin (\theta-\alpha)$, where $\mathrm{R}>0$ and $0 \leq \alpha \leq 90^{\circ}$; (3 marks)

(ii) Hence, solve the equation $5 \sin \theta-12 \cos \theta=6$ for $0 \leq \theta \leq 360^{\circ}$. (4 marks)

Answer

This question focuses on proving trigonometric identities, including one related to triangle angles, and solving trigonometric equations using the R-formula method.

(a) Prove $\frac{1-\cos \theta}{\sin \theta}=\frac{1}{\operatorname{cosec} \theta+\cot \theta}$.

Work with the Right Hand Side (RHS):

RHS = $\frac{1}{\csc \theta + \cot \theta} = \frac{1}{\frac{1}{\sin \theta} + \frac{\cos \theta}{\sin \theta}} = \frac{1}{\frac{1+\cos \theta}{\sin \theta}}$.

RHS = $\frac{\sin \theta}{1+\cos \theta}$.

To show this equals the LHS, multiply numerator and denominator by $(1-\cos \theta)$:

RHS = $\frac{\sin \theta (1-\cos \theta)}{(1+\cos \theta)(1-\cos \theta)} = \frac{\sin \theta (1-\cos \theta)}{1 - \cos^2 \theta}$.

Use $1 - \cos^2 \theta = \sin^2 \theta$.

RHS = $\frac{\sin \theta (1-\cos \theta)}{\sin^2 \theta} = \frac{1-\cos \theta}{\sin \theta}$ = LHS. (Proven)

Alternatively, use half-angle identities: $1-\cos \theta = 2\sin^2(\theta/2)$ and $\sin \theta = 2\sin(\theta/2)\cos(\theta/2)$.

LHS = $\frac{2\sin^2(\theta/2)}{2\sin(\theta/2)\cos(\theta/2)} = \frac{\sin(\theta/2)}{\cos(\theta/2)} = \tan(\theta/2)$.

RHS = $\frac{\sin \theta}{1+\cos \theta}$. Use $1+\cos \theta = 2\cos^2(\theta/2)$.

RHS = $\frac{2\sin(\theta/2)\cos(\theta/2)}{2\cos^2(\theta/2)} = \frac{\sin(\theta/2)}{\cos(\theta/2)} = \tan(\theta/2)$. Since LHS=RHS, identity is proven.

(b) Prove $\sin 2 \mathrm{~A}+\sin 2 \mathrm{~B}+\sin 2 \mathrm{C}=4 \sin A \sin B \sin C$ for a triangle.

In a triangle $A+B+C = \pi$. So $C = \pi - (A+B)$.

LHS = $(\sin 2A + \sin 2B) + \sin 2C$.

Use sum-to-product: $\sin X + \sin Y = 2 \sin(\frac{X+Y}{2}) \cos(\frac{X-Y}{2})$.

$\sin 2A + \sin 2B = 2 \sin(A+B) \cos(A-B)$.

Use $A+B = \pi - C \implies \sin(A+B) = \sin(\pi - C) = \sin C$.

So, $\sin 2A + \sin 2B = 2 \sin C \cos(A-B)$.

LHS = $2 \sin C \cos(A-B) + \sin 2C$.

Use $\sin 2C = 2 \sin C \cos C$.

LHS = $2 \sin C \cos(A-B) + 2 \sin C \cos C$.

Factor out $2 \sin C$: LHS = $2 \sin C [\cos(A-B) + \cos C]$.

Use $C = \pi - (A+B) \implies \cos C = \cos(\pi - (A+B)) = -\cos(A+B)$.

LHS = $2 \sin C [\cos(A-B) - \cos(A+B)]$.

Use identity $\cos(X) - \cos(Y) = -2 \sin(\frac{X+Y}{2}) \sin(\frac{X-Y}{2})$.

Let $X=A-B$ and $Y=A+B$.

$\frac{X+Y}{2} = \frac{A-B+A+B}{2} = \frac{2A}{2} = A$.

$\frac{X-Y}{2} = \frac{A-B-(A+B)}{2} = \frac{-2B}{2} = -B$.

So, $\cos(A-B) - \cos(A+B) = -2 \sin(A) \sin(-B) = -2 \sin A (-\sin B) = 2 \sin A \sin B$.

Substitute back into LHS expression:

LHS = $2 \sin C [2 \sin A \sin B]$.

LHS = $4 \sin A \sin B \sin C$. (Proven)

(c)

(i) Express $5 \sin \theta - 12 \cos \theta$ as $R \sin(\theta - \alpha)$.

$R \sin(\theta - \alpha) = R(\sin \theta \cos \alpha - \cos \theta \sin \alpha) = (R \cos \alpha)\sin \theta - (R \sin \alpha)\cos \theta$.

Compare coefficients:

$R \cos \alpha = 5$.

$R \sin \alpha = 12$.

$R^2 = 5^2 + 12^2 = 25 + 144 = 169 \implies R = \sqrt{169} = 13$.

$\tan \alpha = \frac{R \sin \alpha}{R \cos \alpha} = \frac{12}{5} = 2.4$.

$\alpha = \arctan(2.4)$. Since $\sin \alpha$ and $\cos \alpha$ are positive, $\alpha$ is in the first quadrant.

$\alpha \approx 67.38^{\circ}$.

So, $5 \sin \theta - 12 \cos \theta = 13 \sin(\theta - 67.38^{\circ})$.

(ii) Solve $5 \sin \theta - 12 \cos \theta = 6$ for $0 \leq \theta \leq 360^{\circ}$.

Using part (i): $13 \sin(\theta - 67.38^{\circ}) = 6$.

$\sin(\theta - 67.38^{\circ}) = \frac{6}{13} \approx 0.4615$.

Let $\phi = \theta - 67.38^{\circ}$. Then $\sin \phi = 6/13$.

Principal value: $\phi_1 = \arcsin(6/13) \approx 27.49^{\circ}$.

Since $\sin \phi$ is positive, solutions are in 1st and 2nd quadrants.

1st Quad: $\phi_1 = 27.49^{\circ}$.

2nd Quad: $\phi_2 = 180^{\circ} - 27.49^{\circ} = 152.51^{\circ}$.

Find $\theta = \phi + 67.38^{\circ}$.

$\theta_1 = 27.49^{\circ} + 67.38^{\circ} = 94.87^{\circ}$.

$\theta_2 = 152.51^{\circ} + 67.38^{\circ} = 219.89^{\circ}$.

Consider the range for $\phi = \theta - 67.38^{\circ}$. If $0 \le \theta \le 360$, then $-67.38^{\circ} \le \phi \le 292.62^{\circ}$. Both $\phi_1$ and $\phi_2$ are within this adjusted range.

Solutions are $\theta \approx 94.9^{\circ}$ and $\theta \approx 219.9^{\circ}$.

Trigonometric identities are proven using fundamental definitions, sum-to-product rules, and angle properties. The equation is solved by converting to the R-formula form.

Question 3 (20 marks)

(a) Solve the equation $2 x^{2}-9 x+9=0$, by factorization. (4 marks)

(b) The roots of the equation $x^{2}+6 x+q=0$ are $\alpha$ and $\alpha-1$. Determine the value of $q$. (6 marks)

(c) The roots of the equation $x^{2}+7 x+3=0$ are $\alpha$ and $\beta$. Without solving the equation, form an equation whose roots are $\frac{1}{\alpha^{2}}$ and $\frac{1}{\beta^{2}}$. (10 marks)

Answer

This question deals with solving quadratic equations and manipulating the relationships between roots and coefficients (Vieta's formulas).

(a) Solve $2 x^{2}-9 x+9=0$ by factorization.

We need two numbers that multiply to $2 \times 9 = 18$ and add up to $-9$. These numbers are $-6$ and $-3$.

Rewrite the middle term: $2x^2 - 6x - 3x + 9 = 0$.

Factor by grouping: $2x(x - 3) - 3(x - 3) = 0$.

$(2x - 3)(x - 3) = 0$.

Solutions are $2x - 3 = 0 \implies x = 3/2$ or $x - 3 = 0 \implies x = 3$.

(b) Find $q$ given roots $\alpha$ and $\alpha-1$ for $x^{2}+6 x+q=0$.

From Vieta's formulas for $ax^2+bx+c=0$ (here $a=1, b=6, c=q$):

Sum of roots = $-\frac{b}{a} = -\frac{6}{1} = -6$.

Product of roots = $\frac{c}{a} = \frac{q}{1} = q$.

Sum of roots = $\alpha + (\alpha-1) = 2\alpha - 1$.

Equating the sums: $2\alpha - 1 = -6 \implies 2\alpha = -5 \implies \alpha = -5/2$.

Product of roots = $\alpha (\alpha-1)$.

Equating the products: $q = \alpha (\alpha-1)$.

Substitute the value of $\alpha$: $q = (-\frac{5}{2})(-\frac{5}{2} - 1) = (-\frac{5}{2})(-\frac{5}{2} - \frac{2}{2}) = (-\frac{5}{2})(-\frac{7}{2})$.

$q = \frac{35}{4}$.

(c) Form equation with roots $\frac{1}{\alpha^{2}}$ and $\frac{1}{\beta^{2}}$ given $x^{2}+7 x+3=0$ has roots $\alpha, \beta$.

From the given equation ($a=1, b=7, c=3$):

Sum of roots: $\alpha + \beta = -b/a = -7$.

Product of roots: $\alpha \beta = c/a = 3$.

For the new equation with roots $Y_1 = 1/\alpha^2$ and $Y_2 = 1/\beta^2$, we need the new sum ($S_{new}$) and new product ($P_{new}$).

New Sum: $S_{new} = \frac{1}{\alpha^2} + \frac{1}{\beta^2} = \frac{\beta^2 + \alpha^2}{\alpha^2 \beta^2} = \frac{\alpha^2 + \beta^2}{(\alpha \beta)^2}$.

We need $\alpha^2 + \beta^2$. Use the identity $\alpha^2 + \beta^2 = (\alpha + \beta)^2 - 2\alpha\beta$.

$\alpha^2 + \beta^2 = (-7)^2 - 2(3) = 49 - 6 = 43$.

$S_{new} = \frac{43}{(3)^2} = \frac{43}{9}$.

New Product: $P_{new} = \left(\frac{1}{\alpha^2}\right) \left(\frac{1}{\beta^2}\right) = \frac{1}{\alpha^2 \beta^2} = \frac{1}{(\alpha \beta)^2}$.

$P_{new} = \frac{1}{(3)^2} = \frac{1}{9}$.

The new quadratic equation is of the form $x^2 - (S_{new})x + P_{new} = 0$.

$x^2 - \left(\frac{43}{9}\right)x + \frac{1}{9} = 0$.

Multiply by 9 to clear fractions (optional, but common):

$9x^2 - 43x + 1 = 0$.

Questions involve standard quadratic equation techniques and application of Vieta's formulas to relate roots and coefficients.

Question 4 (20 marks)

(a) Find the middle term in the binomial expansion of $(2 x+3)^{8}$, and determine its value when $x=\frac{1}{12}$. (6 marks)

(b) Expand $(1-3 x)^{-\frac{1}{2}}$ as far as the term in $x^{3}$ and determine the range of values of $x$ for which the expansion is valid. (7 marks)

(c) (i) If $x$ is so small that its fourth and higher powers may be neglected, show that $\sqrt[4]{(1+x)}+\sqrt[4]{(1-x)}=a-b x^{2}$, and determine the values of $a$ and $b$; (5 marks)

(ii) Hence, by putting $x=\frac{1}{16}$ in the result in (c) (i) above, prove that $17^{\frac{1}{4}}+15^{\frac{1}{4}}=3.9985$ approximately. (Mistake in source, should be $17^{1/4}+15^{1/4}$?) (2 marks)

Answer

This question tests the binomial theorem for positive integer and fractional/negative powers, including finding specific terms and using expansions for approximations.

(a) Middle term of $(2x+3)^8$.

The expansion has $n+1 = 8+1 = 9$ terms. The middle term is the $(n/2 + 1)$th term, which is the $(8/2 + 1) = 5$th term.

The general term $T_{r+1} = \binom{n}{r} a^{n-r} b^r$. Here $a=2x, b=3, n=8$.

For the 5th term, $r+1=5 \implies r=4$.

$T_5 = \binom{8}{4} (2x)^{8-4} (3)^4$.

$T_5 = \binom{8}{4} (2x)^4 (3)^4$.

$\binom{8}{4} = \frac{8!}{4!4!} = \frac{8 \times 7 \times 6 \times 5}{4 \times 3 \times 2 \times 1} = 70$.

$T_5 = 70 (16x^4) (81) = 70 \times 16 \times 81 x^4 = 1120 \times 81 x^4 = 90720 x^4$.

The middle term is $90720 x^4$.

Value when $x = 1/12$:

$T_5 = 90720 \left(\frac{1}{12}\right)^4 = 90720 \left(\frac{1}{20736}\right)$.

$T_5 = \frac{90720}{20736}$. Simplify (both divisible by 5184):

$90720 / 5184 = 17.5$

$20736 / 5184 = 4$

$T_5 = 17.5 / 4 = 35 / 8 = 4.375$.

Check calculation: $90720 / 20736 = 4.375$.

Value is $35/8$ or $4.375$.

(b) Expand $(1-3 x)^{-\frac{1}{2}}$ up to $x^3$.

Use $(1+u)^k = 1 + ku + \frac{k(k-1)}{2!}u^2 + \frac{k(k-1)(k-2)}{3!}u^3 + \dots$

Here $u=-3x$ and $k=-1/2$.

$(1-3x)^{-1/2} \approx 1 + (-\frac{1}{2})(-3x) + \frac{-\frac{1}{2}(-\frac{1}{2}-1)}{2}(-3x)^2 + \frac{-\frac{1}{2}(-\frac{1}{2}-1)(-\frac{1}{2}-2)}{6}(-3x)^3$.

$\approx 1 + \frac{3}{2}x + \frac{-\frac{1}{2}(-\frac{3}{2})}{2}(9x^2) + \frac{-\frac{1}{2}(-\frac{3}{2})(-\frac{5}{2})}{6}(-27x^3)$.

$\approx 1 + \frac{3}{2}x + \frac{3/4}{2}(9x^2) + \frac{-15/8}{6}(-27x^3)$.

$\approx 1 + \frac{3}{2}x + \frac{3}{8}(9x^2) + (\frac{-15}{48})(-27x^3)$.

$\approx 1 + \frac{3}{2}x + \frac{27}{8}x^2 + \frac{15 \times 27}{48}x^3$.

Simplify last coefficient: $\frac{15 \times 27}{48} = \frac{5 \times 27}{16} = \frac{135}{16}$.

$(1-3x)^{-1/2} \approx 1 + \frac{3}{2}x + \frac{27}{8}x^2 + \frac{135}{16}x^3$.

Range of validity: Expansion is valid for $|u| < 1$, i.e., $|-3x| < 1$.

$3|x| < 1 \implies |x| < 1/3$.

Valid for $-1/3 < x < 1/3$.

(c)

(i) Show $\sqrt[4]{(1+x)}+\sqrt[4]{(1-x)}=a-b x^{2}$ for small $x$.

Use $(1+u)^k \approx 1 + ku + \frac{k(k-1)}{2}u^2$. Here $k=1/4$.

$(1+x)^{1/4} \approx 1 + \frac{1}{4}x + \frac{\frac{1}{4}(\frac{1}{4}-1)}{2}x^2 = 1 + \frac{1}{4}x + \frac{\frac{1}{4}(-\frac{3}{4})}{2}x^2 = 1 + \frac{1}{4}x - \frac{3}{32}x^2$.

$(1-x)^{1/4} \approx 1 + \frac{1}{4}(-x) + \frac{\frac{1}{4}(-\frac{3}{4})}{2}(-x)^2 = 1 - \frac{1}{4}x - \frac{3}{32}x^2$.

Add the two expansions:

$(1 + \frac{1}{4}x - \frac{3}{32}x^2) + (1 - \frac{1}{4}x - \frac{3}{32}x^2)$

$= (1+1) + (\frac{1}{4}x - \frac{1}{4}x) + (-\frac{3}{32}x^2 - \frac{3}{32}x^2)$.

$= 2 - \frac{6}{32}x^2 = 2 - \frac{3}{16}x^2$.

Comparing with $a - bx^2$, we have $a=2$ and $b=3/16$.

(ii) Use $x=1/16$ to approximate $17^{\frac{1}{4}}+15^{\frac{1}{4}}=3.9985$.

The expression looks like $16^{1/4}(1+x)^{1/4} + 16^{1/4}(1-y)^{1/4}$. Let's match the form from (i).

The expression is $\sqrt[4]{1+x} + \sqrt[4]{1-x}$. If we use $x=1/16$, we get:

$\sqrt[4]{1+1/16} + \sqrt[4]{1-1/16} = \sqrt[4]{17/16} + \sqrt[4]{15/16}$

$= \frac{\sqrt[4]{17}}{\sqrt[4]{16}} + \frac{\sqrt[4]{15}}{\sqrt[4]{16}} = \frac{17^{1/4} + 15^{1/4}}{2}$.

Use the approximation from (c)(i) with $x=1/16$:

$\frac{17^{1/4} + 15^{1/4}}{2} \approx 2 - \frac{3}{16}x^2 = 2 - \frac{3}{16}\left(\frac{1}{16}\right)^2 = 2 - \frac{3}{16(256)} = 2 - \frac{3}{4096}$.

$2 - \frac{3}{4096} = \frac{2 \times 4096 - 3}{4096} = \frac{8192 - 3}{4096} = \frac{8189}{4096}$.

So, $17^{1/4} + 15^{1/4} \approx 2 \times \frac{8189}{4096} = \frac{8189}{2048}$.

Calculate the decimal value: $8189 / 2048 \approx 3.998535...$

Rounding to 4 decimal places gives $3.9985$. (Proven)

Binomial theorem applied correctly for integer and fractional powers, including calculation of specific terms and approximations.

Question 5 (20 marks)

(a) Simplify the expressions:

(i) $\frac{(1-x)^{\frac{1}{2}}-(1-x)^{-\frac{1}{2}}}{(1+x)^{\frac{1}{2}}}$ (Likely typo, denominator should be related) (4 marks)

(ii) $\frac{\log 125-\log 25+\log 5}{\log 625+\frac{1}{2} \log 25}$ (Assuming base 10 or any consistent base) (4 marks)

(b) Solve the equation:

$4^{x}+1=3+2^{x}$ (6 marks)

(c) The application of Kirchoff's laws to a d.c. circuit yielded the simultaneous equations:

$I_{1}-2 I_{2}+I_{3}=0$

$-2 I_{1}+3 I_{2}+2 I_{3}=2$

$3 I_{1}+4 I_{2}-3 I_{3}=14$

Use elimination method to solve the equations. (6 marks)

Answer

This question involves simplifying expressions with fractional indices and logarithms, solving an exponential equation reducible to quadratic, and solving a system of linear equations using elimination.

(a) Simplify expressions.

(i) Simplify $\frac{(1-x)^{\frac{1}{2}}-(1-x)^{-\frac{1}{2}}}{(1+x)^{\frac{1}{2}}}$.

The denominator $(1+x)^{1/2}$ seems unrelated to the numerator terms involving $(1-x)$. Assuming a typo and the denominator should be $(1-x)$, let's simplify the numerator first.

Numerator = $(1-x)^{1/2} - \frac{1}{(1-x)^{1/2}} = \frac{(1-x) - 1}{(1-x)^{1/2}} = \frac{-x}{(1-x)^{1/2}}$.

If the denominator was $(1-x)$, the expression would be $\frac{-x / (1-x)^{1/2}}{1-x} = \frac{-x}{(1-x)^{3/2}}$.

If the denominator was $(1-x)^{1/2}$, the expression would be $\frac{-x / (1-x)^{1/2}}{(1-x)^{1/2}} = \frac{-x}{1-x}$.

If the expression is exactly as written, it cannot be simplified further algebraically in a significant way without more context.

(ii) Simplify $\frac{\log 125-\log 25+\log 5}{\log 625+\frac{1}{2} \log 25}$.

Express numbers as powers of 5:

Numerator: $\log 5^3 - \log 5^2 + \log 5^1 = (3 - 2 + 1)\log 5 = 2 \log 5$.

Denominator: $\log 5^4 + \frac{1}{2} \log 5^2 = 4 \log 5 + \frac{1}{2}(2 \log 5) = 4 \log 5 + 1 \log 5 = 5 \log 5$.

Fraction: $\frac{2 \log 5}{5 \log 5} = \frac{2}{5}$ (assuming $\log 5 \neq 0$).

(b) Solve $4^{x}+1=3+2^{x}$.

Rewrite $4^x$ as $(2^2)^x = (2^x)^2$.

$(2^x)^2 + 1 = 3 + 2^x$.

Let $y = 2^x$. The equation becomes $y^2 + 1 = 3 + y$.

Rearrange into quadratic form: $y^2 - y - 2 = 0$.

Factor the quadratic: $(y-2)(y+1) = 0$.

Solutions for y are $y=2$ or $y=-1$.

Substitute back $y=2^x$.

Case 1: $2^x = 2$. Since $2^1=2$, $x=1$.

Case 2: $2^x = -1$. This has no real solution, as $2^x$ is always positive.

The only real solution is $x=1$.

(c) Solve the system using elimination.

(1) $I_{1}-2 I_{2}+I_{3}=0$

(2) $-2 I_{1}+3 I_{2}+2 I_{3}=2$

(3) $3 I_{1}+4 I_{2}-3 I_{3}=14$

Eliminate $I_1$.

2*(1) + (2): $2(I_1-2I_2+I_3) + (-2I_1+3I_2+2I_3) = 2(0) + 2$

$(2I_1-4I_2+2I_3) + (-2I_1+3I_2+2I_3) = 2$

$-I_2 + 4I_3 = 2 \quad (4)$

3*(1) - (3): $3(I_1-2I_2+I_3) - (3I_1+4I_2-3I_3) = 3(0) - 14$

$(3I_1-6I_2+3I_3) - (3I_1+4I_2-3I_3) = -14$

$-10I_2 + 6I_3 = -14$. Divide by -2: $5I_2 - 3I_3 = 7 \quad (5)$.

Now solve the system in $I_2, I_3$:

(4) $-I_2 + 4I_3 = 2$

(5) $5I_2 - 3I_3 = 7$

Multiply (4) by 5: $-5I_2 + 20I_3 = 10 \quad (4')$

Add (4') and (5): $(-5I_2 + 20I_3) + (5I_2 - 3I_3) = 10 + 7$.

$17I_3 = 17 \implies I_3 = 1$.

Substitute $I_3=1$ into (4): $-I_2 + 4(1) = 2 \implies -I_2 = -2 \implies I_2 = 2$.

Substitute $I_2=2$ and $I_3=1$ into (1): $I_1 - 2(2) + 1 = 0 \implies I_1 - 4 + 1 = 0 \implies I_1 = 3$.

Solution: $I_1 = 3$ A, $I_2 = 2$ A, $I_3 = 1$ A.

Logarithm simplification, solving equations reducible to quadratic, and systematic elimination for linear systems are key skills tested.

Question 6 (20 marks)

(a) Find the coefficient of $x^{6}$ in the binomial expansion of $(3 x+2 y)^{10}$, and determine its value when $x=\frac{1}{2}$ and $y=\frac{1}{3}$. (6 marks)

(b) (i) Determine the first four terms in the binomial expansion of $(3+4 x)^{-\frac{1}{2}}$, and state the values of $x$ for which the expansion is valid. (6 marks)

(ii) Use the binomial theorem to expand $\left(1+\frac{1}{4} x\right)^{\frac{1}{3}}$ as far as the term in $x^{3}$. Hence determine the value of $\sqrt[3]{65}$, correct to four decimal places. (5 marks)

(c) Solve the equation:

$3^{2 x+1}-7\left(3^{x}\right)+2=0$. (3 marks)

Answer

This question focuses on the binomial theorem (finding specific terms, expanding fractional/negative powers, approximation) and solving exponential equations.

(a) Coefficient of $x^6$ in $(3x+2y)^{10}$.

General term $T_{r+1} = \binom{n}{r} a^{n-r} b^r$. Here $a=3x, b=2y, n=10$.

$T_{r+1} = \binom{10}{r} (3x)^{10-r} (2y)^r = \binom{10}{r} 3^{10-r} x^{10-r} 2^r y^r$.

We want the term with $x^6$. So, $10-r = 6 \implies r=4$.

The term is $T_{4+1} = T_5 = \binom{10}{4} 3^{10-4} x^6 2^4 y^4$.

$T_5 = \binom{10}{4} 3^6 x^6 2^4 y^4 = \binom{10}{4} (729)(16) x^6 y^4$.

$\binom{10}{4} = \frac{10!}{4!6!} = \frac{10 \times 9 \times 8 \times 7}{4 \times 3 \times 2 \times 1} = 10 \times 3 \times 7 = 210$.

$T_5 = 210 (729)(16) x^6 y^4 = 210 (11664) x^6 y^4 = 2449440 x^6 y^4$.

The coefficient of $x^6$ is $2449440 y^4$.

Value of the term when $x=1/2, y=1/3$:

$2449440 (1/2)^6 (1/3)^4 = 2449440 \left(\frac{1}{64}\right) \left(\frac{1}{81}\right)$.

$= \frac{2449440}{64 \times 81} = \frac{2449440}{5184} = 472.5$.

(b)

(i) First four terms of $(3+4x)^{-1/2}$.

Factor out 3: $(3(1 + \frac{4}{3}x))^{-1/2} = 3^{-1/2} (1 + \frac{4}{3}x)^{-1/2} = \frac{1}{\sqrt{3}} (1 + \frac{4}{3}x)^{-1/2}$.

Use $(1+u)^k \approx 1 + ku + \frac{k(k-1)}{2!}u^2 + \frac{k(k-1)(k-2)}{3!}u^3$.

Here $u=4x/3, k=-1/2$.

Expansion $\approx \frac{1}{\sqrt{3}} \left[ 1 + (-\frac{1}{2})(\frac{4x}{3}) + \frac{-\frac{1}{2}(-\frac{3}{2})}{2}(\frac{4x}{3})^2 + \frac{-\frac{1}{2}(-\frac{3}{2})(-\frac{5}{2})}{6}(\frac{4x}{3})^3 \right]$.

$\approx \frac{1}{\sqrt{3}} \left[ 1 - \frac{2x}{3} + \frac{3/4}{2}(\frac{16x^2}{9}) + \frac{-15/8}{6}(\frac{64x^3}{27}) \right]$.

$\approx \frac{1}{\sqrt{3}} \left[ 1 - \frac{2x}{3} + \frac{3}{8}(\frac{16x^2}{9}) - \frac{15}{48}(\frac{64x^3}{27}) \right]$.

$\approx \frac{1}{\sqrt{3}} \left[ 1 - \frac{2x}{3} + \frac{2x^2}{3} - \frac{5}{16}(\frac{64x^3}{27}) \right]$.

$\approx \frac{1}{\sqrt{3}} \left[ 1 - \frac{2x}{3} + \frac{2x^2}{3} - \frac{20x^3}{27} \right]$.

$\approx \frac{\sqrt{3}}{3} \left( 1 - \frac{2x}{3} + \frac{2x^2}{3} - \frac{20x^3}{27} \right)$.

Range of validity: $|u| < 1 \implies |\frac{4x}{3}| < 1 \implies |x| < 3/4$. Valid for $-3/4 < x < 3/4$.

(ii) Expand $(1+\frac{1}{4}x)^{1/3}$ up to $x^3$ and evaluate $\sqrt[3]{65}$.

$u=x/4, k=1/3$.

$(1+\frac{x}{4})^{1/3} \approx 1 + \frac{1}{3}(\frac{x}{4}) + \frac{\frac{1}{3}(\frac{1}{3}-1)}{2}(\frac{x}{4})^2 + \frac{\frac{1}{3}(\frac{1}{3}-1)(\frac{1}{3}-2)}{6}(\frac{x}{4})^3$.

$\approx 1 + \frac{x}{12} + \frac{\frac{1}{3}(-\frac{2}{3})}{2}(\frac{x^2}{16}) + \frac{\frac{1}{3}(-\frac{2}{3})(-\frac{5}{3})}{6}(\frac{x^3}{64})$.

$\approx 1 + \frac{x}{12} + \frac{-2/9}{2}(\frac{x^2}{16}) + \frac{10/27}{6}(\frac{x^3}{64})$.

$\approx 1 + \frac{x}{12} - \frac{1}{9}(\frac{x^2}{16}) + \frac{10}{162}(\frac{x^3}{64})$.

$\approx 1 + \frac{x}{12} - \frac{x^2}{144} + \frac{5}{81 \times 64}x^3 = 1 + \frac{x}{12} - \frac{x^2}{144} + \frac{5x^3}{5184}$.

Evaluate $\sqrt[3]{65} = \sqrt[3]{64+1} = \sqrt[3]{64(1 + 1/64)} = 4 (1 + 1/64)^{1/3}$.

This matches the form $(1+x/4)^{1/3}$ if $x/4 = 1/64$, which means $x = 4/64 = 1/16$.

Substitute $x=1/16$ into the expansion:

$(1+1/64)^{1/3} \approx 1 + \frac{(1/16)}{12} - \frac{(1/16)^2}{144} + \frac{5(1/16)^3}{5184}$.

$\approx 1 + \frac{1}{192} - \frac{1/256}{144} + \frac{5/4096}{5184}$.

$\approx 1 + 0.00520833 - \frac{1}{36864} + \frac{5}{21233664}$.

$\approx 1 + 0.00520833 - 0.00002713 + 0.00000024$.

$\approx 1.00518144$.

$\sqrt[3]{65} = 4 (1 + 1/64)^{1/3} \approx 4 \times 1.00518144 \approx 4.02072576$.

Correct to four decimal places: $4.0207$.

(c) Solve $3^{2 x+1}-7\left(3^{x}\right)+2=0$.

Rewrite: $3^1 \cdot 3^{2x} - 7(3^x) + 2 = 0 \implies 3(3^x)^2 - 7(3^x) + 2 = 0$.

Let $y = 3^x$. Equation is $3y^2 - 7y + 2 = 0$.

Factor: $(3y - 1)(y - 2) = 0$.

Solutions for y are $y=1/3$ or $y=2$.

Case 1: $3^x = 1/3 = 3^{-1}$. So $x=-1$.

Case 2: $3^x = 2$. Take logarithms: $x \ln 3 = \ln 2 \implies x = \frac{\ln 2}{\ln 3}$.

$x \approx \frac{0.6931}{1.0986} \approx 0.6309$.

Solutions are $x=-1$ and $x = \frac{\ln 2}{\ln 3} \approx 0.631$.

Binomial theorem applications and solving exponential equations require careful algebraic manipulation and understanding of the underlying principles.

Question 7 (20 marks)

(a) Given the complex numbers $z_{1}=2+3 j, z_{2}=1+2 j$ and $z_{3}=3-4 j$, express $z_{1}+\frac{z_{2} z_{3}}{z_{2}+z_{3}}$ in polar form. (10 marks)

(b) One root of the equation $z^{3}+4 z^{2}+k z+8=0$ is $-1+j \sqrt{3}$. Determine the:

(i) value of $k$; (5 marks)

(ii) other roots. (5 marks)

(c) Solve the equation $z^{3}-1+j \sqrt{3}=0$ (Structure unclear, likely find roots of $z^3 = 1-j\sqrt{3}$).

Answer

This question involves complex number arithmetic, properties of polynomials with real coefficients (conjugate roots), finding unknown coefficients, and finding roots of complex numbers.

(a) Express $Z = z_{1}+\frac{z_{2} z_{3}}{z_{2}+z_{3}}$ in polar form.

First, calculate the components:

$z_2 z_3 = (1+2j)(3-4j) = 1(3) + 1(-4j) + 2j(3) + 2j(-4j) = 3 - 4j + 6j - 8j^2 = 3 + 2j + 8 = 11 + 2j$.

$z_2 + z_3 = (1+2j) + (3-4j) = (1+3) + (2-4)j = 4 - 2j$.

Calculate the fraction: $\frac{z_2 z_3}{z_2 + z_3} = \frac{11+2j}{4-2j}$.

Rationalize: $\frac{(11+2j)(4+2j)}{(4-2j)(4+2j)} = \frac{11(4)+11(2j)+2j(4)+2j(2j)}{4^2 - (2j)^2}$.

$= \frac{44 + 22j + 8j + 4j^2}{16 - (-4)} = \frac{44 + 30j - 4}{16 + 4} = \frac{40 + 30j}{20} = 2 + 1.5j$.

Now calculate $Z = z_1 + (2+1.5j) = (2+3j) + (2+1.5j) = (2+2) + (3+1.5)j = 4 + 4.5j$.

Convert $Z = 4 + 4.5j$ to polar form $r(\cos \theta + j \sin \theta)$.

$r = |Z| = \sqrt{4^2 + (4.5)^2} = \sqrt{16 + 20.25} = \sqrt{36.25}$.

$r \approx 6.0208$.

$\theta = \arg(Z)$. Since real and imaginary parts are positive, $\theta$ is in the 1st quadrant.

$\tan \theta = \frac{4.5}{4} = 1.125$.

$\theta = \arctan(1.125) \approx 48.366^{\circ}$.

Polar form: $Z \approx 6.02 (\cos 48.37^{\circ} + j \sin 48.37^{\circ})$.

(b) Equation $z^{3}+4 z^{2}+k z+8=0$. Root $z_1 = -1+j \sqrt{3}$.

(i) Find $k$.

Since the coefficients (1, 4, k, 8) are real, complex roots must occur in conjugate pairs. Therefore, $z_2 = -1 - j\sqrt{3}$ must also be a root.

The quadratic factor corresponding to these two roots is $(z - z_1)(z - z_2)$.

$= (z - (-1+j\sqrt{3}))(z - (-1-j\sqrt{3}))$

$= ((z+1) - j\sqrt{3})((z+1) + j\sqrt{3})$

$= (z+1)^2 - (j\sqrt{3})^2 = (z^2+2z+1) - (j^2 \cdot 3) = z^2+2z+1 - (-3) = z^2+2z+4$.

Since $z^2+2z+4$ is a factor of $z^3+4z^2+kz+8$, we can use polynomial division or sum/product of roots.

Let the third root be $z_3$.

Sum of roots: $z_1 + z_2 + z_3 = (-1+j\sqrt{3}) + (-1-j\sqrt{3}) + z_3 = -2 + z_3$.

From the polynomial, the sum of roots is $-(\text{coeff of } z^2) / (\text{coeff of } z^3) = -4/1 = -4$.

So, $-2 + z_3 = -4 \implies z_3 = -2$. The third root is real.

Now consider the pairwise products sum or the product of all roots.

Product of roots: $z_1 z_2 z_3 = -(\text{constant term}) / (\text{coeff of } z^3) = -8/1 = -8$.

$z_1 z_2 = (-1+j\sqrt{3})(-1-j\sqrt{3}) = (-1)^2 - (j\sqrt{3})^2 = 1 - (-3) = 4$.

Product check: $(z_1 z_2) z_3 = (4)(-2) = -8$. This matches.

Sum of pairwise products: $z_1 z_2 + z_1 z_3 + z_2 z_3 = (\text{coeff of } z) / (\text{coeff of } z^3) = k/1 = k$.

$k = z_1 z_2 + z_3 (z_1 + z_2)$.

$k = 4 + (-2)((-1+j\sqrt{3}) + (-1-j\sqrt{3})) = 4 + (-2)(-2) = 4 + 4 = 8$.

The value of $k$ is 8.

(ii) Other roots.

We already found them: $z_2 = -1 - j\sqrt{3}$ (conjugate) and $z_3 = -2$ (real root).

(c) Solve $z^{3}-1+j \sqrt{3}=0$.

This means $z^3 = 1 - j\sqrt{3}$. Find the cube roots of $w = 1 - j\sqrt{3}$.

Convert $w$ to polar form $r(\cos \phi + j \sin \phi)$.

$r = |w| = \sqrt{1^2 + (-\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2$.

$\phi = \arg(w)$. Real part positive, imaginary part negative $\implies$ 4th quadrant.

Reference angle $\alpha = \arctan \left|\frac{-\sqrt{3}}{1}\right| = \arctan(\sqrt{3}) = \pi/3$.

Angle $\phi = -\pi/3$ (or $5\pi/3$). Use $\phi = -\pi/3 + 2k\pi$.

$w = 2(\cos(-\pi/3) + j \sin(-\pi/3))$.

Let $z = R(\cos \Theta + j \sin \Theta)$. Then $z^3 = R^3(\cos 3\Theta + j \sin 3\Theta)$.

$R^3 = 2 \implies R = \sqrt[3]{2}$.

$3\Theta = -\pi/3 + 2k\pi \implies \Theta = -\pi/9 + 2k\pi/3$.

Find roots for $k=0, 1, 2$.

$k=0: \Theta_0 = -\pi/9$. $z_0 = \sqrt[3]{2}(\cos(-\pi/9) + j \sin(-\pi/9))$.

$k=1: \Theta_1 = -\pi/9 + 2\pi/3 = -\pi/9 + 6\pi/9 = 5\pi/9$. $z_1 = \sqrt[3]{2}(\cos(5\pi/9) + j \sin(5\pi/9))$.

$k=2: \Theta_2 = -\pi/9 + 4\pi/3 = -\pi/9 + 12\pi/9 = 11\pi/9$. (Equivalent to $11\pi/9 - 2\pi = -7\pi/9$). $z_2 = \sqrt[3]{2}(\cos(11\pi/9) + j \sin(11\pi/9))$.

The roots are $z_k = \sqrt[3]{2} \left( \cos\left(-\frac{\pi}{9} + \frac{2k\pi}{3}\right) + j \sin\left(-\frac{\pi}{9} + \frac{2k\pi}{3}\right) \right)$ for $k=0, 1, 2$.

Complex number arithmetic, polynomial root properties, and De Moivre's theorem for finding roots are applied.

Question 8 (20 marks)

(a) Given $y=\frac{x}{1-x}$, find $\frac{d y}{d x}$ from first principles. (6 marks)

(b) Use implicit differentiation to determine the equations of the:

(i) tangent;

(ii) normal

to the curve $x^{3}+y^{2}+3 x y-2 x+6 y+9=0$ at the point $(1,-1)$. (8 marks)

(c) Determine the stationary points of the curve $f(x)=x^{3}+15 x^{2}+27 x+2$, and state their nature. (6 marks)

Answer

This question covers differentiation from first principles, implicit differentiation to find tangent and normal lines, and finding and classifying stationary points of a polynomial function.

(a) Find $\frac{dy}{dx}$ for $y = \frac{x}{1-x}$ from first principles.

$\frac{dy}{dx} = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$.

$f(x) = \frac{x}{1-x}$.

$f(x+h) = \frac{x+h}{1-(x+h)} = \frac{x+h}{1-x-h}$.

$f(x+h) - f(x) = \frac{x+h}{1-x-h} - \frac{x}{1-x}$.

Combine fractions: $\frac{(x+h)(1-x) - x(1-x-h)}{(1-x-h)(1-x)}$.

Numerator: $(x - x^2 + h - hx) - (x - x^2 - xh) = x - x^2 + h - hx - x + x^2 + xh = h$.

So, $f(x+h) - f(x) = \frac{h}{(1-x-h)(1-x)}$.

$\frac{f(x+h) - f(x)}{h} = \frac{h / [(1-x-h)(1-x)]}{h} = \frac{1}{(1-x-h)(1-x)}$.

Take the limit as $h \to 0$:

$\frac{dy}{dx} = \lim_{h \to 0} \frac{1}{(1-x-h)(1-x)} = \frac{1}{(1-x-0)(1-x)} = \frac{1}{(1-x)^2}$.

(b) Tangent and normal to $x^{3}+y^{2}+3 x y-2 x+6 y+9=0$ at $(1,-1)$.

Differentiate implicitly w.r.t. $x$:

$3x^2 + 2y \frac{dy}{dx} + (3y + 3x \frac{dy}{dx}) - 2 + 6 \frac{dy}{dx} + 0 = 0$.

Group terms with $\frac{dy}{dx}$: $(2y + 3x + 6)\frac{dy}{dx} = 2 - 3x^2 - 3y$.

$\frac{dy}{dx} = \frac{2 - 3x^2 - 3y}{2y + 3x + 6}$.

Evaluate the slope $m_t$ at $(1,-1)$:

$m_t = \frac{2 - 3(1)^2 - 3(-1)}{2(-1) + 3(1) + 6} = \frac{2 - 3 + 3}{-2 + 3 + 6} = \frac{2}{7}$.

(i) Equation of the tangent:

$y - y_1 = m_t(x - x_1)$.

$y - (-1) = \frac{2}{7}(x - 1)$.

$y + 1 = \frac{2}{7}(x - 1)$. Multiply by 7:

$7(y+1) = 2(x-1) \implies 7y + 7 = 2x - 2$.

$2x - 7y - 9 = 0$.

(ii) Equation of the normal:

Slope of normal $m_n = -1/m_t = -1/(2/7) = -7/2$.

$y - y_1 = m_n(x - x_1)$.

$y - (-1) = -\frac{7}{2}(x - 1)$.

$y + 1 = -\frac{7}{2}(x - 1)$. Multiply by 2:

$2(y+1) = -7(x-1) \implies 2y + 2 = -7x + 7$.

$7x + 2y - 5 = 0$.

(c) Stationary points of $f(x)=x^{3}+15 x^{2}+27 x+2$.

Find the first derivative: $f'(x) = 3x^2 + 30x + 27$.

Set $f'(x) = 0$: $3x^2 + 30x + 27 = 0$.

Divide by 3: $x^2 + 10x + 9 = 0$.

Factor: $(x+9)(x+1) = 0$.

Stationary points occur at $x=-9$ and $x=-1$.

Find the corresponding y-values:

$f(-9) = (-9)^3 + 15(-9)^2 + 27(-9) + 2 = -729 + 15(81) - 243 + 2 = -729 + 1215 - 243 + 2 = 245$. Point $(-9, 245)$.

$f(-1) = (-1)^3 + 15(-1)^2 + 27(-1) + 2 = -1 + 15(1) - 27 + 2 = -1 + 15 - 27 + 2 = -11$. Point $(-1, -11)$.

Find the second derivative: $f''(x) = 6x + 30$.

Classify the points:

At $x=-9$: $f''(-9) = 6(-9) + 30 = -54 + 30 = -24$. Since $f'' < 0$, this is a local maximum.

At $x=-1$: $f''(-1) = 6(-1) + 30 = -6 + 30 = 24$. Since $f'' > 0$, this is a local minimum.

Stationary points: Local maximum at $(-9, 245)$ and local minimum at $(-1, -11)$.

Differentiation from first principles, implicit differentiation for tangent/normal, and finding/classifying stationary points are standard calculus techniques.
