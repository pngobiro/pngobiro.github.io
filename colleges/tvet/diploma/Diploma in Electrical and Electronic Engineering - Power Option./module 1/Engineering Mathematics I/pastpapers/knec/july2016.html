Question 1 (20 marks)

(a) Solve the equations:

(i) $5^{6 x+1} \times 25^{x-7}=125$; (4 marks)

(ii) $\quad \log _{3}(x+2)^{2}=2$. (4 marks)

(b) Convert:

(i) $\mathrm{r}=4(1+2 \sin 2 \theta)$ to cartesian form. (Typo? Usually simpler forms) (4 marks)

(ii) $\mathrm{xy}=3$ to polar form. (2 marks)

(c) Three currents $\mathrm{I}_{1}, \mathrm{I}_{2}$ and $\mathrm{I}_{3}$ in amperes flowing in an electric circuit satisfy the following simultaneous equations:

$3 \mathrm{I}_{1}+2 \mathrm{I}_{2}+5 \mathrm{I}_{3}=2$

$3 \mathrm{I}_{1}+3 \mathrm{I}_{2}-2 \mathrm{I}_{3}=4$

$2 \mathrm{I}_{1}-5 \mathrm{I}_{2}-3 \mathrm{I}_{3}=14$

Use elimination method to determine the values of the three currents. (6 marks)

Answer

This question covers solving exponential and logarithmic equations, converting between polar and Cartesian coordinates, and solving a system of linear equations.

(a) Solve equations.

(i) Solve $5^{6 x+1} \times 25^{x-7}=125$.

Express all terms with base 5:

$25 = 5^2$, $125 = 5^3$.

$5^{6x+1} \times (5^2)^{x-7} = 5^3$.

$5^{6x+1} \times 5^{2(x-7)} = 5^3$.

$5^{6x+1} \times 5^{2x-14} = 5^3$.

Use the index law $a^m \times a^n = a^{m+n}$:

$5^{(6x+1) + (2x-14)} = 5^3$.

$5^{8x - 13} = 5^3$.

Equate the exponents: $8x - 13 = 3$.

$8x = 16$.

$x = 2$.

(ii) Solve $\log _{3}(x+2)^{2}=2$.

Convert the logarithmic equation to exponential form:

$(x+2)^2 = 3^2 = 9$.

Take the square root of both sides:

$x+2 = \pm \sqrt{9} = \pm 3$.

Case 1: $x+2 = 3 \implies x = 1$.

Case 2: $x+2 = -3 \implies x = -5$.

Check domain: The argument of the logarithm $(x+2)^2$ must be positive. $(1+2)^2 = 9 > 0$. $(-5+2)^2 = (-3)^2 = 9 > 0$. Both solutions are valid.

Solutions: $x=1, x=-5$.

(b) Convert coordinate forms.

(i) Convert $r=4(1+2 \sin 2 \theta)$ to Cartesian form.

This conversion is unusually complex and might involve terms like $(x^2+y^2)^3$. Let's proceed assuming the equation is correct.

Use $r^2 = x^2+y^2$, $x = r\cos\theta$, $y=r\sin\theta$, $\sin 2\theta = 2\sin\theta\cos\theta$.

$r = 4(1 + 2(2\sin\theta\cos\theta)) = 4(1 + 4\sin\theta\cos\theta)$.

Substitute $\sin\theta = y/r$ and $\cos\theta = x/r$.

$r = 4\left(1 + 4 \frac{y}{r} \frac{x}{r}\right) = 4\left(1 + \frac{4xy}{r^2}\right)$.

$r = 4\left(\frac{r^2 + 4xy}{r^2}\right)$.

Multiply by $r^2$: $r^3 = 4(r^2 + 4xy)$.

Substitute $r^2 = x^2+y^2$. $r = \sqrt{x^2+y^2}$, so $r^3 = (r^2)^{3/2} = (x^2+y^2)^{3/2}$.

$(x^2+y^2)^{3/2} = 4(x^2+y^2 + 4xy)$.

Squaring both sides gives: $(x^2+y^2)^3 = 16(x^2+y^2+4xy)^2$. This is the Cartesian form.

(ii) Convert $xy=3$ to polar form.

Substitute $x = r\cos\theta$ and $y=r\sin\theta$.

$(r\cos\theta)(r\sin\theta) = 3$.

$r^2 \cos\theta \sin\theta = 3$.

Use the identity $\sin 2\theta = 2\sin\theta\cos\theta$, so $\cos\theta\sin\theta = \frac{1}{2}\sin 2\theta$.

$r^2 (\frac{1}{2}\sin 2\theta) = 3$.

$r^2 \sin 2\theta = 6$. This is the polar form.

(c) Solve the system of linear equations.

(1) $3 I_{1}+2 I_{2}+5 I_{3}=2$

(2) $3 I_{1}+3 I_{2}-2 I_{3}=4$

(3) $2 I_{1}-5 I_{2}-3 I_{3}=14$

Eliminate $I_1$.

(1) - (2): $(3I_1+2I_2+5I_3) - (3I_1+3I_2-2I_3) = 2 - 4$.

$-I_2 + 7I_3 = -2 \quad (4)$

To use equation (3), multiply (1) by 2 and (3) by 3:

2*(1): $6I_1+4I_2+10I_3 = 4$

3*(3): $6I_1-15I_2-9I_3 = 42$

Subtract (3') from (1'): $(6I_1+4I_2+10I_3) - (6I_1-15I_2-9I_3) = 4 - 42$.

$19I_2 + 19I_3 = -38$. Divide by 19:

$I_2 + I_3 = -2 \quad (5)$

Now solve the system in $I_2, I_3$:

(4) $-I_2 + 7I_3 = -2$

(5) $I_2 + I_3 = -2$

Add (4) and (5): $(-I_2 + 7I_3) + (I_2 + I_3) = -2 + (-2)$.

$8I_3 = -4 \implies I_3 = -4/8 = -1/2$.

Substitute $I_3 = -1/2$ into (5): $I_2 + (-1/2) = -2 \implies I_2 = -2 + 1/2 = -3/2$.

Substitute $I_2 = -3/2$ and $I_3 = -1/2$ into (1):

$3I_1 + 2(-3/2) + 5(-1/2) = 2$.

$3I_1 - 3 - 5/2 = 2$.

$3I_1 = 2 + 3 + 5/2 = 5 + 2.5 = 7.5$.

$I_1 = 7.5 / 3 = 2.5$.

Solution: $I_1 = 2.5$ A, $I_2 = -1.5$ A, $I_3 = -0.5$ A.

Solutions require applying index laws, logarithm rules, coordinate conversion formulas, and systematic elimination for linear systems.

Question 2 (20 marks)

(a) Prove the identity $\frac{\tan \theta+\sec \theta}{\sec \theta\left(1+\frac{\tan \theta}{\sec \theta}\right)}=1$. (5 marks)

(b) If $\sin \mathrm{A}=\frac{3}{5}$ and $\cos \mathrm{B}=\frac{6}{10}$, where A and B are acute angles, determine:

(i) $\quad \sin (\mathrm{A}-\mathrm{B})$; (4 marks)

(ii) $\cot 2 \mathrm{~A}$. (4 marks)

(c) Given that $8 \cos \theta+6 \sin \theta=\mathrm{R} \sin (\theta+\alpha)$, where $\mathrm{R}>0$ and $0^{\circ} \leq \alpha \leq 90^{\circ}$:

(i) find the values of R and $\alpha$; (3 marks)

(ii) hence, solve the equation $8 \cos \theta+6 \sin \theta=6$ for $0^{\circ} \leq \theta \leq 360^{\circ}$. (4 marks)

Answer

This question focuses on proving trigonometric identities, evaluating compound angles, and solving trigonometric equations using the R-formula method.

(a) Prove $\frac{\tan \theta+\sec \theta}{\sec \theta\left(1+\frac{\tan \theta}{\sec \theta}\right)}=1$.

Simplify the denominator:

Denominator = $\sec \theta \left( 1 + \frac{\tan \theta}{\sec \theta} \right) = \sec \theta \left( 1 + \frac{\sin\theta/\cos\theta}{1/\cos\theta} \right)$.

$= \sec \theta (1 + \sin\theta)$.

Now look at the LHS: $\frac{\tan \theta+\sec \theta}{\sec \theta (1 + \sin\theta)}$.

Express numerator in terms of $\sin\theta, \cos\theta$:

Numerator = $\frac{\sin \theta}{\cos \theta} + \frac{1}{\cos \theta} = \frac{\sin \theta + 1}{\cos \theta}$.

LHS = $\frac{(\sin \theta + 1)/\cos \theta}{(1/\cos \theta) (1 + \sin\theta)}$.

LHS = $\frac{(1 + \sin \theta)/\cos \theta}{(1 + \sin\theta)/\cos \theta} = 1$. (Proven)

(b) Given $\sin A = 3/5$ and $\cos B = 6/10 = 3/5$ (A, B acute).

Find $\cos A$ and $\sin B$.

Since A is acute, $\cos A > 0$. $\cos^2 A = 1 - \sin^2 A = 1 - (3/5)^2 = 1 - 9/25 = 16/25$. So $\cos A = 4/5$.

Since B is acute, $\sin B > 0$. $\sin^2 B = 1 - \cos^2 B = 1 - (3/5)^2 = 16/25$. So $\sin B = 4/5$.

(Note: If $\sin A = \cos B$ and A, B are acute, then $A+B=90^\circ$, so $A$ and $B$ are complementary).

(i) Find $\sin(A-B)$.

$\sin(A-B) = \sin A \cos B - \cos A \sin B$.

$= (3/5)(3/5) - (4/5)(4/5) = 9/25 - 16/25 = -7/25$.

(ii) Find $\cot 2A$.

First find $\tan A = \sin A / \cos A = (3/5) / (4/5) = 3/4$.

Use the double angle formula for tan: $\tan 2A = \frac{2 \tan A}{1 - \tan^2 A}$.

$\tan 2A = \frac{2(3/4)}{1 - (3/4)^2} = \frac{3/2}{1 - 9/16} = \frac{3/2}{(16-9)/16} = \frac{3/2}{7/16}$.

$\tan 2A = \frac{3}{2} \times \frac{16}{7} = \frac{3 \times 8}{7} = \frac{24}{7}$.

$\cot 2A = \frac{1}{\tan 2A} = \frac{1}{24/7} = \frac{7}{24}$.

(c)

(i) Express $8 \cos \theta+6 \sin \theta$ as $R \sin (\theta+\alpha)$.

$R \sin(\theta+\alpha) = R(\sin\theta \cos\alpha + \cos\theta \sin\alpha) = (R\cos\alpha)\sin\theta + (R\sin\alpha)\cos\theta$.

Compare with $6\sin\theta + 8\cos\theta$:

$R \cos \alpha = 6$.

$R \sin \alpha = 8$.

$R^2 = 6^2 + 8^2 = 36 + 64 = 100 \implies R = \sqrt{100} = 10$.

$\tan \alpha = \frac{R \sin \alpha}{R \cos \alpha} = \frac{8}{6} = \frac{4}{3}$.

$\alpha = \arctan(4/3)$. Since $\sin \alpha, \cos \alpha$ are positive, $\alpha$ is in the 1st quadrant.

$\alpha \approx 53.13^{\circ}$.

So, $8 \cos \theta+6 \sin \theta = 10 \sin(\theta + 53.13^{\circ})$.

(ii) Solve $8 \cos \theta+6 \sin \theta=6$ for $0^{\circ} \leq \theta \leq 360^{\circ}$.

Using part (i): $10 \sin(\theta + 53.13^{\circ}) = 6$.

$\sin(\theta + 53.13^{\circ}) = \frac{6}{10} = 0.6$.

Let $\phi = \theta + 53.13^{\circ}$. Then $\sin \phi = 0.6$.

Principal value: $\phi_1 = \arcsin(0.6) \approx 36.87^{\circ}$.

Since $\sin \phi$ is positive, solutions for $\phi$ are in 1st and 2nd quadrants.

1st Quad: $\phi_1 = 36.87^{\circ}$.

2nd Quad: $\phi_2 = 180^{\circ} - 36.87^{\circ} = 143.13^{\circ}$.

Consider solutions in the next cycle as well, as the range for $\phi$ might require it. $\phi_3 = 36.87^\circ + 360^\circ = 396.87^\circ$.

Now find $\theta = \phi - 53.13^{\circ}$.

$\theta_1 = 36.87^{\circ} - 53.13^{\circ} = -16.26^{\circ}$. This is outside the range $0 \le \theta \le 360$. Add $360^\circ$: $343.74^{\circ}$.

$\theta_2 = 143.13^{\circ} - 53.13^{\circ} = 90^{\circ}$.

$\theta_3 = 396.87^{\circ} - 53.13^{\circ} = 343.74^{\circ}$. (Same as $\theta_1 + 360^\circ$).

The solutions in the range $0^{\circ} \leq \theta \leq 360^{\circ}$ are $90^{\circ}$ and $343.7^{\circ}$ (approx).

Proofs involve algebraic manipulation using trig definitions. Compound angle formulas and R-formula are applied correctly.

Question 3 (20 marks)

(a) A committee of 5 is to be chosen from 7 men and 6 women. Find the number of ways in which the committee can be formed so that it contains at least 3 men. (6 marks)

(b) (i) Prove that if $\mathrm{x}^{3}$ and higher power can be neglected, $\sqrt{\frac{1+3 \mathrm{x}}{1-3 \mathrm{x}}}=1+3 \mathrm{x}+\frac{9}{2} \mathrm{x}^{2}$. (6 marks)

(ii) Hence, by letting $\mathrm{x}=\frac{1}{9}$ in (i) above, show that $\sqrt{2}=1 \frac{11}{25}$. (Mistake in target value?) (4 marks)

(c) The resonant frequency $f_{r}=\frac{1}{2 \pi \sqrt{L C}}$. If L increases by $2.4 \%$ and C decreases by $0.7 \%$, determine using the binomial theorem the percentage change in $f_{r}$, correct to one decimal place. (4 marks)

Answer

This question covers combinations, binomial expansion for approximation, and using binomial theorem for percentage change calculations.

(a) Committee with at least 3 men.

Total people: 7 men, 6 women. Committee size: 5.

Possible compositions with at least 3 men:

Case 1: 3 men and 2 women.

Number of ways = $\binom{7}{3} \times \binom{6}{2}$.

$\binom{7}{3} = \frac{7!}{3!4!} = \frac{7 \times 6 \times 5}{3 \times 2 \times 1} = 35$.

$\binom{6}{2} = \frac{6!}{2!4!} = \frac{6 \times 5}{2 \times 1} = 15$.

Ways for Case 1 = $35 \times 15 = 525$.

Case 2: 4 men and 1 woman.

Number of ways = $\binom{7}{4} \times \binom{6}{1}$.

$\binom{7}{4} = \frac{7!}{4!3!} = 35$.

$\binom{6}{1} = 6$.

Ways for Case 2 = $35 \times 6 = 210$.

Case 3: 5 men and 0 women.

Number of ways = $\binom{7}{5} \times \binom{6}{0}$.

$\binom{7}{5} = \binom{7}{2} = \frac{7 \times 6}{2 \times 1} = 21$.

$\binom{6}{0} = 1$.

Ways for Case 3 = $21 \times 1 = 21$.

Total number of ways = Ways(Case 1) + Ways(Case 2) + Ways(Case 3).

Total ways = $525 + 210 + 21 = 756$.

(b)

(i) Approximate $\sqrt{\frac{1+3x}{1-3x}} = (1+3x)^{1/2} (1-3x)^{-1/2}$ up to $x^2$.

Use $(1+u)^k \approx 1 + ku + \frac{k(k-1)}{2} u^2$.

$(1+3x)^{1/2} \approx 1 + \frac{1}{2}(3x) + \frac{\frac{1}{2}(-\frac{1}{2})}{2}(3x)^2 = 1 + \frac{3}{2}x - \frac{1}{8}(9x^2) = 1 + \frac{3}{2}x - \frac{9}{8}x^2$.

$(1-3x)^{-1/2} \approx 1 + (-\frac{1}{2})(-3x) + \frac{-\frac{1}{2}(-\frac{3}{2})}{2}(-3x)^2 = 1 + \frac{3}{2}x + \frac{3/4}{2}(9x^2) = 1 + \frac{3}{2}x + \frac{27}{8}x^2$.

Multiply the expansions, keeping terms up to $x^2$:

$(1 + \frac{3}{2}x - \frac{9}{8}x^2)(1 + \frac{3}{2}x + \frac{27}{8}x^2)$

$\approx 1(1 + \frac{3}{2}x + \frac{27}{8}x^2) + \frac{3}{2}x(1 + \frac{3}{2}x) - \frac{9}{8}x^2(1)$.

$\approx 1 + \frac{3}{2}x + \frac{27}{8}x^2 + \frac{3}{2}x + \frac{9}{4}x^2 - \frac{9}{8}x^2$.

$\approx 1 + (\frac{3}{2}+\frac{3}{2})x + (\frac{27}{8} + \frac{18}{8} - \frac{9}{8})x^2$.

$\approx 1 + 3x + \frac{36}{8}x^2 = 1 + 3x + \frac{9}{2}x^2$. (Proven)

(ii) Use $x=1/9$ to show $\sqrt{2} = 1 \frac{11}{25}$.

The target value $1 \frac{11}{25} = \frac{36}{25} = 1.44$. $\sqrt{2} \approx 1.414$. The target value seems incorrect. Let's calculate the approximation.

Evaluate $\sqrt{\frac{1+3x}{1-3x}}$ with $x=1/9$.

$\sqrt{\frac{1+3(1/9)}{1-3(1/9)}} = \sqrt{\frac{1+1/3}{1-1/3}} = \sqrt{\frac{4/3}{2/3}} = \sqrt{\frac{4}{2}} = \sqrt{2}$.

Now use the approximation $1 + 3x + \frac{9}{2}x^2$ with $x=1/9$.

$1 + 3(1/9) + \frac{9}{2}(1/9)^2 = 1 + \frac{1}{3} + \frac{9}{2}(\frac{1}{81}) = 1 + \frac{1}{3} + \frac{1}{18}$.

$= \frac{18}{18} + \frac{6}{18} + \frac{1}{18} = \frac{25}{18}$.

So, the approximation gives $\sqrt{2} \approx \frac{25}{18}$.

$\frac{25}{18} \approx 1.388...$ This is close to $\sqrt{2} \approx 1.414$, but not equal to $1 \frac{11}{25}$. There might be an error in the question's target value.

(c) Percentage change in $f_r = \frac{1}{2\pi} (LC)^{-1/2}$.

Let $f_0$ be the original frequency. $f_r \propto L^{-1/2} C^{-1/2}$.

New L: $L' = L(1+0.024)$.

New C: $C' = C(1-0.007)$.

New frequency $f'_r \propto (L')^{-1/2} (C')^{-1/2}$.

$f'_r \propto [L(1.024)]^{-1/2} [C(0.993)]^{-1/2}$.

$f'_r \propto L^{-1/2} C^{-1/2} (1.024)^{-1/2} (0.993)^{-1/2}$.

$f'_r \approx f_0 (1+0.024)^{-1/2} (1-0.007)^{-1/2}$.

Use binomial approximation $(1+x)^n \approx 1+nx$.

$(1+0.024)^{-1/2} \approx 1 + (-\frac{1}{2})(0.024) = 1 - 0.012 = 0.988$.

$(1-0.007)^{-1/2} \approx 1 + (-\frac{1}{2})(-0.007) = 1 + 0.0035 = 1.0035$.

$f'_r \approx f_0 (0.988)(1.0035)$.

$0.988 \times 1.0035 = 0.988(1 + 0.0035) = 0.988 + 0.988(0.0035) \approx 0.988 + 0.003458 = 0.991458$.

$f'_r \approx f_0 (0.991458)$.

Change = $f'_r - f_0 \approx f_0 (0.991458 - 1) = -0.008542 f_0$.

Percentage change = $\frac{f'_r - f_0}{f_0} \times 100\% \approx -0.008542 \times 100\% = -0.8542\%$.

Correct to one decimal place: $-0.9\%$.

Alternative using calculus: $\ln f_r = \ln(\frac{1}{2\pi}) - \frac{1}{2}\ln L - \frac{1}{2}\ln C$. $\frac{\delta f_r}{f_r} \approx -\frac{1}{2}\frac{\delta L}{L} - \frac{1}{2}\frac{\delta C}{C}$. $\% \text{ change in } f_r \approx -\frac{1}{2}(\% \text{ change in } L) - \frac{1}{2}(\% \text{ change in } C)$. $\approx -\frac{1}{2}(+2.4\%) - \frac{1}{2}(-0.7\%) = -1.2\% + 0.35\% = -0.85\%$. Rounded to one decimal place: $-0.9\%$.

Combinations involve casework. Binomial expansion is used for approximation, noting potential errors in the question. Percentage change is calculated using binomial approximation or calculus.

Question 4 (20 marks)

(a) Find the inverse function of $f(x)=\frac{-2}{x-5}$. (4 marks)

(b) (i) Show that $\tanh ^{-1} \mathrm{x}=\frac{1}{2} \ln \left(\frac{1+\mathrm{x}}{1-\mathrm{x}}\right)$; (6 marks)

(ii) Hence, determine $\tanh ^{-1} 0.71$, correct to four decimal places. (3 marks)

(c) Solve the equation $2 \sinh x+3 \cosh x=5$, correct to four decimal places. (7 marks)

Answer

This question involves finding the inverse of a rational function, deriving the logarithmic form of an inverse hyperbolic function, and solving a hyperbolic equation using exponential definitions.

(a) Find inverse of $f(x)=\frac{-2}{x-5}$.

Let $y = f(x) = \frac{-2}{x-5}$.

To find the inverse $f^{-1}(y)$, swap $x$ and $y$ and solve for $y$.

$x = \frac{-2}{y-5}$.

Multiply by $(y-5)$: $x(y-5) = -2$.

$xy - 5x = -2$.

$xy = 5x - 2$.

$y = \frac{5x - 2}{x}$.

So, the inverse function is $f^{-1}(x) = \frac{5x - 2}{x}$ (or $5 - \frac{2}{x}$).

(b)

(i) Show $\tanh ^{-1} x=\frac{1}{2} \ln \left(\frac{1+\mathrm{x}}{1-\mathrm{x}}\right)$.

Let $y = \tanh^{-1} x$. Then $x = \tanh y$. Domain is $-1 < x < 1$.

$x = \frac{\sinh y}{\cosh y} = \frac{(e^y - e^{-y})/2}{(e^y + e^{-y})/2} = \frac{e^y - e^{-y}}{e^y + e^{-y}}$.

Multiply numerator and denominator by $e^y$: $x = \frac{e^{2y} - 1}{e^{2y} + 1}$.

Solve for $e^{2y}$: $x(e^{2y} + 1) = e^{2y} - 1$.

$x e^{2y} + x = e^{2y} - 1$.

$x + 1 = e^{2y} - x e^{2y} = e^{2y}(1 - x)$.

$e^{2y} = \frac{1+x}{1-x}$.

Take the natural logarithm: $2y = \ln\left(\frac{1+x}{1-x}\right)$.

$y = \frac{1}{2} \ln\left(\frac{1+x}{1-x}\right)$.

Since $y = \tanh^{-1} x$, we have $\tanh^{-1} x = \frac{1}{2} \ln\left(\frac{1+x}{1-x}\right)$. (Shown)

(ii) Determine $\tanh^{-1} 0.71$.

Use the formula: $\tanh^{-1} 0.71 = \frac{1}{2} \ln\left(\frac{1+0.71}{1-0.71}\right)$.

$= \frac{1}{2} \ln\left(\frac{1.71}{0.29}\right)$.

$\frac{1.71}{0.29} \approx 5.89655$.

$\tanh^{-1} 0.71 = \frac{1}{2} \ln(5.89655)$.

$\ln(5.89655) \approx 1.77436$.

$\tanh^{-1} 0.71 \approx \frac{1}{2}(1.77436) \approx 0.88718$.

Correct to four decimal places: $0.8872$.

(c) Solve $2 \sinh x+3 \cosh x=5$.

Substitute exponential definitions: $\sinh x = \frac{e^x - e^{-x}}{2}$, $\cosh x = \frac{e^x + e^{-x}}{2}$.

$2\left(\frac{e^x - e^{-x}}{2}\right) + 3\left(\frac{e^x + e^{-x}}{2}\right) = 5$.

$(e^x - e^{-x}) + \frac{3}{2}(e^x + e^{-x}) = 5$.

Multiply by 2: $2(e^x - e^{-x}) + 3(e^x + e^{-x}) = 10$.

$2e^x - 2e^{-x} + 3e^x + 3e^{-x} = 10$.

Combine terms: $5e^x + e^{-x} = 10$.

Multiply by $e^x$ to eliminate $e^{-x}$: $5(e^x)^2 + 1 = 10e^x$.

Let $y = e^x$. The equation is $5y^2 + 1 = 10y$.

Rearrange into quadratic form: $5y^2 - 10y + 1 = 0$.

Use the quadratic formula: $y = \frac{-(-10) \pm \sqrt{(-10)^2 - 4(5)(1)}}{2(5)}$.

$y = \frac{10 \pm \sqrt{100 - 20}}{10} = \frac{10 \pm \sqrt{80}}{10}$.

$\sqrt{80} = \sqrt{16 \times 5} = 4\sqrt{5}$.

$y = \frac{10 \pm 4\sqrt{5}}{10} = 1 \pm \frac{2\sqrt{5}}{5}$.

Since $y=e^x$, $y$ must be positive. Both $1 + \frac{2\sqrt{5}}{5}$ and $1 - \frac{2\sqrt{5}}{5}$ need checking. $\sqrt{5} \approx 2.236$. $2\sqrt{5}/5 \approx 4.472/5 \approx 0.894$. Both $1+0.894$ and $1-0.894$ are positive.

Case 1: $e^x = 1 + \frac{2\sqrt{5}}{5}$.

$x = \ln\left(1 + \frac{2\sqrt{5}}{5}\right) \approx \ln(1 + 0.8944) = \ln(1.8944) \approx 0.63887$.

Case 2: $e^x = 1 - \frac{2\sqrt{5}}{5}$.

$x = \ln\left(1 - \frac{2\sqrt{5}}{5}\right) \approx \ln(1 - 0.8944) = \ln(0.1056) \approx -2.24818$.

Solutions correct to four decimal places are $x \approx 0.6389$ and $x \approx -2.2482$.

Finding inverse functions, deriving logarithmic forms of inverse hyperbolic functions, and solving hyperbolic equations require algebraic manipulation and applying definitions.

Question 5 (20 marks)

(a) Given that $\mathrm{Z}_{1}=1+\mathrm{j} 2, \mathrm{Z}_{2}=2-\mathrm{j} 3$ and $\mathrm{Z}_{3}=-4+\mathrm{j} 12$, determine:

(i) $3 Z_{1}+Z_{2}-Z_{3}$; (3 marks)

(ii) $Z_{1}+\frac{Z_{2} Z_{3}}{Z_{2}+Z_{3}}$. (7 marks)

(b) Use Demoivre's theorem to express $\cos ^{6} \theta$ in terms of the cosines of multiples of $\theta$. (6 marks)

(c) (i) If $Z=x+j y$, show that the locus defined by $\arg \left\{\frac{Z+2}{Z}\right\}=\frac{\pi}{4}$ is a circle. (3 marks)

(ii) Hence, determine its centre and radius. (1 mark)

Answer

This question involves complex number arithmetic, using De Moivre's theorem for power reduction, and finding the locus of a complex number defined by its argument.

(a) Complex Number Arithmetic.

(i) $3 Z_{1}+Z_{2}-Z_{3}$.

$= 3(1+2j) + (2-3j) - (-4+12j)$.

$= (3+6j) + (2-3j) + (4-12j)$.

Combine real parts: $3 + 2 + 4 = 9$.

Combine imaginary parts: $6j - 3j - 12j = -9j$.

Result: $9 - 9j$.

(ii) $Z_{1}+\frac{Z_{2} Z_{3}}{Z_{2}+Z_{3}}$.

$Z_2 Z_3 = (2-3j)(-4+12j) = 2(-4) + 2(12j) - 3j(-4) - 3j(12j)$.

$= -8 + 24j + 12j - 36j^2 = -8 + 36j - 36(-1) = -8 + 36j + 36 = 28 + 36j$.

$Z_2 + Z_3 = (2-3j) + (-4+12j) = (2-4) + (-3+12)j = -2 + 9j$.

Fraction: $\frac{28+36j}{-2+9j}$. Rationalize:

$\frac{(28+36j)(-2-9j)}{(-2+9j)(-2-9j)} = \frac{28(-2)+28(-9j)+36j(-2)+36j(-9j)}{(-2)^2 - (9j)^2}$.

$= \frac{-56 - 252j - 72j - 324j^2}{4 - (81j^2)} = \frac{-56 - 324j + 324}{4 - (-81)} = \frac{268 - 324j}{85}$.

Now add $Z_1$: $(1+2j) + \frac{268 - 324j}{85}$.

$= \frac{85(1+2j) + (268 - 324j)}{85} = \frac{(85 + 170j) + (268 - 324j)}{85}$.

$= \frac{(85+268) + (170-324)j}{85} = \frac{353 - 154j}{85}$.

$= \frac{353}{85} - j \frac{154}{85}$.

(b) Express $\cos^6 \theta$ using De Moivre's theorem.

Let $z = \cos \theta + j \sin \theta$. Then $z^{-1} = \cos \theta - j \sin \theta$.

$z + z^{-1} = 2 \cos \theta$.

$z^n = \cos n\theta + j \sin n\theta$. $z^{-n} = \cos n\theta - j \sin n\theta$.

$z^n + z^{-n} = 2 \cos n\theta$.

We need $(2\cos \theta)^6 = (z + z^{-1})^6$.

Expand $(z+z^{-1})^6$ using binomial theorem (coefficients 1 6 15 20 15 6 1):

$= z^6 + 6z^5(z^{-1}) + 15z^4(z^{-1})^2 + 20z^3(z^{-1})^3 + 15z^2(z^{-1})^4 + 6z(z^{-1})^5 + (z^{-1})^6$.

$= z^6 + 6z^4 + 15z^2 + 20 + 15z^{-2} + 6z^{-4} + z^{-6}$.

Group terms: $(z^6 + z^{-6}) + 6(z^4 + z^{-4}) + 15(z^2 + z^{-2}) + 20$.

Substitute $z^n + z^{-n} = 2 \cos n\theta$:

$= (2 \cos 6\theta) + 6(2 \cos 4\theta) + 15(2 \cos 2\theta) + 20$.

$= 2 \cos 6\theta + 12 \cos 4\theta + 30 \cos 2\theta + 20$.

This equals $(2\cos\theta)^6 = 64 \cos^6 \theta$.

So, $64 \cos^6 \theta = 2 \cos 6\theta + 12 \cos 4\theta + 30 \cos 2\theta + 20$.

Divide by 64:

$\cos^6 \theta = \frac{1}{64} (2 \cos 6\theta + 12 \cos 4\theta + 30 \cos 2\theta + 20)$.

$\cos^6 \theta = \frac{1}{32} (\cos 6\theta + 6 \cos 4\theta + 15 \cos 2\theta + 10)$.

(c) Locus for $\arg\left(\frac{Z+2}{Z}\right)=\frac{\pi}{4}$.

(i) Show it's a circle.

Let $W = \frac{Z+2}{Z} = \frac{x+jy+2}{x+jy} = \frac{(x+2)+jy}{x+jy}$.

Rationalize: $W = \frac{((x+2)+jy)(x-jy)}{(x+jy)(x-jy)} = \frac{(x+2)x - (x+2)jy + jyx - jy(jy)}{x^2 - (jy)^2}$.

$W = \frac{x^2+2x - jxy - 2jy + jxy + y^2}{x^2+y^2} = \frac{(x^2+2x+y^2) + j(-2y)}{x^2+y^2}$.

$W = \frac{x^2+y^2+2x}{x^2+y^2} + j \frac{-2y}{x^2+y^2}$.

We are given $\arg(W) = \pi/4$.

$\arg(W) = \arctan\left(\frac{\text{Imaginary Part}}{\text{Real Part}}\right)$.

$\tan(\arg(W)) = \frac{\text{Imaginary Part}}{\text{Real Part}}$.

$\tan(\pi/4) = 1$.

So, $1 = \frac{(-2y)/(x^2+y^2)}{(x^2+y^2+2x)/(x^2+y^2)} = \frac{-2y}{x^2+y^2+2x}$.

$x^2+y^2+2x = -2y$.

Rearrange: $x^2+2x + y^2+2y = 0$.

Complete the square:

$(x^2+2x+1) + (y^2+2y+1) = 0 + 1 + 1$.

$(x+1)^2 + (y+1)^2 = 2$.

This is the equation of a circle.

(ii) Determine centre and radius.

The standard form is $(x-h)^2 + (y-k)^2 = R^2$.

Comparing $(x-(-1))^2 + (y-(-1))^2 = (\sqrt{2})^2$.

Centre $(h, k) = (-1, -1)$.

Radius $R = \sqrt{2}$.

Complex arithmetic, De Moivre's theorem for power reduction, and geometric interpretation of complex number arguments are demonstrated.

Question 8 (20 marks)

(a) Prove the identities:

(i) $\quad \sinh (x+y)=\sinh x \cosh y+\cosh x \sinh y$; (4 marks)

(ii) $\tanh 2 x=\frac{2 \tanh x}{1+\tanh ^{2} x}$ (4 marks)

(b) (i) Express $\tanh ^{-1} x$ in logarithmic form. (4 marks)

(ii) Hence determine the value of $\tanh ^{-1}\left(\frac{1}{2}\right)$, correct to four decimal places. (3 marks)

(c) Solve the equation $3 \cosh ^{2} x-7 \sinh x-1=0$. (5 marks)

Answer

This question deals with proving hyperbolic identities, finding the logarithmic form of inverse hyperbolic functions, and solving equations involving hyperbolic functions.

(a) Prove identities.

(i) Prove $\sinh (x+y)=\sinh x \cosh y+\cosh x \sinh y$.

Use exponential definitions: $\sinh u = \frac{e^u-e^{-u}}{2}$, $\cosh u = \frac{e^u+e^{-u}}{2}$.

RHS = $\left(\frac{e^x-e^{-x}}{2}\right)\left(\frac{e^y+e^{-y}}{2}\right) + \left(\frac{e^x+e^{-x}}{2}\right)\left(\frac{e^y-e^{-y}}{2}\right)$.

Combine over common denominator 4:

RHS = $\frac{1}{4} [ (e^x-e^{-x})(e^y+e^{-y}) + (e^x+e^{-x})(e^y-e^{-y}) ]$.

Expand the products:

Term 1: $e^{x+y} + e^{x-y} - e^{-x+y} - e^{-x-y}$.

Term 2: $e^{x+y} - e^{x-y} + e^{-x+y} - e^{-x-y}$.

Add Term 1 and Term 2:

$(e^{x+y} + e^{x-y} - e^{-x+y} - e^{-x-y}) + (e^{x+y} - e^{x-y} + e^{-x+y} - e^{-x-y})$

$= 2e^{x+y} - 2e^{-x-y} = 2(e^{x+y} - e^{-(x+y)})$.

Substitute back into RHS expression:

RHS = $\frac{1}{4} [ 2(e^{x+y} - e^{-(x+y)}) ] = \frac{e^{x+y} - e^{-(x+y)}}{2} = \sinh(x+y)$ = LHS. (Proven)

(ii) Prove $\tanh 2 x=\frac{2 \tanh x}{1+\tanh ^{2} x}$.

This was already proven in Q1(a)(i). Start with $\tanh(A+B) = \frac{\tanh A + \tanh B}{1 + \tanh A \tanh B}$.

Let $A=x$ and $B=x$.

$\tanh(x+x) = \tanh 2x = \frac{\tanh x + \tanh x}{1 + \tanh x \tanh x} = \frac{2 \tanh x}{1 + \tanh^2 x}$. (Proven)

(b)

(i) Express $\tanh ^{-1} x$ in logarithmic form.

This was derived in Q4(b)(i). Result: $\tanh^{-1} x = \frac{1}{2} \ln\left(\frac{1+x}{1-x}\right)$.

(ii) Determine $\tanh^{-1}(1/2)$.

Use the formula: $\tanh^{-1}(0.5) = \frac{1}{2} \ln\left(\frac{1+0.5}{1-0.5}\right)$.

$= \frac{1}{2} \ln\left(\frac{1.5}{0.5}\right) = \frac{1}{2} \ln(3)$.

$\ln(3) \approx 1.098612$.

$\tanh^{-1}(0.5) \approx \frac{1}{2}(1.098612) \approx 0.549306$.

Correct to four decimal places: $0.5493$.

(c) Solve $3 \cosh ^{2} x-7 \sinh x-1=0$.

Use the identity $\cosh^2 x - \sinh^2 x = 1 \implies \cosh^2 x = 1 + \sinh^2 x$.

Substitute into the equation:

$3(1 + \sinh^2 x) - 7 \sinh x - 1 = 0$.

$3 + 3\sinh^2 x - 7 \sinh x - 1 = 0$.

$3\sinh^2 x - 7 \sinh x + 2 = 0$.

Let $y = \sinh x$. The equation is $3y^2 - 7y + 2 = 0$.

Factor the quadratic: $(3y - 1)(y - 2) = 0$.

Solutions for y are $y=1/3$ or $y=2$.

Case 1: $\sinh x = 1/3$.

$x = \sinh^{-1}(1/3)$. Use $\sinh^{-1} u = \ln(u+\sqrt{u^2+1})$.

$x = \ln\left(\frac{1}{3} + \sqrt{(\frac{1}{3})^2 + 1}\right) = \ln\left(\frac{1}{3} + \sqrt{\frac{1}{9} + 1}\right)$.

$x = \ln\left(\frac{1}{3} + \sqrt{\frac{10}{9}}\right) = \ln\left(\frac{1}{3} + \frac{\sqrt{10}}{3}\right) = \ln\left(\frac{1+\sqrt{10}}{3}\right)$.

$x \approx \ln(\frac{1+3.162}{3}) = \ln(\frac{4.162}{3}) = \ln(1.387) \approx 0.327$.

Case 2: $\sinh x = 2$.

$x = \sinh^{-1}(2) = \ln(2 + \sqrt{2^2+1}) = \ln(2 + \sqrt{5})$.

$x \approx \ln(2 + 2.236) = \ln(4.236) \approx 1.444$.

Solutions: $x = \ln\left(\frac{1+\sqrt{10}}{3}\right) \approx 0.327$ and $x = \ln(2+\sqrt{5}) \approx 1.444$.

Hyperbolic identities are proven using definitions. Inverse function is found via logarithms. The equation is solved by substitution and solving the resulting quadratic.
