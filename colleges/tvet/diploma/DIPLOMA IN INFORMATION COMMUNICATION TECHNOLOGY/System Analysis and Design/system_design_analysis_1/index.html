System Analysis and Design

Duration: 20 hours Level: Intermediate

Course Overview

This course offers a detailed exploration of system analysis and design, covering essential concepts, methodologies, and tools. Students will learn the principles of system development life cycle, problem definition, feasibility studies, system analysis techniques, system design and development, system implementation, maintenance, and documentation. The course also delves into project management in the context of ICT and emerging trends in system analysis and design.

Course Modules

  1. Module 1: Introduction to Systems Analysis and Design

    Overview of systems analysis and design, key terms, information systems components, and types of information systems.

    60 minutes

  2. Module 2: Systems Theory/Concept

    Exploration of systems theory, system characteristics, components, types, and classification of systems.

    90 minutes

  3. Module 3: Systems Development Life Cycle (SDLC)

    In-depth study of the Systems Development Life Cycle, its stages, methodologies, and importance.

    120 minutes

  4. Module 4: Problem Definition

    Understanding problem definition in system analysis, indicators, problem identification methods, and terms of reference.

    90 minutes

  5. Module 5: Feasibility Study

    Comprehensive analysis of feasibility study, its steps, types of feasibility, benefits, and feasibility study report.

    120 minutes

  6. Module 6: Systems Analysis

    Detailed examination of systems analysis, objectives, methods, and system analysis tools.

    150 minutes

  7. Module 7: Systems Design and Development

    Principles of systems design and development, design models, components, methodologies, and tools.

    150 minutes

  8. Module 8: System Implementation

    Exploring system implementation process, techniques, system testing, verification, validation, and user training.

    120 minutes

  9. Module 9: System Maintenance and Review

    Overview of system maintenance, importance, types of maintenance, and system review processes.

    90 minutes

  10. Module 10: System Documentation

    Importance of system documentation, benefits, efficient procedures, and types of documentation.

    60 minutes

  11. Module 11: System Acquisition

    System acquisition methods, choices, criteria for selection, and advantages and disadvantages of each method.

    90 minutes

  12. Module 12: ICT Project Management

    Introduction to ICT project management, objectives, tools, project evaluation, and factors for project success and failure.

    150 minutes

  13. Module 13: Emerging Trends in SAD

    Discussion on emerging trends in system analysis and design and research interests in the field.

    60 minutes

Prerequisites

  • Basic understanding of Information and Communication Technology (ICT).
  • Familiarity with computer systems and software concepts.
