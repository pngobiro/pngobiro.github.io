System Analysis and Design

Duration: Varies by module Level: Diploma in ICT

Course Overview

This course provides a comprehensive introduction to System Analysis and Design, essential for students pursuing a Diploma in ICT at the Nairobi Technical Training Institute. The modules cover fundamental concepts, methodologies, and practical techniques required for effective system development.

🎯 Course Objectives

  • Understand the basic concepts of systems analysis and design.
  • Learn different methodologies for system development life cycle (SDLC).
  • Apply various techniques for problem definition and feasibility study.
  • Master system analysis tools and design principles.
  • Understand system implementation, maintenance, and documentation.
  • Gain knowledge of project management in ICT and emerging trends in SAD.

Course Modules

  1. CHAPTER 1: INTRODUCTION TO SYSTEMS ANALYSIS AND DESIGN

    Introduction to basic terms, data and information, types of information systems, and roles of stakeholders.

    Approx. 60 minutes

  2. CHAPTER 2: SYSTEMS THEORY/CONCEPT

    Exploring systems theory, components, types, and classifications of systems and their properties.

    Approx. 45 minutes

  3. CHAPTER 3: SYSTEMS DEVELOPMENT LIFE CYCLE (SDLC)

    In-depth study of SDLC phases, methodologies, and the importance of each stage in system development.

    Approx. 75 minutes

  4. CHAPTER 4: PROBLEM DEFINITION

    Techniques and methods for effective problem definition and system study initiation.

    Approx. 60 minutes

  5. CHAPTER 5: FEASIBILITY STUDY

    Understanding feasibility studies, their types, benefits, and how to conduct and report them.

    Approx. 90 minutes

  6. CHAPTER 6: SYSTEMS ANALYSIS

    Exploring systems analysis, objectives, methods, and tools used in the analysis phase.

    Approx. 90 minutes

  7. CHAPTER 7: SYSTEMS DESIGN AND DEVELOPMENT

    Principles of system design, design models, methodologies, and tools for system development.

    Approx. 120 minutes

  8. CHAPTER 8: SYSTEM IMPLEMENTATION

    Detailed overview of system implementation phases, testing, user training, and implementation techniques.

    Approx. 120 minutes

  9. CHAPTER 9: SYSTEM MAINTENANCE AND REVIEW

    Importance of system maintenance, types of maintenance, and review processes.

    Approx. 45 minutes

  10. CHAPTER 10: SYSTEM DOCUMENTATION

    Types, importance, and benefits of system documentation in the system lifecycle.

    Approx. 30 minutes

  11. CHAPTER 11: SYSTEM ACQUISITION

    Exploring system acquisition methods and criteria for choosing the right approach.

    Approx. 60 minutes

  12. CHAPTER 12: ICT PROJECT MANAGEMENT

    Introduction to project management in ICT, objectives, tools, and evaluation criteria.

    Approx. 75 minutes

  13. CHAPTER 13: EMERGING TRENDS IN SAD

    Overview of emerging trends and challenges in System Analysis and Design.

    Approx. 30 minutes

Prerequisites

  • Basic understanding of Information and Communication Technology (ICT).
  • Familiarity with basic computer operations and software concepts.
