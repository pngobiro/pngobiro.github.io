(ii) Insert appropriate dimensions of the block.

(c) (i) Save the drawing as bearingblock in KNECEXAM14 folder.

(ii) Print the geometry on an A4 paper.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently. The following describes the general process in a typical CAD program. Specific commands may vary.

(a) Settings:

  • Use the `UNITS` command (or equivalent menu option).
  • Set Length Type to "Decimal".
  • Set Precision to "0.000".
  • Set Insertion scale (or drawing units) to "Millimeters".
  • Use the `LIMITS` command. Enter `0,0` for the lower-left corner and `200,270` for the upper-right corner. Use `ZOOM` command with the `All` option (`Z` enter `A` enter) to display the new limits.

(b) Drawing Isometric View:

  1. Setup: Switch to an isometric drawing mode or viewpoint (e.g., SE Isometric). Ensure Ortho mode (`F8`) and/or Polar Tracking (`F10`) are configured for isometric angles (typically 30, 90, 150 degrees).
  2. Construct Base: Starting from point A as the lowest point (0,0,0), use the `LINE` command or `PLINE` to draw the base outline of the block in the isometric plane, using the dimensions derived from the orthographic views in Figure 1 (e.g., overall length, width).
  3. Extrude/Add Height: Use `LINE` or `COPY` commands to draw vertical lines representing the height of different features. Connect the top points to form the top surfaces.
  4. Draw Features: Construct the sloped surface, the cutout, and the hole using lines drawn along isometric axes, paying close attention to the dimensions provided in the orthographic views. Use `ELLIPSE` command with the `Isocircle` option to draw the circular hole correctly in the isometric plane. Use `TRIM` and `FILLET`/`CHAMFER` if needed.
  5. Dimensioning: Use aligned dimensioning (`DIMALIGNED`) along the isometric axes to show the key dimensions (overall length, width, height, feature locations, hole diameter/location). Note: Dimensioning directly on isometric views can be complex; often dimensions are shown on orthographic views instead. The question asks for dimensions on the block itself.

(c) Saving and Printing:

  1. Use the `SAVEAS` command. Navigate to the KNECEXAM14 folder. Enter the filename "bearingblock" and save.
  2. Use the `PLOT` command. Select printer, choose A4 paper size, set plot area (e.g., 'Extents' or 'Window'), choose scale (e.g., 'Fit to paper' or a specific scale like 1:1), preview, and print.
