Task 1 (20 marks)

(a) Open a computer aided design program and make the following drawing settings:

  1. Units: millimetres;
  2. Precision: 0 decimal place.

(b) Insert two layers each with the following properties:

Layer nameLine weight
Outline0.8 mm
Dimension0.2 mm

(c) (i) Figure 1 shows a 3-D wooden block. Draw the figure as it appears on the outline layer using the dimensions indicated.

(ii) Fully dimension the drawing on the dimension layer. ($5^{1 / 2}$ marks)

(d) Save the drawing as wooden block in the KNECEXAM folder.

(e) Print out later the drawing.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently. The following describes the general process in a typical CAD program. Specific commands may vary based on the software used (e.g., AutoCAD, FreeCAD).

(a) Drawing Settings:

  • Use the appropriate command (e.g., `UNITS`) to set the drawing units to Millimeters.
  • Within the same command or a related settings dialog, set the precision for linear units to 0 decimal places.

(b) Layer Creation:

  • Open the layer management tool (e.g., `LAYER` command).
  • Create a new layer named "Outline". Set its line weight property to 0.80 mm.
  • Create another new layer named "Dimension". Set its line weight property to 0.20 mm.
  • Ensure the "Outline" layer is set as the current layer for drawing the main figure.

(c) Drawing and Dimensioning:

  1. Drawing the Block (Outline Layer): Use drawing commands (e.g., `LINE`, `PLINE`, `RECTANGLE`, potentially 3D commands like `EXTRUDE` or `BOX` if creating a true 3D model, or orthographic projection techniques if drawing 2D views) to replicate the geometry shown in Figure 1, adhering strictly to the provided dimensions. Ensure all lines representing the main object are on the "Outline" layer.
  2. Dimensioning (Dimension Layer): Set the "Dimension" layer as the current layer. Use appropriate dimensioning commands (e.g., `DIMLINEAR`, `DIMALIGNED`, `DIMRADIUS`, `DIMDIAMETER`) to add all necessary dimensions to the drawing as indicated in Figure 1, ensuring they clearly communicate the size and shape of the object. Place dimensions neatly and avoid overcrowding.

(d) Saving:

  • Use the `SAVE` or `SAVEAS` command.
  • Navigate to the KNECEXAM folder created on the desktop.
  • Enter the filename "wooden block" and save the drawing file (e.g., as a .dwg or .dxf file depending on the software).

(e) Printing:

  • Use the `PLOT` or `PRINT` command.
  • Configure the plot settings: select a printer/plotter, choose paper size, set plot scale (e.g., 1:1 or fit to page), define plot area (e.g., Extents or Window), and potentially configure line weights if not using layer-based settings.
  • Preview the plot to ensure correctness.
  • Send the plot to the printer (this step is usually done after the exam).

  3. Balance Sheet (as at 30th Sept): Generate the Balance Sheet report, setting the date to 30/09/2021.

Printing: Use the print function within the software for each generated report.
