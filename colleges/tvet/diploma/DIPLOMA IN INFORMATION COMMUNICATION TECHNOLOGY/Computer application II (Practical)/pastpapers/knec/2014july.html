Task 1

(a) Open a CAD program and make changes to the default settings as follows:

  1. Units in inches
  2. Model limits:
    • Lower left corner <0.000,0.000>
    • Upper right corner <270.000,200.000>
  3. Accuracy: 0.00

(b) Insert two layers each with the following properties:

  1. A layer labelled body with the line weight set to 0.8 mm on which to draw the geometry;
  2. A layer labelled dimension on which to draw all the dimensions.

(c) (i) Figure 1 shows a metal plate drawn with dimensions in inches. Draw on the layer labelled body, the geometry using the dimensions indicated. (10 marks)

(ii) Fully dimension the geometry as it appears. Place the dimensions on a layer labelled dimension.

(d) Save the drawing as metal plate in the KNECEXAM folder to print out later. (1 1/2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently. The following describes the general process in a typical CAD program. Specific commands may vary.

(a) Settings:

  • Use the `UNITS` command (or equivalent menu option).
  • Set Length Type to "Decimal".
  • Set Precision to "0.00".
  • Set Insertion scale (or drawing units) to "Inches".
  • Use the `LIMITS` command. Enter `0,0` for the lower-left corner and `270,200` for the upper-right corner. Use `ZOOM` command with the `All` option (`Z` enter `A` enter) to display the new limits.

(b) Layers:

  • Open the Layer Properties Manager (e.g., `LAYER` command).
  • Create a new layer named "body". Set its Lineweight property to 0.80 mm.
  • Create another new layer named "dimension". Keep default lineweight (or set as specified if different).

(c) Drawing and Dimensioning:

  1. Drawing (body Layer): Make the "body" layer current. Use appropriate 2D drawing commands (`LINE`, `CIRCLE`, `ARC`, `FILLET`, `TRIM`, etc.) to construct the metal plate geometry shown in Figure 1 based precisely on the given dimensions (in inches). Pay attention to the curves, tangents, holes, and overall shape.
  2. Dimensioning (dimension Layer): Make the "dimension" layer current. Use appropriate dimensioning commands (`DIMLINEAR`, `DIMALIGNED`, `DIMRADIUS`, `DIMDIAMETER`, `DIMANGULAR`, etc.) to add all necessary dimensions as shown in the figure. Ensure dimensions are clear, correctly placed, and cover all features.

(d) Saving:

  • Use the `SAVEAS` command.
  • Navigate to the KNECEXAM folder.
  • Enter the filename "metal plate" and save.

Printing (Later): Use the `PLOT` command, configure settings (printer, paper size, plot area - likely 'Extents' or 'Window', scale - likely '1:1' or 'Fit to paper'), preview, and print.
