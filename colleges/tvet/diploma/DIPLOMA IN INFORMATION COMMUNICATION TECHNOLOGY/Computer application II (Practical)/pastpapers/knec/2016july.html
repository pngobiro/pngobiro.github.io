Task 1

(a) Open a CAD program and make the following drawing settings:

  1. Units: millimetres.;
  2. Precision: 1 decimal place.
  3. Model space: 3D Modelling

(b) (i) Draw the 3D drawing as it appears in figure 1.

(ii) Fully dimension the drawing as it appears.

(c) Enlarge the 3D drawing by scale factor 1:3 about the lowest corner of the figure.

(d) Save the drawing as 3Dblock in the KNECEXAM folder to print out later.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently. The following describes the general process in a typical CAD program. Specific commands may vary.

(a) Settings:

  • Use the `UNITS` command (or equivalent menu option).
  • Set Length Type to "Decimal".
  • Set Precision to "1" decimal place.
  • Set Insertion scale (or drawing units) to "Millimeters".
  • Ensure the workspace is set to "3D Modeling" (using workspace switching tools or commands).

(b) Drawing:

  1. Use appropriate 3D drawing commands (`BOX`, `EXTRUDE`, `CYLINDER`, `UNION`, `SUBTRACT`, etc.) to construct the object shown in Figure 1 based precisely on the given dimensions. Pay attention to creating the base, the angled cut, the cylindrical hole, and the smaller rectangular cutout.

(c) Dimensioning:

  • Use dimensioning commands (`DIMLINEAR`, `DIMALIGNED`, `DIMRADIUS`, `DIMDIAMETER`, `DIMANGULAR`, etc.) appropriate for 3D space or specific views (e.g., dimensioning on specific UCS planes or in layout viewports) to add all necessary dimensions as shown in the figure. Ensure dimensions are clear and correctly placed.

(d) Scaling:

  • Use the `SCALE` command.
  • Select the entire 3D object.
  • Specify the base point: Select the lowest corner of the figure (use object snaps like Endpoint).
  • Enter the scale factor: `3`.

(e) Saving:

  • Use the `SAVEAS` command.
  • Navigate to the KNECEXAM folder.
  • Enter the filename "3Dblock" and save.

Printing (Later): Use the `PLOT` command, configure settings for desired views (Isometric, Front), and print.
