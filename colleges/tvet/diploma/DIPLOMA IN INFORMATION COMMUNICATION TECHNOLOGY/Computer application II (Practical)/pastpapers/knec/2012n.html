Task 1

(a) Open a Computer Aided Design program in metric option and apply the following as default settings:

  1. drawing limits: $0.000 \times 0.000, 400.000 \times 250.000$ (1 mark)
  2. decimal: 0 decimal places (1 mark)
  3. snap and grid: 5 (1 mark)
  4. units: millimetres (1 mark)

(b) (i) Figure 1 shows a front and plan elevations of a geometrical block drawn in first angle projection. Draw the two views using the dimensions indicated. Show all the dimensions in the drawing.

(ii) Save the drawing as Geometry2d in the KNECEXAM folder to print out later. (2 marks)

(a) Settings:

  • Use the `UNITS` command (or equivalent menu option).
  • Set Length Type to "Decimal".
  • Set Precision to "0" decimal places.
  • Set Insertion scale (or drawing units) to "Millimeters".
  • Use the `LIMITS` command. Enter `0,0` for the lower-left corner and `400,250` for the upper-right corner. Use `ZOOM` command with the `All` option (`Z` enter `A` enter) to display the new limits.
  • Use the `GRID` command and set spacing to `5`.
  • Use the `SNAP` command and set spacing to `5`. Ensure Snap mode (`F9`) and Grid display (`F7`) are toggled on if desired.

(b) Drawing Orthographic Views (First Angle Projection):

  1. Setup: Determine the layout for first angle projection (Plan view below Front view, Side view typically to the left or right as needed). Use construction lines or offsets to align the views correctly based on the dimensions in Figure 1.
  2. Front Elevation: Draw the view seen directly from the front of the object using the given dimensions.
  3. Plan Elevation: Draw the view seen directly from above the object, aligned vertically below the Front Elevation.
  4. Dimensioning: Add all necessary dimensions (`DIMLINEAR`, `DIMALIGNED`, `DIMRADIUS`, etc.) to both views as shown in Figure 1. Ensure clarity and adherence to dimensioning standards.

(c) Saving:

  • Use the `SAVEAS` command.
  • Navigate to the KNECEXAM folder.
  • Enter the filename "Geometry2d" and save.

Printing (Later): Use the `PLOT` command, configure settings (printer, paper size, plot area - likely 'Extents' or 'Window', scale - likely '1:1' or 'Fit to paper'), preview, and print.
