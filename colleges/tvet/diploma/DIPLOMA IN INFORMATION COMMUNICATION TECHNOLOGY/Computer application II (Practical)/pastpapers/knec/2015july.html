Task 1

(a) Open a CAD program and set the default units to be in millimetres.

(b) Insert two layers each with the following properties:

  1. A layer labelled drawing with the line weight set to 0.7 mm on which to draw a block. (1/2 mark)
  2. A layer labelled dimension with the line weight 0.18 mm on which to insert the dimensions. (1/2 mark)
  3. A layer labelled hidden with the line weight 0.5 mm on which to draw the hidden lines. (1/2 mark)

(c) (i) Figure 1 shows the isometric view of a block. Draw the front elevation, side elevation and plan elevation of the block in third angle projection.

(ii) Fully dimension the drawing on the paper space.

(d) Save the drawing as threeviews in the KNECEXAM folder to print out later. (1 1/2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently. The following describes the general process in a typical CAD program. Specific commands may vary.

(a) Settings:

  • Use the `UNITS` command (or equivalent menu option).
  • Set drawing units to "Millimeters".

(b) Layers:

  • Open the Layer Properties Manager (e.g., `LAYER` command).
  • Create a new layer named "drawing". Set its Lineweight property to 0.70 mm.
  • Create another new layer named "dimension". Set its Lineweight property to 0.18 mm.
  • Create a third layer named "hidden". Set its Lineweight property to 0.50 mm. Set its Linetype to a hidden/dashed line type (e.g., HIDDEN, DASHED). Load the linetype if necessary.

(c) Drawing Orthographic Views (Third Angle Projection):

  1. Setup: Determine the layout for third angle projection (Plan view above Front view, Side view to the right of Front view). Use construction lines (Xlines/Rays) or offsets to align the views correctly based on the isometric drawing (Figure 1).
  2. Front Elevation (drawing Layer): Make the "drawing" layer current. Draw the view seen directly from the front of the object in Figure 1, using the given dimensions. Use the "hidden" layer and appropriate linetype for lines representing features not visible from the front (e.g., the hole, the step).
  3. Plan Elevation (drawing Layer): Make the "drawing" layer current. Draw the view seen directly from above the object, aligned vertically with the Front Elevation. Use the "hidden" layer for features not visible from the top.
  4. Side Elevation (drawing Layer): Make the "drawing" layer current. Draw the view seen directly from the side (typically the right side for third angle), aligned horizontally with the Front Elevation. Use the "hidden" layer for features not visible from the side.
  5. Dimensioning (dimension Layer): Make the "dimension" layer current. Switch to Paper Space (Layout tab). Create viewports for the drawn elevations if not already done. Add all necessary dimensions (`DIMLINEAR`, `DIMALIGNED`, `DIMRADIUS`, etc.) across the different views in Paper Space, ensuring no redundancy and adhering to dimensioning standards. Dimension the overall width, height, depth, feature locations, hole diameter/location, etc., as shown or implied in Figure 1.

(d) Saving:

  • Use the `SAVEAS` command.
  • Navigate to the KNECEXAM folder.
  • Enter the filename "threeviews" and save.

Printing (Later): Use the `PLOT` command from the Paper Space Layout, configure settings (printer, paper size, plot area - likely 'Layout', scale - likely '1:1'), preview, and print.
