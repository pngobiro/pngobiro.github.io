üìö DATA SECURITY

THREATS AND HAZARDS TO DATA

The following are among the threats to data security:

  1. Data may get lost or damaged during a system crash - especially one affecting the hard disk
  2. It can be corrupted as a result of faulty disk drives, or power failures
  3. It can be lost by accidental deletion or overwriting of files
  4. It can be lost or become corrupted by computer viruses
  5. It can be hacked into by unauthorized users and stolen, deleted or altered
  6. It can be destroyed by natural disasters, acts of terrorism, or war
  7. It can be deleted or altered by employees wishing to make money or take revenge on their employer

METHODS OF SECURING DATA

Data security refers to protecting data from destructive forces and unwanted actions of unauthorized users. Data needs to be protected from loss through accidental or malicious deletion, virus attacks hard disk or system failures, unauthorized access etc. Methods of protecting data include:

  1. Use of passwords (Access Authentication): A password is a secret word or string of characters that is used for user authentication/verification before the user can gain access to data.
  2. Right of Access refers to the authorization you have to access different data files. Right of access helps determines who has the right to do what in relation to certain data or information.
  3. Logs and Audit trails: An audit trail is a record showing who has accessed a computer system, when and what operations he or she has performed during a given period of time.
  4. Anti-virus programs are software used to prevent, detect and remove malicious software such as viruses.
  5. Encryption: This is the process of encoding (convert into a coded form) information stored on a device.
  6. Firewalls: A firewall is a software or hardware-based network security system that prevents unauthorized access.
  7. Physical Security: Includes locking offices, alarms, security cameras, etc.
  8. Data Protection Act: Legal framework to regulate data collection, processing, storage, and use.
  9. Data Back-up refers to the copying and archiving of computer data in a secure location for restoration.

Types of Data Back-up

  1. Mirror Back-ups/Disk Mirroring/ RAID: Real-time duplication for continuous availability.
  2. Online Backup: Continuous or frequent backups to a connected storage medium.
  3. Remote Backup: Offsite backup accessible for restore and administration remotely.
  4. Offsite Backup/Vaulting: Storage media kept geographically separate for disaster recovery.
  5. Full Backup (or Reference Backup): Complete backup of all selected files and folders.
  6. Incremental Backup: Backs up only files changed since the last incremental backup.
  7. Differential Backup: Backs up files changed since the last full backup.
  8. Local Backup: Storage medium kept close to the source.

üìù Summary of Data Security

  • Data Threats: System crash, corruption, accidental deletion, viruses, hacking, disasters, employee actions.
  • Security Methods: Passwords, Access Rights, Audit Trails, Anti-virus, Encryption, Firewalls, Physical Security, Legal Frameworks, Backups.
  • Backup Types: Mirror, Online, Remote, Offsite, Full, Incremental, Differential, Local.

üìö Key Takeaways

  • Data security is crucial to protect against various threats and hazards.
  • Methods of securing data range from access controls to physical security and legal frameworks.
  • Data backup is essential for disaster recovery and business continuity.
  • Different backup types offer various levels of protection, speed, and storage efficiency.
