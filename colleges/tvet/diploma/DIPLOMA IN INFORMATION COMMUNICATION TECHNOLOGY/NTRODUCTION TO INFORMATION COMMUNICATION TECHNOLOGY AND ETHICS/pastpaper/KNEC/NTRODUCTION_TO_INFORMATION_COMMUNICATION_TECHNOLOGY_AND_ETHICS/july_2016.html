1. (a) (4 marks)

Differentiate between supercomputers and mainframe computers.

Answer

Supercomputers and mainframe computers, while both high-performance computing systems, are designed for different primary purposes.

Supercomputers: Supercomputers are designed for computationally intensive tasks requiring maximum processing speed and are used for complex scientific and engineering problems. They excel in executing a single, complex program or simulation at the fastest possible speed. Applications include weather forecasting, nuclear research, and advanced scientific simulations.

Mainframe Computers: Mainframe computers are designed for processing very large volumes of data and transactions, emphasizing reliability, security, and scalability for business-critical applications. They are optimized for handling numerous concurrent users and input/output operations. Applications include banking transaction processing, airline reservation systems, and large-scale data warehousing.

1. (d) (6 marks)

Wireless Internet is prone to unauthorized user access. Explain three measures that could be used to mitigate this problem.

Answer

To mitigate unauthorized access to wireless internet networks, several robust security measures can be implemented.

Strong Encryption Protocols (WPA3/WPA2): Utilize strong encryption protocols such as Wi-Fi Protected Access 3 (WPA3) or Wi-Fi Protected Access 2 (WPA2) with Advanced Encryption Standard (AES). WPA3 and WPA2 encryption methods secure wireless communication by encrypting data transmitted over the network, making it extremely difficult for unauthorized users to intercept and decrypt wireless traffic, thus preventing eavesdropping and unauthorized access.

Strong Password and Access Control: Implement a strong and unique password for the Wi-Fi network and regularly change it. Use a complex password that combines uppercase and lowercase letters, numbers, and symbols. Additionally, enable access control features such as MAC address filtering to restrict network access to only authorized devices. Strong passwords and access control mechanisms prevent unauthorized users from easily guessing or cracking the Wi-Fi password and gaining network access.

Disable WPS and Enable Firewall: Disable Wi-Fi Protected Setup (WPS) as it is known to have security vulnerabilities that can be exploited for unauthorized access. Enable the firewall on the wireless router and configure it to block unauthorized incoming and outgoing traffic. A firewall acts as a barrier, monitoring and controlling network traffic, preventing unauthorized access and protecting the network from external threats.

2. (a) (i) (3 marks)

Outline three types of computer buses.

Answer

Computer buses are categorized based on their function and the components they connect. Three main types are:

Data Bus: The data bus carries the actual data being transmitted between the CPU, memory, and peripherals. It is bidirectional and its width (e.g., 32-bit, 64-bit) determines the amount of data that can be transferred simultaneously, impacting system performance.

Address Bus: The address bus is used by the CPU to specify the memory location or I/O port address it wants to access. It is unidirectional (from CPU to memory/peripherals) and its width determines the maximum memory address space the CPU can access.

Control Bus: The control bus transmits control signals from the CPU to other components and vice versa, coordinating and synchronizing their operations. Control signals manage data flow, timing, and system operations, ensuring proper functioning of the computer system.

2. (a) (ii) (4 marks)

Distinguish between north and south bridges as used in computer's chipset.

Answer

Northbridge and Southbridge are components of a traditional chipset architecture, managing different aspects of data flow and component communication on a motherboard.

Northbridge: The Northbridge is the chipset component that directly connects to the CPU, handling high-speed data pathways for performance-critical components. It typically manages communication between the CPU, RAM (memory), and the graphics card (PCIe). The Northbridge is responsible for memory controller functions, front-side bus (FSB) interface, and PCIe lanes for graphics, facilitating high-bandwidth data transfer for demanding tasks.

Southbridge: The Southbridge is the chipset component that handles slower input/output (I/O) functions and peripherals. It manages communication between the CPU and slower peripherals such as hard drives (SATA/IDE), USB ports, audio, network interfaces, BIOS, and expansion slots (PCI). The Southbridge acts as an I/O controller hub, managing a wide range of peripheral devices and functionalities, providing connectivity for slower speed components.

2. (c) (ii) (4 marks)

Computers cannot work without operating systems. Outline four functions of the software to justify this statement

Answer

Operating systems (OS) are indispensable for computer operation. Four critical functions justifying this statement are:

Hardware Management: The OS manages and controls all computer hardware components, including the CPU, memory, storage devices, and peripherals. It acts as an intermediary between hardware and software, allocating resources, managing device drivers, and ensuring hardware components function cohesively. Without an OS, hardware resources cannot be effectively utilized or coordinated.

Process Management: The OS manages processes, which are instances of running programs. It handles process creation, scheduling, execution, and termination, enabling multitasking and efficient resource allocation among applications. Without process management by the OS, applications cannot run concurrently or utilize system resources effectively, rendering computers unusable for typical tasks.

User Interface: The OS provides a user interface (UI) that allows users to interact with the computer system. Whether graphical (GUI) or command-line (CLI), the UI enables users to input commands, launch applications, manage files, and control system settings. Without a user interface provided by the OS, users cannot effectively interact with or operate the computer.

File System Management: The OS manages the file system, organizing and storing data on storage devices. It handles file creation, deletion, organization into directories, and access permissions. The file system management by the OS enables users to store, retrieve, and manage data persistently. Without it, data cannot be organized or reliably stored and accessed.

3. (a) (4 marks)

Outline four categories of computer keyboard keys.

Answer

Computer keyboard keys are typically categorized based on their function and position on the keyboard layout.

Alphanumeric Keys: Alphanumeric keys are the primary keys used for typing letters, numbers, symbols, and punctuation marks. They form the central typing area of the keyboard and are used for text input and data entry.

Function Keys: Function keys (F1 to F12) are located at the top of the keyboard and perform specific commands or functions depending on the operating system and application software. They provide shortcuts for common tasks and application-specific commands.

Modifier Keys: Modifier keys (Shift, Ctrl, Alt, Command/Windows key) are used in combination with other keys to modify their function or input. They are used to type uppercase letters, special symbols, and execute keyboard shortcuts for various commands and operations.

Navigation and Editing Keys: Navigation keys (arrow keys, Page Up, Page Down, Home, End) and editing keys (Insert, Delete, Backspace) are used for cursor movement, document navigation, and text editing tasks. They facilitate efficient text manipulation and document navigation within applications.

3. (c) (ii) (4 marks)

Explain two circumstances that could lead to adaption of a ring topology in computer networking.

Answer

While less common in modern networks, ring topology might be adapted under specific circumstances where its characteristics are advantageous.

Deterministic Network Access and Guaranteed Bandwidth: Ring topology, particularly Token Ring, provides deterministic network access and guaranteed bandwidth allocation through token passing. In environments requiring predictable data transmission timing and guaranteed bandwidth for critical applications, such as industrial control systems or time-sensitive data streams, ring topology's deterministic nature might be considered.

Limited Number of Nodes and Simplified Management: For very small networks with a limited number of nodes and simplified management requirements, ring topology's straightforward cabling and configuration might be adapted. In small, localized networks where simplicity and ease of setup are prioritized over scalability or high performance, a ring topology could be a viable option, although star topology is generally preferred even for small networks today.

4. (a) (ii) (4 marks)

(ii) Differentiate between competing and collaborating conflict management techniques.

Answer

Competing and collaborating are contrasting conflict management techniques that differ in their goals and approaches to conflict resolution.

Competing: Competing conflict management is an assertive and uncooperative approach where one party seeks to win at the expense of the other party. It prioritizes one's own concerns over the concerns of others, often using power, authority, or aggression to achieve dominance. Competing is characterized by a win-lose orientation, where one party aims to impose their solution, potentially damaging relationships and fostering resentment.

Collaborating: Collaborating conflict management is an assertive and cooperative approach where parties work together to find a mutually beneficial solution that satisfies the concerns of all involved. It emphasizes open communication, joint problem-solving, and seeking integrative solutions that address the needs and perspectives of all parties. Collaborating is characterized by a win-win orientation, aiming to achieve mutually satisfying outcomes and strengthen relationships through cooperation and shared goals.

4. (b) (4 marks)

Outline four safety precautions that should be adhered to when charging a laptop computer.

Answer

Adhering to safety precautions when charging a laptop computer is essential to prevent hazards and ensure safe operation.

Use Original or Certified Charger: Always use the original charger provided by the laptop manufacturer or a certified replacement charger that meets the laptop's power requirements. Using incompatible or substandard chargers can lead to overheating, damage to the battery or laptop, and potential fire hazards.

Charge in a Well-Ventilated Area: Charge the laptop in a well-ventilated area to prevent heat buildup. Avoid charging on soft surfaces like beds or pillows that can trap heat and cause overheating. Proper ventilation dissipates heat generated during charging, reducing the risk of overheating and battery damage.

Avoid Charging in Direct Sunlight or High Temperatures: Do not charge the laptop in direct sunlight or excessively hot environments as high temperatures can cause overheating and battery degradation. Charge the laptop in a cool and shaded area to maintain optimal charging temperature and battery health.

Inspect Cables and Connectors Regularly: Regularly inspect the charger cable, connectors, and power adapter for any signs of damage, fraying, or wear. Damaged cables or connectors can pose electrical hazards and should be replaced immediately. Ensure connectors are properly inserted and not forced into the charging port to prevent damage.

7. (d) (i) (4 marks)

Desktop computers come with several sizes of CPU casing. Outline four types of such casings.

Answer

Desktop computer CPU casings (or chassis) come in various sizes and form factors, each designed for different purposes and motherboard types.

Full Tower: Full tower cases are the largest desktop casings, offering maximum expansion capacity, airflow, and component compatibility. They typically support full-sized ATX motherboards and multiple expansion cards, drives, and cooling solutions. Full tower cases are suitable for high-performance desktops, gaming rigs, and servers requiring extensive expandability and cooling.

Mid Tower: Mid tower cases are a balance between size and expandability, being smaller than full towers but still offering good expansion options and airflow. They usually support ATX and microATX motherboards and accommodate a reasonable number of expansion cards and drives. Mid tower cases are popular for mainstream desktops, gaming PCs, and workstations, providing a good balance of size and features.

Mini Tower: Mini tower cases are smaller and more compact than mid towers, designed for space-saving and smaller form factor builds. They typically support microATX and mini-ITX motherboards and have limited expansion slots and drive bays. Mini tower cases are suitable for home and office desktops, media centers, and compact systems where space is a constraint.

Small Form Factor (SFF): Small Form Factor (SFF) cases are the smallest desktop casings, designed for ultra-compact and space-constrained environments. They typically support mini-ITX motherboards and have very limited or no expansion slots and drive bays. SFF cases are used for specialized applications like home theater PCs, embedded systems, and minimalist desktops where size is the primary consideration.

8. (a) (3 marks)

Outline three precautions when handling re-writable Compact Disks.

Answer

Handling re-writable Compact Disks (CD-RWs) requires certain precautions to prevent damage and ensure data integrity.

Handle by the Edges and Center Hole: Always handle CD-RWs by their outer edges or the center hole to avoid touching the data surface. Fingerprints, dust, and scratches on the data surface can interfere with data readability and writeability, potentially causing data errors or damage.

Store in Protective Cases: Store CD-RWs in protective jewel cases or sleeves when not in use. Protective cases shield discs from dust, scratches, sunlight, and physical damage. Proper storage prolongs disc lifespan and maintains data integrity.

Avoid Extreme Temperatures and Humidity: Protect CD-RWs from extreme temperatures, humidity, and direct sunlight. Excessive heat, cold, or humidity can damage the disc material, leading to data loss or disc degradation. Store discs in a cool, dry, and stable environment for optimal data preservation.

8. (c) (ii) (3 marks)

Magnetic disks are popular computer storage medium. Outline three of its advantages that influence this trend.

Answer

Magnetic disks (Hard Disk Drives - HDDs) remain popular computer storage media due to several key advantages.

High Storage Capacity at Lower Cost: Magnetic disks offer high storage capacity at a significantly lower cost per gigabyte compared to solid-state drives (SSDs). HDDs provide large storage volumes at affordable prices, making them cost-effective for storing vast amounts of data, such as operating systems, applications, multimedia files, and backups.

Established Technology and Reliability: Magnetic disk technology is a well-established and mature storage medium with a long history of reliability and widespread use. HDDs have proven durability and data retention capabilities over decades, making them a trusted and reliable storage choice for various applications, from personal computers to enterprise storage systems.

Read and Write Cycle Endurance: Magnetic disks have virtually unlimited read and write cycle endurance compared to SSDs, which have finite write cycle limits. HDDs can endure countless read and write operations without performance degradation or wear-out concerns related to write cycles, making them suitable for applications with intensive read/write workloads and long-term data storage.

8. (d) (6 marks)

Bernard intends to install software on a number of computers in his company. Describe three methods of installation that he could apply.

Answer

Bernard can apply various methods to install software on multiple computers in his company, each with different approaches and suitability.

Manual Installation: Manual installation involves individually installing the software on each computer, typically using installation media (DVD, USB drive) or downloading installers from a network share. Manual installation is suitable for a small number of computers or when customization is required for each installation. It is time-consuming and less efficient for large-scale deployments.

Network-Based Installation (Software Distribution Systems): Network-based installation utilizes software distribution systems or tools to deploy software packages centrally over the network to multiple computers simultaneously. Tools like Microsoft Deployment Toolkit (MDT), System Center Configuration Manager (SCCM), or third-party deployment solutions enable automated software installation, configuration, and updates across the network. Network-based installation is efficient for large-scale deployments and centralized software management.

Disk Imaging and Cloning: Disk imaging and cloning involve creating a master image of a computer with the software pre-installed and then deploying this image to multiple computers. Disk imaging tools create an exact copy of the master computer's hard drive, including the operating system, applications, and settings. Cloning software then duplicates this image onto other computers, rapidly deploying software and configurations to multiple machines. Disk imaging and cloning are efficient for deploying identical software configurations across many computers.
