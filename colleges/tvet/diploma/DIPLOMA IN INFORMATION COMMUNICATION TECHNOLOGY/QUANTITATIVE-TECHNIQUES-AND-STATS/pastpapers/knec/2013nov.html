(a) Skewed Distributions:

(i) Positively Skewed Distribution:

A distribution where the tail on the right side is longer or fatter than the left side. Most values cluster around the lower end. The mean is typically greater than the median, which is greater than the mode (Mean > Median > Mode).

(ii) Negatively Skewed Distribution:

A distribution where the tail on the left side is longer or fatter than the right side. Most values cluster around the higher end. The mean is typically less than the median, which is less than the mode (Mean < Median < Mode).

(b) Quantitative vs. Qualitative Data:
  • Quantitative Data: Data that can be measured numerically and expressed in numbers. It deals with quantities, amounts, or counts. Examples: height (175cm), weight (70kg), age (25 years), number of students (50). It can be further classified as discrete (countable, e.g., number of cars) or continuous (measurable within a range, e.g., height).
  • Qualitative Data: Data that describes qualities or characteristics and cannot typically be measured numerically, though it can be categorized. It deals with descriptions, attributes, labels, or categories. Examples: gender (Male/Female), hair color (Brown/Black), opinions (Agree/Disagree), types of cars (Sedan/SUV).

Answer

(a) Linear Programming Terms:

(i) Non-negativity Condition: This fundamental constraint in linear programming states that the decision variables (representing quantities like products produced, resources used, etc.) cannot take negative values. They must be greater than or equal to zero (e.g., x ≥ 0, y ≥ 0). This reflects the real-world scenario where it's impossible to produce a negative quantity of a product.

(ii) Implication of Slack Variable > 0 at Optimum: A slack variable represents the unused amount of a resource corresponding to a 'less than or equal to' (≤) constraint. If a slack variable is greater than zero in the optimal solution, it means that the resource associated with that constraint is not fully utilized. There is a surplus or leftover amount of that resource, and it is not a binding constraint on the optimal solution.

(b) Normal Distribution (Life Span):

Given: Mean (μ) = 55 years, Standard Deviation (σ) = 8 years.

(i) Proportion between 48 and 60 years:

We need to find P(48 ≤ X ≤ 60).

Convert X values to Z-scores: Z = (X - μ) / σ

  • Z1 (for X=48) = (48 - 55) / 8 = -7 / 8 = -0.875 ≈ -0.88
  • Z2 (for X=60) = (60 - 55) / 8 = 5 / 8 = 0.625 ≈ 0.63

Find P(-0.88 ≤ Z ≤ 0.63) = Area(Z ≤ 0.63) - Area(Z ≤ -0.88)

Using standard normal distribution table:

  • Area(Z ≤ 0.63) ≈ 0.7357
  • Area(Z ≤ -0.88) ≈ 0.1894

P(-0.88 ≤ Z ≤ 0.63) = 0.7357 - 0.1894 = 0.5463

Approximately 54.63% of the population will live between 48 and 60 years.

(ii) Minimum Age for Elders (Oldest 5%):

We need to find the X value corresponding to the top 5% (or the 95th percentile).

This means finding the Z-score such that Area(Z > Zscore) = 0.05, or Area(Z ≤ Zscore) = 0.95.

Looking up 0.95 in the body of the Z-table, the closest Z-score is approximately 1.645.

Convert Z-score back to X value: X = μ + Zσ

X = 55 + (1.645 * 8)

X = 55 + 13.16 = 68.16 years

The minimum age to be considered an elder is approximately 68.16 years.

Answer

(a) Spurious Correlation:

Spurious correlation refers to a statistical relationship between two variables that appears causal but is not. The correlation exists, often due to coincidence or the influence of a third, unobserved variable (a confounding factor or lurking variable) that affects both variables independently. For example, ice cream sales and crime rates might be positively correlated, but the actual cause is likely a third variable, such as warmer weather, which increases both.

(b) Steps in Simplex Method:
  1. Formulate the LP Problem: Define the objective function (to be maximized or minimized) and all constraints based on the problem description.
  2. Convert to Standard Form: Introduce slack variables for '≤' constraints, surplus variables for '≥' constraints, and artificial variables for '=' or '≥' constraints (if needed for initial feasible solution). Ensure all RHS values are non-negative. Rewrite the objective function with all variables on one side (e.g., Z - c1x1 - ... = 0).
  3. Create the Initial Simplex Tableau: Set up a table containing the coefficients of the variables from the objective function and constraints, including slack/surplus/artificial variables. The initial basic feasible solution usually consists of the slack/artificial variables.
  4. Check for Optimality: Examine the objective function row (Z-row). If maximizing, the solution is optimal if all coefficients in the Z-row are non-negative. If minimizing, the solution is optimal if all coefficients are non-positive. If not optimal, proceed to the next step.
  5. Select Entering Variable (Pivot Column): Choose the variable with the most negative coefficient (for maximization) or most positive coefficient (for minimization) in the Z-row. This column is the pivot column.
  6. Select Leaving Variable (Pivot Row): Calculate the ratio of the RHS values to the corresponding positive coefficients in the pivot column for each constraint row. Select the row with the smallest non-negative ratio. This row is the pivot row. The element at the intersection of the pivot column and pivot row is the pivot element.
  7. Perform Pivot Operation: Use elementary row operations to make the pivot element equal to 1 and all other elements in the pivot column equal to 0.
    • Divide the pivot row by the pivot element.
    • For all other rows (including the Z-row), subtract a multiple of the new pivot row to make the element in the pivot column zero. (New Row = Old Row - (Pivot Column Coefficient * New Pivot Row)).
  8. Repeat: Go back to Step 4 (Check for Optimality) with the new tableau and repeat the process until the optimal solution is found.

(a) Maximax vs. Maximin:
  • Maximax Rule:
    • An optimistic decision criterion used under uncertainty (when probabilities of states of nature are unknown).
    • Focuses on the best possible outcome for each alternative.
    • Procedure: Find the maximum payoff for each alternative, then choose the alternative with the overall maximum payoff (the "best of the best").
    • Suitable for risk-seeking decision-makers.
  • Maximin Rule:
    • A pessimistic or conservative decision criterion used under uncertainty.
    • Focuses on the worst possible outcome for each alternative.
    • Procedure: Find the minimum payoff for each alternative, then choose the alternative with the overall maximum of these minimum payoffs (the "best of the worst").
    • Suitable for risk-averse decision-makers aiming to minimize potential losses.
(b) Monte Carlo Simulation:

Monte Carlo simulation is a probabilistic simulation technique that uses repeated random sampling to obtain numerical results. It models systems with inherent uncertainty by defining inputs as probability distributions. By running the simulation many times with different random inputs drawn from these distributions, it generates a range of possible outcomes and their probabilities, allowing for analysis of risk and uncertainty. It's widely used in finance, project management, engineering, and science.

Answer

(a) Assumptions of Linear Programming:
  1. Linearity: The objective function and all constraints must be linear equations or inequalities. This means relationships between variables are directly proportional, with no exponents or cross-products.
  2. Certainty: All parameters in the model (coefficients in the objective function and constraints, resource availability) are known constants and do not change during the period being considered.
  3. Divisibility: Decision variables are continuous and can take on any non-negative fractional value. It assumes that products or resources can be divided into smaller units.
  4. Non-negativity: Decision variables must have non-negative values (≥ 0), as it's generally impossible to produce negative quantities or use negative resources.
  5. Additivity: The total value of the objective function and the total resource usage are the sum of the individual contributions from each decision variable.
*(Note: Only three assumptions were required)*
(b) Importance of Pre-testing in Surveys:

Pre-testing involves administering a questionnaire or survey instrument to a small sample of respondents similar to the target population before the main survey. Its importance includes:

  • Identifying Ambiguities: Helps uncover unclear, confusing, or poorly worded questions that respondents might misinterpret.
  • Assessing Question Flow and Length: Checks if the questionnaire flows logically and isn't too long or fatiguing for respondents.
  • Testing Instructions: Ensures instructions for respondents and interviewers (if applicable) are clear and understandable.
  • Evaluating Response Formats: Determines if the chosen response options (e.g., multiple-choice, scales) are appropriate and capture the intended information.
  • Estimating Time and Cost: Provides a realistic estimate of the time required to complete the survey, aiding in planning and budgeting.
*(Note: Only three points were required)*

(d) Confidence Interval for Proportion:

Given:

  • Sample size (n) = 400
  • Sample proportion (p̂) = 64% = 0.64
  • Confidence Level = 95%

1. Find the Z-value for 95% confidence: Zα/2 = 1.96

2. Calculate the standard error of the proportion (SEp̂):

SEp̂ = √[p̂(1-p̂) / n]

SEp̂ = √[0.64(1-0.64) / 400]

SEp̂ = √[0.64 * 0.36 / 400]

SEp̂ = √[0.2304 / 400]

SEp̂ = √0.000576 = 0.024

3. Calculate the Margin of Error (ME):

ME = Zα/2 * SEp̂

ME = 1.96 * 0.024 = 0.04704

4. Calculate the Confidence Interval (CI):

CI = p̂ ± ME

CI = 0.64 ± 0.04704

Lower Limit = 0.64 - 0.04704 = 0.59296

Upper Limit = 0.64 + 0.04704 = 0.68704

The 95% confidence interval for the population proportion of women satisfied with their marriages in Kenya is approximately (0.593, 0.687) or (59.3%, 68.7%).

Question 7

(a) Outline three ways of administering a questionnaire. (3 marks)

(b) Distinguish between primary data and secondary data as used statistics. (3 marks)

(c) A savings and credit cooperative is giving loans worth Ksh 5,000,000 to members repayable in a period of 10 years. In order to secure the loan, one must have a collateral. One member has a car which depreciates at a reducing balance rate of 9.5% p.a, while another has a piece of land which appreciates at a compound interest rate of 5.5% p.a. Determine the current minimum value of each item to enable each of the two members qualify for this loan. (8 marks)

(d) A mineral water processing plant packages water in bottles labelled as 1500 litres. The Kenya Bureau of Standards took a random sample of 256 bottles from the production line and found them to have a mean capacity of 1497 litres and a standard deviation of 24 litres. The bottles must have the correct capacity because less capacity implies exploitation of the consumer while excess capacity antagonises the factory’s objective to maximise profit. Test whether this label of 1500 litres on the bottles is true at the 5% level of significance. (6 marks)

Answer

(a) Ways of Administering a Questionnaire:
  1. Mail Survey: Questionnaires are sent and returned via postal mail. Allows for wide geographic coverage and respondent anonymity, but often suffers from low response rates and lack of clarification opportunities.
  2. Online Survey: Questionnaires are distributed and completed electronically (e.g., via email links, web forms). Cost-effective, quick data collection, allows multimedia, but can exclude those without internet access and raise concerns about sample representativeness.
  3. Face-to-Face Interview: An interviewer asks questions and records answers in person. Allows for clarification, probing, and observation of non-verbal cues, suitable for complex questions, but is expensive, time-consuming, and prone to interviewer bias.
  4. Telephone Interview: An interviewer asks questions over the phone. Quicker and cheaper than face-to-face, allows clarification, but excludes those without phones and can be perceived as intrusive.
*(Note: Only three ways were required)*
(b) Primary vs. Secondary Data:
  • Primary Data: Data collected firsthand by the researcher specifically for the current study or purpose. It is original data gathered directly from the source through methods like surveys, interviews, observations, or experiments. It is often more relevant and reliable for the specific research question but can be time-consuming and expensive to collect.
  • Secondary Data: Data that has already been collected by someone else for a different purpose but is used by the researcher for their current study. Sources include government publications, company records, academic journals, websites, and previously conducted surveys. It is generally cheaper and quicker to obtain but may not perfectly fit the research needs or be fully reliable/accurate.

(d) Hypothesis Test (Bottle Capacity):

1. Hypotheses:

  • H0: μ = 1500 litres (The label is true)
  • H1: μ ≠ 1500 litres (The label is not true - two-tailed test)

2. Significance Level: α = 5% = 0.05

3. Test Statistic (Large Sample, n=256): Z = (x̄ - μ0) / (s / √n)

  • x̄ = 1497 litres (Sample mean)
  • μ0 = 1500 litres (Hypothesized population mean)
  • s = 24 litres (Sample standard deviation, used as estimate for σ)
  • n = 256 (Sample size)

Z = (1497 - 1500) / (24 / √256)

Z = -3 / (24 / 16)

Z = -3 / 1.5 = -2.00

4. Critical Value(s): For a two-tailed test at α = 0.05, α/2 = 0.025. The critical Z values are ±1.96.

5. Decision Rule: Reject H0 if |Zcalculated| > |Zcritical|.

6. Conclusion: Since |-2.00| > |1.96|, we reject H0.

There is sufficient statistical evidence at the 5% significance level to conclude that the true mean capacity of the bottles is different from the labelled 1500 litres.

(a) Reasons to Prefer Sample Survey over Census:
  1. Cost: Conducting a census (surveying the entire population) is extremely expensive due to the large scale of data collection, processing, and personnel required. A sample survey is significantly cheaper as it involves a smaller group.
  2. Time: Collecting and analyzing data from an entire population takes considerably longer than doing so for a sample. Sample surveys provide results much faster, which is often crucial for timely decision-making.
  3. Accuracy/Quality of Data: With a smaller sample, it's possible to employ more highly trained interviewers, use more sophisticated measurement techniques, and exercise greater supervision and control over the data collection process, potentially leading to higher quality and more accurate data than a large-scale census might achieve.
  4. Feasibility: In cases where the population is very large, geographically dispersed, or difficult to reach, a census might be practically impossible to conduct. A sample survey offers a feasible alternative.
  5. Destructive Testing: If the process of measuring the item destroys it (e.g., testing the lifespan of a lightbulb), a census is impossible, and sampling is the only option.

(b) Discounting Rate vs. Discounting Factor:
  • Discounting Rate (or Discount Rate): This is the interest rate (expressed as a percentage, e.g., 10%) used to calculate the present value of future cash flows. It reflects the time value of money and the risk associated with receiving the cash flow in the future. A higher discount rate implies a higher required return or greater risk, resulting in a lower present value.
  • Discounting Factor (or Present Value Factor): This is a numerical value (less than 1) derived from the discount rate and the time period. It represents the value today of one unit of currency (e.g., $1 or Ksh 1) to be received at a specific point in the future. It is calculated as 1 / (1 + r)n, where 'r' is the discount rate per period and 'n' is the number of periods. Multiplying a future cash flow by its corresponding discount factor gives its present value.
