Sets and Set Theory

Introduction to Sets

A set is a collection of distinct objects. We may consider all the oceans in the world to be a set with the objects being whales, sea plants, sharks, octopus etc. Similarly, all the fresh water lakes in Africa can form a set.

Example

Supposing A to be a set:

A = {4, 6, 8, 13}

The objects in the set, that is, the integers 4, 6, 8 and 13 are referred to as the members or elements of the set. The elements of a set can be listed in any order. For example:

A = {4, 6, 8, 13} = {8, 4, 13, 6}

Sets are always precisely defined. Each element occurs once and only once in a set.

Set Notation

  • The symbol ∈ is used to indicate membership of a set
  • The symbol ∉ represents non-membership
  • To represent that one set is a subset of another set, we use the symbol ⊆

If A = {4, 6, 8, 13}, then:

  • 4 ∈ {4, 6, 8, 13} or 4 ∈ A
  • 16 ∉ A
  • {4, 8} ⊆ A
  • {5, 7} ⊄ A
  • A ⊆ A

Methods of Set Representation

Capital letters are normally used to represent sets. However, there are two different methods for representing members of a set:

i. The Descriptive Method

Involves the description of members of the set in such a way that one can determine the elements of the set without difficulty.

ii. The Enumerative Method

Requires that one writes out all the members of the set within the curly brackets.

For example, the set of numbers 0, 1, 2, 3, 4, 5, 6 and 7 can be represented as:

  • P = {0, 1, 2, 3, 4, 5, 6, 7} (enumerative method)
  • P = {x | x = 0, 1, 2,...7} (descriptive method)
  • P = {x | 0 ≤ x ≤ 7} where x is an integer

Types of Sets

a) Finite and Infinite Sets

A set can be classified as a finite or infinite set, depending upon the number of elements it has:

  • A finite set has a finite number of elements
  • An infinite set has an infinite number of elements

Examples:

  • P = {2, 4, 6,...20} is a finite set with ten elements
  • S = {1, 3, 5,...} is an infinite set

b) Universal Set

The term refers to the set that contains all the elements that an analyst wishes to study.

The notation U or ξ is generally used to denote universal sets.

c) The Null Set or Empty Set

This is a set which contains no elements. It is normally designated by a Greek letter Ø, or { }.

The sets Ø and {Ø} are not the same thing since the former has no elements in it, while the latter has one element in it, namely zero.

d) Equal or Equivalent Sets

Two sets C and D are said to be equal if every member of set C belongs to D and every member of set D also belongs to C.

e) Complement of a Set

The complement of set A is written as A'. This set contains all those elements of universal set which are not in A.

f) Intersection and Union

B ∩ C denotes the intersection of B and C. It is the set containing all those elements which belong to both B and C.

B ∪ C denotes the union of B and C. It is the set containing all elements that belong to either B or C or both.

If B = {5, 8, 11, 20, 25} and C = {1, 3, 5, 7, 9, 11, 13}, then:

B ∩ C = {5, 11}
B ∪ C = {1, 3, 5, 7, 8, 9, 11, 13, 20, 25}

Set Operations and Laws

A simple way of representing sets and relations between sets is by means of the Venn diagram. Venn diagram consists of a rectangle that represents the universal set. Subsets of the universal set are represented by circles drawn within the rectangle, or the universe.

Basic Venn Diagram

Suppose that the universal set is designated by U and the sets A, B and C are subsets of U:

         U
    ┌──────────────┐
    │    ┌─┐  ┌─┐  │
    │    │A│  │B│  │
    │    └─┘  └─┘  │
    │         ┌─┐  │
    │         │C│  │
    │         └─┘  │
    └──────────────┘

Examples with Set Operations

Example 1: Finding Intersection

Given:

Universal set T = {1, 2, 3, 4, 5, 6, 7, 8}
A = {3, 4, 5, 6}
B = {1, 3, 4, 7, 8}

Find the intersection of A and B

The intersection of A and B is the subset of T containing elements that belong to both A and B:

A ∩ B = {3, 4, 5, 6} ∩ {1, 3, 4, 7, 8}
      = {3, 4}

T = {0, 2, 4, 6, 8, 10, 12}

Mutually Exclusive Sets

Two sets are said to be disjointed or mutually exclusive if they have no elements in common (their intersection is empty).

For mutually exclusive sets P and R: P ∩ R = Ø

Example: Multiple Set Unions

Consider the universal set T and its subsets:

T = {a, b, c, d, e, f}
A = {a, d}
B = {b, c, f}
C = {a, c, e, f}

Find:

  1. A ∪ B
  2. A ∪ C
  3. B ∪ C
  4. A ∪ B ∪ C
i)   A ∪ B = {a, d} ∪ {b, c, f} = {a, b, c, d, f}
ii)  A ∪ C = {a, d} ∪ {a, c, e, f} = {a, c, d, e, f}
iii) B ∪ C = {b, c, f} ∪ {a, c, e, f} = {a, b, c, e, f}
iv)  A ∪ B ∪ C = {a, d} ∪ {b, c, f} ∪ {a, c, e, f} = {a, b, c, d, e, f} = T

Complement of a Set

The complement of a set A (denoted as A') contains all elements in the universal set that are not in A.

Example

For the universal set T = {1, 2, 3, 4, 5} and its subsets A = {2, 3} and B = {5}, find:

  1. A'
  2. (A')'
  3. (B')'
i)   A' = {2, 3}' = {1, 4, 5}
ii)  (A')' = ({2, 3}')' = {1, 4, 5}' = {2, 3} = A
iii) (B')' = ({5}')' = {1, 2, 3, 4}' = {5} = B

Laws of Set Algebra

From the Venn diagram where T is the universal set and A its subset, we can deduce the following laws:

i)    A ∪ Ø = A
ii)   A ∪ T = T
iii)  A ∩ A = A
iv)   A ∪ A = A
v)    A ∩ T = A
vi)   A ∪ A' = T
vii)  A ∩ A' = Ø
viii) (A')' = A

Applications

Example 1: Sports Participation

Of the 20 girls in a form:

  • 16 play hockey
  • 12 play tennis
  • 4 play basketball
  • Every girl plays at least one game
  • Two play all three games

Find how many play two and only two games.

Solution

Let's use a Venn diagram to solve this:

              H(16)
    ┌─────────────────┐
    │    x     y     │ T(12)
    │  ┌─────┬─────┐ │
    │  │     │     │ │
    │  │  2  │     │ │
    │  │     │     │ │
    │  └─────┴─────┘ │
    │        z       │
    └─────────────────┘
         B(4)

Where:

  • H = Hockey (16 players)
  • T = Tennis (12 players)
  • B = Basketball (4 players)
  • x, y, z = number playing exactly two games
  • 2 = number playing all three games

Using the diagram:

(14 - x - y) + (10 - y - z) + (2 - x - z) + 2 + x + y + z = 20
28 - x - y - z = 20
Therefore: x + y + z = 8

Answer: 8 girls play exactly two games

Example 2: Voting Election

250 members of a society voted to elect a new chairman with the following conditions:

  • Each member may vote for either one or two candidates
  • Three candidates X, Y, Z stood for election
  • 59 voted for Y only
  • 37 voted for Z only
  • 12 voted for X and Y
  • 14 voted for X and Z
  • 147 voted for either X or Y or both but not for Z
  • 102 voted for Y or Z or both but not for X

Find:

  1. How many voters did not vote
  2. How many voters voted for X only
  3. Who won the elections

Let's use a three-set Venn diagram to solve this:

              X
    ┌─────────────────┐
    │        12       │ Y
    │  P    ┌─┐      │
    │       │ │      │
    │       └─┘ Q    │
    │    14          │ 59
    │         37     │
    └─────────────────┘
         Z

Where:

  • P = number who voted for X only
  • Q = number who voted for both Y and Z

Solution Steps:

  1. Finding P and Q: 
    P + 12 + 59 = 147 ⟹ P = 76    (from X or Y or both but not Z)
Q + 59 + 37 = 102 ⟹ Q = 6     (from Y or Z or both but not X)
  2. Number who did not vote: 
    Total members = 250
Voters = P + 12 + 14 + 59 + 6 + 37
       = 76 + 12 + 14 + 59 + 6 + 37
       = 204
Did not vote = 250 - 204 = 46
  3. Total votes for each candidate: 
    X votes = 76 + 12 + 14 = 102
Y votes = 12 + 59 + 6 = 77
Z votes = 37 + 14 + 6 = 57

Conclusion

X won the election with 102 votes, compared to Y's 77 votes and Z's 57 votes.

Key Points

  • A set is a collection of distinct objects
  • Order of elements does not matter in a set
  • Elements appear only once in a set
  • Set membership is denoted using special symbols (∈, ∉, ⊆)
