Central Tendency

Measures of Central Tendency

Specific Objectives

At the end of this topic, the trainee should be able to:

  • Define the measures of central tendency
  • State the properties of the measure of central tendency
  • Determine the measures of central tendency

Introduction

Measures of central tendency show the tendency of some central value around which data tends to cluster. These are statistical values which tend to occur at the centre of any well-ordered set of data.

Important: When these measures occur, they do not always indicate the exact centre of the data.

Objectives of Averaging

  • To get one single value that describes the characteristics of the entire data
  • To facilitate comparison

Properties of Good Measures

  • Easy to understand (since statistical methods are designed to simplify complexity)
  • Simple to compute
  • Based on all observations
  • Rigidly defined with one and only one interpretation
  • Capable of further algebraic treatment
  • Have sampling stability
  • Not unduly affected by extreme values

Important Statistical Terms

i) Class Limits

Numerical values which limit the extent of a given class. All observations in a given class fall within the interval bounded by these limits.

Example: In a class 15-19, 15 and 19 are class limits

ii) Class Boundaries

Statistical boundaries that separate one class from another. Determined by:

Class Boundary = (Lower limit + Next upper limit) √∑ 2

Example: For classes 15-19 and 20-24, the boundary is (19 + 20) √∑ 2 = 19.5

iii) Class Mid-points

Values marking the center of a given class.

Mid-point = (Upper limit + Lower limit) √∑ 2

iv) Class Interval/Width

The difference between an upper class boundary and lower class boundary. Measures the length of a given class.

Types of Central Tendency Measures

  • Arithmetic mean
  • Median
  • Mode
  • Geometric mean
  • Harmonic mean

Arithmetic Mean

The arithmetic mean is the most popular and widely used measure for representing entire data with one value. It's commonly known as average or mean.

Formula

Mean = Œ£X √∑ n

Where:

  • X = values in the dataset
  • Œ£ = summation
  • n = number of observations

Example: Ungrouped Data

Find the mean of: 60, 80, 90, 120

Mean = (60 + 80 + 90 + 120) √∑ 4

= 350 √∑ 4

= 87.5

The arithmetic mean is very useful because it:

  • Represents the values of most observations in the population
  • Describes the population well in terms of magnitudes attained by most members

Merits of Arithmetic Mean

  • Possesses most characteristics of a good average
  • Utilizes all the observations given
  • Very useful statistic in terms of applications (e.g., hypothesis testing, quality control)
  • Best representative of data from a normal population
  • Can be determined accurately using mathematical formulas

Limitations of Arithmetic Mean

  • Difficult to compute without assumptions about class intervals for open-ended classes
  • May give wrong impression if data is not from a 'normal' population
  • Can give unrealistic values with discrete variables (e.g., average number of children being 4.4)
  • Can be unduly affected by extreme values

The Median

The median is a statistical value located at the center of a given set of data when organized in order of magnitude or size.

Example: Ungrouped Data

Original data: 14, 17, 9, 8, 20, 32, 18, 14.5, 13

Ordered data: 8, 9, 13, 14, 14.5, 17, 18, 20, 32

Median = 14.5 (middle number)

The median divides the data into two equal halves - equal number of observations above and below it.

Methods for Finding Median in Grouped Data

  1. Graphical method using cumulative frequency curve (ogive)
  2. Using formula method

  • Finding the value corresponding to N/2 on y-axis (where N is total frequency)

Formula Method for Median

Median = L + i/f(m-c)

Where:

  • L = Lower class boundary of the class containing the median
  • i = Class interval
  • f = Frequency of the class containing the median
  • m = (n/2)th or (n+1/2)th position
  • c = Cumulative frequency of the class before median class

Example Solution

Position of median = (n+1)/2 = (146+1)/2 = 73.5

= 60 + 7.29

= 67.29

Merits of Median

  • Not influenced by extreme values
  • Sum of deviations from median is minimum
  • Shows the true centre of data
  • Can be extended to determine quartiles
  • Easy to define
  • Can be obtained easily from cumulative frequency curve
  • Useful in determining degrees of skewness

Limitations of Median

  • Not determined by all observations
  • Not capable of algebraic treatment
  • Less reliable
  • Affected by sampling fluctuation
  • Can give imaginary values with even number of observations
  • Formula computation not easily understood by businessmen
  • Limited applications in business environment

Quartiles, Deciles, and Percentiles

These measures follow the same computational procedure as the median. For grouped data, specific formulas are used for each measure.

Quartiles

Q1 = L + i/f(m-c)

Where:

  • Q1 = First or lower quartile
  • L = Lower class limit of the lower quartile group
  • i = Class interval of the lower quartile group
  • f = Frequency of the lower quartile group
  • m = n/4 (where n is total frequency)
  • c = Cumulative frequency of the group preceding the lower quartile group

Deciles

D2 = L + i/f(m-c)

Where:

  • D2 = Second decile
  • m = 2n/10

Percentiles

P40 = L + i/f(m-c)

Where:

  • P40 = Fortieth percentile
  • m = 40n/100

Mode

Mode is defined as a value within a frequency distribution which has the highest frequency. In business activities, mode is particularly important as it helps identify items in highest demand (e.g., footwear, clothes, construction materials).

Methods for Finding Mode in Grouped Data

  1. Graphical method using histogram
  2. Computation method using formula

Sometimes a single mode may not exist. In such cases, we refer to the class with the highest frequency as the modal class.

Merits of Mode

  • Not affected by extreme values
  • Can be easily used for qualitative phenomena
  • Meaningful as an average
  • Can be determined from incomplete data if highest frequency observations are known
  • Has several applications in business
  • Easy to define
  • Can be determined easily from a graph

Limitations of Mode

  • Cannot always be computed
  • Determination can be cumbersome for large ungrouped data
  • Formula calculation unfamiliar to most business people
  • May sometimes be non-existent or have multiple modes

Harmonic Mean

Harmonic mean is used to determine the average growth rates for natural economies. It is defined as the reciprocal of the average of the reciprocals of all the values.

Formula

HM = n √∑ Œ£(1/x)

Or written as:

HM = 1/[1/n(1/x‚ÇÅ + 1/x‚ÇÇ + ... + 1/x‚Çô)]

where n is the number of values and x represents each value in the dataset

Example: Economic Growth Rates

The economic growth rates of five countries were given as 20%, 15%, 25%, 18%, and 5%

HM = 1 √∑ [1/5(1/20 + 1/15 + 1/25 + 1/10 + 1/5)]

= 1 √∑ [0.2(0.05 + 0.07 + 0.04 + 0.10 + 0.2)]

= 1 √∑ 0.092

= 10.86%

Merits and Limitations

The harmonic mean shares the same merits and limitations as the arithmetic mean.

Key Points

  • Central tendency measures help identify the central or typical value in a dataset
  • Good measures should be easy to understand and compute
  • Understanding class limits, boundaries, and intervals is crucial for data analysis
  • Different types of means serve different purposes in statistical analysis
