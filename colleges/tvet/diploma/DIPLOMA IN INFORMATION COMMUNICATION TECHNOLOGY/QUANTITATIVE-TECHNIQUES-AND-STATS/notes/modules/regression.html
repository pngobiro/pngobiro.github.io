Regression Analysis

Basic Concepts

Regression is a concept that refers to the changes which occur in the dependent variable as a result of changes in the independent variable. This analysis helps us:

  • Understand relationships between variables
  • Predict values of one variable based on another
  • Analyze the nature and strength of relationships
  • Make informed business decisions

Business Applications

Knowledge of regression is particularly valuable in business statistics where:

  • We need to analyze how changes in independent variables affect dependent variables
  • Most business activities involve relationships between multiple variables
  • Predictions of dependent variables are needed for decision making
  • Example: Predicting expenditure based on income levels

Correlation vs. Regression Analysis

Correlation Regression

Simple Regression Equation

General Form:

y = a + bx

Where:

  • y = Dependent variable
  • a = y-intercept (constant)
  • b = Slope of the line
  • x = Independent variable

This equation is determined using the "method of least squares" technique

Line of Best Fit

The regression line represents the best-fitting straight line through the data points, minimizing the distance between the actual y-values and the predicted values.

Multiple Linear Regression Models

Sometimes more than one factor influences the dependent variable.

Example: Production Cost Factors

Cost of production per week in a department depends on multiple factors:

  • Total number of hours worked
  • Raw material used during the week
  • Total number of items produced
  • Number of hours spent on repair and maintenance

Key Points:

  • Multiple factors should be used for accurate cost prediction
  • Simple scatter diagrams cannot show relationships between multiple factors
  • More complex models needed for multiple variable analysis

Non-Linear Relationships

When scatter diagrams and correlation coefficients don't indicate linear relationships, consider non-linear models.

Two Common Non-Linear Models:

  1. Exponential Model:

    y = abÀ£

    Linearization:

    • Take log of both sides: log y = log a + x log b
    • Let Y = log y, A = log a, B = log b
    • Results in linear form: Y = A + Bx
  2. Geometric Model:

    y = ax·µá

    Linearization:

    • Take log of both sides: log y = log a + b log x
    • Let Y = log y, A = log a, X = log x
    • Results in linear form: Y = A + bX

After transforming to linear form, use standard linear regression techniques (method of least squares) to determine coefficients.
