Internet Based Programming

Duration: Varies Level: Diploma

Course Overview

This module provides study notes for the Internet Based Programming course, designed for Information Technology Diploma students at Kisii National Polytechnic. It covers fundamental concepts, web programming technologies, HTML coding, web authoring tools, JavaScript and ASP, web security, and emerging trends in internet-based programming.

Course Modules

  1. Module 1: Introduction to Internet-Based Programming

    Explore the basics of internet programming, the World Wide Web, and the role of websites in organizations.

    Approx. 30 minutes

  2. Module 2: Web Programming

    Learn about web programming concepts, approaches, languages, and common web programming interfaces.

    Approx. 45 minutes

  3. Module 3: HTML Coding

    Dive into HTML coding, covering structure, tags, attributes, elements, and essential HTML tags.

    Approx. 1 hour

  4. Module 4: Web Authoring and Design Tools

    Discover web design tools, features, authoring protocols, and characteristics of good web design, including an introduction to Adobe Dreamweaver, Flash, and Fireworks.

    Approx. 1.5 hours

  5. Module 5: Java Script and Active Server Pages

    Understand Java Script and ASP, data input/output procedures, and implementation examples.

    Approx. 1 hour

  6. Module 6: Web Security

    Examine web security, issues, challenges, and security measures to protect web applications.

    Approx. 45 minutes

  7. Module 7: Emerging Trends in Internet Based Programming

    Explore emerging trends, challenges, and solutions in internet-based programming.

    Approx. 30 minutes

Prerequisites

  • Basic computer literacy
  • Understanding of internet concepts
