Structured Programming Notes

Duration: 8 hours Level: Beginner

Course Overview

This course provides a comprehensive introduction to structured programming concepts using the C programming language. It covers fundamental programming principles, program design, data structures, and file handling, equipping learners with the essential skills for software development.

  • Understand the principles of structured programming.
  • Learn the basics of the C programming language, including syntax, data types, and operators.
  • Master control structures for decision making and repetition in C programs.
  • Explore functions and subprograms for modular programming.
  • Gain knowledge of data structures like arrays, linked lists, stacks, and queues.
  • Learn file handling techniques in C for data persistence.
  • Understand the importance of software documentation.

Course Modules

  1. Module 1: Introduction to Structured Programming

    Overview of structured programming, history of programming languages, and software considerations.

    45 minutes

  2. Module 2: Program Design and Development

    Program development cycle, design concepts, and design tools like pseudo-code and flowcharts.

    60 minutes

  3. Module 3: Program Structure - Part 1

    Structure of a C program, source code files, and basic elements.

    45 minutes

  4. Module 4: Program Structure - Part 2

    C data types, variables, constants, type casting and operators (Arithmetic, Increment/Decrement).

    60 minutes

  5. Module 5: Program Structure - Part 3 & Operators

    Operators (Assignment, Relational, Logical, Conditional), Precedence and Associativity.

    45 minutes

  6. Module 6: Control Structures

    Sequence, selection (if, if-else, switch), and repetition structures (while, for, do-while).

    60 minutes

  7. Module 7: Subprograms, Arrays and Sorting

    Functions, Arrays, Declaring, Initializing and Accessing Arrays, Sorting Techniques.

    75 minutes

  8. Module 8: Advanced Data Structures and File Handling

    Searching Arrays, Linked Lists, Pointers, C Strings, Queues, Stacks, File Handling, Software Documentation.

    90 minutes

Prerequisites

  • Basic computer literacy.
  • Familiarity with basic mathematical concepts.
