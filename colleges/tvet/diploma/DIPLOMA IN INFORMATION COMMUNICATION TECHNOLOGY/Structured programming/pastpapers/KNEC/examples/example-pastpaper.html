Question 1 (15 marks)

(a) Outline the function of each of the following commands as used in a CAD program:

(i) units;

(ii) divide;

(iii) group;

(iv) break. (4 marks)

(b) State three types of mechanical components in robots which facilitate their movement in their environment. (3 marks)

(c) A geometry drawn in a CAD program could not fit on an A4 paper when printed. Outline three ways in which this drawing could be made to fit on this paper. (3 marks)

(d) The following list of balances has been extracted from the books of Kakai Moli, a sole trader, as at 30 June 2018:

Stock (1 July 2017) Ksh 300,820
Purchases Ksh 1,700,800
Sales Ksh 4,100,160
Drawings Ksh 110,000

Additional information:

  1. Stock as at 30 June 2018: Ksh 400,660
  2. Drawings of stock amounted to Ksh 36,400

Prepare the trading account for Kakai Moli for the period ended 30 June 2018. (5 marks)

Answer

(a) CAD commands functions:

(i) Units:

  • Sets the measurement units for the drawing
  • Controls precision and display format of dimensions

(ii) Divide:

  • Splits an object into equal segments
  • Creates evenly spaced points along an object

(iii) Group:

  • Combines multiple objects into a single entity
  • Allows collective manipulation of objects

(iv) Break:

  • Splits an object at a specified point
  • Creates gaps in continuous objects

(b) Robot mechanical components for movement:

  1. Actuators
    • Motors and servos
    • Controls physical movement
  2. Joints and linkages
    • Connects moving parts
    • Provides flexibility
  3. Wheels or tracks
    • Enables ground movement
    • Provides stability

(c) Making CAD drawing fit on A4:

  1. Scale the drawing
    • Adjust viewport scale
    • Use scale command
  2. Modify page setup
    • Adjust paper orientation
    • Change plot scale
  3. Use paper space layout
    • Create custom viewport
    • Adjust viewport scale

Question 2 (15 marks)

(a) The following errors and omissions have been identified in the books of Jimmy Enterprise Ltd:

  • Ksh 200,000 cash received from sale of an old motor vehicle had been correctly entered in the cash book and debited in the discount received account.
  • A cash payment of Ksh 40,000 for wages entered in the cash book had not been entered in the wages account.

Show the necessary entries in the journal to correct these errors. (4 marks)

(b) With the aid of an example in each case, distinguish between trim and extend commands as used in a CAD program. (6 marks)

(c) Phone manufacturers have created phones which allow drivers to use voice commands to activate phones while driving.

(i) State two hardware requirements for these types of phones. (2 marks)

(ii) Outline three challenges that may be faced by using voice commanded phones. (3 marks)

Answer

(a) Journal entries:

                Dr  Motor Vehicle Disposal Account    200,000
                Cr  Discount Received Account                 200,000
                (To correct wrong entry of vehicle sale)

                Dr  Wages Account                     40,000
                Cr  Suspense Account                          40,000
                (To record wages payment omitted from wages account)

(b) Trim vs. Extend commands:

Trim command:

  • Shortens objects to meet a boundary edge
  • Example: Trimming overlapping lines at intersections
  • Removes excess portions of objects

Extend command:

  • Lengthens objects to meet a boundary edge
  • Example: Extending lines to meet at a corner
  • Fills gaps between objects

(c) Voice command phones:

(i) Hardware requirements:

  1. High-quality microphone
    • Noise cancellation
    • Clear voice capture
  2. Processor capable of real-time voice processing
    • Fast response time
    • Adequate processing power

(ii) Challenges of voice commanded phones:

  1. Accuracy issues
    • Background noise interference
    • Accent recognition problems
  2. Response time delays
    • Processing lag
    • Network dependencies
  3. Limited command vocabulary
    • Restricted functionality
    • Command memorization needed

Question 3 (15 marks)

(a) Thuran Enterprises purchased office furniture on 1 July 2017 for Ksh 1,000,000. The furniture is depreciated at 20% per annum on net book value.

Show the necessary entries in the following accounts for the year ended 30th June 2018:

(i) Accumulated depreciation account

(ii) Asset account (5 marks)

(b) Describe each of the following components of a Geographic Information System:

(i) software;

(ii) data;

(iii) methods. (6 marks)

(c) Figure 2 is a drawing created by modifying Figure 1 in a computer aided design program. Outline the procedure to be followed to achieve this. (4 marks)

Answer

(a) Account entries:

                Asset Account (Office Furniture)
                2017                     Ksh        2018                    Ksh
                July 1 Bank         1,000,000      June 30 Balance c/d  1,000,000
                                   _________                            _________
                                   1,000,000                           1,000,000
                2018
                July 1 Balance b/d 1,000,000

                Accumulated Depreciation Account
                2018                     Ksh        2018                    Ksh
                June 30 Balance c/d   200,000      June 30 P&L           200,000
                                    ________                             ________
                                    200,000                              200,000

Calculation: 20% of 1,000,000 = 200,000

(b) GIS components:

(i) Software:

  • Programs for data manipulation and analysis
  • Interface for user interaction
  • Tools for spatial analysis and mapping

(ii) Data:

  • Spatial data (maps, coordinates)
  • Attribute data (descriptions, properties)
  • Metadata (data about data)

(iii) Methods:

  • Procedures for data collection
  • Analysis techniques
  • Quality control processes

(c) CAD modification procedure:

  1. Copy original drawing
  2. Use rotate command to change orientation
  3. Scale drawing if needed
  4. Apply mirror command for symmetry

May 1 2018: Bought a motor vehicle on credit from Kawe Motors Ltd for Ksh 679,000

Specify next point or [Close/Undo]: 0,25

Specify next point or [Close/Undo]: C

(4 marks)

(c) Raster vs. Vector data:

Raster Data Vector Data
Grid of pixels/cells Points, lines, polygons
Example: Satellite imagery Example: Road networks
Better for continuous data Better for discrete features
Resolution dependent Scale independent

Question 5 (15 marks)

(a) Differentiate between a cashflow statement and an income statement as used in accounting. (3 marks)

(b) Explain three application areas of computer vision as used in artificial intelligence. (6 marks)

(c) Keli Company intends to buy ABC company. Explain three indicators that Keli management should use to determine if ABC is profitable. (6 marks)

Answer

(a) Cashflow vs. Income statement:

Cashflow Statement Income Statement
Shows actual cash movements Shows profitability
Records receipts and payments Records income and expenses
Indicates liquidity position Indicates operational efficiency

(b) Computer vision applications:

  1. Medical imaging
    • Diagnostic assistance
    • Disease detection
    • Medical scan analysis
  2. Security and surveillance
    • Face recognition
    • Motion detection
    • Object tracking
  3. Quality control
    • Product inspection
    • Defect detection
    • Assembly line monitoring

(c) Profitability indicators:

  1. Financial ratios
    • Profit margin
    • Return on investment
    • Return on assets
  2. Growth trends
    • Revenue growth
    • Market share increase
    • Customer base expansion
  3. Cash flow analysis
    • Operating cash flow
    • Free cash flow
    • Working capital management

Question 6 (15 marks)

(a) State three examples of geographic information system applications (GIS) in the market. (3 marks)

(b) Explain three main parts of the robot shown in Figure 3. (6 marks)

(c) Outline a circumstance when each of the following types of search algorithms is used in artificial intelligence:

(i) Uninformed search;

(ii) Informed search. (2 marks)

(d) Outline the steps to be followed to insert text as it appears in the 3-D model shown in Figure 4. (4 marks)

Answer

(a) GIS applications in market:

  1. ArcGIS
    • Professional mapping and analysis
    • Industry standard solution
  2. QGIS
    • Open-source GIS platform
    • Free and community-supported
  3. MapInfo
    • Business mapping solution
    • Location intelligence tools

(b) Main parts of robot:

  1. Control system (head)
    • Processing unit
    • Sensors and cameras
  2. Manipulator (arms)
    • Movement mechanism
    • Grippers or tools
  3. Base/mobility system
    • Support structure
    • Movement platform

(c) Search algorithm circumstances:

(i) Uninformed search:

  • When no information about goal state is available
  • For complete exploration of search space

(ii) Informed search:

  • When heuristic information is available
  • For optimizing search efficiency

(d) Steps to insert 3D text:

  1. Select 3D text tool
    • Choose text command
    • Set text style
  2. Specify text location
    • Select insertion point
    • Set orientation
  3. Set text properties
    • Height and width
    • Depth for 3D effect
  4. Apply text formatting
    • Font and size
    • Material and texture
