CHAPTER 2: NUMBER SYSTEM AND BINARY ARITHMETIC

üî¢ Introduction to Number System

Definition: Number system is the way to represent a number in different forms.

Types of Number system:

  1. 1.Binary Number System: It is the number system with base value 2 means it has only two digits to represent the data. The digits are ( 0,1 ). E.g. $00,01,10,11,100 \ldots$.
  2. 2.Decimal Number System: It is the number system with base value 10 means it has 10digits to represent the data. The digits are(0-9). Eg. 0,1,2,3,4,5,6 $\qquad$
  3. 3.Octal Number System: It is the number system with base value 8 means it has 8 digits to represent the data. The digits are ( 0-7).
  4. 4.Hexadecimal Number System : It is the number system with base value 16 means it has 16 digits to represent the data. The digits are (0-15). Eg. 0,1,2,3......., $9, A, B, C, D, E, F$

üîÑ Conversion between Number System

To convert a decimal number into binary number system. Follow the following Steps:-

  1. Step-1:Divide the Number by 2 (as 2 is the base of the binary number system).
  2. Step-2:Collect the remainder.
  3. Step-3:Divide the quotient again with 2.
  4. Step-4:Repeat the step 2 & 3 until the quotient is 0.
  5. Step-5:Start from bottom, read the sequence of remainders upwards to the top.

To convert hexadecimal to binary umber, simply replace each hexadecimal bit with its 4 bit equivalent binary bit

i.e. - $37_{16}=0011 \ 0111_2$

$-\mathrm{C} 4_{16}=1100 \ 0100_2$

To convert a binary number to its hexadecimal equivalent, simply group the binary bits at groups of 4. if necessary, may have to add 0's to complete the groups

Note

The leading zero that is added to complete the MSB assist us in making 4 bit binary group.

The grouping of a 4 bit binary number is referred as binary coded hexadecimal
