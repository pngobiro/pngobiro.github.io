üé≤ Combinations

A combination is selection of some given elements in which order does not matter.

The number of all combinations of $n$ things, taken $r$ at a time is -

$$_{n} C_{r}=\frac{n!}{r!(n-r)!}$$

Problem 1

Find the number of subsets of the set $\{1,2,3,4,5,6\}$ having 3 elements.

Solution

The cardinality of the set is 6 and we have to choose 3 elements from the set. Here, the ordering does not matter. Hence, the number of subsets will be $6 C 3=\frac{6!}{3!(6-3)!} = \frac{6 \times 5 \times 4}{3 \times 2 \times 1} = 20$.

Problem 2

There are 6 men and 5 women in a room. In how many ways we can choose 3 men and 2 women from the room?

Solution

The number of ways to choose 3 men from 6 men is $6 C 3 = \frac{6!}{3!3!} = 20$ and the number of ways to choose 2 women from 5 women is $5 C_{2} = \frac{5!}{2!3!} = 10$

Hence, the total number of ways is - $6 C_{3} \times 5 C_{2}=20 \times 10=200$

Problem 3

How many ways can you choose 3 distinct groups of 3 students from total 9 students?

Solution

Let us number the groups as 1,2 and 3

For choosing 3 students for $1^{\text {st }}$ group, the number of ways - $9 C 3 = \frac{9!}{3!6!} = 84$

The number of ways for choosing 3 students for $2^{\text {nd }}$ group after choosing 1 st group - $6 C 3 = \frac{6!}{3!3!} = 20$

The number of ways for choosing 3 students for $3^{\text {rd }}$ group after choosing $1^{\text {st }}$ and $2^{\text {nd }}$ group - $3 C 3 = \frac{3!}{3!0!} = 1$

Hence, the total number of ways = $9 C_{3} \times 6 C_{3} \times 3 C_{3}=84 \times 20 \times 1=1680$

However, since the groups are distinct but not ordered (group 1, group 2, group 3 is the same as group 2, group 1, group 3), we must divide by the number of ways to order the groups (3!).

Corrected ways = $1680 / 3! = 1680 / 6 = 280$

üìê Pascal's Identity

Pascal's identity, first derived by Blaise Pascal in $19^{\text {th }}$ century, states that the number of ways to choose $k$ elements from $n$ elements is equal to the summation of number of ways to choose ( $k-1$ ) elements from ( $n-1$ ) elements and the number of ways to choose $k$ elements from $n-1$ elements.

Mathematically, for any positive integers k and n : ${ }_{n} C_{k}={ }_{n-1} C_{k-1}+{ }_{n-1} C_{k}$

Proof -

$$\begin{aligned} _{n-1} C_{k-1}+{ }_{n-1} C_{k} & = \frac{(n-1)!}{(k-1)!(n-1-(k-1))!} + \frac{(n-1)!}{k!(n-1-k)!} \\ & = \frac{(n-1)!}{(k-1)!(n-k)!} + \frac{(n-1)!}{k!(n-k-1)!} \\ & = \frac{(n-1)!}{(k-1)!(n-k-1)!} \left( \frac{1}{n-k} + \frac{1}{k} \right) \\ & = \frac{(n-1)!}{(k-1)!(n-k-1)!} \left( \frac{k + (n-k)}{k(n-k)} \right) \\ & = \frac{(n-1)!}{(k-1)!(n-k-1)!} \left( \frac{n}{k(n-k)} \right) \\ & = \frac{n(n-1)!}{k(k-1)!(n-k)(n-k-1)!} \\ & = \frac{n!}{k!(n-k)!} \\ & = { }_{n} C_{k} \end{aligned}$$

‚ûïn Binomial Theorem

A binomial is a polynomial with two terms

What happens when we multiply a binomial by itself ... many times?

Example: a+b

$\mathbf{a}+\mathbf{b}$ is a binomial (the two terms are $\mathbf{a}$ and $\mathbf{b}$ )

Let us multiply a+b by itself using Polynomial Multiplication :

$$(a+b)(a+b)=a^{2}+2 a b+b^{2}$$

Now take that result and multiply by $\mathbf{a}+\mathbf{b}$ again:

$$\left(a^{2}+2 a b+b^{2}\right)(a+b)=\mathbf{a}^{3}+\mathbf{3a}^{2} \mathbf{b}+\mathbf{3} \mathbf{a b}^{2}+\mathbf{b}^{3}$$

And again:

$$\left(a^{3}+3 a^{2} b+3 a b^{2}+b^{3}\right)(a+b)=\mathbf{a}^{4}+4 a^{3} b+6 \mathbf{a}^{2} \mathbf{b}^{2}+4 \mathbf{a} b^{3}+\mathbf{b}^{4}$$

The calculations get longer and longer as we go, but there is some kind of pattern developing.

That pattern is summed up by the Binomial Theorem:

$$\begin{gathered} (a+b)^{n}=\sum_{k=0}^{n}\binom{n}{k} a^{n-k} b^{k} \\ \text { The Binomial Theorem } \end{gathered}$$

Don't worry ... it will all be explained!

And you will learn lots of cool math symbols along the way.

Exponents

First, a quick summary of Exponents.

An exponent says how many times to use something in a multiplication.

Example: $8^{2}=8 \times 8=64$

An exponent of $\mathbf{1}$ means just to have it appear once, so we get the original value:

Example: $8^{1}=8$

An exponent of $\mathbf{0}$ means not to use it at all, and we have only 1:

Example: $8^{0}=1$

Exponents of (a+b)

Now on to the binomial.

We will use the simple binomial a+b, but it could be any binomial.

Let us start with an exponent of $\mathbf{0}$ and build upwards.

Exponent of 0: When an exponent is 0 , we get 1:

$$(a+b)^{0}=1$$

Exponent of 1: When the exponent is 1, we get the original value, unchanged:

$$(a+b)^{1}=a+b$$

Exponent of 2: An exponent of 2 means to multiply by itself.

$$(a+b)^{2}=(a+b)(a+b)=a^{2}+2 a b+b^{2}$$

Exponent of 3: For an exponent of 3 just multiply again:

$$(a+b)^{3}=\left(a^{2}+2 a b+b^{2}\right)(a+b)=a^{3}+3 a^{2} b+3 a b^{2}+b^{3}$$

We have enough now to start talking about the pattern.

The Pattern

In the last result we got:

$$a^{3}+3 a^{2} b+3 a b^{2}+b^{3}$$

Now, notice the exponents of a. They start at 3 and go down: $3, 2, 1, 0$ :

Likewise the exponents of $b$ go upwards: $0, 1, 2, 3$ :

If we number the terms 0 to $n$, we get this:

k=0k=1k=2k=3
$a^{3}$$a^{2}$$a$1
1$b$$b^{2}$$b^{3}$

Which can be brought together into this:

$$a^{n-k} b^{k}$$

How about an example to see how it works:

Example: When the exponent, $n$, is 3 .

The terms are:

k=0:k=1:k=2:k=3:
$a^{n-k} b^{k} = a^{3-0} b^{0} = \mathbf{a}^{3}$ $a^{n-k} b^{k} = a^{3-1} b^{1} = \mathbf{a}^{2} \mathbf{b}$ $a^{n-k} b^{k} = a^{3-2} b^{2} = \mathbf{a b}^{2}$ $a^{n-k} b^{k} = a^{3-3} b^{3} = \mathbf{b}^{3}$

It works like magic!
