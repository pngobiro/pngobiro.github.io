Question 1

(a) (i) Outline two assumptions in linear interpolation.

(ii) Define the term finite difference table as applied in numerical analysis.

(b) Differentiate between the terms iteration and interpolation as methods of mathematical estimation.

(c) (i) Define the term model as used in science and business.

(ii) Justify the need for the use of models in science and business.

(d) The data in Table 1 represents the number of students with indiscipline cases at Jitahidi High School in the year 2014.

Term 1 Term 2 Term 3
Form 1 10 16 20
Form 2 23 35 30
Form 3 30 25 40
Form 4 16 24 10

Represent the data using each of the following:

(i) Grouped bar chart;

(ii) Percentage component bar chart.

Answer

(a) (i) Assumptions in Linear Interpolation:

  • Linearity: The function being interpolated is assumed to behave linearly between the two known data points.
  • Continuity: The function is assumed to be continuous within the interval of interpolation.

(a) (ii) Finite Difference Table:

A finite difference table is a tabular arrangement of function values (y) for a set of equally spaced independent variable values (x), along with their successive differences (Δy, Δ²y, Δ³y, ...). It helps identify patterns, approximate derivatives, and construct interpolation polynomials.

(b) Iteration vs. Interpolation:

  • Iteration: A repetitive process where each step refines a previous result to approximate a solution (e.g., Newton-Raphson method).
  • Interpolation: A method to estimate unknown values that fall *between* known data points, assuming a specific relationship between them (e.g., linear interpolation).

(c) (i) Model:

A model is a simplified representation or abstraction of a real-world system, process, or object, designed to capture essential features for analysis, prediction, or understanding.

(c) (ii) Need for Models:

  • To simplify complex systems for better understanding.
  • To predict future behavior or outcomes.
  • To simulate scenarios or test hypotheses without affecting the real system.
  • To optimize processes or designs.
  • To communicate complex ideas effectively.

(b) Given the linear function defined by the general equation:

Answer

(a) General Binomial Theorem:

For any real number n and |x| < 1:

(a+b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k

Expanding this gives:

(a+b)^n = a^n + \binom{n}{1} a^{n-1}b + \binom{n}{2} a^{n-2}b^2 + ... + \binom{n}{n-1} ab^{n-1} + b^n

Where \( \binom{n}{k} = \frac{n!}{k!(n-k)!} \) is the binomial coefficient.

(b) Explanation of Constants in y = ax + b:

  • a (Slope/Gradient): Represents the rate of change of y with respect to x. It indicates how much y increases (if a > 0) or decreases (if a < 0) for a one-unit increase in x. If a = 0, the line is horizontal.
  • b (Y-intercept): Represents the value of y when x is equal to zero. It is the point where the line crosses the y-axis.

Answer

(a) Mean:

The mean (or arithmetic mean) is a measure of central tendency calculated by summing all the values in a dataset and dividing by the number of values. It represents the average value of the dataset.

(b) Binomial Expansion of (2x + 3y)⁶:

Using the binomial theorem \( (a+b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k \) with a=2x, b=3y, n=6.

\binom{6}{0} (2x)^6 (3y)^0 = 1 \times 64x^6 \times 1 = 64x^6

\binom{6}{1} (2x)^5 (3y)^1 = 6 \times 32x^5 \times 3y = 576x^5y

\binom{6}{2} (2x)^4 (3y)^2 = 15 \times 16x^4 \times 9y^2 = 2160x^4y^2

\binom{6}{3} (2x)^3 (3y)^3 = 20 \times 8x^3 \times 27y^3 = 4320x^3y^3

\binom{6}{4} (2x)^2 (3y)^4 = 15 \times 4x^2 \times 81y^4 = 4860x^2y^4

\binom{6}{5} (2x)^1 (3y)^5 = 6 \times 2x \times 243y^5 = 2916xy^5

\binom{6}{6} (2x)^0 (3y)^6 = 1 \times 1 \times 729y^6 = 729y^6

Expansion in ascending powers of x (descending powers of y):

(2x + 3y)^6 = 729y^6 + 2916xy^5 + 4860x^2y^4 + 4320x^3y^3 + 2160x^4y^2 + 576x^5y + 64x^6

Expansion in ascending powers of y (descending powers of x):

(2x + 3y)^6 = 64x^6 + 576x^5y + 2160x^4y^2 + 4320x^3y^3 + 4860x^2y^4 + 2916xy^5 + 729y^6

(The question asks for ascending powers of x, so the first expansion listed is technically correct, though usually ascending powers of the second term is implied if not specified.)

Question 4

(a) Differentiate between absolute value and positional value of a digit as used in number systems.

(b) A commercial bank uses a ten digit system to assign account numbers to its customers. The first three digits represent the branch number. The account number uses the format "0dd-ddddddd", where:

  • the first character must be the digit zero
  • the second character must be a non-zero digit
  • third character must be any numeric digit
  • the fourth character must be a non-zero digit
  • the fifth up to the tenth characters must be any numeric digits

Determine the maximum number of:

(i) branches which this system can accommodate in general;

(ii) customers which this system can accommodate in general;

(iii) customers, which this system can currently accommodate if the bank has a total of 26 branches.

(c) Explain each of the following logic gates illustrating with logical circuits and truth tables:

(i) inclusive OR gate;

(ii) AND gate;

(iii) exclusive OR gate.

Answer

(a) Absolute vs. Positional Value:

  • Absolute Value (Face Value): The inherent value of a digit itself, regardless of its position in a number (e.g., the absolute value of '7' is always seven).
  • Positional Value (Place Value): The value a digit represents based on its position within a number. This value is the digit's absolute value multiplied by the base raised to the power of its position (e.g., in 73410, the positional value of '7' is 7 * 102 = 700).

(b) Bank Account Number System:

Format: 0dd-ddddddd (3 digits for branch, 7 for customer within branch)

Branch part: 0dd

  • 1st digit: Must be 0 (1 possibility)
  • 2nd digit: Must be non-zero (1-9, so 9 possibilities)
  • 3rd digit: Any numeric digit (0-9, so 10 possibilities)

Customer part: ddddddd

  • 4th digit: Must be non-zero (1-9, so 9 possibilities)
  • 5th-10th digits: Any numeric digit (0-9, so 10 possibilities each)

(i) Maximum Branches:

The branch number is determined by the first three digits (0dd).

Number of branches = (Possibilities for 1st digit) * (Possibilities for 2nd digit) * (Possibilities for 3rd digit)

Number of branches = 1 * 9 * 10 = 90 branches

(ii) Maximum Customers (General):

This refers to the total unique account numbers possible across all potential branches.

Total customers = (Max Branches) * (Max Customers per Branch)

Max Customers per Branch (ddddddd part):

= (Possibilities for 4th) * (Possibilities for 5th) * ... * (Possibilities for 10th)

= 9 * 10 * 10 * 10 * 10 * 10 * 10 = 9 * 106 = 9,000,000

Total customers = 90 * 9,000,000 = 810,000,000 customers

(iii) Maximum Customers (26 Branches):

If the bank only uses 26 specific branch codes out of the possible 90.

Total customers = (Number of Actual Branches) * (Max Customers per Branch)

Total customers = 26 * 9,000,000 = 234,000,000 customers

(i) Inclusive OR Gate:

  • Explanation: Outputs TRUE (1) if at least one of its inputs is TRUE (1). Outputs FALSE (0) only if all inputs are FALSE (0).

  • Explanation: Outputs TRUE (1) if the inputs are different (one is TRUE, the other is FALSE). Outputs FALSE (0) if the inputs are the same (both TRUE or both FALSE).

Question 5

(a) Explain each of the following terms used in probability:

(i) independent events;

(ii) mutually exclusive events.

(b) Solve the following simultaneous equations using either elimination or substitution method:

5x - 3y = 27
7y - 2x = 24

(c) A room has a rectangular floor with a perimeter measuring 42 feet and a surface area of 108 square feet. Determine the lengths of its two sides.

(d) A soda manufacturing company analysed its past financial records and derived its revenue function given by the equation R(x) = 12x - x² - 15, and the cost function given by the equation C(x) = 15 - x, where x represents the number of crates in hundred thousands with revenue and cost in ten million Kenya shillings. Determine the following about the company:

(i) The profit function;

(ii) The break-even point(s) for the production;

(iii) The number of crates of soda that maximises profit and the corresponding maximum profit.

Answer

(a) Probability Terms:

  • (i) Independent Events: Two events are independent if the occurrence or non-occurrence of one event does not affect the probability of the other event occurring. Mathematically, P(A and B) = P(A) * P(B).
  • (ii) Mutually Exclusive Events: Two events are mutually exclusive (or disjoint) if they cannot both occur at the same time. If one event happens, the other cannot. Mathematically, P(A and B) = 0.

(b) Simultaneous Equations:

Eq 1: 5x - 3y = 27

Eq 2: 7y - 2x = 24 => -2x + 7y = 24

Using Elimination Method:

Multiply Eq 1 by 2: 10x - 6y = 54

Multiply Eq 2 by 5: -10x + 35y = 120

Add the two new equations:

(10x - 6y) + (-10x + 35y) = 54 + 120

29y = 174

y = \frac{174}{29} = 6

Substitute y = 6 into Eq 1:

5x - 3(6) = 27

5x - 18 = 27

5x = 45

x = \frac{45}{5} = 9

Solution: x = 9, y = 6

(c) Rectangular Floor Dimensions:

Let length = L, width = W.

Perimeter = 2(L + W) = 42 => L + W = 21 => L = 21 - W

Area = L * W = 108

Substitute L from the perimeter equation into the area equation:

(21 - W) * W = 108

21W - W² = 108

W² - 21W + 108 = 0

Factor the quadratic equation:

(W - 9)(W - 12) = 0

So, W = 9 or W = 12.

If W = 9, then L = 21 - 9 = 12.

If W = 12, then L = 21 - 12 = 9.

The lengths of the two sides are 9 feet and 12 feet.

(d) Soda Company Analysis:

Revenue R(x) = 12x - x² - 15

Cost C(x) = 15 - x

(x is in hundred thousands of crates, R and C are in ten million KSh)

(i) Profit Function P(x):

P(x) = R(x) - C(x)

P(x) = (12x - x² - 15) - (15 - x)

P(x) = 12x - x² - 15 - 15 + x

P(x) = -x² + 13x - 30

(ii) Break-Even Point(s):

Break-even occurs when Profit P(x) = 0 (or R(x) = C(x)).

-x² + 13x - 30 = 0

Multiply by -1: x² - 13x + 30 = 0

Factor the quadratic:

(x - 3)(x - 10) = 0

Break-even points are at x = 3 and x = 10.

(This means break-even occurs at 300,000 crates and 1,000,000 crates).

(iii) Maximum Profit:

Maximum profit occurs at the vertex of the parabolic profit function P(x) = -x² + 13x - 30.

The x-coordinate of the vertex is given by \( x = -\frac{b}{2a} \), where a = -1 and b = 13.

x = -\frac{13}{2(-1)} = -\frac{13}{-2} = 6.5

The number of crates that maximizes profit is x = 6.5 (which corresponds to 650,000 crates).

Maximum Profit P(6.5):

P(6.5) = -(6.5)² + 13(6.5) - 30

P(6.5) = -42.25 + 84.5 - 30

P(6.5) = 12.25

The maximum profit is 12.25 (which corresponds to 122.5 million KSh).

Question 6

(a) (i) Explain the use of parity bit in a computer character coding system.

(ii) Differentiate between odd parity and even parity.

(b) Perform each of the following binary arithmetic operations:

(i) 110100112 + 10101012

(ii) 111011012 - 101100112

(c) A computer security survey was conducted on data backup media by ordinary users in tertiary academic institutions. A random sample of 250 users was taken, and the results were as follows:

  • 136 users back up data on flash memory
  • 108 users back up data on compact disk
  • 90 users back up data on the Internet
  • 52 users back up data on flash memory and compact disk
  • 46 users back up data on flash memory and the Internet
  • 34 users back up data on compact disk and the Internet
  • 38 do not back up their data on any of the media

(i) Represent this information using a Venn diagram.

(ii) Determine the number of users in the study who back up data on:

I. all the three media;

II. flash memory or compact disk but not the Internet;

III. flash memory and compact disk but not the Internet.

Answer

(a) Parity Bit:

(i) Use: A parity bit is an extra bit added to a binary code (like a character code) to detect errors during data transmission or storage. It ensures that the total number of '1' bits in the code (including the parity bit itself) is either always even (even parity) or always odd (odd parity).

(ii) Odd vs. Even Parity:

  • Odd Parity: The parity bit is set to '1' if the number of '1's in the data bits is even, making the total count of '1's (including the parity bit) odd. It's set to '0' if the number of '1's in the data bits is already odd.
  • Even Parity: The parity bit is set to '1' if the number of '1's in the data bits is odd, making the total count of '1's (including the parity bit) even. It's set to '0' if the number of '1's in the data bits is already even.

(b) Binary Arithmetic:

(i) Addition: 110100112 + 10101012

   11010011
+  01010101  (Padded with leading zero)
-----------
  100101000

Result: 1001010002

(ii) Subtraction: 111011012 - 101100112

Using 2's complement for subtraction (assuming 8-bit representation for simplicity, though the numbers fit):

Number to subtract: 10110011

1's complement: 01001100

2's complement: 01001100 + 1 = 01001101

Add the 2's complement to the first number:

   11101101
+  01001101
-----------
  100111010

Discard the overflow carry bit (the leading '1').

Result: 001110102 (or simply 1110102)

(c) Data Backup Survey:

Let F = Flash, C = Compact Disk, I = Internet. Total Sample = 250.

Given:

  • |F| = 136
  • |C| = 108
  • |I| = 90
  • |F ∩ C| = 52
  • |F ∩ I| = 46
  • |C ∩ I| = 34
  • |Neither| = 38

Total who use at least one medium = Total Sample - |Neither| = 250 - 38 = 212.

Using the Principle of Inclusion-Exclusion for three sets:

|F ∪ C ∪ I| = |F| + |C| + |I| - |F ∩ C| - |F ∩ I| - |C ∩ I| + |F ∩ C ∩ I|

212 = 136 + 108 + 90 - 52 - 46 - 34 + |F ∩ C ∩ I|

212 = 334 - 132 + |F ∩ C ∩ I|

212 = 202 + |F ∩ C ∩ I|

|F ∩ C ∩ I| = 212 - 202 = 10

Now calculate regions for the Venn diagram:

  • Only F = |F| - |F∩C only| - |F∩I only| - |F∩C∩I|
    • |F∩C only| = |F ∩ C| - |F∩C∩I| = 52 - 10 = 42
    • |F∩I only| = |F ∩ I| - |F∩C∩I| = 46 - 10 = 36
    • Only F = 136 - 42 - 36 - 10 = 48
  • Only C = |C| - |F∩C only| - |C∩I only| - |F∩C∩I|
    • |C∩I only| = |C ∩ I| - |F∩C∩I| = 34 - 10 = 24
    • Only C = 108 - 42 - 24 - 10 = 32
  • Only I = |I| - |F∩I only| - |C∩I only| - |F∩C∩I|
    • Only I = 90 - 36 - 24 - 10 = 20

Check: 48 + 32 + 20 + 42 + 36 + 24 + 10 + 38 = 250. Correct.

  • Only F (F \ (C ∪ I)): 48
  • Only C (C \ (F ∪ I)): 32
  • Only I (I \ (F ∪ C)): 20
  • Outside all circles (Neither): 38

I. All three media: |F ∩ C ∩ I| = 10 users

II. Flash memory or compact disk but not the Internet:

= (Only F) + (Only C) + (F and C only)

(b) Given two matrices:

Determine the following:

(a) Questionnaire Administration Methods:

  1. Face-to-Face Interview: The enumerator asks questions directly and records the respondent's answers. Allows for clarification but can be time-consuming and prone to interviewer bias.
  2. Telephone Interview: Similar to face-to-face but conducted over the phone. Quicker and covers wider geography but lacks visual cues and may have lower response rates.
  3. Mail Questionnaire: Questionnaires are mailed to respondents who fill them out and return them. Cost-effective for large samples but often suffers from low response rates and lack of clarification.
  4. Online Questionnaire: Distributed via email or web link. Convenient, low cost, allows for complex skip patterns, but requires internet access and may exclude certain demographics.

(b) Car Fuel Consumption:

Let d = distance covered in kilometres.

(i) Models:

Car A:

Car B:
