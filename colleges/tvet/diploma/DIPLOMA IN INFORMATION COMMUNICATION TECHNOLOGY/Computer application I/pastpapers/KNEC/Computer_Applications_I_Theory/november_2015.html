COMPUTER APPLICATIONS I (THEORY)

Exam Code: 2920/102A

Duration: 2 hours

Period: November 2015

1. (a) (i) (2 marks)

Explain the function of each of the following features of a word processing software: (i) gutter margin;

Answer

A gutter margin adds extra space to the side margin or top margin of a document that will be bound. This additional space ensures that text is not obscured by the binding.

1. (a) (ii) (2 marks)

(ii) hyphenation.

Answer

Hyphenation is a feature that breaks words at the end of lines to improve text flow and readability, especially in narrow columns or justified text. It reduces excessive whitespace and creates a more uniform text appearance.

1. (b) (3 marks)

(b) Outline three advantages of using a database management system instead of the conventional computer filing system.

Answer

Reduced Data Redundancy: DBMS minimizes duplication of data by storing data in a structured and organized manner, unlike file systems where the same data may be repeated in multiple files.

Improved Data Integrity and Consistency: DBMS enforces integrity constraints and rules, ensuring data accuracy and consistency across the database. File systems lack centralized control, leading to potential inconsistencies.

Enhanced Data Sharing and Access: DBMS allows multiple users to access and share data concurrently while managing security and access controls. File systems typically have limited sharing capabilities and lack robust security features.

1. (c) (2 marks)

(c) Explain the function of macros as used in spreadsheet programs.

Answer

Macros in spreadsheet programs automate repetitive tasks by recording a sequence of actions. These recorded actions can then be replayed with a single command, saving time and ensuring consistency in operations like formatting, calculations, or data manipulation.

1. (d) (6 marks)

(d) Joel intends to design a data entry form using a database software. Explain three categories of controls he could use to achieve his objective.

Answer

Text Boxes: These controls allow users to enter alphanumeric data such as names, addresses, or descriptions. They are versatile for capturing free-form text input and can be configured for single-line or multi-line entries, with options for input masks to enforce specific data formats.

Option Buttons (Radio Buttons): Used to select one option from a predefined set of mutually exclusive choices. They are ideal for fields where only one selection is valid, such as gender or product category, ensuring data consistency and limiting input errors.

Combo Boxes (Dropdown Lists): These provide a dropdown list of predefined values, allowing users to select one option. Combo boxes are effective for fields with a limited number of valid entries, like city or country, conserving form space and ensuring data integrity by restricting input to valid choices.

2. (a) (3 marks)

2. (a) A government ICT officer has recommended the installation of wireless LAN over wired LAN. Outline three reasons that may have formed his decision.

Answer

Increased Mobility and Flexibility: Wireless LAN (WLAN) allows users to connect to the network from anywhere within the coverage area without physical cables, providing greater mobility and flexibility compared to wired LAN.

Reduced Installation Costs and Complexity: WLAN eliminates the need for extensive cabling infrastructure, reducing installation costs and complexity, especially in existing buildings or environments where running cables is difficult or expensive.

Scalability and Easier Expansion: Expanding a WLAN is generally easier and more scalable than a wired LAN. Adding new devices to a wireless network is simpler as it doesn't require running new cables, facilitating easier network expansion and adaptation to changing needs.

2. (b) (4 marks)

(b) Differentiate between smartart and clipart as used in presentation software.

Answer

SmartArt: SmartArt is a feature in presentation software that allows users to create visually appealing diagrams and charts to represent data or ideas. It offers pre-designed templates for various types of graphics like lists, processes, hierarchies, and relationships, which are customizable in terms of text, colors, and styles, making it suitable for structured information display.

Clipart: Clipart refers to a collection of pre-made images or illustrations that can be inserted into presentations to enhance visual appeal or convey specific themes. Clipart images are generally static and decorative, used to add visual interest or reinforce a message, but are not typically used for displaying structured data or complex information.

Key Differences: SmartArt is for visualizing data and structured information using customizable, dynamic graphics, while clipart is for decorative, static images to enhance visual appeal. SmartArt focuses on information representation, and clipart focuses on visual enrichment.

2. (c) (iii) (2 marks)

(iii) Suppose a user entered the formula SUMIF(B2:B5,"<25,000",A2:A5) in cell C6. State the expected output.

Answer

The expected output of the formula SUMIF(B2:B5,"<25000",A2:A5) in cell C6 would be 0.

This is because the SUMIF function is set to sum values in the range A2:A5 where the corresponding cells in B2:B5 are less than 25,000. However, the property values are in column B and commissions are in column C. The formula is incorrectly referencing the property values (Column B) for the condition and then attempting to sum the property values again (Column A) based on a commission condition that will never be met as commissions are all greater than 25,000. Since no property values meet the condition of corresponding commission being less than 25,000, the sum will be zero.

3. (a) (4 marks)

3. (a) Differentiate between magnetic and optical storage media, giving two examples in each case.

Answer

Magnetic Storage Media: Magnetic storage media stores data by magnetizing a ferromagnetic material. Data is accessed by reading and writing magnetic patterns on the surface of the media. Magnetic media is generally sensitive to magnetic fields and physical impact. Examples include Hard Disk Drives (HDDs) and Magnetic Tapes.

Optical Storage Media: Optical storage media uses lasers to read and write data on reflective surfaces. Data is encoded as variations in reflectivity on the disc surface, and read by sensing the reflection of laser light. Optical media is less susceptible to magnetic fields but can be damaged by scratches or dust. Examples include CDs, DVDs, and Blu-ray Discs.

Key Differences: Magnetic storage uses magnetic fields for data storage and retrieval, while optical storage uses lasers. Magnetic media is generally faster for read/write operations but more sensitive to magnetic interference. Optical media is more durable against magnetic fields but can be slower and more susceptible to physical damage from scratches.

3. (b) (3 marks)

(b) Outline three utility programs used in computer systems.

Answer

Antivirus Software: Utility programs designed to protect computer systems from malware, such as viruses, worms, and Trojans. They scan files, monitor system behavior, and remove or quarantine malicious software to maintain system security and integrity.

Disk Defragmentation Tools: These utilities reorganize fragmented files on a hard drive to improve file access speed and overall system performance. By placing related file parts closer together, they reduce the time it takes for the drive to read and write data.

Backup and Restore Utilities: Programs that allow users to create copies of their data for safekeeping and system recovery. They enable users to back up important files, folders, or entire systems and restore them in case of data loss or system failure, ensuring data preservation and business continuity.

3. (c) (4 marks)

(c) Joseph has been advised by his supervisor to appropriately align the text on a memo he was formatting using a word-processing software. Outline four types of alignments he could use.

Answer

Left Alignment: Text is aligned along the left margin, creating a ragged right edge. This is the most common alignment for body text as it is generally considered the easiest to read.

Center Alignment: Text is centered between the left and right margins. This alignment is often used for titles, headings, or short lines of text to create a balanced and formal appearance.

Right Alignment: Text is aligned along the right margin, creating a ragged left edge. This alignment is typically used for dates, addresses, or captions, often to position them at the side of a page or column.

Justify Alignment: Text is aligned along both the left and right margins, creating straight edges on both sides. This alignment is achieved by adjusting the spacing between words and sometimes letters, often used in formal documents and newspapers for a clean, block-like text appearance.

4. (b) (3 marks)

(b) Differentiate between portrait and landscape print options used in desktop publishing.

Answer

Portrait Orientation: In portrait orientation, the page is oriented vertically, meaning the height of the page is greater than its width. This is the standard orientation for documents like letters, reports, and essays, and is suitable for text-heavy documents.

Landscape Orientation: In landscape orientation, the page is oriented horizontally, meaning the width of the page is greater than its height. This orientation is often used for presentations, brochures, certificates, and documents containing wide tables or images, as it provides more horizontal space.

Key Differences: Portrait is vertical (taller than wide), ideal for text documents. Landscape is horizontal (wider than tall), better for visual content and wide layouts.

4. (d) (4 marks)

(d) Figure 2 shows a rehearsal toolbar in a presentation program. Use it to answer the question that follows.

Answer

(i) Previous: This button, typically represented by a left arrow, restarts the timing for the current slide. If clicked during slide rehearsal, it resets the timer for the current slide back to zero, allowing the presenter to re-time the slide.

5. (a) (i) (3 marks)

5. (a) (i) Outline three types of user interfaces provided by an operating system.

Answer

Graphical User Interface (GUI): A user-friendly interface that uses visual elements like icons, menus, and windows to interact with the operating system. Users interact by pointing and clicking with a mouse or trackpad, making it intuitive and easy to use, especially for beginners. Examples include Windows, macOS, and desktop environments in Linux.

Command Line Interface (CLI): A text-based interface where users interact with the operating system by typing commands. It provides precise control over system operations and is often preferred by advanced users and system administrators for scripting and automation. Examples include the Terminal in macOS and Linux, and Command Prompt in Windows.

Touch Interface: User interfaces designed for touch-based devices like smartphones and tablets. They use touch gestures such as taps, swipes, and pinches for interaction and are optimized for mobile use. Examples include Android and iOS interfaces.

5. (a) (ii) (3 marks)

(ii) Differentiate between tailor made and off-the-shelf programs as used in computer software.

Answer

Tailor-made (Custom) Programs: Tailor-made programs are software applications specifically developed to meet the unique requirements of a particular organization or user. They are designed from scratch, offering features and functionalities precisely aligned with the client's needs, providing a highly customized solution.

Off-the-shelf Programs: Off-the-shelf programs are pre-packaged software applications that are developed for general use and sold to the public. They cater to a broad range of users and offer standardized features, making them readily available and typically less expensive than custom software.

Key Differences: Tailor-made software is custom-built for specific needs, offering high customization but higher cost and development time. Off-the-shelf software is pre-built for general use, readily available and cheaper, but offers limited customization.

5. (d) (3 marks)

(d) Given that cells B2, B3 and B4 of a spreadsheet program contain the formulas; = D3*E6,=A$8 and =$D$5 respectively. State the formulas, as they would appear when copied into cells E2, E3 and E5 respectively.

Answer

When copied to cell E2: The formula =D3*E6 will change to =G3*H6. Relative references (D3 and E6) adjust based on the relative movement of the copy operation (3 columns to the right).

When copied to cell E3: The formula =A$8 will remain =A$8. The absolute row reference ($8) ensures that the row number does not change when copied, but the relative column reference (A) will change to D because E is 3 columns to the right of B. Therefore, it becomes =D$8.

When copied to cell E5: The formula =$D$5 will remain =$D$5. Both column ($D) and row ($5) are absolute references, so they do not change regardless of where the formula is copied.

6. (a) (2 marks)

6. (a) Outline four layout guides available in a desktop publishing program.

Answer

Margin Guides: These guides define the boundaries of the printable area on a page, setting the space between the page edges and the content. They help in maintaining consistent white space and preventing content from being too close to the page borders.

Column Guides: Column guides divide the page into vertical columns, useful for creating layouts with multiple columns of text or images, like in newspapers or magazines. They ensure consistent column widths and spacing across the document.

Row Guides: Row guides divide the page horizontally into rows, aiding in aligning content horizontally across the page. They are helpful for grid-based layouts and ensuring elements are evenly spaced vertically.

Ruler Guides: User-defined guides that can be placed anywhere on the page, both horizontally and vertically. These guides assist in precise alignment of objects and text, allowing for custom grid systems and accurate placement of elements.
