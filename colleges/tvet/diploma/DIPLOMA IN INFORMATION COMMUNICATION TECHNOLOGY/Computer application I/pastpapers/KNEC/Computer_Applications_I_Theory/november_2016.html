Exam Code: 2920/102A

Duration: 2 hours

Period: November 2016

1. (a) (2 marks)

Define each of the following terms as used in DTP programs. (i) master pages;

Answer

Master pages: These are template pages that define the layout and elements that should appear on every page of a document, ensuring consistency.

1. (a) (ii) (2 marks)

thumbnails.

Answer

Thumbnails: These are small preview images of pages or graphics, used for quick identification and navigation within a document or image collection.

1. (b) (2 marks)

Explain each of the following terms as used in spreadsheet programs. (i) what-if analysis;

Answer

What-if analysis: This is a feature that allows users to change input values in a spreadsheet model to see how they affect calculated results, enabling scenario planning and impact assessment.

1. (b) (ii) (2 marks)

template.

Answer

Template: This is a pre-designed spreadsheet file with formatting, formulas, and sometimes sample data, used as a starting point to create new spreadsheets with a consistent structure.

1. (c) (i) (3 marks)

Lucie intends to present her statistical data using charts. (i) Outline three types of charts she could use to achieve her objective.

Answer

Three types of charts Lucie could use are:

Bar Chart: Effective for comparing discrete categories of data, with bars representing the magnitude of each category.

Pie Chart: Useful for showing proportions of a whole, where each slice of the pie represents a percentage of the total.

Line Chart: Best for displaying trends over time or continuous data, showing changes and patterns in data points connected by lines.

1. (c) (ii) (4 marks)

(ii) Explain two reasons that could have led to her intention.

Answer

Two reasons for Lucie's intention could be:

Visual Clarity: Charts can present statistical data in a visually appealing and easily understandable format. Complex data sets are often difficult to interpret in tables, but charts can highlight patterns, trends, and comparisons quickly, making the information more accessible to an audience.

Enhanced Communication: Charts are powerful communication tools that can convey key insights from data more effectively than raw numbers. They help to emphasize important findings, support arguments, and engage the audience by presenting data in a compelling and memorable way, facilitating better data-driven storytelling.

2. (b) (4 marks)

Differentiate between the functions of autocorrect and autocomplete features as used in aword processing program.

Answer

Autocorrect:

Autocorrect is a feature that automatically corrects common typing errors as you type. It works by comparing typed words against a built-in dictionary and a list of predefined corrections for common misspellings, automatically replacing errors with correct spellings or predefined text. Autocorrect enhances typing accuracy by fixing mistakes in real-time without user intervention after typing.

Autocomplete:

Autocomplete, also known as word completion, predicts and suggests words or phrases as you type. It anticipates what you are about to type based on the first few characters and provides a list of suggestions to complete the word or phrase. Autocomplete speeds up typing and reduces effort by allowing users to select from suggested completions instead of typing full words or phrases manually.

Difference:

Autocorrect fixes errors automatically after they are typed, focusing on error correction. Autocomplete predicts and suggests words during typing to speed up input and reduce typing effort, focusing on efficiency and speed.

2. (c) (2 marks)

With the aid of a diagram in each case, explain the following database models. (i) network;

Answer

Network Database Model:

The network database model is an extension of the hierarchical model, allowing records to have multiple parent records and multiple child records, forming a graph-like structure. This model uses pointers to connect related records, enabling more complex relationships beyond the tree-like hierarchy. It provides flexibility in representing many-to-many relationships.

2. (c) (ii) (2 marks)

hierarchical.

Answer

Hierarchical Database Model:

The hierarchical database model organizes data in a tree-like structure, where each record has a single parent record and multiple child records. Relationships are one-to-many, forming a hierarchy from a root record down to subordinate records. Data is accessed by navigating down the hierarchy.

3. (a) (ii) (4 marks)

Ann intends to use each of the following operating system tools: - User accounts; - Device management. Explain the function of each of the system tools.

Answer

User accounts:

User accounts in an operating system manage individual access and permissions to the computer system. Each user account has unique login credentials and settings, allowing multiple users to share a computer while maintaining separate files, preferences, and security settings. User accounts enable user-specific customization and enhance system security through access control.

Device management:

Device management tools in an operating system control and configure hardware devices connected to the computer. This includes installing, updating, and troubleshooting device drivers, configuring device settings, and managing hardware resources. Device management ensures that all hardware components function correctly and communicate effectively with the operating system, enabling proper system operation and hardware utilization.

4. (b) (4 marks)

Leah has created the company's database that holds information about the employees' payroll with the following fields: employee no, employee name, age, basic salary, date of birth and department. She intends to search the following information from the database. (i) details of employees whose salaries are over 10,000 but less than 20,000;

Answer

Criteria for search (i):

To find details of employees with salaries over 10,000 but less than 20,000, Leah should use a query with the following criteria for the 'basic salary' field:

WHERE basic salary > 10000 AND basic salary < 20000

This SQL-like WHERE clause will filter records to include only employees whose basic salary is within the specified range.

4. (b) (ii) (4 marks)

(ii) details of employees whose ages are equal to or more than 45 years. State the criteria that she could use for each search to achieve her objective.

Answer

Criteria for search (ii):

To find details of employees aged 45 years or older, Leah should use a query with the following criteria for the 'age' field:

WHERE age >= 45

This SQL-like WHERE clause will filter the employee records to show only those where the age is 45 years or greater.

4. (d) (4 marks)

Lilly intends to design a brochure for her college using a DTP program. Explain two character spacing options that she could use when typing text.

Answer

Two character spacing options Lilly could use are:

Kerning: Kerning adjusts the space between specific pairs of characters to improve visual spacing and readability. It fine-tunes the space between characters that appear too close or too far apart, enhancing typographic appearance.

Tracking: Tracking (or character spacing) uniformly adjusts the spacing between all characters in a selected block of text. It increases or decreases the overall density of text, affecting the visual appearance and readability of larger text blocks, useful for fitting text into a given space or creating stylistic effects.

5. (a) (2 marks)

Explain each of the following terms as used in databases. (i) referential integrity;

Answer

Referential integrity: This is a database constraint that ensures relationships between tables remain consistent. It requires that foreign keys in one table must correspond to primary keys in a related table, preventing orphaned records and maintaining data accuracy across related tables.

5. (a) (ii) (2 marks)

relationship;

Answer

Relationship: In databases, a relationship defines how tables are linked and associated with each other. Relationships can be one-to-one, one-to-many, or many-to-many, specifying how records in different tables are logically connected and how data can be joined and accessed across tables.

5. (a) (iii) (2 marks)

normalization.

Answer

Normalization: This is the process of organizing data in a database to reduce redundancy and improve data integrity. It involves dividing larger tables into smaller tables and defining relationships between them to minimize data duplication and dependencies, making the database more efficient, flexible, and easier to maintain.

6. (b) (4 marks)

Catherine has recommended the use of a word processing program over the type-writer currently being used for processing the company's text-based documents. Outline four reasons for her recommendation.

Answer

Four reasons for Catherine's recommendation are:

Efficiency and Speed: Word processors allow for faster text entry, editing, and formatting compared to typewriters. Features like cut, copy, paste, and automated formatting tools significantly speed up document creation and modification.

Error Correction and Editing: Word processors offer easy error correction with features like spell check, grammar check, and revision tracking. Mistakes can be easily corrected without retyping entire pages, unlike typewriters where errors are permanent and require manual correction or retyping.

Formatting and Presentation: Word processors provide extensive formatting options for text and page layout, including fonts, styles, paragraph formatting, and page design. This allows for creating professional-looking documents with enhanced visual appeal, which is limited with typewriters offering basic formatting.

Storage and Retrieval: Documents created in word processors can be saved electronically, allowing for easy storage, retrieval, and sharing. Electronic files are easily backed up and can be accessed and distributed digitally, contrasting with paper documents from typewriters that require physical storage and are harder to share and manage.

6. (c) (4 marks)

With the aid of a sketch, differentiate between stroke and fill as used in DTP programs.

Answer

Stroke:

Stroke refers to the outline or border of an object in DTP programs. It defines the visual edge of shapes, text, or paths. Stroke attributes include color, thickness (weight), and style (solid, dashed, dotted). Strokes are used to define the contours and boundaries of elements, making them visually distinct.

Fill:

Fill refers to the interior area of an object, enclosed by its stroke. Fill determines the color, pattern, or gradient that occupies the space within a shape or text. Fill provides the visual substance to objects, coloring or texturing their inner regions. Objects can have a fill without a stroke, or a stroke without a fill, or both.

Difference:

Stroke is the outline defining the edge, while fill is the color or pattern occupying the area within that outline. Stroke defines the boundary, fill defines the content within the boundary.
