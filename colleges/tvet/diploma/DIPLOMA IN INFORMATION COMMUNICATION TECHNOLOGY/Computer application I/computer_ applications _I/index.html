Computer Applications I

Duration: 8 hours Level: Beginner

Course Overview

This course is designed to equip learners with fundamental knowledge and skills in computer applications. It covers essential topics from basic computer software and operating systems to practical applications like word processing, spreadsheets, databases, desktop publishing, presentations, and internet & email usage. By the end of this course, students will be able to effectively use common computer applications for academic and professional tasks.

Course Modules

  1. Module 1: Introduction to Computer Applications

    Explore the basics of computer applications, software types, and their classifications.

    60 minutes

  2. Module 2: Introduction to Operating System Environment

    Learn about operating systems, their functions, types, and command usage.

    90 minutes

  3. Module 3: Word Processing

    Master word processing skills, from basic actions to advanced formatting and document proofing.

    120 minutes

  4. Module 4: Spreadsheet

    Understand spreadsheets, basic Excel skills, formulas, functions, and data management.

    90 minutes

  5. Module 5: Database

    Introduction to databases, concepts, Microsoft Access, and database operations.

    90 minutes

  6. Module 6: Desktop Publishing

    Explore desktop publishing principles and Microsoft Publisher features.

    60 minutes

  7. Module 7: Presentations

    Learn presentation software basics and creating effective presentations with Microsoft PowerPoint.

    90 minutes

  8. Module 8: Internet and Email

    Understand internet services, browsing, and effective use of email communication.

    60 minutes

  9. Module 9: Emerging Issues and Trends

    Overview of emerging issues and trends in computer applications.

    30 minutes
