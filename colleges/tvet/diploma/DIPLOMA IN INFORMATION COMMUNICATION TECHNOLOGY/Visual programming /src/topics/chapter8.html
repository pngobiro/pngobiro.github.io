CHAPTER 8: DATA STRUCTURES

Introduction to Data Structures in Visual Basic

Data structures are organized formats for efficiently storing, accessing, and manipulating data in Visual Basic applications. They provide ways to represent complex data relationships and organize information in memory. Visual Basic supports several built-in data structures that help developers manage data effectively.

Arrays

Arrays are the most fundamental data structures in Visual Basic, allowing you to store multiple values of the same data type in a single variable.

Fixed-Size Arrays

Dim numbers(4) As Integer  ' Creates an array with 5 elements (0 to 4)
numbers(0) = 10
numbers(1) = 20
numbers(2) = 30
numbers(3) = 40
numbers(4) = 50

Dynamic Arrays

Dim dynamicArray() As String  ' Declare dynamic array
ReDim dynamicArray(10)     ' Resize array to 11 elements
ReDim Preserve dynamicArray(20)  ' Resize while preserving data

Multi-dimensional Arrays

Dim matrix(3, 3) As Integer  ' 4x4 two-dimensional array
' Accessing elements
matrix(0, 0) = 1  ' First element
matrix(3, 3) = 16 ' Last element

Collections

Collections provide more flexible ways to store and manage groups of objects compared to arrays. Visual Basic offers several collection types:

ArrayList

Dim list As New ArrayList()
list.Add("First Item")
list.Add(123)
list.Add(True)
' ArrayList can store different data types

Dictionary

Dictionaries store key-value pairs, allowing quick lookup of values using unique keys.

Dim dict As New Dictionary
dict.Add "Name", "John"
dict.Add "Age", 30
dict.Add "City", "New York"

' Accessing values
MsgBox dict("Name")  ' Displays "John"

Collection

The Collection object is a simple way to store ordered items:

Dim col As New Collection
col.Add "First"
col.Add "Second"
col.Add "Third"

' Accessing items by index
Debug.Print col(2)  ' Prints "Second"

User-Defined Types (UDT)

User-Defined Types allow you to create custom data structures that group related data together:

Type Person
    FirstName As String
    LastName As String
    Age As Integer
    Email As String
End Type

' Using the UDT
Dim employee As Person
employee.FirstName = "John"
employee.LastName = "Doe"
employee.Age = 30
employee.Email = "john.doe@example.com"

Queues and Stacks

Queue

A Queue implements First-In-First-Out (FIFO) behavior:

Dim queue As New Queue
queue.Enqueue "First"
queue.Enqueue "Second"
queue.Enqueue "Third"

' Remove and return the first item
Dim item As String = queue.Dequeue()  ' Returns "First"

Stack

A Stack implements Last-In-First-Out (LIFO) behavior:

Dim stack As New Stack
stack.Push "First"
stack.Push "Second"
stack.Push "Third"

' Remove and return the last item
Dim item As String = stack.Pop()  ' Returns "Third"

Linked Lists

Linked Lists are dynamic data structures where each element points to the next element:

Public Class Node
    Public Value As String
    Public NextNode As Node
End Class

' Creating a linked list
Dim firstNode As New Node
firstNode.Value = "First"

Dim secondNode As New Node
secondNode.Value = "Second"
firstNode.NextNode = secondNode

Dim thirdNode As New Node
thirdNode.Value = "Third"
secondNode.NextNode = thirdNode

Working with Data Structures

Iterating Through Data Structures

' Array iteration
Dim numbers(4) As Integer
For i = 0 To UBound(numbers)
    Debug.Print numbers(i)
Next i

' Collection iteration
For Each item In collection
    Debug.Print item
Next item

' Dictionary iteration
For Each key In dict.Keys
    Debug.Print key & ": " & dict(key)
Next key

Searching and Sorting

' Array sorting
Array.Sort(numbers)

' Array searching
Dim index As Integer = Array.IndexOf(numbers, searchValue)

' Collection searching
If dict.ContainsKey("Name") Then
    Debug.Print "Found: " & dict("Name")
End If
📝

Key Points

  • Arrays provide fixed and dynamic storage for elements of the same type
  • Collections offer flexible storage for different types of data
  • User-Defined Types allow creation of custom data structures
  • Queues and Stacks provide specialized data access patterns
  • Proper selection of data structure impacts program performance
