Question 1

(a) State the function of each of the following controls in a Visual Basic program.

  • (i) pointer;
  • (ii) frame;
  • (iii) ole;
  • (iv) combobox.

(b) With the aid of an example, distinguish between the Name property and the Caption property as used in a Visual Basic program.

(c) Explain the following statements used to trap errors at run time in a Visual Basic program.

  • (i) On Error Goto;
  • (ii) On Error Resume;
  • (iii) On Error Goto 0.

(d) Table I shows a list of colors that should be displayed on a form when a user enters a specified capital letter corresponding to the color. Use it to answer the question that follows.

LETTER BACKGROUND COLOUR
R RED
G GREEN
B BLUE
Y YELLOW
ANY OTHER LETTER NOT VALID COLOUR

Write a Visual Basic program that would accept the letter and change the background colour of the form to the corresponding colour when a user clicks on a command button. Use an If statement and attach the code to a command button.

Answer

(a) Functions of controls:

  • (i) Pointer: Used to select and move controls on a form during design time.
  • (ii) Frame: Used to group related controls together.
  • (iii) OLE: Used to embed objects from other applications into a Visual Basic program.
  • (iv) Combobox: A combination of a textbox and a listbox, allowing the user to either type in a value or select one from the list.

(b) Name vs Caption property:

  • Name: The internal name of the control, used to refer to it in code.
  • Caption: The text that is displayed on the control.

Example:

A command button might have a Name of "cmdOK" and a Caption of "OK".

(c) Error trapping statements:

  • (i) On Error Goto: Specifies a label to jump to when an error occurs.
  • (ii) On Error Resume Next: Continues execution with the statement following the one that caused the error.
  • (iii) On Error Goto 0: Disables error handling.

(d) Visual Basic program:


Private Sub Command1_Click()
    Dim letter As String
    letter = UCase(Text1.Text)

    If letter = "R" Then
        Form1.BackColor = vbRed
    ElseIf letter = "G" Then
        Form1.BackColor = vbGreen
    ElseIf letter = "B" Then
        Form1.BackColor = vbBlue
    ElseIf letter = "Y" Then
        Form1.BackColor = vbYellow
    Else
        MsgBox "Not a valid colour"
    End If
End Sub

(c) Write a Visual Basic program that prompts a user to enter three values into an array through an input box. The program should then arrange then display the values using a textbox control array. Use a for next statement.

Answer

(a) Functions of statements:

  • (i) Assignment: Assigns a value to a variable.
  • (ii) Executable: Performs an action, such as calculating a value or displaying output.
  • (iii) Declaration: Declares a variable, specifying its name and data type.

(b) String vs Boolean data type:

  • String: Used to store text. Example: Dim name As String = "John Doe"
  • Boolean: Used to store true/false values. Example: Dim isEligible As Boolean = True

(c) Visual Basic program:


Private Sub Command1_Click()
    Dim values(2) As String
    Dim i As Integer

    For i = 0 To 2
        values(i) = InputBox("Enter value " & i + 1)
        Text1(i).Text = values(i)
    Next i

    'Sort the array (Bubble Sort)
    Dim temp As String
    For i = 0 To 1
        For j = i + 1 To 2
            If values(i) > values(j) Then
                temp = values(i)
                values(i) = values(j)
                values(j) = temp
            End If
        Next j
    Next i

    'Display the sorted values
    For i = 0 To 2
        Text1(i).Text = values(i)
    Next i
End Sub

    Next i

Question 3

(a) Outline two advantages of a Visual Basic Programming language.

(b) (i) Describe the term data bound control form as used in Visual Basic.

(ii) Outline four types of database applications that would be linked to a program written in Visual Basic programming language.

(iii) Explain two roles of a Visual Data manager in a Visual Basic Programming language.

(c) Distinguish between a simple combo box and a drop down list combo box.

(d) Mary has developed a payroll system for Ufunzi organization. Explain three bencfits of testing the system before delivering it to the owners.

Answer

(a) Advantages of Visual Basic Programming language:

  • Rapid Application Development (RAD): Visual Basic provides a visual design environment that allows developers to quickly create user interfaces and connect them to code.
  • Ease of Use: Visual Basic is generally easier to learn and use than traditional text-based languages.

(b) (i) Data bound control form:

A data-bound control form is a form that is connected to a data source, such as a database table. The controls on the form are automatically updated with data from the data source, and changes made to the controls are automatically saved back to the data source.

(ii) Types of database applications:

  • Customer relationship management (CRM) systems
  • Inventory management systems
  • Accounting systems
  • Human resource management (HRM) systems

(iii) Roles of a Visual Data manager:

  • Creating and managing databases.
  • Creating and managing tables.

(c) Simple combo box vs Drop down list combo box:

  • Simple combo box: Allows the user to either type in a value or select one from the list.
  • Drop down list combo box: Only allows the user to select a value from the list.

(d) Benefits of testing the system:

  • To identify and fix errors before the system is released to the users.
  • To ensure that the system meets the users' needs and requirements.
  • To improve the system's reliability and performance.

Question 5

(a) (i) State the purpose of each of the following properties of a timer control in Visual Basic programming language.

I interval;

II enabled.

(ii) Explain giving a reason why it is important to set AutoSize property of a picture box to true when designing programs in Visual Basic.

(b) Distinguish between module level variable and local variable as used in a Visual Basic program.

(c) Explain three steps to be followed when building a report in a Visual Basic program.

(d) Write a Visual Basic program that would accept five characters into an array one after the other using an inputbox(). The program should then display the characters in the reverse order on a picture box. Attach the code to a command button. Use for next statement

Answer

(a) (i) Timer control properties:

  • I interval: Specifies the time interval, in milliseconds, between Timer events.
  • II enabled: Determines whether the Timer control is enabled or disabled.

(ii) Importance of setting AutoSize property to true:

Setting the AutoSize property of a picture box to true ensures that the picture box automatically adjusts its size to fit the image it contains, preventing the image from being clipped or distorted.

(b) Module level vs local variable:

  • Module level variable: Declared outside any procedure and is accessible from anywhere within the module.
  • Local variable: Declared within a procedure and is only accessible within that procedure.

(c) Steps to be followed when building a report:

  • Connect to the data source.
  • Design the report layout.
  • Add controls to the report to display the data.

(d) Visual Basic program:


Private Sub Command1_Click()
    Dim chars(4) As String
    Dim i As Integer
    Dim output As String

    For i = 0 To 4
        chars(i) = InputBox("Enter character " & i + 1)
        Next i

    For i = 4 To 0 Step -1
        output = output & chars(i)
    Next i

    Picture1.Print output
End Sub
