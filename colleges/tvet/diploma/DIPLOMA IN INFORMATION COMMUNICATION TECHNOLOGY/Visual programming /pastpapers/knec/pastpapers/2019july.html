Question 1

(a) Outline a circumstance under which each of the following controls may be used in a Visual Basic program.

  • (i) Timer;
  • (ii) Picturebox;
  • (iii) Checkbox.

(b) Explain the function of each of the following in a Visual Basic Integrated Development Environment.

  • (i) Menu bar;
  • (ii) Object browser;
  • (iii) Form layout window.

(c) Distinguish between caption and name properties as used in a Visual Basic Program.

(d) Write a program in Visual Basic that uses functions to generate acceptable remarks based on the average score that a student obtains in an examination as shown in Table 1. The program should display the remarks on a label. Use select case statement. Attach the program to a command button.

Average Score Remarks
From 0 to 299 Mean below average
From 200 to 299 Average mean
From 300 to 399 Good mean
From 400 to 499 Excellent mean
Any value Illegal entry

Answer

(a) Circumstances for control usage:

  • (i) Timer: To execute code at regular intervals, create animations, or schedule tasks.
  • (ii) Picturebox: To display graphics, act as a container for other controls, or serve as a drawing surface.
  • (iii) Checkbox: To allow users to select or deselect an option (true/false, yes/no). Multiple checkboxes can be selected simultaneously.

(b) Function of IDE elements:

  • (i) Menu bar: Provides access to commands for managing the project, editing code, debugging, and other IDE functions.
  • (ii) Object browser: Allows viewing the objects, properties, methods, events, and constants available in object libraries (like VB's own library or external components).
  • (iii) Form layout window: Shows a miniature representation of the screen and allows positioning the application's forms as they will appear at runtime.

(c) Caption vs Name properties:

  • Caption: The visible text displayed on a control (like a button, label, or form title bar). This is what the user sees.
  • Name: The internal identifier used in the code to refer to the control. This is what the programmer uses.

(d) Visual Basic program (Exam Remarks):


Private Sub Command1_Click()
    Dim averageScore As Single
    Dim remarks As String

    ' Assuming average score is obtained from a TextBox named txtAverageScore
    averageScore = Val(txtAverageScore.Text) 

    remarks = GetRemarks(averageScore)

    ' Assuming a Label named lblRemarks to display the output
    lblRemarks.Caption = remarks 
End Sub

Function GetRemarks(score As Single) As String
    Select Case score
        Case 0 To 199 ' Corrected range based on table logic
            GetRemarks = "Mean below average"
        Case 200 To 299
            GetRemarks = "Average mean"
        Case 300 To 399
            GetRemarks = "Good mean"
        Case 400 To 499 ' Assuming 499 is the max possible average
            GetRemarks = "Excellent mean"
        Case Else
            GetRemarks = "Illegal entry"
    End Select
End Function

Question 2

(a) Outline the function of each of the following properties of a form.

  • (i) borderstyle;
  • (ii) windowstate;
  • (iii) backcolour.

(b). With the aid of an example in each case, distinguish between a method and an event as used in a Visual Basic Program.

(c) The following is an error statement in a Visual Basic program. Use it to answer the question that follows:

Explain the function of the part labelled (i), (ii) and (iii) when used in a program statement.

(d) (i) Determine the output from each of the following Visual Basic logical statements.

  • I If (expression1) is true and (expression2) is true;
  • II If (expression1) is false or (expression2) is true.

(ii) Write a program in Visual Basic that accepts the number x through the use of a textbox. If the value of the expression \(x^{3}-x\) is greater or equals to 100 , the program displays the message "A big number!" on a label, otherwise output "A small number!".

Answer

(a) Form properties:

  • (i) BorderStyle: Determines the type of border and elements (like title bar, minimize/maximize buttons) the form has (e.g., Fixed Single, Sizable, None).
  • (ii) WindowState: Determines the initial state of the form when it appears (Normal, Minimized, Maximized).
  • (iii) BackColor: Sets the background color of the form.

(b) Method vs Event:

  • Method: An action that an object can perform. It's a procedure associated with an object. Example: `List1.AddItem "Apple"` (AddItem is a method of the ListBox object).
  • Event: An action or occurrence recognized by an object, often initiated by the user or the system, to which the program can respond. Example: `Command1_Click()` (Click is an event of the CommandButton object).

(c) Function of labelled parts (Error Statement Figure):

  • (i) Object: Refers to the object (e.g., control, form) that generated the error.
  • (ii) Number: A unique integer value identifying the specific runtime error that occurred.
  • (iii) Description: A string providing a textual description of the error.

(d) (i) Output of logical statements:

  • I: True
  • II: True

(ii) Visual Basic program (Big/Small Number):


Private Sub Command1_Click() ' Assuming a command button triggers the check
    Dim x As Double
    Dim result As Double

    x = Val(Text1.Text) ' Assuming Text1 is the textbox for input

    result = x ^ 3 - x

    If result >= 100 Then
        Label1.Caption = "A big number!" ' Assuming Label1 for output
    Else
        Label1.Caption = "A small number!"
    End If
End Sub

Question 3

(a) Outline three activities that can be done on the project window in a Visual Basic Program.

(b) Explain each of the following terms as used in variable declarations in a Visual Basic Program.

  • (i) lifetime:
  • (ii) scope;
  • (iii) datatype.

(c) Distinguish between variablename = InputBox ("message") and variablename = Val (InputBox ("message ")) Statement as used in Visual Basic programming.

(d) (i) Write a Visual Basic statement that would display the value 40976 formatted to each of the following:

  • I. date;
  • II. currency.

(ii) The length of the hypotenuse of a right-angle triangle is given by the formula \(c=\sqrt{a^{2}+b^{2}}\) where a and b are the perpendicular sides and c is the hypotenuse

Write a program in Visual Basic that would accept the values \(a\) and \(b\) through the use of a text box, compute the hypotenuse and display the results on a message box.

Answer

(a) Activities on the Project window:

  • View the files (forms, modules, etc.) that make up the project.
  • Add new files to the project.
  • Remove existing files from the project.
  • Open files for viewing code or design.

(b) Variable declaration terms:

  • (i) Lifetime: The period for which a variable retains its value (e.g., procedure level vs. module/static level).
  • (ii) Scope: The region of the code where a variable can be accessed (e.g., local to a procedure, module level, or global/public).
  • (iii) Datatype: Specifies the kind of data a variable can hold (e.g., Integer, String, Date, Boolean) and the amount of memory it occupies.

(c) InputBox vs Val(InputBox):

  • variablename = InputBox("message"): Assigns the user's input directly to the variable as a String data type.
  • variablename = Val(InputBox("message")): Assigns the user's input to the variable after attempting to convert it to a numeric (Double) data type using the `Val` function. If the input cannot be interpreted as a number, `Val` returns 0.

(d) (i) Formatting statements:

  • I. Date: MsgBox Format(40976, "Short Date") or MsgBox CDate(40976) (VB interprets numbers as dates based on days since 30/12/1899)
  • II. Currency: MsgBox Format(40976, "Currency")

(ii) Visual Basic program (Hypotenuse):


Private Sub Command1_Click() ' Assuming a command button triggers calculation
    Dim a As Double, b As Double, c As Double

    a = Val(Text1.Text) ' Assuming Text1 for side a
    b = Val(Text2.Text) ' Assuming Text2 for side b

    If a <= 0 Or b <= 0 Then
        MsgBox "Sides must be positive numbers.", vbExclamation
        Exit Sub
    End If

    c = Sqr(a ^ 2 + b ^ 2)

    MsgBox "The hypotenuse (c) is: " & c
End Sub

Question 4

(a) Outline four methods of a list box as used in a Visual Basic program.

(b) Distinguish between on error go to 0 and on error go to -1 statements in a Visual Basic program.

(c) Explain the reasons for applying each of the following in an array in a Visual Basic program.

  • (i) Control array;
  • (ii) Array bound;
  • (iii) Redim statement.

(d) Write a program in Visual Basic with a label and text box named labell and Textl respectively that uses a procedure to performs the following sequence of events when the command button is clicked. Use with statements.

  • TextBox 1 is disabled;
  • The background colour of TextBox 1 becomes yellow;
  • Labell becomes visible;
  • Text on the label to have a blue colour;
  • The font of text on the label to be bold.

Answer

(a) ListBox methods:

  • AddItem: Adds an item to the list.
  • RemoveItem: Removes an item from the list by its index.
  • Clear: Removes all items from the list.
  • List: Accesses an item in the list by its index.

(b) On Error GoTo 0 vs On Error GoTo -1:

  • On Error GoTo 0: Disables any active error handler in the current procedure. Subsequent errors will be fatal unless another error handler is enabled.
  • On Error GoTo -1: Clears the error object (Err object) but does not disable the current error handler. This is generally not recommended or standard practice; `Resume Next` or `Resume [Label]` are used to continue after handling an error. (Note: `On Error GoTo -1` is not standard VB error handling).

(c) Reasons for array features:

  • (i) Control array: To group multiple controls of the same type, allowing them to share event procedures and simplifying code when performing similar actions on related controls.
  • (ii) Array bound (LBound, UBound): To determine the lower and upper index limits of an array, essential for iterating through all elements without causing errors.
  • (iii) Redim statement: To create or resize a dynamic array at runtime when the exact size is not known beforehand or needs to change. `Preserve` keyword can optionally retain existing data.

(d) Visual Basic program (Modify controls):


Private Sub Command1_Click()
    Call ModifyControls
End Sub

Sub ModifyControls()
    With Text1
        .Enabled = False
        .BackColor = vbYellow
    End With

    With Label1
        .Visible = True
        .ForeColor = vbBlue
        .Font.Bold = True
    End With
End Sub

Question 5

(a) State the correct data type in Visual Basic programming for each of the following description of statements.

  • (i) The height of a student (in cm );
  • (ii) The number of children in a class;
  • (iii) The academic grade of a student;
  • (iv) Country status whether poor or rich.

(b) Ali a programmer created a Visual Basic program and ran it in his computer, he observed the error" object not found "displayed on the screen. Explain two causes of this error.

(c) Explain each of the following as used in data structures:

  • (i) bubble sort;
  • (ii) binary search;
  • (iii) Tree traversal.

(d) Write a program in Visual Basic that would display the following pattern on a picturebox:

1 2 3 4 5 6
1 2 3 4 5
1 2 3 4
1 2 3
1 2
1

Answer

(a) Data types:

  • (i) Height: Single or Double (allows decimals)
  • (ii) Number of children: Integer or Long
  • (iii) Academic grade: String (e.g., "A", "B+")
  • (iv) Country status: Boolean (True for rich, False for poor, or vice versa) or String ("Poor", "Rich")

(b) Causes of "Object not found" error:

  • Misspelled object name: Referring to a control or object using an incorrect name in the code.
  • Object not loaded/created: Trying to access properties or methods of an object (like a form) that hasn't been loaded into memory or an object variable that hasn't been instantiated (using `Set obj = New ...`).

(c) Data structure concepts:

  • (i) Bubble sort: A simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. Passes continue until no swaps are needed.
  • (ii) Binary search: An efficient search algorithm for sorted arrays. It repeatedly divides the search interval in half, comparing the middle element with the target value to determine which half to continue searching.
  • (iii) Tree traversal: The process of visiting (checking and/or updating) each node in a tree data structure exactly once. Common methods include pre-order, in-order, and post-order traversal.

(d) Visual Basic program (Pattern):


Private Sub Command1_Click() ' Assuming attached to a command button
    Dim i As Integer, j As Integer
    Dim lineStr As String

    Picture1.Cls ' Clear the picture box

    For i = 6 To 1 Step -1 ' Outer loop for rows (6 down to 1)
        lineStr = "" ' Reset string for each line
        For j = 1 To i ' Inner loop for numbers in the row
            lineStr = lineStr & j & " "
        Next j
        Picture1.Print Trim(lineStr) ' Print the line, trimming trailing space
    Next i
End Sub

Question 6

(a) Outline three ways in which a project can be started in a Visual Basic program window.

(b) Distinguish between adcmdtext and adcmdtable values as used ADO Data controls in a Visual Basic Program.

(c) Figure 1 shows a dialog box used to prompt one to choose one the three mechanisms for connecting to a database.

Explain the circumstance that may lead one to use each of the options labelled (i), (ii) and (iii).

(d) Write a program in Visual Basic that would compute the summation of integers from 20 to 50 and display the results on a label. Use a For loop statement.

Answer

(a) Ways to start a project in VB:

  • Selecting "Standard EXE" (or another project type) from the "New Project" dialog box when VB starts.
  • Choosing "File" -> "New Project" from the menu bar.
  • Opening an existing project file (.vbp).

(b) adCmdText vs adCmdTable:

  • adCmdText: Indicates that the `RecordSource` property contains an SQL query string to be executed.
  • adCmdTable: Indicates that the `RecordSource` property contains the name of a database table whose records are to be retrieved.

(c) Circumstances for connection options:

  • (i) Use Data Link File: When connection parameters (provider, server, database, credentials) are stored in an external Universal Data Link (.udl) file. Useful for easy deployment and modification without changing code.
  • (ii) Use ODBC Data Source Name: When an ODBC Data Source Name (DSN) has been pre-configured on the system through the ODBC Data Source Administrator, defining the driver and connection details for a specific database.
  • (iii) Use Connection String: When providing all necessary connection parameters directly as a string within the application code or control property. Offers flexibility but might require code changes if connection details change.

(d) Visual Basic program (Summation):


Private Sub Command1_Click() ' Assuming attached to a command button
    Dim i As Integer
    Dim sum As Long ' Use Long for potentially large sums

    sum = 0
    For i = 20 To 50
        sum = sum + i
    Next i

    Label1.Caption = "Summation from 20 to 50: " & sum ' Assuming Label1 for output
End Sub

Question 7

(a) Outline four events of an ADO Data control as used in Visual Basic Programming.

(b) Distinguish between public and local array declaration in a Visual Basic program.

(c) Describe the two properties to be set on a textbox in order to display data from a database in Visual Basic.

(d) Write a program in Visual Basic that accepts a name and a score of six students in a test through the use of an inputbox(0). The program should then display the names scores and the average scores on a picture box. Attach the code to a command button.

Answer

(a) ADO Data control events:

  • MoveComplete: Occurs after the current record pointer changes.
  • WillChangeRecord: Occurs before the current record pointer changes.
  • RecordChangeComplete: Occurs after one or more records change.
  • Error: Occurs if an error happens during a data access operation.

(b) Public vs Local array declaration:

  • Public array: Declared using the `Public` keyword, usually in a standard module. It is accessible from anywhere in the entire project (all forms and modules).
  • Local array: Declared within a specific procedure (Sub or Function) using the `Dim` keyword. It is only accessible within that procedure.

(c) Textbox properties for data display:

  • DataSource: Specifies the data control (e.g., ADO Data Control) that provides the connection to the database and the recordset.
  • DataField: Specifies the specific field (column) from the recordset (defined in the DataSource) whose value should be displayed in the textbox.

(d) Visual Basic program (Student scores):


Private Sub Command1_Click()
    Dim names(5) As String ' Array for 6 names
    Dim scores(5) As Single ' Array for 6 scores
    Dim i As Integer
    Dim totalScore As Single
    Dim averageScore As Single

    totalScore = 0
    For i = 0 To 5
        names(i) = InputBox("Enter name for student " & (i + 1) & ":")
        scores(i) = Val(InputBox("Enter score for " & names(i) & ":"))
        totalScore = totalScore + scores(i)
    Next i

    averageScore = totalScore / 6

    Picture1.Cls ' Clear picture box
    Picture1.Print "Name" & vbTab & "Score"
    For i = 0 To 5
        Picture1.Print names(i) & vbTab & scores(i)
    Next i
    Picture1.Print "Average Score: " & Format(averageScore, "0.00")
End Sub

Question 8

(a) Distinguish between a listbox and a combobox as used in Visual Basic.

(b) Write an algorithm to implement each of the following in a stack data structures:

  • (i) Check if it is full;
  • (ii) Add an element.

(c) Eric a programmer intends to create an application in Visual Basic program and would like to connect a database to bound controls on the forms. Describe two controls that would help him achieve his goal.

(d) Write a program in Visual Basic that on command button click, the program add the values from three text boxes into a database table with the fields named, \(AdNo\), StudName and StudClass respectively using the ADO command. The program should then update the database and display the message "data added" on a messagebox 0 .

Answer

(a) Listbox vs Combobox:

  • Listbox: Displays a list of items where the user can select one or multiple items. The list is always visible.
  • Combobox: Combines a textbox with a dropdown list. The user can either type a value or select one from the dropdown list.

(b) Stack algorithms:

(Assuming a fixed-size array implementation `stackArray` with size `MAX_SIZE` and a `top` pointer initialized to -1)

  • (i) Check if full (IsFull): 
    
Function IsFull()
  If top = MAX_SIZE - 1 Then
    Return True
  Else
    Return False
  End If
End Function
  • (ii) Add an element (Push): 
    
Procedure Push(item)
  If IsFull() Then
    Print "Stack Overflow"
  Else
    top = top + 1
    stackArray[top] = item
  End If
End Procedure

(c) Controls for database connection:

  • ADO Data Control (Adodc): Provides a visual way to connect to various data sources using ADO and allows binding other controls to display data.
  • DataGrid Control (or MSFlexGrid/MSHFlexGrid): Can be bound to a data control (like ADODC) to display records in a tabular format.

(d) Visual Basic program (Add data using ADO):


' Assumes an ADO Data Control named Adodc1 is on the form and connected
' Assumes TextBoxes named txtAdNo, txtStudName, txtStudClass

Private Sub Command1_Click()
    On Error GoTo AddErrorHandler

    With Adodc1.Recordset
        .AddNew ' Create a new record buffer
        .Fields("AdNo").Value = Val(txtAdNo.Text) ' Use Val() if AdNo is numeric
        .Fields("StudName").Value = txtStudName.Text
        .Fields("StudClass").Value = txtStudClass.Text
        .Update ' Save the new record to the database
    End With

    MsgBox "Data added", vbInformation ' Use vbInformation, not 0

    Exit Sub

AddErrorHandler:
    MsgBox "Error adding data: " & Err.Description, vbCritical
End Sub
