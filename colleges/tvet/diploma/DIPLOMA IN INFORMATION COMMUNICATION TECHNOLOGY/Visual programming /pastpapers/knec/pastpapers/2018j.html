Question 1

(a) State a toolbox control that would be used to perform each of the following in a Visual Basic program.

  • (i) Select an object placed on a form;
  • (ii) Hold pictures created in art packages;
  • (iii) Drawing a circle;
  • (iv) Display bar on a form.

(b) Distinguish between image and ole controls as used in a Visual Basic program.(4 marks)

(c) (i) Describe a user defined function as used in programming.

(ii) With the aid of an example in each case, explain the function of the following string function statements as used in a Visual Basic program.

  • (I) Right ();
  • (II) Ltrim();
  • (III) Mid 0 .

(d) Write a program in Visual Basic that would allow a user to enter a string through the use of an inputbox function. The program should then display; the string on a listbox control and the total number of items entered in listbox on a label. Attach the code to a command button.

Answer

(a) Toolbox controls:

  • (i) Pointer
  • (ii) PictureBox or Image
  • (iii) Shape
  • (iv) ProgressBar or ScrollBar (depending on the type of bar)

(b) Image vs OLE controls:

  • Image control: Primarily used to display pictures (BMP, JPG, GIF, etc.). It's lightweight but has fewer properties and methods compared to PictureBox.
  • OLE control: Used to embed or link objects from other applications (like Word documents, Excel spreadsheets) into the VB form. It allows interaction with the embedded object's application.

(c) (i) User-defined function:

A block of code written by the programmer to perform a specific task and return a value. It can be called from other parts of the program.

(ii) String functions:

  • (I) Right(string, length): Returns a specified number of characters from the right side of a string. Example: Right("Visual", 3) returns "ual".
  • (II) LTrim(string): Removes leading spaces from a string. Example: LTrim(" Hello") returns "Hello".
  • (III) Mid(string, start, [length]): Returns a specified number of characters from a string, starting at a specified position. Example: Mid("Visual", 2, 3) returns "isu".

(d) Visual Basic program:


Private Sub Command1_Click()
    Dim userInput As String
    userInput = InputBox("Enter a string:")

    If userInput <> "" Then
        List1.AddItem userInput
        Label1.Caption = "Total items: " & List1.ListCount
    End If
End Sub

Question 3

(a) Explain the function of menu control array as used in a menu editors,

(b) Distinguish between the drivelisthox and dirlistbox controls as used in a Visual Basic program.

(c) Write a program in Visual Basic programing language that would calculate the volume of a cuboid given that the height is 20 , length is 45 and the width is 35 . The program should then display the results on the form. Attach the code to a command button click event.

(d) Ali a programming student would like to write a program that would pass. 10 values into touction () Votum an array.

(ii) State the most appropriate iteration statement for the program giving a reason for your answer.

(iii) Incase he wants to increase the number of values at run time from 10 to 30 .State the Visual Basic statement that would help him achieve his goal.

Answer

(a) Menu control array:

A menu control array allows multiple menu items to share the same name and event procedure. Each menu item in the array is distinguished by its unique Index property. This simplifies code by allowing a single event procedure to handle clicks for related menu items.

(b) DriveListBox vs DirListBox:

  • DriveListBox: Displays a list of available disk drives on the system.
  • DirListBox: Displays a list of directories (folders) within a specified path, often linked to a DriveListBox.

(c) Visual Basic program (Cuboid volume):


Private Sub Command1_Click()
    Dim height As Integer, length As Integer, width As Integer
    Dim volume As Long

    height = 20
    length = 45
    width = 35

    volume = CalculateVolume(height, length, width)

    Form1.Print "The volume of the cuboid is: " & volume
End Sub

Function CalculateVolume(h As Integer, l As Integer, w As Integer) As Long
    CalculateVolume = h * l * w
End Function

(d) (ii) Most appropriate iteration statement:

A `For...Next` loop is most appropriate because the number of values (10) is known beforehand. It provides a clear start, end, and increment for the loop.

(iii) Statement to increase array size:

`ReDim Preserve arrayName(29)` (Assuming the array is 0-based and needs to hold 30 elements, index 0 to 29. `Preserve` keeps existing data.)

Question 4

(a) (i) Explain two reasons that could make Ali to use dynamic arrays in his program.

(ii) Explain each of the following terms as used in database connection.

  • (I) Database engine;
  • (II) Database control;
  • (III) Recordset.

(b) Distinguish between a the dimensional array and a control array as used in a Visual Basic program.

(c) Table 1 show a criteria used by a member's club to award discount to its clients. The entry fee is Kshs 1000 per head. Use it to answer the question that follows.

Write a program in Visual Basic programing language that would accept the age of a member and the status of a member through text boxes. The program should then compute the discount allowed and display the status and the discount on a form. Use an If statement.

Answer

(a) (i) Reasons to use dynamic arrays:

  • Unknown Size: When the exact number of elements needed is not known at design time and may vary during runtime.
  • Memory Efficiency: To allocate memory only when needed and release it when the array is no longer required, especially for potentially large arrays.

(ii) Database connection terms:

  • (I) Database engine: The underlying software component that manages the database, handling data storage, retrieval, and manipulation (e.g., Microsoft Jet for Access).
  • (II) Database control (e.g., Data Control, ADO Data Control): A visual control placed on a form that provides a connection to a database and allows navigation through records, often used to bind other controls to data fields.
  • (III) Recordset: An object representing a set of records retrieved from a database table or as the result of a query. It allows manipulation (viewing, adding, editing, deleting) of these records.

(b) Three-dimensional array vs Control array:

  • Three-dimensional array: A data structure that stores elements in a grid with three indices (e.g., `Dim myArray(x, y, z)`). All elements must be of the same data type.
  • Control array: A group of controls on a form that share the same name but have different Index properties. They share event procedures. The controls themselves can be of the same type (e.g., an array of TextBoxes).

(c) Visual Basic program (Club discount):


Private Sub Command1_Click()
    Dim age As Integer
    Dim status As String
    Dim discountRate As Single
    Dim discountAmount As Double
    Dim entryFee As Double

    entryFee = 1000
    age = Val(txtAge.Text) ' Assuming txtAge for age
    status = UCase(txtStatus.Text) ' Assuming txtStatus for status, convert to uppercase

    If age >= 1 And age <= 10 Then
        If status = "JUNIOR" Then
            discountRate = 0.045 ' 4.5%
        Else
            discountRate = 0 ' Or handle invalid status
        End If
    ElseIf age >= 11 And age <= 18 Then
        If status = "SENIOR" Then
            discountRate = 0.025 ' 2.5%
        Else
            discountRate = 0 ' Or handle invalid status
        End If
    ElseIf age >= 19 And age <= 30 Then
        If status = "ADULT" Then
            discountRate = 0.015 ' 1.5%
        Else
            discountRate = 0 ' Or handle invalid status
        End If
    Else ' Above 30
        Form1.Print "Status: Not allowed in the club"
        Form1.Print "Discount: N/A"
        Exit Sub ' Exit if not allowed
    End If

    discountAmount = entryFee * discountRate

    Form1.Print "Status: " & status
    Form1.Print "Discount Allowed: Ksh " & Format(discountAmount, "0.00")

End Sub

(a) Outline four characteristics of a Visual Programming language.

(b) Write the standard prefix for each of the following Visual Basic programming language data types:

  • (i) Boolean;
  • (ii) Currency;
  • (iii) Byte;
  • (iv) Object.

(c) (i) A student used Textboxes to enter two integers into a Visual Basic program in order to get their sum. When the program was executed the result generated was a concatenation of the two integers. Explain a reason that could have led to this observation.

(ii) Distinguish between static and module level variables as used in Visual Basic programming language.

(d) A programmer designed a Visual Basic interface that would capture details of a student as follows:

  • Registration Number and Name through the use text boxes.
  • Age group in years (15-17,18-20,21-23) through the use of option buttons
  • Gender (Male or Female) through the use of option buttons

When the fields are entered and a command button is clicked, the program displays the Registration Number, Name, Gender and the statement "You are under age" if the age group is \(15-17\) otherwise "You are Mature" through the use of a message box.

  • (i) Sketch the interface;
  • (ii) Write a Visual Basic program to implement the interface.

Answer

(a) Characteristics of Visual Programming language:

  • Graphical User Interface (GUI) based development.
  • Event-driven programming model.
  • Use of visual components and controls.
  • Rapid Application Development (RAD) capabilities.

(b) Standard prefixes:

  • (i) Boolean: bln
  • (ii) Currency: cur
  • (iii) Byte: byt
  • (iv) Object: obj

(c) (i) Reason for concatenation:

The program treated the values entered into the Textboxes as strings rather than numbers. The '+' operator in Visual Basic performs string concatenation if both operands are strings. The student needed to convert the Text property of the Textboxes to a numeric type (e.g., using `Val()`) before performing addition.

(ii) Static vs Module level variables:

  • Static variable: A local variable declared with the `Static` keyword. It retains its value between calls to the procedure in which it is declared. Its scope is local to the procedure.
  • Module level variable: Declared in the Declarations section of a module. Its scope can be the entire module (Private) or the entire project (Public). Its lifetime is the duration the module is loaded.

(d) (i) Interface Sketch:

(A sketch showing a form with labels and textboxes for "Registration Number" and "Name", three option buttons for age groups (15-17, 18-20, 21-23) likely within a frame, two option buttons for gender (Male, Female) likely within another frame, and a command button labelled "Submit" or similar.)

(ii) Visual Basic program:


Private Sub Command1_Click()
    Dim regNo As String, studentName As String, gender As String, ageGroupMsg As String

    regNo = txtRegNo.Text ' Assuming textbox name is txtRegNo
    studentName = txtName.Text ' Assuming textbox name is txtName

(a) (i) Outline the stage in programing when each of the following errors can be detected.

  • I. syntax errors
  • III. logical errors.

(ii) Explain the error in each of the following Visual Basic statements.

  • I. \(a / x=y / \theta\)
  • II. for ( \(x=0 ; x<10 ; x++\) ) \{ msgbox("Hello word") \}

(b) With the aid of an example, explain a way in which a public procedure can be accessed from a form in Visual Basic programming.

(c) State the Visual Basic control represented by each of the following names:

  • (i) tmrTask
  • (ii) shpTask
  • (iii) fraTask
  • (iv) drvTask.

(d) Every month, a salesperson is paid a fixed amount of Ksh 10,000 . The salesperson is paid an additional commission at the rate of \(5 \%\) of the individual sales greater or equal to Ksh 30.000 or suffers a pay cut of Ksh 1000 if the sales fall below Ksh. 30,000 . The total earning is taxed at a rate of \(15 \%\).

Write a Visual Basic program that allows a user to enter the amount of sales through the use of a text box. The program then computes the commission, the tax deducted and the net pay that the salesperson will receive. The net pay for the month should then be displayed on a label.

Answer

(a) (i) Stage when errors can be detected:

  • I. Syntax errors: Detected during compilation.
  • III. Logical errors: Detected during testing or runtime.

(ii) Errors in Visual Basic statements:

  • I. a / x=y / \theta: Invalid syntax. Division by zero is possible if x or theta is zero. Also, theta is not a valid variable name.
  • II. for ( x=0 ; x<10 ; x++ ) \{ msgbox("Hello word") \}: Invalid syntax. This is C++ syntax, not Visual Basic.

(b) Accessing a public procedure from a form:

Declare the procedure as Public in a module. Then, call the procedure from the form using the module name and procedure name. Example:


'In Module1.bas
Public Sub MyPublicProcedure()
    MsgBox "Hello from Module1!"
End Sub

'In Form1.frm
Private Sub Command1_Click()
    Module1.MyPublicProcedure ' Call the public procedure
End Sub

(c) Visual Basic controls represented by names:

  • (i) tmrTask: Timer control
  • (ii) shpTask: Shape control
  • (iii) fraTask: Frame control
  • (iv) drvTask: DriveListBox control

(d) Visual Basic program:


Private Sub Command1_Click()
    Dim sales As Double, commission As Double, tax As Double, netPay As Double, totalEarning As Double

    sales = Val(Text1.Text)

    If sales >= 30000 Then
        commission = sales * 0.05
    Else
        commission = -1000 ' Pay cut
    End If

    totalEarning = 10000 + commission
    tax = totalEarning * 0.15
    netPay = totalEarning - tax

    Label1.Caption = "Net Pay: Ksh " & netPay
End Sub

(c) Visual Basic program (Prime number check):


Private Sub Command1_Click()
    Dim num As Integer, i As Integer, isPrime As Boolean

    num = Val(InputBox("Enter an integer value:"))

    If num <= 1 Then
        MsgBox "Not a prime number"
        Exit Sub
    End If

    isPrime = True
    For i = 2 To num - 1
        If num Mod i = 0 Then
            isPrime = False
            Exit For
        End If
    Next i

    If isPrime Then
        MsgBox num & " is a prime number"
    Else
        MsgBox num & " is not a prime number"
    End If
End Sub
