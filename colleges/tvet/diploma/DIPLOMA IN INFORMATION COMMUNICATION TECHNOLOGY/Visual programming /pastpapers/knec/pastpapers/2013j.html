Question 1

(a) Explain the effect of setting MDIchild property to true in Visual programming.

(b) Assuming that the company you work for has asked you to identify a programming software to develop a payroll system. State three likely reasons that would lead you to chose Visual Programming language.

(c) Under what circumstance would each of the following errors occur in Visual Basic programming:

  • (i) syntax errors;
  • (ii) run-time errors;
  • (iii) logic errors.

(d) The three sides of a triangle having the dimensions a, b and c, must satisfy the following two conditions:

  • All the three side lengths must be positive;
  • the sum of any two side lengths must be greater than the third side length i.e. \((a+b)>c,(a+c)>b\) and \((b+c)>a\)

Write a visual basic program that captures the three sides of a triangle by way of three text boxes, test the two conditions and return an appropriate message if any of the conditions is invalid. The program should then use a function and the dimensions to calculate the area of the triangle and then display on a form,

Attach the code to a command button click event.

Hint: The area of the triangle is given by the formula:

\(\text { area }=\sqrt{s(s-a)(s-b)(s-c)} \text { where } s=\frac{a+b+c}{2}\)

Answer

(a) Setting MDIchild property to true:

Setting the MDIchild property to true makes a form a child form of an MDI (Multiple Document Interface) form. This means the form will be displayed within the MDI parent form's client area and will be closed when the MDI parent form is closed.

(b) Reasons to choose Visual Programming language:

  • Rapid Application Development (RAD): Visual programming languages provide a visual design environment that allows developers to quickly create user interfaces and connect them to code.
  • Ease of Use: Visual programming languages are generally easier to learn and use than traditional text-based languages.
  • GUI Development: Visual programming languages are well-suited for developing applications with graphical user interfaces (GUIs).

(c) Types of errors:

  • Syntax errors: Occur when the code violates the rules of the programming language.
  • Run-time errors: Occur during the execution of the program, such as division by zero or accessing an invalid memory location.
  • Logic errors: Occur when the code does not produce the expected results due to a flaw in the program's logic.

(d) Visual Basic program:


Private Sub Command1_Click()
    Dim a As Single, b As Single, c As Single, s As Single, area As Single

    a = Val(Text1.Text)
    b = Val(Text2.Text)
    c = Val(Text3.Text)

    If a <= 0 Or b <= 0 Or c <= 0 Then
        MsgBox "Side lengths must be positive."
        Exit Sub
    End If

    If (a + b <= c) Or (a + c <= b) Or (b + c <= a) Then
        MsgBox "The sum of any two side lengths must be greater than the third side length."
        Exit Sub
    End If

    s = (a + b + c) / 2
    area = Sqr(s * (s - a) * (s - b) * (s - c))

    Form1.Print "Area of the triangle: " & area
End Sub
