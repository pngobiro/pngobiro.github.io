Question 1

(a) Outline the function of each of the following tool box features in a Visual Basic program:

  • (i) pointer;
  • (ii) frame;
  • (iii) ole.

(b) Explain each of the following statements used in error trapping in a Visual Basic program:

  • (i) On Error Goto label;
  • (ii) On Error Resume Next;
  • (iii) On Error Goto 0.

(c) Explain the function of each of the following IDE interface of a Visual Basic program:

  • (i) Form Window;
  • (ii) Property Window;
  • (iii) Form Layout window.

(d) Write a Visual Basic program that would accept a character through the use of a text box. The program should then change the background colour of the current form based on the captured character colour criteria shown in Table 1. Use If...Then...Else statement and attach the code to a command button.

Character Form Colour
R Red
G Green
B Blue
Y Yellow
any other letter not valid colour

Table 1

Answer

(a) Tool box features:

  • (i) Pointer: Used to select and move controls on a form during design time. It's the default tool in the toolbox.
  • (ii) Frame: A container control that can group related controls together, providing visual organization and functional grouping.
  • (iii) OLE: Object Linking and Embedding control that allows embedding objects from other applications into a Visual Basic form.

(b) Error trapping statements:

  • (i) On Error Goto label: Enables error handling by directing program flow to a specified label when an error occurs.
  • (ii) On Error Resume Next: Specifies that when an error occurs, execution continues with the statement immediately following the statement that caused the error.
  • (iii) On Error Goto 0: Disables any enabled error handling in the current procedure and resets error handling to the default condition.

(c) IDE interface components:

  • (i) Form Window: The design surface where forms are created and modified, allowing visual placement and arrangement of controls.
  • (ii) Property Window: Displays and allows modification of the properties of the currently selected form or control.
  • (iii) Form Layout Window: Shows the position of forms on the screen at runtime, allowing adjustment of the startup position of forms.

(d) Program for changing form color:

Private Sub cmdChangeColor_Click()
    Dim userChar As String
    
    userChar = UCase(txtInput.Text)  ' Convert to uppercase
    
    Select Case userChar
        Case "R"
            Me.BackColor = vbRed
        Case "G"
            Me.BackColor = vbGreen
        Case "B"
            Me.BackColor = vbBlue
        Case "Y"
            Me.BackColor = vbYellow
        Case Else
            MsgBox "Not a valid colour character!", vbExclamation
    End Select
End Sub

Question 2

(a) State the Visual Basic type conversion functions used to convert a string into each of the following types of data:

  • (i) Boolean data;
  • (ii) Currency data;
  • (iii) Date data;
  • (iv) Integer data.

(b) State a circumstance that necessitates the use of each of the following statements in a Visual Basic program:

  • (i) Option Explicit;
  • (ii) With...End With.

(c) Distinguish between the Do While... Loop and Do... Loop While statements as used in Visual Basic program. Give a syntax in each case.

(d) The vertical height (h) in metres of a particle projected vertically upwards with an initial velocity \(100 \mathrm{~m} / \mathrm{s}\) with respect to time (t) is obtained using the formula \(h=100 t+4.9 \mathrm{t}^{2}\).

Write a Visual Basic program that prompts a user to enter the time (t) using input box function. The program should then use a function to compute the height in metres and display the results in a picture box. Attach the code to a command button.

Answer

(a) Type conversion functions:

  • (i) Boolean data: CBool(string)
  • (ii) Currency data: CCur(string)
  • (iii) Date data: CDate(string)
  • (iv) Integer data: CInt(string)

(b) Circumstances for statements:

  • (i) Option Explicit: Necessitated when you want to force explicit declaration of all variables, preventing accidental use of undeclared variables.
  • (ii) With...End With: Used to simplify code when working with a single object's properties or methods, avoiding repetitive object references.

(c) Do While...Loop vs Do...Loop While:

  • Do While...Loop: Executes the code block as long as the condition is true. The condition is checked at the beginning of the loop.
  • Do...Loop While: Executes the code block at least once, and then continues to execute as long as the condition is true. The condition is checked at the end of the loop.

Syntax:


'Do While...Loop
Do While condition
    'Code to execute
Loop

'Do...Loop While
Do
    'Code to execute
Loop While condition

Answer

(a) Form methods:

  • Show: Displays the form.
  • Hide: Hides the form.
  • Print: Prints text on the form.
  • Cls: Clears the form of any printed text or graphics.

(b) Argument vs Parameter:

  • Argument: The actual value passed to a procedure when it is called.
  • Parameter: A variable declared in the procedure's definition that receives the value of an argument.

(c) Data report controls:

  • (i) rptlabel: Used to display static text on the report.
  • (ii) rptfunction: Used to perform calculations and display the results on the report.

(d) Program for delete event:


Private Sub dtaPhonebook_Validate(Index As Integer, Cancel As Boolean)
    If dtaPhonebook.Recordset.EOF Then
        MsgBox "Record does not exist!", vbCritical, "Delete Error"
        Cancel = True
    Else
        If MsgBox("Delete this record?", vbYesNo + vbQuestion, "Confirm Delete") = vbYes Then
            dtaPhonebook.Recordset.Delete
            dtaPhonebook.Recordset.MoveNext
            If dtaPhonebook.Recordset.EOF Then dtaPhonebook.Recordset.MoveLast
            MsgBox "Record deleted!", vbInformation, "Delete Success"
        End If
    End If
End Sub

  • DataField: Specifies the field in the database to which the text box is bound.
  • DataSource: Specifies the data control that provides access to the database.

(b) Datagrid properties:

  • (i) AllowArrows: Determines whether the user can use the arrow keys to navigate the DataGrid control.
  • (ii) AllowDelete: Determines whether the user can delete rows from the DataGrid control.

  • (ii) Option base: Used to declare the default lower bound for array subscripts (either 0 or 1).
  • (iii) Preserve: Used with ReDim to retain the existing data in an array when resizing it.

(a) Outline the function of each of the following methods of a recordset as used in a Visual Basic program:

  • (i) synchronize;
  • (ii) rowstatus;
  • (iii) edit;
  • (iv) update.

(b) (i) Explain the term function call as used in Visual Basic programming language.

(ii) Classify each of the following Visual Basic operators as either Assignment Operators, or Comparison Operators or Arithmetic Operators.

(c) Explain the function of the following features in a menu editor of a Visual Basic program.

  • (i) separator bars;
  • (ii) access keys;
  • (iii) shortcut keys.

(d) Figure 2 shows a form designed in a Visual Basic program used to capture the preferred destination sites in Kenya by a tourist. Use it to answer the question that follows.

Write a Visual Basic program that uses control array check boxes for site options. When a user clicks a command button, the text "Karibu Kenya" appears as the title and selected destination sites are displayed on the list box.

Answer

(a) Recordset methods:

  • (i) Synchronize: Refreshes the data in the Recordset object from the underlying database.
  • (ii) RowStatus: Returns a value that indicates the status of a row in a Recordset object.
  • (iii) Edit: Puts the current record in edit mode, allowing changes to be made.
  • (iv) Update: Saves any changes made to the current record in the Recordset object.

(b) (i) Function call:

A function call is the act of invoking a function or procedure in a program. It involves passing control to the function, along with any required arguments, and receiving a return value (if any) after the function has executed.

(ii) Operator classification:

  • Assignment Operators: =, +=, -=, *=, /=
  • Comparison Operators: =, <>, <, >, <=, >=
  • Arithmetic Operators: +, -, *, /, \, Mod, ^

(c) Menu editor features:

  • (i) Separator bars: Used to create visual divisions between menu items, improving organization and readability.
  • (ii) Access keys: A keyboard shortcut (usually Alt + a letter) that allows users to quickly access a menu item.
  • (iii) Shortcut keys: A keyboard shortcut (e.g., Ctrl + C) that directly executes a menu item's function, bypassing the need to open the menu.

(d) Program to compute commission:


Private Sub cmdComputeCommission_Click()
    Dim sale As Single, commission As Single
    
    sale = InputBox("Enter sale amount:")
    
    Select Case sale
        Case Is < 10000
            commission = sale * 0.05
        Case Is >= 10000 And Is < 20000
            commission = sale * 0.1
        Case Is >= 20000 And Is < 30000
            commission = sale * 0.2
        Case Is >= 30000
            commission = sale * 0.3
        Case Else
            MsgBox "Invalid Value", vbCritical
            Exit Sub
    End Select
    
    MsgBox "Commission = " & commission, vbInformation
End Sub

(c) Program using dynamic array:


Private Sub cmdStoreValues_Click()
    Dim values() As Single, value As Single, count As Integer
    
    count = 0
    Do
        value = InputBox("Enter a value (negative to terminate):")
        If value < 0 Then
            ReDim Preserve values(count)
            values(count) = value
            Exit Do
        Else
            ReDim Preserve values(count)
            values(count) = value
            count = count + 1
            ReDim Preserve values(count) 'Prepare for next value
        End If
    Loop While True
    
    'Display the stored values (optional)
    Dim i As Integer
    For i = 0 To UBound(values)
        Debug.Print values(i)
    Next i
End Sub

Question 8

(a) Outline four uses of the data manager in a Visual Basic program.

(b) Distinguish between step over and step into as used in Visual Basic programming language.

(c) Explain a circumstance under which each of the following Visual Basic program features would be used:

  • (i) immediate window;
  • (ii) quick watch;
  • (iii) watch windows.

(d) The following is a run time extract of a Visual Basic program used to display cumulative sum of integers from 1 to 100. Write a Visual Basic program to display $t$ output as it appears in figure 4. Use Do...loop and use the tab function to separate columns.

Answer

(a) Uses of data manager:

  • Creating new databases.
  • Modifying existing database structures.
  • Adding, deleting, and modifying data in a database.
  • Creating and managing indexes.

(b) Step over vs Step into:

  • Step Over: Executes the next line of code, stepping over any function or procedure calls.
  • Step Into: Executes the next line of code, stepping into any function or procedure calls to debug its code.

(c) Program features usage:

  • (i) Immediate window: Used to execute single lines of code, test expressions, and inspect variable values during debugging.
  • (ii) Quick Watch: Used to quickly inspect the value of a variable or expression during debugging.
  • (iii) Watch windows: Used to monitor the values of variables and expressions over time during debugging.
