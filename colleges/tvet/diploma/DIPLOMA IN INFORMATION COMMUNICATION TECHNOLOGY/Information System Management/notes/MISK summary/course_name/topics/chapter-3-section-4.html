Decision Support Systems

🎯 Overview and Characteristics

💡 Decision Support Systems

Information systems for the management level that combine data and sophisticated analytical models on data analysis tools to support semi-structured decision making.

📊

Key Characteristics

  • 👥 Developed with manager participation to support decision-making
  • 🔄 Built to be modified to adapt to user requirements
  • 🎯 Directly supports the decision making process
  • 📈 Can project future situations using "WHAT IF" and "GOAL SEEKING" modes
  • 📊 Has extensive analytical capabilities using logical and mathematical manipulation
  • 🔄 Combines internal and external information in its databases
  • 📊 Has graphic facilities to portray decision situations

⚙️ Components of DSS

💾

Data Management System

Relies on a variety of internal and external databases. The extraction procedures are specified by a specialist rather than an end-user paying attention to data consistency across multiple support systems.

🔄

Model Management System

A model is an abstract representation that illustrates the components or relationships of a given situation. Models can be physical, mathematical, verbal, statistical, or optimization models.

Types of Models:

  • 📈 Market response models: Show how sales depends on factors such as price and promotion
  • 🔄 Simulation models: Generate input values randomly from probability distribution
  • 📊 Optimization models: Developed for resource allocation to maximize profits or minimize costs
💬

Dialogue Management System

Provides a flexible interface between the human decision maker and the computer system. Supports multiple forms of input and output, with capabilities including:

  • 💻 Natural language input specification
  • 🗣️ Voice activation
  • 🖼️ Graphics capabilities
  • 🪟 Windows functionality for multiple activities

📋 When to Use DSS

  • 📊 When there is vast amount of data to be processed
  • 🔄 When there is a large amount of computation or data manipulation with complex interrelationships
  • ❓ When there is an analysis of stages requiring the "WHAT IF" approach
  • 🧠 When human judgment is required
  • 👥 When communication is required and several people contributing expertise are involved
  • 📈 When the decision maker is comfortable with quantitative methods

🛠️ Building DSS

💡 DSS Technologies:
  • ⚙️ Specific DSS: The actual system managers work with during decision making, built using DSS generators or tools
  • 💻 DSS Generators: Software packages providing capabilities for building specific DSS (e.g., Express, PC Express, Focus, Lotus 1-2-3, MS Excel)
  • 🔧 DSS Tools: Programming languages and tools that can be used as building blocks (e.g., APL, SPSS, SAS)
🔑

Key Points to Remember

  • 📌

    DSS combines data and analytical models to support semi-structured decision making.

  • 📌

    Three main components: Data Management, Model Management, and Dialogue Management.

  • 📌

    Best used when dealing with complex data relationships and requiring extensive analysis.

  • 📌

    Can be built using various technologies from specific systems to general programming tools.
