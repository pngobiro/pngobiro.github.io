Computer Crime and Security Auditing

🚨 Computer Crime

⚠️ Definition

Computer crime refers to any illegal act involving computers as either tools or targets. Modern organizations heavily depend on systems like electronic funds transfer, ICU, and air traffic control, making them vulnerable to computer-related crimes.

🔍

Common Types of Computer Crime

  • 👤 Impersonation: Unauthorized system access by posing as legitimate users
  • 🔓 Unauthorized Access: Breaking into systems without permission
  • 💰 Financial Fraud: Using systems for monetary theft
  • 🗑️ Data Theft: Stealing sensitive information through scavenging or wire-tapping

📋 Security Auditing

💡 Purpose of Auditing

Information systems are audited to verify organizations' accounting records and financial statements, as these are now primarily managed through information systems. Both external and internal auditors perform these checks.

🔍

Audit Process

  1. 📝 Study the system and documentation
  2. 🔄 Perform compliance testing on sample transactions
  3. ✔️ Validate financial record totals
  4. 📊 Report findings to management
🎯

Types of IS Audit

  • 🛠️ System Development Audits: Ensuring controls are embedded in design
  • ⚙️ Operational Audits: Checking system controls and their implementation
  • 📱 Application Audits: Validating individual information systems
