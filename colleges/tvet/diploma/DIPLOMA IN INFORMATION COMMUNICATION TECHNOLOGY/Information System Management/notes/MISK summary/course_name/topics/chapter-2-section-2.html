Control Principles and System Types

‚öôÔ∏è Control Overview

üìã Control Process

Control is the process of ensuring that operations proceed according to plan and at the most basic level-this is done by comparing the actual results or output of the system against a target and using any differences found to adjust the input side of the system so as to bring activities in line with the target. In practice the target may be termed a norm, a budget a standard, a performance or stock level and so on.

üîÑ Types of Control Systems

‚û°Ô∏è

Open Loop Control System

This is a simple open loop control system. Its operation is very simple, when an input signal directs the control. Common examples include washing machines, light switches, gas ovens, etc.

‚ö†Ô∏è Drawback:

The drawback of an open loop control system is that it is incapable of making automatic adjustments. Even when the magnitude of the output is too big or too small, the system will not make the appropriate adjustments. For this reason, an open loop control system is not suitable for use as a complex control system.

üîÑ

Closed Loop Control System

Sometimes, we may use the output of the control system to adjust the input signal. This is called feedback. Feedback is a special feature of a closed loop control system. A closed loop control system compares the output with the expected result or command status, and then it takes appropriate control actions to adjust the input signal.

üí° Key Feature:

A closed loop system is always equipped with a sensor, which is used to monitor the output and compare it with the expected result. A well-designed system can often increase the accuracy of the output.

‚Ü©Ô∏è Feedback Types and Loops

üìù Feedback Definition

This is the gathering of information on past performance from the output of a system, department or process and using it to govern future performance by adjusting the input size of the system.

Types of Feedback

  • ‚ûï Positive Feedback: This causes a system to amplify an adjustment or an action. It acts in the same direction as the measured deviation i.e. this reinforces the way the system is moving and if thought to be beneficial. For example, if an advertisement increased the sales further advertisements may be considered.
  • ‚ûñ Negative Feedback: This is feedback seeking fluctuation around a norm or a standard. The corrective action is on the opposite side of the deviation e.g. production quantities below the plan may require an increase in labor.

Feedback Loop Components

  • üìä SENSOR: This is the measuring and recording device for example automatic metering or some kind of paper work.
  • üîÑ COMPARATOR: This is the means by which comparisons of actual results and the plan are achieved e.g. A clerk or a computer program.
  • üë• EFFECTOR: This could be a manager or supervisor acting on the report to make adjustments.

‚è© Feed Forward and Control Timing

‚ö°

Feed Forward

Where a self-regulating feedback system is not able to control a process adequately it may be feasible to use feed forward. This is where monitoring at some early stage of a system or process may indicate that an adjustment should be made at a later stage of the process, prior to the final output. Feed forward is not an automatic process and requires management intervention for it to operate successfully and, consequently, it does not have the degree of 'automatic' control inherent in a feedback system.

‚ö†Ô∏è Control Limitations:

Managers may be unable to effect control due to the following reasons:

  • ‚ùå The subject covered is outside manager's control.
  • ‚è∞ The information comes too late for effective action to be taken.
  • üìä Insufficient details are provided.
  • ‚ùì The information that is provided is in a form that isn't understood.
üîë

Key Points to Remember

  • üìå

    Control systems can be either open loop (no automatic adjustments) or closed loop (with feedback mechanisms).

  • üìå

    Feedback can be positive (amplifying) or negative (corrective) depending on the desired outcome.

  • üìå

    Feedback loops consist of sensors, comparators, and effectors working together.

  • üìå

    Feed forward requires management intervention and monitoring at early stages.
