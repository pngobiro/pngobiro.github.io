Task 1 (20 marks)

(a) (i) Open a word processing program and create the following document as it appears. Save it as beeffarming in the KNECEXAM folder to print out later.

Ngombe Bora Company

This is a company that educates farmers on how to keep good breeds of livestock. It deals with both beef and dairy production.

Beef production systems which farmers can practice include:

  • Nomadic Pastoralism
  • Ranching
  • Agro-pastoralism
  • Feedlot system

Nomadic Pastoralism: It involves a seasonal pattern of movement.

Ranching: This is a form of beef production system practiced within a defined unit of land.

Agro-Pastoralism: This is a production system where beef farming is practiced alongside crop farming.

Feedlot system: These are units where young steers are intensively put on a feeding regime purposely to fatten.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Open Excel. Enter title in A1 (merged A1:E1). Enter headers in A2:E2. Enter data A3:C8. Save as `sales.xlsx`.

(b)

  1. Total Selling Price (D3): `=B3*C3`.
  2. Drag fill handle down D3 to D8.
  3. Grand Total (D9): `=SUM(D3:D8)`. Enter label in A9.

(c)

  1. Remarks (E3): `=IF(OR(AND(A3="Beef",C3>1500),AND(A3="Milk",C3>1500),AND(A3="Hide and Skin",C3>50),AND(OR(A3="Steers",A3="Heifers",A3="Bulls"),C3>20)),"Good order","Bad order")`. Drag down E3 to E8.
  2. Right-click Sheet1 tab > Move or Copy > Create a copy > OK.

(d)

  1. Go to Sheet1 (2). Select Product names (A3:A8) and Total Selling Price (D3:D8).
  2. Insert > Charts > Pie > Pie.
  3. Add Chart Title (e.g., "Total Selling Price by Product"). Add Data Labels (e.g., Category Name, Percentage). Position chart.

(e) Save `sales.xlsx`. Prepare Sheet1 and Sheet1 (2) (with chart) for printing.
