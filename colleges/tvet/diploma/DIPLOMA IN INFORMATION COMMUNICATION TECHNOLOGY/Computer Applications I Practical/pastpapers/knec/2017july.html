Task 2 (20 marks)

Figure 1 shows a worksheet extract of Esther's termly budget. Use it to answer the questions that follow.

(a) (i) Open a spreadsheet program and key in the data in sheet 1 as it appears. Save the workbook as Ebudget in the KNECEXAM folder. (Marks implied)

(ii) Using a function and cell references only, calculate each of the following:

  1. Total expenses for each month;
  2. Total income for each month. (Marks implied)

(iii) Format all the values as numbers, using a 1000 separator. (Mark implied)

(iv) Create an embedded column chart showing Total income and Total expenses for the four months. (Mark implied)

(v) Use a function to display the value for each of the following for the three months (Jan, Feb, Mar):

  1. the highest income;
  2. the lowest expenses. (Marks implied)

(b) Figure 2 shows a worksheet extract of stock items for New Qwish Enterprise Ltd. Use it to answer the questions that follow.

(i) Enter the data as it appears in Sheet2 of the Ebudget workbook. (Marks implied)

(ii) Using an appropriate function and cell addresses only, determine the Stock Value of each item. (Stock Value = Quantity * Price per item) (Marks implied)

(iii) Assume that each item is discounted at 10%, using cell referencing only, determine the new price per item. (Marks implied)

(c) Using the subtotal function, determine the stock value of each category of items. (Marks implied)

(d) (i) Copy the contents of Sheet 2 to Sheet 3 in cell range A1:F15. (Mark implied)

(ii) Name the range as Qwishstock. (Mark implied)

(iii) Format the range named Qwishstock as follows:

  1. apply an outline border to the range;
  2. apply a 45 degrees alignment to the column headings. (Marks implied)

(e) Save the changes on sheet 3 to print out later. (Mark implied)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Esther's Budget (Sheet1):

  1. Open Microsoft Excel.
  2. In Sheet1, enter the data from Figure 1.
  3. Calculations:
    • Total Expenses (e.g., B10): `=SUM(B6:B9)`. Drag across to E10.
    • Total Income (e.g., B4): `=SUM(B2:B3)`. Drag across to E4.
  4. Formatting: Select all numerical value cells (e.g., B2:E10). Right-click > Format Cells > Number > Check 'Use 1000 Separator (.)', Decimal places: 0. Click OK.
  5. Chart: Select Month headers (B1:E1). Hold Ctrl and select Total Expenses row (B10:E10) and Total Income row (B4:E4). Go to Insert > Charts > Column > Clustered Column. Position the chart.
  6. Highest/Lowest:
    • Highest Income (e.g., in G4): `=MAX(B4:D4)`
    • Lowest Expenses (e.g., in G10): `=MIN(B10:D10)`
  7. Save workbook as `Ebudget.xlsx` in `KNECEXAM`.

(b-e) New Qwish Stock (Sheet2 & Sheet3):

  1. Go to Sheet2. Enter data from Figure 2.
  2. Stock Value (Column F, starting F2): Formula: `=C2*D2`. Drag down.
  3. New Price (Column G, starting G2): Formula: `=D2*(1-10%)`. Drag down. Format as currency if needed.
  4. Subtotals (Sheet2): Sort data by Category (Column B). Go to Data > Subtotal. 'At each change in': Category, 'Use function': Sum, 'Add subtotal to': Stock Value. Click OK.
  5. Copy to Sheet3: Select A1:F15 (adjust range if needed) on Sheet2 (with subtotals removed: Data > Subtotal > Remove All). Copy (Ctrl+C). Go to Sheet3. Paste (Ctrl+V) into A1.
  6. Name Range: Select A1:F15 on Sheet3. In the Name Box (left of formula bar), type `Qwishstock` and press Enter.
  7. Formatting (Qwishstock range on Sheet3):
    • Select the range `Qwishstock`. Right-click > Format Cells > Border > Outline. Click OK.
    • Select the header row (A1:F1). Right-click > Format Cells > Alignment > Orientation: 45 degrees. Click OK.
  8. Save the `Ebudget.xlsx` workbook. Prepare Sheet3 for printing.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Database, Tables, Relationships:

  1. Open MS Access. Create Blank Database named `XYZ.accdb` in `KNECEXAM`.
  2. Create Tables (Design View):
    • tblvehicle: Vehicle_RegNo (Text 10, PK), Vehicle_Type (Text 15), Vehicle_Make (Text 20), Vehicle_Price (Currency). Save.
    • customertbl: Customer_ID (Text 10, PK), Customer_Name (Text 50), Customer_Address (Number, Integer), Customer_Town (Text 25), Vehicle_RegNo (Text 10 - *FK*). Save.
    • tblpayment: PaymentID (AutoNumber, PK), Customer_Name (Text 50), Vehicle_RegNo (Text 10 - *FK*), Vehicle_Price (Currency), Amount_Paid (Currency). Save. *(Note: Added PK. Ideally, this table should link via Customer_ID, not name)*.
  3. Relationships: Database Tools > Relationships. Add `tblvehicle` and `customertbl`. Drag `Vehicle_RegNo` from `tblvehicle` to `Vehicle_RegNo` in `customertbl`. Enforce Integrity. Create. *(Relationship with tblpayment is ambiguous due to using Customer Name instead of ID)*. Close & Save.

(b) Forms and Data Entry:

  1. Select `tblvehicle` > Create > Form. Save as `frmvehicle`.
  2. Select `customertbl` > Create > Form. Save as `frmcustomer`.
  3. Select `tblpayment` > Create > Form. Save as `frmpayment`.
  4. Open each form and enter the respective data.

(b.iii) Query (qrybalance):

  1. Create > Query Design. Add `tblpayment`.
  2. Add fields: `Customer Name`, `Vehicle_RegNo`, `Vehicle_Price`, `Amount_Paid`.
  3. In a new field column, type: `Balance: [Vehicle_Price]-[Amount_Paid]`.
  4. Run to test. Save as `qrybalance`.

(b.iv) Report (rptcleared):

  1. Select `qrybalance` query. Create > Report Wizard.
  2. Add fields: `Customer Name`, `Vehicle_RegNo`, `Balance`. Click Next.
  3. No grouping/sorting. Click Next twice.
  4. Choose layout. Title: `rptcleared`. Finish.
  5. Open in Design View. Go to Report Design > Property Sheet. Select 'Report' from dropdown. Go to Data tab > Filter property. Enter `[Balance]=0`. Set Filter On Load to Yes.
  6. Adjust layout. Save.

(c) Printing Prep:

  1. Open `qrybalance` (Datasheet View). Prepare for printing.
  2. Open `rptcleared` (Report View or Print Preview). Prepare for printing.
