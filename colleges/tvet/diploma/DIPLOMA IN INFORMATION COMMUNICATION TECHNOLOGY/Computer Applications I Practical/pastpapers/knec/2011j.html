Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Data Entry and Setup:

  1. Open Microsoft Excel (or a similar spreadsheet program).
  2. In Sheet1, enter the data exactly as shown in Figure 1. Ensure correct cell alignment and formatting (e.g., dates).
  3. Save the workbook as `sales.xlsx` (or `.ods`) inside the `KNECEXAM` folder created on the Desktop.
  4. Go to Sheet2. Enter the data from Figure 2 (Retail Prices).
  5. Rename Sheet1 to `Sales` (Right-click tab > Rename).
  6. Rename Sheet2 to `Prices` (Right-click tab > Rename).

(b) Calculations (in 'Sales' sheet):

  1. Total Sold per Newspaper: Assuming sales data is in columns B to E (Jan to Apr) starting from row 3 for "The Mwananchi", in cell F3 enter the formula: `=SUM(B3:E3)`. Drag the fill handle down to calculate for other newspapers. Add a column header "Total Sold" in F2.
  2. Total Amount per Newspaper:
    • You need the price from the 'Prices' sheet. Assuming newspaper names in 'Sales' (A3:A6) match 'Prices' (A2:A5) and prices are in 'Prices' (B2:B5).
    • In cell G3 (Sales sheet), use `VLOOKUP` to get the price and multiply by the total sold (F3). Formula: `=F3*VLOOKUP(A3,Prices!$A$2:$B$5,2,FALSE)`.
    • Format column G as Currency (Kshs). Add a column header "Total Amount (Kshs)" in G2. Drag the fill handle down.

(c) 3-D Pie Chart (in 'Sales' sheet):

  1. Select the range containing the newspaper names (e.g., A3:A6) and their corresponding sales for February (e.g., C3:C6). Hold Ctrl to select non-adjacent ranges.
  2. Go to Insert > Charts > Pie > 3-D Pie.
  3. Chart Title: Click on the default title and change it to "Total Sales for The Month of February 2011".
  4. Legend: Click the '+' sign next to the chart (Chart Elements) > Legend > Bottom.
  5. Data Labels: Click '+' > Data Labels > More Options... > Check 'Percentage' and uncheck 'Value'. Position as desired (e.g., Outside End).
  6. Position the chart appropriately within the 'Sales' sheet.

(d) Show Formulas and Save:

  1. Show Formulas:
    • Go to the 'Sales' sheet. Navigate to the 'Formulas' tab > Formula Auditing group > Click 'Show Formulas'. Adjust column widths if needed.
    • Go to the 'Prices' sheet. Click 'Show Formulas'. (Likely no formulas here, but follow the instruction).
  2. Save the workbook (`sales.xlsx`).
  3. Prepare for printing both sheets showing formulas (File > Print). Remember to switch back to showing values after printing if needed (click 'Show Formulas' again).

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Database and Table Creation:

  1. Open Microsoft Access (or a similar database program like LibreOffice Base).
  2. Create a new Blank Database.
  3. Save the database as `Inventory.accdb` (or `.odb`) inside the `KNECEXAM` folder on the Desktop.
  4. Create Table:
    • Go to the 'Create' tab and click 'Table Design'.
    • Enter the field names, data types, and properties as specified:
      • ISBN: Short Text (Size 20), Caption: ISBN Number. Right-click and set as Primary Key.
      • TTTLE: Short Text (Size 50).
      • AUTHOR: Short Text (Size 20).
      • YEAR: Number (Field Size: Integer).
      • LIST_PRICE: Currency (Format: Currency, Decimal Places: 2, Symbol: Kshs - may need custom format if Kshs not default).
      • PUBLISHER: Short Text (Default size is usually fine).
    • Save the table design with the name `books`.
  5. Create Form:
    • Select the `books` table in the Navigation Pane.
    • Go to the 'Create' tab and click 'Form'. Access will create a basic form.
    • Switch to Layout View or Design View.
    • Arrange the fields and labels to match the layout shown in the figure (stacked layout). Adjust spacing and alignment.
    • Add a Form Header and type "REDEEMED BOOK SELLERS LTD" and "BOOKS DETAILS ENTRY FORM". Format as needed (bold, center).
    • Ensure all fields (ISBN Number, TTTLE, AUTHOR, YEAR, LIST_PRICE, PUBLISHER) are present and correctly labeled.
    • Save the form as `Books`.
  6. Enter Data:
    • Open the `Books` form (or the `books` table in Datasheet View).
    • Enter the four records provided in the table. Ensure data types match (e.g., enter only numbers for Year, currency for List Price).

(b) Query and Report Creation:

  1. Create Query:
    • Go to the 'Create' tab and click 'Query Design'.
    • Add the `books` table. Close the 'Show Table' dialog.
    • Double-click each field name in the `books` table box to add them to the design grid below (ISBN, TTTLE, AUTHOR, YEAR, LIST_PRICE, PUBLISHER).
    • In the 'Criteria' row under the YEAR field, type `2001`.
    • In the 'Criteria' row under the ISBN field, type `*243*` (using wildcards `*` to find '243' anywhere within the ISBN).
    • Run the query (Click 'Run' in the 'Design' tab) to test it. It should show the two books from 2001 with '243' in the ISBN.
    • Save the query as `YearOfPublications`.
  2. Create Report:
    • Select the `YearOfPublications` query in the Navigation Pane.
    • Go to the 'Create' tab and click 'Report'. Access will generate a basic report.
    • Switch to Layout View or Design View.
    • Grouping: If not already grouped, go to 'Group & Sort' in the 'Design' tab. Add a group on 'YEAR'. Ensure the Year header is displayed.
    • Sorting: Within the 'Group & Sort' pane, add a sort on 'ISBN' (Ascending).
    • Adjust layout, field widths, and headings as needed for clarity. Add a report title like "Publications Year Report".
    • Save the report as `Publications Year`.

(c) Printing:

  1. Open each object (table `books`, query `YearOfPublications`, form `Books`, report `Publications Year`) one by one.
  2. Go to File > Print > Print Preview to check the layout.
  3. Go to File > Print to print each object.
