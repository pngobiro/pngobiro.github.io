Task 2 (Spreadsheet - 25 Marks)

Figure 1 shows an extract from a spreadsheet program. Use it to answer the questions that follow.

Question 2 (a)

(i) Open a spreadsheet program and create the following worksheet as it appears.

(ii) Save the worksheet as Sales in the KNECEXAM folder to print out later. (1 mark)

(iii) Insert three rows above the column headings and key in the following header each line per row.

  • Maisha Mapya Investment Company
  • Down Town Branch
  • Sales for the month of October 2011
(2 marks)

(iv) Merge and center the header inserted in (iii) across cells A to I. (1 mark)

Question 2 (b)

(i) Table 1 shows the percentage commission earned by the salesmen with respect to their sales in column C.

Using cell addresses and a logical function, compute the commission for each salesman. (2 marks)

(ii) Given that PAYE in column E is deducted from the commission earned at the rate of 15%. Using cell addresses only, compute the PAYE payable by each salesman. (1 mark)

(iii) Net pay in column F is given by commission less PAYE. Using cell addresses only, compute the net pay for each salesman. (1 mark)

Question 2 (c)

Maisha Mapya Company Ltd management categorizes the salesmen according to Net Pay earned. A salesman who earns a net pay of more than Ksh. 30000 is categorized as a supervisor, a salesman who earns between Ksh. 10000-29999 is categorized as an assistant supervisor otherwise one is categorized as an assistant salesman.

(i) In column G type in the title Category.

(ii) Using an appropriate function insert the corresponding category for each salesman.

Question 2 (d)

(i) Create an embedded line chart with markers to represent the Net Pay against the salesmen. (2 marks)

(ii) Save the worksheet as Salesmod in the KNECEXAM folder to print out late the worksheet and the chart on the same page. (2 marks)

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a)

  1. Open Excel. Recreate the table from Figure 1.
  2. Save as `Sales.xlsx` in KNECEXAM.
  3. Insert 3 rows above row 1.
  4. Enter headers in A1, A2, A3.
  5. Merge & Center A1:I1, A2:I2, A3:I3.

(b) Assuming Sales are in C5:C10, Commission in D5, PAYE in E5, Net Pay in F5:

  1. Commission (D5): `=IF(C5>=50000, C5*10%, IF(C5>=30000, C5*7%, C5*5%))`. Drag down.
  2. PAYE (E5): `=D5*15%`. Drag down.
  3. Net Pay (F5): `=D5-E5`. Drag down.

(c) Assuming Net Pay is in F5:F10, Category in G5:

  1. Enter "Category" in G4.
  2. Category (G5): `=IF(F5>30000, "Supervisor", IF(F5>=10000, "Assistant Supervisor", "Assistant Salesman"))`. Drag down.

(d)

  1. Select Salesman names (e.g., A5:A10) and Net Pay (F5:F10).
  2. Insert > Charts > Line > Line with Markers.
  3. Format chart (titles, labels). Ensure it's embedded on the same sheet.
  4. Save As `Salesmod.xlsx` in KNECEXAM. Prepare for printing.

Question 3 (b)

(i) Create a query showing PatientNo, PatientName, DoctorsName and WardName for all female patients. Save the query as PatientDetails to print out later.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a)

  1. Create Database: Open Access > Blank Database > Name: `Hospital.accdb` > Save in KNECEXAM.
  2. Create Tables (Design View):
    • PatientData: `PatientNo` (Short Text, PK), `PatientName` (Short Text), `Sex` (Short Text), `Age` (Number - Byte/Integer).
    • DoctorData: `DoctorNo` (Short Text, PK), `DoctorName` (Short Text), `PatientNo` (Short Text - For Relationship).
    • WardData: `WardNo` (Short Text, PK), `WardName` (Short Text), `PatientNo` (Short Text - For Relationship).
    • *Note: Changed Number fields to Text for IDs based on sample data.*
  3. Relationships: Database Tools > Relationships. Add tables. Drag `PatientData.PatientNo` to `DoctorData.PatientNo`. Drag `PatientData.PatientNo` to `WardData.PatientNo`. Enforce Integrity if desired (though data suggests one-to-one, which might be unusual). Save layout.
  4. Create Forms: Select each table > Create > Form. Save as `PatientData`, `DoctorData`, `WardData`.
  5. Enter Data: Use forms to enter the provided data.

(b)

  1. Query (PatientDetails): Create > Query Design. Add `PatientData`, `DoctorData`, `WardData`. Add fields: `PatientData.PatientNo`, `PatientName`, `DoctorName`, `WardName`. Criteria for `Sex`: "Female". Save as `PatientDetails`.
  2. Report (PatientRep): Create > Query Design. Add `PatientData`, `WardData`. Add fields: `PatientData.PatientNo`, `PatientName`, `WardName`. Criteria for `Age`: `Between 20 And 30`. Save query (e.g., `qryAgeReport`). Create > Report Wizard > Select `qryAgeReport`. Add fields. No grouping/sorting. Title: `PatientRep`. Finish. Save.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a)

  1. Open PowerPoint. Create slides based on content in the image for Table 2 (likely 5 slides). Use appropriate layouts.
  2. Save as `Graduation.pptx` in KNECEXAM.
  3. Insert New Slide: After slide 2, Insert > New Slide > Title Only. Title: "Courses Offered".
  4. Create Hierarchy: On the new slide (now slide 3), Insert > SmartArt > Hierarchy. Choose a suitable layout (e.g., Organization Chart) and recreate the structure from the image.
  5. Animation:
    • Slide 1: Animations tab > Select Title > Add Animation > Entrance > Fly In.
    • Slide 2: Animations tab > Select Title/Content > Add Animation > Emphasis > Spin.
  6. Transitions:
    • Slide 2: Transitions tab > Uncover > Effect Options: From Top.
    • Slide 5: Transitions tab > Box > Effect Options: In.
  7. Slide Show Timing: Transitions tab > Timing group > Check "After:", set to 00:05.00 > Apply To All.
  8. Save As `GraduationMod.pptx`. Prepare for printing (File > Print > Settings > Handouts: 3 Slides).
