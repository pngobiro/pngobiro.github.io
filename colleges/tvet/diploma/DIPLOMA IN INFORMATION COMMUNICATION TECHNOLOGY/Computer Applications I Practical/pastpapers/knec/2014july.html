A word processor is a computer application used for the production (including composition, editing, formatting and possibly printing) of any sort of printable material.

Word processors usually feature a monochrome display and the ability to save documents on memory cards or CD ROMs. Other models have introduced innovations such as spell-checking programs, increased formatting options, and dot-matrix printing.

Word processing implies the presence of text manipulation functions that extend beyond a basic ability to enter and change text, such as automatic generation of: batch mailings; indices of keywords; tables of contents; tables of figures; cross-referencing; footnote numbering.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Data Entry and Title:

  1. Open Microsoft Excel.
  2. Enter the data from Figure 1 into Sheet1, starting from cell A1.
  3. Select Row 1, right-click > Insert.
  4. Select cells A1:I1 in the new row, click 'Merge & Center'.
  5. Type "SUPERIOR STATIONERY LIMITED" in the merged cell A1.
  6. Save the workbook as `Task2.xlsx` in the `KNECEXAM` folder.
  7. *Note: Original row numbers mentioned in the question are now increased by 1.*

(b) Calculations and Formatting:

  1. Calculations:
    • Cost (Column D, starting D3): `=B3*C3`. Drag down to D8.
    • Sales (Column G, starting G3): `=E3*F3`. Drag down to G8.
    • Total Cost (Cell D9): `=SUM(D3:D8)`.
    • Total Sales (Cell G9): `=SUM(G3:G8)`.
  2. Formatting:
    • Select C3:E8 and G3:H8 (H column calculated next). Right-click > Format Cells > Currency > Symbol: KES, Decimal places: 2. Click OK.
    • Select D3:D9 (Cost column + Total). Go to Home tab > Font group > Fill Color > Choose Green.
  3. New Selling Price:
    • (Column H, starting H3): `=E3*(1+$B$11)`. Drag down to H8. (Ensure H3:H8 are formatted as KES currency with 2 decimal places).
  4. Stock Balance and Remarks:
    • Stock Balance (Implied column, e.g., J, starting J3): `=B3-F3`. Drag down to J8.
    • Remarks (Column I, starting I3): `=IF(J3<150,"Reorder",IF(J3>400,"Excess","OK"))`. Drag down to I8.

(c) Chart:

  1. Select Item names (A3:A8). Hold Ctrl and select Cost (D3:D8) and Sales (G3:G8).
  2. Go to Insert > Charts > Column > 3-D Clustered Column.
  3. Right-click the chart > Move Chart > New sheet > Name it `Sheet2`. Click OK.

(d) Printing Prep:

  1. Go to Sheet1. Formulas tab > Formula Auditing group > Show Formulas. Adjust column widths.
  2. Save the workbook (`Task2.xlsx`).
  3. Prepare Sheet1 (showing formulas) and Sheet2 (chart) for printing.

Answer

Disclaimer: Answers are AI-generated and may contain errors. Verify independently.

(a) Create Database:

  1. Open Microsoft Access.
  2. Create a new Blank Database.
  3. Save it as `Task3.accdb` in the `KNECEXAM` folder.

(b) Create Tables & Relationships:

  1. Create Tables (in Design View):
    • DoctorData: DoctorCode (Text 15, PK), DoctorName (Text 25). Save.
    • PatientData: PatientID (Text 15, PK), PatientName (Text 25), PatientSex (Text 10). Save.
    • DrugData: DrugCode (Text 15, PK), DrugName (Text 50), Price (Currency). Save.
    • PatientVisitation: VisitationID (AutoNumber, PK - *Implied*), VisitationDate (Date/Time), PatientID (Text 15), PatientName (Text 25), DoctorCode (Text 15), DrugCode (Text 15). Save.
  2. Relationships:
    • Go to Database Tools > Relationships. Add all four tables.
    • Drag `PatientID` from `PatientData` to `PatientID` in `PatientVisitation`. Enforce Referential Integrity. Create.
    • Drag `DoctorCode` from `DoctorData` to `DoctorCode` in `PatientVisitation`. Enforce Referential Integrity. Create.
    • Drag `DrugCode` from `DrugData` to `DrugCode` in `PatientVisitation`. Enforce Referential Integrity. Create.
    • Close and save the layout.

(c) Create Forms & Enter Data:

  1. Create Forms:
    • Select `DoctorData` table > Create tab > Form. Save as `DoctorDetails`.
    • Select `PatientData` table > Create tab > Form. Save as `PatientDetails`.
    • Select `DrugData` table > Create tab > Form. Save as `DrugDetails`.
    • Select `PatientVisitation` table > Create tab > Form. Save as `PatientVisitation`.
  2. Enter Data:
    • Open each form (`DoctorDetails`, `PatientDetails`, `DrugDetails`, `PatientVisitation`) and enter the corresponding data provided. Note: You might need to manually type PatientName in the PatientVisitation form as lookup wasn't specified in table design, or modify the form/table later to add the lookup.

(d) Create Report:

  1. Go to Create tab > Query Design.
  2. Add `PatientVisitation`, `PatientData`, `DoctorData`, `DrugData` tables.
  3. Add the following fields to the grid: `PatientName` (from PatientData), `PatientID` (from PatientData or PatientVisitation), `DoctorName` (from DoctorData), `DrugName` (from DrugData), `DrugCode` (from DrugData or PatientVisitation).
  4. Go to Create tab > Report Wizard.
  5. Select the query you just created as the source.
  6. Add all available fields to the 'Selected Fields' box. Click Next.
  7. Do not add grouping levels. Click Next.
  8. Do not add sorting. Click Next.
  9. Choose a layout (e.g., Tabular). Click Next.
  10. Title the report `PatientReport`. Click Finish.
  11. Adjust column widths and layout in Design or Layout view if needed.
  12. Save the report. Prepare for printing.
