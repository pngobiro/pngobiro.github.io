Chapter 4: Data Types and Operations

Understanding Data Types and Basic Operations in Excel

Types of Data in Spreadsheets

1. Labels (Text)

  • Labels are texts consisting of alphanumeric characters that can be entered into a cell
  • Examples: Item codes like "Salary", Names like "John"
  • In Excel, Text can be any combination of numbers, spaces, & nonnumeric characters (e.g., 10A19, 27AXY, 12-976, 2084675)

2. Values (Numbers)

  • Values consist of numerals & mathematical formulas entered into a cell
  • In Excel, a number can contain only: digits 0 to 9, +, -, (, ), /, $, %

3. Formulas

  • A Formula is a sequence of values, cell references, functions & arithmetic operators whose calculation results in a numeric value
  • It is an equation that performs operations on worksheet data
  • Formulas can perform mathematical operations such as addition, subtraction, division and multiplication
  • Used to tell Excel how you want a particular value to be computed

4. Functions

  • A special command which you can type into your formula to perform arithmetic operations
  • An inbuilt equation used for calculations
  • A short predefined formula used to perform a given specific task
  • Functions can be used to perform both simple and complex calculations

Cell Operations

Editing Cell Contents

  1. Double-click on the cell that contains the data you want to edit

    -Or-

    Click in the cell, then press F2

  2. Edit the cell contents:
    • Use Left or Right Arrow keys to move the insertion point
    • Use Backspace or Delete to erase characters
  3. Press ENTER to confirm changes, then use Arrow keys to move to another cell

Saving Workbooks

Purpose:

  • To use the worksheet at a later time
  • Periodic saving (e.g., every 1 minute) helps prevent data loss in case of power failure

To Save a New Workbook

  1. Click the Save button on the Standard toolbar

    -OR-

    On the File menu, choose Save (or press CTRL+S)

  2. In the File name box, enter a name for the worksheet
  3. In the Save in list, select the drive and/or folder location
  4. Click the Save button

To Save a Copy of a Workbook

  1. Open the workbook you want to copy
  2. On the File menu, click Save As...
  3. Enter a new name for the file
  4. Select a different location if needed
  5. Click the Save button

Automatic Saving

  1. On the Tools menu, click Options
  2. Click the Save tab
  3. Select the Save AutoRecover info every checkbox
  4. Enter the desired time interval in minutes

Opening and Closing Workbooks

Opening a Saved Worksheet

Purpose:

  • Continue working on an incomplete worksheet
  • View existing data
  • Update periodically changing information
  1. Choose Open on the File menu

    -OR-

    Click the Open button on the Standard toolbar (or press CTRL+O)

  2. In the Look in list, select the drive or folder
  3. Locate and open the folder containing the file
  4. Double-click the file you want to open

Exiting Excel

To exit from Excel when finished working:

  • On the File menu, click Exit
  • -OR- Press ALT+F4
  • -OR- Click the Close button on the Title bar

Note: If the workbook was not saved before closing, Excel will prompt you to save changes.
