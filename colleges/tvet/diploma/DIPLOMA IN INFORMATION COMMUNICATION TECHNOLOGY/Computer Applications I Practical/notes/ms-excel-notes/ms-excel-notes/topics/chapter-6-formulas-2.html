Chapter 6: Advanced Functions

Understanding Advanced Mathematical and Statistical Functions

Trigonometric Functions

COS Function

The COS function gives the cosine of a given angle.

Syntax: COS(number)

number - is the angle in radians. To convert degrees to radians, multiply by PI()/180

Examples:

=COS(1.047) Result: 0.500171 =COS(60 * PI() / 180) Result: 0.5

SIN Function

The SIN function gives the sine of a given angle.

Syntax: SIN(number)

Examples:

=SIN(PI() / 2) Result: 1 =SIN(30 * PI() / 180) Result: 0.5

TAN Function

The TAN function gives the tangent of a given angle.

Syntax: TAN(number)

Examples:

=TAN(0.785) Result: 0.99920 =TAN(45 * PI() / 180) Result: 1

Statistical Functions

AVERAGE Function

Calculates the arithmetic mean of a set of numbers.

Syntax: AVERAGE(number1,number2,...)

Note: Empty cells, logical values, or text are ignored, but cells with zero are included.

Example: For cells A2:A6 containing values 10, 7, 9, 27, & 4:

=AVERAGE(A2:A6) Result: 11.4 =AVERAGE(A2:A4,A6) Result: 7.5 =AVERAGE(A2:A5, 5) Result: 11.6

MIN and MAX Functions

Find the smallest (MIN) or largest (MAX) value in a set.

Syntax: MIN(number1,number2,...) Syntax: MAX(number1,number2,...)

Example: For cells A2:A6 containing values 10, 7, 9, 27 & 2:

=MAX(A2:A6) Result: 27 =MIN(A2:A6) Result: 2 =MAX(A2:A6,30) Result: 30 =MIN(A2:A6,0) Result: 0

MEDIAN Function

Gives the middle value in a set of numbers.

Syntax: MEDIAN(number1,number2,...)

Note: For an even number of values, MEDIAN calculates the average of the two middle numbers.

Example: For cells A2:A7 containing values 1, 2, 3, 4, 5, 6:

=MEDIAN(A2:A6) Result: 3 (middle of first 5 numbers) =MEDIAN(A2:A7) Result: 3.5 (average of 3 and 4)

Counting Functions

COUNT Function

Counts the number of cells containing numbers.

Syntax: COUNT(value1,value2,...)

Note: Empty cells, logical values, text, or error values are ignored.

Examples:

=COUNT(A2:A6) Counts numbers in range =COUNT(A2:A3,A6) Counts numbers in specific cells

Logical Functions

NOT Function

Reverses the logical value of its argument.

Syntax: NOT(logical)

Examples:

=NOT(FALSE) Result: TRUE =NOT(1+1=2) Result: FALSE

AND Function

Returns TRUE if all arguments are TRUE.

Syntax: AND(logical1,logical2,...)

Examples:

=AND(TRUE,TRUE) Result: TRUE =AND(TRUE,FALSE) Result: FALSE =AND(2+2=4,2+3=5) Result: TRUE

OR Function

Returns TRUE if any argument is TRUE.

Syntax: OR(logical1,logical2,...)

Examples:

=OR(TRUE) Result: TRUE =OR(1+1=1,2+2=5) Result: FALSE =OR(TRUE,FALSE,TRUE) Result: TRUE
