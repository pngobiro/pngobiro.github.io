Chapter 8: Advanced Charts and Printing

Advanced Chart Features and Printing Options

Modifying Charts

Changing Chart Type

  1. Click the chart to activate the Chart menu
  2. On the Chart menu, click Chart Type
  3. Click the Standard Types tab
  4. Select the desired chart type
  5. Click OK

Changing Chart Options

  1. Click the chart
  2. On the Chart menu, click Chart Options
  3. Use the various tabs to modify:
    • Titles
    • Legends
    • Data labels
    • Axes

Modifying Data Series

  1. Click the chart
  2. On the Chart menu, click Source Data
  3. On the Series tab, select the data series to modify
  4. Make changes in the Name box or by selecting new worksheet cells

Working with Chart Data

Changing the Data Range

  1. Click the chart
  2. On the Chart menu, click Source Data
  3. Click the Data Range tab
  4. Select new range in the worksheet

Including New Data

To add information not already in the chart:

  1. Select the range you want to add
  2. Position the mouse pointer along the edge of the selected range
  3. Drag the range into the chart

Note: Excel will automatically update the chart to include the new range.

Deleting Data

To delete data from both worksheet and chart:

  • Delete the data from the worksheet - chart updates automatically

To delete data from chart only:

  1. Click the data series you want to delete
  2. Press the DELETE key

Formatting Charts

Text Formatting

  1. Click the chart text or select characters to format
  2. Use the Formatting toolbar to:
    • Change font
    • Modify font size
    • Change font color
    • Apply Bold, Italic, or Underline

Moving and Resizing

  • To move: Click and drag the chart item to a new location
  • To resize: Drag the sizing handles until the item is the desired size

Printing in Excel

Purpose of Printing:

  • Create paper copies to present to others
  • Maintain a paper filing system alongside computer records

Page Setup

Control the appearance of printed worksheets through Page Setup options:

  • Define page breaks
  • Fit large worksheets to single pages
  • Add descriptive information
  • Define rows/columns to print on each page
  • Change print order of worksheets

Setting Page Margins

  1. Select the worksheet to print
  2. On the File menu, click Page Setup
  3. Click the Margins tab
  4. Enter margin sizes for:
    • Top
    • Bottom
    • Left
    • Right
    • Header
    • Footer

Print Options

Setting Paper Orientation

  1. In Page Setup dialog box, click the Page tab
  2. Under Orientation, select:
    • Portrait
    • Landscape

Print Quality

Note: Draft quality increases printing speed by ignoring formatting and most graphics.

Previewing Before Printing

  1. Click Print Preview on the File menu
  2. Review the preview
  3. Make adjustments if needed
  4. Click Print to proceed or Close to return
