Chapter 3: Excel Interface and Layout

Working with Microsoft Excel's Interface and Basic Operations

Positioning Data in a Cell

Adjusting the alignment of cell contents helps to distinguish different types of information in cells. By default:

  • Text data is aligned to the left in a cell
  • Numbers, dates & times are aligned to the right

To Align Data

  1. Select the cells with the data you want to reposition
  2. On the Formatting toolbar, click the appropriate alignment button:
    • Align Left button - to align text to the left of the cell
    • Center button - to center text in the cell
    • Align Right button - to align text to the right

Merging Cells Across Columns

You can merge headings across the top of a range of cells. When you merge cells, the selected cells are combined into one cell, spreading the content of one cell over many cells.

Steps to Merge Cells

  1. Copy the data you want into the upper-leftmost cell within the range
  2. Select the cells you want to merge
  3. Click the Merge and Center button on the Formatting toolbar

    -OR-

    On the Format menu, click Cells, click the Alignment tab, then select the Merge cells checkbox

Warning: Excel places only the data in the upper-leftmost cell of the selected range into the resulting merged cell. If there is data in other cells, the data is deleted when you merge the cells.

Hiding Rows or Columns

Hiding rows or columns prevents the display and printing of data held in particular rows or columns. This is useful when your rows or columns contain confidential formulas not necessary in the printed report.

To Hide Rows or Columns

  1. Select the rows or columns you want to hide
  2. On the Format menu, point to Row or Column, then click Hide

To Display Hidden Rows or Columns

  1. For rows: Select the row below and the row above the hidden rows
  2. For columns: Select the column to the left and the column to the right of the hidden columns
  3. On the Format menu, point to Row or Column, then click Unhide

Tip: If the first row or column is hidden:

  1. Click Go To on the Edit menu
  2. In the Reference box, type A1, and click OK
  3. On the Format menu, point to Row or Column, then click Unhide

Applying Borders, Shadings & Patterns

You can apply borders to cells, shade cells with a background color, or shade cells with a color pattern to:

  • Distinguish between different types of information
  • Make the worksheet more visually appealing
  • Draw attention to important data

To Apply Borders

  1. Select the range of cells
  2. On the Format menu, select Cells
  3. Click on the Border tab
  4. Under Presets section, select the Border style
  5. To change the line style, click a style in the Style list
  6. Specify the color from the Color drop-down list
  7. Click OK

To Apply or Remove Cell Shadings

  1. Select the cells to modify
  2. On the Format menu, click Cells
  3. Click the Patterns tab
  4. In the Cell shading box, click the desired color
  5. To include a background pattern, select from the Pattern box
  6. Click OK
