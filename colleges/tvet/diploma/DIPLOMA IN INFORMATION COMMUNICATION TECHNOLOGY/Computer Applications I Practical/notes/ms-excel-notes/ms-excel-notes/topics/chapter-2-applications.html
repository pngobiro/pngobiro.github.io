Chapter 2: Applications and Features

Application Areas and Common Features of Spreadsheets

Application Areas of Spreadsheets

1. Accounting

Spreadsheets provide an easy & streamlined means of financial management. They are mostly used by Accountants to record their daily transactions & also keep financial records.

For example, a spreadsheet can be used to:

  • Record sales & purchases
  • Calculate profits
  • Produce Invoices and compile financial statements
  • Prepare budgets
  • Monitor payments from customers
  • Detect aged debtors
  • Track the value of assets over time (Appreciation and Depreciation)

Note: Most spreadsheet programs come with inbuilt functions such as SUM, AVERAGE, PRODUCT, etc, which enable the Accountant to carry out daily accounting tasks easily.

2. Data Management

A spreadsheet enables information to be produced easily and kept up-to-date.

Features include:

  • Create, edit, save, retrieve and print worksheet data & records
  • Arrange data neatly in tabular structure
  • Link related data across worksheets
  • Sorting (arranging worksheet records in a particular order)
  • Filtering (displaying only records that meet given conditions)
  • Forms for data entry & viewing
  • Total/Subtotal functions

3. Scientific Applications

Spreadsheet programs can be used by Scientists & Researchers to compile & analyse their results.

4. Statistical Analysis

Spreadsheets provide data analysis tools for complex statistical analyses and chart generation.

Examples of statistical functions include:

  • AVERAGE - calculates the mean of a set of values
  • MEDIAN - gives the value in the middle of a set of values

These operations can be used by:

  • Teachers to compile student marks and produce results
  • Clerks & Secretaries to create and manipulate tables of figures

5. Forecasting (What-if Analysis)

The automatic recalculation feature enables "What if" analysis - a technique to find out the effect of changing certain values in a worksheet on other cells.

This method can be used for financial forecasting, budgeting, and cost analysis.

Common Features of Electronic Spreadsheets

  1. Ability to create, edit, save & retrieve worksheets
  2. Inbuilt functions & formulae for calculations
  3. Automatic recalculation when values change
  4. Sort and filter data capabilities
  5. Data validation facility
  6. Chart facility for visual representation
  7. Data formatting options
  8. SOLVER facility for resource optimization
  9. Column width & Row height adjustment
  10. Hide/unhide rows & columns
  11. Quick worksheet printing
  12. Pre-designed Templates
  13. Data summarization using Consolidation and Pivot tables

Templates

A Template is a document that acts as a blueprint or outline for other documents of the same type. It contains standard text, graphics & formatting that will be used in all documents of this type.

Differences between Electronic Spreadsheet and Calculator

An electronic spreadsheet:

  • Has more memory than calculator
  • Is able to perform complex logical operations
  • Uses the large storage capacity of the computer
  • Has a large working area
