Chapter 7: Creating Charts

Understanding and Creating Charts in Excel

Introduction to Charts

Purpose: Charts are used to present data effectively. They make relationships among numbers easy for users to see by turning numbers into shapes that can be compared to one another.

For instance, rather than having to analyze several columns of worksheet numbers, you can see at a glance whether:

  • Sales are falling or rising over quarterly periods
  • How actual sales compare to projected sales
  • Distribution patterns in your data

Different Types of Charts and Their Uses

1. Line Chart

A Line graph is used to show trends over time. Best for:

  • Visualizing data trends
  • Showing changes over time
  • Comparing multiple series over time

2. Bar Chart

Used to show comparisons between different items. Ideal for:

  • Comparing Sales and Target figures
  • Showing horizontal comparisons
  • Displaying negative values

3. Stacked Bar Chart

Used to show the distribution of data within categories. Perfect for:

  • Showing distribution of sales by month
  • Comparing performance of different salesmen
  • Displaying part-to-whole relationships

4. Pie Chart

Shows the distribution of parts to a whole. Best for:

  • Showing percentages of a total
  • Displaying market share
  • Representing proportional data

5. Scatter Charts

Used to show relationships between two variables. Ideal for:

  • Showing correlations
  • Identifying patterns
  • Scientific data analysis

6. Column Charts

Similar to bar charts but with vertical bars. Best for:

  • Comparing values across categories
  • Showing changes over time
  • Comparing different items

Creating a Simple Chart

Basic Steps

  1. Enter the data you want to be represented in the chart on the worksheet
  2. Select the cells or range that contains the data
  3. On the Insert menu, click Chart (or click the Chart Wizard button)

For Non-Continuous Ranges:

  1. Select the first group of cells
  2. Hold down CTRL
  3. Select additional cell groups

Note: The nonadjacent selections must form a rectangle.

Using the Chart Wizard

  1. Select Chart Type
    • Click Standard Types or Custom Types tab
    • Choose chart category and sub-type
    • Review description and sample
  2. Specify Data Range
    • Confirm or modify selected range
    • Choose data series in rows or columns
    • Review sample chart
  3. Set Chart Options
    • Add chart title
    • Add axis titles
    • Configure legend

Understanding Chart Components

Legend

A box that identifies the patterns or colors assigned to the data series or categories in a chart.

Axes

Lines that border the plot area, providing a frame of reference for measurement or comparison:

  • Value Axis (Y-axis): Usually vertical, displays data values
  • Category Axis (X-axis): Usually horizontal, displays categories

Note: Excel creates the axis values automatically from your worksheet data.
