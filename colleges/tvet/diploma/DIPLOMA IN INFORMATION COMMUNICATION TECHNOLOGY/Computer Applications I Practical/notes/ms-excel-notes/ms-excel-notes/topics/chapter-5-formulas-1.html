Chapter 5: Basic Formulas and Operations

Understanding Basic Mathematical Operations in Excel

Basic Arithmetic Operations

Adding Numbers

Method 1: Direct Addition

Type numbers directly into a cell:

=D5+E5+F5 -OR- =5+15+20

Adding Numbers in a Contiguous Range

Use the AutoSum (Î£) button on the Standard toolbar:

  1. Click a cell below the column of numbers or to the right of the row of numbers
  2. Click AutoSum on the Standard toolbar
  3. Excel suggests a formula
  4. Press ENTER to accept or modify the range as needed

Using the SUM Function

For adding numbers not in a contiguous row or column:

Syntax: SUM(number1,number2,...)

Examples:

  • =SUM(A2:A4) - adds values in range A2 to A4
  • =SUM(A2:A4, 15) - adds range plus additional value
  • =SUM(A2, A4, 2) - adds specific cells and value

Subtracting Numbers

Basic subtraction formula:

=10-5

Example with cell references:

=A2-A3

Calculating Running Balance

For banking transactions:

=SUM(F6,D7,-E7)

Where:

  • F6 contains previous balance
  • D7 contains deposit amount
  • E7 contains withdrawal amount

Working with Percentages

To Increase a Number by Percentage

=F5*(1+5%)

Using cell reference for percentage:

=F5*(1+$F$2)

Note: The reference to F2 uses absolute cell reference ($F$2) so the formula can be copied without changing the reference.

Multiplication and Division

Multiplying Numbers

Use the asterisk (*) operator or the PRODUCT function.

Syntax for PRODUCT:

PRODUCT(number1,number2,...)

Examples:

  • =5*10
  • =A2*B2
  • =(5+2)*3

Practical Example: Calculating Cost

To calculate cost of items:

=B2*C2 (where B2 is Quantity and C2 is Price)

Important: Don't forget the '=' symbol at the start of the formula, or Excel will treat it as text.

Dividing Numbers

Basic division formula:

=10/5

Using cell references:

=A2/A3

Special Mathematical Functions

Square Root (SQRT)

Syntax:

SQRT(number)

Examples:

  • =SQRT(16) results in 4
  • =SQRT(B2) calculates square root of value in B2

Note: SQRT returns #NUM! error for negative numbers.

Power Function

Syntax:

POWER(number,power)

Examples:

  • =5^2 or =POWER(5,2) for 5 squared
  • =5^3 or =POWER(5,3) for 5 cubed
  • =POWER(4,5/4) for fractional powers
