Topic 10: Working with Graphics and Printing

üéØ Changing the Weight, Color and Style of a Line.

  1. 1.Select the object whose line weight, color or style you want to change.
  2. 2.Click on Format, select AutoShape, then click the Colors and Lines tab.
  3. 3.Select a line from the Style box. To use a dashed line, then select it under Dashed.
  4. 4.Use the Up & Down arrows in the Weight box to adjust the line weight.
  5. 5.To apply a different color to the line, select it from the Color drop-down list.
  6. 6.Click OK.

üéØ Callouts

These are special AutoShapes into which you can add text. They can be used to draw attention to an important point or to highlight information that you want to emphasize.

üéØ Adding WordArt to a document.

WordArt is a text object that can be created with ready-made effects, and to which additional formatting options can be applied.

  1. 1.Click on Insert, point to Picture, then click WordArt.
  2. 2.In the WordArt Gallery dialog box, click the WordArt effect you want, then click OK.
  3. 3.In the Edit WordArt Text dialog box, type the text you want.
  4. 4.To change the font type, select a font in the Font list. To change the font size, select a size in the Size list. To make text bold or italic, click the Bold or Italic button.
  5. 5.Click OK.

üéØ Changing the text in WordArt.

  1. 1.Double-click the WordArt object you want to change.
  2. 2.In the Edit WordArt Text dialog box, type the new text, then click OK.

üéØ Formatting WordArt text.

  1. 1.Click the WordArt, click on Format, then choose WordArt (or, right click the WordArt, then choose Format WordArt).
  2. 2.Use the tabs on the Format WordArt dialog box to change the WordArt text.

üéØ INSERTING A CLIPART (PICTURE) FROM THE MICROSOFT CLIP GALLERY

The Microsoft Clip Organizer contains a variety of useful drawings, photographs, sounds, videos, and other media files called clips that you can insert and use in presentations, publications, and other Microsoft Office documents.

Inserting a clip from the Clip Organizer.

A Clip is a single media file including art, sound, animation, or movies.

  1. 1.Position the insertion point where you want your clip to be inserted.
  2. 2.Click on Insert, point to Picture, then click Clip Art.
  3. 3.At the bottom of the Insert Clip Art task pane, click Clip Organizer. Use the Clip Organizer to browse through Microsoft Office clip collections.
  4. 4.Click the Pictures tab. In the Categories list box, select the category of Clip Art pictures you want, e.g., Plants. A preview of images appears for that category.
  5. 5.In the Results box, click the media clip you want to insert, then drag it into your open document.

üéØ INSERTING AN IMAGE FROM A SCANNER.

A Scanner is a data capture device that transforms a hardcopy or real graphical object into digital form.

To scan an object;

  1. 1.The computer must be connected to a Scanner.
  2. 2.Position the insertion point where you want your picture to appear.
  3. 3.Click on Insert, point to Picture, then click From Scanner or Camera.
  4. 4.To scan, follow the instructions that come with your scanner.
  5. 5.When the image appears on the screen, edit or format it as required the same way you would with a ClipArt.

üéØ INSERTING A PICTURE FROM ANOTHER FILE.

  1. 1.Position the insertion point where you want the picture to be placed.
  2. 2.Click on Insert, point to Picture, then click From File.
  3. 3.Locate the file that contains the image you want to insert, e.g., from a floppy disk, CD, etc.
  4. 4.Select the object you want to insert, then click the Insert button.
  5. 5.When the image appears on the screen, edit or format it as required the same way you would with a ClipArt.

üéØ Changing the size of a graphic.

After you insert a picture, you may find that it is too big or too small for your needs.

To change the size of the object;

Method 1:

  1. 1.Select the graphic, then drag the sizing handles away from or toward the center until the object is the shape and size you want.

Sizing handles are the small circles or squares that appear at the corners and sides of a selected object.

Method 2:

  1. 1.Select the graphic.
  2. 2.Click on Format, select Picture, then click the Size tab.
  3. 3.Under Size and rotate, adjust the sizes in the Height and Width boxes.
  • -To resize the picture by a specific percentage, under Scale, enter the percentages you want in the Height and Width boxes.
  • -To maintain the ratio between the height & width of the object when resizing, select the Lock aspect ratio checkbox.
  1. 4.Click OK.

üéØ Rotating an AutoShape, a picture, or a WordArt.

  1. 1.Select the drawing object you want to rotate.
  2. 2.To rotate the object to any angle, drag the Rotate handle (U) on the object in the direction you want to rotate it, then click outside the object to set the rotation.
  3. 3.To rotate the object $90^{\circ}$ to the left or the right,
  • -On the Drawing toolbar, click Draw, point to Rotate or Flip, then click Rotate Left or Rotate Right.

üéØ Deleting a drawing object, Text Box or WordArt.

  1. 1.Click to select the object you want to delete, then press the Delete key.

üéØ Create a mirror image of an object.

  1. 1.Click the AutoShape, picture, Clip Art, or WordArt you want to duplicate.
  2. 2.Click on Edit, click Copy, and then click Paste.
  3. 3.On the Drawing toolbar, click Draw, point to Rotate or Flip, then click Flip Horizontal or Flip Vertical.
  4. 4.Drag and position the duplicate object so that it mirrors the original object.

üéØ Importing objects into a word processor.

Importing refers to getting ready-made objects and text from other applications into a word processor.

Difference between Object Linking and Embedding.

In Linking, the object imported can only be edited in the original application from which it was created; In Embedding, the object imported becomes part of the current application, and can be edited there.
