Topic 8: Working with Tables

üéØ CREATING MS-WORD TABLES.

A Table is a feature in Ms-Word made up of rows & columns which intersect to form cells that the user can enter text and graphics.

  • -A Cell is a box formed by the intersection of a row & column in a worksheet or a table, in which the user can enter information.
  • -A Row is a horizontal arrangement of cells; a Column is a vertical arrangement of cells.

Importance of creating tables.

  • ‚àöTables are used to organize and enhance the display of information such as timetables.
  • ‚àöTables make it easier to read & understand information that would otherwise have to be written in a repetitive and lengthy fashion.

Note. To perform calculations in a table, the table cells must be referenced as A1, A2, B1, B2, etc, with the letter representing the column & the number representing a row.

Methods of creating a new table.

Method 1: Using the Insert Table button on the Standard toolbar.

  1. 1.Click where you want the table to be created.
  2. 2.Click the Insert Table button on the Standard toolbar,. A grid appears below the button.
  3. 3.Drag over the grid to select the number of Rows & Columns you want, then release the Mouse button.

Method 2: Using the Insert - Table command on the Table menu.

  1. 1.Click on Table, point to Insert, then click Table.
  2. 2.Under Table size, specify the number of Columns & Rows.
  3. 3.Under AutoFit behavior, choose an option to adjust the table size.
  4. 4.To use an in-built table format, click the AutoFormat button, then select a table style.
  5. 5.Click the OK button.

Method 3: Using the Draw Table tool for creating more complex tables.

  1. 1.Click on Table, then choose Draw Table.

The Tables and Borders toolbar appears, and the Mouse pointer changes to a Pencil.

  1. 2.Drag the pointer to draw the outline of the table. E.g., to define the outer boundaries for the table, draw a rectangle.
  2. 3.Draw the column and row lines inside the rectangle.

After creating the table, click in a cell and start typing text.

  • -To move the text cursor from one cell to the next in the same row, press the TAB key.
  • -To move text cursor one column to the left in the same row (to move to the previous cell), press SHIFT + TAB.

Inserting text before a table.

  1. 1.If you had already inserted the table, click in the upper-left cell in the first row of the table.
  2. 2.Press the ENTER key.

Selecting parts of a Table.

To select:

  • -A cell - point to the left edge of the cell, and then click.
  • -A row - click to the left of the row.
  • -A column - click the column's top border.
  • -Multiple cells, rows, or columns - drag across the cell, row, or column.
  • -The entire table - click the table move handle.
  • -Text in the next cell - press TAB key.
  • -Text in the previous cell - press SHIFT+TAB keys.

Note. To select a cell, row, column or the entire table;

  1. 1.Click in the cell, row, column or table.
  2. 2.Click on Table, point to Select, then click Table, Column, Row or Cell.

Changing the width of a Column in a Table.

Change the width of a column enhances the appearance of the table, e.g., if the cells are wider than the text they contain.

  1. 1.Point on the boundary of the column whose width you want to change until the pointer becomes a double-headed arrow.
  2. 2.Drag the boundary to the right or left until the column is the width you want.

Making the columns in a table automatically fit the contents.

  1. 1.Click in the table.
  2. 2.Click on Table, point to AutoFit, then click AutoFit to Contents.

Changing the Height of a Row in a table.

In Ms-Word documents, the height of each row in a table depends on the contents of the cells in that row and the paragraph spacing before or after text.

In addition, pressing the ENTER key when the text cursor is in a cell will increase the height of the row.

  1. 1.Point on the boundary of the row whose height you want to change until the pointer becomes a double-headed arrow, and then drag the boundary.

Making multiple columns or rows the same size.

  1. 1.Select the columns or rows you want to make the same size.
  2. 2.On the Table menu, point to AutoFit, then click Distribute Columns Evenly or Distribute Rows Evenly.

Changing the column width or row height to a specific measurement.

  1. 1.Click a cell in the column or row.
  2. 2.Click on Table, click Table Properties, then click the Column tab or Row tab.
  3. 3.In the Size section, select the preferred column or height.

Adding Rows and Columns to a table.

You can add more rows and/or columns to an existing table.

To insert Rows;

  1. 1.Select the same number of rows as the number you want to insert, e.g., to insert 3 new rows, select 3 existing rows.
  2. 2.Click on Table, point to Insert, then click Rows Above or Rows Below.

Tip. To quickly add a row at the end of a table, click in the last cell of the last row, then press the TAB key.

To insert Columns;

  1. 1.Select the same number of columns as the number you want to insert.
  2. 2.Click on Table, point to Insert, then click Columns to the Right or Columns to the Left.

Adding a column to the right of the last column in a table;

  1. 1.Click in the last column.
  2. 2.Click on Table, point to Insert, then click Columns to the Right.

Clearing the contents of a table.

Clearing cells removes the cell contents (formulas and data), or number formats leaving the cells on the table blank.

  1. 1.Select the cells, rows or columns whose contents you want to clear, then press DELETE key.

Deleting cells, rows, or columns from a table.

You can delete any no. of adjacent cells, rows, or columns from a table if you do not need the information in them.

Deleting removes the cells, rows and columns from the document and shifts the surrounding cells to fill the space.

  1. 1.Select the cells, rows, or columns you want to delete.
  2. 2.Click on Table, point to Delete, then click Columns, Rows, or Cells.

If you are deleting cells, the Delete Cells dialog box appears.

Select:To:
Shift cells leftMove cells from the right of the selected cells to replace the cells being deleted.
Shift cells upMove cells from below the selected cells to replace the cells being deleted.
Delete entire rowDelete the row(s).
Delete entire columnDelete the column(s).

Deleting a table and its contents.

Deleting removes the cell contents together with the cells themselves.

  1. 1.Click inside the table.
  2. 2.Click Table, point to Delete, then click Table.

Displaying or hiding table Gridlines.

Gridlines are the thin lines that indicate the cell boundaries in a table. Gridlines do not print when you print a document.

  1. 1.On the Table menu, click Show Gridlines or Hide Gridlines.

Adding Borders & Shading to table cells.

You can apply your own borders and shading to a table. This is because Ms-Word does not print the gridlines that divide the table cells on your screen.

  1. 1.Selecting the table cells you want to apply borders around.
  2. 2.Click on Format, then choose Borders and Shading.
  3. 3.In the resulting dialog box, click the Borders tab or the Shading tab, and then select the options you want.
  4. 4.In the Apply to: box, select Table or Cell.
  5. 5.Click the OK button.

Note. To apply in-built/automatic table formats (borders) around table cells;

  1. 1.Click in the table.
  2. 2.Click on Table, then choose Table AutoFormat.
  3. 3.In the Table styles box, click the table format you want.
  4. 4.Under Apply special formats to and Formats to apply, select the options you want, then click the Apply button.

Merging of cells in a table.

Merging of cells refers to combining 2 or more cells in the same row or column to form a single cell.

Importance.

  • ‚àöYou can merge several cells horizontally to create a table heading that spans several columns.
  • ‚àöTo create larger cells in a table without increasing the height or width of existing cells.
  1. 1.Select the cells you want to merge. This can be adjacent rows or adjacent columns.
  2. 2.Click on Table, then choose Merge Cells.

Note. When you merge several cells in a column to create a vertically oriented table heading that spans several rows, you can change the orientation of the heading text.

  1. 1.Select the heading text that you want to span several rows.
  2. 2.On the View menu, point to Toolbars, then click Tables and Borders.
  3. 3.Click the Change Text Direction button.

Splitting a cell into multiple cells in a table.

This refers to subdividing one or more cells into parts.

  1. 1.Click in a cell, or select multiple cells that you want to split.
  2. 2.Click on Table, then choose Split Cells.
  3. 3.In the resulting dialog box, specify the no. of Rows and/or Columns that the selected cells will be split into.
  4. 4.Click the OK button.

Numbering the cells in a table.

  1. 1.Select the cells you want to number.

To number the beginning of each row, select only the first column in the table by clicking the column's top border.

  1. 2.Click on Format, then choose Bullets and Numbering.
  2. 3.Click the Numbered tab, then select a number format from the list.

Aligning a table on a page.

You can change the horizontal placement of a table within a page, e.g., center a table between the left & right page margins.

  1. 1.Select the table.
  2. 2.Click on Table, choose Table Properties, then click the Table tab.
  3. 3.Under Alignment, select Left, Center or Right.
  4. 4.Click the OK button.

Splitting a table into parts.

When a long table falls within a page break, Ms-Word allows the page break to divide the row between the two pages.

You can adjust the table to make sure that the information appears as you want it to when the table spans multiple pages.

To split a table into 2 parts;

  1. 1.Click the row that you want to be the first row of the second table.
  2. 2.Click on Table, then choose Split Table.

Converting a Table into lines of Text.

  1. 1.Select the entire table, or the rows and columns you want to convert to text.
  2. 2.Click on Table, point to Convert, then click Table to Text.

A dialog box is displayed that requires you to specify how the text will be separated after conversion.

  1. 3.In the Separate text with: box, select the character to be used as a separator, i.e., Paragraph marks, Tabs, or Commas.
  2. 4.Click the OK button.

Converting lines of Text into a Table.

  1. 1.Select the text you want to convert to a table.
  2. 2.Click on Table, point to Convert, then click Text to Table.

A dialog box is displayed that requires you to specify the number of rows and columns.

  1. 3.In the Separate text at: box, specify whether the table will be defined by Paragraphs, Tabs, or Commas.
  2. 4.Click the OK button.

PERFORMING ARITHMETIC CALCULATIONS IN A MS-WORD TABLE.

It is possible to perform mathematical calculations such as Sum, Product, and Count, if a table has numerical figures.

  1. 1.Click in the cell in which you want the result to be placed.
  2. 2.Click on Table, then choose Formula. A formula dialog box is displayed.
  3. 3.If Ms-Word proposes a formula that you do not want to use, delete it from the Formula box, making sure that you don't delete the equal sign.
  4. 4.In the Paste function box, click a function. A Function is a mathematical formula that accomplishes calculations in a table.

E.g., to add numbers, click SUM.

  1. 5.To refer to the contents of a cell in the formula, type the cell references in the parentheses. Use a Comma to separate references to individual cells, or a Colon to separate the first & the last cells in a selected range.

Examples;

  • -To add the numbers in cells A1, A3 & C2, type: =SUM(a1, a3,c2)
  • -To find the average of values in cells A1, A2, B1, B2, C1 & C2, type: =Average(a1:c2)
  1. 6.In the Number format box, enter a format for the numbers, e.g., to display the numbers as a decimal percentage, click 0.00%.
  2. 7.Click the OK button.

Sorting in tables.

Sorting refers to arranging of text, numbers or data in a list or table according to some predetermined sequence.

  1. 1.Click on any cell in the table you want to sort.
  2. 2.Click on Table, then choose Sort. The entire table is automatically selected, and the Sort dialog box appears.
  3. 3.Select the type of data to be used when sorting, i.e., Text, Number or Date.
  4. 4.Under Sort By, select the column number corresponding to the column you want to use as the basis for sorting.
  5. 5.Select the Ascending or Descending option button.

Ascending will sort in increasing order (lowest to highest or A-Z); Descending will sort in decreasing order (highest to lowest or Z-A).

  1. 6.If you want to use additional columns for sorting, enter the column in the Then by boxes.
  2. 7.Click the OK button.
