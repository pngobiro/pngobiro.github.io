Chapter 4: Creating MS-Access Database

üìÅ CREATING A MS-ACCESS DATABASE

Database is a collection of data and objects, such as tables, queries, forms, reports, related to a particular topic or purpose.

üíæ MS-ACCESS DATABASE FILES

Using Ms-Access, you can manage all your information from a single database file.

Database File:

  • ‚Ä¢ A collection of logically related records.
  • ‚Ä¢ A collection of all tables and objects used to manage data.

A Database file consists of rows and columns.

Within the file, you can use database objects such as:

  • ‚Ä¢ Tables - to store your data.
  • ‚Ä¢ Queries - to find and retrieve any data you want.
  • ‚Ä¢ Forms - to view, add, and update data in tables.
  • ‚Ä¢ Reports - to analyze or print data in a specific layout.

üõ†Ô∏è Methods of creating an Ms-Access Database file

(A). Create a new Blank database file

You can create a Blank database & then add the Tables, Forms, Reports & other objects later.

This requires the user to define each database element separately allowing him/her to create a self-defined database.

  1. 1. Click New on the File menu or on the Standard toolbar (or press CTRL+N).
  2. 2. Under New, click Blank Database, then click the OK button.
  3. 3. In the File New Database dialog box, specify a name & location for the database file, then click on the Create button to display the Database window.
  4. 4. Click on the tab that has the object you want to create, e.g., Tables, and follow the instructions provided in the resultant dialog box.

Note. All Ms-Access databases are saved with an automatic extension of .mdb

(B). Create a database file using the Database Wizard

The Database Wizard allows the user to create in one operation the required Tables, Forms, and Reports for the type of database you choose. The wizard provides a set of databases where the user is free to modify them according to his/her needs.

  1. 1. On the File menu, click New.
  2. 2. Under New from template, click General templates.
  3. 3. Click the Databases tab, then double-click the icon for the kind of database you want to create, (or click the icon, then click the OK button).
  4. 4. In the File New Database dialog box, specify a name & location for the database.
  5. 5. Click on the Create button, then follow the instructions in the Wizard to define your new database.
