Introduction to HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It describes the structure of web pages using markup.

Key Characteristics

  • Not a programming language, but a markup language
  • Uses tags to describe page content
  • Forms the foundation of web documents
  • Interpreted by web browsers
  • Maintained by the World Wide Web Consortium (W3C)

HTML Structure

Basic HTML Document Structure

<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Essential HTML Tags

  • <html> - Defines the root of an HTML document
  • <head> - Contains metadata about the document
  • <title> - Specifies the document title
  • <body> - Defines the document's body

HTML Tags Classification

Tag Types

Paired Tags

Have opening and closing tags

Example: <p>Text</p>

Unpaired Tags

Stand-alone tags without closing tag

Example: <br>, <hr>

Functional Categories

  • Formatting Tags: Control text appearance (<b>, <i>, <u>)
  • Structure Tags: Define document structure (<html>, <head>, <body>)
  • Form Tags: Create interactive forms (<form>, <input>)
  • Table Tags: Create tabular layouts (<table>, <tr>, <td>)
  • List Tags: Create ordered and unordered lists (<ul>, <ol>, <li>)

HTML Elements and Attributes

HTML elements consist of tags and their content, while attributes provide additional information about elements.

Attribute Syntax

<tag attribute="value">Content</tag>

Common Attributes

  • class: Specifies CSS classes
  • id: Provides unique identifier
  • style: Defines inline CSS
  • src: Specifies source for media elements
  • href: Defines hyperlink references
