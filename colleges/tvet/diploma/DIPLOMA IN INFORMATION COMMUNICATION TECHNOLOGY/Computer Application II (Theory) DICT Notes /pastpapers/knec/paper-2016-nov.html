Question 1 (15 marks)

(a) Define each of the following terms as used in financial accounting:

(i) cash account; (2 marks)

(ii) balance sheet. (2 marks)

(b) When Jane was browsing through a financial accounting application folder, she noted the files named KIKOTL.QBB and KIKOTL.QBW. Distinguish between the information in the two files. (4 marks)

(c) DesignBlue is an architectural design company for houses. The company uses computer-aided design (CAD) templates for their new tasks.

(i) Define the term template; (2 marks)

(ii) Explain benefits that the company is likely to gain from using this approach. (2 marks)

(iii) Outline three advantages the company would gain by using computer-aided design (CAD) programs instead of the manual system. (3 marks)

Answer

(a) Financial accounting terms:

(i) Cash account:

  • Record of all cash transactions
  • Shows cash receipts and payments
  • Tracks business's cash position

(ii) Balance sheet:

  • Financial statement showing financial position
  • Lists assets, liabilities, and owner's equity
  • Snapshot of business at specific point

(b) QuickBooks file types:

KIKOTL.QBB KIKOTL.QBW
Backup file Working file
Archive/safety copy Active company file
Used for data recovery Used for daily operations

(c) CAD templates:

(i) Template definition:

  • Pre-designed drawing format
  • Standard layouts and settings
  • Reusable design elements

(ii) Benefits of using templates:

  1. Increased efficiency
    • Faster project starts
    • Standardized workflow
  2. Consistency in design
    • Uniform presentation
    • Quality control

(iii) Advantages of CAD over manual system:

  1. Accuracy
    • Precise measurements
    • Error reduction
  2. Easy modifications
    • Quick changes
    • Design flexibility
  3. Storage and retrieval
    • Digital archiving
    • Easy access to past designs

Question 3 (15 marks)

(a) List four stakeholders who may have interests in financial accounting reports produced by a bank. (2 marks)

(b) Determine the owner's equity for a company which has the following balances in their books of accounts:

  • Land worth Ksh.1,000,000
  • Machinery worth Ksh.2,050,000
  • Cash amounting to Ksh. 2,900,000
  • Loan borrowed worth Ksh.5,000,000

(3 marks)

(c) Explain how artificial intelligence is used in each of the following areas:

(i) Planning. (2 marks)

(ii) Scheduling. (2 marks)

(d) Figure 2 shows a geometrical diagram drawn using a computer-aided design (CAD) program. Use it to answer the questions that follow:

(i) Identify four types of dimensions applied in the drawing. (2 marks)

(ii) Outline the steps taken to create portion of the drawing labelled X. (2 marks)

(iii) A copy of the drawing is required to be enlarged by factor 2. Describe the procedure to be followed. (2 marks)

Answer

(a) Bank report stakeholders:

  1. Shareholders
  2. Regulators (Central Bank)
  3. Customers/Depositors
  4. Potential investors

(b) Owner's equity calculation:

                Assets = Liabilities + Owner's Equity
                Land + Machinery + Cash = Loan + Owner's Equity
                1,000,000 + 2,050,000 + 2,900,000 = 5,000,000 + Owner's Equity
                5,950,000 = 5,000,000 + Owner's Equity
                Owner's Equity = 950,000

(c) AI applications:

(i) Planning:

  • Optimal path finding
  • Resource allocation
  • Strategy development

(ii) Scheduling:

  • Task optimization
  • Resource management
  • Time allocation

(d) CAD drawing:

(i) Dimension types:

  1. Linear dimensions
  2. Angular dimensions
  3. Radial dimensions
  4. Diameter dimensions

(ii) Creating portion X:

  1. Draw basic shape
  2. Use appropriate commands
  3. Add dimensions

(iii) Enlarging procedure:

  1. Select entire drawing
  2. Use SCALE command
  3. Enter scale factor 2

Question 6 (15 marks)

(a) In the following list of computer-aided design (CAD) program commands:

Explode Viewport Join Spline UCS

(i) identify those which do not alter the shape of the drawing. (1 mark)

(ii) In each case, state the use of the commands identified in (i). (1 mark)

(b) Describe each of the following pricing methods used in inventory controls:

(i) FIFO; (2 marks)

(ii) LIFO; (2 marks)

(c) Distinguish between a note and an invoice as used in accounting. (3 marks)

(d) Describe each of the following search techniques used in Artificial Intelligence:

(i) breadth-first search; (2 marks)

(ii) bidirectional search; (2 marks)

(iii) hill-climbing search. (2 marks)

Answer

(a) CAD commands:

(i) Non-shape altering commands:

  • Viewport
  • UCS (User Coordinate System)

(ii) Commands usage:

Command Use
Viewport Controls view display
UCS Changes coordinate system

(b) Inventory pricing methods:

(i) FIFO (First In, First Out):

  • Oldest inventory sold first
  • Cost flow assumption
  • Reflects physical flow

(ii) LIFO (Last In, First Out):

  • Newest inventory sold first
  • Higher cost flow in inflation
  • Tax advantages

(c) Note vs. Invoice:

Note Invoice
Written promise to pay Bill for goods/services
Single payment document Detailed transaction record
Used for loans Used for sales

(d) AI search techniques:

(i) Breadth-first search:

  • Explores all nodes at current depth
  • Complete but memory intensive

(ii) Bidirectional search:

  • Searches from both start and goal
  • Meets in the middle

(iii) Hill-climbing search:

  • Moves toward better states
  • Local optimization strategy
