Question 1 (15 marks)

(a)(i) State three examples of financial statements prepared in accounting. (3 marks)

(ii) Explain the main function of financial accounting. (2 marks)

(b) State five areas of application for each of the following types of lines in a CAD drawing:

(i) continuous-thick lines;

(ii) continuous-thin lines. (6 marks)

(c) Distinguish between vector and raster data types as used in the GIS software. (4 marks)

Answer

(a)(i) Financial statements examples:

  1. Balance Sheet/Statement of Financial Position
  2. Income Statement/Profit and Loss Account
  3. Cash Flow Statement

(ii) Main function of financial accounting:

  • Recording, analyzing, and reporting financial transactions
  • Providing accurate financial information for decision making
  • Ensuring compliance with accounting standards and regulations

(b) CAD line applications:

(i) Continuous-thick lines:

  • Visible outlines and edges
  • Object boundaries
  • Main contours
  • Primary features
  • Cut lines in sections

(ii) Continuous-thin lines:

  • Dimension lines
  • Leader lines
  • Hatching
  • Grid lines
  • Reference lines

(c) Vector vs. Raster data in GIS:

Vector Data Raster Data
Uses points, lines, and polygons Uses grid of pixels/cells
Ideal for discrete features Better for continuous data
Scale independent Resolution dependent
Smaller file size Larger file size

Question 4 (15 marks)

(a)(i) Explain the term First-In-First-Out inventory control strategy. (2 marks)

(ii) Outline a circumstance where First-In-First-Out strategy could be used in companies to control inventory. (2 marks)

(b) Outline three deductions made from company employees' salaries. (3 marks)

(c) Outline four responsibilities that an officer at the ministry of agriculture can perform using a GIS program. (4 marks)

(d) The following are the sugar stock details of a biscuit factory:

  • maximum usage of 5,000 units per week
  • maximum lead time of 5 weeks
  • minimum usage of 1,000 units per week
  • minimum lead time is 3 weeks
  • reorder quantity is 12,000

Calculate the maximum level of stocks of sugar that should be held by the factory at any given time. (4 marks)

Answer

(a)(i) FIFO strategy:

  • Items purchased first are sold first
  • Ensures oldest stock is used before newer stock

(ii) FIFO application circumstances:

  • Perishable goods management
  • Food and beverage industry
  • Products with expiration dates

(b) Salary deductions:

  1. Statutory deductions
    • Income tax (PAYE)
    • National Social Security Fund (NSSF)
    • National Health Insurance Fund (NHIF)

(c) GIS applications in agriculture:

  1. Crop monitoring
    • Track crop health
    • Monitor growth patterns
  2. Land use planning
    • Soil analysis
    • Irrigation planning
  3. Resource allocation
    • Water management
    • Fertilizer distribution
  4. Yield prediction
    • Harvest forecasting
    • Production planning

(d) Maximum stock level calculation:

                Maximum stock level = Reorder level + Reorder quantity
                
                Reorder level = Maximum usage × Maximum lead time
                             = 5,000 × 5
                             = 25,000 units
                
                Maximum stock level = 25,000 + 12,000
                                  = 37,000 units

Question 5 (15 marks)

(a) Explain the circumstance under which each of the following commands would be used in CAD application:

(i) mtext;

(ii) ortho. (4 marks)

(b) Distinguish between breadth-first search and bidirectional search techniques used in artificial intelligence. (4 marks)

(c) With the aid of a diagram, outline the features of a journal in accounting. (7 marks)

Answer

(a) CAD commands usage:

(i) Mtext (Multiline text):

  • Creating detailed text annotations
  • Adding multiple lines of text
  • Formatting text with different styles

(ii) Ortho:

  • Drawing straight horizontal/vertical lines
  • Creating perpendicular elements
  • Maintaining 90-degree angles

(b) Search techniques comparison:

Breadth-first Search Bidirectional Search
Explores all nodes at current depth Searches from both start and goal
Sequential level by level Two searches meet in middle
Complete but memory intensive More efficient for large spaces
Guarantees optimal solution Reduces search space

(c) Journal features:

                                    JOURNAL
                Date | Particulars | Folio | Debit | Credit
                -----+-------------+--------+--------+-------
                     | Entry desc. |  Ref  |  Amt  |  Amt
  • Date column: Chronological order
  • Particulars: Transaction description
  • Folio: Reference number
  • Debit column: Amount to be debited
  • Credit column: Amount to be credited
  • Narration: Brief explanation of entry
  • Double entry format

Question 6 (15 marks)

(a) Outline four challenges in the development of systems that can understand natural language in AI. (4 marks)

(b) Explain each of the following accounting terms:

(i) return inwards;

(ii) direct expenses;

(iii) carriage inwards. (6 marks)

(c) Figure 2 shows a diagram created in a CAD program. Explain the connotation of the:

(i) line segment AB; (1 mark)

(ii) hatched portions labelled C. (1 mark)

(d) Sketch the shape that would be formed from the following sequence of computer-aided design (CAD) commands and input. (3 marks)

Line Specify first point: 0,0
Specify next point or [Undo]: 0,800
Specify next point or [Undo]: 1000,800
Specify next point or [Undo]: 1000,1600
Specify next point or [Undo]: 2200,1600
Specify next point or [Undo]: 2200,0
Specify next point or [Close/Undo]: c

Answer

(a) Natural language processing challenges:

  1. Ambiguity in language
    • Multiple word meanings
    • Context-dependent interpretation
  2. Grammar complexity
    • Irregular structures
    • Language exceptions
  3. Cultural nuances
    • Idioms and expressions
    • Regional variations
  4. Context understanding
    • Situational awareness
    • Implicit meanings

(b) Accounting terms:

(i) Return inwards:

  • Goods returned by customers
  • Reduces sales revenue
  • Also called sales returns

(ii) Direct expenses:

  • Costs directly related to production
  • Can be traced to specific products
  • Part of prime cost

(iii) Carriage inwards:

  • Transportation cost of purchases
  • Added to purchase cost
  • Part of cost of goods

(c) CAD diagram elements:

(i) Line segment AB:

  • Reference line or dimension line
  • Measurement reference

(ii) Hatched portions C:

  • Cross-section area
  • Material indication

(d) CAD shape:

                2200,1600
                    +----------------+
                    |                |
                    |                |
1000,800   +--------+                |
           |                         |
           |                         |
0,0        +-------------------------+
                                 2200,0

The commands create a complex polygon with specified coordinates, closed with the 'c' command.
