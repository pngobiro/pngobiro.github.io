Question 1 (15 marks)

(a) A CAD student was advised to combine a collection of objects into a block of entity. Outline four benefits that the student would achieve from this advice. (4 marks)

(b) JMZ is a garment shop dealing with T shirts. The shop made the following transactions:

1/2/2014purchased 200 units each at Ksh. 1000
3/2/2014purchased 150 units each at Ksh. 900
5/2/2014purchased 225 units each at Ksh. 1200

(i) Determine the average cost of inventory (3 marks)

(ii) If on 7/2/2014 the company sold 250 units using the FIFO strategy, determine:

I. the cost of goods that the company sold (2 marks)

II. the cost of ending inventory. (2 marks)

(c) Below are commands and successive values entered in an AutoCAD program:

LINE
LINE Specify next point or [Undo]: 0,0
LINE Specify next point or [Undo]: 6,0
LINE Specify next point or [Undo]: 6,-3
LINE Specify next point or [Undo]: 10,-3
LINE Specify next point or [Undo]: 10,2
LINE Specify next point or [Undo]: 6,2
LINE Specify next point or [Undo]: 6,6
LINE Specify next point or [Undo]: 0,6
LINE Specify next point or [Undo]: close

Sketch the resultant drawing. (4 marks)

Answer

(a) Benefits of combining objects into blocks:

  1. Improved file management
    • Reduced file size
    • Better organization
  2. Easy modification
    • Modify all instances at once
    • Consistent changes
  3. Increased efficiency
    • Quick reuse of components
    • Faster drawing creation
  4. Standardization
    • Consistent appearance
    • Uniform elements

(b) Inventory calculations:

(i) Average cost calculation:

                Total Cost = (200 × 1000) + (150 × 900) + (225 × 1200)
                          = 200,000 + 135,000 + 270,000
                          = 605,000

                Total Units = 200 + 150 + 225 = 575

                Average Cost = 605,000 ÷ 575 = Ksh. 1052.17 per unit

(ii) FIFO calculations:

I. Cost of goods sold:

                First 200 units at 1000 = 200,000
                Next 50 units at 900 = 45,000
                Total cost = 245,000

II. Ending inventory:

                Remaining 100 units at 900 = 90,000
                225 units at 1200 = 270,000
                Total ending inventory = 360,000

(c) AutoCAD drawing:

                (0,6)---(6,6)
                   |      |
                   |      |
                (0,0)---(6,0)--(10,2)
                         |      |
                         |      |
                      (6,-3)--(10,-3)

The drawing forms an irregular shape with perpendicular and angled lines.

  2. Accounts payable

    • Overcharging correction
    • Discount application

(d) CAD benefits for architectural company:

Question 3 (15 marks)

(a)(i) Define the term artificial intelligence as used in computers. (1 mark)

(ii) Distinguish between intelligent agent and physical agent as used in artificial intelligence. (4 marks)

(b) Management of Kitungu Company was reluctant in implementing computerised accounting system. Outline four reasons that may be informing their decision. (4 marks)

(c) An Engineer intends to create geometric block in a CAD program. Outline three methods in which he can specify the locations of points in the program. (6 marks)

Answer

(a)(i) Artificial Intelligence definition:

  • Computer systems that simulate human intelligence
  • Ability to learn, reason, and self-correct
  • Problem-solving capabilities like humans

(ii) Intelligent vs. Physical agents:

Intelligent Agent Physical Agent
Software-based Hardware-based
Makes decisions Executes actions
Processes information Interacts with environment
Virtual existence Physical existence

(b) Reluctance for computerized accounting:

  1. Cost concerns
    • Implementation expenses
    • Training costs
  2. Staff resistance
    • Fear of job loss
    • Change resistance
  3. Security concerns
    • Data breaches
    • System vulnerabilities
  4. Technical complexity
    • Learning curve
    • System maintenance

(c) CAD point specification methods:

  1. Direct coordinate entry
    • Absolute coordinates (X,Y)
    • Relative coordinates (@X,Y)
  2. Mouse input
    • Click at desired location
    • Snap to grid/objects
  3. Object snaps
    • Endpoint snap
    • Midpoint snap
    • Intersection snap

Auto-CAD, Picasa, IDEAS, CATIA, GIMP Fluent, CorelDraw, Transys, Adobe Photoshop, Hyper mesh
