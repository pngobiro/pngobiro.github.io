Question 1 (4 marks)

Explain each of the following terms as used in CAD applications:

(i) spline; (2 marks)

(ii) Crosshairs. (2 marks)

Answer

(i) Spline:

  • Smooth curve through control points
  • Used for creating flowing curves
  • Mathematically defined curve shape

(ii) Crosshairs:

  • Cursor showing X and Y axes
  • Precision pointing device
  • Aids in accurate placement

Question 5 (4 marks)

(a) Jane, a new accountant has been advised to use double-entry system for her company. Outline two advantages of this move. (2 marks)

(b) State the function of each of the following dimensions used in CAD application:

(i) linear; (1 mark)

(ii) angular. (1 mark)

Answer

(a) Double-entry advantages:

  1. Error detection
    • Balance verification
    • Accuracy checking
  2. Complete records
    • Both sides of transactions
    • Better audit trail

(b) CAD dimensions:

(i) Linear dimension:

  • Measures straight distances
  • Shows horizontal/vertical measurements

(ii) Angular dimension:

  • Measures angles between lines
  • Shows rotational measurements
