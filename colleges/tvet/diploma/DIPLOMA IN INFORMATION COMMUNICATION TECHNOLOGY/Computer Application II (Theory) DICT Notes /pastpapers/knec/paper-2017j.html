INSTRUCTIONS TO CANDIDATES

Answer any FIVE of the following EIGHT questions.

Question 1 (20 marks)

(a) Outline four advantages of using computers in data processing. (4 marks)

(b) Explain each of the following data processing methods: (4 marks)

(i) Online

(ii) Real-time

Answer

(a) Advantages of using computers in data processing:

Speed: Computers process data much faster than humans.

Accuracy: Computers are less prone to errors than humans.

Storage: Computers can store large amounts of data.

Automation: Computers can automate repetitive tasks.

(b) Data processing methods:

Online: Data is processed immediately upon input.

Real-time: Data is processed as it is generated, with minimal delay.

Question 2 (20 marks)

(a) Describe two types of charts that can be created using a spreadsheet program. (4 marks)

Answer

(a) Types of charts:

Bar Chart: Used to compare values across categories.

Pie Chart: Used to show proportions of a whole.
