Question 1 (15 marks)

(a) Describe the concept of going concern as used in accounting. (2 marks)

(b) State three elements of vector data as used in GIS. (3 marks)

(c) Arthur intends to procure a computer to be used for a Turing test during an artificial intelligence practice session. Explain three properties that the computer should possess. (6 marks)

(d) Explain two sources of spatial data in GIS. (4 marks)

Answer

(a) Going concern concept:

  • Business will continue operating indefinitely
  • No intention or need to liquidate
  • Basis for asset valuation and depreciation

(b) Vector data elements:

  1. Points
    • Zero-dimensional features
    • Discrete locations
  2. Lines
    • One-dimensional features
    • Linear features
  3. Polygons
    • Two-dimensional features
    • Area features

(c) Turing test computer properties:

  1. Natural Language Processing
    • Understand human language
    • Generate coherent responses
  2. Knowledge Representation
    • Store information
    • Access relevant data
  3. Learning Capability
    • Adapt to new information
    • Improve responses

(d) Spatial data sources:

  1. Primary sources
    • GPS surveys
    • Field measurements
  2. Secondary sources
    • Existing maps
    • Satellite imagery

Question 2 (15 marks)

(a) Define the term geo-statistics as used in GIS. (2 marks)

(b) Abel intends to identify the shortest distance between four towns in his county using a search method. Outline three factors that he should consider when choosing a search method in artificial intelligence. (3 marks)

(c) With the aid of sketches, describe the following types of dimensions as used in CAD applications: (6 marks)

(i) aligned;

(ii) baseline;

(d) Differentiate between asset value and liability balances as used in accounting programs. (4 marks)

Answer

(a) Geo-statistics definition:

  • Statistical methods for spatial data analysis
  • Tools for analyzing geographic patterns
  • Spatial relationship measurements

(b) Search method factors:

  1. Problem complexity
    • Number of nodes
    • Search space size
  2. Time constraints
    • Processing speed
    • Response requirements
  3. Accuracy needs
    • Optimal vs. satisfactory solution
    • Precision requirements

(c) CAD dimensions:

(i) Aligned dimension:

  • Parallel to measured feature
  • Follows object orientation

(ii) Baseline dimension:

  • Multiple dimensions from common point
  • Uniform reference point

(d) Asset vs. Liability balances:

Asset Value Liability Balance
Resources owned Obligations owed
Debit balance Credit balance
Economic benefits Future payments
