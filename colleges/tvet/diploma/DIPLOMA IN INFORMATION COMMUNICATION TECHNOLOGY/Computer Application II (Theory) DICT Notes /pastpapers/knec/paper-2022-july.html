Question 1 (15 marks)

(a) State the meaning of each of the following terms as used in artificial intelligence:

(i) autonomous robot;

(ii) rule;

(iii) forward chaining; (3 marks)

(b) Explain each of the following terms as used in a Geographic Information System:

(i) resolution;

(ii) remote sensing. (4 marks)

(c) Distinguish between break and explode commands as used in computer-aided design (CAD) program. (4 marks)

(d) Outline four ways an agricultural extension officer could use Geographic Information System in livestock farming. (4 marks)

Answer

(a) AI terms:

(i) Autonomous robot: A robot that operates independently and makes decisions without continuous human intervention, using sensors and programming to navigate and perform tasks.

(ii) Rule: A conditional statement in AI that defines specific actions to be taken when certain conditions are met, typically in the form "IF condition THEN action".

(iii) Forward chaining: A reasoning method in AI where the system starts with known facts and applies rules to reach new conclusions, moving from premises to conclusions.

(b) GIS terms:

(i) Resolution:

  • The level of detail or precision in spatial data
  • Determines the smallest feature that can be distinguished in the data
  • Usually measured in units per pixel for raster data

(ii) Remote sensing:

  • Collection of data about objects from a distance
  • Uses sensors on satellites or aircraft
  • Captures information through electromagnetic radiation

(c) Break vs. Explode commands in CAD:

Break command:

  • Splits a single object into two parts at a specified point
  • Maintains the original properties of the object in both parts
  • Used for creating gaps or dividing lines

Explode command:

  • Breaks down a complex object into its constituent parts
  • Separates combined objects into individual elements
  • Converts blocks or polylines into basic entities

(d) GIS in livestock farming:

  1. Grazing land management
    • Mapping pasture areas
    • Monitoring grass quality and availability
  2. Disease monitoring
    • Tracking disease outbreaks
    • Planning vaccination programs
  3. Herd movement tracking
    • Planning optimal grazing routes
    • Monitoring animal distribution
  4. Resource allocation
    • Planning water points
    • Managing feeding stations

Question 2 (15 marks)

(a) Outline three types of accounts commonly found in double entry book-keeping. (3 marks)

(b) A CAD user opted to enable each of the following features in a CAD program while drawing:

(i) ortho mode;

(ii) object snap mode;

Explain the effect of each while drawing. (4 marks)

(c) Explain two processes that can be applied to vector data in a Geographic Information System. (4 marks)

(d)(i) Explain the term bank-reconciliation statement as used in accounting. (2 marks)

(ii) State a way in which each of the following items are treated in a bank reconciliation statement:

- uncredited cheques;

- bank charges. (2 marks)

Answer

(a) Types of accounts in double entry book-keeping:

  1. Personal accounts - Records transactions with individuals or organizations
  2. Real accounts - Deals with assets and properties
  3. Nominal accounts - Records income, expenses, and losses

(b) CAD features effects:

(i) Ortho mode:

  • Constrains drawing to horizontal and vertical lines
  • Forces lines to be drawn at 90-degree angles
  • Helps maintain precise perpendicular relationships

(ii) Object snap mode:

  • Automatically snaps cursor to specific points on objects
  • Ensures precise connections between elements
  • Helps maintain accuracy in drawing

(c) Vector data processes in GIS:

  1. Buffer analysis
    • Creates zones around features
    • Used for proximity analysis
  2. Overlay analysis
    • Combines multiple vector layers
    • Creates new datasets based on spatial relationships

(d) Bank reconciliation:

(i) Bank reconciliation statement:

  • A document that compares and adjusts bank statement with cash book
  • Explains differences between bank and book balances

(ii) Treatment of items:

Uncredited cheques:

  • Deducted from bank statement balance
  • Added to reconciliation as outstanding items

Bank charges:

  • Deducted from cash book balance
  • Recorded as an expense in the accounts

(b) Areas for intelligent robots in dangerous jobs:

  1. Nuclear facilities
    • Handling radioactive materials
    • Maintenance in radioactive zones
  2. Deep-sea operations
    • Underwater exploration
    • Pipeline maintenance
  3. Hazardous material handling
    • Chemical processing
    • Explosive disposal

(c) GIS components for setup:

  1. Hardware infrastructure
    • High-performance computers
    • Storage systems
    • Input/output devices
  2. Software systems
    • GIS software packages
    • Database management systems
    • Analysis tools

(d) Capital vs. Revenue expenditures:

Capital Expenditures Revenue Expenditures
Long-term investments Day-to-day operational expenses
Creates or improves assets Maintains existing assets
Benefits extend beyond one year Benefits consumed within one year
Capitalized in balance sheet Expensed in income statement

Question 5 (15 marks)

(a) The management of Movo company was advised to apply double entry bookkeeping in their accounting procedures.

(i) Explain the meaning of double entry bookkeeping. (2 marks)

(ii) Outline three benefits that the company gains from using this method of accounting. (3 marks)

(b) Describe each of the following terms as used in artificial intelligence:

(i) expert system;

(ii) supervised machine learning. (4 marks)

(c) With the aid of examples, differentiate between spatial data and non-spatial data as used in Geographic Information System. (6 marks)

Answer

(a) Double entry bookkeeping:

(i) Meaning:

  • An accounting system where each transaction affects at least two accounts
  • Every debit entry must have a corresponding credit entry

(ii) Benefits:

  1. Error detection
    • Easy to spot mistakes through trial balance
    • Ensures balanced books
  2. Complete financial picture
    • Shows both sources and uses of funds
    • Provides comprehensive view of transactions
  3. Better financial control
    • Helps prevent fraud
    • Improves accountability

(b) AI terms:

(i) Expert system:

  • Computer program that emulates decision-making of human experts
  • Uses knowledge base and inference engine
  • Provides expert-level solutions in specific domains

(ii) Supervised machine learning:

  • Training method where AI learns from labeled data
  • System is provided with input-output pairs
  • Learns to predict outputs for new inputs

(c) Spatial vs. Non-spatial data:

Spatial Data Non-spatial Data
Location-specific information Attribute information
Example: Coordinates of a building Example: Building name
Example: Road network layout Example: Road names
Example: River course Example: River flow rate

Question 6 (15 marks)

(a) State a circumstance that would necessitate the use of each of the following CAD commands:

(i) lweight;

(ii) limits;

(iii) oops. (3 marks)

(b) Explain each of the following types of learning as used in artificial intelligence:

(i) deductive learning;

(ii) abductive learning. (4 marks)

Answer

(a) CAD commands usage:

(i) lweight (lineweight):

  • When different line thicknesses are needed for clarity
  • When creating technical drawings with varying line importance

(ii) limits:

  • When setting up drawing boundaries
  • When defining the drawing area size

(iii) oops:

  • When needing to undo multiple operations
  • When reverting accidental changes

(b) Types of AI learning:

(i) Deductive learning:

  • Learning through logical reasoning from general to specific
  • Uses established rules to reach conclusions
  • Based on proven principles and facts

(ii) Abductive learning:

  • Learning by forming best explanations for observations
  • Makes educated guesses based on incomplete information
  • Involves inference to the best explanation
