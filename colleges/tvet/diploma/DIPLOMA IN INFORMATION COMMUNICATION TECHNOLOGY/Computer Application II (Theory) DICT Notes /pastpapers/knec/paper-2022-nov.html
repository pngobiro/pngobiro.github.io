Question 1 (15 marks)

(a) Categorize each of the following items as either Income (I), Expense (E), Asset (A), Liability (L), or Capital (C):

(i) Accounts payable

(ii) Rent received

(iii) Cash

(iv) Drawings

(v) Building

(vi) Fees received (3 marks)

(b) Distinguish between blind search and uninformed search as used in artificial intelligence. (4 marks)

(c) Sketch the output that would be generated when the following sequence of commands in a Computer Aided Design program are executed:

multiple _pline
0,0
50,50
80,80
0,80
0,0

(4 marks)

(d) Explain two ways in which Geographic Information Systems (GIS) could be used in the execution of rescue missions. (4 marks)

Answer

(a) Item categorization:

  1. Accounts payable - (L) Liability
  2. Rent received - (I) Income
  3. Cash - (A) Asset
  4. Drawings - (C) Capital
  5. Building - (A) Asset
  6. Fees received - (I) Income

(b) Blind Search vs. Uninformed Search:

These are actually the same thing, often used interchangeably, but with the following characteristics:

  • No additional information about the search space beyond the problem definition
  • Explores nodes without any estimate of their proximity to the goal
  • Examples include breadth-first and depth-first search
  • Contrasts with informed search which uses heuristic information

(c) Output sketch:

      (80,80)
     /     |
    /      |
   /       |
  /        |
(0,0)-----(0,80)

This creates a triangular shape with a vertical line on the right side.

(d) GIS in rescue missions:

  1. Route Optimization
    • Determining fastest paths to reach victims
    • Avoiding obstacles and hazardous areas
  2. Resource Management
    • Tracking rescue team locations
    • Coordinating emergency response units

Question 2 (15 marks)

(a) Explain the application of artificial intelligence in each of the following areas:

(i) email filters;

(ii) search engines;

(iii) e-commerce website. (6 marks)

(b) The following information relates to Tomina Traders stock records for the year 2021:

  • Stock as at 01/01/2021: Ksh 3,323,500
  • Purchases: Ksh 25,027,000
  • Returns outwards: Ksh 1,246,000
  • Carriage inwards: Ksh 57,000
  • Carriage outwards: Ksh 538,000
  • General distribution expenses: Ksh 284,000
  • Stock as at 31/12/2021: Ksh 4,589,000

Determine the cost of sales for the year for Tomina Traders. (5 marks)

(c) State one circumstance necessitating enabling of each of the following features in a CAD program:

(i) object snap;

(ii) dynamic input. (4 marks)

Answer

(a) AI applications:

Email filters:

  • Spam detection using machine learning
  • Message categorization into primary, social, promotions

Search engines:

  • Natural language processing for query understanding
  • Ranking algorithms based on relevance and user behavior

E-commerce website:

  • Personalized product recommendations
  • Customer behavior prediction and analysis

(b) Cost of Sales calculation:

Opening Stock (01/01/2021) 3,323,500
Add: Purchases 25,027,000
Add: Carriage inwards 57,000
Less: Returns outwards (1,246,000)
Less: Closing Stock (31/12/2021) (4,589,000)

Note: Carriage outwards and general distribution expenses are not included in cost of sales calculation.

(c) CAD features usage:

Object snap:

  • When precise connections between objects are required
  • When drawing elements need to align perfectly

Dynamic input:

  • When working with precise measurements and angles

Question 3 (15 marks)

(a) Explain the function of each of the following objects in a GIS:

(i) coordinate data objects;

(ii) vector objects. (4 marks)

(b) Outline the function of each of the following financial statements:

(i) balance sheet;

(ii) profit and loss account;

(iii) cash flow statement. (3 marks)

(c) Explain a reason for switching between model space window and paper space window while designing in a CAD program. (4 marks)

(d) State the relationship between machine learning and artificial intelligence. (4 marks)

Answer

(a) GIS objects:

Coordinate data objects:

  • Store location information in x, y (and sometimes z) coordinates
  • Define precise positions of geographic features

Vector objects:

  • Represent features using points, lines, and polygons
  • Store spatial data in geometric form with associated attributes

(b) Financial statements:

Balance sheet:

  • Shows financial position at a specific point in time
  • Lists assets, liabilities, and owner's equity

Profit and loss account:

  • Shows financial performance over a period
  • Details income, expenses, and resulting profit or loss

Cash flow statement:

  • Shows movement of cash in and out of business
  • Categorizes cash flows into operating, investing, and financing activities

(c) Model space vs. Paper space:

  • Model space is used for creating and editing the actual design
  • Paper space is used for layout and printing preparation
  • Switching allows separation of design work from presentation setup
  • Enables multiple views and scales of the same design

(d) Machine Learning and AI relationship:

  • Machine learning is a subset of artificial intelligence
  • AI is the broader concept of machines being able to carry out tasks intelligently
  • Machine learning focuses on the ability to learn from data without explicit programming
  • ML provides the learning capabilities that make AI systems adaptive and improve over time

Question 4 (15 marks)

(a) Explain each of the following types of Artificial Intelligence (AI) used in computers:

(i) reactive machines AI;

(ii) limited memory AI;

(iii) self-aware AI. (6 marks)

(b) Differentiate between first in first out (FIFO) and last in first out (LIFO) stock valuation methods. (4 marks)

(c) With the aid of an illustration in each case, describe the following types of dimensions used in CAD design:

(i) radial;

(ii) arc length. (5 marks)

Answer

(a) Types of AI:

Reactive machines AI:

  • Most basic form of AI
  • Responds to current situations without past memory
  • Cannot learn from experience or use past information

Limited memory AI:

  • Can use past experiences to inform future decisions
  • Stores data for a limited time
  • Example: Self-driving cars using recent driving data

Self-aware AI:

  • Theoretical type of AI with consciousness
  • Understanding of own existence
  • Currently does not exist in practice

(b) FIFO vs LIFO:

FIFO LIFO
First items purchased are first ones sold Most recently purchased items are sold first
Better reflects physical flow of goods May not match actual movement of inventory

Question 5 (15 marks)

(a) State two differences between human intelligence and artificial intelligence. (4 marks)

(b) Explain the term intelligent agent as used in Artificial Intelligence. (2 marks)

(c) State four advantages of using vector data in Geographic Information System. (4 marks)

(d) Dalili Ltd acquired a new mainframe computer system for Ksh 600,000 on 1 November 2019. The computer's estimated useful life is five years, at the end of which it is expected to have a scrap value of Ksh 54,500.

If the company's financial year ends on 31 March and straight-line depreciation is applied on a monthly basis, what is the depreciation charge on the computer in the profit and loss account for the year ending 31 March 2021? (5 marks)

Answer

(a) Human vs Artificial Intelligence:

Human Intelligence Artificial Intelligence
Learns through experience and adaptation Learns through programmed algorithms and data
Has emotional and creative capabilities Limited to logical and programmed responses

(b) Intelligent Agent:

  • A system that perceives its environment and takes actions to maximize its chances of success
  • Operates autonomously and adapts to changes in its environment

(c) Advantages of vector data in GIS:

  1. Compact data storage
  2. Maintains quality when scaled
  3. Precise representation of boundaries
  4. Easy to edit and manipulate

(d) Depreciation calculation:

Step 1: Calculate annual depreciation

  • Cost: Ksh 600,000
  • Scrap value: Ksh 54,500
  • Depreciable amount: Ksh 545,500
  • Useful life: 5 years
  • Annual depreciation: Ksh 109,100 (545,500 ÷ 5)
  • Monthly depreciation: Ksh 9,091.67 (109,100 ÷ 12)

Year ending 31 March 2021:

  • 12 months × Ksh 9,091.67
  • = Ksh 109,100

Question 6 (15 marks)

(a) Outline three functions of a Geographical Information System (GIS). (3 marks)

(b) A designer opted to use the block feature in a CAD program while creating a design. Outline two reasons for this preference. (2 marks)

(c) Skyweb GIS Company has been advised to hire the following categories of employees:

(i) cartographers;

(ii) GIS Programmer;

(iii) remote sensing specialists.

Explain the services that each of the above employees could provide to the company. (6 marks)

(d) Explain each of the following accounting terms:

(i) provision for depreciation;

(ii) historical cost accounting basis. (4 marks)

Answer

(a) GIS Functions:

  1. Data capture and storage of geographic information
  2. Spatial analysis and mapping
  3. Data visualization and presentation

(b) Reasons for using blocks in CAD:

  1. Reusability - Same components can be used multiple times
  2. Efficiency - Changes to block affect all instances simultaneously

(c) Employee roles:

Cartographers:

  • Create and design maps
  • Ensure proper visual representation of geographic data

GIS Programmer:

  • Develop GIS applications and tools
  • Maintain and customize GIS software

Remote Sensing Specialists:

  • Analyze satellite and aerial imagery
  • Extract and interpret geographic data from remote sources

(d) Accounting terms:

Provision for depreciation:

  • Accumulated amount set aside to account for asset value reduction
  • Represents total depreciation charged to date

Historical cost accounting basis:

  • Records assets at original purchase cost
  • Maintains this value regardless of current market value
