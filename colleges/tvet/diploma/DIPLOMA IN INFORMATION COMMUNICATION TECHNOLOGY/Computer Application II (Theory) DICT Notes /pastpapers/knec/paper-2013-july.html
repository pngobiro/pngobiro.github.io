Question 1 (15 marks)

(a) Explain each of the following intelligence agent system qualities:

(i) autonomics; (2 marks)

(ii) adaptivity; (2 marks)

(iii) sociability. (2 marks)

(b) Define each of the following terms as used in financial accounting:

(i) capital (1 mark)

(ii) balancing; (1 mark)

(iii) journal; (1 mark)

(iv) ledger. (1 mark)

(c) Figure 1, Figure 2 and Figure 3 show the front, end and plan elevations in third angle projection of a solid block respectively. Sketch the isometric view of the solid taking X as the lowest point in the drawing. (5 marks)

Answer

(a) Intelligence agent qualities:

(i) Autonomics:

  • Self-governing behavior
  • Independent decision-making
  • No direct human intervention

(ii) Adaptivity:

  • Ability to learn from experience
  • Modify behavior based on feedback
  • Environment responsiveness

(iii) Sociability:

  • Interaction with other agents
  • Communication capabilities
  • Collaborative behavior

(b) Financial accounting terms:

(i) Capital:

  • Owner's investment in business
  • Net worth of business

(ii) Balancing:

  • Equalizing debits and credits
  • Finding account differences

(iii) Journal:

  • Chronological record of transactions
  • Primary book of entry

(iv) Ledger:

  • Book of final entry
  • Collection of accounts

(c) Isometric drawing:

[Note: A proper isometric drawing would show:

  • 30-degree angles for isometric axes
  • Point X at lowest position
  • Proper proportions from given views
  • Hidden lines where needed
  • Consistency with front, end, and plan views

Question 2 (15 marks)

(a)(i) List two examples of accounting software other than quick books. (1 mark)

(ii) Explain one advantage of using accounting software programs in bookkeeping. (2 marks)

(b) Outline three differences between vector and raster data types as used in Geographic Information Systems (GIS). (6 marks)

(c) Table 1 shows the financial status of three business ventures owned by John as at 31st July 2011.

Assets Liabilities Capital
1 25,000 14,000 A
2 35,000 B 2,000
3 C 300,000 350,000

Answer

(a)(i) Accounting software examples:

  1. Sage
  2. Tally

(ii) Accounting software advantage:

  • Automated calculations
  • Reduced error rate
  • Time efficiency

(b) Vector vs. Raster data differences:

Vector Data Raster Data
Points, lines, polygons Grid cells/pixels
Scalable without quality loss Resolution dependent
Small file size Larger file size

(c) Financial calculations:

Using Assets = Liabilities + Capital:

                1. 25,000 = 14,000 + A
                   A = 11,000

                2. 35,000 = B + 2,000
                   B = 33,000

                3. C = 300,000 + 350,000
                   C = 650,000

                Totals:
                Total Assets = 25,000 + 35,000 + 650,000 = 710,000
                Total Liabilities = 14,000 + 33,000 + 300,000 = 347,000
                Total Capital = 11,000 + 2,000 + 350,000 = 363,000
