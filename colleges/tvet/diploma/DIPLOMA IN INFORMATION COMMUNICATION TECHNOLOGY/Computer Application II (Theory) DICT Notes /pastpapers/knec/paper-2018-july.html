Question 2 (15 marks)

(a) Explain the function of each of the following commands as used in a CAD program:

(i) Mirror; (2 marks)

(ii) Fillet. (2 marks)

(b) Describe the most appropriate procedure to be followed in order to obtain figure 2 from figure 1 other than using copy command in a CAD program. (3 marks)

(c) With the aid of example, distinguish between Vector data and attribute data as used in Geographical Information Systems. (5 marks)

(d) The goal of artificial Intelligence is to make machines that have human capabilities by improving their computational abilities. State three challenges that face AI in this quest. (3 marks)

Answer

(a) CAD commands:

(i) Mirror command:

  • Creates a reversed copy of selected objects
  • Uses a mirror line as reflection axis
  • Maintains symmetry in designs

(ii) Fillet command:

  • Creates rounded corners between intersecting lines
  • Smooths sharp corners with specified radius
  • Useful for creating realistic transitions

(b) Procedure to obtain figure 2:

  1. Use array command
    • Select polar array option
    • Specify center point
    • Set number of copies and angle
  2. Alternative method: Rotate and copy
    • Select original object
    • Rotate with copy option
    • Specify rotation angle

(c) Vector vs. Attribute data:

Vector Data Attribute Data
Spatial information Descriptive information
Example: Road network Example: Road name, type
Points, lines, polygons Properties, characteristics
Location data Associated information

(d) AI challenges:

  1. Natural language understanding
    • Context interpretation
    • Language ambiguity
  2. Pattern recognition
    • Complex visual processing
    • Real-world variability
  3. Common sense reasoning
    • General knowledge application
    • Contextual decision making

Question 3 (15 marks)

(a) Outline the function of each of the following personnel in Geographic Information Systems industry: (4 marks)

(i) Cartographers;

(ii) Database managers;

(iii) Programmers;

(iv) Spatial analysts;

(b) Figure 3 shows a drawing created using a CAD program. Indicate three types of dimensions and their respective sections where they can be used in the drawing. (3 marks)

(c) With the aid of an example, explain the function of each of the following features on a Geographic Information System map: (4 marks)

(i) lines;

(ii) areas.

(d) State two differences between a trading account and a profit and loss account. (4 marks)

Answer

(a) GIS personnel functions:

(i) Cartographers:

  • Design and create maps
  • Ensure map accuracy and readability
  • Apply cartographic principles

(ii) Database managers:

  • Maintain spatial databases
  • Ensure data integrity
  • Manage data access and security

(iii) Programmers:

  • Develop GIS applications
  • Create custom tools and scripts
  • Maintain system functionality

(iv) Spatial analysts:

  • Perform spatial analysis
  • Create analytical models
  • Interpret spatial patterns

(b) CAD dimension types:

  1. Linear dimensions
    • For straight lengths
    • Horizontal and vertical measurements
  2. Radial dimensions
    • For circles and arcs
    • Diameter and radius measurements
  3. Angular dimensions
    • For angles between lines
    • Angular measurements

(c) GIS map features:

(i) Lines:

  • Represent linear features
  • Example: Roads, rivers, boundaries
  • Show connections and routes

(ii) Areas:

  • Represent regions or polygons
  • Example: Land parcels, administrative areas
  • Show coverage and extent

(d) Trading vs. Profit and Loss Account:

Trading Account Profit and Loss Account
Shows gross profit/loss Shows net profit/loss
Deals with direct costs Deals with indirect costs
Related to trading activities Related to all business activities
First part of final accounts Second part of final accounts

Question 4 (15 marks)

(a) Outline the function of each of the following basic elements of Geographic Information Systems: (6 marks)

(i) Software;

(ii) Hardware;

(iii) Procedures/Methods;

(b) Explain the term contra entry as used in the cash book. (3 marks)

(c) Explain three reasons that cause the difference between cash book and bank statement balances. (6 marks)

Answer

(a) GIS basic elements:

(i) Software:

  • Processes spatial data
  • Provides analysis tools
  • Enables map creation and visualization

(ii) Hardware:

  • Physical computing components
  • Input/output devices
  • Storage and processing units

(iii) Procedures/Methods:

  • Standardized workflows
  • Data collection protocols
  • Analysis techniques

(b) Contra entry:

  • An entry that appears on both sides of the cash book
  • Used to record direct transfers between cash and bank accounts
  • Maintains double-entry principle without affecting other accounts

(c) Cash book and bank statement differences:

  1. Timing differences
    • Outstanding cheques
    • Deposits in transit
    • Different recording dates
  2. Direct bank entries
    • Bank charges
    • Standing orders
    • Direct debits/credits
  3. Errors
    • Bank errors
    • Cash book errors
    • Recording mistakes

Question 6 (15 marks)

(a) Explain the function of each of the following drawing spaces used in a CAD program: (4 marks)

(i) model-space;

(ii) paper-space.

(b) Outline five benefits of using paper-space feature when creating a drawing in a CAD program. (5 marks)

(c) Explain three bookkeeping errors that can be revealed when a trial balance balances. (6 marks)

Answer

(a) CAD drawing spaces:

(i) Model-space:

  • Primary drawing environment
  • Creates actual size models
  • Full 3D modeling capability

(ii) Paper-space:

  • Layout and plotting environment
  • Creates multiple views
  • Manages print layouts

(b) Paper-space benefits:

  1. Multiple views
    • Different scale views
    • Various perspectives
  2. Layout flexibility
    • Custom page setups
    • Title block integration
  3. Print optimization
    • Scale control
    • Plot settings
  4. Annotation management
    • Dimension control
    • Text placement
  5. Drawing organization
    • Multiple layouts
    • Sheet management

(c) Bookkeeping errors despite balanced trial balance:

  1. Complete omission
    • Transaction not recorded at all
    • Both debit and credit missing
  2. Equal errors
    • Same amount wrong on both sides
    • Balancing maintained despite error
  3. Compensating errors
    • Different errors that cancel out
    • Net effect maintains balance
