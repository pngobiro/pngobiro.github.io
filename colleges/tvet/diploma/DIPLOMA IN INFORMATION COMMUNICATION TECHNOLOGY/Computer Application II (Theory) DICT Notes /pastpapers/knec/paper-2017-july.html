Question 1 (15 marks)

(a) State a command that an architect would use to perform each of the following tasks in a CAD program:

(i) Separate the line segments of a square object;

(ii) Connect two straight edges with an arc that is tangent to the objects which has a specified radius;

(iii) Group selected lines as one entity;

(iv) Measure the distance between two points;

(v) Draw a regular pentagon. (5 marks)

(b) The government has been advised to install a virtual reality system for training her citizens on disaster preparedness.

(i) Outline two output devices that the system should have, giving an example in each case. (4 marks)

(ii) Outline two examples of software development tools, which could be used to develop the system. (2 marks)

(c) Explain two disadvantages of first in first out pricing method of an inventory control system. (4 marks)

Answer

(a) CAD commands:

  1. Separate line segments: EXPLODE command
    • Breaks compound objects into basic elements
  2. Connect with tangent arc: FILLET command
    • Creates rounded corners with specified radius
  3. Group lines: GROUP command
    • Combines multiple objects into single entity
  4. Measure distance: DIST command
    • Calculates distance between points
  5. Draw pentagon: POLYGON command
    • Creates regular polygon with 5 sides

(b) Virtual Reality system:

(i) Output devices:

  1. Visual output
    • VR headsets/HMDs
    • Large display screens
  2. Audio output
    • Spatial audio speakers
    • 3D positional headphones

(ii) Software development tools:

  1. Unity3D
  2. Unreal Engine

(c) FIFO disadvantages:

  1. Cost tracking complexity
    • Requires detailed record keeping
    • Multiple price layers to track
  2. Market value mismatch
    • Historical costs may not reflect current values
    • Can understate inventory value in inflation

Loan Kshs. 500,000; Land Kshs. 1,500,000; Cash Kshs 20,000; Owners equity Kshs. 900,000; Machinery Kshs. 80,000 and creditors.

Question 5 (15 marks)

(a) Outline six properties that could be set on a line in a CAD program. (3 marks)

(b) Explain three reasons for the increased use of robots in manufacturing plants. (6 marks)

(c) Table 1 shows the usage of stock items A and B. Calculate the Minimum stock level for each stock item. (6 marks)

Item A B
Ordering quantities 500 units 900 units
Maximum reorder period for emergency 2 weeks 2 weeks
Normal usage 90 units per week 120 units per week

Answer

(a) Line properties in CAD:

  1. Color
  2. Line type (solid, dashed, etc.)
  3. Line weight (thickness)
  4. Layer assignment
  5. Scale factor
  6. Transparency

(b) Reasons for increased robot use:

  1. Improved efficiency
    • 24/7 operation capability
    • Consistent performance
  2. Cost reduction
    • Lower labor costs
    • Reduced errors
  3. Safety improvement
    • Handling hazardous materials
    • Working in dangerous environments

(c) Minimum stock level calculation:

                For Item A:
                Minimum stock = Normal usage × Maximum reorder period
                             = 90 × 2
                             = 180 units

                For Item B:
                Minimum stock = Normal usage × Maximum reorder period
                             = 120 × 2
                             = 240 units

Question 6 (15 marks)

(a) Explain each of the following terms as used in expert systems:

(i) forward chaining;

(ii) priori knowledge. (4 marks)

(b) Distinguish between spatial and attribute data as used in GIS, giving an example in each case. (5 marks)

(c) Explain each of the following terms as used in financial accounting, giving an example in each case:

(i) asset;

(ii) liability;

(iii) expense. (6 marks)

Answer

(a) Expert system terms:

(i) Forward chaining:

  • Data-driven reasoning process
  • Starts with available data and works toward conclusions
  • Uses IF-THEN rules to reach goals

(ii) Priori knowledge:

  • Knowledge independent of experience
  • Built-in rules and facts
  • Fundamental system knowledge

(b) Spatial vs. Attribute data:

Spatial Data Attribute Data
Location information Descriptive information
Example: Road network Example: Road name, type
Example: Building location Example: Building use, age

(c) Financial accounting terms:

(i) Asset:

  • Resource owned by business
  • Has economic value
  • Example: Buildings, equipment

(ii) Liability:

  • Obligation to pay
  • Future economic sacrifice
  • Example: Loans, accounts payable

(iii) Expense:

  • Cost of operations
  • Reduces owner's equity
  • Example: Rent, salaries
