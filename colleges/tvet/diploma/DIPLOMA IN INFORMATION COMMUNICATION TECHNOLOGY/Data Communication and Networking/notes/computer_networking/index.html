CHAPTER 2: COMPUTER NETWORKING

Duration: Estimated 4 hours Level: Beginner

Course Overview

This course unit specifies the competencies required to perform computer Networking. It involves Identification of network types, Connection of networking devices, configuration of network devices, network testing, configuration of LAN network type and monitoring network connectivity.

Course Modules

  1. Module 1: Identify Network Type and Components

    Learn to identify different types of networks, network components, topologies, transmission media, and the benefits of computer networking.

    Estimated 1 hour

  2. Module 2: Connect Network Devices

    Understand network connection media, characteristics, and how to connect devices to a network.

    Estimated 1 hour

  3. Module 3: Configure Network Devices

    Explore network software installation, TCP/IP and Ethernet architectures, and device configuration.

    Estimated 1 hour

  4. Module 4: Configure LAN Network Types

    Learn to set up and configure Local Area Networks, including components and network protocols.

    Estimated 1 hour

  5. Module 5: Perform Network Testing

    Understand network testing plans, tools, component testing, and network performance evaluation.

    Estimated 30 minutes

Prerequisites

  • Basic computer literacy.
  • Familiarity with operating systems (Windows recommended).
