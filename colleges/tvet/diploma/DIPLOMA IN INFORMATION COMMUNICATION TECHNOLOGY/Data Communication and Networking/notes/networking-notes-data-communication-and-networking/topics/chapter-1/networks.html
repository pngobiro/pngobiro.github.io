Networks

A network is a set of devices (often referred to as nodes) connected by communication links. A node can be a computer, printer, or any other device capable of sending and/or receiving data generated by other nodes on the network.

📊 Network Criteria

A network must be able to meet certain criteria. The most important of these are performance, reliability, and security.

Performance

  • ⏱️
    Transit Time: The amount of time required for a message to travel from one device to another.
  • ⚡
    Response Time: The elapsed time between an inquiry and a response.
  • 📈
    Factors Affecting Performance: Number of users, type of transmission medium, capabilities of connected hardware, and efficiency of software.

Reliability

  • 🔄
    Measured by frequency of failure, recovery time from failures, and network robustness in catastrophic situations.

Security

  • 🔒
    Protection: Securing data from unauthorized access and damage
  • 🛡️
    Implementation: Policies and procedures for recovery from breaches and data losses

🔌 Types of Connections

A network is two or more devices connected through links. A link is a communications pathway that transfers data from one device to another. There are two possible types of connections: point-to-point and multipoint.

Point-to-Point Connection

  • 🔗
    Provides a dedicated link between two devices, with the entire capacity reserved for transmission between those two devices.
  • 📡
    Implementation: Usually uses actual wire/cable, but can also use microwave or satellite links
  • 🎮
    Example: TV remote control establishing a point-to-point connection with the television's control system

Multipoint Connection

  • 🌐
    More than two specific devices share a single link
  • ⚡
    Channel Sharing:
    • Spatially shared: Several devices can use the link simultaneously
    • Time-shared: Users must take turns using the connection
📝

Key Points to Remember

  • 🔑

    Network performance is measured through transit time and response time.

  • 🔑

    Reliability considers failure frequency, recovery time, and catastrophic robustness.

  • 🔑

    Point-to-point connections provide dedicated links, while multipoint connections allow resource sharing.
