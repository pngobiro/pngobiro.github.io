Question 6

(a) Outline two parts of Ethernet MAC address.

(b) For each of the following functions of Ethernet frame, identify the field responsible:

  1. detects errors in an Ethemet frame;
  2. describes which higher-level protocol is used;
  3. notifies the destination to get ready;
  4. shows frame MAC address of the origin.

(c) Carrier Sense Media Access/Collusion Detection uses several steps that guarantee delivery of data packet from sender to receiver host. Outline four such steps.

(d) (i) Outline four factors that should be considered before installing communication software in a computer.

(ii) Margret was diagnosing a tablet that was not connecting to the company's WIFI. Outline three possible causes and solutions for the problem.

Answer

(a) Parts of Ethernet MAC address:

  • Organizationally Unique Identifier (OUI)
  • Network Interface Controller specific

(b) Functions of Ethernet frame:

  • (i) detects errors in an Ethernet frame: Frame Check Sequence (FCS)
  • (ii) describes which higher-level protocol is used: Type/Length field
  • (iii) notifies the destination to get ready: Preamble
  • (iv) shows frame MAC address of the origin: Source Address

(c) Steps that guarantee delivery of data packet:

  • Sender listens for carrier
  • If channel is idle, sender transmits
  • If collision occurs, sender stops transmitting
  • Sender waits random time before retransmitting

(d) (i) Factors to consider before installing communication software:

  • System requirements
  • Compatibility
  • Security
  • Licensing

(ii) Possible causes and solutions for tablet not connecting to WIFI:

  • Cause: Incorrect password. Solution: Verify and re-enter password.
  • Cause: Weak signal. Solution: Move closer to router.
  • Cause: Tablet not configured for DHCP. Solution: Configure tablet to use DHCP.

Question 8

(a) (i) Outline two characteristics of WAN.

(ii) Distinguish between unicast IPv6 address and anycast IPv6 address.

(b) Paul noted that Cyclic Redundancy Check (CRC) is an error-detecting scheme that is commonly used in data communication systems. Outline four advantages that could be influencing this trend.

(c) Every Ethernet uses Address Resolution Protocol for it to function properly. Explain two functions of this protocol.

(d) Figure 2 shows a TCP/IP model. Use it to answer the questions that follows.

(i) Explain the function of the layers labelled (i) and (ii).

(ii) List four protocols that could be used in the application layer.

Answer

(a) (i) Characteristics of WAN:

  • Large geographical area
  • High bandwidth

(ii) Distinguish between unicast IPv6 address and anycast IPv6 address:

  • Unicast: One-to-one communication
  • Anycast: One-to-nearest communication

(b) Advantages that could be influencing this trend:

  • High reliability
  • Efficient error detection
  • Simple implementation
  • Suitable for burst errors

(c) Functions of Address Resolution Protocol:

  • Resolves IP addresses to MAC addresses
  • Maintains a table of IP-to-MAC address mappings

(d) (i) Function of the layers labelled (i) and (ii):

  • (i) Transport Layer: Provides reliable data transfer
  • (ii) Internet Layer: Handles routing of data packets

(ii) List four protocols that could be used in the application layer:

  • HTTP
  • FTP
  • SMTP
  • DNS
