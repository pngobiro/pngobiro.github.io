Question 1

(a) (i) State two advantages of multiplexing in data communication systems. (2 marks)

(ii) Distinguish between frequency division multiplexing (FDM) and time division multiplexing (TDM).

(b) Explain two types of transmission impairment.

(c) Mary designed a network for her company by connecting the existing Local Area Networks to the Internet. Explain the reason for using the following hardware devices in the task:

  • (i) bridge;
  • (ii) router;
  • (iii) NIC.

(d) Bidii Company Ltd. is considering replacing their network cables with fibre optic cables. Outline four challenges that they are likely to face.

Answer

(a)

(i) Advantages of multiplexing in data communication systems:

  • Efficient utilization of bandwidth - allows multiple signals to share a single transmission medium
  • Cost-effective - reduces the number of physical transmission lines needed

(ii) Distinction between FDM and TDM:

  • Frequency Division Multiplexing (FDM):
    • Divides bandwidth into frequency ranges
    • Each signal uses its own frequency band
    • Signals transmit simultaneously
    • Used in analog systems like radio/TV
  • Time Division Multiplexing (TDM):
    • Divides transmission time into slots
    • Each signal gets full bandwidth for its time slot
    • Signals transmit sequentially
    • Used in digital systems

(b)

Types of transmission impairment:

  • Attenuation: Signal weakening over distance
    • Signal loses power as it travels
    • Requires amplifiers or repeaters
    • More significant at higher frequencies
  • Noise: Unwanted signals that interfere with data
    • Thermal noise from electronic components
    • Electromagnetic interference (EMI)
    • Cross-talk from adjacent cables

(c)

Hardware devices and their purposes:

  • Bridge:
    • Connects and filters traffic between LANs
    • Reduces collision domains
    • Operates at Data Link Layer (Layer 2)
    • Uses MAC addresses for filtering
  • Router:
    • Connects different networks (LAN to Internet)
    • Routes packets using IP addresses
    • Determines optimal path for data
    • Provides network security
  • NIC (Network Interface Card):
    • Physical interface between computer and network
    • Contains unique MAC address
    • Handles data conversion
    • Manages network protocol implementation

(d)

Challenges of replacing network cables with fiber optic cables:

  1. High Initial Cost:
    • Expensive cables and connectors
    • Specialized equipment needed
    • Higher installation costs
  2. Installation Expertise:
    • Requires specialized training
    • Complex splicing and termination
    • Precise alignment needed
  3. Physical Vulnerability:
    • More fragile than copper cables
    • Susceptible to physical damage
    • Special handling required
  4. Equipment Compatibility:
    • May need new network hardware
    • Interface conversion required
    • Legacy system integration challenges

Question 2

(a) Explain each of the following terms as used in feedback-based flow control techniques of transport protocol:

  • (i) stop-and-wait;
  • (ii) sliding window.

(b) Distinguish between peer-to-peer and client-server networks.

(c) Excel Company Ltd. uses wireless transmission media to communicate to their clients using broadcast method.

  • (i) Explain the term broadcast;
  • (ii) Describe two types of wireless transmission that could be used by the company.

(d) Explain three circumstances that would necessitate the implementation of encryption network security system.

Answer

(a) Flow control techniques:

  • Stop-and-wait:
    • Basic flow control method where sender transmits one frame and waits
    • Next frame sent only after receiving acknowledgment (ACK)
    • Simple but inefficient for high bandwidth networks
    • Suitable for simple, low-speed connections
  • Sliding window:
    • Advanced flow control allowing multiple frames in transit
    • Uses window size to limit unacknowledged frames
    • Improves throughput and efficiency
    • Better utilization of available bandwidth

(b) Peer-to-peer vs Client-server networks:

Peer-to-peer Networks Client-server Networks
Each computer has equal status and capabilities Clear distinction between servers and clients
Decentralized resource management Centralized resource management
No dedicated server required Dedicated servers provide resources
Better for small networks Better for large networks

(c)

(i) Broadcast:

  • Communication method where data is sent to all devices on network
  • Single transmission reaches multiple recipients
  • Uses broadcast address (all 1s in host portion)
  • Efficient for one-to-many communication

(ii) Types of wireless transmission:

  1. Radio Frequency (RF):
    • Uses electromagnetic waves (3 kHz to 300 GHz)
    • Good for wide area coverage
    • Can penetrate walls and obstacles
    • Used in Wi-Fi, mobile networks
  2. Microwave:
    • Higher frequency transmission (1-300 GHz)
    • Requires line of sight
    • Higher bandwidth capacity
    • Used for point-to-point links

(d) Circumstances requiring encryption:

  1. Sensitive Data Transmission:
    • Financial transactions
    • Personal information transfer
    • Confidential business data
  2. Compliance Requirements:
    • Data protection regulations
    • Industry standards (PCI-DSS)
    • Privacy laws
  3. Security Threats:
    • Man-in-the-middle attacks
    • Data interception risks
    • Network eavesdropping

Question 3

(a) (i) Outline two components that can be used as Data Terminating Equipment in a computer network.

(ii) Explain each of the following error detection methods:

  • (I) parity check;
  • (II) cyclic redundancy check.

(b) (i) An organization that is located in ABY building has designed their network using 10 Base T Ethernet Standard. Explain two characteristics of 10 base \(T\) cable in Ethernet technology.

(ii) Explain two circumstances that would necessitate the use of Bluetooth in an office.

(c) With an aid of an example in each case, explain three modes of data transmission. (6 marks)

Answer

(a)

(i) Data Terminating Equipment (DTE) components:

  • Computers/Workstations:
    • End devices that process and generate data
    • Interface with users and applications
    • Connect to network through NICs
  • Printers/Terminals:
    • Output devices in the network
    • Process received data
    • Network-enabled peripherals

(ii) Error detection methods:

  • Parity Check:
    • Adds one bit to make number of 1s even or odd
    • Simple but only detects single-bit errors
    • Cannot detect multiple bit errors
    • Used in simple serial communications
  • Cyclic Redundancy Check (CRC):
    • Uses polynomial division
    • Generates checksum for data block
    • Can detect burst errors
    • More reliable than parity check

(b)

(i) 10 Base T Ethernet cable characteristics:

  • Physical Characteristics:
    • Uses Category 3 or higher UTP cable
    • Maximum segment length of 100 meters
    • RJ-45 connectors for termination
  • Transmission Characteristics:
    • 10 Mbps data rate
    • Baseband transmission
    • Manchester encoding

(ii) Circumstances for Bluetooth use in office:

  • Mobile Device Integration:
    • Wireless headsets for calls
    • Mobile phone connectivity
    • Tablet/smartphone data transfer
  • Peripheral Connectivity:
    • Wireless keyboards and mice
    • Portable printers
    • Presentation devices

(c) Data transmission modes with examples:

  1. Simplex Transmission:
    • One-way communication only
    • Example: TV broadcasting - station transmits, viewers only receive
    • No feedback or return channel
  2. Half-duplex Transmission:
    • Two-way communication, one direction at a time
    • Example: Walkie-talkies - users take turns to speak
    • Requires turn-taking protocol
  3. Full-duplex Transmission:
    • Simultaneous two-way communication
    • Example: Telephone conversation - both parties can speak at once
    • Most efficient for interactive communication

    • Token-passing protocol
    • Nodes transmit only when they have the token
    • No collisions
    • Deterministic access

Question 6

(a) (i) List four types of OSPF protocol routers.

(ii) Explain one function of each of the following Internet protocols:

  • (I) SMTP;
  • (II) FTP.

(b) Figure 1 shows a frame format of IEEE 802.3. Use it to answer the question that follows:

(i) Identify the fields labelled (i), (ii), and (iii).

(ii) Explain the function of the field labelled "Frame Check sequence".

(c) Explain the importance of the following in data communication:

  • (i) line coding;
  • (ii) scrambling.

(d) Describe the function of the following network devices:

  • (i) gateway;
  • (ii) switch.

Answer

(a)

(i) Types of OSPF routers:

  1. Internal Routers
  2. Area Border Routers (ABR)
  3. Backbone Routers
  4. Autonomous System Boundary Routers (ASBR)

(ii) Internet protocol functions:

  • SMTP (Simple Mail Transfer Protocol): Used for sending email messages between servers.
  • FTP (File Transfer Protocol): Used for transferring files between a client and a server.

(b)

(i) IEEE 802.3 Frame Format fields:

  • (i) Preamble
  • (ii) Start Frame Delimiter (SFD)
  • (iii) Destination Address

(ii) Frame Check Sequence (FCS):

  • Error detection code used to verify the integrity of the frame.
  • Detects errors introduced during transmission.

(c)

Importance in data communication:

  • Line Coding: Converts digital data into a suitable signal for transmission over a physical medium.
  • Scrambling: Prevents long sequences of 0s or 1s, ensuring clock synchronization and reducing DC components.

(d)

Network device functions:

  • Gateway: Connects networks with different protocols, enabling communication between them.
  • Switch: Forwards data packets between devices within a network, improving efficiency and reducing collisions.

(a) List four examples of Network Operating System software.

(b) Differentiate between \(BCC\) and \(CC\) as used in email programs.

(c) (i) Define the following terms as used in the Internet:

  • (I) hypertext;
  • (II) cookie;
  • (III) chat;
  • (IV) website.

(ii) Explain two uses of antivirus software.

(d) Figure 2 shows a coaxial cable used for data communication. Use it to answer the questions that follow.

  • (i) Identify the parts labelled (i), (ii), (iii) and (iv)
  • (ii) Explain two challenges experienced while using the cable.

Answer

(a)

Examples of Network Operating System software:

  1. Windows Server
  2. Linux
  3. macOS Server
  4. UNIX

(b)

BCC vs. CC:

  • CC (Carbon Copy): Sends a copy of the email to the specified recipients, and all recipients can see who else received the email.
  • BCC (Blind Carbon Copy): Sends a copy of the email to the specified recipients, but the other recipients cannot see who else received the email.

(c)

(i) Definitions:

  • Hypertext: Text that contains links to other documents or resources.
  • Cookie: A small piece of data stored on a user's computer by a website to remember information about the user.
  • Chat: Real-time text-based communication between two or more people over the Internet.
  • Website: A collection of related web pages, images, videos, and other digital assets that are hosted on a web server and accessible through a web browser.

(ii) Uses of antivirus software:

  • Detecting and removing malware
  • Preventing malware infections

(d)

(i) Parts of the coaxial cable:

  • (i) Outer jacket
  • (ii) Shielding
  • (iii) Insulation
  • (iv) Conductor

(ii) Challenges of using coaxial cable:

  • Limited bandwidth compared to fiber optic cable
  • Susceptible to interference
