Question 1 (marks)

(a) Explain three advantages of extranet networks in organizations.

(b) A network design should meet several user needs. Outline four examples of such needs.

(c) Differentiate between information security and network infrastructure security.

(d) Routing protocols use routing tables to establish the source and destination of data packets in transit. Explain three routing path determination methods used by these protocols.

Answer

(a) Three advantages of extranet networks in organizations:

  1. Enhanced collaboration with business partners - allows secure sharing of data and resources with external stakeholders
  2. Cost reduction - reduces communication and transaction costs by digitizing business processes
  3. Improved customer service - provides customers with self-service access to information and support

(b) Four examples of user needs in network design:

  1. Performance - adequate speed and response time
  2. Reliability - minimal downtime and data loss
  3. Scalability - ability to grow with increasing users/data
  4. Security - protection of sensitive data and resources

(c) Difference between information security and network infrastructure security:

  • Information security focuses on protecting data confidentiality, integrity and availability regardless of where it resides
  • Network infrastructure security focuses specifically on protecting network devices, connections and protocols that transport data

(d) Three routing path determination methods:

  1. Static routing - manually configured fixed routes
  2. Dynamic routing - automatically updated routes based on network conditions
  3. Default routing - catches traffic with no specific route defined

Question 2 (marks)

(a)

  1. Outline two characteristics of a point to point protocol (PPP).
  2. Distinguish between network and host addresses as used in IP sub-netting.

(b) With the aid of a diagram, describe a time-division multiplexing (TDM).

(c) Copper cables offer different throughputs in data communication. Outline four factors that influence the cables'.

(d) With aid of a diagram, describe TCP handshake.

Answer

(a)(i) Two characteristics of PPP:

  1. Provides connection authentication
  2. Supports multiple network layer protocols

(ii) Distinction between network and host addresses:

  • Network address identifies the network segment/subnet (network portion of IP)
  • Host address identifies specific device within that network (host portion of IP)

(b) Time-division multiplexing (TDM):

TDM is a method where multiple data streams share a medium by allocating different time slots to each stream. Each source gets the full bandwidth for its designated time slot. The multiplexer switches between inputs in a round-robin fashion.

(c) Four factors influencing copper cable throughput:

  1. Cable length - longer distances reduce throughput
  2. Signal interference/noise - affects signal quality
  3. Cable quality/category - higher categories support better throughput
  4. Environmental factors - temperature, humidity affect performance

(d) TCP handshake:

TCP three-way handshake consists of:

  1. SYN: Client sends synchronization packet
  2. SYN-ACK: Server acknowledges and sends its synchronization
  3. ACK: Client acknowledges server's synchronization

Question 3 (marks)

(a) Explain two types of cookies as used in the Internet.

(b) Joan was requested to patch communication software in her company's computers. Explain two reasons that could have influenced the request.

(c) Use of password expiration features in data communication systems is unpopular. Explain two limitations of this feature.

(d)

  1. Wireless networks are commonly used in business firms. Outline four benefits of the network.
  2. Explain two types of topology diagrams used in a local area network (LAN).

Answer

(a) Two types of cookies:

  1. Session cookies - temporary cookies that expire when browser closes
  2. Persistent cookies - stored on device and remain until deleted or expiration date

(b) Two reasons for software patching:

  1. Security vulnerability fixes - patches address discovered security threats
  2. Bug fixes - patches correct software errors and improve stability

(c) Two limitations of password expiration features:

  1. Users tend to choose simple variations of old passwords
  2. Frequent changes increase likelihood of users writing down passwords

(d)(i) Four benefits of wireless networks:

  1. Mobility - users can connect from anywhere within range
  2. Easy installation - no need for physical cable infrastructure
  3. Scalability - easy to add new devices
  4. Cost-effective - reduces cabling costs and maintenance

(ii) Two types of LAN topology diagrams:

  1. Physical topology - shows actual physical layout and connections
  2. Logical topology - shows data flow and network addressing

Question 4 (marks)

(a) Outline two limitations of CSMA/CD.

(b) Figure 1 shows an Ethernet frame. Explain the functions of each of the parts labelled (i) and (ii).

(c) Star topology is very popular in data communication systems. Outline four advantages that could justify the statement.

(d)

  1. Describe two types of user interfaces in network operating systems. (4 marks)
  2. Troubleshooting a network involves several stages. Explain three generic stages used by network administrators.

Answer

(a) Two limitations of CSMA/CD:

  1. Performance degrades with heavy network traffic due to collisions
  2. Limited effective distance due to collision detection timing requirements

(b) Functions of Ethernet frame parts:

  • Part (i) Preamble: Synchronizes receiver clock and indicates frame start
  • Part (ii) CRC: Cyclic Redundancy Check - detects transmission errors in the frame

(c) Four advantages of star topology:

  1. Easy troubleshooting - faults can be isolated quickly
  2. High reliability - failure of one node doesn't affect others
  3. Easy expansion - new devices can be added easily
  4. Centralized management - network control from central hub/switch

(d)(i) Two types of user interfaces in network operating systems:

  1. Command Line Interface (CLI) - text-based interface for command input
  2. Graphical User Interface (GUI) - visual interface with icons and menus

(ii) Three generic stages of network troubleshooting:

  1. Problem identification - gathering symptoms and user reports
  2. Problem isolation - narrowing down the possible causes
  3. Problem resolution - implementing and testing solutions

Question 5 (marks)

(a)

  1. Outline any three layers of TCP/IP model.
  2. Distinguish between router and bridge as used in networking.

(b)

  1. Outline three factors, other than cost, that should be considered when selecting a fibre optic media.
  2. Network administrators use several documents when troubleshooting a network. Describe two documents commonly used.

(c) Deno was required to troubleshoot a physical layer of a network for a client. Explain three possible symptoms that he could have realized.

Answer

(a)(i) Three layers of TCP/IP model:

  1. Application Layer - handles high-level protocols (HTTP, FTP, SMTP)
  2. Transport Layer - provides end-to-end communication (TCP, UDP)
  3. Network Layer - handles addressing and routing (IP)

(ii) Distinction between router and bridge:

  • Router operates at network layer (Layer 3), uses IP addresses for routing between different networks
  • Bridge operates at data link layer (Layer 2), uses MAC addresses to connect network segments within same network

(b)(i) Three factors for selecting fiber optic media:

  1. Distance requirements - maximum length needed
  2. Bandwidth requirements - data capacity needed
  3. Environmental conditions - exposure to interference/harsh conditions

(ii) Two documents used in network troubleshooting:

  1. Network topology diagram - shows network layout and connections
  2. Network baseline documentation - shows normal network performance metrics

(c) Three possible physical layer symptoms:

  1. No link light on network interface
  2. Intermittent connectivity
  3. Poor signal quality or high error rates

  2. DSL (Digital Subscriber Line)

  4. Satellite

Question 7 (marks)

(a) Difference between contention-based and controlled media access methods. (4 marks)

(b) Describe three functions of presentation layer as used in OSI.

(c) IP protocol is phenomenon when configuring a network. Explain three of its general features.

(d) Most corporate prefers distributed network operating systems. Explain two benefits that could be influencing this trend. (4 marks)

Answer

(a) Difference between contention-based and controlled media access methods:

  • Contention-based methods: Devices compete for access to the medium, with potential collisions (e.g., CSMA/CD). No central coordination.
  • Controlled access methods: Access to medium is coordinated, with devices taking turns based on a predetermined scheme (e.g., token passing). Eliminates collisions.

(b) Three functions of presentation layer in OSI:

  1. Data encryption/decryption for security
  2. Data compression/decompression for efficient transmission
  3. Data format translation between different systems

(c) Three general features of IP protocol:

  1. Connectionless service - packets routed independently
  2. Best effort delivery - no guarantee of packet arrival
  3. Media independent - works across different network types

(d) Two benefits of distributed network operating systems:

  1. Improved reliability through redundancy - if one server fails, others continue operation
  2. Better resource utilization - workload can be balanced across multiple servers

Question 8 (marks)

(a) Explain two functions of network interface card in a computer.

(b)

  1. Outline two types of transmission impairments.
  2. Distinguish between network control and link control protocols as used in Ppp.

(c) Cloud computing is an emerging technology in business firms. Explain two types of clouds that could be used.

(d) Explain three circumstances that would necessitate the implementation of frame relays in corporate networks.

Answer

(a) Two functions of network interface card:

  1. Physical addressing - provides unique MAC address for network identification
  2. Data framing - prepares data for transmission over the network medium

(b)(i) Two types of transmission impairments:

  1. Attenuation - weakening of signal strength over distance
  2. Noise - unwanted signals that interfere with data transmission

(ii) Distinction between network control and link control protocols in PPP:

  • Network Control Protocol (NCP) - handles configuration and negotiation of network layer protocols
  • Link Control Protocol (LCP) - manages link establishment, configuration, and testing

(c) Two types of cloud computing:

  1. Public cloud - services provided over public internet by third-party providers
  2. Private cloud - dedicated cloud infrastructure for a single organization

(d) Three circumstances for implementing frame relay:

  1. Need for cost-effective WAN connectivity between multiple sites
  2. Requirement for flexible bandwidth allocation (committed information rate)
  3. Applications requiring reliable data transfer with minimal overhead
