Question 1

(a) (i) List three criteria that could be used to classify data communication networks.

(ii) Distinguish between a powered-over-Ethernet (PoE) and a non-PoE switch.

(b) (i) Outline three factors that should be considered when selecting a switch, other than cost.

(ii) Bottom-up troubleshooting approach could be used by network administrators to establish a network fault. Outline four disadvantages they are likely to realize while using this approach.

(c) Implementation of any network design should follow several design principles. Describe three principles that should be adhered to.

(d) (i) Outline two categories of wireless networks.

(ii) Explain the term encapsulation as applied to transport protocols.

Answer

(a) (i) Criteria to classify data communication networks:

  • Geographical Coverage:
    • Local Area Network (LAN): Covers a small area like an office or home.
    • Metropolitan Area Network (MAN): Covers a city or metropolitan region.
    • Wide Area Network (WAN): Covers a large geographical area, potentially global.
  • Network Topology:
    • Star: Central node connects all other nodes.
    • Ring: Each node connects to exactly two other nodes, forming a ring.
    • Bus: All nodes connect to a single cable.
    • Mesh: Nodes are interconnected with many redundant connections between network nodes.
  • Transmission Technology:
    • Wired: Uses cables like Ethernet or fiber optic.
    • Wireless: Uses radio waves, microwaves, or infrared signals.

(a) (ii) PoE vs Non-PoE Switch:

  • PoE (Power over Ethernet) Switch:
    • Provides electrical power along with data over Ethernet cables.
    • Supports devices like IP phones, wireless access points, and IP cameras without needing separate power supplies.
  • Non-PoE Switch:
    • Only transmits data.
    • Connected devices require separate power sources.

(b) (i) Factors to consider when selecting a switch (other than cost):

  • Port Density and Speed:
    • Number of ports needed to connect devices.
    • Speed requirements (e.g., Gigabit Ethernet).
  • Management Capabilities:
    • Support for VLANs, QoS, and SNMP.
    • Web-based or command-line interface for configuration.
  • Scalability:
    • Ability to add more switches or stack them.
    • Support for future expansion.

(b) (ii) Disadvantages of bottom-up troubleshooting:

  • Time-Consuming:
    • Checking each component from the physical layer upwards can be lengthy.
  • May Miss Systemic Issues:
    • Focusing on individual components may overlook broader network problems.
  • Requires Extensive Knowledge:
    • Needs expertise at each layer of the OSI model.
  • Inefficient for Complex Problems:
    • Not suitable for diagnosing complex, multi-layered issues.

(c) Network design principles:

  • Hierarchical Design:
    • Implementing core, distribution, and access layers.
  • Modularity:
    • Using modular components for easy scaling and maintenance.
  • Redundancy:
    • Including backup paths and components for fault tolerance.

(d) (i) Categories of wireless networks:

  • Wireless Local Area Network (WLAN):
    • Covers a limited area, such as a home, office, or campus.
  • Wireless Personal Area Network (WPAN):
    • Covers a very small area, typically within a few meters.

(d) (ii) Encapsulation:

  • Encapsulation:
    • Process of adding headers and trailers to data as it moves down the protocol stack.

Question 2

(a) (i) Distinguish between blogs and portals as used in the Internet.

(ii) Figure 1 shows a TCP/IP termination process. Identify appropriate descriptor for each process labelled (i), (ii), (iii) and (iv).

(b) CFG polytechnic intends to install a wireless network that uses radio frequency for transmission: Outline four characteristics of the signal in the network.

(c) Farasi College has introduced Internet services to its students. Explain two challenges that the college is likely to experience.

Answer

(a) (i) Blogs vs Portals:

  • Blogs:
    • Personal or professional websites with chronological entries.
  • Portals:
    • Gateway websites providing organized access to various services and resources.

(a) (ii) TCP/IP termination process descriptors:

  • (i) FIN:
    • Initiates connection termination.
  • (ii) ACK:
    • Acknowledgment of the FIN segment.
  • (iii) FIN:
    • Second FIN segment to complete termination.
  • (iv) ACK:
    • Final acknowledgment.

(b) Characteristics of radio frequency signals:

  • Frequency:
    • Operating range in the electromagnetic spectrum.
  • Amplitude:
    • Signal strength and power.
  • Wavelength:
    • Distance between signal peaks.
  • Phase:
    • Timing relationship between signals.

(c) Challenges in providing Internet services:

  • Bandwidth Limitations:
    • Insufficient bandwidth for all students.
  • Security Threats:
    • Unauthorized access to network resources.

Question 3

(a) Explain each of the following terms as used in the Internet:

(i) cookies;

(ii) history.

(b) Network operating systems provide critical functions in data communication systems. Outline four typical functions that are likely to be realized.

(c) Logical security measures are normally recommended approaches of preventing network insecurity incidences. Explain two such measures.

(d) (i) Explain this two sub layers of the data link layer used by LAN.

(ii) Every network topology requires careful considerations before installation. Outline four factors that should consider before the installation, other than cost.

Answer

(a) (i) Cookies:

  • Definition:
    • Small text files that websites store on a user's computer.
  • Purpose:
    • Remember information about the user, such as preferences and login details.
  • Functionality:
    • Help websites recognize returning users and customize content.

(a) (ii) History:

  • Definition:
    • A list of websites that a user has visited.
  • Purpose:
    • Allows users to easily revisit previously accessed pages.
  • Functionality:
    • Maintained by web browsers for easy navigation and tracking.

(b) Network operating system functions:

  • File and Printer Sharing:
    • Allows users to share files and printers across the network.
  • User Authentication and Access Control:
    • Manages user accounts and permissions.
  • Network Resource Management:

    • Allocates and manages network resources such as bandwidth and storage.
  • Protocol Support and Network Services:

    • Supports various network protocols and services such as DNS and DHCP.

(c) Logical security measures:

  • Firewalls:

    • Control network traffic based on predefined rules.
  • Intrusion Detection Systems (IDS):

    • Monitor network traffic for suspicious activity.

(d) (i) Sublayers of the data link layer:

  • Logical Link Control (LLC):

    • Provides an interface to the network layer.
  • Media Access Control (MAC):

    • Controls access to the physical medium.

(d) (ii) Factors for network topology considerations:

  • Scalability:

    • Ease of adding new devices to the network.
  • Reliability:

    • Fault tolerance and redundancy.
  • Security:

    • Access control and data protection.
  • Manageability:

    • Ease of configuration and troubleshooting.

Question 4

(a) (i) State the function of the network layer in the OSI model.

(ii) Describe the function of the following network devices:

  1. Hub;
  2. Repeater.

(b) Explain three reasons for using fibre optic cables in data communication.

(c) (i) State the difference between IPv4 and IPv6.

(ii) Convert the IPv4 address 192.168.1.10 into binary form.

(d) Explain the importance of each of the following in data communication:

  1. Data integrity;
  2. Data security.

Answer

(a) (i) Function of the network layer:

  • Function:

    • Provides routing and forwarding of data packets between different networks.

(a) (ii) Network devices:

  • I. Hub:

    • A simple device that connects multiple Ethernet devices together, acting as a common connection point for devices in a network.
  • II. Repeater:

    • A device that amplifies or regenerates an incoming signal before retransmitting it, extending the distance over which a signal can be transmitted.

    • (b) Reasons for using fibre optic cables:

    • High Bandwidth:

      • Fibre optic cables can carry much more data than traditional copper cables.
  • Low Signal Loss:

    • Signals can travel longer distances without significant degradation.
  • Immunity to Electromagnetic Interference:

    • Fibre optic cables are not affected by EMI, ensuring reliable data transmission.

    • (c) (i) IPv4 vs IPv6:

    • IPv4:

      • 32-bit address space, limiting the number of unique addresses.
  • IPv6:

    • 128-bit address space, providing a vastly larger number of unique addresses.

    • (c) (ii) IPv4 address in binary form:

    • 192.168.1.10:

      • 11000000.10101000.00000001.00001010

    (d) Importance in data communication:

    • I. Data integrity:

      • Ensures that data remains accurate and consistent throughout its lifecycle.
  • II. Data security:

    • Protects data from unauthorized access and modification, maintaining confidentiality and trust.

    • Question 5

    (a) (i) State the Nyquist theorem.

    (ii) A network has a bandwidth of 10 MHz. If the lowest signal is 10 MHz, determine the highest signal.

    (b) Explain the importance of the following transmission media in data communication:

    1. Coaxial cable;
    2. Microwave.

    (c) (i) State the function of a modem in data communication.

    (ii) Explain the difference between synchronous and asynchronous data transmission.

    (d) (i) State two functions of a gateway in data communication.

    (ii) Explain the difference between TCP and UDP protocols.

    Answer

    (a) (i) Nyquist theorem:

    • Nyquist theorem:

      • States that the sampling rate must be at least twice the highest frequency component of the signal to accurately reconstruct the signal.

    (a) (ii) Highest signal:

    • Highest signal:

      • Given bandwidth = 10 MHz and lowest signal = 10 MHz, the highest signal is 20 MHz.

    (b) Importance of transmission media:

    • I. Coaxial cable:

      • Provides a shielded transmission medium for reliable data communication, commonly used for cable TV and Ethernet networks.
  • II. Microwave:

    • Enables wireless communication over long distances, used in cellular networks and satellite communication.

    • (c) (i) Function of a modem:

    • Function of a modem:

      • Modulates and demodulates signals to enable data transmission over telephone lines or other communication channels.

    (c) (ii) Synchronous vs Asynchronous data transmission:

    • Synchronous:

      • Data is transmitted in continuous streams with precise timing, requiring a clock signal for synchronization.
  • Asynchronous:

    • Data is transmitted in individual characters or bytes with start and stop bits, not requiring a continuous clock signal.

    • (d) (i) Functions of a gateway:

    • Functions of a gateway:

      • Connects different networks with different protocols, enabling communication between them.
      • Performs protocol conversion and address mapping.

    (d) (ii) TCP vs UDP protocols:

    • TCP:

      • A connection-oriented protocol that provides reliable, ordered, and error-checked delivery of data.
  • UDP:

    • A connectionless protocol that provides fast, but unreliable delivery of data.

    • Question 6

    (a) (i) State the meaning of the term multiplexing as used in data communication.

    (ii) Outline three advantages of using multiplexing techniques in data transmission.

    (b) (i) State the meaning of the term protocol as used in data communication.

    (ii) Explain the function of each of the following protocols:

    1. HTTP;
    2. SMTP.

    (c) (i) State the meaning of the term network topology.

    (ii) Describe the following network topologies:

    1. Star topology;
    2. Ring topology.

    (d) (i) State the meaning of the term computer network security.

    (ii) Outline three measures that can be used to enhance computer network security.

    Answer

    (a) (i) Meaning of multiplexing:

    • Multiplexing:

      • A technique that combines multiple signals into one signal over a shared medium.

    (a) (ii) Advantages of multiplexing:

    • Efficient Use of Bandwidth:

      • Maximizes the use of available bandwidth by transmitting multiple signals simultaneously.
  • Reduced Infrastructure Costs:

    • Reduces the number of physical links required for data transmission.
  • Increased Network Capacity:

    • Enhances the capacity of the network to handle more data traffic.

    • (b) (i) Meaning of protocol:

    • Protocol:

      • A set of rules and procedures that govern the transmission of data in a network.

    (b) (ii) Function of protocols:

    • I. HTTP:

      • Used for transferring web pages and other content between a web server and a web browser.
  • II. SMTP:

    • Used for sending email messages between email servers.

    • (c) (i) Meaning of network topology:

    • Network topology:

      • The arrangement of the elements (links, nodes, etc.) of a communication network.
