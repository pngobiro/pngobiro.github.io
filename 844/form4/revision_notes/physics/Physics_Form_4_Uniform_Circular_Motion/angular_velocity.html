Angular Velocity

  • If a body moving in a circular path turns through an angle θ radians in time ‘t’, we define angular velocity omega (ω), as the rate of change of the angular displacement ω = θ/t,
  • The SI unit of angular velocity is radians per second (rads-1).
  • Since the radian measure is a ratio we can write it as second-1 (s-1).

    Relation between linear velocity and angular velocity
